# 海关清关与合规技能

# TL;DR

**天猫国际清关的核心是在1210保税备货与9610直邮两条路径中选择最适合的通关模式，通过提前完成商品备案、准确归类HS编码、备齐报关单证，配合三单对碰机制，实现高效合规通关，同时建立清关异常分级响应体系以应对查验、退运等突发情况。**

---

> 适用渠道：天猫国际（Tmall Global）
> 适用角色：天猫国际商家运营、供应链专员、合规专员
> 适用场景：新商家入驻、选品合规、大促备货、清关异常处置

---

## 一、核心概念速览

### 1.1 什么是清关？

清关（Customs Clearance），即货物从境外进入中国境内时，向海关申报、接受监管、缴纳税费并获得放行的全过程。对于天猫国际商家而言，清关是跨境商品进入消费者手中的必经关卡。

### 1.2 天猫国际清关的特殊性

天猫国际采用 **1210 保税备货模式** 和 **9610 直邮模式** 两种主流清关路径：

| 模式 | 全称 | 适用场景 | 时效 | 税费特点 |
|------|------|----------|------|----------|
| **1210 保税备货** | 保税跨境贸易电子商务 | 大促备货、热销标品 | 较快（仓内清关） | 先备货后申报，批次清关 |
| **9610 直邮** | 跨境贸易电子商务 | 非标品、小批量 | 较慢（单件清关） | 单件申报，逐票清关 |

> ⚠️ **关键提示**：选择哪种模式直接影响备货成本和清关时效。大促期间优先推荐 1210 模式，可有效缓解峰值通关压力。

## 二、跨境电商进口清关全流程

### 2.1 标准清关流程（1210 保税备货模式）

```
境外发货 → 国际运输 → 口岸入境 → 保税仓入库
→ 订单生成 → 申报清单 → 海关审核 → 放行
→ 国内配送 → 消费者签收
```

**Step 1：境外发货**
- 商家境外仓库根据预测销量备货
- 组柜装箱，贴好唛头和箱单

**Step 2：国际运输**
- 海运/空运/铁运至中国口岸
- 常用口岸：上海、宁波、广州、杭州（菜鸟保税仓）

**Step 3：口岸入境**
- 货代向海关申报进境货物舱单
- 换取提货单（D/O）

**Step 4：保税仓入库**
- 货物进入保税物流中心（B型）
- 仓库接收、上架，库存数据同步至海关系统

**Step 5：订单生成**
- 消费者在天猫国际下单
- 订单信息实时推送至海关系统（支付单、物流单、订单三单对碰）

**Step 6：申报清单**
- 电商平台推送申报清单（包含商品信息、价格、数量、收件人等）
- 支付企业推送支付单
- 物流企业推送物流单
- **三单对碰**：三单信息匹配一致后方可申报

**Step 7：海关审核**
- 海关对申报信息进行审核
- 系统自动审核（95%以上）+ 人工审核（抽检）

**Step 8：放行与配送**
- 审核通过，海关系统下达放行指令
- 仓库拣货出库，快递配送至消费者

### 2.2 直邮模式清关流程（9610 模式）

```
境外仓库 → 国际小包/专线 → 邮政/商业快件渠道
→ 海关监管场所 → 电子申报 → 核验放行 → 国内派送
```

**核心差异**：9610 模式下，每一票订单单独申报，适用小批量、多SKU 的非标品。

## 三、商品备案与准入要求

### 3.1 商品备案的重要性

在天猫国际上架进口商品，**必须在海关完成备案**，否则商品无法正常清关。备案是清关的前置条件。

### 3.2 备案类型

| 备案类型 | 适用商品 | 备案主体 | 有效期 |
|----------|----------|----------|--------|
| **首次进口备案** | 首次在中国申报的进口食品/化妆品/保健品 | 境外生产企业或境内进口商 | 长期有效（变更需重新备案） |
| **跨境电商备案** | 通过跨境电商渠道进口的正面清单内商品 | 电商平台 + 物流企业 + 支付企业 | 与企业经营周期一致 |
| **原产地备案** | 需享受协定税率的商品（如RCEP成员国） | 进口商 | 随货物批次 |

### 3.3 正面清单管理

跨境电商进口商品须在 **《跨境电子商务零售进口商品清单（正面清单）》** 范围内。目前清单涵盖：

- 母婴用品（奶粉、纸尿裤、奶瓶等）
- 化妆品（护肤、彩妆，需具备国家药监局备案凭证）
- 食品（保健品、配方奶粉、零食等）
- 服饰鞋包
- 数码电器（部分，需3C认证）
- 家居用品

> 🔴 **禁区**：以下商品 **不在正面清单范围内**，**禁止通过跨境电商渠道进口**：
> - 仿真枪械、管制刀具
> - 乳制品（生乳、鲜乳除外，配方奶粉在清单内）
> - 旧服装、旧电子产品
> - 部分药品（如精神类药品）
> - 未经准入的境外农产品

### 3.4 化妆品特殊要求

化妆品是天猫国际的核心品类，备案要求尤为严格：

**普通化妆品**：需在国家药监局完成 **进口非特殊用途化妆品备案**（备案号格式：`津妆/沪妆/粤妆XXXXXXXX号`）

**特殊用途化妆品**：需申请 **进口特殊用途化妆品行政许可批件**（批件号格式：`国妆特进字JXXXXXXXX`）

> ⚠️ **风险提示**：2021年起，进口化妆品全面实施 **功效宣称评价**，商家需在商品详情页展示功效依据，否则面临下架风险。

### 3.5 婴幼儿配方奶粉要求

- 须持有 **境外生产企业注册号**（海关总署注册）
- 须在配方注册有效期 内（有效期5年）
- 外包装须加贴中文标签，且内容须与注册配方一致
- 禁止标注"最接近母乳"等功效性宣传语

## 四、报关操作要点

### 4.1 报关单证体系

清关过程中需要准备以下核心单证：

| 单证类型 | 出具方 | 用途 |
|----------|--------|------|
| **发票（Invoice）** | 境外卖方 | 申报商品价格，海关审价依据 |
| **装箱单（Packing List）** | 境外卖方 | 货物数量、重量、包装信息 |
| **提单（B/L）或空运单（AWB）** | 船公司/航空公司 | 货物物权凭证 |
| **原产地证（CO/Form E/Form RCEP等）** | 境外官方机构 | 享受协定关税税率 |
| **进口货物报关单** | 报关行/商家 | 向海关正式申报 |
| **商品编码（HS Code）** | 商家/报关行 | 海关商品归类，确定税率 |
| **检疫证书/卫生证书** | 生产国官方 | 部分食品/动植物产品必需 |

### 4.2 HS 编码归类技巧

HS 编码是清关的核心，它决定了：

- 进口税率
- 是否需要特殊许可证件
- 是否需要3C认证/检疫审批

**常见天猫国际商品 HS 编码参考：**

| 商品 | HS 编码 | 关税税率 | 备注 |
|------|---------|----------|------|
| 婴幼儿配方奶粉 | 19011010 | 0% | 跨境电商综合税 9.1% |
| 纸尿裤 | 96190010 | 0% | 跨境电商综合税 9.1% |
| 精华液（化妆品） | 33039900 | 1% | 跨境电商综合税 25% |
| 运动鞋 | 64041100 | 12% | 跨境电商综合税 25% |
| 扫地机器人 | 85081100 | 0% | 需3C认证 |

> ⚠️ **重要**：HS 编码归类错误会导致 **税差损失** 或 **通关延误**，建议商家与专业报关行确认编码。

### 4.3 海关审价机制

海关可能对申报价格提出质疑，常见原因：

- 申报价格明显低于市场行情
- 存在关联交易（关联公司之间的定价）
- 特许权使用费未计入完税价格

**应对策略**：
1. 提供完整的交易链条证明（合同、发票、付款记录）
2. 提供同类商品的市场价格参考
3. 如涉及品牌授权，提供授权协议和特许权使用费说明

### 4.4 税费计算

跨境电商进口综合税计算公式：

**综合税 = 购买单价 × 数量 × 综合税率**

| 商品类别 | 综合税率 | 优惠税率（经认证） |
|----------|----------|-------------------|
| 奶粉/尿裤/部分食品 | 9.1% | 0% |
| 化妆品（高档） | 50% | 25% |
| 化妆品（普通） | 25% | 11.2% |
| 服饰/鞋/包 | 25% | 11.2% |
| 数码电器 | 13% | 13% |

> 💡 **省税技巧**：同一订单商品尽量选择同一税率档次，避免高税率商品拉高整单税费。

## 五、清关异常处理

### 5.1 常见清关异常类型

| 异常类型 | 表现 | 常见原因 |
|----------|------|----------|
| **查验** | 货物被海关扣留等待检查 | 随机布控 / 商品信息存疑 |
| **价格质疑** | 海关要求提供价格证明材料 | 申报价格偏低 |
| **归类质疑** | 海关对 HS 编码提出异议 | 编码归类存在争议 |
| **单证不全** | 申报时被退单 | 缺少必要许可证件或证书 |
| **退运** | 货物被要求退返境外 | 违禁品 / 无法补齐证件 |
| **扣货** | 货物被海关扣押 | 涉嫌侵权 / 质量问题 |

### 5.2 查验应对流程

**Step 1：收到查验通知**
- 物流通知 / 海关系统推送查验指令
- 确认查验类型：**机查**（扫描仪检查）或 **人工查**（开箱检查）

**Step 2：准备材料**
- 发票、装箱单、合同
- 产品说明书、成分表
- 原产地证明
- 品牌授权书（如涉及品牌商品）
- 检测报告（针对食品、化妆品）

**Step 3：配合海关**
- 安排人员或报关行现场配合开箱
- 如实回答海关问询，不隐瞒不伪造
- 提供补充说明材料

**Step 4：等待结果**
- 机查：通常1-3个工作日
- 人工查：通常3-7个工作日
- 查验正常 → 放行
- 查验异常 → 可能涉及补税、罚款或退运

> 💡 **加速技巧**：与查验海关保持良好沟通，提供材料越完整，查验时间越短。

### 5.3 退运处理流程

如商品无法在中国境内完成清关，需退运境外：

**前提条件**：海关出具《不准入境决定书》或退运处理决定

**流程**：
1. 向海关申请退运
2. 办理出口申报（退运商品不征税，但需正常报关）
3. 货物出库，运回境外仓库
4. 评估是否更换包装/补齐证件后再次进口

> ⚠️ **成本警示**：退运成本极高（国际运费+仓储费+操作费），建议在发货前充分确认商品合规性。

### 5.4 清关时效优化策略

| 优化方向 | 具体措施 |
|----------|----------|
| **提前备案** | 商品在上架前完成所有备案，避免货到港后等待备案 |
| **三单对碰优化** | 确保订单、支付、物流数据实时同步，避免申报延迟 |
| **预归类预审** | 大促前与海关提前沟通商品归类，减少查验概率 |
| **保税仓选择** | 选择通关效率高的保税仓库（如杭州、宁波） |
| **合规自查** | 定期自查商品信息、价格、标签是否符合最新法规 |

## 六、大促清关专项指南

### 6.1 双11 / 618 清关备战

**提前30天**：完成所有新增SKU的备案；与报关行确认爆品HS编码；与保税仓预约入仓仓位

**提前15天**：完成主力商品批量入仓；与海关确认大促期间通关绿色通道政策

**提前7天**：三单系统压力测试；备足通关保证金；设置清关异常应急预案

**大促期间**：安排专人监控清关状态；设置异常预警阈值（超时未放行自动告警）

### 6.2 爆仓应对

当保税仓库存不足时，可启用 **"前店后仓"** 模式：
- 直邮模式作为补充渠道
- 部分订单转至香港仓发货（再从香港清关进境）
- 与平台协商临时提高保税库存预警水位

## 九、常见问题速答

**Q1：商品已备案，清关时还需要提供备案号吗？**
> ✅ 需要。在申报清单中需填写商品备案编号，系统会自动关联核验。

**Q2：同一商品，不同批次进口税率不同是怎么回事？**
> 可能是原产地不同导致。若享受RCEP等协定税率，需提供对应原产地证；若一般贸易进口则按MFN税率执行。

**Q3：清关被海关查验，会产生额外费用吗？**
> 查验本身不收费，但由此产生的仓储费、滞港费（如货物滞留码头）需商家承担。

**Q4：商品被下架，是否可以继续清关已下单的货物？**
> 原则上已生成订单且完成申报的可以继续清关，但新订单将无法再申报。建议立即处理商品合规问题。

**Q5：进口税率比预期高很多怎么办？**
> 核实HS编码是否正确；若归类无误则需如实缴税；若发现归类错误可向海关申请复核，但需提供充分依据。

# 场景分析

### 典型场景

**场景一：新商家首次入驻——清关全流程认知**
新商家入驻天猫国际，不了解进口清关全链路，需要从零搭建清关体系，包括选择清关模式、完成商品备案、签约报关行和保税仓、对接三单系统。

**场景二：新品上架前——商品合规准入判断**
商家计划引入某品牌化妆品或保健品，需要确认该商品是否在正面清单内、是否需要药监局备案、备案流程和周期如何，规避货到港后无法清关的风险。

**场景三：货物到港后——清关延误处理**
货物已到达中国口岸，但清关进度停滞，需要判断是查验、价格质疑、归类质疑还是单证不全，并快速采取针对性措施。

**场景四：大促备战——批量通关保障**
双11/618大促前，需要在有限时间内完成大量商品入仓通关，涉及批量备案、仓位预约、系统压力测试等协调工作。

### 核心挑战

- **合规不确定性**：正面清单动态调整，商品准入判断存在模糊地带
- **时效压力**：大促期间通关时效要求高，批量货物清关瓶颈明显
- **信息不对称**：商家与海关/报关行之间存在专业壁垒，沟通效率低
- **成本波动**：查验产生的仓储费和滞港费难以预估，影响毛利

### 正确认知

- 清关不是"事后处理"，而是"前置规划"——商品备案应在选品阶段完成
- 1210保税模式适合爆款标品，9610直邮适合非标长尾，两者可以并行使用
- 查验本身不收费，但延误的时效成本和资金占用成本巨大
- 三单对碰（订单、支付单、物流单）是跨境电商清关的核心数据机制，不容忽视

# 决策框架

### Step 1：确定进口模式

**决策节点**：商品特征 + 备货量 + 时效要求

```
热销标品 + 大批量备货 + 大促备战 → 优先1210保税模式
非标品 + 小批量 + 长尾SKU + 新品测试 → 优先9610直邮模式
高价值商品（客单价>800元）+ 低量试销 → 9610直邮（降资金占用）
```

### Step 2：商品合规准入检查

**决策节点**：正面清单 + 备案要求 + 许可证件

```
Step 2.1：确认商品是否在《跨境电商零售进口正面清单》内
  ├─ 不在清单内 → 该商品无法通过跨境电商渠道进口，考虑一般贸易
  └─ 在清单内 → 继续Step 2.2

Step 2.2：确认商品类目对应的备案/许可要求
  ├─ 化妆品 → 药监局备案（普通/特殊用途）
  ├─ 婴幼儿配方奶粉 → 配方注册 + 境外生产企业注册
  ├─ 食品/保健品 → 进口食品境外生产企业注册
  ├─ 3C类产品 → 3C认证
  └─ 其他品类 → 核查是否需要检疫/进口许可

Step 2.3：确认HS编码和进口税率
  └─ 提前与报关行确认，避免归类争议
```

### Step 3：报关单证准备

**决策节点**：单证完整性 + 准确性

```
基础单证：发票、装箱单、提单/空运单、合同
特殊单证（按需）：原产地证、品牌授权书、检测报告
├─ 原产地为RCEP成员国 → 准备Form R，享受协定税率
└─ 其他 → 一般MFN税率
```

### Step 4：清关渠道选择与报关行对接

**决策节点**：成本 + 时效 + 通关历史

```
保税仓选择：优先通关效率高的仓库（杭州、宁波、上海）
报关行选择：查看历史通关时效、查验率、处理异常能力
```

### Step 5：通关状态监控与异常处理

**决策节点**：通关状态 + 异常类型

```
正常放行 → 继续履约配送
├─ 收到查验通知 → 立即准备全套单证，配合海关，优先机查通道
├─ 价格质疑 → 提供采购发票、价格说明、市场行情参考
├─ 归类质疑 → 提供产品说明书、成分表，申请复核
├─ 单证不全 → 补齐材料，重新申报
└─ 退运通知 → 评估退运成本，与报关行商议替代方案
```

# 专业建议

### 策略层：构建合规前置体系

1. **建立商品合规准入SOP**：将备案前置到选品阶段，选品必查正面清单和备案要求，避免选品完成后再发现无法清关
2. **建立供应商资质清单**：对每个供应商的境外生产企业注册状态、配方注册有效期进行追踪，设置到期预警
3. **双通道并行策略**：爆款使用1210保税模式，非标长尾使用9610直邮，两者互补而非互斥
4. **定期合规自查机制**：每季度对在售商品进行一次合规扫描，检查备案是否有效、标签是否符合最新法规

### 执行层：提升通关效率

1. **预归类预审**：大促前与海关预沟通爆品的HS编码归类，减少查验概率（提前30天启动）
2. **三单系统稳定性保障**：与大仓、支付机构、物流企业做系统联调，确保峰值下单时三单对碰不卡壳
3. **备足通关保证金**：大促前在保税仓账户预存足够保证金，避免峰值期间因保证金不足导致通关延误
4. **建立应急报关行资源池**：除了主报关行，建立2-3家备用报关行联系方式，遇到主报关行处理能力不足时可以快速切换

### 风险层：异常处置准备

1. **查验应急预案**：提前准备产品说明书（中英文）、成分表、品牌授权书PDF，随时可提供给海关
2. **价格争议应对**：与供应商签订的价格协议（Commercial Agreement）留存，作为申报价格的支撑文件
3. **退运成本测算表**：对每个SKU提前测算退运成本，作为是否冒险重运的决策依据
4. **建立海关关系档案**：记录与各口岸海关沟通的历史，便于后续快速找到对接人

# 质量检查

### 执行前检查（Pre-Clearance Checklist）

- [ ] 商品已确认在《跨境电商零售进口正面清单》范围内
- [ ] 所有需要备案的商品已完成海关/药监局备案，备案号有效
- [ ] HS编码已与报关行确认，税率已核实
- [ ] 发票、装箱单、提单/空运单等基础单证齐全
- [ ] 化妆品已具备药监局备案凭证（备案号或批件号）
- [ ] 婴幼儿配方奶粉具备配方注册号和境外生产企业注册号
- [ ] 1210模式：保税仓仓位已预约，入仓时间已确认
- [ ] 9610模式：海外仓已确认发货，直邮渠道已对接
- [ ] 三单系统（订单/支付单/物流单）联调测试通过

### 执行中检查（In-Process Checklist）

- [ ] 货物入仓后保税仓数据与海关系统同步正常
- [ ] 申报清单信息与订单信息完全一致，无金额/数量偏差
- [ ] 物流状态实时可查，通关节点有记录
- [ ] 异常告警阈值已设置（超时未放行自动通知）

### 执行后检查（Post-Clearance Checklist）

- [ ] 清关完成时间在预期时效内（1210模式≤3天，9610模式≤7天）
- [ ] 无补税/罚款记录，毛利未受意外侵蚀
- [ ] 消费者收货时效符合预期，DSR物流评分未下滑
- [ ] 本次清关经验已沉淀至操作SOP

### 质量评分标准

| 维度 | 优秀（9-10分） | 合格（7-8分） | 需改进（<7分） |
|------|-------------|------------|------------|
| 合规完整性 | 全品类备案齐全，无遗漏 | 主力SKU已备案，长尾遗漏 | 存在未备案商品上架 |
| 通关时效 | 正常放行≤24h | ≤3天 | 超过3天或发生查验 |
| 成本控制 | 零额外税费 | 少量正常关税 | 补税/罚款/退运 |
| 异常处理 | 有预案，24h内解决 | 有处理但耗时较长 | 被动等待无响应 |
