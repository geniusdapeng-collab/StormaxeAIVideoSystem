# TL;DR

拆解天猫国际跨境供应链全链路成本结构，掌握定价方法论，实现商品利润最大化。

# 场景分析

## 典型场景
- 选品阶段：计算某个商品有没有利润空间
- 大促前：制定活动价格，保证利润
- 月度复盘：分析为什么这个月利润下滑
- 新品定价：确定新品的合理价格区间

## 核心挑战
1. **成本项繁多**：采购、物流、关税、平台费、推广费...不知道该算哪些
2. **定价两难**：定高了没销量，定低了没利润
3. **利润模糊**：GMV很高但算下来不赚钱
4. **隐性成本**：算账时发现有很多没考虑到的费用

## 正确认知
- **不是GMV为王，是净利润为王**
- **定价决定生死，成本决定利润**
- **每个成本项都要抠，才能挤出利润**
- **数据是基础：没有准确的成本数据，一切都是空谈**

# 决策框架

## 第一步：识别成本类型

### 固定成本 vs 变动成本

| 类型 | 说明 | 示例 |
|------|------|------|
| 固定成本 | 不随销量变化的成本 | 仓储固定费、人员成本 |
| 变动成本 | 随销量变化的成本 | 物流费、关税、平台佣金 |

### 成本归属
- **直接成本**：可以直接归因到某个商品的
- **间接成本**：需要分摊的共同成本

## 第二步：建立成本模型

### 标准成本表模板

```
商品成本表
├── 商品名称：
├── SKU编码：
└── 成本项：
    ├── 采购成本（CIF价）
    ├── 头程物流
    ├── 关税
    ├── 跨境综合税
    ├── 保税仓仓储费
    ├── 操作费
    ├── 国内配送费
    ├── 平台佣金
    ├── 推广费（可选）
    └── 其他费用
    = 总成本
```

## 第三步：制定定价策略

### 定价公式
```
售价 = 成本 ÷ (1 - 目标毛利率)
```

### 定价考虑因素
1. 成本底线（不能亏本）
2. 市场竞品价格
3. 品牌定位
4. 平台活动要求

## 商品定价策略

### 定价方法

#### 成本定价法
```
售价 = 采购成本 ÷ (1 - 毛利率 - 各项费率之和)
```

适用场景：竞争不激烈、有独特价值的商品

#### 市场定价法
```
参考竞品价格 ± 差异化价值
```

适用场景：竞争激烈的标品

#### 价值定价法
```
售价 = 用户感知价值 - 用户预期
```

适用场景：差异化强、品牌溢价高的商品

### 定价层级

#### 引流款
- 目的：吸引流量、提升排名
- 毛利率：0-5%
- 要求：性价比极高

#### 利润款
- 目的：贡献主要利润
- 毛利率：25-40%
- 要求：差异化明显

#### 形象款
- 目的：提升品牌形象
- 毛利率：50%+
- 要求：高端定位、稀缺性

### 大促定价策略

#### 双11定价要求
| 活动 | 折扣要求 | 建议毛利空间 |
|------|----------|--------------|
| 官方立减 | 15-30% | 10-15% |
| 红包补贴 | 5-10% | 5-8% |
| 跨店满减 | 10-20% | 5-10% |

#### 定价检查清单
- [ ] 活动价是否低于成本？
- [ ] 叠加优惠后是否有利润？
- [ ] 与竞品相比是否有优势？
- [ ] 是否符合品牌定位？

## 盈利能力分析

### 核心指标

#### 毛利率
```
毛利率 = (售价 - 成本) ÷ 售价 × 100%
```

#### 净利率
```
净利率 = (收入 - 所有成本) ÷ 收入 × 100%
```

#### ROI（投资回报率）
```
ROI = 利润 ÷ 投入成本 × 100%
```

### 盈亏平衡分析

#### 盈亏平衡点
```
盈亏平衡销量 = 固定成本 ÷ (单价 - 单位变动成本)
```

#### 利润规划
| 目标利润 | 需要的毛利率 | 说明 |
|----------|--------------|------|
| 保本 | 0% | 仅覆盖成本 |
| 健康利润 | 15-20% | 可持续发展 |
| 高利润 | 25%+ | 投入推广/新品 |

### 盈利能力诊断

#### 常见问题诊断
| 问题 | 症状 | 原因 | 解决方案 |
|------|------|------|----------|
| 成本过高 | 毛利率<10% | 采购/物流议价弱 | 优化供应链 |
| 定价偏低 | 竞品价格更低 | 市场定位问题 | 差异化定价 |
| 费用过高 | 净利率低 | 推广/运营费用高 | 精细化运营 |
| 滞销积压 | 库存周转慢 | 备货不精准 | 优化备货 |

### 盈利能力分析模板

```
月度利润分析
├── 销售收入
├── 成本
│   ├── 采购成本
│   ├── 物流成本
│   ├── 税务成本
│   └── 平台成本
├── 费用
│   ├── 推广费
│   ├── 人员费
│   └── 其他费
├── 毛利润 = 收入 - 成本
├── 净利润 = 毛利润 - 费用
├── 毛利率 = 毛利润 ÷ 收入
└── 净利率 = 净利润 ÷ 收入
```

## 成本优化实战

### 采购成本优化

#### 策略1：源头直采
- 减少中间环节
- 直接对接品牌/工厂
- 成本降低10-20%

#### 策略2：批量采购
- 集中订单
- 阶梯价格
- 成本降低5-15%

#### 策略3：长期合作
- 框架协议
- 年度返点
- 成本降低3-10%

### 物流成本优化

#### 策略1：物流模式组合
| 销量 | 推荐模式 | 成本 |
|------|----------|------|
| <500/月 | 直邮 | 灵活、低风险 |
| 500-2000/月 | 保税+直邮 | 平衡 |
| >2000/月 | 保税为主 | 规模效应 |

#### 策略2：整合头程
- 多SKU拼柜
- 集中发货
- 成本降低10-30%

#### 策略3：快递协议价
- 阶梯运量
- 月结账期
- 成本降低15-25%

### 税务成本优化

#### 策略1：商品结构优化
- 合理搭配高低税率商品
- 控制单次订单金额
- 税负降低5-10%

#### 策略2：保税模式
- 大批量入保税仓
- 综合税负更低
- 税负降低10-20%

### 运营成本优化

#### 策略1：自动化
- 系统自动处理订单
- 减少人工干预
- 效率提升50%+

#### 策略2：精细化推广
- 数据驱动投放
- ROI>1才是有效投放
- 推广费降低20-30%

#### 策略3：退货率控制
- 提升商品质量
- 详情页描述准确
- 退货率<5%
