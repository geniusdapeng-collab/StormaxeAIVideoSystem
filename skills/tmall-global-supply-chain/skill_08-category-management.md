# TL;DR

**品类供应链规划的核心是通过三层金字塔模型构建科学的商品结构，结合SKU生命周期管理和量化采购计划，在保障供应稳定性的同时不断优化商品组合，实现GMV增长与库存周转的平衡，最终提升整体品类竞争力。**

---

# 场景分析

### 典型场景

**场景一：新季度品类结构调整**
运营团队需要在季度初制定品类结构调整计划，决定哪些品类重点投入、哪些品类收缩优化。

**场景二：SKU优化大作战**
某品类在售SKU 500个，但GMV增长停滞，需要识别哪些SKU该保留、哪些该淘汰。

**场景三：大促采购计划制定**
双11大促前，需要制定全品类采购备货计划，确定各品类备货量和资金需求。

**场景四：新品引进决策**
供应链团队收到多个新品引进提案，需要评估哪个新品值得引进、首批备货量多少合适。

### 核心挑战

- **SKU数量的两难**：SKU少怕错过机会、SKU多怕管不过来
- **备货的博弈**：备多怕滞销、备少怕断货，大促前尤其难决策
- **生命周期的把控**：新品什么时候该扩量、成熟品什么时候该淘汰，缺乏量化标准
- **供应商配合度**：核心供应商交期不稳定，影响备货计划的执行
- **数据孤岛**：销售数据、库存数据、采购数据不连通，难以做出科学决策

### 正确认知

- **品类结构比SKU数量更重要**：20%的SKU贡献80%的GMV，重点维护头部SKU
- **生命周期管理要量化**：不是感觉，而是基于动销率、毛利率、周转天数等指标决策
- **采购计划要算安全库存**：不是拍脑袋，而是基于数据模型计算
- **新品引进要设试错期**：30天试销期是判断新品是否值得扩量的黄金标准
- **品类规划是动态的**：不是一次性的季度计划，而是持续迭代的过程

### Step 3：采购计划制定

**安全库存计算公式：**
```
安全库存 = (最大日销量 - 平均日销量) × 备货天数 + 最低库存

补货量 = 预期销量 + 安全库存 - 当前库存 - 在途库存
```

**决策树：**
```
Step 3.1：计算基础备货量
  └─ 基础备货量 = 日均销量 × 补货周期

Step 3.2：设置安全库存
  └─ 安全库存 = Z值 × 日均销量 × √补货周期

Step 3.3：判断采购紧急程度
  ├─ 库存=0 → P0紧急：立即下单+空运
  ├─ 库存<7天销售 → P1高优：海运+空运组合
  ├─ 库存<14天销售 → P2正常：常规海运
  └─ 库存充足 → P3优化：暂缓采购
```

### Step 4：生命周期决策

**各阶段量化标准：**
| 阶段 | 核心指标 | 达标标准 | 不达标处理 |
|-----|---------|---------|-----------|
| 引进期（1-3月） | 30天动销率 | ≥15% | 分析原因或淘汰 |
| 成长期（3-6月） | 月销量增速 | ≥30% | 加大投入或调整策略 |
| 成熟期（6-18月） | 毛利率+周转天数 | 毛利≥25%，周转≤45天 | 优化成本或升级产品 |
| 衰退期 | 连续3月销量下滑 | >20% | 促销清仓或下架 |

# 专业建议

### 策略层：品类结构优化

1. **三层金字塔模型**：形象品类10%（高曝光）、利润品类30%（高毛利）、流量品类60%（高复购）
2. **品类动态调整**：每季度复盘品类结构，根据市场变化及时调整
3. **新品引进漏斗**：建立"提案-评估-试销-扩量-淘汰"全链路管理
4. **跨品类协同**：寻找高关联品类进行组合运营，提升客单价

### 执行层：SKU精细管理

1. **SKU分级运营**：A类SKU重点维护保障库存，D/F类SKU快速淘汰释放精力
2. **动销率监控**：30天零动销SKU立即启动预警，60天零动销启动清仓
3. **生命周期管理**：引进期设30天试错期，成熟期关注毛利和周转平衡
4. **库存健康度管理**：设置库存天数预警线（健康<45天，预警45-60天，危险>60天）

### 风控层：采购风险管控

1. **供应商分级管理**：S/A级供应商优先保障，B/C级控制订单量
2. **安全库存动态调整**：根据供应商交期波动、销量趋势动态调整
3. **补货周期追踪**：每月统计供应商实际交期vs承诺交期偏差
4. **大促备货专项**：大促前60天启动备货，核心爆款双供应商备货

# 品类供应链规划

## 一、品类结构规划

### 1.1 品类分层模型

天猫国际品类结构采用**三层金字塔模型**，实现从战略到执行的分级管理：

```
                    ┌─────────────────────┐
                    │     形象品类         │  ← 10% SKU
                    │  (Flagship/Showcase) │
                    │  高曝光、高毛利、引领趋势 │
                    └─────────────────────┘
           ┌───────────────────────────────┐
           │         利润品类              │  ← 30% SKU
           │      (Profit Drivers)         │
           │   稳定流量、成熟品类、高复购   │
           └───────────────────────────────┘
  ┌─────────────────────────────────────────┐
  │              流量品类                    │  ← 60% SKU
  │         (Traffic Builders)              │
  │   引流款、高性价比、市场竞争、GMV贡献   │
  └─────────────────────────────────────────┘
```

### 1.2 品类规划核心维度

| 评估维度 | 指标说明 | 评估方法 |
|---------|---------|---------|
| **市场规模** | 品类在天猫国际的整体GMV盘子 | 行业报告 + 平台数据 |
| **增长趋势** | 品类同比/环比增速 | 生意参谋 + 历史数据 |
| **竞争强度** | 品牌集中度、头部占比 | 竞品分析 |
| **毛利空间** | 品类平均毛利率 | 成本结构分析 |
| **供应链稳定性** | 货源保障、物流时效 | 供应商评估 |
| **政策风险** | 跨境政策、监管要求 | 政策跟踪 |

### 2.2 SKU分层管理机制

根据**销售贡献**与**利润贡献**两个维度，将SKU分为四类：

| SKU类型 | 占比 | 运营策略 | 资源倾斜 |
|---------|-----|---------|---------|
| **明星SKU** | 20% | 重点维护、持续补货、流量加持 | 最高优先级 |
| **现金牛SKU** | 30% | 稳定供应、控成本、提效率 | 高优先级 |
| **问题SKU** | 30% | 重点分析、差异化运营、转化提升 | 中优先级 |
| **淘汰SKU** | 20% | 促销清仓、逐步下架、优化结构 | 逐步退出 |

#### 【A类SKU - 重点维护】
- ✓ 保障库存充足率≥95%

- ✓ 参加平台核心营销活动
- ✓ 持续追踪竞品价格策略
- ✓ 用户评价管理与维护

#### 【D类SKU - 促销清仓】
- ✓ 评估库存结构与库龄
- ✓ 制定阶梯式促销方案
- ✓ 申请专项清仓资源位
- ✓ 设置预警机制防止超卖
- ✓ 完成后及时下架归档

#### 【F类SKU - 直接淘汰】
- ✓ 核对库存清零
- ✓ 完成商品下架
- ✓ 解除供应商合约（如需）
- ✓ 更新品类结构数据
- ✓ 复盘淘汰原因

### 3.2 采购模式选择

| 模式 | 适用场景 | 优势 | 劣势 | 资金占用 |
|-----|---------|-----|-----|---------|
| **保税仓备货** | 爆款、高频品 | 时效快、体验好 | 资金占用大 | 高 |
| **海外仓直邮** | 中等销量品 | 库存灵活 | 时效较慢 | 中 |
| **集货直邮** | 低销量品、新品试水 | 零库存风险 | 物流成本高 | 低 |
| **柔性供应链** | 趋势品、非标品 | 快速响应 | 成本较高 | 灵活 |

### 3.3 采购计划计算模型

#### 3.3.1 安全库存计算

```
安全库存 = (最大日销量 - 平均日销量) × 备货天数 + 最低库存

示例：
最大日销量 = 100件
平均日销量 = 60件
备货天数 = 7天
最低库存 = 200件

安全库存 = (100 - 60) × 7 + 200 = 480件
```

#### 3.3.2 采购量计算

```
采购量 = 预期销量 + 安全库存 - 当前库存 - 在途库存

示例：
预期30天销量 = 1800件
安全库存 = 480件
当前库存 = 600件
在途库存 = 200件

采购量 = 1800 + 480 - 600 - 200 = 1480件
```

#### 3.3.3 备货周期规划

| 采购模式 | 备货周期 | 建议提前下单时间 |
|---------|---------|----------------|
| 保税仓备货 | 7-15天 | 大促前45天 |
| 海外仓直邮 | 15-30天 | 大促前60天 |
| 集货直邮 | 5-10天 | 大促前30天 |

## 五、实战案例与最佳实践

### 5.1 案例一：美妆品类SKU结构优化

**背景：** 某美妆品类在售SKU 500个，但GMV增长停滞，毛利率持续下滑。

**问题诊断：**
- SKU数量过多（500个），但有效SKU仅占40%
- 尾部SKU（200个）贡献不足5% GMV，却占用50%运营精力
- 形象品类缺失，高端线不足

**优化方案：**
1. **SKU精简：** 从500个优化至300个（淘汰200个低效SKU）
2. **结构重塑：** 
   - 形象品类（10%）：引入3个高端小众品牌
   - 利润品类（30%）：聚焦50个核心爆款
   - 流量品类（60%）：优化性价比产品线

**优化效果：**
- SKU精简40%，运营效率提升60%
- 毛利率从28%提升至35%
- GMV同比增长25%

### 5.2 案例二：大促采购计划制定

**背景：** 某季度大促（双11），需制定全品类采购备货计划。

**计划制定过程：**

```
Step 1: 数据预测
├── 历史双11数据：GMV 5000万
├── 同比增长预期：30%
├── 目标GMV：6500万

Step 2: 品类拆解
├── 母婴用品：预计GMV 2000万（占比30%）
├── 美妆护肤：预计GMV 2500万（占比38%）
└── 保健食品：预计GMV 2000万（占比32%）

Step 3: 采购量计算
├── 安全库存天数：15天
├── 平均毛利率目标：30%
└── 采购资金需求：4550万

Step 4: 供应商协同
├── 核心供应商提前60天启动
├── 物流链路预演
└── 应急备货方案
```

**关键成功因素：**
- 提前45天启动采购计划
- 核心爆款双倍备货
- 设置10%弹性库存应对超卖

## 七、常见问题处理

### Q1: 如何平衡SKU数量与运营效率？
**A:** 采用"二八法则"——20%的核心SKU贡献80%的GMV。重点维护头部SKU，减少尾部SKU的运营投入，逐步淘汰长期低效SKU。

### Q2: 新品引进后多久可以判断是否成功？
**A:** 一般30天可以初步判断。核心指标：动销率≥15%、转化率≥3%、好评率≥90%。达标则进入成长期，不达标则分析原因后决策。

### Q3: 保税仓库存积压如何处理？
**A:** 
1. 优先内部消化：员工内购、KOL赠送
2. 跨品类清仓：搭配其他品类促销
3. 平台资源置换：申请清仓专场
4. 临期处理：临近保质期前60天启动清仓

### Q4: 如何应对供应商交期延误？
**A:**
1. 提前识别：建立供应商KPI监控体系
2. 备选方案：每个核心SKU至少有2家供应商
3. 分批到货：拆分订单降低单批延误风险
4. 安全库存：关键SKU设置更高安全库存
