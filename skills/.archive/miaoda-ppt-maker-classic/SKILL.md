## 🎨 经典商务风格特点

### 配色方案
| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | 深棕红 #2D1B1B | 经典、稳重 |
| 强调 | 金色 #DAA520 | 传统中国风 |
| 文字 | 米白色 #FFF8E6 | 高对比、易读 |
| 辅助 | 橄榄绿/深红 | 成功/警告 |

### 排版风格
- **对称工整**：左右对齐，视觉平衡
- **顶部金色标题栏**：统一风格，层次分明
- **适度留白**：不激进，保持呼吸感
- **经典装饰**：金色分隔线，圆点列表

### 图表风格
| 经典版 | 年轻版 |
|--------|--------|
| 经典柱状图 | 创意图表 |
| 传统饼图 | 旭日图 |
| 稳重折线图 | 动态图表 |
| 达成率进度条 | 仪表盘 |
| 经典数据表格 | 卡片式 |

### 内容风格
- ✅ 强调KPI达成率
- ✅ 同比/环比数据对比
- ✅ 排名和荣誉
- ✅ 团队表彰
- ✅ 稳重务实的语言
- ❌ 避免：emoji过多、表情包、太多网络用语

## 📊 图表数据格式

### 柱状图
```json
{
  "type": "bar",
  "title": "季度销售额趋势",
  "categories": ["Q1", "Q2", "Q3", "Q4"],
  "values": [280, 320, 350, 330]
}
```

### 饼图
```json
{
  "type": "pie",
  "title": "销售构成",
  "segments": {
    "线上直销": 45,
    "线下渠道": 30,
    "代理分销": 15,
    "其他": 10
  }
}
```

### 折线图
```json
{
  "type": "line",
  "title": "月度增长趋势",
  "categories": ["1月", "2月", "3月", "4月"],
  "values": [85, 92, 88, 95]
}
```

### 对比图
```json
{
  "type": "comparison",
  "title": "同比数据对比",
  "categories": ["Q1", "Q2", "Q3", "Q4"],
  "data_2025": [280, 320, 350, 330],
  "data_2024": [210, 250, 280, 290]
}
```

### KPI仪表盘
```json
{
  "type": "kpi",
  "title": "核心KPI达成情况",
  "items": [
    {"title": "年度目标完成率", "value": "78%"},
    {"title": "净利润增长率", "value": "12.5%"},
    {"title": "客户满意度", "value": "92%"},
    {"title": "团队稳定性", "value": "95%"}
  ]
}
```

### 数据表格
```json
{
  "type": "table",
  "title": "部门绩效排名",
  "headers": ["排名", "部门", "完成率", "得分"],
  "rows": [
    ["1", "销售一部", "98%", "95"],
    ["2", "销售二部", "92%", "88"]
  ]
}
```
