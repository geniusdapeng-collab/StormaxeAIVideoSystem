## 🎨 核心特性

### 12种专业图表

| 类型 | 图表 | 适用场景 |
|------|------|----------|
| **能力分析** | 雷达图、仪表盘 | 多维度评估、KPI展示 |
| **流程分析** | 漏斗图 | 转化漏斗、用户路径 |
| **趋势分析** | 组合图（柱+线） | 趋势+对比双视图 |
| **构成分析** | 堆叠柱状图 | 多品类占比变化 |
| **决策分析** | 矩阵图、金字塔 | 四象限、层级结构 |
| **项目管理** | 甘特图、热力图 | 进度跟踪、区域分析 |

### 四套配色主题

| 主题 | 配色 | 适用场景 |
|------|------|----------|
| Executive | 深蓝+金色 | 商务汇报（默认） |
| Spark | 深紫+青色 | 科技创业 |
| Terminal | 深灰+绿色 | 技术极客 |
| Clean | 白色+红色 | 简约专业 |

## 📐 页面布局规范

### 图表页布局（左侧图表60% + 右侧洞察40%）

```
┌─────────────────────────────────────────────────────────┐
│  页面标题                                                    │
├──────────────────────────┬────────────────────────────┤
│                          │  📊 核心洞察                       │
│    【专业图表】            │  ─────────                    │
│    雷达图/漏斗图           │  • 综合得分：85分               │
│    甘特图/热力图           │  • 优势能力：采购 92分          │
│    矩阵图/金字塔           │  • 待提升：成本 78分            │
│    ...                    │  💡 建议                       │
│    60%                   │  聚焦提升成本控制能力             │
│                          │  40%                          │
└──────────────────────────┴────────────────────────────┘
```

### 留白规范
- 边距统一：左右0.8英寸
- 内容占比：≤60%
- 元素间距：≥0.3英寸

### 字体规范
- 页面标题：32pt 粗体
- 洞察标题：18pt 粗体
- 正文内容：13-14pt
- 辅助说明：11-12pt

## ⚙️ 使用流程

### 1. 接收需求
识别用户需要制作PPT的意图

### 2. 内容采集
- 用户直接提供数据/内容
- 读取飞书文档自动提取
- 读取PDF文件提取内容

### 3. 数据处理
- 解析用户提供的数据
- 转换为标准图表格式
- 生成洞察分析

### 4. PPT生成
- 选择合适的主题配色
- 应用图表+文字配合布局
- 生成PPTX文件

### 5. 发送交付
- 发送文件给用户
- 简要说明内容结构

1. 数据格式是否符合Schema定义
2. 图表类型是否适合数据特征
3. 文件是否正常生成

生成失败时自动降级：
- 图表失败 → 使用文字列表
- 主题失败 → 使用默认主题
