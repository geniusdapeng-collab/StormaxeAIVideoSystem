## 2. 指标框架

### AI产品指标层次

| 层次 | 说明 | 示例 |
|------|------|------|
| 北极星 | 核心成功指标 | DAU/转化率 |
| 过程指标 | 用户行为路径 | 输入率/完成率 |
| 质量指标 | AI效果好坏 | 准确率/满意度 |
| 业务指标 | 商业价值 | 收入/续费率 |

### 指标矩阵

```
1. 用户维度：活跃/留存/流失
2. 体验维度：响应/等待/完成
3. 质量维度：准确/召回/相关
4. 商业维度：收入/成本/ROI
```

## 3. 专业建议

### A. 核心指标

#### 选择方法
```
1. 业务目标：产品目标是什么
2. 用户价值：用户获得什么
3. 可量化：能够被测量
4. 可行动：能指导决策
5. 可追踪：持续监测
```

### B. 质量指标

#### AI特有指标
```
1. 准确率：正确答案比例
2. 召回率：覆盖程度
3. 响应时间：处理速度
4. 成功率：完成比例
5. 相关性：结果相关性
```

### C. 监控体系

#### 监控内容
```
1. 核心指标：日/周/月趋势
2. 异常预警：指标突变告警
3. 细分维度：按用户/场景分析
4. 对比分析：版本/策略对比
5. 归因分析：找影响因素
```

### D. 分析方法

#### 分析方法
```
1. 漏斗分析：转化路径
2. 留存分析：用户粘性
3. 分布分析：数据分布
4. 归因分析：贡献度
5. 相关分析：因素关系
```
