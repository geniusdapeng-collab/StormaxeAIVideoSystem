## 2. 增长框架

### AARRR模型

| 阶段 | 说明 | 关键指标 |
|------|------|----------|
| Acquisition | 获取 | 获客成本 |
| Activation | 激活 | 首次使用 |
| Retention | 留存 | 留存率 |
| Revenue | 变现 | 收入 |
| Referral | 推荐 | 裂变系数 |

### AI产品增长特点

```
1. 体验即营销：效果好口碑好
2. 工具属性强：容易分享
3. API可扩展：生态获客
4. 效果可量化：数据驱动
```

## 3. 专业建议

### A. 获客渠道

#### 渠道选择
```
1. 内容营销：博客/社媒
2. SEO/ASO：搜索流量
3. 口碑传播：用户推荐
4. 合作获客：生态合作
5. 付费投放：精准获客
```

### B. 激活策略

#### 激活要点
```
1. 首次体验：惊艳时刻
2. 新手引导：快速上手
3. 价值呈现：立即见效
4. 成就反馈：正向激励
5. 社交证明：他人认可
```

### C. 留存提升

#### 方法策略
```
1. 产品价值：持续有用
2. 习惯培养：使用场景
3. 会员体系：权益激励
4. 社区运营：归属感
5. 持续优化：体验升级
```

### D. 裂变增长

#### 裂变方法
```
1. 邀请奖励：双赢机制
2. 内容分享：自然传播
3. 协作功能：共同使用
4. 模板/提示词：分享价值
5. 比赛活动：社交互动
```
