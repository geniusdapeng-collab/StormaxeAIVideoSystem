## TL;DR

AI产品 = 明确能力边界 + 渐进式呈现 + 适当人机协作 + 容错设计

## 1. AI产品设计原则

### 核心原则
| 原则 | 说明 |
|------|------|
| 透明 | 让用户知道是AI |
| 可控 | 用户可修正AI输出 |
| 可靠 | 减少幻觉和错误 |
| 及时 | 响应速度合理 |

## 2. AI交互模式

### 对话式
- 适用：信息查询、任务执行
- 设计：清晰输入、格式化输出

### 建议式
- 适用：辅助决策
- 设计：多选项+可解释

### 自动式
- 适用：Routine任务
- 设计：可干预+可撤回
