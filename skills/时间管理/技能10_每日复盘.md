## TL;DR

每日复盘 = 回顾今天 + 分析得失 + 总结收获 + 计划明天 + 持续改进

## 1. 复盘认知

### 1.1 什么是复盘
```
复盘定义：
复盘源于围棋术语，指对局结束后回顾整个对局，分析得失，学习改进。

每日复盘：
每天工作结束时，回顾今天做了什么，有什么收获，有什么教训，明天要做什么。

复盘的价值：
├── 巩固收获：把今天的经验变成能力
├── 发现问题：及时发现自己的不足
├── 改进提升：找到改进的方向
├── 避免重复：不犯同样的错误
├── 积累知识：把碎片经验系统化
└── 促进成长：持续改进，日积月累
```

### 1.2 为什么要复盘
```
复盘的重要性：
├── 经验变成能力：做过的事不白做
├── 吃一堑长一智：错误不能白犯
├── 建立闭环：有行动有反思
├── 持续进步：每天进步一点点
├── 减少遗忘：好经验不会忘记
└── 目标清晰：知道自己在做什么

不复盘的后果：
├── 做过的就忘了
├── 犯过的错还会再犯
├── 不知道自己做了什么
├── 没有积累和成长
└── 原地踏步
```

### 1.3 复盘常见问题
```
复盘常见问题：
├── 没时间：太忙没时间复盘
├── 不知道写什么：复盘太形式化
├── 写流水账：没有深度思考
├── 只写问题：忽略做得好的
├── 复盘完不改进：复盘没效果
└── 三天打鱼：不能坚持
```

## 2. 复盘内容

### 2.1 核心复盘内容
```
每日复盘核心内容：

1. 今天完成了什么？
├── 最重要的3件事
├── 完成的任务
└── 产出的成果

2. 今天做得好的？
├── 做得好的点
├── 有效的做法
└── 可以继续的方法

3. 今天做得不好的？
├── 没做好的点
├── 遇到的问题
└── 需要改进的

4. 明天要做什么？
├── 最重要的3件事
├── 需要解决的问题
└── 计划尝试的方法

5. 一个收获/感悟
├── 学到的新东西
├── 新的认知
└── 触动自己的点
```

```
深度复盘内容（可选）：

人际关系：
├── 今天和谁有重要互动？
├── 做得好/不好的沟通？
├── 需要改进的关系？
└── 明天要维护的关系？

时间使用：
```


## 3. 复盘技巧

### 3.1 复盘时间
```
复盘时间选择：

晚间复盘（推荐）：
├── 时间：下班后或睡前
├── 时长：10-15分钟
├── 优点：记忆新鲜，容易回顾
└── 适合：大多数人

早晨复盘：
├── 时间：第二天早晨
├── 时长：10分钟
├── 优点：可以用复盘指导一天
└── 适合：喜欢早晨规划的人

固定时间：
├── 建议固定一个时间
├── 形成习惯
├── 不要太晚（太累不想写）
└── 也不要太早（还没结束一天）
```

### 3.2 复盘深度
```
复盘深度：

表面复盘（新手）：
├── 做了什么任务
├── 完成情况
└── 明天计划

中等复盘（进阶）：
├── 任务完成情况
├── 做得好/不好的点
├── 原因分析
├── 改进措施
└── 明天计划

深度复盘（高阶）：
├── 全方位回顾
├── 原因深度分析
├── 认知提升
├── 方法论总结
├── 跨领域联系
└── 行动计划

建议：
├── 新手从简单开始
├── 不要追求一步到位
├── 慢慢增加深度
└── 关键是持续
```

### 3.3 复盘记录
```
复盘记录：

记录方式：
├── 纸笔：简单直接
├── 备忘录：手机随时记
├── 笔记APP：Notion/印象笔记
├── 复盘日记本：专门的复盘本
└── 语音记录：适合不爱打字

记录原则：
├── 及时：当天记下
├── 具体：写具体的事
├── 真实：不要自我欺骗
├── 有行动：有改进措施
└── 可回顾：定期翻看

回顾频率：
├── 每天写
├── 每周翻看
├── 每月总结
└── 每季度回顾
```

## 5. 复盘习惯养成

### 5.1 建立习惯
```
建立复盘习惯：

开始阶段：
├── 从简单的开始
├── 先写3件事
├── 固定时间
├── 固定地点
└── 不要追求完美

坚持阶段：
├── 每天坚持
├── 即使只有5分钟
├── 不要中断
├── 形成自动化
└── 让它成为习惯

优化阶段：
├── 增加深度
├── 增加维度
├── 定期回顾
├── 持续改进
└── 找到适合自己的方式
```

### 5.2 克服困难
```
复盘困难及应对：

没时间：
├── 只需要10-15分钟
├── 固定时间，雷打不动
├── 可以边写边回顾
└── 不写才浪费时间

不知道写什么：
├── 从"今天做了3件事"开始
├── 慢慢增加维度
├── 不要追求完美
└── 写流水账也比不写好

写了没效果：
├── 关键是执行改进措施
├── 定期翻看以前的复盘
├── 找到规律和模式
└── 复盘+行动=成长

太累不想写：
├── 把它变成习惯
├── 养成自动化
├── 把它当作一天的结束仪式
└── 5分钟就能写完
```
