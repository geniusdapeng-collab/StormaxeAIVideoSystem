## TL;DR

日程规划 = 确定目标 + 列出任务 + 分配时间 + 留有缓冲 + 灵活调整

## 1. 日程规划认知

### 1.1 为什么规划重要
```
日程规划的价值：
├── 明确目标：知道今天要完成什么
├── 合理分配：把时间分配给不同任务
├── 保护时间：不被临时任务占据
├── 减少焦虑：按计划执行更安心
├── 提高效率：有计划减少决策时间
└── 便于复盘：知道自己做了什么
```

### 1.2 规划常见问题
```
日程规划常见问题：
├── 计划太满：没有留buffer
├── 不切实际：计划完不成
├── 太笼统：没有具体任务
├── 缺少重点：什么事都安排
├── 不会调整：变化就打乱计划
└── 只列不执行：计划成空想
```

## 2. 日程规划步骤

### 2.1 早晨规划（10分钟）
```
早晨规划步骤：

1. 查看日历
   └── 今天有什么会议/约定了什么

2. 确定3件最重要的事
   └── 今天必须完成的3件事

3. 评估时间
   └── 预估每件事需要的时间

4. 分配时间段
   └── 把任务安排到具体时间

5. 预留buffer
   └── 预留1-2小时应急

6. 设定目标
   └── 今天要达成什么
```

### 2.2 规划模板
```
每日计划模板：

日期：XX月XX日 周X
今日目标：...

【最重要的事】（3件）
1. XX任务 - 预计XX时间
2. XX任务 - 预计XX时间
3. XX任务 - 预计XX时间

【例行任务】
- 邮件处理：XX时间
- 团队站会：XX时间
- 汇报更新：XX时间

【时间安排】
9:00-10:00 | 最重要的事1
10:00-11:00 | 最重要的事2
11:00-12:00 | 邮件/沟通
12:00-13:00 | 午餐休息
13:00-15:00 | 最重要的事3
15:00-16:00 | 例行任务
16:00-17:00 | 缓冲/机动
17:00-18:00 | 收尾/明日计划

【预留buffer】1小时

【注意事项】
-
```

### 2.3 时间分配原则
```
时间分配原则：

黄金时间分配：
├── 9-11点：最重要的工作
├── 2-4点：次重要的工作
└── 4-6点：收尾和次要工作

时间块原则：
├── 把同类任务放在一起
├── 减少切换成本
└── 效率更高

Buffer原则：
├── 预留10-20%的时间
├── 处理意外任务
└── 避免计划太满

灵活原则：
├── 计划不是死的
├── 根据实际情况调整
└── 保持弹性
```

## 3. 任务安排

### 3.1 任务分类安排
```
任务分类安排：

深度工作（需要专注）：
├── 安排在黄金时间
├── 整块时间（1-2小时）
├── 关闭通知
└── 最重要的事优先

协作沟通：
├── 安排在协作时段
├── 集中处理
└── 避免打断深度工作

日常事务：
├── 安排在低效时段
├── 碎片时间处理
└── 批量处理

学习成长：
├── 安排固定时间
├── 早晨或午休后
└── 每天坚持
```

### 3.2 安排检查清单
```
任务安排检查：
□ 最重要的事安排在黄金时间了吗？
□ 每件事有预估时间吗？
□ 有预留buffer时间吗？
□ 有留出休息时间吗？
□ 有考虑会议/约定了的时间吗？
□ 计划可行吗（不过于饱满）？
□ 灵活性够吗（能应对变化）？
```

## 4. 计划执行

### 4.1 执行原则
```
计划执行原则：

开始原则：
├── 从最重要的事开始
├── 不要从简单的事开始
├── 开始了就专注做

执行原则：
├── 专注当下：只做这一件事
├── 时间控制：控制时间不超时
├── 记录状态：完成或延后

结束原则：
├── 标记完成
├── 调整计划
├── 记录收获
└── 进入下一项
```

### 4.2 应对打断
```
被打断处理：

计划内打断：
├── 评估紧急程度
├── 记录打断
├── 决定是立即处理还是延后
└── 调整计划

计划外任务：
├── 问自己：真的紧急吗？
├── 记录下来
├── 放入"之后做"清单
├── 除非紧急，否则按计划执行
└── 重要的事完成后处理
```

### 4.3 调整计划
```
计划调整：

正常调整：
├── 任务提前完成：加入新任务或休息
├── 任务延后：延后到之后的时段
└── 紧急插入：替换较低优先级任务

大幅调整：
├── 重新评估优先级
├── 推迟部分任务到明天
├── 必要时加班完成
└── 反思计划合理性

调整记录：
├── 记录调整原因
├── 统计计划完成率
└── 持续优化
```

## 5. 晚间收尾

### 5.1 收尾步骤
```
晚间收尾（10分钟）：

1. 检查完成
   └── 标记今天完成的任务

2. 记录未完成
   └── 把未完成的移到明天

3. 明日计划
   └── 初步规划明天最重要的3件事

4. 简单复盘
   └── 今天有什么收获/教训

5. 准备明早
   └── 准备需要的资料/工具
```

### 5.2 复盘问题
```
每日复盘问题：
□ 今天完成了什么？
□ 有什么没完成？为什么？
□ 有什么做得好可以继续？
□ 有什么需要改进？
□ 明天最重要的一件事是什么？
```
