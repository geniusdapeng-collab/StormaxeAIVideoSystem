## TL;DR

番茄工作法 = 25分钟专注 + 5分钟休息 + 循环往复 + 持续改进

## 1. 番茄工作法原理

### 1.1 什么是番茄工作法
```
番茄工作法定义：
一种时间管理方法，通过将工作时间划分为25分钟的专注时段（番茄钟）和短暂的休息时间，来提升专注力和工作效率。

核心原理：
├── 专注单一任务：一次只做一件事
├── 时间盒子：设定固定时长的专注时间
├── 主动休息：休息是为了更好地工作
├── 即时反馈：记录完成情况
└── 持续迭代：不断改进工作方法
```

### 1.2 番茄工作法价值
```
番茄工作法价值：
├── 提升专注：减少分心
├── 克服拖延：降低启动压力
├── 时间感知：建立时间感
├── 减少焦虑：专注于当下
├── 提升效率：减少切换成本
└── 持续产出：保持稳定输出
```

### 1.3 适合场景
```
适合场景：
├── 深度工作：需要专注的创造性工作
├── 学习备考：看书、学习新知识
├── 写作编辑：写文章、写代码
├── 项目规划：方案设计、思考
└── 任务分解：处理大任务

不适合场景：
├── 紧急响应：需要立即处理的事
├── 简短任务：5分钟内能完成的
├── 沟通协作：需要讨论的会议
└── 日常事务：查看邮件、简单回复
```

## 2. 基础操作

### 2.1 标准流程
```
标准番茄钟流程：

1. 准备阶段（1-2分钟）
   ├── 选择任务
   ├── 预估需要几个番茄钟
   └── 设定计时器（25分钟）

2. 工作阶段（25分钟）
   ├── 专注工作
   ├── 不被打断
   └── 不做其他事

3. 休息阶段（5分钟）
   ├── 站起来活动
   ├── 喝水休息
   └── 远离工作

4. 循环重复
   ├── 每4个番茄钟后长休息（15-30分钟）
   └── 记录完成情况
```

### 2.2 关键规则
```
番茄工作法规则：

核心规则：
├── 一个番茄钟不可分割
├── 番茄钟结束后立刻休息
├── 番茄钟内只做一件事
├── 番茄钟数量记录在纸上
└── 完成4个番茄钟后长休息

打断规则：
├── 打断分为内部和外部
├── 内部打断：记录下来，稍后处理
├── 外部打断：告知、协商、计划
└── 被打断的番茄钟作废
```

### 2.3 计时方法
```
计时方法：
├── 手机APP：Forest、潮汐、Focus
├── 电脑软件：Pomodone、Be Focused
├── 物理计时：厨房计时器
├── 在线工具：tomato-timer.com
└── 简单方法：手机闹钟

选择原则：
├── 简单易用：不要花时间设置
├── 有提醒：不会忘记休息
├── 记录功能：能记录完成数量
└── 适合自己：不同人适合不同工具
```

## 3. 进阶技巧

### 3.1 任务预估
```
任务预估方法：
├── 预估任务需要几个番茄钟
├── 记录实际用了几个
├── 分析差距原因
└── 持续调整预估

预估记录表：
| 任务 | 预估 | 实际 | 备注 |
|------|------|------|------|
| 写方案 | 3个 | 4个 | 调研花了时间 |
| 写代码 | 2个 | 2个 | 预估准确 |
| 回邮件 | 1个 | 0.5个 | 很快就处理完了 |
```

### 3.2 打断处理
```
内部打断处理：
├── 识别打断：意识到自己想做其他事
├── 记录打断：把想法写下来
├── 快速评估：紧急吗？重要吗？
├── 继续工作：回到当前番茄钟
└── 安排处理：为打断任务安排时间

外部打断处理：
├── 感谢对方：谢谢你的联系
├── 快速评估：紧急吗？
├── 如果紧急：
│   ├── 结束当前番茄钟
│   ├── 处理紧急事项
│   └── 重新开始新番茄钟
└── 如果不紧急：
    ├── 告知你的专注时间
    ├── 约定稍后处理
    └── 继续当前工作
```

### 3.3 长休息利用
```
长休息（15-30分钟）建议：
├── 身体活动：站起来、走动
├── 补充能量：喝水、吃点东西
├── 眼睛休息：看看远处
├── 简单思考：回顾刚才的工作
└── 处理打断：处理之前记录的小事

长休息禁忌：
├── 不要看手机（容易停不下来）
├── 不要继续想工作的事
├── 不要开始新的任务
└── 不要过度休息（30分钟内）
```

## 4. 实际应用

### 4.1 工作日应用
```
工作日番茄钟安排：

早晨高效时段（9-11点）：
├── 2-3个番茄钟处理最重要的事
├── 每个番茄钟后休息5分钟
└── 不要一上来就处理邮件

上午协作时段（11-12点）：
├── 减少番茄钟
├── 改为处理日常事务
└── 准备午餐

下午高效时段（2-4点）：
├── 2-3个番茄钟
├── 处理复杂任务
└── 保持专注

下班前（5-6点）：
├── 收尾工作
├── 回复必要邮件
└── 规划明天
```

### 4.2 任务类型应用
```
不同任务的番茄钟设置：

深度工作类（方案、写作、代码）：
├── 每次2-3个连续番茄钟
├── 安排在黄金时间
├── 中间只休息5分钟
└── 长休息在4个之后

学习类（读书、课程）：
├── 每次1-2个番茄钟
├── 番茄钟之间复习
├── 做好笔记
└── 适当延长休息

日常事务类（邮件、审批）：
├── 不使用完整番茄钟
├── 用碎片时间处理
└── 记录但不计时
```

### 4.3 调整适应
```
适应阶段建议：

第一周（适应期）：
├── 先从1-2个番茄钟开始
├── 不要追求数量
└── 专注感受工作节奏

第二周（习惯期）：
├── 增加到3-4个番茄钟
├── 尝试完整的工作日
└── 记录完成情况

第三周（优化期）：
├── 找到自己的节奏
├── 调整番茄钟时长（有些人适合30分钟）
├── 总结适合自己的方法
└── 持续优化
```

## 5. 常见问题

### 5.1 被频繁打断
```
被频繁打断处理：
├── 告知他人：设定专注时间
├── 物理隔离：去会议室、戴耳机
├── 工具辅助：显示"忙碌"状态
├── 批量处理：把同类事集中
├── 优化时间：把重要时段空出来
└── 接受现实：有些打断无法避免
```

### 5.2 觉得25分钟太长
```
调整番茄钟时长：
├── 20分钟：适合刚开始练习
├── 25分钟：标准时长
├── 30分钟：适合深度工作
├── 45分钟：适合长时间专注
└── 找到适合自己的：不是所有人都适合25分钟

重要的是：
├── 专注：质量比数量重要
├── 坚持：持续工作比时长重要
└── 迭代：不断调整找到最佳
```

### 5.3 休息时不想工作
```
休息时心理调整：
├── 明确休息目的：休息是为了更好工作
├── 换个脑子：让负责工作的脑区休息
├── 做简单的事：喝水、走动、伸懒腰
├── 不刷手机：刷手机反而更累
├── 奖励自己：完成了番茄钟再休息
└── 调整心态：休息不是浪费时间
```
