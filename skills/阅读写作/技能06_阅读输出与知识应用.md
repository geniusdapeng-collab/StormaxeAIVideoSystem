## 2. 输出框架

### 输出层次

| 层次 | 形式 | 价值 |
|------|------|------|
| 笔记 | 摘录/心得 | 加深理解 |
| 摘要 | 总结/归纳 | 结构化 |
| 评论 | 思考/评价 | 批判思维 |
| 应用 | 实践/行动 | 转化为能力 |
| 分享 | 讲解/教授 | 深化理解 |

### 输出流程

```
1. 阅读：获取信息
2. 理解：消化吸收
3. 整理：结构化
4. 输出：表达分享
5. 应用：实践行动
```

## 3. 专业建议

### A. 输出方式

#### 常用方式
```
1. 读书笔记：摘录+感想
2. 书评文章：系统评价
3. 思维导图：可视化
4. 知识卡片：原子化
5. 讲解视频：教授他人
```

### B. 知识应用

#### 应用方法
```
1. 问题导向：解决实际问题
2. 行动清单：列出要做的
3. 实践练习：刻意应用
4. 项目实践：整合应用
5. 反思总结：持续改进
```

### C. 输出习惯

#### 习惯培养
```
1. 强制输出：读后必写
2. 定时输出：固定时间
3. 量化目标：数量要求
4. 公开承诺：社交压力
5. 反馈迭代：持续改进
```

### D. 费曼学习

```
1. 选择主题：确定要教的
```
