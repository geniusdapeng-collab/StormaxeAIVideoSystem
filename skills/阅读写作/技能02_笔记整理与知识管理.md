## 2. 笔记体系框架

### 笔记分类

| 类型 | 内容 | 特点 |
|------|------|------|
| 灵感笔记 | 想法/灵感 | 零散/快速 |
| 读书笔记 | 阅读收获 | 结构化 |
| 学习笔记 | 课程/培训 | 系统化 |
| 工作笔记 | 项目/会议 | 实用导向 |
| 生活笔记 | 日记/感悟 | 个人化 |

### 笔记结构

```
1. 收集箱：所有未整理
2. 分类文件夹：按主题
3. 标签系统：多维度
4. 索引：快速定位
5. 归档：历史保存
```

## 3. 专业建议

### A. 笔记方法

#### 常用方法
```
1. 康奈尔笔记：分区记录
2. 子弹笔记：简洁快速
3. 卡片笔记：原子化
4. 思维导图：可视化
5. Zettelkasten：链接网络
```

### B. 知识组织

#### 组织原则
```
1. 分类清晰：主题明确
2. 标签丰富：多维度
3. 链接关联：建立连接
4. 版本管理：历史追溯
5. 搜索优化：便于查找
```

### C. 工具选择

#### 工具对比
```
1. 纸质笔记本：专注/手写
2. 印象笔记：功能全面
3. Notion：结构化强
4. Obsidian：链接强大
5. 语雀：团队协作
```

### D. 整理习惯

#### 整理流程
```
1. 每日：收集箱清空
2. 每周：分类整理
3. 每月：标签优化
4. 每季：知识复盘
5. 每年：归档清理
```
