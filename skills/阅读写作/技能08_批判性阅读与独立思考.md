## 2. 批判思维框架

### 批判要素

| 要素 | 内容 | 问题示例 |
|------|------|----------|
| 论题 | 讨论什么 | 核心问题？ |
| 论点 | 什么观点 | 主旨是什么？ |
| 论据 | 什么证据 | 证据可靠？ |
| 论证 | 如何推理 | 逻辑成立？ |
| 结论 | 得出什么 | 结论合理？ |

### 思考层次

```
1. 理解：作者说什么
2. 分析：怎么说的
3. 评价：说得对吗
4. 应用：对我有什么用
5. 创造：能启发什么新想法
```

## 3. 专业建议

### A. 质疑方法

#### 质疑清单
```
1. 论点清晰吗？
2. 证据是什么？
3. 证据够吗？
4. 推理有效吗？
5. 有偏见吗？
6. 有其他解释吗？
7. 结论合理吗？
```

### B. 论证分析

#### 分析方法
```
1. 识别论证结构
2. 评估前提假设
3. 检查推理逻辑
4. 评估证据强度
5. 识别逻辑谬误
```

### C. 常见谬误

#### 识别谬误
```
1. 人身攻击：攻击人不是观点
2. 稻草人：曲解观点
3. 诉诸权威：权威≠真理
4. 诉诸情感：情感≠事实
5. 非黑即白：过度简化
```

### D. 独立思考

#### 思考习惯
```
1. 多角度：换位思考
2. 查证：验证信息来源
3. 记录：写下思考过程
4. 讨论：与人交流观点
5. 反思：定期回顾判断
```
