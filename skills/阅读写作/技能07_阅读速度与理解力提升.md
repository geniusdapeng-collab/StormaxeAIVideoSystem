## TL;DR

效率提升核心：扩展视幅→减少回视→消除默读→理解优先→刻意练习，五步提升阅读效率

## 2. 阅读效率框架

### 影响因素

| 因素 | 影响速度 | 影响理解 |
|------|----------|----------|
| 视幅 | 大→快 | 大→好 |
| 回视 | 少→快 | 少→连贯 |
| 默读 | 无→快 | 无→理解 |
| 专注度 | 高→快 | 高→深 |
| 词汇量 | 大→快 | 大→准 |

### 速度基准

```
普通阅读：200-300字/分钟
快速阅读：500-800字/分钟
速读：1000+字/分钟
```

## 3. 专业建议

### A. 视幅扩展

#### 扩展方法
```
1. 指读法：手指引导
2. 缩略法：减少回视
3. 组块阅读：扩大每次摄入
4. 边缘扩展：利用周边视觉
5. 训练工具：速读软件
```

### B. 减少回视

#### 回视控制
```
1. 理解基础：词汇量
2. 预读训练：预测内容
3. 专注训练：减少分心
4. 阅读信心：相信理解
5. 标记法：标记位置
```

### C. 消除默读

#### 默读控制
```
1. 节奏阅读：用节拍器
2. 视觉化：图像代替声音
3. 快速扫描：不读出声
4. 注意力转移：专注文字
5. 渐进训练：逐步加速
```

### D. 理解优先

#### 理解策略
```
1. 预览：了解结构
2. 提问：带着问题读
3. 联系：联系已知
4. 总结：每段总结
5. 复习：及时回顾
```
