写作基础核心：明确目的→搭建结构→填充内容→优化表达→反复修改，五步写出好文章

## 2. 写作框架

### 文章结构

```
1. 标题：吸引眼球
2. 开头：引入主题
3. 主体：展开论述
4. 结尾：总结升华
```

### 写作流程

| 阶段 | 内容 | 时间占比 |
|------|------|----------|
| 准备 | 明确目的/收集素材 | 20% |
| 起草 | 写初稿/搭框架 | 30% |
| 修改 | 优化内容/结构调整 | 30% |
| 润色 | 语言打磨/校对 | 20% |

## 3. 专业建议

### A. 文章结构

#### 常见结构
```
1. 总分总：总-分-总
2. 并列式：几个并列观点
3. 递进式：层层深入
4. 对比式：正反对比
5. 问题-解决：问题+方案
```

### B. 表达技巧

#### 语言原则
```
1. 简洁：用词精准
2. 具体：避免抽象
3. 生动：形象化表达
4. 逻辑：条理清晰
5. 连贯：过渡自然
```

### C. 写作方法

#### 写作步骤
```
1. 明确写作目的
2. 确定目标读者
3. 收集相关素材
4. 搭建文章框架
5. 填充具体内容
6. 修改优化
7. 校对检查
```

### D. 常见问题

#### 问题解决
```
1. 写不出来：自由写作/先写大纲
2. 表达不清：多用例子/具体描述
3. 结构混乱：先搭框架
4. 语言枯燥：多读好文章/积累词汇
5. 逻辑混乱：使用逻辑连接词
```
