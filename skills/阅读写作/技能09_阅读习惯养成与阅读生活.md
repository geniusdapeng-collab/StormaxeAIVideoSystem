## 2. 习惯框架

### 微习惯策略

| 原则 | 说明 | 示例 |
|------|------|------|
| 微小 | 从最小开始 | 每天1页 |
| 可行 | 容易做到 | 穿睡衣前读 |
| 灵活 | 不固定形式 | 有空就读 |
| 积累 | 积少成多 | 一年50本 |

### 时间安排

```
1. 固定时间：
   - 晨起后15分钟
   - 午休前15分钟
   - 睡前15分钟

2. 碎片时间：
   - 通勤时间
   - 等候时间
   - 排队时间
```

## 3. 专业建议

### A. 开始策略

#### 入门方法
```
1. 从简单开始：感兴趣的书
2. 从少量开始：每天5页
3. 从固定时间开始：形成规律
4. 从习惯叠加开始：搭配已有习惯
5. 从轻松环境开始：减少阻力
```

### B. 维持策略

#### 持续方法
```
1. 记录进度：打卡记录
2. 设置提醒：固定提醒
3. 创造环境：书放在可见处
4. 加入社群：互相督促
5. 奖励自己：完成目标奖励
```

### C. 障碍克服

#### 解决困难
```
1. 没时间：碎片时间利用
2. 读不进去：换感兴趣的书
3. 容易中断：微习惯策略
4. 拖延：设定具体时间
5. 懈怠：回顾阅读收获
```

### D. 阅读生活

#### 融入生活
```
1. reading角落：固定阅读空间
2. reading清单：待读书单
3. reading社交：参加读书会
4. reading输出：分享读后感
5. reading旅行：探访书店/图书馆
```
