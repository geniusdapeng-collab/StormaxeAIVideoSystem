## 2. 素材框架

### 素材类型

| 类型 | 内容 | 收集方法 |
|------|------|----------|
| 故事 | 人物/事件 | 观察/记录 |
| 案例 | 真实/虚构 | 阅读/收集 |
| 数据 | 统计/研究 | 搜索/阅读 |
| 金句 | 名言/感悟 | 摘录/创作 |
| 观点 | 见解/思考 | 记录/整理 |

## 3. 专业建议

### A. 收集方法

#### 日常收集
```
1. 随身记录：手机备忘录
2. 阅读摘录：好句好段
3. 观察记录：生活发现
4. 搜索积累：网络资源
5. 思考记录：想法感悟
```

### B. 创意激发

#### 激发方法
```
1. 自由写作：想到什么写什么
2. 随机词刺激：随机组合
3. 提问练习：如果...会怎样
4. 跨界思考：不同领域结合
5. 转换视角：不同角度看待
```

### C. 整理方法

#### 整理原则
```
1. 定期整理：周/月回顾
2. 分类清晰：便于查找
3. 标签管理：多维度
4. 定期删减：去芜存菁
5. 关联建立：建立连接
```

### D. 创意转化

#### 转化技巧
```
1. 组合创新：素材+素材
2. 延伸扩展：观点+案例
3. 角度创新：旧素材新角度
4. 场景迁移：跨领域应用
5. 归纳演绎：从具体到抽象
```
