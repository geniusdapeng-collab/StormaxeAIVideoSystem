## 2. 选品框架

### 产品分类

| 品类 | 特点 | 利润空间 |
|------|------|----------|
| 主粮 | 高频刚需 | 20-30% |
| 零食 | 高频复购 | 30-40% |
| 营养品 | 利润高 | 40-50% |
| 罐头 | 搭配销售 | 25-35% |

### 选品原则

```
1. 需求导向：宠物主需要什么
2. 品质优先：安全第一
3. 差异化：避免同质竞争
4. 利润合理：可持续经营
5. 供应链稳定：持续供货
```

## 3. 专业建议

### A. 供应商选择

#### 评估维度
```
1. 资质齐全：生产/经营许可
2. 产品质量：检测报告
3. 供货能力：产能/库存
4. 价格竞争力：议价空间
5. 服务支持：售后/培训
```

### B. 品质把控

#### 品质标准
```
1. 原料溯源：来源可查
2. 配方科学：营养均衡
3. 生产资质：合规生产
4. 检测报告：定期送检
5. 包装规范：符合标准
```

### C. 谈判策略

#### 谈判要点
```
1. 首批量：谈首批起订量
2. 价格：谈阶梯价格
3. 账期：谈付款周期
4. 售后：退换货政策
5. 独家：区域/渠道独家
```

### D. 库存管理

#### 管理方法
```
1. 安全库存：避免断货
2. 先进先出：避免过期
3. 库存预警：自动提醒
4. 盘点定期：账实相符
5. 滞销处理：及时清货
```
