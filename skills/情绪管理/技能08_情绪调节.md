## 1. 情绪调节的基础

### 1.1 什么是情绪调节
```
情绪调节的定义：
情绪调节是指个体对情绪的体验、表达和发生时机进行影响和操控的过程。

情绪调节的目标：
├── 减少负面情绪体验
├── 增加正面情绪体验
├── 维持情绪稳定
├── 提高情绪智力
└── 改善人际关系

情绪调节的重要性：
├── 促进心理健康
├── 改善人际关系
├── 提高工作效率
├── 增强幸福感
├── 促进身体健康
└── 提升决策质量

情绪调节不是：
├── 不是压抑情绪
├── 不是消除情绪
├── 不是情绪控制
├── 不是让情绪消失
└── 而是学会与情绪相处

情绪调节的目标：
接纳情绪 → 调节强度 → 选择表达
```

### 1.2 情绪调节的策略分类
```
情绪调节策略分类（Gross模型）：

先行聚焦策略（在情绪产生前）：
├── 情境选择：选择或回避情境
├── 情境修正：改变当前情境
├── 注意调配：转移或集中注意
└── 认知改变：改变对情境的解释

反应聚焦策略（在情绪产生后）：
├── 反应调节：调节情绪反应
├── 生理调节：调节身体反应
├── 行为调节：调节外在表现
└── 体验调节：调节内在体验

有效策略：
├── 情境选择：主动选择环境
├── 情境修正：改变环境
├── 认知重评：改变思维方式
├── 表达抑制：暂时控制表情
└── 情绪聚焦：关注积极面

无效策略：
├── 压抑情绪
├── 逃避情境
├── 反刍思维
├── 灾难化思维
└── 情绪化应对

## 2. 生理调节

### 2.1 呼吸调节
```
呼吸调节法：

腹式呼吸：
步骤：
1. 躺下或坐直
2. 一只手放在胸口，一只放在腹部
3. 用鼻子吸气（4秒），感受腹部鼓起
4. 屏住呼吸（2秒）
5. 用嘴巴慢慢呼气（6-8秒）
6. 感受腹部下降
7. 重复5-10次

要点：
├── 专注于呼吸的感觉
├── 延长呼气时间
├── 每天练习10-15分钟
├── 感到压力时随时使用
└── 慢慢建立习惯

---

快速呼吸法（应对急性焦虑）：
步骤：
1. 快速浅呼吸（1-2分钟）
2. 过度换气导致轻度头晕
3. 然后做深长缓慢的呼气
4. 重复3-5次
5. 身体会放松

注意：
├── 不要过度换气太久
├── 感到不适立即停止
├── 最好有人在旁
└── 不适合有心血管问题的人

---

规律呼吸练习：
目标：建立稳定的呼吸模式
1. 每天固定时间练习
2. 至少10-15分钟
3. 专注于呼吸
4. 逐步延长时间
5. 融入日常生活
```

### 2.2 身体放松
```
渐进性肌肉放松（PMR）：

原理：
肌肉紧绷会产生紧张感
通过先紧绷后放松来体验放松感

步骤：
1. 从脚开始
2. 收紧肌肉（5-7秒）
3. 感受紧张感
4. 放松肌肉（15-20秒）
5. 感受放松感
6. 继续下一个部位

顺序：
脚 → 小腿 → 大腿 → 臀部
→ 腹部 → 胸部 → 背部
→ 手 → 前臂 → 上臂
→ 肩膀 → 颈部 → 脸部

注意：
├── 在安静的地方
├── 穿宽松的衣服
├── 不要过度紧绷
├── 感受放松而不是紧绷
└── 每天练习15-20分钟

---

其他放松技术：

想象放松：
1. 想象一个让你放松的地方
2. 用所有感官感受
3. 在那里停留10-15分钟
4. 让身体慢慢放松

---

温热放松：
1. 泡热水澡
2. 热敷
3. 喝热饮
4. 温暖促进放松

---

身体活动：
1. 快走
2. 慢跑
3. 瑜伽
4. 太极
5. 跳舞
```

## 3. 认知调节

### 3.1 认知重评
```
认知重评的定义：
认知重评是通过改变对情绪诱发事件的解释和理解来改变情绪体验。

重评 vs 压抑：
├── 重评：改变想法
├── 压抑：抑制情绪
├── 重评：有效降低情绪
├── 压抑：可能适得其反
└── 重评是更健康的策略

重评的步骤：

步骤1：识别自动思维
"当我遇到这个情况时，我在想什么？"
记录下你的自动想法

步骤2：评估思维
├── 这是事实还是推测？
├── 有证据支持吗？
├── 我在灾难化吗？
├── 有其他解释吗？
└── 这个想法有帮助吗？

步骤3：生成替代思维
├── 更平衡的版本
├── 更现实的版本
├── 更温和的版本
├── 更接纳的版本
└── 更积极的版本

步骤4：应用替代思维
将替代思维应用到情况中
感受情绪的变化

---

重评示例：

情况：同事没有回复你的消息

自动思维：
"他一定对我有意见了"

重评：
"他可能只是很忙"
"也许他没看到消息"
"这不一定是针对我的"
"即使他有意见，也可以沟通解决"

情绪变化：
焦虑 → 平静/理解
```

### 3.2 接纳与正念
```
接纳情绪：

接纳的要点：
├── 情绪是暂时的
├── 情绪是人类正常体验
├── 抗拒情绪会加强情绪
├── 接纳情绪会减少情绪
└── 情绪本身不会伤害你

接纳练习：
1. 注意情绪
2. 不要评判
3. 给情绪命名
4. 允许情绪存在
5. 观察情绪变化
6. 不需要改变或消除

---

正念调节：

什么是正念：
有意识地、不评判地专注于当下

正念调节情绪的步骤：

1. 暂停
   停止当前活动
   深呼吸几次

2. 觉察
   注意当下的体验
   注意身体感受
   注意情绪

3. 不评判
   不要评判好坏
   只是观察
   允许事物如其所是

4. 标签
   给情绪贴标签
   "这是焦虑"
   "这是悲伤"

5. 扩展
   给情绪空间
   不要紧抓或推开
   允许情绪流动

6. 继续
   带着觉察继续活动
   或者继续停留在正念中

正念练习：
每天5-10分钟
从短时间开始
保持规律练习
慢慢增加时长
```
