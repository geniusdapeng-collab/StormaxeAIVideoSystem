## TL;DR

情绪管理 = 觉察情绪 + 接纳情绪 + 调节情绪 + 表达情绪

## 1. 情绪基础认知

### 1.1 情绪分类
```
基本情绪：
├── 喜：开心、快乐、兴奋
├── 怒：生气、愤怒、烦躁
├── 哀：悲伤、难过、沮丧
├── 惧：害怕、焦虑、担忧
└── 惊：惊讶、震惊

复杂情绪：
├── 愧疚：做了对不起别人的事
├── 嫉妒：别人比自己好
├── 委屈：觉得不公平
├── 尴尬：做了丢脸的事
└── 失望：期望没达到
```

### 1.2 情绪影响
```
情绪对工作的影响：
├── 积极情绪：效率高、创意多、人际关系好
├── 消极情绪：效率低、易出错、影响协作
└── 极端情绪：决策失误、破坏关系

情绪影响范围：
├── 工作效率
├── 决策质量
├── 人际关系
├── 身体健康
└── 职业发展
```

## 2. 情绪觉察

### 2.1 觉察方法
```
情绪觉察信号：
├── 身体信号：心跳加快、呼吸急促、肌肉紧张
├── 行为信号：语速变化、动作变化、表情变化
├── 思维信号：想法消极、钻牛角尖
└── 情绪信号：明显感觉到某种情绪

自我观察：
□ 我现在感觉怎么样？
□ 我的情绪是什么？
□ 这个情绪有多强烈？
□ 是谁/什么事引发了这个情绪？
□ 我想怎么处理这个情绪？
```

### 2.2 情绪日记
```
情绪日记模板：
日期：XX月XX日
时间：XX:XX
情境：发生了XX事
情绪：感到XX（情绪名称）
强度：1-10分
原因：我觉得XX
处理：我做了XX
结果：情绪变得XX
```

## 3. 情绪调节技巧

### 3.1 即时调节
```
快速冷静技巧：
├── 深呼吸：4-7-8呼吸法（吸4秒-屏7秒-呼8秒）
├── 数数：从1数到10
├── 暂时离开：去洗手间冷静一下
├── 喝水：喝杯水让自己冷静
└── 微笑：强迫自己微笑

延迟反应：
1. 意识到自己情绪激动
2. 告诉对方"我需要想一想"
3. 离开现场，找安静地方
4. 等冷静后再回来沟通
```

### 3.2 认知调节
```
ABC情绪理论：
├── A（Activating event）：事件
├── B（Belief）：信念/想法
└── C（Consequence）：结果/情绪

调节方法：
├── 质疑想法：这个想法是真的吗？
├── 换个角度：还有别的解释吗？
├── 找证据：有什么证据支持这个想法？
├── 后果分析：如果这样想会怎样？
└── 更理性：更平衡的想法是什么？
```

### 3.3 压力管理
```
压力来源：
├── 工作量：任务太多、太难
├── 时间：截止日期临近
├── 人际：冲突、矛盾
├── 职业发展：晋升、加薪
└── 工作环境：噪音、拥挤

压力缓解方法：
├── 分解任务：大任务分解小步骤
├── 时间管理：合理安排
├── 寻求支持：找人倾诉
├── 放松技巧：冥想、运动
└── 调整期望：设定合理目标
```

## 4. 职场情绪场景

### 4.1 被批评
```
被批评的反应：
❌ 立即反驳
❌ 找借口辩解
❌ 情绪崩溃
❌ 记仇报复

正确反应：
✅ 先听完，不要打断
✅ 感谢对方反馈
✅ 问清楚具体问题
✅ 表示会改进
✅ 事后复盘改进
```

### 4.2 被抢功
```
被抢功的反应：
❌ 当场大吵
❌ 背后抱怨
❌ 消极怠工
❌ 记仇报复

正确反应：
✅ 冷静，不要当场发作
✅ 私下沟通，说明情况
✅ 保留证据，保护自己
✅ 以后注意提前汇报
✅ 评估关系，决定是否继续合作
```

### 4.3 与领导冲突
```
与领导冲突的处理：
1. 尊重领导权威
2. 选择合适时机沟通
3. 用数据和事实说话
4. 提供解决方案
5. 服从决定（即使不完全认同）
6. 事后复盘，避免再次冲突
```

## 5. 积极心态培养

### 5.1 心态调整
```
积极心态：
├── 成长思维：失败是学习机会
├── 主动负责：是我的问题我来解决
├── 感恩心态：感谢帮助过你的人
├── 接纳不完美：没有人是完美的
└── 关注能改变的：无法改变的事放下

心态自检：
□ 我关注的是问题还是解决方案？
□ 我是在抱怨还是在解决？
□ 我能控制的是什么？
□ 一年后这还重要吗？
□ 有没有值得感谢的？
```

### 5.2 能量管理
```
能量来源：
├── 身体：睡眠、运动、饮食
├── 情绪：积极关系、兴趣爱好
├── 思维：学习、挑战、成长
└── 精神：价值观、使命感

每日能量管理：
├── 早晨：计划、锻炼
├── 上午：最重要的工作
├── 中午：休息、放松
├── 下午：协作、沟通
└── 晚上：复盘、休息
```

需要寻求帮助的信号：

├── 无法正常工作
├── 有伤害自己念头
