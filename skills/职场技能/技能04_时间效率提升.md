## TL;DR

时间效率 = 计划先行 + 优先级管理 + 专注执行 + 定期复盘

## 1. 时间管理基础

### 1.1 时间管理矩阵

| | 紧急 | 不紧急 |
|---|------|--------|
| **重要** | 🔥 立即做 | 📅 计划做 |
| **不重要** | ⚡ 委托做 | ❌ 删除做 |

### 1.2 每日时间分配
```
高效工作日
├── 深度工作（4小时）← 最重要的工作
├── 沟通协作（2小时）← 会议、邮件、沟通
├── 日常事务（1小时）← 例行工作
└── 学习成长（1小时）← 反思、总结、学习
```

## 2. 计划管理

### 2.1 每日计划
```
早晨10分钟：计划当天工作
├── 确定3件最重要的事
├── 预估每件事的时间
├── 预留缓冲时间
└── 留出应急时间

晚上10分钟：复盘当天工作
├── 完成情况
├── 未完成原因
├── 改进措施
└── 明日计划
```

### 2.2 周计划
```
周初30分钟：规划本周工作
├── 明确本周核心目标
├── 分解为每日任务
├── 预留重点项目时间
└── 安排协作沟通时间

周末30分钟：复盘本周工作
├── 目标完成情况
├── 收获与教训
└── 下周改进计划
```

## 3. 优先级管理

### 3.1 每日三件事法则
```
原则：
1. 每天只选3件最重要的事
2. 优先完成这3件事
3. 其他事情利用剩余时间

选择标准：
□ 这件事对目标贡献大吗？
□ 今天不做会怎样？
□ 做完会有什么收益？
```

### 3.2 艾森豪威尔矩阵应用
```
任务分类
├── 象限1（重要+紧急）：立即执行
│   └── 危机、紧急问题、deadline
├── 象限2（重要+不紧急）：计划执行
│   └── 规划、预防、增值工作
├── 象限3（不重要+紧急）：尽量委托
│   └── 干扰、某些会议、某些邮件
└── 象限4（不重要+不紧急）：删除
    └── 浪费时间、无意义社交
```

## 4. 专注力提升

### 4.1 深度工作法
```
深度工作定义：不受干扰的全神贯注工作

深度工作时间安排：
├── 早晨黄金时间（9-11点）
├── 下午高效时间（2-4点）
└── 每次至少90分钟

深度工作技巧：
1. 关闭所有通知
2. 告知他人不要打扰
3. 准备水、咖啡
4. 设定明确目标
5. 完成后休息15分钟
```

### 4.2 番茄工作法
```
标准番茄钟：
├── 工作25分钟
├── 休息5分钟
├── 每4个番茄钟休息15-30分钟

应用步骤：
1. 选择任务
2. 设定25分钟倒计时
3. 专注工作，不中断
4. 计时结束休息5分钟
5. 循环
```

### 4.3 战胜拖延
```
拖延原因：
├── 任务太大，不知从何开始
├── 完美主义，害怕做不好
├── 任务无聊，缺乏动力
└── 目标不清，不知道为什么做

战胜拖延方法：
1. 分解任务：大任务分解为小步骤
2. 2分钟原则：2分钟能做的事立刻做
3. 设定截止：给自己设deadline
4. 改变环境：远离干扰源
5. 奖励机制：完成后给自己奖励
```

## 5. 会议效率

### 5.1 减少无效会议
```
会议筛选：
□ 这个会议必须开吗？
□ 能用邮件/消息解决吗？
□ 我必须参加吗？

无效会议特征：
❌ 无明确目标
❌ 无准备议程
❌ 人数太多
❌ 讨论无边界
❌ 无结论无行动
```

### 5.2 高效会议
```
会前准备：
├── 明确会议目标
├── 准备会议议程
├── 提前发送资料
└── 确保必要人员参加

会中控制：
├── 按时开始
├── 严格按议程
├── 控制讨论范围
├── 记录关键结论
└── 明确行动项

会后跟进：
├── 发送会议纪要
├── 跟踪行动项
└── 及时反馈结果
```
