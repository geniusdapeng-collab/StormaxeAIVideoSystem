## 2. 沟通协作能力

### 2.1 职场表达技巧

#### 清晰表达四步法
```
1. 结论先行：先说结论，再说原因
2. 逻辑清晰：按时间/结构/重要性排列
3. 数据支撑：用数据说话，更有说服力
4. 换位思考：考虑听众关注点
```

#### 常用表达场景

| 场景 | 方法 | 示例 |
|------|------|------|
| 汇报工作 | 结果→过程→问题 | "项目已完成，ROI达15%，遇到XX问题已解决" |
| 跨部门沟通 | 目标→需求→共识 | "为了达成Q3目标，需要XX部门配合" |
| 会议发言 | 观点→论据→行动 | "我认为应该XX，理由是YY，建议ZZ" |
| 书面沟通 | 目的→内容→要求 | "请查收XX，如有问题请在YY前反馈" |

### 2.2 职场倾听技巧

#### 高效倾听五步
```
1. 专注：放下手机，眼神交流
2. 理解：不打断，记录要点
3. 确认：复述对方观点确认理解
4. 反馈：表达理解和共情
5. 行动：明确下一步行动
```

### 2.3 跨部门协作

#### 协作成功要素
```
1. 明确目标：共同目标是什么
2. 清晰职责：各自负责什么
3. 及时沟通：定期同步进展
4. 互相理解：尊重其他部门立场
5. 共同庆功：成果一起分享
```

## 3. 时间管理能力

### 3.1 日程规划

#### 每日时间分配
```
├── 深度工作（2-4小时）← 最重要的工作
├── 沟通协作（2-3小时）← 会议、邮件、沟通
├── 日常事务（1-2小时）← 例行工作
└── 学习成长（0.5-1小时）← 反思、总结、学习
```

#### 时间块工作法
```
1. 早晨30分钟：计划当天工作
2. 上午9-11点：深度工作（最重要的事）
3. 上午11-12点：会议和沟通
4. 下午2-4点：深度工作或协作
5. 下午4-5点：收尾和明日计划
```

### 3.2 优先级管理

#### 重要紧急矩阵

| | 紧急 | 不紧急 |
|---|------|--------|
| **重要** | ⏰ 立刻做 | 📅 计划做 |
| **不重要** | ⚡ 委托做 | ❌ 删除做 |

#### 每日三件事法则
```
1. 每天早上确定3件最重要的事
2. 优先完成这3件事
3. 其他事情利用剩余时间处理
```

### 3.3 效率提升技巧

#### 克服拖延
```
1. 2分钟原则：2分钟内能做的事立刻做
2. 分解任务：大任务分解为小步骤
3. 设置截止：给自己设Deadline
4. 消除干扰：关掉无关通知
5. 奖励机制：完成后给自己奖励
```
