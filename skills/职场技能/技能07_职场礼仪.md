## TL;DR

职场礼仪 = 守时 + 尊重 + 诚信 + 专业 + 适度

## 1. 基本礼仪原则

### 1.1 核心原则
```
职场礼仪5原则：
1. 守时：准时是基本的尊重
2. 尊重：尊重他人时间、工作、人格
3. 诚信：说到做到，诚实守信
4. 专业：工作场合保持专业形象
5. 适度：不过不失，恰到好处
```

### 1.2 礼仪自检
```
每日礼仪自检：
□ 准时到达
□ 着装得体
□ 手机静音
□ 守信用
□ 微笑问候
□ 主动帮助
□ 及时回复
□ 感谢他人
```

## 2. 形象礼仪

### 2.1 着装规范
```
着装原则：
├── 干净整洁：衣服干净、无异味
├── 得体合适：符合场合要求
├── 适度正式：比公司要求稍正式
└── 个人风格：适度展现个性

场合着装：
├── 日常办公：商务休闲
├── 商务会议：正式商务装
├── 外出拜访：正式商务装
├── 团建活动：商务休闲
└── 周五便装：根据公司规定
```

### 2.2 仪容仪表
```
仪容要求：
├── 整洁干净：头发、面部、指甲
├── 气味清新：口气、体味
├── 妆容得体：女性适度化妆
└── 配饰简约：不过于夸张
```

## 3. 沟通礼仪

### 3.1 语言礼仪
```
礼貌用语：
├── 见面：你好、早上好
├── 请求：请、麻烦你
├── 感谢：谢谢、辛苦了
├── 道歉：对不起、请原谅
├── 告别：再见、回头聊
└── 赞美：做得真好、很棒

禁忌语言：
❌ 粗话、脏话
❌ 抱怨、负面的话
❌ 八卦、谣言
❌ 讽刺、挖苦
❌ 不尊重的话
```

### 3.2 邮件礼仪
```
邮件礼仪：
├── 主题：清晰明确
├── 称谓：Dear XX / 你好 XX
├── 正文：简洁有条理
├── 附件：命名规范
├── 签名：信息完整
└── 发送前检查收件人

邮件检查清单：
□ 主题是否清晰
□ 称谓是否恰当
□ 内容是否简洁
□ 重点是否突出
□ 附件是否上传
□ 收件人是否正确
```

### 3.3 即时通讯礼仪
```
消息礼仪：
├── 及时回复：如无法回复先确认收到
├── 分段发送：长消息分段
├── 重要信息：加@提醒
├── 时间注意：非工作时间避免发消息
└── 内容规范：避免纯表情、纯语音
```

## 4. 会议礼仪

### 4.1 会议参与
```
会议礼仪：
├── 准时：提前5分钟进入
├── 静音：进入会议静音
├── 专注：认真听、记笔记
├── 发言：举手或等待邀请
├── 手机：静音或关机
└── 离场：必须先说明原因
```

### 4.2 会议发言
```
发言礼仪：
├── 简明扼要：不说废话
├── 有理有据：观点有支撑
├── 尊重他人：不打断别人
├── 控制时间：不超时
└── 积极贡献：提供有价值意见
```

## 5. 用餐礼仪

### 5.1 工作餐
```
工作餐礼仪：
├── AA制：一般各自付账
├── 邀请：谁邀请谁付账
├── 点菜：考虑多数人口味
├── 敬酒：适度、不强迫
└── 用餐：保持优雅、不出声
```

### 5.2 商务宴请
```
商务宴请礼仪：
├── 邀请：提前发出正式邀请
├── 座位：主人坐主位
├── 点菜：询问客人喜好
├── 敬酒：按座次顺序
├── 话题：避免敏感话题
└── 结束：主人宣布结束
```

## 6. 社交礼仪

### 6.1 称呼礼仪
```
称呼原则：
├── 正式场合：职称/姓名
├── 熟悉后：名字
├── 不确定：先生/女士
├── 记住：不要叫错名字
└── 尊重：使用敬称

常见称呼：
├── 领导：张总、王经理
├── 同事：张姐、王哥
├── 客户：张先生、李女士
└── 长辈：张老师、李前辈
```

### 6.2 介绍礼仪
```
介绍顺序：
├── 先卑后尊：先介绍位低者
├── 先主后宾：先介绍主人
├── 先男后女：先介绍男性

介绍内容：
├── 姓名
├── 公司/部门
├── 职位
└── 与会关系
```
