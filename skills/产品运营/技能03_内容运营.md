## TL;DR

内容运营 = 内容定位 + 选题规划 + 内容生产 + 内容分发 + 数据分析

## 1. 内容定位

### 1.1 内容定位方法
```
内容定位三要素：

目标用户：
├── 用户是谁
├── 用户画像
├── 用户需求
├── 用户痛点
└── 内容偏好

产品价值：
├── 产品解决什么问题
├── 产品核心功能
├── 产品差异化
├── 产品使用场景
└── 产品品牌调性

内容调性：
├── 专业/通俗
├── 严肃/活泼
├── 官方/亲民
├── 深度/浅出
└── 风格统一

内容定位公式：
"为XX用户，提供XX价值的内容"
例如：
"为职场人，提供职场技能提升的内容"
"为妈妈群体，提供育儿知识的内容"
```

### 1.2 内容规划
```
内容规划矩阵：

| 内容类型 | 占比 | 目的 | 频率 |
|----------|------|------|------|
| 干货内容 | 40% | 价值输出 | 每周2篇 |
| 产品内容 | 30% | 产品推广 | 每周1篇 |
| 热点内容 | 20% | 流量获取 | 随时 |
| 互动内容 | 10% | 用户互动 | 每周1篇 |

内容主题规划：
├── 常青树话题：长期有价值的
├── 热点话题：当下热点
├── 产品话题：产品相关
├── 用户话题：用户关心的
└── 竞品话题：行业相关
```

## 2. 选题策划

### 2.1 选题来源
```
选题来源渠道：

用户需求：
├── 用户调研问卷
├── 用户反馈收集
├── 评论区需求
├── 私信需求
└── 客服问题

数据分析：
├── 阅读量高的内容
├── 搜索关键词
├── 用户停留时长
├── 收藏量高的内容
└── 分享量高的内容

竞品分析：
├── 竞品热门内容
├── 行业热点话题
├── 平台热门内容
└── KOL选题方向

热点追踪：
├── 节假日热点
├── 行业热点
├── 社会热点
├── 平台热点
└── 突发事件

自创选题：
├── 日常积累
├── 团队脑暴
├── 专家建议
└── 内容延伸
```

### 2.2 选题方法
```
选题评估模型：

ICE评分法：
├── Impact（影响力）：覆盖用户量
├── Confidence（信心）：选题把握度
├── Ease（难易）：内容生产难度
└── 得分 = I × C × E

PASS评分法：
├── Popular（热度）：用户关注度
├── Accurate（准确）：选题匹配度
├── Simple（简单）：内容可执行
├── Strong（有力）：内容说服力
└── 综合评分

选题优先级：
├── P0：高ICE + 高价值
├── P1：高ICE + 中价值
├── P2：中ICE + 中价值
└── P3：低ICE + 低价值

选题清单：
├── 标题是否吸引人
├── 内容是否有价值
├── 用户是否需要
├── 是否符合定位
├── 是否有传播点
└── 是否能持续生产
```

## 3. 内容生产

### 3.1 内容形式
```
内容形式分类：

图文内容：
├── 文章：深度文章、短文章
├── 图集：图片合集
├── 信息图：数据可视化
├── 问答：Q&A形式
└── 测评：产品测评

视频内容：
├── 短视频：15-60秒
├── 中视频：1-10分钟
├── 长视频：10分钟+
├── 直播：实时互动
└── Vlog：个人记录

音频内容：
├── 播客：音频节目
├── 语音：语音文章
├── 音乐：音乐内容
└── 声音特效：趣味音频

互动内容：
├── 测试：性格测试、趣味测试
├── 投票：意见投票
├── 问答：互动问答
├── 抽奖：内容互动
└── 游戏：内容游戏
```

```

1. 选题确定

   └── 选题确认

   ├── 大纲设计

   ├── 内容编辑

4. 内容发布
   ├── 标题优化
   ├── 封面设计
   ├── 标签设置
   └── 发布上传

5. 效果追踪
   ├── 数据监控
   ├── 用户反馈
   ├── 内容优化
   └── 经验总结

时间安排：
├── 日常内容：提前3-5天准备
├── 深度内容：提前1-2周准备
├── 热点内容：快速响应，24小时内

```
