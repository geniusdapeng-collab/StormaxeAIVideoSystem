## TL;DR

数据运营 = 指标体系 + 数据监控 + 数据分析 + 数据驱动

## 1. 指标体系

### 1.1 核心指标
```
运营核心指标体系：

用户指标：
├── 规模指标：DAU/MAU/注册用户数
├── 质量指标：留存率、活跃度
├── 来源指标：新增用户、渠道分布
└── 价值指标：LTV、ARPU

业务指标：
├── 流量指标：PV/UV/访问时长
├── 转化指标：转化率、跳出率
├── 交易指标：GMV、订单量、客单价
└── 复购指标：复购率、购买频次

产品指标：
├── 功能使用：功能渗透、功能活跃
├── 体验指标：加载速度、崩溃率
├── 内容指标：内容生产、内容消费
└── 互动指标：点赞、评论、分享
```

### 1.2 指标定义
```
指标定义模板：

指标名称：DAU（日活跃用户数）
指标定义：当日启动过APP的用户数
计算公式：COUNT(DISTINCT user_id) WHERE date = TODAY
数据来源：用户行为日志
更新频率：每日
指标类型：核心指标
历史均值：XX万
波动范围：±5%
异常阈值：±10%
负责人：XX
```

```

├── 绿色：正常（波动±5%）
├── 黄色：关注（波动±10%）
├── 橙色：警告（波动±20%）
└── 红色：紧急（波动±30%+）

预警处理：
├── 确认数据准确性

├── 评估影响范围
```
