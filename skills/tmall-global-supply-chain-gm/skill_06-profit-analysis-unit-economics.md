# TL;DR

不懂单元经济的GM不是好GM——每一笔订单、每一个SKU、每一个品类，都必须算清楚账，用数据驱动决策。
# 场景分析

## 典型场景
- 整体账面上赚钱，但不知道哪个品类在拖累
- 销售额增长了30%，但利润反而下滑了
- 不知道推广费该不该花，ROI怎么算才算清楚
- 大促GMV很好看，但算上所有费用其实是亏损的

## 正确认知
- **GM最应该盯的不是GMV，是净利润**
- **单元经济是生死线，卖一单亏一单的模式不可持续**
- **成本结构分析比成本绝对值更重要**
- **财务数据不是月底才看，是每天都要追踪**

# 决策框架

## 第一步：分析对象确定

```
分析维度选择：

├─ 整体P&L（GM级别）
│   └─ 月度/季度复盘，评估整体盈利能力
├─ 品类P&L（品类总监级别）
│   └─ 哪个品类在赚钱，哪个在拖累
├─ SKU P&L（运营总监级别）
│   └─ 商品级盈利能力，优化商品结构
└─ 订单单元经济（运营专员级别）
    └─ 单笔订单是否盈利，ROI是否合理

分析频率：
├─ 日常：核心指标仪表盘（日更）
├─ 周度：推广ROI追踪（周更）
├─ 月度：品类P&L复盘（月度经营会）
└─ 季度：全维度深度分析（战略会议）
```

## 第二步：数据收集

### 数据清单

| 数据类型 | 数据来源 | 频率 | 负责人 |
|----------|----------|------|--------|
| GMV/订单量 | 店铺后台 | 日 | 运营专员 |
| 毛利率 | 财务系统 | 月 | 财务专员 |
| 推广花费 | 推广后台 | 日 | 运营专员 |
| 退货率 | 客服系统 | 周 | 客服主管 |
| 库存周转 | ERP | 周 | 供应链 |
| 现金流 | 财务系统 | 月 | 财务总监 |

### 数据准确性校验

```
原始数据 → 数据清洗 → 归因处理 → 可用数据

├─ 原始数据校验：
│   ├─ GMV：退款后GMV vs 退款前
│   ├─ 成本：采购成本含税+物流+关税
│   └─ 推广费：各渠道花费汇总（含退款的扣减）
│
├─ 归因处理：
│   ├─ 多渠道归因：各渠道按比例分配GMV
│   └─ 时间窗口：7天/14天/30天归因
│
└─ 数据交叉验证：
    ├─ GMV × 毛利率 ≈ 毛利额？
    └─ 推广花费 ÷ GMV × 100 = 推广费率？
```

## 第三步：单元经济测算

```
单笔订单盈亏计算：

Step 1：收入端
→ 客单价 = 销售额 ÷ 订单数
→ 净收入 = GMV × 实际转化率

Step 2：成本端
→ 商品成本 = 采购价 + 关税 + 头程
→ 平台扣点 = GMV × 扣点率
→ 履约成本 = 尾程配送 + 包装 + 操作费
→ 退货成本 = GMV × 退货率 × 退货处理成本率
→ 获客成本 = 推广花费 ÷ 归因订单数

Step 3：利润测算
→ 毛利 = 净收入 - 商品成本
→ 毛利率 = 毛利 ÷ 净收入
→ 净贡献 = 毛利 - 平台扣点 - 履约成本 - 退货成本 - 获客成本

Step 4：判断
→ 净贡献 > 0 ✅ 盈利
→ 净贡献 = 0 ⚠️ 盈亏平衡，密切关注
→ 净贡献 < 0 ❌ 亏损，需立即优化

盈亏平衡点公式：
最低客单价 = (商品成本 + 平台扣点 + 履约成本) ÷ 毛利率
```

## 第四步：问题诊断

### 问题分解树

```
净利润低于目标？
├─ 毛利低？
│   ├─ 毛利率低 → 定价太低 or 商品成本高
│   │   └─ 对比竞品定价，对比供应商报价
│   └─ GMV低 → 流量少 or 转化低
│       └─ 检查流量结构，页面转化率
├─ 费用高？
│   ├─ 推广费高 → ROI低
│   │   └─ 哪些渠道/关键词ROI差
│   ├─ 退货率高 → 商品质量/描述问题
│   │   └─ 分析退货原因TOP3
│   └─ 仓储费高 → 库存周转慢
│       └─ 检查滞销SKU
└─ 资金效率低？
    ├─ 库存周转天数高 → 备货策略问题
    └─ 账期问题 → 结算方式优化
```

# 专业建议

## P&L管理

### 跨境电商P&L结构

```
营业收入
  └─ 商品销售额（GMV × 实际转化率）

营业成本
  ├─ 商品采购成本
  ├─ 关税+增值税+消费税
  ├─ 头程物流费
  └─ 头程保险费

毛利润 = 营业收入 - 营业成本

营业费用
  ├─ 平台扣点（天猫佣金+支付宝费）
  ├─ 推广费用（直通车+超级推荐+钻展+淘客）
  ├─ 仓储费用（保税仓+操作费）
  ├─ 尾程配送费
  ├─ 退货处理费
  ├─ 人员成本
  ├─ 运营成本（包材+损耗+其他）
  └─ 管理费用

营业利润 = 毛利润 - 营业费用

净利润 = 营业利润 - 税前利润 - 所得税
```

### 品类P&L示例

```
美妆品类P&L（2026年Q1）

一、收入
├─ GMV：1500万
├─ 实际转化率：65%
└─ 净收入：975万

二、商品成本
├─ 采购成本：600万（40% of GMV）
├─ 关税：45万（3% of GMV）
├─ 头程物流：75万（5% of GMV）
└─ 总成本：720万

三、毛利润
└─ 255万（17% of GMV，margin=26%）

四、营业费用
├─ 平台扣点：112.5万（7.5% of GMV）
├─ 推广费用：225万（15% of GMV）
├─ 仓储尾程：90万（6% of GMV）
├─ 退货处理：22.5万（1.5% of GMV）
├─ 人员成本：45万（3% of GMV）
└─ 总费用：495万

五、营业利润
└─ 255万 - 495万 = -240万（亏损16%）

六、改善建议
├─ 毛利率过低 → 提升中高端占比
├─ 推广费过高 → 优化ROI，提升自然流量
└─ 退货率高 → 提升商品质量
```

## 单元经济分析

### 订单级单元经济

```
单元经济 = 每笔订单的贡献 - 每笔订单的成本

每笔订单贡献 = 客单价 × 毛利率
每笔订单成本 = 获客成本 + 履约成本 + 退货成本

示例：
- 客单价：200元
- 商品毛利率：30%
- 订单贡献：200 × 30% = 60元

- 获客成本（按CAC）：20元
- 平台扣点：200 × 7.5% = 15元
- 尾程配送：10元
- 退货成本：200 × 5% × 1.5% = 1.5元
- 订单总成本：46.5元

单元经济 = 60 - 46.5 = 13.5元（盈利）

盈亏平衡点：获客成本上限 = 客单价 × 毛利率 - 履约成本 = 60 - 26.5 = 33.5元
```

## 成本结构分析

### 全成本结构拆解

```
跨境电商全成本 = 商品成本 + 运营成本 + 资金成本

一、商品成本（占GMV的45-55%）
├─ 采购成本：35-45%
├─ 关税/综合税：5-8%
└─ 头程物流：3-5%

二、平台运营成本（占GMV的25-35%）
├─ 平台扣点：6-8%
├─ 推广费：8-15%
├─ 仓储尾程：4-6%
└─ 退货/损耗：1-3%

三、资金成本（占GMV的2-5%）
├─ 库存资金占用：1-3%
└─ 账期成本：1-2%

盈亏平衡线：
毛利率 > 运营成本率 + 资金成本率
即：毛利率 > 40-50%
```

### 成本优化路径

| 成本项 | 优化空间 | 优化策略 |
|--------|----------|----------|
| 采购成本 | 5-15% | 集中采购、供应商整合 |
| 关税 | 3-10% | FTA利用、商品归类优化 |
| 头程物流 | 5-20% | 批量发货、物流商比价 |
| 推广费 | 10-30% | ROI优化、渠道整合 |
| 仓储尾程 | 3-10% | 仓库优化、批量操作 |
| 退货 | 10-30% | 质量改进、描述准确 |

## 资金效率分析

### 现金循环周期

```
现金循环周期 = 库存周转天数 + 应收账款天数 - 应付账款天数

示例：
- 采购付款：Day 0
- 生产+头程：Day 1-30
- 入库上架：Day 30
- 销售等待：Day 30-37
- 收到款项：Day 37-44

库存周转天数：37天
应收账款天数：0天（平台直接结算）
应付账款天数：30天

现金循环周期 = 37 - 0 - 30 = 7天（优秀！）
```

### ROI深度分析

```
ROI = (收益 - 投入) ÷ 投入 × 100%

营销ROI计算：
├─ 广义ROI：GMV贡献 ÷ 营销投入（包含归因比例）
├─ 侠义ROI：直接转化GMV ÷ 营销投入
└─ 真实ROI：归因后毛利贡献 ÷ 营销投入

示例（侠义ROI）：
- 直通车花费：100万
- 直通车带来GMV：350万
- 侠义ROI = 350/100 = 3.5倍

示例（真实ROI）：
- 归因比例：70%
- 归因GMV：350 × 70% = 245万
- 商品毛利率：30%
- 毛利贡献：245 × 30% = 73.5万
- 真实ROI = (73.5 - 100) / 100 = -26.5%（亏损！）

结论：只看GMV会误判，必须看归因毛利！
```
