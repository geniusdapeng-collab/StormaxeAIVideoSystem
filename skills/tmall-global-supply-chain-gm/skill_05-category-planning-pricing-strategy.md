# TL;DR

品类结构是GM的方向盘，定价策略是GM的利润密码——品类规划决定你能赚多少钱，定价决定你能卖多少货。
# 场景分析

## 典型场景
- 毛利率越来越低，但不知道哪个品类在拖累整体
- SKU越来越多，但80%的销量来自20%的商品
- 定价总是被竞品牵着走，没有自己的节奏
- 新品引入成功率不到30%，库存压了一堆

## 正确认知
- **品类结构决定生意质量，定价决定生意思路**
- **不是SKU越多越好，精 SKU > 多 SKU**
- **定价不是越低越好，是找到利润最大化的价格**
- **新品引入要有漏斗，成功率低不可怕，可怕的是不知道为什么不成功**

# 决策框架

## 第一步：品类现状诊断

### 品类健康度评估

```
品类问题诊断树：

是新品引入还是存量优化？
├─ 新品引入
│   └─ 进入"新品引入决策流"
└─ 存量优化
    ├─ 哪个指标出问题？
    │   ├─ 毛利率低 → 成本优化 or 定价调整
    │   ├─ GMV占比低 → 流量提升 or 淘汰替换
    │   ├─ SKU积压 → 促销清仓
    │   └─ 动销率低 → 优化选品
    └─ 问题严重程度？
        ├─ 毛利占比<20%的品类拖累整体 → 立即优化
        └─ 毛利占比>20%的品类问题 → 制定改善计划
```

### 品类结构健康度矩阵

| 指标 | 健康值 | 预警值 | 危险值 |
|------|--------|--------|--------|
| A类爆款GMV占比 | 60% | <50% | <40% |
| SKU淘汰率（年） | 15-20% | <10% | 0% |
| 新品成功率 | >50% | <30% | <20% |
| 品类动销率 | >80% | <60% | <40% |
| 品类集中度 | <60%（单一品类） | >70% | >80% |

## 第二步：定价决策

### 定价策略选择

```
用户价格敏感度评估？
├─ 高（竞品比价频繁）
│   └─ 竞品定价 + 成本底线保护
│       → 参考竞品价格 × (0.95-1.05)
│       → 毛利率底线：不能低于15%
└─ 低（品牌/品质驱动）
    └─ 价值定价 + 品牌溢价
        → 用户感知价值 × 溢价系数（1.1-1.3）
        → 参考品牌溢价空间

市场阶段判断？
├─ 新品期（引入期）
│   └─ 渗透定价：低价切入，抢占市场份额
│       → 可接受毛利率：10-20%
│       → 目标：6个月内进入品类Top 20
├─ 成长期
│   └─ 竞争定价：随竞品动态调整
│       → 毛利率：20-30%
│       → 目标：稳定市场份额
├─ 成熟期
│   └─ 利润定价：榨取最大利润
│       → 毛利率：30-40%
│       → 目标：贡献最大毛利额
└─ 衰退期
    └─ 清仓定价：快速出清库存
        → 不考虑毛利，快速变现
```

### 价格调整决策

| 调整场景 | 调整方向 | 调整幅度 | 决策权限 |
|----------|----------|----------|----------|
| 竞品降价>10% | 跟价 | ≤竞品降幅 | 运营总监 |
| 竞品降价>20% | 差异化应对 | 评估后决策 | GM |
| 大促申报 | 降价 | 价格力要求 | 运营总监 |
| 滞销清仓 | 大幅降价 | 亏损可控 | GM |
| 新品测试 | 低价试销 | <成本10% | GM |

## 第三步：新品引入决策

```
新品引入决策流：

Step 1：市场机会判断
→ 市场规模 > 1亿/年？
→ 市场增速 > 10%/年？
→ 竞争度：玩家<10家 or 有差异化机会？
→ 是 → 进入Step 2；否 → 放弃

Step 2：产品匹配评估
→ 与现有品类协同？互补 or 竞争？
→ 供应链可控性？供应商稳定？备货周期？
→ 产品差异化？卖点明确？

Step 3：财务测算
→ 毛利率 > 25%？
→ 首年GMV贡献 > 100万？
→ 投资回报期 < 6个月？
→ 是 → 进入Step 4；否 → 重新评估 or 放弃

Step 4：小批量测试
→ 引入50-100件试销
→ 30天内验证：加购率>5%？转化率>3%？
→ 是 → 正式引入；否 → 复盘原因，决定继续 or 放弃

Step 5：规模化决策
→ 测试数据支撑规模化？
→ 资金充足备货？
→ 有推广计划？
→ 是 → 正式引入，全力推广
```

## 第四步：SKU淘汰决策

```
SKU淘汰触发条件（任一满足即触发）：

├─ 滞销：90天无动销 → 直接淘汰
├─ 低效：月GMV < 5000元 → 降级观察3个月，仍不达标则淘汰
├─ 亏损：连续3月毛利率 < 15% → 整改2个月，不达标则淘汰
├─ 违规：累计违规处罚 > 2次 → 直接淘汰
└─ 结构冲突：与高毛利替代品重叠 → 评估后决策

淘汰决策权限：
├─ 普通SKU：运营总监审批
├─ 爆款/形象款：GM审批
└─ 战略品牌SKU：评估影响后决策
```

# 专业建议

## 品类结构规划

### 品类分层（ABC模型）

| 层级 | 定位 | SKU占比 | GMV占比 | 毛利率目标 |
|------|------|---------|---------|------------|
| A类爆款 | 流量担当 | 10% | 60% | 25-30% |
| B类利润款 | 利润担当 | 30% | 30% | 35-45% |
| C类长尾 | 补充品类 | 60% | 10% | 20-25% |

### 品类组合策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 流量型 | 低价引流量，带动其他品类 | 新店/冲规模 |
| 利润型 | 高毛利，贡献主要利润 | 成熟店铺 |
| 形象型 | 高品质，树立品牌形象 | 品牌旗舰店 |
| 防御型 | 竞品有我也得有 | 竞争激烈品类 |

## 定价策略

### 定价方法论

| 方法 | 公式 | 适用 |
|------|------|------|
| 成本定价 | 价格 = 成本 × (1+毛利率) | 通用 |
| 竞品定价 | 价格 = 竞品价格 × (1 ± 调整系数) | 竞争市场 |
| 价值定价 | 价格 = 用户感知价值 - 溢价空间 | 高端品牌 |
| 渗透定价 | 低价切入，快速占领市场 | 新品类/新市场 |

### 定价决策树

```
用户价格敏感度？
├── 高 → 竞品定价 + 成本底线
└── 低 → 价值定价 + 品牌溢价

竞品价格？
├── 比我们低 → 差异化 + 价值说明
├── 差不多 → 成本定价 + 促销空间
└── 比我们高 → 优势定价 + 抢夺份额
```

### 价格带策略

| 价格带 | 定位 | SKU占比 | 毛利率 |
|--------|------|---------|--------|
| 引流价格带 | 入门款/小样 | 15% | 10-15% |
| 主流价格带 | 主力款 | 60% | 25-35% |
| 高端价格带 | 形象款/升级款 | 20% | 40-50% |
| 超高端 | 顶级款/限量款 | 5% | >50% |

## SKU管理

### SKU生命周期

```
引入期（0-3个月）→ 成长期（3-12个月）→ 成熟期（12-24个月）→ 衰退期（>24个月）
    ↓                    ↓                    ↓                ↓
  测试市场            冲销量             稳利润           淘汰/清仓
```

### SKU评估矩阵

| 维度 | 高销量+高毛利 | 高销量+低毛利 | 低销量+高毛利 | 低销量+低毛利 |
|------|--------------|--------------|--------------|--------------|
| 策略 | 主力培养 | 控制成本 | 提升转化 | 立即淘汰 |
| 操作 | 保持库存 | 减少投入 | 推广加码 | 清仓处理 |

### SKU淘汰机制

| 淘汰条件 | 触发标准 | 处理方式 |
|----------|----------|----------|
| 滞销 | 90天无动销 | 清仓处理 |
| 低效 | 月GMV<5000元 | 降级或淘汰 |
| 亏损 | 连续3月毛利率<15% | 整改或淘汰 |
| 违规 | 违规处罚>2次 | 直接淘汰 |

## 新品引入

### 新品评估框架

| 评估维度 | 权重 | 评估内容 |
|----------|------|----------|
| 市场机会 | 25% | 市场规模、增速、竞争度 |
| 产品力 | 25% | 差异化、品质、卖点 |
| 供应链 | 20% | 供应商稳定性、成本、备货周期 |
| 利润空间 | 20% | 毛利率、费用率、ROI |
| 契合度 | 10% | 与品类结构互补性 |

### 新品引入流程

```
Step 1：市场调研
→ 市场规模、增速、竞争分析
→ 目标用户需求调研

Step 2：产品评估
→ 供应商评估
→ 产品测试
→ 成本测算

Step 3：小批量测试
→ 引入10-50件试销
→ 收集用户反馈
→ 验证市场反应

Step 4：决策
→ 测试数据达标 → 正式引入
→ 测试数据不达标 → 放弃

Step 5：规模化
→ 正式采购备货
→ 制定推广计划
→ 持续追踪优化
```

### 新品成功率提升

| 策略 | 说明 |
|------|------|
| 轻量化测试 | 先用小批量验证，降低试错成本 |
| 快节奏迭代 | 小步快跑，快速验证快速调整 |
| 用户共创 | 让用户参与产品定义 |
| 数据驱动 | 用数据验证而非直觉 |

## 品类增长策略

|------|------|----------|
