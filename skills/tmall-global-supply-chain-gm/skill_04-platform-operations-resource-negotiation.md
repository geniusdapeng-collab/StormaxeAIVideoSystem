# TL;DR

平台关系是GM的重要外部资产——会做人的运营拿到的资源可能是竞品的3倍，这是一项需要持续经营的关系型工作。
# 场景分析

## 典型场景
- 同样是报活动，竞品每次都能上，自己却总是被拒
- 小二每次沟通都很客气，但资源就是不倾斜
- 平台规则突然变了，措手不及，损失惨重
- 想争取KA资源，但不知道找谁、怎么谈

## 正确认知
- **平台关系是战略性资产，不是靠人情靠吃喝**
- **能为平台创造价值的商家，永远不缺资源**
- **规则变化要提前预判，不能等通知了才反应**
- **GM应该亲自抓平台关系，而不是只让下属对接**

# 决策框架

## 第一步：资源需求识别

### 资源分类矩阵

| 资源类型 | 优先级 | 获取难度 | GM参与必要性 |
|----------|--------|----------|--------------|
| S级大促主会场 | P0 | 极高 | GM必争 |
| A级会场坑位 | P0 | 高 | GM重点参与 |
| 类目坑位 | P1 | 中 | 运营总监跟进 |
| 认证标识 | P1 | 低 | 达标即可 |
| 数据权限 | P2 | 低 | 日常申请 |

### 需求澄清

```
资源争取三问：
├─ 为什么是这个资源？
│   → 与GMV目标的关联是什么？
├─ 凭什么是你？
│   → 过往GMV贡献？配合度？创新力？
└─ 准备了什么？
    → 货品/价格/流量/活动方案？
```

## 第二步：关系评估

### 平台关系健康度自测

| 评估维度 | 优秀 | 合格 | 危险 |
|----------|------|------|------|
| 小二响应速度 | 即时 | <24h | >3天无回应 |
| 信息获取 | 提前透露 | 通知同步 | 公告才知道 |
| 资源分配 | 主动推荐 | 公平竞争 | 每次被拒 |
| 问题解决 | 全力支持 | 按规处理 | 推诿拖延 |
| 年度会议 | GM出席 | 总监出席 | 不被邀请 |

### 关系修复信号

| 危险信号 | 应对策略 |
|----------|----------|
| 小二更换 | 立即重新建立关系 |
| 信息变慢 | 主动约见，了解原因 |
| 活动被拒 | 复盘原因，3个月内改善 |
| 资源被竞品抢 | 分析差距，针对性提升 |

## 第三步：申报方案准备

```
申报决策流程：

Step 1：确认申报资质
→ DSR达标？动销率达标？违规清零？
→ 不达标 → 先整改达标再申报

Step 2：确定申报策略
→ 主推款（流量款/利润款/形象款）
→ 价格力（全网最低 or 历史最低）
→ 备货量（活动销量的1.5倍）

Step 3：材料准备
→ 货品方案（主推+新品+库存）
→ 价格方案（折扣+优惠券+满减）
→ 流量方案（付费+私域+内容）
→ 服务承诺（发货时效+客服响应）

Step 4：沟通小二
→ 提前沟通，争取内部推荐
→ 说明我们的优势和支持意愿
→ 准备好被挑战，有应对方案

Step 5：结果跟进
→ 申报成功 → 全力执行
→ 申报失败 → 了解原因，争取下次机会
→ 被候补 → 保持跟进，实时备货
```

## 第四步：规则变化应对

```
规则预警 → 影响评估 → 快速响应 → 反馈优化

Step 1：预警获取（日常）
→ 行业小二提前透露（非正式渠道）
→ 平台官方公告
→ 同行交流群
→ 历史规律预判（如大促前规则收紧）

Step 2：影响评估（24小时内）
→ 对GMV影响 → 正面/负面/中性
→ 对成本影响 → 费用增加/减少
→ 对竞品影响 → 我们相对受益/受损
→ 整体判断：值得重视 or 可以接受

Step 3：快速响应（48小时内）
→ 合规调整：若违规，立即整改
→ 沟通争取：若误判，提供证据申诉
→ 备选方案：若影响大，准备Plan B

Step 4：反馈优化
→ 向平台反馈执行中发现的困难
→ 提供合理化建议
→ 争取参与规则制定讨论
→ 建立预警清单，下次提前准备
```

| 层级 | 角色 | 关系频率 | GM参与度 |

### 关系经营策略

| 策略 | 具体做法 | 频率 |
|------|----------|------|
| 价值创造 | 提供市场洞察、消费者反馈 | 月度 |
| 活动共创 | 联合策划、提前沟通 | 按需 |
| 业绩证明 | 用数据说话，证明合作价值 | 月度 |
| 人情关怀 | 节日问候、日常关心 | 日常 |
| 及时响应 | 平台要求第一时间响应 | 日常 |

### 价值贡献模型

```
你为平台创造的价值 = 你能获得的资源

价值贡献维度：
├─ GMV贡献（平台GMV占比）
├─ 品牌影响力（行业排名/声量）
├─ 创新玩法（新品/内容/直播）
├─ 用户体验（DSR/满意度）
└─ 合作配合（活动参与度/规则遵守）

KA商家标准：
├─ GMV贡献：类目Top 10
├─ 品牌力：知名品牌或高成长
├─ 配合度：规则遵守+积极配合
└─ 创新力：愿意尝试新玩法
```

## 活动申报

### 申报核心要素

| 要素 | 说明 | 评审权重 |
|------|------|----------|
| 价格力 | 全网最低 or 历史最低 | 30% |
| 货品力 | 备货量充足/爆款打造 | 25% |
| 店铺力 | DSR高/动销好 | 20% |
| 配合度 | 活动规则配合 | 15% |
| 历史表现 | 过往活动GMV达成率 | 10% |

### 申报方案模板

```
活动申报方案：

一、活动定位
本次活动的核心目标和预期效果

二、销售目标
- GMV目标：X万（同比+XX%）
- 订单数：X万单
- 客单价：XX元

三、货品规划
├─ 主推款：X款，预计贡献GMV XX%
├─ 新品：X款，作为亮点
└─ 库存：活动销量的X倍

四、价格策略
├─ 主推款折扣：X折（历史最低）
├─ 优惠券：满XX减XX
└─ 价格力：符合申报要求

五、流量规划
├─ 付费流量：XX万
├─ 私域流量：触达X万人
└─ 内容预热：X篇种草内容

六、配合承诺
- 发货时效：48小时100%发货
- 客服响应：<30秒
- DSR保障：维持4.8以上
```
