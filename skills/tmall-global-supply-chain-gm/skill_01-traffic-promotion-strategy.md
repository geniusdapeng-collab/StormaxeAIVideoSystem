# TL;DR

流量是GM的命脉——掌握从年度流量规划到大促投放策略，让每一分推广费都带来最大GMV回报。

# 场景分析

## 典型场景
- 大促GMV目标1个亿，但流量只够支撑6000万，缺口从哪来？
- 推广费花了100万，但ROI只有1.5，算不过来账
- 大促报名报不上，看着竞品吃肉自己喝汤
- 日常流量持续下滑，不知道是行业问题还是自己的问题

## 核心挑战
1. **流量成本高**：获客成本每年涨20%，ROI越来越难维持
2. **大促流量争抢**：僧多粥少，资源位竞争激烈
3. **ROI难优化**：不知道推广费该往哪砸
4. **活动申报难**：不了解平台规则，报不上名

## 正确认知
- **流量运营是GM的一号工程**
- **流量不只是花钱买，更是规划和运营出来的**
- **大促成功=流量×转化率×客单价，三者缺一不可**
- **ROI是流量运营的北极星指标**

# 决策框架

## 第一步：流量结构诊断

### 流量来源拆解

```
全店流量 = 搜索流量 + 推荐流量 + 付费流量 + 内容流量 + 私域流量

各渠道占比参考：
├─ 搜索流量：35-45%（自然搜索+直通车）
├─ 推荐流量：25-35%（超级推荐/钻展）
├─ 付费流量：15-20%（直通车/钻展/淘客）
├─ 内容流量：5-10%（直播/短视频/图文）
└─ 私域流量：3-5%（会员/社群）
```

### 流量健康度评估

| 指标 | 健康值 | 预警值 | 问题诊断 |
|------|--------|--------|----------|
| 自然流量占比 | >50% | <40% | 付费依赖症 |
| 搜索流量占比 | >30% | <20% | 搜索权重低 |
| 付费ROI | >3.0 | <2.0 | 推广效率低 |
| 加购率 | >8% | <5% | 商品吸引力弱 |
| 转化率 | >3% | <2% | 页面或流量质量问题 |

## 第二步：制定流量目标

### 流量目标分解

```
GMV目标拆解：

GMV目标 = 流量目标 × 转化率目标 × 客单价目标

示例：
- GMV目标：1亿
- 客单价：200元
- 转化率目标：3%
- 所需订单数：1亿÷200 = 50万单
- 所需访客数：50万÷3% = 1667万访客

各渠道流量目标：
├─ 搜索流量：600万（36%）
├─ 推荐流量：500万（30%）
├─ 付费流量：350万（21%）
├─ 内容流量：150万（9%）
└─ 私域流量：67万（4%）
```

### 流量增长路径

| 阶段 | 目标 | 策略 |
|------|------|------|
| 基础建设 | 流量稳定 | 优化搜索权重 |
| 快速增长 | 流量+30% | 加大推广投入 |
| 精细运营 | 流量+50% | 优化流量结构 |
| 生态扩张 | 流量+100% | 多渠道布局 |

## 第三步：预算分配

### 推广预算分配

```
年度推广预算 = 年度GMV目标 × 推广费率

示例：
- 年度GMV目标：1亿
- 推广费率：8%
- 年度推广预算：800万

月度预算分配：
├─ 1月（春节）：50万（淡季，控制投入）
├─ 2月（开年）：60万（爬坡期）
├─ 3-4月（日常）：60万/月
├─ 5月（618预热）：80万
├─ 6月（618）：120万
├─ 7-9月（日常）：60万/月
├─ 10月（双11预热）：100万
├─ 11月（双11）：150万
└─ 12月（年货节）：60万
```

### 渠道预算分配

| 渠道 | 预算占比 | 适用场景 |
|------|----------|----------|
| 直通车 | 40% | 搜索流量、爆款推广 |
| 超级推荐 | 25% | 推荐流量、新客获取 |

### 大促流量节奏

```
大促当天流量曲线：

8-10点（第一波）：流量高峰，维持高转化
12-14点（午间）：流量低谷，加大优惠

GMV最大化策略：
├─ 0点开门红 → 前N小时优惠

```

## 推广ROI优化

### ROI计算框架

```
ROI = GMV贡献 ÷ 推广花费

GMV贡献 = 成交金额 × 归因比例

示例：
- 直通车花费：100万
- 直通车带来GMV：350万
- 归因比例：70%（考虑多渠道影响）
- GMV贡献：350万 × 70% = 245万
- ROI = 245万 ÷ 100万 = 2.45
```

### ROI提升策略

| 策略 | 操作方法 | 效果 |
|------|----------|------|
| 关键词优化 | 保留ROI>2的词 | 提升整体ROI |
| 人群优化 | 圈定高转化人群 | ROI↑20% |
| 时间段优化 | 高转化时间段重点投 | 效率↑ |
| 落地页优化 | 承接页相关性↑ | 转化率↑ |

### ROI警戒线

| 商品类型 | ROI警戒线 | 盈亏平衡 |
|----------|----------|----------|
| 高毛利（>50%） | >1.5 | 覆盖成本 |
| 中毛利（30-50%） | >2.0 | 有利润 |
| 低毛利（<30%） | >3.0 | 需要规模效应 |
