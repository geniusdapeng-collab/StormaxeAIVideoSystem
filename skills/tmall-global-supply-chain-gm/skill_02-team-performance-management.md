# TL;DR

从GM视角设计团队绩效体系——让OKR承接战略，让KPI驱动执行，让团队既有方向感又有战斗力。

# 场景分析

## 典型场景
- 年初定了1个亿GMV目标，但不知道怎么拆解到团队，每个人不知道自己的方向
- KPI定了但没人认真执行，考核流于形式
- 员工抱怨KPI不合理，做得多扣得多
- 团队离职率高，好的留不住，差的赶不走

## 核心挑战
1. **战略落地难**：公司目标到团队目标层层衰减，最后变成各干各的
2. **考核不公平**：定量指标容易一刀切，定性指标容易主观
3. **激励失效**：奖金发了但没激励效果，员工觉得理所当然
4. **人才断层**：骨干走了能力带不走，新人成长太慢

## 正确认知
- **绩效管理的目的是驱动业务成功，不是管控员工**
- **OKR定方向，KPI管执行，两者缺一不可**
- **最好的激励不是钱，是让员工看到自己成长的机会**
- **GM的第一职责是培养人，而不是只用人不培养人**

# 决策框架

## 第一步：组织架构设计

```
跨境供应链团队架构（参考）

总经理
├── 运营总监
│   ├── 运营经理（2人）
│   │   ├── 运营专员（4人）
│   │   └── 助理（2人）

### 岗位职责梳理

| 岗位 | 核心职责 | 关键输出 |
|------|----------|----------|
| 运营总监 | 整体运营、GMV负责 | GMV达成 |
| 运营经理 | 品类运营、活动策划 | 品类GMV |
| 运营专员 | 执行层面 | 日常运营数据 |

| 客服主管 | 客服团队、DSR | 满意度、投诉率 |

## 第二步：目标分解（OKR体系）

### OKR vs KPI

| 对比 | OKR | KPI |
|------|-----|-----|
| 目的 | 激发挑战 | 衡量执行 |
| 设定 | 自下而上 | 自上而下 |
| 评分 | 0-1分 | 百分比 |
| 失败容忍 | 鼓励失败 | 不容忍失败 |
| 适用 | 创新/探索类 | 常规/执行类 |

### GM级OKR示例

```
O1：实现年度GMV 1.2亿，同比增长40%
KR1：Q1-Q4 GMV分别为2500万/3000万/3200万/3300万
KR2：月度复购率从15%提升至22%
KR3：引入10个新品牌/品类

O2：打造行业领先的供应链体系
KR1：库存周转天数从60天降至45天
KR2：48小时发货率从92%提升至98%
KR3：供应链成本率从38%降至34%

O3：建立高绩效团队
KR1：核心岗位流失率<10%
KR2：人均GMV产出提升30%
KR3：团队满意度>90%
```

### 团队级OKR示例

```
运营经理OKR（Q2）：

O：打造2个千万级爆款品类
KR1：美妆品类GMV突破2000万（贡献率>30%）
KR2：至少2个SKU月销过500万
KR3：新品上市成功率>60%（新品上架30天内日销>1万）

O：提升大促运营能力
KR1：618大促GMV达到800万（同比+50%）
KR2：大促ROI达到3.0以上
KR3：大促加购转化率>40%
```

# 专业建议

## 绩效评估机制

### 评估周期

| 类型 | 周期 | 内容 |
|------|------|------|
| 日追踪 | 每日 | 核心数据仪表盘 |
| 周复盘 | 每周 | OKR进度review |
| 月评估 | 每月 | 月度绩效打分 |
| 季OKR | 每季 | OKR复盘+制定 |
| 年终评 | 每年 | 年度综合评估 |

```

| OKR | 自评 | 主管评 |

| 等级 | 比例 | 描述 | 奖金系数 |
|------|------|------|----------|
| S | 5% | 超越期望 | 1.5-2.0× |
| A | 20% | 超出期望 | 1.2-1.5× |
| B | 50% | 达到期望 | 1.0× |
| C | 20% | 接近期望 | 0.6-0.8× |
| D | 5% | 未达期望 | 0-0.5× |

## 激励机制

### 薪酬结构

```
薪酬结构 = 固定工资 + 绩效奖金 + 项目奖金 + 长期激励

固定工资：70%（保障基本生活）
绩效奖金：15%（月度/季度发放）
项目奖金：10%（大促/重点项目）
长期激励：5%（年度奖金/期权）
```

### 奖金方案

#### 月度绩效奖金
```
月度奖金 = 月薪 × 绩效系数 × 奖金系数

绩效系数 = KPI得分÷100
奖金系数 = 部门奖金池÷Σ(个人月薪×绩效系数)
```

#### 大促专项奖金
```
大促GMV奖金：
├─ 达成率>120% → GMV×0.5%
├─ 达成率100-120% → GMV×0.3%
└─ 达成率80-100% → GMV×0.1%
```

### 非物质激励

| 激励类型 | 具体方式 | 适用场景 |
|----------|----------|----------|
| 荣誉激励 | 月度之星、荣誉墙 | 精神认可 |
| 成长激励 | 外出培训、导师带教 | 能力提升 |
| 授权激励 | 独立操盘项目 | 信任认可 |
| 晋升激励 | 明确的晋升通道 | 长期发展 |

## 人才管理

### 人才盘点

| 类型 | 定义 | 策略 |
|------|------|------|
| 明星员工 | 高绩效+高潜力 | 重点培养、晋升 |
| 核心员工 | 高绩效+中潜力 | 保留、激励 |
| 高潜力 | 低绩效+高潜力 | 培养、轮岗 |
| 待改进 | 低绩效+低潜力 | 绩效改进或淘汰 |

### 人才梯队

```
梯队建设：

第一梯队（高管储备）：
- 现任总监/经理
- 培养方向：GM/总监

第二梯队（骨干力量）：
- 现任高级专员
- 培养方向：经理/高级经理

第三梯队（高潜人才）：
- 新晋专员/管培生
- 培养方向：高级专员

替补计划：
- 每个关键岗位至少有1个继任者
- 继任准备度：Ready Now / 1-2年 / 3年以上
```

### 离职预防

| 预警信号 | 应对措施 |
|----------|----------|
| 连续2个月绩效下降 | 1:1沟通，了解原因 |
| 突然减少加班 | 关注动态，及时沟通 |
| 团队核心成员情绪低落 | 管理者反思+调整 |
| 竞品挖人 | 及时回应+激励保留 |
