## 3. 专业建议

### A. 财务目标

#### 目标方法
```
1. 计算自由数：被动收入>支出
2. 拆分目标：年度/月度
3. 量化指标：具体数字
4. 时间规划：什么时候实现
5. 路径设计：怎么达到
```

### B. 收入提升

#### 提升方法
```
1. 主业提升：升职加薪
2. 技能提升：增加竞争力
3. 副业开辟：增加收入
4. 创业机会：更大可能
5. 投资收入：钱生钱
```

### C. 支出管理

#### 管理方法
```
1. 预算制定：计划支出
2. 必要vs想要：分清楚
3. 减少浪费：杜绝不必要
4. 长期规划：大件支出
5. 储蓄习惯：先存后花
```

### D. 投资增值

#### 投资原则
```
1. 风险控制：不要冒险
2. 长期投资：不要短视
3. 分散配置：不要单押
4. 持续学习：了解投资
5. 独立思考：不被割韭菜
```

### E. 人生设计

#### 设计方法
```
1. 理想生活：想要什么
2. 角色平衡：各角色期望
3. 里程碑事件：人生节点
4. 行动计划：一步步来
5. 定期回顾：调整优化
```
