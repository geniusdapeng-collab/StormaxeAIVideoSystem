## 2. 思维模型框架

### 核心思维模型

| 模型 | 用途 | 场景 |
|------|------|------|
| 第一性原理 | 追根溯源 | 创新/突破 |
| 框架思考 | 结构化 | 分析/规划 |
| 逆向思维 | 反向推演 | 问题解决 |
| 系统思维 | 整体视角 | 复杂问题 |
| 批判思维 | 质疑验证 | 判断/决策 |

### 思维层次

```
1. 收集信息：事实层面
2. 分析理解：逻辑层面
3. 框架应用：结构层面
4. 深度洞察：本质层面
5. 创新突破：创造层面
```

## 3. 专业建议

### A. 第一性原理

#### 应用方法
```
1. 质疑假设：问"为什么"
2. 回归本质：最基础是什么
3. 重新构建：从头开始
4. 突破限制：不受现有约束
5. 创新方案：新方法
```

### B. 框架思考

#### 应用方法
```
1. 选择框架：适合的模型
2. 结构分解：拆解要素
3. 要素分析：每个部分
4. 整合输出：形成方案
5. 检验框架：是否适用
```

### C. 决策方法

#### 决策方法
```
1. 明确目标：想要什么
2. 收集信息：了解情况
3. 方案选项：列出选择
4. 评估利弊：优缺点
5. 果断决定：选择执行
```

### D. 批判思维

#### 应用方法
```
1. 质疑信息：来源可靠吗
2. 检验逻辑：论证合理吗
3. 多元视角：其他角度呢
4. 证据验证：有证据吗
5. 保持开放：随时修正
```
