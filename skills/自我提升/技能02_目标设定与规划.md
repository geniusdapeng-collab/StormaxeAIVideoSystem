## TL;DR

目标规划核心：愿景锚定→目标拆解→计划制定→资源配置→进度追踪，五步实现目标

## 1. 场景分析

### 请求摘要
用户需要设定目标和制定规划

### 显性需求
- 科学设定目标
- 制定可行计划

### 场景分类
- `scenario_classification`: 目标管理
- `urgency`: normal

## 2. 目标规划框架

### 目标层次

| 层次 | 时间 | 示例 |
|------|------|------|
| 愿景 | 3-5年 | 想成为什么样的人 |
| 年目标 | 1年 | 今年要达成什么 |
| 月计划 | 1月 | 本月重点任务 |
| 周任务 | 1周 | 本周要做什么 |
| 日行动 | 每天 | 今日待办 |

### 目标原则（SMART）

```
1. Specific：具体明确
2. Measurable：可衡量
3. Achievable：可达成
4. Relevant：相关性
5. Time-bound：有时间限制
```

## 3. 专业建议

### A. 愿景锚定

#### 锚定方法
```
1. 理想生活：描述理想一天
2. 人生角色：各角色期望
3. 墓志铭：希望留下什么
4. 榜样分析：想成为谁
5. 逆向推演：从结果倒推
```

### B. 目标拆解

#### 拆解方法
```
1. 终极目标：最终要什么
2. 阶段目标：关键里程碑
3. 量化指标：具体数字
4. 最小行动：第一步是什么
5. 依赖关系：先做什么
```

### C. 计划制定

#### 制定方法
```
1. 任务清单：要做什么事
2. 时间估算：需要多久
3. 资源需求：需要什么
4. 责任到人：谁来做
5. 截止时间：什么时候完成
```

### D. 进度追踪

#### 追踪方法
```
1. 定期回顾：周/月检查
2. 量化指标：完成多少
3. 偏差分析：差在哪里
4. 调整优化：怎么改进
5. 庆祝进步：认可成就
```
