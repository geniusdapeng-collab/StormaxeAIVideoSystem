| 识别 | 感知情绪 | 觉察 |

## 3. 专业建议

### A. 情绪识别

#### 识别方法
```
1. 身体扫描：感受身体
2. 情绪词汇：命名情绪
3. 触发事件：什么引发的
4. 情绪日记：记录情绪
5. 他人反馈：别人怎么说
```

### B. 情绪调节

#### 调节方法
```
1. 暂停：先停下来
2. 呼吸：深呼吸
3. 接纳：不抵抗情绪
4. 重构：换个角度看
5. 表达：合适方式
```

### C. 压力应对

#### 应对方法
```
1. 问题解决：直面解决
2. 情绪释放：运动/倾诉
3. 放松技巧：冥想/按摩
4. 社会支持：找人帮忙
5. 调整预期：降低要求
```

### D. 心理韧性

#### 培养方法
```
1. 成长思维：困难是成长
2. 积极解释：看到好的一面
3. 核心价值：知道什么重要
4. 社会连接：有人支持
5. 自我关怀：对自己好点
```
