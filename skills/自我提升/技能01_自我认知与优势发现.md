自我认知核心：价值观探索→性格特点→能力盘点→优势发现→个人定位，五步清晰认识自己

### 场景分类

## 2. 自我认知框架

### 认知维度

| 维度 | 内容 | 方法 |
|------|------|------|
| 价值观 | 什么最重要 | 价值排序 |
| 性格 | 我是什么样的人 | 性格测试 |
| 能力 | 我擅长什么 | 成就事件 |
| 优势 | 我的核心竞争力 | 优势识别 |
| 动机 | 我为什么做 | 动机挖掘 |

### 认知流程

```
1. 自我反思：我是谁
2. 他人反馈：别人眼中的我
3. 成就事件：做得好什么
4. 失败教训：做不好什么
5. 模式识别：反复出现什么
```

## 3. 专业建议

### A. 价值观探索

#### 探索方法
```
1. 价值卡片：排序练习
2. 人生阶段：各阶段重视
3. 榜样分析：佩服谁
4. 临终视角：什么重要
5. 冲突选择：面临选择时
```

### B. 性格特点

#### 了解方法
```
1. 性格测试：MBTI/大五
2. 行为模式：日常表现
3. 压力反应：困境中
4. 社交风格：与人相处
5. 工作偏好：喜欢什么
```

### C. 能力盘点

#### 盘点方法
```
1. 技能清单：会什么技能
2. 成就事件：做成过什么
3. 失败教训：做错过什么
4. 他人评价：别人怎么赞
5. 持续投入：什么做不腻
```

### D. 优势发现

#### 发现方法
```
1. 优势测试：盖洛普优势
2. 心流体验：做什么忘我
3. 快速学习：什么一学就会
4. 自然而然：什么不用刻意
5. 能量来源：做什么有劲
```
