## 2. 时间管理框架

### GTD流程

| 步骤 | 内容 | 行动 |
|------|------|------|
| 收集 | 所有任务 | 收集箱 |
| 处理 | 分类整理 | 分类 |
| 组织 | 整理归类 | 项目/日历 |
| 回顾 | 定期检查 | 周回顾 |
| 执行 | 行动 | 做 |

### 时间管理原则

```
1. 重要 vs 紧急：优先做重要的
2. 80/20法则：20%带来80%成果
3. 能量管理：精力好时做重要事
4. 批量处理：同类事一起做
5. 保护时间：留出专注时间
```

## 3. 专业建议

### A. 优先级排序

#### 排序方法
```
1. 艾森豪威尔矩阵：重要/紧急四象限
2. ABC法则：A最重要先做
3. 关键结果：盯着最重要的
4. 说不勇气：敢于拒绝
5. 委托他人：非核心可委托
```

### B. 时间块规划

#### 规划方法
```
1. 留出缓冲：不要排太满
2. 主题时间：同类事集中
3. 黄金时间：高效时段做要事
4. 会议控制：固定时间
5. 休息安排：定时休息
```

### C. 专注执行

#### 专注方法
```
1. 番茄工作法：25分钟专注
2. 手机隔离：远离干扰
3. 环境整理：干净整洁
4. 任务单一：一次做一件
5. 完成标准：定义"做完"
```

### D. 复盘优化

#### 复盘方法
```
1. 每日复盘：15分钟回顾
2. 时间日志：记录去向
3. 分析浪费：找出低效
4. 持续改进：优化方法
5. 习惯固化：形成routine
```
