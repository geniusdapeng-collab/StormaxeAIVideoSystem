## TL;DR

安全教育核心：危险识别→安全意识→求助技能→自我保护→应急处理，五步培养安全小卫士

## 1. 场景分析

### 显性需求

## 2. 安全教育框架

### 各年龄段安全重点

| 年龄段 | 安全重点 | 教育方式 |
|--------|----------|----------|
| 0-3岁 | 居家安全 | 家长防护 |
| 3-6岁 | 认知危险 | 绘本/游戏 |
| 6-12岁 | 自我保护 | 情景演练 |

### 安全教育要素

```
1. 识别：认识危险
2. 预防：远离危险
3. 应对：怎么办
4. 求助：找谁帮忙
5. 心理：自我保护意识
```

## 3. 专业建议

### A. 危险识别

#### 教育方法
```
1. 环境排查：家里/户外
2. 绘本学习：安全绘本
3. 情景模拟：角色扮演
4. 案例分享：适当讲解
5. 规则制定：安全红线
```

### B. 居家安全

#### 防护要点
```
1. 尖锐物品：放到高处
2. 电器插座：保护盖
3. 窗户阳台：防护栏
4. 药品化学品：锁起来
5. 厨房安全：远离火源
```

### C. 户外安全

#### 安全要点
```
1. 交通安全：遵守规则
2. 水安全：远离水域
3. 陌生人：不轻信
4. 走失防护：记住信息
5. 游乐设施：安全使用
```

### D. 自我保护

#### 保护技能
```
1. 身体界限：隐私保护
2. 求助技能：找警察/工作人员
3. 记住信息：家庭地址电话
4. 紧急电话：110/120/119
5. 信任直觉：感觉不对就跑
```
