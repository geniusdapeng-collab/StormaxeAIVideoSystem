## 2. 学习能力框架

### 核心学习能力

| 能力 | 说明 | 培养重点 |
|------|------|----------|
| 专注力 | 注意力 | 抗干扰 |
| 记忆力 | 记忆能力 | 方法技巧 |
| 思维力 | 思考能力 | 逻辑思维 |
| 观察力 | 观察能力 | 细节发现 |
| 表达力 | 表达能力 | 输出能力 |

### 学习能力要素

```
1. 动力：愿不愿意学
2. 能力：能不能学会
3. 方法：会不会学习
4. 习惯：能不能坚持
5. 环境：有没有条件
```

## 3. 专业建议

### A. 专注力培养

#### 培养方法
```
1. 限时任务：番茄钟法
2. 环境简单：减少干扰
3. 兴趣引导：从喜欢开始
4. 正向反馈：及时鼓励
5. 运动锻炼：提升专注
```

### B. 记忆力培养

#### 培养方法
```
1. 理解记忆：先理解后记忆
2. 联想记忆：构建联系
3. 重复复习：间隔重复
4. 图像记忆：形象化
5. 记忆宫殿：高级技巧
```

### C. 思维力培养

#### 培养方法
```
1. 提问引导：多问为什么
2. 思维游戏：棋类、拼图
3. 分类归纳：找规律
4. 逆向思考：反过来想
5. 问题解决：独立思考
```

### D. 学习习惯

#### 培养方法
```
1. 固定时间：作息规律
2. 固定流程：学习步骤
3. 准备充分：书桌整洁
4. 及时复习：温故知新
5. 总结复盘：定期回顾
```
