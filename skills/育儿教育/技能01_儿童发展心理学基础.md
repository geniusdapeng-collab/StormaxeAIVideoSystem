## 2. 发展心理学框架

### 各年龄段特点

| 年龄段 | 发展重点 | 关键任务 |
|--------|----------|----------|
| 0-1岁 | 安全感 | 依恋关系 |
| 1-3岁 | 自主性 | 分离个体化 |
| 3-6岁 | 主动性 | 游戏与想象 |
| 6-12岁 | 勤奋感 | 学习能力 |
| 12-18岁 | 自我认同 | 青春期发展 |

### 发展领域

```
1. 生理发展：身体成长
2. 认知发展：思维能力
3. 情绪发展：情感调节
4. 社交发展：人际关系
5. 人格发展：自我形成
```

## 3. 专业建议

### A. 认知发展

#### 发展特点
```
1. 感知运动阶段：0-2岁，通过感觉运动认识世界
2. 前运算阶段：2-7岁，语言和符号思维发展
3. 具体运算阶段：7-11岁，逻辑思维开始
4. 形式运算阶段：11岁+，抽象思维发展
```

### B. 情绪发展

#### 发展特点
```
1. 基本情绪：喜、怒、哀、惧
2. 情绪理解：理解他人情绪
3. 情绪调节：学会控制情绪
4. 情绪表达：适宜方式表达
```

### C. 社交发展

#### 发展特点
```
1. 亲子关系：依恋类型
2. 同伴关系：社交技能
3. 性别角色：性别认同
4. 社会规范：规则意识
```

### D. 发展支持

#### 支持方法
```
1. 敏感回应：及时回应需求
2. 安全环境：提供探索空间
3. 适宜挑战：略高于当前能力
4. 积极鼓励：正向反馈
5. 稳定关系：主要养育人稳定
```
