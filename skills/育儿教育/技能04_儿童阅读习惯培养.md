## 2. 阅读培养框架

### 各年龄段阅读重点

| 年龄段 | 阅读重点 | 形式 |
|--------|----------|------|
| 0-1岁 | 布书、卡片 | 家长读 |
| 1-3岁 | 绘本、翻翻书 | 亲子共读 |
| 3-6岁 | 绘本、桥梁书 | 亲子共读+自主 |
| 6-12岁 | 文字书 | 自主阅读 |

### 阅读能力发展

```
1. 翻书期：会翻书页
2. 图期：看图理解
3. 字期：认识文字
4. 读本：自主阅读
5. 精读：深入理解
```

## 3. 专业建议

### A. 阅读环境

#### 环境营造
```
1. 阅读角：舒适专属空间
2. 书架：随手可取
3. 书香：家长以身作则
4. 氛围：固定阅读时间
5. 多元：接触各类书籍
```

### B. 亲子共读

#### 共读技巧
```
1. 固定时间：养成习惯
2. 声情并茂：吸引注意
3. 互动提问：加深理解
4. 指读文字：识字作用
5. 角色扮演：趣味性强
```

### C. 兴趣引导

#### 引导方法
```
1. 投其所好：选择孩子喜欢
2. 关联兴趣：与动画、游戏关联
3. 悬念设置：吸引往下看
4. 成就激励：阅读成就感
5. 小伙伴影响：同伴带动
```

### D. 习惯坚持

#### 坚持方法
```
1. 固定节奏：每天坚持
2. 目标可行：循序渐进
3. 家长陪伴：一起阅读
4. 正向反馈：鼓励为主
5. 打造仪式：阅读仪式感
```
