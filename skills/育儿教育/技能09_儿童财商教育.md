## 2. 财商教育框架

### 各年龄段教育重点

| 年龄段 | 教育重点 | 方式 |
|--------|----------|------|
| 3-6岁 | 认识金钱 | 购物游戏 |
| 6-10岁 | 零花钱管理 | 自主管理 |
| 10-18岁 | 理财实践 | 银行账户 |

### 财商要素

```
1. 认识：金钱是什么
2. 理解：钱从哪里来
3. 管理：怎么花
4. 储蓄：存钱习惯
5. 投资：钱生钱
```

## 3. 专业建议

### A. 认识金钱

#### 教育方法
```
1. 实物教学：认识钱币
2. 购物参与：一起买东西
3. 价格认知：了解价格
4. 交换概念：物物交换
5. 劳动价值：赚钱不易
```

### B. 零花钱管理

#### 管理方法
```
1. 定时定量：固定时间给
2. 自主管理：自己保管
3. 记录支出：记账习惯
4. 分配比例：存/花/捐
5. 预算概念：量入为出
```

### C. 储蓄习惯

#### 培养方法
```
1. 储蓄罐：可视化储蓄
2. 目标储蓄：想要什么
3. 银行账户：开户体验
4. 利息概念：钱会变多
5. 延迟满足：等一等
```

### D. 理性消费

#### 培养方法
```
1. 需求 vs 想要：区分
2. 价格比较：性价比
3. 等待降温：冷静期
4. 分享与捐赠：爱心消费
5. 体验消费：比物质重要
```
