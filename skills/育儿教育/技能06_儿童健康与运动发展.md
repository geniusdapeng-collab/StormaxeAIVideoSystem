## 2. 健康运动框架

### 各年龄段运动重点

| 年龄段 | 运动重点 | 推荐活动 |
|--------|----------|----------|
| 0-1岁 | 抬头、翻身 | 被动操 |
| 1-3岁 | 走、跑、跳 | 户外跑跳 |
| 3-6岁 | 协调性 | 球类、平衡木 |
| 6-12岁 | 体能技能 | 游泳、球类 |

### 健康要素

```
1. 营养：均衡饮食
2. 运动：适量锻炼
3. 睡眠：充足休息
4. 心理：健康心态
5. 习惯：规律作息
```

## 3. 专业建议

### A. 运动发展

#### 发展要点
```
1. 大运动：抬头、坐、爬、走、跑
2. 精细运动：抓握、画画、写字
3. 平衡感：平衡木、滑梯
4. 协调性：球类、跳舞
5. 力量感：攀爬、跳跃
```

### B. 运动培养

#### 培养方法
```
1. 户外时间：每天2小时
2. 运动兴趣：从游戏开始
3. 循序渐进：逐步提高
4. 陪伴运动：亲子互动
5. 多元体验：各种运动
```

### C. 健康饮食

#### 饮食要点
```
1. 均衡营养：荤素搭配
2. 规律饮食：定时定量
3. 零食控制：健康零食
4. 喝水习惯：多喝白开水
5. 饮食习惯：细嚼慢咽
```

### D. 疾病预防

#### 预防方法
```
1. 疫苗接种：按时接种
2. 卫生习惯：勤洗手
3. 季节防护：增减衣物
4. 定期体检：关注发育
5. 常见病护理：家庭护理
```
