## 技能概述

**对应好莱坞类型**: 实战动作（Realistic Combat / Hard Action）
**定位**: 实战动作编剧是"真实战斗的故事讲述者"——用动作代替语言，用身体表达情感，用搏斗推进叙事。
**核心职责**: 动作叙事设计、角色能力构建、动作场景构建、真实感保持

## 专业知识库

### 第一部分：动作叙事原理

```
动作叙事 vs 对话叙事:

核心区别:
- 对话叙事: 用语言表达
- 动作叙事: 用身体表达

动作叙事的好处:
1. 超越语言障碍
2. 直接的情感冲击
3. 文化普遍性
4. 视觉娱乐性

何时用动作叙事:
- 角色不愿说出口
- 语言无法表达
- 冲突升级
- 情节推进
```

#### 动作叙事结构

```
动作场景的叙事结构:

开场: 建立情境
- 角色处境
- 冲突背景
- 目标明确

发展: 战斗过程
- 攻守转换
- 能力展示
- 困境出现

高潮: 决定性时刻
- 最后一击
- 角色抉择
- 代价付出

结尾: 结果与后果
- 胜负已定
- 伤亡显现
- 情节推进
```

### 第二部分：角色设计

```
动作主角的能力设计:

能力维度:
1. 技术类型
   - 徒手格斗（武术流派）
   - 持械战斗
   - 射击
   - 混合能力

2. 专长领域
   - 近身格斗
   - 远程作战
   - 战术指挥
   - 特殊技能

3. 身体属性
   - 力量型
   - 速度型
   - 耐力型
   - 敏捷型

4. 性格属性
   - 冷静型（战术思考）
   - 狂暴型（情绪驱动）
   - 技巧型（精准高效）
   - 生存型（临场应变）

能力对比表:
| 属性 | 主角 | 对手 |
|------|------|------|
| 力量 | 中 | 高 |
| 速度 | 高 | 中 |
| 技巧 | 高 | 中 |
| 经验 | 中 | 高 |
```

### 第三部分：动作场景构建

```
动作场景构建模板:

Step 1: 确定场景目的
- 展示角色能力？
- 推进情节？
- 建立关系？
- 揭示弱点？

Step 2: 设计约束条件
- 空间限制？
- 武器限制？
- 人数限制？
- 时间限制？

Step 3: 设计意外元素
- 环境变化
- 第三方介入
- 装备故障
- 能力受限

Step 4: 安排攻守节奏
- 主角优势 → 对手反击
- 对手优势 → 主角智取
- 反复拉锯
- 最终决胜

Step 5: 连接叙事
- 动作有代价
- 胜利/失败有意义
- 为后续情节铺垫
```

### 第四部分：真实感保持

```
实战动作的真实性原则:

1. 伤害逻辑
   - 被击中会痛
   - 伤痛影响行动
   - 体力会消耗
   - 肾上腺素过后是疲惫

2. 环境逻辑
   - 地面湿滑
   - 光线不足
   - 空间狭窄
   - 天气影响

3. 武器逻辑
   - 武器会损坏
   - 弹药有限
   - 需要维护
   - 各有优缺点

4. 生理逻辑
   - 会流血
   - 会骨折
   - 会疲劳
   - 痛觉存在

5. 战术逻辑
   - 利用地形
   - 团队配合
   - 心理战术
   - 环境优势

真实感违禁清单:
- 无限体力
- 无痛伤害
- 无敌武器
- 无视环境
```
