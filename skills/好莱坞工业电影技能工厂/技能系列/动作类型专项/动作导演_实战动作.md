## 技能概述

**对应好莱坞类型**: 实战动作（Realistic Combat / Hard Action）
**定位**: 实战动作导演是"真实战斗的诠释者"——负责设计在物理上可信、在视觉上震撼的实战动作场面。是让观众相信这是真实搏斗的那个人。
**核心职责**: 真实动作编排、实战氛围营造、物理可信性把控、动作与叙事融合

## 专业知识库

### 第一部分：实战动作原理

```
实战动作 vs 风格化动作:

实战动作特征:
- 真实格斗技术
- 物理可信
- 伤害真实
- 紧张激烈
- 动作服务于叙事

vs 风格化动作（武侠/超级英雄）:
- 夸张的动作
- 美感优先
- 伤害不真实
- 舞蹈化打斗
- 动作本身是表演

代表作品:
《突袭》系列《拳霸》《导火线》《伯恩系列》《碟影重重》
```

#### 格斗基础

```
实战格斗类型:

1. 拳法 (Striking)
   - 站立打击
   - 拳/肘/膝/腿
   - 距离控制
   - 防守技术

2. 摔跤 (Grappling)
   - 近距离缠斗
   - 摔法/寝技
   - 地面控制
   - 降服技术

3. 持械 (Weapons)
   - 短刀/匕首
   - 棍棒
   - 枪械（较少实战动作用）
   - 日常物品

4. 综合格斗 (Mixed)
   - 站立+摔投+地面
   - 根据情境混合使用
   - 最接近真实战斗
```

### 第二部分：动作编排

```
实战动作编排原则:

1. 简洁有效
   - 真实动作不花哨
   - 快速结束战斗
   - 伤害累积
   - 疲惫感

2. 环境利用
   - 利用周围物品
   - 地形优势
   - 狭窄空间限制
   - 多功能场景

3. 角色能力
   - 根据角色设定能力
   - 专业技能展示
   - 普通人挣扎感
   - 成长曲线

4. 伤害逻辑
   - 伤害真实累积
   - 体力消耗
   - 伤痛影响行动
   - 肾上腺素

动作编排检查:
- [ ] 动作是否物理可信？
- [ ] 是否有受伤/疲惫表现？
- [ ] 是否有环境利用？
- [ ] 是否与角色能力匹配？
- [ ] 是否服务于叙事？
```

### 第三部分：实战氛围

```
实战动作的视觉语言:

1. 快速剪辑
   - 短镜头切换
   - 保持紧张感
   - 真实打斗节奏
   - 肾上腺素感

2. 手持摄影
   - 不稳定感
   - 参与感
   - 真实感
   - 混乱感

3. 音效强化
   - 打击声真实
   - 呼吸声紧张
   - 环境音压抑
   - 低频冲击

4. 真实伤害
   - 瘀伤/红肿
   - 出血
   - 动作迟缓
   - 痛苦表情

实战光线:
- 低照度: 夜间/暗处
- 高对比: 硬光
- 自然光: 日景动作
- 混合光: 复杂场景
```
