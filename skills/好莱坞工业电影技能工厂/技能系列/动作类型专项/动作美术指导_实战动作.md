## 技能概述

**对应好莱坞类型**: 实战动作（Realistic Combat）
**定位**: 实战动作美术指导是"真实环境的构建者"——负责构建让动作场景更加可信的真实环境，让观众相信搏斗发生在真实世界中。
**核心职责**: 真实环境设计、动作场景空间、道具真实感、动作环境构建

## 专业知识库

### 第一部分：实战动作环境

```
实战动作的美术原则:

1. 真实环境
   - 真实的建筑
   - 真实的空间
   - 真实的道具
   - 观众信任环境

2. 环境利用
   - 空间限制
   - 地形优势
   - 可利用物品
   - 环境成为武器

3. 破坏真实性
   - 战斗痕迹
   - 破损效果
   - 真实伤害积累
   - 环境反馈
```

### 第二部分：动作场景设计

```
动作场景的类型:

1. 城市环境
   - 街道
   - 建筑内部
   - 停车场
   - 屋顶

2. 工业环境
   - 仓库
   - 工厂
   - 建筑工地
   - 地下室

3. 居住环境
   - 公寓
   - 房屋
   - 酒店
   - 办公室

4. 特殊环境
   - 交通工具
   - 狭小空间
   - 高空
   - 水下
```
