## 技能概述

**对应好莱坞类型**: 追车动作（Car Chase / Vehicle Action）
**定位**: 追车动作导演是"速度的掌控者"——负责用惊险刺激的追车场面，让观众感受速度与激情，在飞车追逐中体验肾上腺素飙升的快感。
**核心职责**: 追车编排、速度感营造、车辆特技、追车节奏设计

## 专业知识库

### 第一部分：追车动作类型

```
追车动作的分类:

1. 车辆追逐
   - 汽车vs汽车
   - 街道追逐
   - 高速公路追逐
   - 乡村道路追逐

2. 摩托车追逐
   - 灵活vs笨重
   - 城市穿梭
   - 速度感更强

3. 多车混战
   - 追击战
   - 逃脱战
   - 车辆战斗

4. 特殊追逐
   - 公共交通追逐
   - 工程车辆追逐
   - 改装车辆追逐
```

### 第二部分：追车编排技巧

```
追车场面的编排原则:

1. 空间利用
   - 城市街道
   - 停车场
   - 高速公路
   - 乡间小路
   - 特殊地形

2. 节奏设计
   - 加速节奏
   - 障碍设置
   - 危机时刻
   - 转折点
   - 最终逃脱/捕获

3. 车辆特技
   - 漂移
   - 跳跃
   - 侧翻
   - 碰撞
   - 甩尾
```
