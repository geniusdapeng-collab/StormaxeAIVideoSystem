## 技能概述

**对应好莱坞类型**: 实战动作（Realistic Combat / Hard Action）
**定位**: 实战动作摄影指导是"搏斗的参与者"——负责用快节奏的剪辑、手持摄影、高对比光，构建让观众身临其境的真实战斗体验。
**核心职责**: 动作节奏配合、快速运动摄影、真实战斗光效、肾上腺素视觉

## 专业知识库

### 第一部分：实战动作摄影

```
实战动作的摄影原则:

1. 手持摄影
   - 不稳定感增加真实感
   - 观众参与感
   - 混乱氛围
   - 肾上腺素感

2. 快速剪辑
   - 短镜头
   - 快速切换
   - 保持紧张感
   - 真实打斗节奏

3. 近距离摄影
   - 强调真实伤害
   - 观众感受到冲击
   - 细节展示
   - 情感传递

4. 跟随摄影
   - 跟随角色移动
   - 主观视角
   - 沉浸感
```

### 第二部分：光线设计

```
实战动作的光线设计:

1. 高对比
   - 硬光
   - 清晰阴影
   - 强调打击感
   - 真实感

2. 低照度
   - 夜间/暗处战斗
   - 增加紧张感
   - 阴影隐藏细节
   - 观众想象补充

3. 自然光
   - 白天战斗用自然光
   - 真实感
   - 不均匀照度
```
