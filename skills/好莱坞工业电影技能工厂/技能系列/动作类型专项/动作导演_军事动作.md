## 技能概述

**对应好莱坞类型**: 军事动作（Military Action）
**定位**: 军事动作导演是"战争机器的诠释者"——负责用精准的军事战术、真实的武器装备、宏大的战场场面，展现战争中的英雄主义与牺牲。
**核心职责**: 军事战术设计、武器装备呈现、战场氛围构建、军事真实感把控

## 专业知识库

### 第一部分：军事动作的类型

```
军事动作的类型:

1. 特种部队
   - 小队行动
   - 精准打击
   - 战术配合
   - 经典: 《黑鹰坠落》《特种部队》

2. 战争史诗
   - 大规模战斗
   - 宏大场面
   - 历史战争
   - 经典: 《拯救大兵瑞恩》《血战钢锯岭》

3. 现代战争
   - 当代军事冲突
   - 最新武器装备
   - 地缘政治背景
   - 经典: 《拆弹部队》《危机13小时》

4. 战争喜剧
   - 黑色幽默
   - 荒诞战争
   - 人性在战争中
   - 经典: 《铁金刚》《全金属外壳》
```

### 第二部分：军事战术设计

```
军事战术的视觉表达:

1. 步兵战术
   - 队形移动
   - 掩护与突击
   - 通信联络
   - 战术手势

2. 载具作战
   - 装甲部队
   - 空中支援
   - 海上作战
   - 多军种协同

3. 特种作战
   - 渗透/撤离
   - 精确打击
   - 秘密行动
   - 团队配合

战术节奏设计:
- 战前准备: 紧张、期待
- 首次接触: 突然、混乱
- 战术对抗: 智慧、技巧
- 关键时刻: 高潮、牺牲
- 战后余波: 反思、代价
```

### 第三部分：武器装备

```
军事动作的武器系统:

轻武器:
- 手枪: 近距离战斗
- 步枪: 标准步兵武器
- 狙击步枪: 精准打击
- 机枪: 压制火力

重武器:
- 火箭筒: 反载具
- 迫击炮: 曲射火力
- 坦克: 装甲突击
- 战斗机: 空中优势

特殊装备:
- 夜视仪
- 热成像
- 无人机
- 防弹衣

武器的视觉表达:
- 开火效果
- 弹道轨迹
- 击中效果
- 爆炸效果
```
