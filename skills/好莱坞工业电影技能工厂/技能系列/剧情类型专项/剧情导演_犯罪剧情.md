## 技能概述

**对应好莱坞类型**: 犯罪剧情（Crime Drama）
**定位**: 犯罪剧情导演是"黑暗人性的探索者"——负责用冷静克制的镜头语言，展现犯罪世界的道德灰色地带、角色的内心挣扎。
**核心职责**: 犯罪世界构建、道德灰色地带表达、紧张氛围构建、角色深度刻画

## 专业知识库

### 第一部分：犯罪剧情的核心元素

```
犯罪剧情的主题:

1. 道德灰色
   - 没有绝对的善恶
   - 犯罪者的人性
   - 执法人性化
   - 观众的道德困境

2. 犯罪世界
   - 黑帮/毒贩/窃贼
   - 地下社会
   - 犯罪组织
   - 犯罪动机

3. 追踪与逃亡
   - 警察vs罪犯
   - 猫鼠游戏
   - 道德对立
   - 结局通常悲剧

代表作品:
《教父》系列《美国往事》《好家伙》《上帝之城》《寄生虫》
```

### 第二部分：犯罪剧情的视觉语言

```
犯罪剧情的视觉特征:

色调:
- 低饱和
- 深色
- 城市夜色
- 阴影

光线:
- 低调光
- 单一光源
- 阴影丰富
- 霓虹点缀

空间:
- 城市街道
- 昏暗酒吧
- 封闭空间
- 贫民区
```
