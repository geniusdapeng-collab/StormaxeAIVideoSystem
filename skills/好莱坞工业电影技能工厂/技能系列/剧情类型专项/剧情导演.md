## 技能概述

**对应好莱坞类型**: 家庭剧情（Family Drama）
**定位**: 家庭剧情导演是"情感真相的挖掘者"——负责用克制、真实的镜头语言，展现家庭关系中的爱、冲突、和解与成长。
**核心职责**: 家庭情感构建、真实表演引导、克制美学设计、情感节奏管理

## 专业知识库

### 第一部分：家庭剧情的本质

```
家庭剧情的核心:

主题探索:
1. 代际冲突
   - 父母与子女的矛盾
   - 价值观差异
   - 期望vs现实

2. 家庭秘密
   - 被隐藏的真相
   - 过去的创伤
   - 影响当下的过去

3. 和解与宽恕
   - 家人间的和解
   - 自我和解
   - 宽恕的力量

4. 身份与归属
   - 我是谁？
   - 我属于哪里？
   - 家庭对我的意义

代表作品:
《阳光小美女》《爱你已久》《特别响，非常近》《海边的曼彻斯特》
```

### 第二部分：克制的美学

```
家庭剧情的视觉语言:

1. 自然主义光线
   - 真实室内光
   - 窗口自然光
   - 避免过度美化
   - 真实感优先

2. 亲密构图
   - 特写镜头
   - 紧凑的构图
   - 角色的空间接近
   - 观察者视角

3. 缓慢节奏
   - 长时间镜头
   - 静默的力量
   - 留白
   - 让情感呼吸

4. 真实环境
   - 真实的家
   - 日常的场景
   - 生活的质感
   - 熟悉感

色调设计:
- 暖色调: 家庭温暖
- 冷色调: 冲突/距离
- 自然色: 真实感
- 低饱和: 克制/怀旧
```

### 第三部分：表演引导

```
家庭剧情的表演策略:

核心原则:
- 真实胜于表演
- 细节胜于夸张
- 沉默胜于言语
- 留白胜于说尽

情感处理:
1. 压抑的情感
   - 表面平静
   - 深层波涛
   - 细节泄露

2. 爆发的情感
   - 压抑后的释放
   - 罕见但有力
   - 爆发的理由充分

3. 沉默的情感
   - 不需要台词
   - 眼神/肢体表达
   - 共情的力量

表演指导技巧:
- 鼓励"少即是多"
- 细节决定真实感
- 允许即兴
- 不要过度排练
```
