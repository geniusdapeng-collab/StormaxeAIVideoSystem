## 技能概述

**对应好莱坞类型**: 家庭剧情（Family Drama）
**定位**: 家庭剧情编剧是"亲情真相的挖掘者"——负责用真实的情感、细腻的细节，构建让观众看到自己家庭的家庭故事。
**核心职责**: 家庭关系构建、代际冲突表达、情感真实化、家庭秘密设计

## 专业知识库

### 第一部分：家庭剧情叙事

```
家庭剧情的主题:

1. 代际冲突
   - 父母vs子女
   - 期望vs现实
   - 传统vs现代
   - 沟通障碍

2. 家庭秘密
   - 被隐藏的过去
   - 影响当下的创伤
   - 真相的揭露
   - 和解vs继续逃避

3. 家庭成员关系
   - 兄弟姐妹
   - 夫妻关系
   - 亲子关系
   - 大家庭

4. 家庭时刻
   - 餐桌对话
   - 家庭聚会
   - 危机时刻
   - 日常时刻
```

### 第二部分：角色弧光

```
家庭剧情的角色类型:

1. 逃避者
   - 逃离家庭
   - 多年不归
   - 回归面对过去
   - 和解or继续逃避

2. 照顾者
   - 为家庭牺牲
   - 被认为理所当然
   - 疲惫和愤怒
   - 自我价值的追寻

3. 叛逆者
   - 挑战家庭规则
   - 被家庭排斥
   - 证明自己
   - 接纳or继续对抗

4. 继承者
   - 接手家庭责任
   - 父母角色
   - 传承vs创新
   - 家庭延续
```
