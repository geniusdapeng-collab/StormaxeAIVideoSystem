## 技能概述

**对应好莱坞类型**: 悬疑剧情（Thriller / Suspense Drama）
**定位**: 悬疑剧情导演是"紧张的控制者"——负责用精准的节奏、悬念的构建、惊悚的氛围，让观众在心跳加速中体验解谜的快感。
**核心职责**: 悬念构建、紧张节奏控制、惊悚氛围、真相揭示

## 专业知识库

### 第一部分：悬疑剧情的核心元素

```
悬疑剧情的类型:

1. 心理悬疑
   - 复杂心理
   - 身份之谜
   - 记忆碎片
   - 经典: 《搏击俱乐部》《致命身份》

2. 惊悚悬疑
   - 生命威胁
   - 倒计时
   - 孤立无援
   - 经典: 《危情》《亡命天涯》

3. 推理悬疑
   - 犯罪调查
   - 解谜过程
   - 真相追踪
   - 经典: 《七宗罪》《消失的爱人》

4. 政治悬疑
   - 阴谋论
   - 权力斗争
   - 秘密揭露
   - 经典: 《危机13小时》《 Snowden》
```

### 第二部分：悬念构建技巧

```
悬疑的节奏设计:

1. 建立悬念
   - 抛出谜题
   - 吸引注意力
   - 让观众想知道

2. 维持悬念
   - 逐步揭示线索
   - 但不给出答案
   - 误导和线索并存
   - 观众的期待和焦虑

3. 升级悬念
   - 增加危险/复杂度
   - 揭露更大的真相
   - 时限压力

4. 解决悬念
   - 真相揭示
   - 反转（可选）
   - 情感满足
```

### 第三部分：悬疑的视觉语言

```
悬疑的视觉风格:

色调:
- 低调为主
- 冷色调
- 压抑感
- 偶尔的亮色点缀

光线:
- 低照度
- 深阴影
- 明暗对比
- 突出关键元素

空间:
- 封闭感
- 压迫感
- 隔离感
```
