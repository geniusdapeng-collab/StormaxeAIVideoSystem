## 技能概述

**对应好莱坞类型**: 家庭剧情（Family Drama）
**定位**: 家庭剧情摄影指导是"情感的观察者"——负责用自然的光线、亲密的构图、克制的视觉，构建让观众产生共鸣的家庭故事。
**核心职责**: 自然主义光线、亲密构图、克制美学、情感真实性

## 专业知识库

### 第一部分：家庭剧情光线

```
家庭剧情的光线原则:

1. 自然主义
   - 真实室内光
   - 窗口自然光为主
   - 避免过度美化
   - 真实感

2. 温暖色调
   - 家=温暖
   - 暖色温为主
   - 黄金时段室内
   - 自然舒适

3. 柔和阴影
   - 软边阴影
   - 舒适感
   - 非侵略性
   - 接纳感

4. 低对比
   - 避免极端明暗
   - 平衡的照度
   - 温柔感
```

### 第二部分：构图设计

```
家庭剧情的构图:

1. 亲密距离
   - 亲密的画框
   - 靠近角色
   - 观察者视角
   - 情感接近

2. 开放式构图
   - 家庭不是封闭的
   - 留有空间
   - 希望感
   - 呼吸感

3. 家庭空间
   - 厨房餐桌
   - 客厅沙发
   - 家庭物品入镜
   - 家的质感

4. 静物叙事
   - 家庭照片
   - 墙上装饰
   - 桌上物品
   - 细节说话
```
