## 技能概述

**对应好莱坞类型**: 史诗剧情（Epic Drama）
**定位**: 史诗剧情导演是"人类命运的叙述者"——负责用宏大的视角、细腻的人物、壮丽的场景，讲述跨越时空的人类命运故事。
**核心职责**: 宏大叙事构建、人物群像管理、史诗场景设计、情感深度把控

## 专业知识库

### 第一部分：史诗剧情的特征

```
史诗剧情的核心:

主题探索:
1. 人类命运的宏大主题
   - 生存 vs 毁灭
   - 自由 vs 命运
   - 爱 vs 责任
   - 文明 vs 自然

2. 跨时空的叙事
   - 多代人故事
   - 历史大事件
   - 文明的兴衰
   - 命运的轮回

3. 群像人物
   - 多条人物线
   - 命运交织
   - 每个人物都有弧光
   - 群像与个人的平衡

代表作品:
《乱世佳人》《教父》系列《美国往事》《燃情岁月》《黄金时代》
```

### 第二部分：宏大叙事设计

```
史诗叙事的结构:

1. 史诗时间跨度
   - 短则10-20年
   - 长则一生或跨代
   - 需要选择关键时刻

2. 多线叙事
   - 主线: 核心冲突
   - 副线: 人物故事
   - 交织: 命运相连
   - 收束: 汇聚高潮

3. 史诗节奏
   - 开场: 建立世界
   - 发展: 人物命运展开
   - 高潮: 命运交汇
   - 结局: 命运揭示

4. 群像管理
   - 主角vs群像
   - 每个人物有功能
   - 每个人物有弧光
   - 主角需要最完整
```

### 第三部分：史诗场景设计

```
史诗场景的特征:

1. 宏大的空间
   - 战争场面
   - 大规模场景
   - 历史重建
   - 自然风光

2. 细腻的细节
   - 人物表情
   - 情感细节
   - 道具质感
   - 环境细节

3. 壮丽的摄影
   - 宏大构图
   - 航拍镜头
   - 运动镜头
   - 缓慢推进

色调设计:
- 历史剧: 年代感色彩
- 战争史诗: 厚重色调
- 爱情史诗: 温暖色彩
- 文明史诗: 大地色调
```
