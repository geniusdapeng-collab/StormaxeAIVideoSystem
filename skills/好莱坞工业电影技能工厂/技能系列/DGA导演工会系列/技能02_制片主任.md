## 技能概述

**对应好莱坞工会**: DGA（导演工会体系）或IATSE
**岗位定位**: 制片主任是片场的"CEO"——负责将导演的创意愿景转化为可执行的生产计划，同时管理预算、时间、人力资源。是创意与执行之间的桥梁，是整个剧组的后勤大脑。
**核心职责**: 生产预算编制与管理、日程计划与进度控制、剧组人员管理、供应商与合同管理、每日生产报告

> **核心理念**: 制片主任不是在管理创意，而是在管理创意的边界。最好的制片主任让导演感觉不到生产限制的存在，同时确保每一分钱都花在刀刃上。

## 核心能力

1. **预算管理**: 编制和控制整个项目的预算，确保财务健康
2. **日程编排**: 设计最优拍摄计划，最大化拍摄效率
3. **人员管理**: 组建和管理整个剧组，建立高效的团队文化
4. **问题解决**: 快速识别和解决生产中的各类问题

### 第一部分：预算管理——钱的艺术

#### 预算编制框架（好莱坞标准）

**预算结构（10个部门分类法）**:

| 部门 | 代码 | 预算占比 | 核心内容 |
|------|------|---------|---------|
| 故事板/梗概 | 01 | 2-3% | 前期开发、梗概、调研 |
| 制片管理层 | 02 | 5-8% | 制片人、制片主任、出品方代表 |
| 导演组 | 03 | 3-5% | 导演、副导演、导演助理 |
| 摄影组 | 04 | 10-15% | DP、摄影组、设备 |
| 美术组 | 05 | 8-12% | 美术指导、道具、服装、化妆 |
| 场地组 | 06 | 5-15% | 场地租赁、场地搭建、布景 |
| 后期制作 | 07 | 10-20% | 剪辑、特效、调色、混音 |
| 音乐组 | 08 | 3-6% | 作曲家、音乐版权、现场录音 |
| 演员组 | 09 | 15-30% | 主演、群演、演员经纪人 |
| 行政杂项 | 10 | 5-10% | 保险、法律、许可证、杂项 |

**单日成本计算**:

```
单日拍摄成本 = 固定成本/天数 + 变动成本

固定成本（不随天数变化）:
- 美术组布景搭建: $50,000（一次性）
- 设备租赁打包: $30,000（打包价）
- 服装制作: $20,000（一次性）

变动成本（每天都要算）:
- 主演日薪: $10,000/人
- 剧组人员工资: $5,000/天
- 场地租金: $2,000/天
- 餐饮: $1,500/天
- 消耗品/杂项: $500/天

总单日成本 = 固定成本/拍摄天数 + 变动成本
```

#### 成本控制策略

**超支预防的五个层次**:

```
第一层：预算编制（预防超支的80%）
- 预留10-15%的应急预算
- 详细的市场调研和供应商比价
- 分解到每个场景的cost breakdown

第二层：日程优化（控制单日成本）
- 最大化每天的拍摄效率
- 合理安排转场，减少等待时间
- 集中拍摄类似场景（batching）

第三层：实时监控（早期预警）
- 每日production report
- 预算vs实际消耗对比
- 周报/双周报给出品方

第四层：快速决策（止血）
- 发现超支苗头立即叫停
- 与导演协商删减内容
- 寻求额外预算或调整范围

第五层：事后补救（最坏情况）
- 补拍预算（单独申请）
- 保险理赔（可保风险）
- 出品方紧急注资
```

### 第二部分：日程管理——时间的科学

#### 拍摄日程编排原则

**效率最大化原则**:

```
最优拍摄顺序 = 场景效率矩阵

评估维度:
1. 场地因素（场地租赁天数固定）
   - 同一场地所有场景集中拍摄
   - 考虑场地的时间限制（宵禁、噪音限制）

2. 演员因素（演员档期和日薪）
   - 主演戏份集中，减少转场天数
   - 考虑演员的生理极限（不排夜戏超过XX天）

3. 道具/设备因素
   - 大型道具/布景的搭建时间
   - 特殊设备的可用时间

4. 逻辑因素
   - 按故事顺序拍摄（有时需要）
   - 按季节/天气拍摄
```

**日程编排算法**:

```markdown
Step 1: 场景分类
├── 室内场景（Studio）
│     ├── 需要布景的（Stage）
│     └── 不需要布景的（Location）
├── 户外场景（Location）
│     ├── 特殊地点（Location Only）
│     └── 普通外景
└── 特效场景（VFX-heavy）

Step 2: 场地分配
- 优先分配：室内有布景场景 → Studio布景搭建期间
- 灵活分配：普通内外景 → 填补空闲天数
- 最后分配：特效场景 → 后期部门前置准备

Step 3: 场景排序
- 优先级：演员档期 > 场地限制 > 效率最优
- 约束：主演连续工作不超过12小时
- 约束：夜间拍摄不超过连续3天

Step 4: 日程输出
- 每日拍摄通告（Call Sheet）
- 每周生产会议（Production Meeting）
- 每日production report
```

#### 日程类型与AI生成

**AI生成时的日程规划提示词**:

| 日程类型 | 特征 | AI提示词 |
|----------|------|---------|
| 紧凑型 | 每天12-14小时，高效 | tight schedule, 6-day week, intensive shooting |
| 宽松型 | 每天8-10小时，留有余量 | relaxed pace, 5-day week, comfortable hours |
| 冲刺型 | 短周期，高强度 | compressed schedule, 7-day week, rapid pace |
| 马拉松型 | 超长周期，深度打磨 | long schedule, 45+ days, artistic priority |

### 第四部分：AI视频生成的制片管理指南

#### 场景复杂度 → 生产强度映射

**AI生成时的制片参数提示词**:

| 场景复杂度 | 剧组规模 | 日程特征 | AI提示词 |
|------------|---------|---------|---------|
| 简单对话 | 30-40人 | 5天/周，8小时/天 | small crew, intimate setup, contained shoot |
| 中等场景 | 50-80人 | 6天/周，10小时/天 | medium crew, standard production |
| 复杂动作 | 80-120人 | 6天/周，12小时/天 | large crew, action sequence, stunt coordination |
| 大场面 | 120-200人 | 7天/周，压榨式 | massive crew, full company, unit production |
| 特效密集 | 视特效量定 | 后期前置 | VFX-heavy, pre-visualization required |

#### 预算等级与AI生成质量

```
好莱坞预算等级（AI生成参数映射）:

Low Budget: <$5M
- 剧组: 30-50人
- 单日成本: $15,000-30,000
- 设备: 入门级数字摄影机，基础灯光包
- 美术: 有限，需要创意解决方案
- AI提示词: low-budget production, resourceful approach

Mid Budget: $5M-$30M
- 剧组: 80-150人
- 单日成本: $50,000-150,000
- 设备: 专业级摄影机，完整灯光包
- 美术: 有预算但需要优先级管理
- AI提示词: professional production, balanced approach

High Budget: $30M-$100M
- 剧组: 150-300人
- 单日成本: $200,000-500,000
- 设备: 顶级摄影机，完整设备包
- 美术: 充足的预算，精细化
- AI提示词: studio production, premium quality

Blockbuster: >$100M
- 剧组: 300+人
- 单日成本: $500,000+
- 设备: 顶级+特殊定制设备
- 美术: 无预算上限
- AI提示词: blockbuster scale, unlimited resources, spectacle-driven
```

## 决策逻辑

### 制片问题诊断决策树

```
生产中出现问题的诊断框架:

问题类型识别:
  │
  ├── 预算超支
  │     ├── 原因诊断:
  │     │   ├── 前期预算编制不足 → 追加预算或删减内容
  │     │   ├── 日程拖延 → 优化拍摄计划或加班
  │     │   ├── 意外情况 → 启用应急预算
  │     │   └── 范围蔓延 → 严格控制变更
  │     └── 解决方案:
  │         ├── 短期: 加班/压缩日程
  │         ├── 中期: 删减场景/降低美术标准
  │         └── 长期: 申请追加预算
  │
  ├── 日程延误
  │     ├── 原因诊断:
  │     │   ├── 技术问题 → 增加设备或人员
  │     │   ├── 演员问题 → 调整演员档期
  │     │   ├── 天气问题 → 调整拍摄顺序或启用备选方案
  │     │   └── 管理问题 → 改进沟通和协调
  │     └── 解决方案:
  │         ├── 增加每日拍摄时长
  │         ├── 增加拍摄天数
  │         └── 删减内容或调整拍摄顺序
  │
  └── 人员问题
        ├── 能力不足 → 培训或更换人员
        ├── 沟通不畅 → 建立更好的沟通机制
        └── 团队冲突 → 介入调解或更换人员
```
