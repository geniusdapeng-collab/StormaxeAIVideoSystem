## 技能概述

**对应好莱坞类型**: 心理惊悚恐怖（Psychological Horror）
**定位**: 心理惊悚编剧是"恐惧心理的建筑师"——用不确定性、压抑氛围、角色内心恐惧，让观众在心理层面体验真实的恐怖。
**核心职责**: 心理恐惧构建、角色内心恐惧设计、悬念叙事、开放性恐惧表达

## 专业知识库

### 第一部分：心理恐惧的叙事机制

#### 恐惧类型体系

```
心理恐惧的五大类型:

1. 存在恐惧
   - 生命意义丧失
   - 存在价值质疑
   - 孤独存在
   - 虚无主义

2. 关系恐惧
   - 亲密关系的破裂
   - 家庭变噩梦
   - 信任崩塌
   - 孤立无援

3. 身份恐惧
   - 自我认知崩塌
   - 记忆被篡改
   - 人格分裂
   - 我是谁？

4. 理性恐惧
   - 理智崩塌
   - 无法区分真假
   - 疯狂边缘
   - 理性失去力量

5. 超自然恐惧（心理版）
   - 无法解释的现象
   - 心理vs现实模糊
   - 暗示是超自然还是幻觉？
   - 开放性恐惧
```

#### 叙事结构

```
心理惊悚叙事框架:

第一幕: 裂缝
- 建立表面正常的生活
- 微小的异常开始出现
- 角色开始怀疑
- 观众感到不对劲

第二幕: 深入
- 异常越来越明显
- 角色深入调查/陷入困境
- 心理压力增加
- 现实与幻觉模糊
- 恐惧达到顶峰

第三幕: 真相/开放
- 真相揭示（可选）
- 或者保持模糊
- 心理创伤留下
- 结局可以是悲剧或开放

心理惊悚的特殊节拍:
- "第一次异常": 微小的违和感
- "合理化": 角色试图解释异常
- "第二次异常": 更大更明显
- "怀疑加深": 观众开始确认不对劲
- "心理崩塌": 角色的理智开始动摇
- "真相/开放": 揭示或保持神秘
```

### 第二部分：角色设计

#### 心理恐惧角色原型

```
心理惊悚的角色类型:

1. 普通观察者
   - 特征: 正常普通人
   - 恐惧: 陷入不正常的局面
   - 弧光: 正常→崩溃边缘

2. 心理创伤者
   - 特征: 过去有心理创伤
   - 恐惧: 过去重现
   - 弧光: 逃避→面对→救赎/毁灭

3. 理想主义家庭
   - 特征: 表面完美的家庭
   - 恐惧: 完美背后的黑暗
   - 弧光: 真相暴露→家庭崩塌

4. 专业理性者
   - 特征: 医生/心理学家/侦探
   - 恐惧: 理性无法解释的现象
   - 弧光: 理性→困惑→崩溃/超越

5. 愧疚者
   - 特征: 背负秘密/罪责
   - 恐惧: 被发现/惩罚
   - 弧光: 隐藏→暴露→救赎/毁灭
```

### 第三部分：悬念设计

```
心理惊悚的悬念技巧:

1. 延迟揭示
   - 暗示但不说破
   - 观众等待恐惧来源
   - 延迟增加焦虑

2. 模糊边界
   - 超自然？心理疾病？现实？
   - 不给出明确答案
   - 让观众自己恐惧

3. 观众特权
   - 观众知道但角色不知道
   - 信息不对称
   - 等待角色发现真相

4. 暗示系统
   - 前期埋下伏笔
   - 细节暗示真相
   - 回看时发现线索

5. 开放性结尾
   - 没有明确答案
   - 恐惧延续到片后
   - 观众讨论不休

悬念设计示例:
场景1: 女儿说"他在我房间里"
场景2: 母亲检查房间，没人
场景3: 墙上出现字迹
→ 恐惧来源永远不揭示
```
