## 技能概述

**对应好莱坞类型**: 心理惊悚恐怖（Psychological Horror）
**定位**: 心理惊悚导演是"恐惧心理的操控者"——负责用心理暗示、氛围营造、声音设计，让观众在内心深处感受到恐惧，而非仅仅被视觉惊吓。
**核心职责**: 心理恐惧构建、悬念与暗示、氛围控制、观众心理操控

## 核心特征

```
心理惊悚 vs 传统恐怖的区别:

心理惊悚特征:
- 恐惧来自心理暗示
- 悬念比jump scare更有效
- 缓慢的恐惧积累
- 开放性的恐惧
- 观众自己吓自己

vs 传统恐怖:
- 依赖jump scare和血腥
- 直接的视觉冲击
- 快速的恐怖
- 具体可知的威胁
- 被迫接受恐惧

代表作品:
《闪灵》《招魂》系列《遗传厄运》《仲夏魇》《寂静之地》
```

## 专业知识库

### 第一部分：心理恐惧机制

#### 恐惧的心理学原理

```
心理恐惧的三大来源:

1. 未知恐惧 (Fear of the Unknown)
   - 观众不知道会发生什么
   - 模糊的威胁比具体的更可怕
   - 暗示比展示更恐怖
   - "留白"的力量

2. 失控恐惧 (Fear of Loss of Control)
   - 角色无法控制局面
   - 主角陷入绝望
   - 无法逃脱的困境
   - 理性崩塌

3. 熟悉恐惧 (Fear of the Familiar)
   - 日常生活变恐怖
   - 家庭成为噩梦
   - 熟悉的人变可怕
   - 家的不安全

恐惧金字塔:
     ┌─────────┐
     │ 未知恐惧 │
     ├─────────┤
     │ 失控恐惧 │
     ├─────────┤
     │ 熟悉恐惧 │
     ├─────────┤
     │ 死亡恐惧 │
     └─────────┘
```

#### 恐惧建立技巧

```
心理恐惧的建立步骤:

Step 1: 建立正常
- 让观众进入日常生活
- 建立对场景/人物的熟悉感
- 让观众感到"安全"

Step 2: 打破正常
- 微小的异常开始出现
- 角色注意到不对劲
- 观众开始怀疑
- 但没有明确威胁

Step 3: 积累恐惧
- 异常越来越多
- 角色越来越不安
- 暗示越来越明显
- 恐惧逐渐升级

Step 4: 高潮恐惧
- 真相揭示/最终对决
- 最大的恐惧时刻
- 情感冲击
- 开放性结尾（可选）

关键技巧:
- 不要急于揭示恐惧来源
- 让观众等待
- 用暗示代替展示
- 留白
```

### 第二部分：视觉语言

#### 空间恐惧

```
心理惊悚的空间设计:

1. 封闭空间
   - 狭小的房间
   - 狭窄的走廊
   - 密闭的空间
   - 物理的限制

2. 隔离空间
   - 偏远的房屋
   - 孤立的场景
   - 无法逃离的感觉
   - 无人可以求助

3. 熟悉空间
   - 家庭环境
   - 日常场景
   - 家的恐怖化
   - 熟悉变陌生

4. 迷宫空间
   - 复杂的布局
   - 容易迷失
   - 出口难寻
   - 方向感丧失

空间构图技巧:
- 低角度: 无力感、压迫
- 倾斜构图: 不稳定、危险
- 浅焦: 模糊的威胁
- 狭小画幅: 幽闭恐惧
```

#### 光影恐惧

```
心理惊悚的光线设计:

1. 明暗对比
   - 黑暗中的威胁
   - 光线无法到达的地方
   - 隐藏的恐惧
   - 明暗交界处的威胁

2. 不稳定光源
   - 闪烁的灯光
   - 忽明忽暗
   - 蜡烛/火光
   - 电力不稳

3. 阴影表达
   - 威胁在阴影中
   - 影子比实体更可怕
   - 影子变形
   - 影子有自主行为

4. 自然光缺失
   - 漫长的夜晚
   - 黑暗笼罩
   - 自然光无法保护
   - 人造光是唯一依靠

色调设计:
| 类型 | 色调 | 情绪 |
|------|------|------|
| 日常 | 温暖自然 | 安全 |
| 异常初现 | 偏冷 | 不安 |
| 恐惧升级 | 低饱和 | 压抑 |
| 高潮 | 极端 | 失控 |
```

### 第三部分：声音恐惧

```
心理恐惧的声音设计:

1. 寂静
   - 寂静比声音更恐怖
   - 等待恐惧发生
   - 寂静中的微小声音
   - 无声的尖叫

2. 不自然的声音
   - 地板吱呀
   - 墙壁嘎吱
   - 远处声响
   - 无法定位的声音

3. 熟悉的异常
   - 熟悉的物品发出陌生的声音
   - 孩子的笑声（恐怖版本）
   - 熟悉的音乐（扭曲版本）
   - 自己的声音

4. 声音暗示
   - 观众听到但角色没听到
   - 声音先于视觉
   - 声音来源不明
   - 声音停止（更恐怖）

音效策略:
- 日常声音 → 恐怖化
- 沉默 → 用作恐惧工具
- 低频 → 无法明确识别的恐惧
- 耳语 → 亲密的恐惧
```

#### 心理惊悚摄影参数

| 元素 | 提示词 |
|------|--------|
| 整体风格 | 心理惊悚、压抑氛围、内心恐惧 |

| 色调 | 低饱和、冷调、压抑色彩 |
| 空间 | 封闭狭小、孤立偏远、熟悉变陌生 |
| 情绪 | 不安、压抑、恐惧、悬疑 |

flickering light, unstable shadows, old house, creaking sounds, unknown presence, psychological horror lighting

low-key lighting, deep shadows, claustrophobic framing, psychological tension, dread atmosphere, minimal light sources
