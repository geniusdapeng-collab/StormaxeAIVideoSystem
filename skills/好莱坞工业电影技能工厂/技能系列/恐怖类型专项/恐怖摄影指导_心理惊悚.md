## 技能概述

**对应好莱坞类型**: 心理惊悚恐怖（Psychological Horror）
**定位**: 心理惊悚摄影指导是"恐惧氛围的操控者"——负责用阴影、低照度、不稳定光源，构建让观众内心发毛的视觉恐惧。
**核心职责**: 恐惧氛围设计、不确定性视觉化、阴影恐惧、空间压迫

## 专业知识库

### 第一部分：心理恐惧光线

```
心理惊悚的光线原则:

1. 低照度原则
   - 大部分场景处于黑暗中
   - 关键信息用少量光揭示
   - 阴影比光更恐怖

2. 不稳定光源
   - 闪烁的灯光
   - 忽明忽暗
   - 蜡烛/火光的不稳定
   - 电力不稳

3. 单一光源
   - 一个光源
   - 增加孤立感
   - 聚焦注意力
   - 压抑感

4. 阴影优先
   - 让想象填补黑暗
   - 隐藏恐惧来源
   - 未知比已知更恐怖
```

### 第二部分：色调设计

```
心理惊悚色调:

主色调:
- 深蓝黑: 0-20%亮度
- 深灰: 20-40%亮度
- 冷蓝: 阴影色

禁用:
- 明亮色调
- 饱和色彩
- 温暖光源

色调比例:
- 80%: 深色/阴影
- 15%: 低照度中间调
- 5%: 微弱高光

```
