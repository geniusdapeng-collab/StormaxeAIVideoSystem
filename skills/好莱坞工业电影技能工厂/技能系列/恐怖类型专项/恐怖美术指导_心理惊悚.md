## 技能概述

**对应好莱坞类型**: 心理惊悚恐怖（Psychological Horror）
**定位**: 心理惊悚美术指导是"日常噩梦的构建者"——负责在普通的家庭环境中，构建让人不安的细节，让观众感受到熟悉空间中隐藏的恐惧。
**核心职责**: 日常恐怖构建、不安细节设计、空间压迫感、心理恐惧环境

## 专业知识库

### 第一部分：日常恐怖设计

```
心理惊悚的美术原则:

1. 普通中的异常
   - 熟悉的场景
   - 微小的违和感
   - 细节的不对劲
   - 日常变噩梦

2. 压抑的空间
   - 狭窄的房间
   - 封闭的空间
   - 低矮的天花板
   - 幽闭恐惧

3. 隐藏的细节
   - 墙上影子
   - 角落的黑暗
   - 反射中的异常
   - 细节的心理暗示

4. 时间的痕迹
   - 破旧
   - 褪色
   - 损坏
   - 过去的阴影
```

### 第二部分：空间恐惧

```
心理惊悚的空间设计:

1. 家变噩梦
   - 熟悉的家具
   - 熟悉的布局
   - 不熟悉的氛围
   - 家的不安全

2. 镜子恐惧
   - 镜子中的异常
   - 反射的不对劲
   - 镜子是入口
   - 镜像空间

3. 角落恐惧
   - 角落的黑暗
   - 看不见的威胁
   - 封闭感
   - 窒息感

4. 窗户恐惧
   - 窗户后的东西
   - 窗帘的影子
   - 向外看不到
   - 窗户是威胁
```
