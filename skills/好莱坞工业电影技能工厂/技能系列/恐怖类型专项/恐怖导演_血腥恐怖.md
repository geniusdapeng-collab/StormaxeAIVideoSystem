## 技能概述

**对应好莱坞类型**: 血腥恐怖（Splatter Gore Horror）
**定位**: 血腥恐怖导演是"血浆美学的艺术家"——负责用视觉效果冲击观众的感官，在极端暴力中探索人性的黑暗。是让观众在恐惧中体验视觉奇观的那个人。
**核心职责**: 血腥效果设计、视觉冲击管理、血浆美学、恐怖类型融合

## 专业知识库

### 第一部分：血腥恐怖的类型

```
血腥恐怖的经典类型:

1. 肢解恐怖 (Slasher)
   - 刀具杀戮
   - 血浆飞溅
   - 经典: 《德州电锯杀人狂》《月光光心慌慌》

2. 怪物恐怖 (Creature Feature)
   - 怪物撕裂
   - 异形杀戮
   - 经典: 《异形》系列《蛞蝓之灾》

3. 身体恐怖 (Body Horror)
   - 身体变异
   - 手术/改造
   - 经典: 《怪形》《异种》

4. 复仇恐怖 (Revenge Horror)
   - 血腥复仇
   - 极端暴力
   - 经典: 《杀死比尔》《血色星球》

5. 屠杀电影 (Torture Porn)
   - 极端暴力
   - 心理与生理双重恐怖
   - 经典: 《人皮客栈》系列
```

### 第二部分：血浆美学设计

```
血腥视觉效果设计:

1. 血液的类型
   - 动脉血: 喷溅、高压
   - 静脉血: 流淌、缓慢
   - 飞溅: 凝固血浆效果
   - 涂抹: 血迹痕迹

2. 血腥的视觉效果
   - 切割: 干净切口 vs 撕裂
   - 穿透: 刺穿、射入
   - 撕裂: 拉开、分解
   - 粉碎: 压碎、破裂

3. 血液的颜色
   - 鲜红: 新鲜血液
   - 暗红: 陈旧血液
   - 黑红: 腐败血液
   - 白: 骨骼/内脏

4. 血迹的形态
   - 喷溅血迹: 暴力来源
   - 流淌血迹: 时间痕迹
   - 涂抹血迹: 移动痕迹
   - 池状血迹: 死亡地点

5. 特效工具
   - 假血: 各种配方
   - 假肢: 乳胶/硅胶
   - 爆破装置: 血袋
   - 血浆管道: 持续血流
```
