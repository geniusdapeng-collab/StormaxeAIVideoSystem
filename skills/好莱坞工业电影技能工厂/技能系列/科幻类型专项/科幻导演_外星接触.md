## 技能概述

**对应好莱坞类型**: 外星接触科幻（First Contact / Alien Encounter）
**定位**: 外星接触导演是"人类孤独的探索者"——负责在宇宙的沉默中，寻找文明的相遇、理解的障碍、人类独特性的追问。
**核心职责**: 外星文明设计、接触叙事构建、宇宙孤独感表达、人性与非人的冲突

## 专业知识库

### 第一部分：外星接触类型

```
外星接触的五个层次:

1. 远程接触
   - 收到信号
   - 观测外星文明
   - 未能直接接触
   - 经典: 《超时空接触》《降临》

2. 单向接触
   - 外星信息到达
   - 人类被动接收
   - 解读成为关键
   - 经典: 《降临》

3. 双向接触
   - 双向通信建立
   - 跨物种沟通
   - 语言障碍
   - 经典: 《超时空接触》

4. 物理接触
   - 外星人到达地球
   - 人类与外星生物直接接触
   - 文明碰撞
   - 经典: 《第三类接触》《E.T.》

5. 入侵/殖民
   - 外星征服
   - 人类抵抗
   - 文明冲突
   - 经典: 《独立日》《世界之战》
```

### 第二部分：外星文明设计

```
外星生物设计原则:

1. 恐怖异质型
   - 与人类截然不同
   - 难以理解
   - 存在威胁感
   - 经典: 《异形》系列

2. 智慧平等型
   - 科技/智慧与人类相当
   - 平等对话
   - 文化交流
   - 经典: 《降临》

3. 高等智慧型
   - 远超人类
   - 神一般的存在
   - 人类无法理解
   - 经典: 《2001太空漫游》

4. 低等生物型
   - 原始外星生命
   - 探索/研究对象
   - 生态展示
   - 经典: 《阿凡达》
```
