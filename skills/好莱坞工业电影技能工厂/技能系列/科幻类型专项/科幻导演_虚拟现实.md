## 技能概述

**对应好莱坞类型**: 虚拟现实科幻（Virtual Reality / Simulation）
**定位**: 虚拟现实导演是"真实边界的探索者"——负责用视觉语言模糊真实与虚拟的边界，让观众质疑"什么是真实"。
**核心职责**: 虚拟现实世界构建、真实vs虚拟表达、模拟世界设计、数字美学

## 专业知识库

### 第一部分：虚拟现实世界构建

```
虚拟现实类型:

1. 完全沉浸
   - 100%虚拟世界
   - 脱离肉体的体验
   - 数字形态存在
   - 经典: 《黑客帝国》

2. 增强现实
   - 虚拟叠加现实
   - 现实增强
   - 混合世界
   - 经典: 《攻壳机动队》

3. 记忆世界
   - 进入他人记忆
   - 重现过去
   - 心理学应用
   - 经典: 《全面启动》

4. 游戏世界
   - 沉浸式游戏
   - 玩家进入游戏
   - 死亡=真实死亡
   - 经典: 《头号玩家》
```

### 第二部分：双重世界设计

```
虚拟现实的视觉表达:

真实世界:
- 自然光
- 自然色彩
- 物理规律
- 模糊/不完美
- 情感真实

虚拟世界:
- 过度锐利
- 超饱和色彩
- 几何化
- 数字纹理
- 完美但空洞

对比技巧:
- 同时呈现两个世界
- 色彩对比
- 构图差异
- 运动差异

虚拟世界美学:
- 几何化
- 网格/线条
- 数字瀑布流
- 低多边形
- 超现实色彩
```
