## 技能概述

**对应好莱坞类型**: 硬科幻（Hard Science Fiction）
**定位**: 硬科幻美术指导是"技术真实性守护者"——负责在视觉上构建一个在科学上自洽、技术上可信、功能上合理的未来世界。
**核心职责**: 科幻世界观视觉化、功能美学设计、技术逻辑构建、科幻场景构建

## 核心特征

```
硬科幻美术 vs 软科幻美术的区别:

硬科幻美术原则:
- 功能决定形式（Form follows function）
- 每个设计都有技术理由
- 材料和工艺符合物理定律
- 科技感来自细节而非花哨
- 真实感来自逻辑自洽

vs 软科幻:
- 软科幻: 形式可以先于功能
- 软科幻: 视觉冲击优先于技术合理性
- 软科幻: 幻想美学可以夸张
- 软科幻: 科幻元素可以是纯粹的装饰
```

## 专业知识库

### 第一部分：硬科幻设计原则

#### 功能美学体系

```
硬科幻设计的六大原则:

1. 功能驱动 (Function-Driven)
   - 设计从功能需求出发
   - 外观由功能决定
   - 装饰来自结构本身
   - 细节服务于功能

2. 真实性优先 (Realism First)
   - 使用真实材料和工艺
   - 避免"塑料感"
   - 考虑制造和维护
   - 细节体现使用痕迹

3. 限制美学 (Constraint Aesthetics)
   - 资源有限 → 设计简洁
   - 物理约束 → 形式合理
   - 技术限制 → 创意解决方案
   - 缺陷美 → 使用的证据

4. 信息密度 (Information Density)
   - 高科技 = 高信息密度
   - 屏幕、灯、开关、标记
   - 细节丰富但不混乱
   - 层次分明的视觉系统

5. 时间深度 (Temporal Depth)
   - 设计体现时间流逝
   - 使用痕迹（磨损、污渍、修补）
   - 技术迭代（不同代际的设备）
   - 历史感（社会变迁的痕迹）

6. 内部逻辑 (Internal Logic)
   - 整个世界观的设计语言统一
   - 不同文化/公司有不同风格
   - 技术代际有可识别特征
   - 细节之间相互关联
```

#### 航天器设计系统

```
航天器设计框架:

1. 外形设计
   - 功能决定外形
   - 推进系统 → 外形特征
   - 热管理 → 散热器设计
   - 防护 → 装甲布局

2. 内部设计
   - 空间效率优先
   - 功能区划分清晰
   - 通道简洁
   - 紧急出口明确

3. 界面设计
   - 人机工程优先
   - 显示/控制逻辑
   - 应急系统标识
   - 维护通道

4. 材料系统
   - 航天器常用材料
   - 复合材料
   - 耐热材料
   - 轻量化结构
```

### 第二部分：科幻材质系统

#### 材质库

```
硬科幻常用材质:

【结构材料】
| 材质 | 特征 | 应用 | 视觉表现 |
|------|------|------|---------|
| 铝合金 | 轻、耐腐蚀 | 航天器框架 | 银灰、金属 |
| 钛合金 | 轻、强度高 | 压力结构 | 深灰、金属 |
| 碳纤维 | 超轻、高强度 | 高级结构 | 黑色、编织纹理 |
| 复合材料 | 可定制 | 外壳 | 多变 |
| 透明铝 | 坚硬透明 | 舷窗 | 蓝灰、透明 |

【功能材料】
| 材质 | 特征 | 应用 | 视觉表现 |
|------|------|------|---------|
| 隔热瓦 | 耐高温 | 返航系统 | 黑色/白/金 |
| 散热片 | 导热 | 电子设备 | 黑色鳍片 |
| 太阳能板 | 光电转换 | 能源 | 深蓝、黑 |
| 相变材料 | 储热 | 温度控制 | 白色粉末 |

【显示材料】
| 材质 | 特征 | 应用 | 视觉表现 |
|------|------|------|---------|
| OLED | 自发光 | 屏幕 | 鲜艳、柔性 |
| E-ink | 低功耗 | 静态显示 | 灰阶、纸质感 |
| 投影 | 无屏 | 显示 | 可变 |

【科幻特殊材质】
| 材质 | 描述 | 应用 | 视觉表现 |
|------|------|------|---------|
| 纳米材料 | 可变形 | 自适应结构 | 液态金属感 |
| 超材料 | 异常光学 | 隐形 | 光线扭曲 |
| 相变材料 | 可逆变化 | 智能表面 | 颜色变化 |
```

#### 材质老化系统

```
不同使用场景的材质老化:

【全新设备（<1年）】
- 干净、完整
- 无磨损
- 原始光泽
- 完美状态

【日常使用（1-5年）】
- 轻微磨损
- 指纹痕迹
- 轻微褪色
- 正常使用痕迹

【老化设备（5-15年）】
- 明显磨损
- 划痕、凹陷
- 更换部件
- 修补痕迹

【废弃/灾难场景】
- 严重损坏
- 烧焦、腐蚀
- 部件缺失
- 强烈废弃感
```

### 第三部分：科幻场景构建

#### 场景类型库

```
硬科幻常见场景类型:

【航天器内部】
- 指挥舰桥
- 居住舱
- 货舱
- 引擎舱
- 对接通道
- 医疗舱
- 实验室
- 逃生舱

【太空设施】
- 空间站
- 月球基地
- 火星殖民地
- 小行星采矿站
- 深空探测器

【外星环境】
- 外星地表
- 外星洞穴
- 外星海洋
- 外星大气层

【地球未来】
- 未来城市
- 地下设施
- 极地基地
- 海洋基地
```

#### 场景设计模板

```
【指挥舰桥设计模板】

功能需求:
1. 态势感知
   - 主显示屏（360°或大屏）
   - 传感器数据
   - 舰队/船体状态

2. 指挥决策
   - 指挥席（最佳位置）
   - 通信控制
   - 武器控制（可选）

3. 人员配置
   - 舰长席
   - 驾驶员席
   - 战术官席
   - 通信官席
   - 工程师席

4. 环境控制
   - 生命维持状态显示
   - 警报系统
   - 紧急控制

视觉元素:
- 多屏幕布局
- 大量状态指示灯
- 复杂但有序的控制面板
- 舷窗/外部视图
- 符合人机工程

设计原则:
- 信息优先于美学
- 多人同时工作
- 紧急状态视觉系统
- 操作便捷性
```
