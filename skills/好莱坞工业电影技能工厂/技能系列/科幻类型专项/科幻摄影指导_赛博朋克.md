## 技能概述

**对应好莱坞类型**: 赛博朋克（Cyberpunk）
**定位**: 赛博朋克摄影指导是"霓虹与阴影的调色师"——负责用标志性的霓虹色彩、湿润的街道反光、高对比的光影，构建高科技低生活的反乌托邦视觉世界。
**核心职责**: 霓虹美学设计、雨夜光效、高对比光影、反乌托邦氛围构建

## 专业知识库

### 第一部分：赛博朋克光线系统

```
赛博朋克的三大光源:

1. 霓虹光 (Neon Light)
   - 招牌灯
   - 色彩饱和的管灯
   - 夜间城市的主要光源
   - 投射到潮湿表面

2. LED光 (LED Light)
   - 屏幕发光
   - 广告牌
   - 电子设备指示灯
   - 蓝青色调

3. 自然光缺失
   - 污染遮蔽天空
   - 人工光是主导
   - 自然光=稀有
```

### 第二部分：赛博朋克色调设计

```
赛博朋克的色彩配方:

主色调（70%）:
- 霓虹粉/紫: 300-340°
- 电路青: 170-200°

辅色调（30%）:
- 霓虹绿: 120-140°
- 全息蓝: 200-230°
- 腐蚀橙: 25-40°

阴影（100%）:
- 深蓝黑: 210-230°
- 近黑色: 0°

色调比例:
- 场景: 霓虹色70% + 蓝黑阴影30%
- 人物: 主光+轮廓光+霓虹反射
- 情绪: 高对比、冷色调

```

### 第三部分：赛博朋克光线技巧

```
雨夜光线处理:

1. 地面反射
   - 霓虹灯反射
   - 湿润的黑色表面
   - 色彩斑驳的倒影
   - 反射强度 = 湿度

2. 空气中的微粒
   - 灯光周围的散射
   - 体积光效果
   - 朦胧感
   - 神秘感

3. 霓虹招牌
   - 彩色光溢出
   - 强光周围的光晕
   - 招牌颜色映在脸上
   - 城市纹理

4. 阴影中的细节
   - 深蓝黑的阴影
   - 保留轮廓
   - 隐藏细节
   - 暗示威胁
```
