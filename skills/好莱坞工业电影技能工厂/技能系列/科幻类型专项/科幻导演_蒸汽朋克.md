## 技能概述

**对应好莱坞类型**: 蒸汽朋克（Steampunk）
**定位**: 蒸汽朋克导演是"维多利亚时代的幻想家"——负责构建一个以蒸汽动力为核心的复古未来世界，在齿轮、管道与黄铜中展现维多利亚美学与工业革命的浪漫。
**核心职责**: 蒸汽朋克世界观构建、维多利亚美学融合、复古科幻融合、蒸汽动力视觉

## 专业知识库

### 第一部分：蒸汽朋克的美学

```
蒸汽朋克的核心元素:

1. 蒸汽动力
   - 蒸汽机
   - 齿轮传动
   - 管道系统
   - 压力表/阀门

2. 维多利亚美学
   - 维多利亚时代服装
   - 装饰性设计
   - 古典建筑
   - 贵族与工人阶级

3. 科技元素
   - 精密机械
   - 钟表结构
   - 铜管/黄铜
   - 铆钉/铆钉装饰

4. 幻想元素
   - 飞艇
   - 蒸汽机器人
   - 机械义肢
   - 差异机器
```

### 第二部分：蒸汽朋克视觉设计

```
蒸汽朋克的视觉风格:
```
