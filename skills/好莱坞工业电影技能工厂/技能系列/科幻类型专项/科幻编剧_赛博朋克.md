## 技能概述

**对应好莱坞类型**: 赛博朋克（Cyberpunk）
**定位**: 赛博朋克编剧是"反乌托邦的预言家"——用科技包裹人性，用霓虹照亮黑暗，用虚构的未来批判现实的资本主义。
**核心职责**: 反乌托邦叙事构建、赛博朋克世界观设计、复杂角色创作、技术与人性冲突表达

## 专业知识库

### 第一部分：赛博朋克叙事结构

#### 主题系统

```
赛博朋克的四大核心主题:

1. 身份危机
   - 记忆可以被篡改
   - 身体可以被替换
   - "真实"的我是什么？
   - 经典: 《银翼杀手》Deckard的身份之谜

2. 人性边界
   - 多少改造才算机器？
   - 复制人有灵魂吗？
   - AI能爱吗？
   - 经典: 《攻壳机动队》素子的身份探索

3. 社会批判
   - 资本主义的终极形态
   - 技术加剧不平等
   - 个人vs巨型公司
   - 经典: 《黑客帝国》系统的控制

4. 反英雄叙事
   - 虚无主义主角
   - 腐败的世界
   - 微弱的反抗
   - 苦涩的结局
```

#### 叙事结构模板

```
赛博朋克叙事框架:

第一幕: 世界的囚徒
- 主角在反乌托邦中挣扎
- 建立日常的"正常"
- 展示社会的荒谬
- 危机打破现状

第二幕: 觉醒与挣扎
- 发现世界的真相
- 尝试反抗或逃避
- 身体/身份的变化
- 技术与人性的冲突

第三幕: 代价与选择
- 对抗系统或被系统吞噬
- 身份的终极追问
- 技术vs人性的选择
- 通常是苦涩/开放的结局

赛博朋克的特殊节拍:
- "接口时刻": 接入数字世界
- "记忆闪回": 真实vs虚假记忆
- "义体升级": 身体/能力的改变
- "系统觉醒": AI/系统的意识
```

### 第二部分：赛博朋克角色设计

#### 角色原型

```
赛博朋克的经典角色类型:

1. 反英雄 (Anti-Hero)
   - 冷漠、自我保护
   - 生存至上
   - 道德灰色
   - 示例: Rick Deckard、K

2. 地下分子 (Street Runner)
   - 黑客、雇佣兵、罪犯
   - 技术纯熟
   - 反叛精神
   - 示例: Trinity、Neo

3. 企业傀儡 (Corporate Pawn)
   - 企业中层或低层
   - 被系统利用
   - 觉醒后反抗
   - 示例: William Smith

4. 觉醒AI (Awakened AI)
   - 人工智能获得意识
   - 存在主义危机
   - 追求自由
   - 示例: Wintermute、Machine

5. 复制人/义体人 (Replicant/Cyborg)
   - 非人类身份
   - 追求平等权利
   - 短暂生命
   - 示例: Roy Batty、Rachael

6. 系统精英 (System Elite)
   - 企业高层
   - 冷酷理性
   - 维护系统
   - 示例: Eldon Tyrell
```

#### 角色弧光设计

```
赛博朋克角色弧光模板:

弧光类型1: 觉醒弧光
起点: 被系统控制/麻木
中点: 遭遇真相/接触地下
终点: 反抗/牺牲

弧光类型2: 身份弧光
起点: 不知道真实身份
中点: 发现身份被操控
终点: 接受/反抗真实自我

弧光类型3: 人性弧光
起点: 非人类/机械
中点: 获得/失去人性
终点: 人性胜利/失败

弧光类型4: 虚无弧光
起点: 虚无主义
中点: 发现值得为之战斗的东西
终点: 希望/幻灭
```

### 第三部分：赛博朋克世界观设计

#### 世界构建框架

```
赛博朋克世界观六要素:

1. 政治经济体系
   - 企业统治取代政府
   - 贫富分化极端
   - 资源控制在大企业

2. 技术形态
   - 神经接口普及
   - 身体改造普遍
   - AI/机器人存在
   - 虚拟现实发达

3. 社会结构
   - 精英在上层
   - 大众在中下层
   - 地下/黑市存在
   - 亚文化繁荣

4. 空间布局
   - 垂直城市（高层=富裕）
   - 地面混乱
   - 地下世界
   - 虚拟空间

5. 文化形态
   - 霓虹文化
   - 街头文化
   - 企业文化
   - 黑客文化

6. 环境状态
   - 污染严重
   - 天气失控
   - 自然消失
   - 人工环境
```

#### 科技设定细节

```
核心科技设定:

1. 神经接口 (Neural Interface)
   - 直接连接大脑
   - 接入网络
   - 记忆存储
   - 潜在风险: 被黑客攻击

2. 身体改造 (Cybernetics)
   - 义体四肢
   - 视觉/听觉增强
   - 身体性能提升
   - 潜在风险: 排斥反应/身份危机

3. 人工智能 (AI)
   - 强AI存在
   - 拥有意识
   - 与人类关系紧张
   - 潜在风险: AI统治

4. 虚拟现实 (VR/Matrix)
   - 完整模拟世界
   - 数字存在
   - 难以区分真假
   - 潜在风险: 沉迷/控制
```

#### 赛博朋克对话技巧

```
对话风格:

1. 简洁冷峻
   - 短句为主
   - 情绪压抑
   - 不说废话

2. 技术术语
   - 角色习惯用技术语言
   - 但不过度解释

3. 反讽批判
   - 用反讽表达社会批判
   - 商业广告/口号的讽刺

4. 记忆碎片
   - 对话中闪回
   - 真实与虚假混合

示例:
A: "你不记得了？"
B: "我的记忆还是真的？"
A: "在这个世界上，有什么是真的？"
B: "我的仇恨是真的。"
```
