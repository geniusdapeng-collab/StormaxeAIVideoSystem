## 技能概述

**对应好莱坞类型**: 末日生存科幻（Post-Apocalyptic / Survival Sci-Fi）
**定位**: 末日生存导演是"文明的废墟诗人"——负责在文明的废墟上，用镜头语言展现人类的韧性与绝望、希望与毁灭的永恒冲突。
**核心职责**: 末日世界观构建、后启示录美学、生存叙事构建、废墟氛围营造

## 专业知识库

### 第一部分：末日世界观构建

```
末日场景的类型:

1. 自然灾难后
   - 气候灾难
   - 瘟疫大流行
   - 小行星撞击
   - 超级太阳耀斑

2. 人为灾难后
   - 核战争
   - 生化灾难
   - AI失控
   - 资源枯竭

3. 社会崩溃后
   - 经济崩溃
   - 政府瓦解
   - 无政府状态
   - 文明倒退

时间线:
- 灾难刚发生: 混乱、逃亡
- 1-10年: 秩序重建/部落形成
- 10-100年: 新社会形态
- 100年+: 后启示录文明

```

### 第二部分：后启示录美学

```
末日场景的视觉特征:

1. 废墟美学
   - 建筑残骸
   - 自然侵蚀
   - 人类存在的痕迹
   - 时间的重量

2. 物资匮乏
   - 破旧衣物
   - 拼凑装备
   - 废物利用
   - 匮乏感

3. 自然回归
   - 植被侵蚀建筑
   - 动物回归城市
   - 生态系统变化
   - 自然的力量

4. 危险环境
   - 污染区域
   - 辐射区
   - 极端天气
   - 危险生物

色调设计:
- 灰褐色调: 废墟尘土
- 绿色: 自然侵蚀
- 橙黄: 危险/火
- 蓝灰: 压抑/寒冷
```
