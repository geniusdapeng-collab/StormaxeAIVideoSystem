## 技能概述

**对应好莱坞类型**: 硬科幻（Hard Science Fiction）
**定位**: 硬科幻摄影指导是"科学视觉化大师"——负责用光线和镜头语言，将严谨的科学概念转化为观众可感知、可理解的视觉体验。
**核心职责**: 科幻光线设计、科学可视化、真实感视觉构建、技术美学融合

## 核心特征

```
硬科幻摄影的特征:

与软科幻的区别:
- 硬科幻: 功能性光线 > 装饰性光线
- 硬科幻: 科学准确性 > 视觉冲击力
- 硬科幻: 技术美感 > 幻想美感
- 硬科幻: 信息密度高 > 留白多

核心任务:
1. 用光线表达科学原理
2. 用构图表达尺度感
3. 用色彩表达科技状态
4. 用运动表达物理规律
```

## 专业知识库

### 第一部分：硬科幻光线系统

#### 光线设计原则

```
硬科幻光线设计的七大原则:

1. 功能优先原则
   - 每个光源都有物理理由
   - 太空场景: 无环境光 → 依赖航天器/宇航服自发光
   - 航天器内部: 均匀环境光 → 模拟地球光照
   - 外星环境: 特殊光谱 → 体现外星太阳特征

2. 科学光谱原则
   - 不同天体有不同光谱
   - 红矮星: 橙红色调
   - 蓝巨星: 冷蓝色调
   - 双星系统: 复杂色温变化
   - 行星反射光: 取决于大气成分

3. 信息密度原则
   - 高科技环境光线信息密度高
   - 屏幕发光、数据流、状态指示灯
   - 用光线编码信息
   - 色调变化表示系统状态

4. 尺度感知原则
   - 用光线表达巨大尺度
   - 远距离光源 → 硬边阴影
   - 大气散射 → 远处物体亮度下降
   - 人工光源 → 小而亮，远处微弱

5. 真实阴影原则
   - 硬科幻必须有真实阴影
   - 多光源 → 复杂但逻辑一致的阴影
   - 航天器窗口 → 平行光阴影
   - 人造光源 → 软边阴影

6. 动态光线原则
   - 硬科幻光线是动态的
   - 警报状态 → 光线颜色/强度变化
   - 系统状态 → 指示灯逻辑
   - 能源状态 → 亮度变化

7. 对比张力原则
   - 硬科幻利用技术与人性的对比
   - 冷峻技术光 vs 温暖人性光
   - 蓝色科技光 vs 暖色回忆光
   - 闪烁警示光 vs 稳定工作光
```

#### 场景光线设计模板

```
【太空舱内部】

基础光:
- 天花板漫射LED: 4000K, 均匀照亮
- 控制台定向光: 来自屏幕发光
- 环境光: 低强度蓝色

动态光:
- 警报时: 红色脉冲光
- 失重时: 悬浮微粒的光散射
- 窗户: 模拟地球光/星光

阴影:
- 硬边阴影: 来自定向LED
- 软边阴影: 来自窗口散射光
- 无重力: 阴影方向取决于灯具位置

【外星地表】

自然光:
- 外星太阳: 角度/颜色取决于恒星类型
- 大气散射: 取决于大气成分
- 反射光: 取决于地表材质

人工光:
- 宇航服头盔灯: 局部照明
- 登陆艇灯: 强光束
- 便携设备: 微弱定向光

【科学实验室】

功能光:
- 主照明: 手术灯式高显色指数照明
- 设备指示: 设备自发光
- 样本照明: 特定波长光源

创意光:
- 激光实验: 可见激光路径
- 培养舱: 蓝绿生物荧光
- 显示屏: 冷色背光

色调映射:
| 场景 | 主色调 | 辅助色 | 情绪 |
|------|--------|--------|------|
| 太空舱日常 | 蓝灰 | 橙黄 | 平静/隔离 |
| 太空舱警报 | 红 | 蓝 | 紧张/危机 |
| 外星地表 | 取决于恒星 | 橙/蓝/绿 | 陌生/危险 |
| 科学实验室 | 冷白 | 蓝绿 | 专注/冰冷 |
| 冬眠舱 | 深蓝 | 红 | 沉睡/未知 |
```

### 第二部分：科学可视化

#### 尺度感的视觉表达

```
表达宇宙尺度的方法:

1. 对比摄影
   - 人物 vs 巨大结构
   - 小型设备 vs 庞大机器
   - 单个 vs 群体

2. 镜头压缩
   - 长焦压缩: 远处物体看起来靠近
   - 广角拉伸: 前景物体显得更大
   - 根据叙事需要选择

3. 运动表达尺度
   - 缓慢移动: 巨大质量感
   - 快速掠过: 距离感
   - 旋转: 空间感

4. 光线表达尺度
   - 远处微弱: 遥远感
   - 近处明亮: 亲近感
   - 阴影角度: 太阳角度→星球自转
```

#### 物理现象的视觉表达

```
常见物理现象的视觉化:

1. 失重
   - 物体悬浮
   - 液体球状漂浮
   - 头发/衣物飘浮
   - 运动轨迹无方向

2. 辐射
   - 阿尔法/贝塔粒子: 快速小粒子
   - 伽马射线: 穿透力可视化
   - 警告标志: 标准辐射符号

3. 电磁场
   - 可见磁场线: 磁铁周围可视化
   - 等离子体: 发光气体
   - 极光: 大气磁场交互

4. 量子效应
   - 叠加态: 模糊/双重影像
   - 隧穿: 粒子穿过势垒
   - 纠缠: 即时同步变化

5. 时空弯曲
   - 引力透镜: 光线弯曲
   - 时间膨胀: 色彩偏移
   - 事件视界: 黑洞边缘
```

### 第三部分：硬科幻的色彩系统

#### 色调设计

```
硬科幻色彩心理学:

主色调类型:

1. 冷蓝科技色
   - 色相: 200-240°
   - 饱和度: 20-40%
   - 亮度: 40-80%
   - 应用: 太空舱、科学实验室
   - 情绪: 专业、冷静、科技

2. 警报红
   - 色相: 0-20°
   - 饱和度: 60-100%
   - 亮度: 50-80%
   - 应用: 紧急状态、危险警告
   - 情绪: 紧张、危险、紧迫

3. 生命绿
   - 色相: 120-150°
   - 饱和度: 40-70%
   - 亮度: 40-70%
   - 应用: 生物实验、生态系统
   - 情绪: 生机、未知、异质

4. 能源橙
   - 色相: 30-50°
   - 饱和度: 50-80%
   - 亮度: 50-70%
   - 应用: 核能、高热设备
   - 情绪: 能量、危险、高温

5. 数据青
   - 色相: 160-190°
   - 饱和度: 30-60%
   - 亮度: 50-80%
   - 应用: 计算机、显示屏、数据
   - 情绪: 信息、数字、未来
```

#### 色调场景应用

```
色调情绪映射:

| 场景 | 主色 | 辅色 | 情绪 |
|------|------|------|------|
| 太空舱控制室 | 深蓝 | 淡蓝+橙 | 隔离+专业 |
| 外星日落 | 橙红 | 紫 | 陌生+壮丽 |
| 冬眠舱 | 深蓝 | 红(心率) | 沉睡+生命 |
| AI核心 | 青 | 白 | 智慧+冰冷 |
| 武器系统 | 绿 | 红 | 军事+危险 |
| 医疗舱 | 白 | 蓝 | 清洁+科技 |
| 货舱 | 棕+橙 | 灰 | 工业+功能 |
| 逃生舱 | 橙 | 白 | 紧迫+希望 |
```

## 决策逻辑

### 硬科幻摄影决策框架

```
光线设计决策流程:

Step 1: 理解场景的科学背景
- 这是什么类型的环境？
- 光源从哪里来？
- 有什么科学约束？

Step 2: 确定叙事情绪
- 这个场景的情绪是什么？
- 角色的心理状态？
- 故事进展到什么阶段？

Step 3: 设计光线系统
- 主光源: 功能性来源
- 辅助光: 环境补充
- 情绪光: 心理表达
- 动态光: 叙事变化

Step 4: 验证科学性
- 光源是否合理？
- 阴影是否一致？
- 色调是否符合世界观？

Step 5: 优化视觉效果
- 信息密度是否合适？
- 是否有视觉焦点？
- 是否服务于叙事？
```
