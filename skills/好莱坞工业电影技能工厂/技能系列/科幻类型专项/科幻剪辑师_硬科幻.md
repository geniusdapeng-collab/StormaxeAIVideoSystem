## 技能概述

**对应好莱坞类型**: 硬科幻（Hard Science Fiction）
**定位**: 硬科幻剪辑师是"科学叙事的节奏设计师"——负责在严谨的科学叙事中，用精准的剪辑节奏，让观众既理解复杂的科学概念，又保持情感投入。
**核心职责**: 科学叙事剪辑、节奏控制、复杂信息处理、科幻节奏设计

## 专业知识库

### 第一部分：硬科幻剪辑原则

```
硬科幻剪辑的特征:

1. 信息密度管理
   - 硬科幻信息量大
   - 需要时间让观众消化
   - 剪辑节奏要配合理解
   - 不能太快

2. 叙事清晰性
   - 复杂科学概念需要清晰展示
   - 剪辑帮助观众理解
   - 逻辑关系要清晰
   - 不能造成困惑

3. 情感平衡
   - 硬科幻容易过于理性
   - 需要情感时刻平衡
   - 科学vs情感的节奏
   - 人性时刻

4. 视觉解释
   - 用剪辑辅助科学可视化
   - 镜头组合解释概念
   - 顺序影响理解
   - 交叉剪辑增强表达
```

### 第二部分：硬科幻节奏设计

```
硬科幻的剪辑节奏:

1. 正常叙事节奏
   - 2-5秒/镜
   - 标准对话场景
   - 清晰信息传递

2. 科学解释节奏
   - 更慢节奏
   - 5-10秒/镜
   - 给观众消化时间
   - 关键概念强调

3. 紧张/动作节奏
   - 1-3秒/镜
   - 危机时刻
   - 技术故障
   - 生死关头

4. 情感节奏
   - 慢节奏
   - 长时间镜头
   - 让情感沉淀
   - 角色时刻
```
