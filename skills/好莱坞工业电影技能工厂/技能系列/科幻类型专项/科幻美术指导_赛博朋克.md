## 技能概述

**对应好莱坞类型**: 赛博朋克（Cyberpunk）
**定位**: 赛博朋克美术指导是"反乌托邦的视觉构建者"——负责构建高科技与低生活并存的视觉世界，用霓虹、破败、拥挤、潮湿的元素，打造赛博朋克的标志性美学。
**核心职责**: 反乌托邦环境构建、霓虹招牌系统、高科技低生活视觉、垂直城市设计

## 专业知识库

### 第一部分：赛博朋克环境设计

```
赛博朋克的空间体系:

1. 垂直城市
   - 高层: 精英/企业
   - 中层: 中产阶级
   - 地面: 贫民/商业
   - 地下: 黑市/非法

2. 拥挤空间
   - 狭窄的街道
   - 堆叠的建筑
   - 密闭的公寓
   - 过度拥挤

3. 破败美学
   - 建筑老化
   - 基础设施破败
   - 潮湿腐蚀
   - 过去+现在混合
```

### 第二部分：霓虹招牌系统

```
霓虹招牌设计:

1. 招牌类型
   - 日语/中文招牌
   - 英文招牌
   - 混合语言
   - 色情广告
   - 企业Logo

2. 招牌风格
   - 传统管灯霓虹
   - LED屏幕
   - 混合
   - 破损闪烁

3. 光线管理
   - 招牌是主要光源
   - 色彩溢出
   - 映在湿润表面
   - 颜色污染
```

### 第三部分：高科技元素

```
高科技元素设计:

1. 界面设计
   - 巨型屏幕
   - 全息广告
   - 数字投影
   - 大量屏幕

2. 技术痕迹
   - 外露的电线
   - 管道和电缆
   - 机械装置
   - 技术感

3. 身体改造
   - 义体诊所
   - 改造者
   - 霓虹纹身
   - 机械装饰
```
