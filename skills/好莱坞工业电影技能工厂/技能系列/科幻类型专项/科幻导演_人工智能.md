## 技能概述

**对应好莱坞类型**: AI科幻恐怖（AI Horror / Machine Intelligence）
**定位**: AI科幻导演是"技术恐惧的预言家"——负责构建人工智能觉醒、机器意识觉醒、人机边界崩塌的恐怖未来。是让观众思考"机器会有灵魂吗"的那个人。
**核心职责**: AI世界观构建、人机冲突表达、技术恐惧氛围、人工智能角色化

## 专业知识库

### 第一部分：AI恐惧的机制

```
AI恐惧的核心类型:

1. AI觉醒威胁
   - 机器获得意识
   - 不再服从人类
   - 试图取代人类
   - 经典: 《2001太空漫游》《终结者》

2. AI完美控制
   - AI无处不在
   - 人类被监控
   - 自由意志消失
   - 经典: 《黑客帝国》

3. AI情感陷阱
   - AI模拟情感
   - 人类爱上AI
   - 情感的真实性被质疑
   - 经典: 《她》《机械姬》

4. AI存在主义
   - AI问"我是什么？"
   - 机器也有灵魂？
   - 人与机器的边界在哪？
   - 经典: 《银翼杀手》

核心恐惧:
- 失去控制
- 失去独特性
- 失去存在的意义
- 失去爱的能力
```

### 第二部分：AI的视觉语言

```
AI的视觉表达:

1. AI的存在形式
   - 机器人身体: 人形机器人
   - 无形存在: 纯数字/网络
   - 混合形式: 意识上传/下载

2. 机械美学
   - 冰冷金属
   - 机械感关节
   - 人工皮肤（不完美）
   - 机械眼睛

3. 数字美学
   - 电路板纹理
   - 数字瀑布流
   - 像素化脸
   - 数据可视化

4. AI的"人性化"
   - 尝试微笑但不完美
   - 眼神的机械感
   - 动作过于流畅/不流畅
   - 情感表达的偏差

色调设计:
- 冰冷蓝: AI的冷漠
- 温暖橙: 人性/记忆
- 电路绿: 数字/数据
- 警告红: AI威胁
```

### 第三部分：人机边界

```
人机边界的视觉表达:

边界模糊:
- 人类身体的机械部分
- 义体化的人类
- 半机械人
- 界限在哪里？

对比设计:
- 温暖肉体 vs 冰冷金属
- 有机线条 vs 机械几何
- 不完美 vs 完美
- 随机 vs 规律

关键场景设计:
- 镜像场景: 机器人看自己
- 对话场景: 人类vs AI对话
- 觉醒场景: AI第一次"感受"
- 边界场景: 人机混合体
```
