## 技能概述

**交叉维度**: 表演风格 × 卡萨维茨 × 粗粝真实 × 微表情
**定位**: 这是"卡萨维茨粗粝真实"的表演风格系统——约翰·卡萨维茨是"方法演技"的先驱，他的电影以"粗糙、真实、未经雕琢"闻名。卡萨维茨的微表情是"真实的肌肉抽搐"——不是表演出来的微表情，而是演员真正进入角色后，自然发生的肌肉抽搐和微表情。
**核心功能**: 提供卡萨维茨粗粝真实的表演系统、真实肌肉抽搐捕捉、粗糙美学、微表情的自然发生技巧、以及AI视频生成的粗粝真实提示词

## 第一部分：卡萨维茨粗粝真实的科学基础

### 什么是"卡萨维茨粗粝真实"？——卡萨维茨的表演哲学

```
【卡萨维茨的表演哲学】

"摄影机记录的是真实发生的那一刻，不是假装发生的那一刻。"

卡萨维茨粗粝真实的核心理念：
- 真实：不是"表演"情绪，而是让情绪"发生"
- 粗糙：接受所有的不完美
- 自然：让演员真正进入角色，而不是表演角色

【卡萨维茨的著名真实场景】

场景1：亲密——卡罗尔的"真实崩溃"
- 卡萨维茨：卡罗尔在多次拍摄后真正的情绪崩溃
- 微表情：真实的肌肉抽搐、眼泪、声音破裂
- 风格：真实 vs 表演

场景2：受威胁的人——吉娜的"真实对话"
- 卡萨维茨：吉娜和罗伯特之间的真实张力
- 微表情：真实的紧张、真实的不安
- 风格：真实的人际关系

场景3：面孔——查尔斯斯的"私人时刻"
- 卡萨维茨：查尔斯斯在镜头前的私人时刻
- 微表情：真实的孤独、真实的不安全感
- 风格：角色的私人时刻

场景4：夫君J的四个月——的"真实生活"
- 卡萨维茨：一家人的真实生活片段
- 微表情：真实的家庭互动
- 风格：生活的真实切片

【粗粝真实的肌肉层面】

真实 vs 表演：
- 表演：肌肉按照"正确"的方式运动
- 真实：肌肉自然抽搐，不按照"正确"的方式

卡萨维茨的微表情特点：
- 不完美：接受所有的不完美
- 抽搐：肌肉的自然抽搐
- 破裂：微表情的突然破裂
```

### 卡萨维茨微表情的"自然发生"系统

```
【卡萨维茨微表情的"自然发生"系统】

自然发生 = 让情绪自己发生

三层：

Layer 1：准备
- 卡萨维茨：让演员长时间待在角色中
- 目的：让情绪自然浮现

Layer 2：捕捉
- 卡萨维茨：用摄影机捕捉情绪真正发生的瞬间
- 目的：真实的微表情

Layer 3：接受
- 卡萨维茨：接受所有的不完美
- 目的：粗糙美学

【卡萨维茨的微表情编码】

编码1：自然抽搐
- 肌肉：自然抽搐，不是"表演"
- 特点：不完美

编码2：突然破裂
- 微表情：突然破裂，不是"计划"
- 特点：意外

编码3：声音
- 声音：真实的声音破裂
- 特点：真实
```
