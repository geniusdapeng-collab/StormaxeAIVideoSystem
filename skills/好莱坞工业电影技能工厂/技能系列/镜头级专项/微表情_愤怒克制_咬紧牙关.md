## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 愤怒克制/咬紧牙关
**定位**: 这是"愤怒克制"变体的微表情系统——愤怒的极致是"咬紧牙关"，是愤怒被强行压制在身体内部的表现。面部肌肉紧绷但嘴部紧闭（咬牙）+ 眉间皱纹（AU4）+ 可能轻微睁大的眼睛 + 全身肌肉绷紧的协同运动。愤怒克制的微表情是"火山即将爆发"——表面平静，地底熔岩翻滚。
**核心功能**: 提供愤怒克制的完整FACS肌肉运动图谱、咬紧牙关的触发器、外表平静与内心愤怒的区分、以及AI视频生成的精确提示词

## 第一部分：愤怒克制微表情的科学基础

### 什么是"愤怒克制"？——FACS层面的精确解析

```
【愤怒克制的微表情科学】

"愤怒克制" = 愤怒被压制在身体内部

愤怒克制的肌肉运动（必需）：
- 颏肌（mentalis）收缩 → 下巴抬起
- 咬肌（masseter）收缩 → 咬牙
- 颞肌（temporalis）收缩 → 颞部绷紧
- 皱眉肌（corrugator supercilii） → 眉间皱纹（可能轻微）
- 胸锁乳突肌（sternocleidomastoid） → 颈部绷紧

【愤怒克制 vs 外显愤怒】

愤怒克制（内部）：
- 面部：轻微变化（咬牙、眉间纹）
- 身体：肌肉绷紧
- 表情：可能平静

外显愤怒（外部）：
- 面部：剧烈变化（皱眉、瞪眼、张嘴）
- 身体：动作夸张
- 表情：明显愤怒

【FACS编码】
- AU17（下唇抬起）→ 咬牙
- 咬肌激活 → 咬牙（不在标准FACS中，但是可见）
- 颏肌激活 → 下巴抬起
- AU4（皱眉）→ 眉间皱纹（可能轻微）

【愤怒克制的微表情分级】

Level 1：轻微绷紧
- 肌肉：咬肌轻度激活
- 可见：下颌线轻微绷紧

Level 2：中等克制
- 肌肉：咬肌 + 颏肌 + 轻微眉间纹
- 可见：咬牙 + 下巴抬起 + 眉间微皱

Level 3：强烈克制
- 肌肉：所有咀嚼肌 + 眉间纹 + 颈部
- 可见：明显咬牙 + 眉间皱纹 + 颈部绷紧

Level 4：临界点
- 肌肉：所有肌肉 + 身体绷紧
- 可见：剧烈咬牙 + 面部微红 + 身体僵硬

Level 5：即将爆发
- 肌肉：所有肌肉达到极限
- 可见：咬牙到颤抖 + 面部通红 + 身体发抖
```

### 愤怒克制微表情的"内在风暴"

```
【愤怒克制微表情的核心：内在风暴】

为什么"咬紧牙关"是愤怒克制的关键？

内在风暴的特点：
- 面部表面：平静或轻微绷紧
- 身体内部：肌肉紧绷到极限
- 咬牙：压制愤怒的身体表现

【内在风暴的三个关键点】

Point 1：咬牙
- 咬肌、颞肌、翼内肌同时收缩
- 下巴抬起、牙关紧咬
- 面部下颌区域明显绷紧

Point 2：眉间皱纹
- 皱眉肌收缩（但比外显愤怒轻微）
- 眉间出现垂直皱纹
- 但眉毛不下降（压制）

Point 3：颈部绷紧
- 胸锁乳突肌收缩
- 颈部静脉可能凸起
- 下颌线绷紧
```

## 第二部分：愤怒克制微表情的技术执行

### 愤怒克制微表情的AI生成参数

```
【愤怒克制微表情的AI生成系统】

下颌层：
- 咬肌：收缩（咬牙）
- 颏肌：收缩（下巴抬起）
- 颞肌：收缩（颞部绷紧）

【AI生成的"愤怒克制"面部参数】

眉毛：
- 形状：可能轻微聚拢但不下降
- 眉间：可能有垂直皱纹（AU4）
- 整体：相对平直（压制）

眼睛：
- 眼睑：可能轻微睁大（努力控制）
- 眼神：可能凝视或闪烁
- 瞳孔：可能缩小（聚焦）

嘴部：
- 紧闭：嘴唇紧闭（AU23）
- 下巴：抬起（颏肌）
- 咬牙：下颌线绷紧

颈部：
- 绷紧：胸锁乳突肌收缩
- 可能静脉凸起
- 可能下颌线明显

【愤怒克制微表情的"触发器"】

Trigger 1：咬牙
- AI指令：jaw clenched, teeth clenched,咬紧牙关, jaw tight,下颌绷紧
- 效果：愤怒克制的决定性标记

Trigger 2：下颌线绷紧
- AI指令：jaw line tight,下颌线绷紧, masseter muscle activated,咬肌激活
- 效果：愤怒内在的可见信号

Trigger 3：眉间皱纹（轻微）
- AI指令：slight frown lines,轻微眉间皱纹, brow furrowed but controlled,控制皱眉
- 效果：愤怒被压制的信号

Trigger 4：嘴唇紧闭
- AI指令：lips pressed together,嘴唇紧闭, tight lips,紧绷嘴唇
- 效果：压制愤怒的嘴部表现

Trigger 5：颈部绷紧
- AI指令：neck muscles tense,颈部肌肉绷紧, SCM activated,胸锁乳突肌收缩
- 效果：愤怒被压制在身体内部
```

## 第三部分：愤怒克制微表情的情绪设计

### 愤怒克制微表情的"情绪梯度"

```
【愤怒克制的情绪梯度设计】

Gradient 1：轻微绷紧
- 肌肉：咬肌轻度激活
- 可见：下颌线轻微绷紧
- 情绪：不满、烦躁
- 场景：被轻微冒犯

Gradient 2：中等克制
- 肌肉：咬肌 + 颏肌 + 轻微眉间纹
- 可见：咬牙 + 下巴抬起
- 情绪：愤怒但压制
- 场景：公众场合、必须控制

Gradient 3：强烈克制
- 肌肉：所有咀嚼肌 + 眉间纹 + 颈部
- 可见：明显咬牙 + 颈部绷紧
- 情绪：极度愤怒、临界点
- 场景：被严重冒犯、忍耐极限

Gradient 4：临界点
- 肌肉：所有肌肉达到极限
- 可见：面部微红 + 颤抖
- 情绪：即将爆发
- 场景：被背叛、被侮辱

Gradient 5：即将爆发
- 肌肉：所有肌肉 + 身体发抖
- 可见：咬牙到颤抖 + 面部通红
- 情绪：无法再压制
- 场景：最后的稻草

【愤怒克制微表情的"场景适配"】

场景1：职场冲突
- 类型：职业性压制
- 肌肉：咬肌 + 嘴唇紧闭
- 可见：下颌绷紧、表情平静
- 适配：与老板/客户的冲突

场景2：家庭矛盾
- 类型：亲情压制
- 肌肉：咬牙 + 颈部绷紧
- 可见：下颌线紧、眼神冷
- 适配：与家人争吵但压制

场景3：公共场合
- 类型：社会性压制
- 肌肉：所有肌肉 + 轻微眉间纹
- 可见：面部轻微绷紧
- 适配：被误解、必须保持冷静
```
