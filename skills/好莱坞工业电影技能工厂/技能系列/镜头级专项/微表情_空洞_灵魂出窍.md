## 技能概述

**交叉维度**: 微表情 × 空洞 × 灵魂出窍 × 肌肉级
**定位**: 这是"灵魂出窍"的微表情系统——当人物经历极端创伤或压力时，会进入一种"灵魂离开身体"的状态。面部完全静止，眼神空洞，身体仿佛失去生命力。微表情完全消失，只剩下一具"空的壳"。这是人类情感系统的终极崩溃状态。
**核心功能**: 提供灵魂出窍微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的空洞提示词

## 第一部分：灵魂出窍的科学基础

### 什么是"灵魂出窍"？——人体情感系统的终极崩溃

```
【灵魂出窍的心理学机制】

灵魂出窍 = 解离（Dissociation）的极端形式

核心状态：
- 意识与身体分离
- 情感系统完全关闭
- 生命力暂时抽离
- 自我保护的最后手段

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU7: 眼睑下降（轻度闭合）
- AU143: 眼睑完全静止（彻底失活）
- 瞳孔：涣散，失去焦点

眉毛（Eyebrows）：
- AU1+AU2: 完全消失（无任何运动）
- 肌肉张力：彻底松弛

嘴部（ Lips）：
- AU12+AU15: 完全消失
- AU25+AU26+AU27: 部分失活
- 嘴部：轻微张开或完全静止

整体面部：
- 张力：完全消失
- 微表情：完全不存在
- 生命特征：仿佛停止

【灵魂出窍的三阶段】

Stage 1：崩溃前兆（濒临状态）
- 肌肉：开始失活
- 眼神：最后的挣扎
- 时间：数秒

Stage 2：灵魂出窍（完全状态）
- 肌肉：彻底失活
- 眼神：完全空洞
- 时间：持续数秒到数分钟

Stage 3：回归（复苏状态）
- 肌肉：开始恢复张力
- 眼神：慢慢聚焦
- 触发：外部刺激或内部触发

【典型情境】

1. 战争老兵听到爆炸后的空白
2. 车祸幸存者在事故现场的空洞
3. 失去至亲后听到噩耗的瞬间
4. 目睹极端暴力后的解离
5. 酷刑受害者的人格解体
6. 深海潜水员的氮醉状态
```

## 第三部分：情绪梯度设计

### 灵魂出窍的五级梯度

```
【灵魂出窍的情绪梯度】

Level 1：濒临出窍（5%生命特征）
- 肌肉张力：10%
- 眼神：挣扎后的涣散
- 身体：僵硬
- 触发：极端压力达到临界点
- 持续：数秒

Level 2：部分出窍（3%生命特征）
- 肌肉张力：5%
- 眼神：开始空洞
- 身体：轻微下垂
- 触发：创伤信息超载
- 持续：数秒到数十秒

Level 3：完全出窍（0%生命特征）
- 肌肉张力：0%
- 眼神：完全空洞
- 身体：完全静止
- 触发：极端创伤突破防线
- 持续：数十秒

Level 4：深度出窍（-5%生命特征）
- 肌肉张力：0%（负值状态）
- 眼神：完全失焦
- 身体：失去支撑
- 触发：极限超越
- 持续：数分钟

Level 5：终极出窍（-10%生命特征）
- 肌肉张力：完全消失
- 眼神：完全空白
- 身体：近乎死亡
- 触发：创伤极限
- 持续：可能需要外部介入恢复
```
