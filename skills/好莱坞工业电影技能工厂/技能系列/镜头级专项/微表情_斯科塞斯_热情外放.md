## 技能概述

**交叉维度**: 表演风格 × 斯科塞斯 × 热情外放 × 微表情
**定位**: 这是"斯科塞斯热情外放"的表演风格系统——马丁·斯科塞斯的电影以"意大利裔美国人的热情"著称，他的角色经常在极端情绪下爆发出强烈的、微表情丰富的表达。斯科塞斯的微表情是"情感的火山"——情绪不是被压制，而是被彻底地、完全地表达出来，从内心的深处喷涌而出。
**核心功能**: 提供斯科塞斯热情外放的表演系统、意大利式情感表达、情绪的彻底释放、微表情的丰富变化、以及AI视频生成的热情风格提示词

## 第一部分：斯科塞斯热情外放的科学基础

### 什么是"斯科塞斯热情外放"？——斯科塞斯的表演哲学

```
【斯科塞斯的表演哲学】

"情感不是要压制的东西，而是要拥抱的东西。"

斯科塞斯热情外放的核心：
- 外放：情感被完全表达，不压制
- 意大利式：地中海文化中的高情感表达
- 家庭：情感与家庭、荣誉紧密相连

【斯科塞斯的著名热情场景】

场景1：愤怒的公牛——杰克对弟弟的解释
- 斯科塞斯：杰克解释为什么弟弟不理解他
- 微表情：愤怒、悲伤、渴望、绝望，完全混合
- 风格：意大利式的情感爆发

场景2：出租车司机——特拉维斯的"政治演说"
- 斯科塞斯：特拉维斯对贝茨的政治演说
- 微表情：愤怒、挫败、渴望，完全外放
- 风格：孤独者的愤怒爆发

场景3：好伙伴——约翰尼对父亲的愤怒
- 斯科塞斯：约翰尼对父亲说"我不再为你工作了"
- 微表情：愤怒、解放、自由
- 风格：儿子对父亲的反叛

场景4：赌城风云——山姆的"女人演讲"
- 斯科塞斯：山姆解释为什么他爱阿姬
- 微表情：爱情、激情、绝望，完全外放
- 风格：男人的爱与痛苦

【意大利式情感的肌肉层面】

外放 = 情感完全表达

肌肉层面：
- 眉毛：强烈下降（愤怒）或提升（悲伤）
- 眼睛：表达所有情绪
- 嘴部：表达所有情绪
- 身体：参与表达
```

### 斯科塞斯微表情的"火山系统"

```
【斯科塞斯微表情的"火山系统"】

火山 = 情感从内部喷发

三层：

Layer 1：积累
- 情感：在内部积累
- 肌肉：准备爆发

Layer 2：爆发
- 情感：从内部喷发
- 微表情：完全外放

Layer 3：余韵
- 情感：爆发后的余韵
- 微表情：慢慢平息

【斯科塞斯的外放编码】

编码1：完全表达
- 微表情：所有情绪都完全表达
- 不是压制，是释放

编码2：意大利式手势
- 手势：与微表情同步
- 身体：参与表达

编码3：家族情感
- 情感：与家庭紧密相连
- 微表情：家族的情感
```

## 第三部分：斯科塞斯热情外放的情绪设计

### 斯科塞斯微表情的"外放情绪"

```
【斯科塞斯外放的情绪设计】

情绪1：愤怒（外放）
- 斯科塞斯：意大利式的愤怒爆发
- 微表情：完全表达，眉毛下降，嘴部张开
- 场景：家庭冲突

情绪2：悲伤（外放）
- 斯科塞斯：意大利式的悲伤表达
- 微表情：完全表达，眼泪、声音
- 场景：亲人去世

情绪3：爱（外放）
- 斯科塞斯：意大利式的爱
- 微表情：完全表达，温柔但强烈
- 场景：爱情表白

情绪4：骄傲（外放）
- 斯科塞斯：意大利式的骄傲
- 微表情：完全表达，挺胸抬头
- 场景：成功时刻

【斯科塞斯经典的"积累-爆发"结构】

结构：积累→爆发→余韵

积累：
- 情感：在内部积累
- 角色：压制，但越来越难

爆发：
- 情感：从内部喷发
- 角色：完全表达
- 微表情：丰富变化

余韵：
- 情感：平息
- 角色：疲惫但释放
- 微表情：慢慢平静
```
