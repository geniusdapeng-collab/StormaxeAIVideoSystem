## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 幸福/喜悦
**定位**: 这是"幸福"情绪的微表情系统——基于FACS（面部动作编码系统），精确到每一块面部肌肉的激活。幸福的微表情不是"笑"，是"杜兴微笑"的精确肌肉运动：眼轮匝肌（orbicularis oculi）的收缩 + 颧大肌（zygomaticus major）的提升，让眼睛变成月牙形，让鱼尾纹出现，让"真笑"和"假笑"在肌肉层面产生可区分的差异。
**核心功能**: 提供幸福的完整FACS肌肉运动图谱、眼眶周围微表情触发器、真笑与假笑的肌肉层面区分、以及AI视频生成的精确提示词

## 第一部分：幸福微表情的科学基础

### 什么是"真笑"？——FACS层面的精确解析

```
【幸福的微表情科学】

"幸福" = 杜兴微笑（Duchenne Smile）

杜兴微笑的肌肉运动（必需）：
- 颧大肌（zygomaticus major）提升 → 嘴角上扬
- 眼轮匝肌（orbicularis oculi）收缩 → 眼睛变成月牙形（眼尾皱纹）
- 眼轮匝肌（orbicularis oculi）的收缩是"真笑"和"假笑"的唯一区分

【FACS编码】
- AU6（眼轮匝肌内侧收缩）→ 眉毛下压，眼角褶皱
- AU12（颧大肌收缩）→ 嘴角向上向后拉动
- AU25（嘴唇分开）→ 允许露出牙齿

【"真笑"和"假笑"的肌肉层面区分】

真笑（杜兴微笑）：
- 颧大肌 + 眼轮匝肌 → 眼睛变小（眉毛下压）→ 鱼尾纹出现
- 肌肉协同：同时激活，不是顺序激活
- 时间：自然过渡，不会突然出现

假笑（社交微笑）：
- 只有颧大肌 → 嘴角上扬，眼睛不变
- 肌肉协同：只有嘴角，眉毛和眼睛不动
- 时间：可以突然开始突然结束

【幸福的微表情分级】

Level 1：微微笑（嘴角微动）
- FACS：AU12轻度激活
- 肌肉：只有颧大肌
- 可见：嘴角上扬1-2mm

Level 2：轻笑（嘴角+轻微眼动）
- FACS：AU12 + AU6轻度激活
- 肌肉：颧大肌 + 眼轮匝肌内侧
- 可见：嘴角上扬 + 眼尾微皱

Level 3：微笑（杜兴微笑）
- FACS：AU12 + AU6完全激活
- 肌肉：颧大肌 + 眼轮匝肌完全收缩
- 可见：标准月牙眼 + 鱼尾纹 + 脸颊提升

Level 4：大笑（完全激活）
- FACS：AU12 + AU6 + AU25
- 肌肉：全部微笑肌肉 + 嘴唇分开
- 可见：露出上牙 + 脸颊提升 + 眼周皱纹

Level 5：狂喜（极强激活）
- FACS：AUC（无法编码）= 完全微笑
- 肌肉：所有微笑肌肉 + 眉毛提升
- 可见：眉毛提升 + 完全杜兴 + 可能流泪
```

### 幸福微表情的"眼眶三角区"

```
【幸福微表情的核心：眼眶三角区】

为什么眼眶是幸福的关键？

眼眶周围（眼轮匝肌区域）：
- 眼轮匝肌是"不自主"的——无法假装收缩
- 当眼轮匝肌收缩，眉毛会被向下拉动
- 眉毛下压 + 眼尾褶皱 = "眼角的皱纹" = 鱼尾纹

【眼眶三角区的三个关键点】

Point 1：眼角（眼角皱纹/鱼尾纹）
- 眼轮匝肌收缩 → 眼角出现斜向皱纹
- 皱纹从眼角向外辐射，45度角
- 真笑的皱纹是双侧的、对称的

Point 2：眉毛（眉毛下压）
- 眼轮匝肌收缩 → 眉毛内侧被向下拉动
- 眉毛呈倒八字形（倒V）
- 假笑时眉毛不下压

Point 3：脸颊（脸颊提升）
- 颧大肌收缩 → 脸颊提升
- 脸颊提升会让下眼睑的皮肤鼓起
- 会在眼睛下方形成轻微的"卧蚕"
```

## 第二部分：幸福微表情的技术执行

### 幸福微表情的AI生成参数

```
【幸福微表情的AI生成系统】

骨骼肌肉层：
- 颧大肌：激活，嘴角向上向后拉动
- 眼轮匝肌：收缩，眼尾皱纹出现
- 提口角肌：轻度激活，辅助嘴角上扬
- 颏肌：轻度激活，嘴唇微微分开

【AI生成的"幸福"面部参数】

眉毛：
- 内侧：向下压（眼轮匝肌收缩）
- 外侧：保持位置或微升
- 整体：呈倒八字形
- 错误：平直或向上（那是惊讶）

眼睛：
- 眼睑：下眼睑提升（被脸颊推起）
- 眼角：鱼尾纹（眼轮匝肌收缩）
- 瞳孔：正常或轻微放大（愉悦时）
- 错误：睁大（那是惊讶）

鼻部：
- 鼻翼：轻微扩张（愉快时）
- 错误：收缩（那是厌恶）

嘴部：
- 嘴角：向上向后拉动（颧大肌）
- 嘴唇：自然分开或紧闭
- 牙齿：可能露出（上唇提升）
- 错误：向下（那是悲伤）

脸颊：
- 提升：明显提升
- 皱纹：鼻唇沟加深
- 错误：无提升（那是假笑）

【幸福微表情的"触发器"】

Trigger 1：眼轮匝肌收缩
- AI指令：orbicularis oculi activated, crow's feet, eye corners wrinkled
- 效果：真笑 vs 假笑的决定性差异

Trigger 2：眉毛倒八字
- AI指令：inner eyebrows depressed, inverted V shape eyebrows
- 效果：真笑的眉毛状态

Trigger 3：脸颊提升
- AI指令：cheeks raised, cheekbones prominent, under-eye tissue bulging
- 效果：丰满的"苹果肌"效果

Trigger 4：嘴角上扬
- AI指令：mouth corners pulled up and back, zygomaticus major activated
- 效果：标准微笑

Trigger 5：自然过渡
- AI指令：smile emerges naturally, muscles activate simultaneously
- 效果：避免"突然出现"的假笑感
```

## 第三部分：幸福微表情的情绪设计

### 幸福微表情的"情绪梯度"

```
【幸福的情绪梯度设计】

Gradient 1：满足（轻度幸福）
- 肌肉：AU12轻度激活
- 表情：嘴角微上扬，眼睛正常
- 情绪：平静、满足、放松
- 场景：安静的下午、读完一本好书

Gradient 2：愉悦（中等幸福）
- 肌肉：AU12 + AU6轻度激活
- 表情：嘴角上扬 + 眼尾微皱
- 情绪：愉快、温暖、舒适
- 场景：见到老朋友、吃到美食

Gradient 3：快乐（标准幸福）
- 肌肉：AU12 + AU6完全激活
- 表情：标准杜兴微笑
- 情绪：开心、欢乐、幸福
- 场景：收到礼物、实现目标

Gradient 4：欣喜（强烈幸福）
- 肌肉：AUC（完全激活）+ AU25
- 表情：大范围微笑 + 露出牙齿
- 情绪：狂喜、兴奋、激动
- 场景：婚礼、获奖、意外惊喜

Gradient 5：狂喜（极强幸福）
- 肌肉：AUC + 所有辅助肌肉
- 表情：完全杜兴 + 眉毛提升 + 可能流泪
- 情绪：极乐、忘我、幸福到哭
- 场景：久别重逢、梦想成真

【幸福微表情的"场景适配"】

场景1：浪漫时刻
- 幸福类型：温柔的微笑
- 肌肉：AU12 + AU6轻度（不过于夸张）
- 眼神：眼神温柔，注视对方
- 适配：接吻前、凝视时刻

场景2：胜利时刻
- 幸福类型：自信的笑
- 肌肉：AU12 + AU6完全 + 眉毛微升
- 眼神：眼神坚定，略微俯视
- 适配：获奖、击败对手

场景3：家庭时刻
- 幸福类型：温暖的笑
- 肌肉：AU12 + AU6完全
- 眼神：眼神柔和，看向家人
- 适配：家庭团聚、孩子出生

场景4：独处时刻
- 幸福类型：微微笑
- 肌肉：AU12轻度
- 眼神：眼神向内，回忆美好
- 适配：回忆、感恩、独处
```
