## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 惊讶凝固/时间停止
**定位**: 这是"惊讶凝固"变体的微表情系统——惊讶的极致是"时间停止"，是极度震惊导致整个世界都慢下来的感觉。眼睛睁到最大（AU5）+ 嘴巴张开（AU26）+ 全身僵住 + 可能周围环境"慢动作"的协同表达。惊讶凝固的微表情是"意识的瞬间空白"——大脑在那一瞬间停止了所有处理。
**核心功能**: 提供惊讶凝固的完整FACS肌肉运动图谱、时间停止感的触发器、震惊与普通惊讶的区分、以及AI视频生成的精确提示词

## 第一部分：惊讶凝固微表情的科学基础

### 什么是"惊讶凝固"？——FACS层面的精确解析

```
【惊讶凝固的微表情科学】

"惊讶凝固" = 意识瞬间空白

惊讶凝固的肌肉运动（必需）：
- 额肌（frontalis）→ 整个眉毛向上（AU1+AU2）
- 提上睑肌（levator palpebrae superioris）→ 眼睛睁到最大（AU5）
- 下颌肌（masseter）→ 嘴巴张开（AU26）
- 全身：僵住、不动

【惊讶凝固 vs 普通惊讶】

惊讶凝固（极度）：
- 程度：最大睁眼、最大张嘴
- 持续：可能持续更长时间
- 身体：完全僵住

普通惊讶（正常）：
- 程度：正常睁眼、正常张嘴
- 持续：短暂后恢复
- 身体：可能有轻微动作

【FACS编码】
- AU1+AU2（眉毛）→ 整个眉毛向上
- AU5（眼睑）→ 上眼睑大幅提升
- AU26（下颌）→ 下颌下降，嘴巴大张

【惊讶凝固的微表情分级】

Level 1：极度震惊
- 肌肉：AU1+AU2+AU5+AU26最大激活
- 可见：眼睛和嘴巴完全张开
- 身体：完全僵住

Level 2：时间停止
- 肌肉：最大激活 + 环境慢动作
- 可见：周围事物也仿佛停止
- 身体：完全静止

Level 3：意识空白
- 肌肉：所有惊讶肌肉
- 可见：眼神空洞
- 身体：可能轻微摇晃

Level 4：完全僵住
- 肌肉：所有惊讶肌肉 + 身体僵硬
- 可见：全身不动
- 身体：僵硬

Level 5：灵魂出窍
- 肌肉：所有惊讶肌肉
- 可见：仿佛灵魂离开身体
- 身体：完全失去反应
```

### 惊讶凝固微表情的"时间停止"

```
【惊讶凝固微表情的核心：时间停止】

为什么"时间停止"是惊讶凝固的关键？

时间停止的特点：
- 大脑：在那一瞬间停止处理
- 身体：完全僵住
- 感知：世界仿佛慢下来

【时间停止的三个关键点】

Point 1：眼睛睁到最大
- 提上睑肌最大激活
- 上眼睑提升到极限
- 巩膜大面积暴露

Point 2：嘴巴大张
- 下颌下降到极限
- 嘴巴完全张开
- 可能呈现O形

Point 3：全身僵住
- 所有肌肉冻结
- 身体失去反应
- 仿佛时间停止
```
