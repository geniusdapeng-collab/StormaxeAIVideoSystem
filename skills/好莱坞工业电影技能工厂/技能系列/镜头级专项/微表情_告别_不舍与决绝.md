**交叉维度**: 微表情 × 告别 × 不舍与决绝 × 肌肉级
**定位**: 这是"告别"的微表情系统——当两个人在分离的瞬间，面部同时承载着不舍与决绝的对抗性情绪。一只手可能还握着对方，另一只手已经开始推开。眼眶可能泛着泪光，但眼神已经开始看向远方。这是电影中最具情感张力的时刻之一，每一个面部肌肉的细微变化都在述说着"我想留下，但我必须走"。
**核心功能**: 提供告别时刻的肌肉级科学解析、FACS编码、情绪对抗技巧、以及AI视频生成的告别提示词

## 第一部分：告别的科学基础

### 什么是"不舍与决绝"？——告别时刻的情感神经机制

```
【告别时刻的心理学机制】

告别时刻 = 不舍（想要留下）+ 决绝（必须离开）+ 对抗性共存

核心状态：
- 身体两侧的不一致（拉 vs 推）
- 眼眶与眼神的对抗（泪 vs 看远方）
- 嘴角与眉毛的对抗（想笑 vs 想哭）
- 内在撕裂与外在平静的对比

【FACS肌肉级解析】

眼睑（Eyelids）：
- 眼睑：轻微下垂（试图掩盖泪光）
- 眨眼频率：加快（情绪激动）
- 眼轮匝肌：轻微收缩（抑制眼泪）

眉毛（Eyebrows）：
- AU1: 内侧提升（悲伤/不舍）
- AU4: 下降（努力控制）
- 不对称：一边更高（倔强/不服输）

眼眶（Orbital）：
- 眼眶轮匝肌：轻微收缩（抑制哭泣）
- 内侧眉毛抬起+外侧下降 = "痛苦但不说"

嘴角（Lips）：
- AU12+15: 一边上扬+一边下沉（苦笑/对抗）
- AU24: 嘴唇抿紧（压抑情绪）
- 不对称嘴角：一半想笑，一半想哭

整体：
- 眼神：看向远方（决绝）+ 回看对方（不舍）
- 头部：轻微转向远方（离开）+ 轻微转回（留恋）

【告别时刻的两型】

Type 1：不舍主导型
- 特点：悲伤更明显，但努力控制
- 表现：眼含泪光、嘴角颤抖、声音哽咽
- 场景：与爱人告别、与亲人告别

Type 2：决绝主导型
- 特点：外表平静，内心翻涌
- 表现：眼神坚定、嘴角紧抿、声音平稳
- 场景：战士出征、英雄离别

【典型情境】

1. 火车站月台告别
2. 机场安检口告别
3. 医院病房告别
4. 战场上的诀别
5. 深夜门口的最后一面
6. 雨中转身离去
7. 不回头的告别
8. 强忍泪水的告别
```

## 第三部分：告别类型设计

### 告别时刻的五型设计

```
【告别时刻的五型设计】

Type 1：恋人告别
- 特点：最深的不舍，最痛苦的决绝
- 情绪：心碎+希望+绝望混合
- 场景：异地恋/战争/死亡
- 代表：《英国病人》的最终告别

Type 2：亲子告别
- 特点：最本能的不舍，最无奈的决绝
- 情绪：心碎+责任+爱
- 场景：送孩子远行/父母去世
- 代表：《人生大事》的父女告别

Type 3：战友告别
- 特点：最克制的告别，最深的情感
- 情绪：战友情+使命+牺牲
- 场景：战场/葬礼/出发前
- 代表：《拯救大兵瑞恩》的最后告别

Type 4：自我告别
- 特点：与过去的自己告别
- 情绪：释然+不舍+决心
- 场景：人生转折点/决定改变
- 代表：《搏击俱乐部》的觉醒时刻

Type 5：永别
- 特点：知道再也不会见面
- 情绪：绝望+接受+爱
- 场景：临终/消失/离开
- 代表：《泰坦尼克号》的诀别

【五级告别强度设计】

Level 1：短暂分离（10%强度）
- 特点：明天就见面
- 微表情：轻微不舍
- 氛围：轻松但有留恋

Level 2：短期分离（30%强度）
- 特点：几周或几个月
- 微表情：明显不舍
- 氛围：情绪明显但不失控

Level 3：长期分离（60%强度）
- 特点：几年或更久
- 微表情：强烈不舍+努力控制
- 氛围：情绪激烈但压抑

Level 4：永久分离（85%强度）
- 特点：知道再也不会见面
- 微表情：极度不舍+完全控制
- 氛围：心碎但平静

Level 5：生死分离（100%强度）
- 特点：死亡或消失
- 微表情：绝望+接受+爱
- 氛围：超越悲伤，化为永恒
```
