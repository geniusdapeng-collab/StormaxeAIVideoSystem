## 技能概述

**交叉维度**: 微表情 × 战栗 × 不祥预感 × 肌肉级
**定位**: 这是"不祥预感"的微表情系统——当人物感知到即将发生的危险、厄运或不幸时，会产生一种本能的战栗反应。这种战栗从脊柱开始，蔓延到全身，伴随着皮肤的鸡皮疙瘩和眉毛的急剧收缩。这是一种原始的、本能的情绪反应，跨越文化和物种。
**核心功能**: 提供不祥预感微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的战栗提示词

## 第一部分：不祥预感的科学基础

### 什么是"不祥预感"？——原始本能的预警系统

```
【不祥预感的心理学机制】

不祥预感 = 本能预警系统（Primitive Warning System）

核心状态：
- 本能感知到危险
- 身体进入预警状态
- 战栗作为信号
- 生存本能的激活

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU1+AU2: 急剧提升（恐惧反应）
- AU4+AU5+AU6+AU7: 眉毛急剧提升+眼睑提升+眼睛张大
- 瞳孔：急剧放大（肾上腺素反应）
- 编码：[AU1+AU2+AU4+AU5+AU6+AU7]

眉毛（Eyebrows）：
- AU1: 内侧提升（恐惧的独特标志）
- AU2: 外侧提升（惊奇）
- AU4: 聚拢（担忧/危险感知）
- 组合：急剧的AU1+AU2+AU4组合

嘴部（Lips）：
- AU26: 下巴下降（轻微张口，战栗反应）
- AU25: 嘴唇拉紧（紧绷的嘴唇）
- 组合：张开的嘴+紧绷的嘴唇=不祥预感

全身（Body）：
- 战栗（Shiver）：从脊柱开始的身体震颤
- 鸡皮疙瘩（Piloerection）：皮肤起疙瘩
- 呼吸：变浅变快

【战栗的三阶段】

Stage 1：感知（直觉阶段）
- 时间：0.1-0.5秒
- 身体：开始警觉
- 微表情：眉毛急剧变化

Stage 2：战栗（本体阶段）
- 时间：0.5-2秒
- 身体：战栗开始
- 微表情：全身反应

Stage 3：应对（决策阶段）
- 时间：2-5秒
- 身体：战斗/逃跑准备
- 微表情：根据决策变化

【典型情境】

1. 听到黑暗中传来的不明声音
2. 预感亲人出事前的本能感知
3. 危险逼近前的直觉
4. 进入废弃建筑时的不安
5. 预感坏消息即将到来
6. 暴风雨来临前的自然反应
7. 看见危险生物时的本能反应
```

## 第三部分：情绪梯度设计

### 不祥预感的五级梯度

```
【不祥预感的情绪梯度】

Level 1：轻微警觉（10%强度）
- 眉毛：轻微提升
- 身体：几乎无变化
- 鸡皮疙瘩：无
- 触发：模糊的不安感
- 持续：1-3秒

Level 2：明显不安（30%强度）
- 眉毛：明显提升+聚拢
- 身体：轻微战栗
- 鸡皮疙瘩：轻微
- 触发：感知到某种危险
- 持续：3-8秒

Level 3：强烈预警（60%强度）
- 眉毛：急剧提升+聚拢
- 眼睛：张开
- 身体：明显战栗
- 鸡皮疙瘩：明显
- 触发：明确感知危险
- 持续：5-15秒

Level 4：极度恐惧（85%强度）
- 眉毛：最大提升
- 眼睛：完全睁开
- 身体：剧烈战栗
- 鸡皮疙瘩：强烈
- 触发：危险迫近
- 持续：10-30秒

Level 5：本能崩溃（100%强度）
- 眉毛：超出正常范围
- 眼睛：最大化
- 身体：无法控制的颤抖
- 鸡皮疙瘩：遍布全身
- 触发：极端危险
- 持续：直到危险解除或崩溃
```
