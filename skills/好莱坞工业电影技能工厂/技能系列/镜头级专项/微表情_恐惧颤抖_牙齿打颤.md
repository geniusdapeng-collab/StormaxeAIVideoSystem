## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 恐惧颤抖/牙齿打颤
**定位**: 这是"恐惧颤抖"变体的微表情系统——恐惧的极致是"牙齿打颤"，是身体在极端恐惧下失去控制的本能反应。眼睛睁大（AU5）+ 嘴唇颤抖 + 牙齿打颤 + 全身颤抖的协同运动。恐惧颤抖的微表情是"身体的背叛"——意志无法控制肉体，所有本能都在尖叫着逃跑。
**核心功能**: 提供恐惧颤抖的完整FACS肌肉运动图谱、牙齿打颤的触发器、身体失控与意志挣扎的区分、以及AI视频生成的精确提示词

## 第一部分：恐惧颤抖微表情的科学基础

### 什么是"恐惧颤抖"？——FACS层面的精确解析

```
【恐惧颤抖的微表情科学】

"恐惧颤抖" = 身体失控的本能反应

恐惧颤抖的肌肉运动（必需）：
- 提上睑肌（levator palpebrae superioris）→ 眼睛睁大（AU5）
- 额肌（frontalis） → 眉毛提升（AU1+AU2）
- 口轮匝肌（orbicularis oris） → 嘴唇颤抖
- 咀嚼肌：颞肌、咬肌 → 牙齿打颤
- 全身肌肉：不自主颤抖

【恐惧颤抖 vs 普通恐惧】

恐惧颤抖（极端）：
- 身体：失控颤抖、牙齿打颤
- 面部：无法控制的抖动
- 意识：可能清醒或混乱

普通恐惧（正常）：
- 身体：僵硬或逃跑反应
- 面部：可控的表情变化
- 意识：清醒、警觉

【FACS编码】
- AU1+AU2（眉毛）→ 提升
- AU5（眼睑）→ 睁大
- 口轮匝肌颤抖 → 嘴唇颤抖
- 咀嚼肌颤抖 → 牙齿打颤

【恐惧颤抖的微表情分级】

Level 1：紧张
- 肌肉：全身轻微绷紧
- 可见：肌肉轻微颤抖
- 牙齿：正常

Level 2：发抖
- 肌肉：全身中等颤抖
- 可见：四肢发抖
- 牙齿：可能轻微磕碰

Level 3：颤抖
- 肌肉：全身明显颤抖
- 可见：无法控制的抖动
- 牙齿：明显打颤

Level 4：剧烈颤抖
- 肌肉：全身剧烈抖动
- 可见：站立困难、牙齿剧烈磕碰
- 牙齿：剧烈打颤

Level 5：崩溃（极度恐惧）
- 肌肉：完全失控
- 可见：瘫软、无法站立
- 牙齿：剧烈磕碰、无法控制
```

### 恐惧颤抖微表情的"失控信号"

```
【恐惧颤抖微表情的核心：失控信号】

为什么"颤抖"是恐惧的关键？

失控的特点：
- 意志无法控制身体
- 肌肉不自主收缩
- 牙齿相互碰撞

【失控信号的三个关键点】

Point 1：牙齿打颤
- 颞肌和咬肌不自主收缩
- 上下牙齿相互碰撞
- 发出"咯咯"声

Point 2：嘴唇颤抖
- 口轮匝肌不自主收缩
- 嘴唇微微抖动
- 可能影响说话

Point 3：全身颤抖
- 四肢肌肉不自主收缩
- 可能无法站立
- 可能瘫软
```
