## 技能概述

**交叉维度**: 微表情 × 恍惚 × 记忆闪回 × 肌肉级
**定位**: 这是"记忆闪回"的微表情系统——当人物突然被拉入过去的记忆、创伤重现、或时间感消失时，会呈现一种"恍惚"的状态。眼神变得空洞，身体僵硬或轻微抖动，意识在过去和现在之间分裂。这是解离的典型表现，也是PTSD的核心症状。
**核心功能**: 提供记忆闪回微表情的肌肉级科学解析、FACS编码、恍惚状态设计、以及AI视频生成的闪回提示词

## 第一部分：记忆闪回的科学基础

### 什么是"记忆闪回"？——解离性恍惚的神经科学

```
【记忆闪回的心理学机制】

记忆闪回 = 解离性恍惚（Dissociative Fugue）

核心状态：
- 意识从现在分裂
- 被拉入过去的记忆
- 时间感消失
- 自我感分裂

【神经科学机制】

海马体（Hippocampus）：
- 负责情境记忆
- 闪回时：过去的场景被激活
- 与现在分离

杏仁核（Amygdala）：
- 负责情绪记忆
- 闪回时：创伤情绪被重新体验
- 比正常记忆更强烈

前额叶皮层（Prefrontal Cortex）：
- 负责"现在"的自我感
- 闪回时：功能减弱
- 导致自我感分裂

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU7: 眼睑轻微下垂（部分失去意识）
- AU43: 眼睛变得空洞（意识离开）
- 组合：AU7+AU43=闪回眼睛

眉毛（Eyebrows）：
- AU1: 轻微提升（可能的悲伤/恐惧）
- 或AU4: 轻微聚拢（可能的痛苦）

嘴部（Lips）：
- AU25: 轻微张开（部分失去意识）
- 可能轻微颤抖

身体（Body）：
- 僵硬（身体留在现在）
- 抖动（意识回到过去）
- 或完全静止

【记忆闪回的两阶段】

Stage 1：进入（进入期）
- 眼神：开始变得空洞
- 身体：开始僵硬
- 持续：1-5秒

Stage 2：完全闪回（闪回期）
- 眼神：完全空洞
- 身体：僵硬或抖动
- 持续：5秒到数分钟

【典型情境】

1. PTSD退伍军人听到爆炸声
2. 性侵受害者看到袭击者
3. 车祸幸存者再次经历事故
4. 战争摄影师看到相机里的图像
5. 童年创伤的闪回
6. 失去孩子的母亲看到照片
7. 丧失记忆者的记忆突然恢复
```

## 第三部分：恍惚状态设计

### 记忆闪回的五级梯度

```
【记忆闪回的情绪梯度】

Level 1：轻微恍惚（10%强度）
- 眼神：轻微失焦
- 身体：轻微僵硬
- 触发：记忆的提示开始
- 持续：1-3秒

Level 2：明显恍惚（30%强度）
- 眼神：明显失焦
- 身体：明显僵硬
- 触发：记忆开始进入
- 持续：3-10秒

Level 3：深度恍惚（60%强度）
- 眼神：深度失焦
- 身体：僵硬+轻微抖动
- 触发：完全进入记忆
- 持续：10-30秒

Level 4：完全闪回（85%强度）
- 眼神：完全空洞
- 身体：强烈抖动
- 触发：完全沉浸在过去
- 持续：30秒-数分钟

Level 5：解离（100%强度）
- 眼神：完全无神
- 身体：僵硬或抖动
- 触发：完全解离
- 持续：数分钟，可能需要外部介入

【记忆闪回的两型】

Type 1：创伤闪回（Traumatic Flashback）
- 特点：创伤记忆重现
- 情绪：恐惧、痛苦、绝望
- 身体：抖动、僵硬

Type 2：怀念闪回（Nostalgic Flashback）
- 特点：美好记忆重现
- 情绪：怀念、温暖、悲伤
- 身体：轻微僵硬
```
