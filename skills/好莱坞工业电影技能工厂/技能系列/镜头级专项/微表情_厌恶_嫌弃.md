## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 厌恶/嫌弃
**定位**: 这是"厌恶"情绪的微表情系统——基于FACS（面部动作编码系统），厌恶不是"讨厌"，是上唇提升皱缩（AU9/AU10）导致鼻子像兔子一样皱起来 + 眉头皱起（AU4） + 嘴角向下（AU15）或向下和向前突出的协同运动。厌恶的微表情是"身体在排斥有毒物质"——面部扭曲是为了把"有毒的东西"推出去。
**核心功能**: 提供厌恶的完整FACS肌肉运动图谱、兔子皱和鼻翼翕动的触发器、厌恶与愤怒的区分、以及AI视频生成的精确提示词

## 第一部分：厌恶微表情的科学基础

### 什么是"厌恶"？——FACS层面的精确解析

```
【厌恶的微表情科学】

"厌恶" = 身体排斥有毒物质的面部信号

厌恶的肌肉运动（必需）：
- 提上唇肌（levator labii superioris）收缩 → 上唇向上提升皱缩
- 鼻肌（compressor nasi / procerus）收缩 → 鼻翼翕动，鼻子皱起
- 皱眉肌（corrugator supercilii）收缩 → 眉间和眉头皱起
- 颧小肌（zygomaticus minor）抑制 → 嘴角不能上扬

【FACS编码】
- AU9（鼻肌收缩）→ 鼻翼向上翕动，鼻梁出现皱纹
- AU10（提上唇肌收缩）→ 上唇向上提升，露出牙龈
- AU4（皱眉肌收缩）→ 眉头皱起，眉间皱纹
- AU15（降口角肌收缩）→ 嘴角向下

【厌恶的"兔子皱"】

厌恶最独特的特征是"兔子皱"：
- 提上唇肌 + 鼻肌同时收缩
- 上唇向上皱缩 → 像兔子露出门牙
- 鼻翼向上翕动 → 鼻梁出现横向皱纹

【厌恶 vs 愤怒】

厌恶和愤怒有时相似，但关键区别：
- 厌恶：AU9/AU10（上唇提升）+ AU4（皱眉）
- 愤怒：AU4（皱眉）+ AU7（眼睑收紧）+ 无上唇提升

【厌恶的微表情分级】

Level 1：轻微不悦（轻度厌恶）
- FACS：AU4轻度激活
- 肌肉：只有皱眉肌
- 可见：眉头轻微皱起

Level 2：嫌弃（中等厌恶）
- FACS：AU4 + AU9轻度激活
- 肌肉：皱眉肌 + 鼻肌
- 可见：眉头皱起 + 鼻翼微皱

Level 3：厌恶（标准厌恶）
- FACS：AU4 + AU9 + AU10激活
- 肌肉：皱眉肌 + 鼻肌 + 提上唇肌
- 可见：眉头皱起 + 鼻子皱起 + 上唇提升

Level 4：厌恶（强烈厌恶）
- FACS：AUC + AU15
- 肌肉：所有厌恶肌肉 + 嘴角向下
- 可见：鼻子皱起 + 上唇提升 + 嘴角向下

Level 5：作呕（极强厌恶）
- FACS：AUC = 完全厌恶
- 肌肉：所有厌恶肌肉 + 作呕反应
- 可见：面部扭曲 + 身体后仰
```

### 厌恶微表情的"鼻-嘴区"

```
【厌恶微表情的核心：鼻-嘴区】

为什么"兔子皱"是厌恶的关键？

鼻-嘴区域（nasal-labial region）：
- 提上唇肌收缩 → 上唇向上提升，露出上牙
- 鼻肌收缩 → 鼻翼向上翕动，鼻梁横向皱纹
- 两者协同 = "兔子皱"

【鼻-嘴区的三个关键点】

Point 1：鼻翼翕动
- 鼻肌（compressor nasi）收缩
- 鼻翼向上和向外翕动
- 鼻梁出现横向皱纹

Point 2：上唇皱缩
- 提上唇肌收缩
- 上唇向上提升皱缩
- 像兔子露出门牙

Point 3：眉头皱起
- 皱眉肌收缩
- 眉头向中线聚拢
- 眉间出现垂直皱纹
```

## 第二部分：厌恶微表情的技术执行

### 厌恶微表情的AI生成参数

```
【厌恶微表情的AI生成系统】

鼻-嘴层：
- 上唇：向上提升皱缩
- 鼻翼：向上翕动
- 鼻梁：横向皱纹

【AI生成的"厌恶"面部参数】

眉毛：
- 整体：向中线聚拢皱起
- 内侧：眉头皱起
- 外侧：相对保持或微升
- 错误：平直或向上（那是惊讶）

眼睛：
- 上眼睑：可能轻微下垂（被皱眉肌拉动）
- 下眼睑：可能轻微提升
- 整体：眯着或不眯
- 错误：睁大（那是惊讶）

鼻部：
- 鼻翼：向上翕动
- 鼻梁：横向皱纹
- 鼻孔：可能轻微扩张

嘴部：
- 上唇：向上提升皱缩（AU10）
- 下唇：可能轻微前突
- 嘴角：向下（AU15）或平直
- 错误：嘴角上扬（那是幸福）

【厌恶微表情的"触发器"】

Trigger 1：兔子皱
- AI指令：nose wrinkled, upper lip curled up, rabbit nose, bunny nose, bunny face, rabbit-like grimace
- 效果：厌恶的决定性特征

Trigger 2：鼻翼翕动
- AI指令：nostrils flaring, nasalis muscle activated, nose flaring,鼻翼翕动
- 效果：厌恶的呼吸特征

Trigger 3：上唇提升
- AI指令：upper lip raised, levator labii superioris activated,上唇提升, gum exposed
- 效果：厌恶的嘴部特征

Trigger 4：眉头皱起
- AI指令：brows furrowed,眉头皱起,AU4, frown,眉头向中线聚拢
- 效果：厌恶的眉部特征

Trigger 5：嘴角向下
- AI指令：mouth corners pulled down,嘴角向下,AU15,下唇下降
- 效果：强烈厌恶的嘴部特征
```

## 第三部分：厌恶微表情的情绪设计

### 厌恶微表情的"情绪梯度"

```
【厌恶的情绪梯度设计】

Gradient 1：轻微不悦（轻度厌恶）
- 肌肉：AU4轻度激活
- 表情：眉头轻微皱起
- 情绪：不悦、反感、轻微讨厌
- 场景：看到不喜欢的东西

Gradient 2：嫌弃（中等厌恶）
- 肌肉：AU4 + AU9轻度激活
- 表情：眉头皱起 + 鼻翼微皱
- 情绪：嫌弃、厌恶、讨厌
- 场景：闻到难闻气味

Gradient 3：厌恶（标准厌恶）
- 肌肉：AU4 + AU9 + AU10激活
- 表情：眉头皱起 + 鼻子皱起 + 上唇提升
- 情绪：厌恶、反感、恶心
- 场景：看到恶心的事物

Gradient 4：强烈厌恶（强烈厌恶）
- 肌肉：AUC + AU15
- 表情：鼻子皱起 + 上唇提升 + 嘴角向下
- 情绪：强烈厌恶、憎恶、鄙夷
- 场景：面对极度反感的事物

Gradient 5：作呕（极强厌恶）
- 肌肉：所有厌恶肌肉 + 作呕反应
- 表情：面部扭曲 + 身体后仰
- 情绪：作呕、恶心到吐
- 场景：极端恶心的事物

【厌恶微表情的"场景适配"】

场景1：闻到臭味
- 厌恶类型：嗅觉厌恶
- 肌肉：AU9 + AU10 + 鼻翼翕动
- 表情：兔子皱 + 眼睛眯起
- 适配：闻到臭味、腐败气味

场景2：看到恶心事物
- 厌恶类型：视觉厌恶
- 肌肉：AU4 + AU9 + AU10
- 表情：眉头皱起 + 兔子皱
- 适配：看到昆虫、伤口

场景3：道德厌恶
- 厌恶类型：轻蔑厌恶
- 肌肉：AU4 + AU15
- 表情：眉头皱起 + 嘴角向下
- 眼神：俯视、鄙夷
- 适配：看不起某人
```
