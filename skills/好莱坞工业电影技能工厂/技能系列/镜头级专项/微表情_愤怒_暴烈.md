## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 愤怒/暴烈
**定位**: 这是"愤怒"情绪的微表情系统——基于FACS（面部动作编码系统），愤怒不是"凶"，是皱眉肌和眼轮匝肌的协同收缩导致眉毛下降并挤在一起（眉间皱纹）+ 下眼睑紧张提升 + 嘴唇紧闭（嘴角向前或向后收紧）的协同运动。愤怒的微表情是"身体的战斗准备"——面部肌肉的收缩是为攻击做准备的本能反应。
**核心功能**: 提供愤怒的完整FACS肌肉运动图谱、眉间皱纹与咬牙的触发器、愤怒与敌意的区分、以及AI视频生成的精确提示词

## 第一部分：愤怒微表情的科学基础

### 什么是"愤怒"？——FACS层面的精确解析

```
【愤怒的微表情科学】

"愤怒" = 身体攻击准备的面部信号

愤怒的肌肉运动（必需）：
- 皱眉肌（corrugator supercilii）收缩 → 眉毛向中线拉动并下降
- 眼轮匝肌（orbicularis oculi）收缩 → 眼睑紧张，下眼睑提升
- 颏肌（mentalis）收缩 → 下巴抬起（攻击姿态）
- 降下唇肌（depressor labii inferioris）激活 → 下唇微微抬起（咬牙）
- 颧大肌（zygomaticus major）抑制 → 嘴角不向上

【FACS编码】
- AU4（皱眉）→ 眉毛向中线下降并挤在一起
- AU7（眼睑收紧）→ 眼睑紧张
- AU9（鼻肌收缩）→ 鼻梁上方皱纹
- AU10（上唇提升）→ 上唇提升，露出牙龈
- AU17（下唇抬起）→ 下唇中央抬起
- AU23（嘴唇收紧）→ 嘴唇紧闭

【愤怒的"三角区"】

愤怒的三个核心区域：
- 眉间：皱眉肌收缩 → 垂直皱纹（眉间皱纹）
- 眼睛：眼轮匝肌收缩 → 眼睛变窄，下眼睑提升
- 嘴部：咬牙 → 嘴唇紧闭或张开露出牙齿

【愤怒的微表情分级】

Level 1：烦恼（轻度愤怒）
- FACS：AU4轻度激活
- 肌肉：只有皱眉肌
- 可见：眉毛轻微下降，眉间微皱

Level 2：不满（中等愤怒）
- FACS：AU4 + AU7激活
- 肌肉：皱眉肌 + 眼轮匝肌
- 可见：眉毛下降 + 眼睛变窄

Level 3：愤怒（标准愤怒）
- FACS：AU4 + AU7 + AU9激活
- 肌肉：皱眉肌 + 眼轮匝肌 + 鼻肌
- 可见：眉间皱纹 + 眼睛变窄 + 鼻梁皱纹

Level 4：暴怒（强烈愤怒）
- FACS：AUC + AU10 + AU23
- 肌肉：所有愤怒肌肉 + 嘴唇变化
- 可见：咬牙切齿 + 嘴唇紧闭或张开

Level 5：狂怒（极强愤怒）
- FACS：AUC = 完全愤怒
- 肌肉：所有愤怒肌肉 + 身体反应
- 可见：面部通红 + 血管暴起 + 颤抖
```

### 愤怒微表情的"三角区"

```
【愤怒微表情的核心：三角区】

为什么眉间是愤怒的关键？

眉间（glabella）：
- 皱眉肌的起点
- 当皱眉肌收缩，两条眉毛被拉向中线
- 眉毛挤在一起 → 眉间出现垂直皱纹

【三角区的三个关键点】

Point 1：眉间皱纹
- 皱眉肌收缩 → 眉间出现2-4条垂直皱纹
- 皱纹深度 = 愤怒程度
- 皱纹是愤怒的"信号枪"

Point 2：眼睛变窄
- 眼轮匝肌收缩 → 眼睛横向上变窄
- 下眼睑提升 → 眼睛纵向也变小
- 眼睛变窄 = 攻击前的聚焦

Point 3：咬牙
- 颏肌收缩 → 下巴抬起
- 降下唇肌激活 → 下唇中央抬起
- 咬牙 = 抑制攻击的尝试
```

## 第二部分：愤怒微表情的技术执行

### 愤怒微表情的AI生成参数

```
【愤怒微表情的AI生成系统】

眼睑层：
- 上眼睑：向下压（被皱眉肌拉动）
- 下眼睑：紧张提升
- 眼睛：横向变窄，纵向也变小

【AI生成的"愤怒"面部参数】

眉毛：
- 整体：向中线下降并挤在一起
- 内侧：明显下降（皱眉肌）
- 外侧：也下降，但不如内侧明显
- 错误：平直或向上（那是惊讶）

眼睛：
- 上眼睑：向下压
- 下眼睑：提升并紧张
- 瞳孔：可能缩小（聚焦）
- 错误：睁大（那是恐惧）

鼻部：
- 鼻梁：横向皱纹（鼻肌收缩/AU9）
- 鼻翼：可能扩张（愤怒时用嘴呼吸）

嘴部：
- 紧闭：嘴唇收紧（AU23）
- 张开：上唇提升，露出牙龈（AU10）
- 咬牙：下唇中央抬起（AU17）
- 错误：嘴角向上（那是幸福）

【愤怒微表情的"触发器"】

Trigger 1：眉间皱纹
- AI指令：glabella wrinkles, vertical frown lines, corrugated brow, eyebrows drawn together
- 效果：愤怒的决定性标记

Trigger 2：眼睛变窄
- AI指令：eyes narrowed, eye aperture reduced, lower eyelid raised, eye tense
- 效果：愤怒眼睛的特征

Trigger 3：咬牙
- AI指令：jaw clenched, chin raised, mentalis contracted, lower lip elevated
- 效果：抑制攻击的面部信号

Trigger 4：鼻梁皱纹
- AI指令：transverse nasal wrinkles, procerus contracted, nose wrinkled
- 效果：愤怒的鼻部特征

Trigger 5：下唇抬起
- AI指令：lower lip central raised, depressor labii inferioris activated,咬牙
- 效果：强烈愤怒的嘴部特征
```

## 第三部分：愤怒微表情的情绪设计

### 愤怒微表情的"情绪梯度"

```
【愤怒的情绪梯度设计】

Gradient 1：烦恼（轻度愤怒）
- 肌肉：AU4轻度激活
- 表情：眉毛轻微下降，眉间微皱
- 情绪：不满、烦躁、恼怒
- 场景：被打扰、轻微冒犯

Gradient 2：不满（中等愤怒）
- 肌肉：AU4 + AU7激活
- 表情：眉毛下降 + 眼睛变窄
- 情绪：不快、生气、恼火
- 场景：被欺骗、被轻视

Gradient 3：愤怒（标准愤怒）
- 肌肉：AU4 + AU7 + AU9激活
- 表情：眉间皱纹 + 眼睛变窄 + 鼻梁皱纹
- 情绪：愤怒、恼怒、激动
- 场景：被侮辱、被伤害

Gradient 4：暴怒（强烈愤怒）
- 肌肉：AUC + AU10 + AU23
- 表情：咬牙切齿 + 嘴唇变化
- 情绪：暴怒、狂怒、愤慨
- 场景：被背叛、极度不公

Gradient 5：狂怒（极强愤怒）
- 肌肉：所有愤怒肌肉 + 身体反应
- 表情：面部通红、血管暴起、颤抖
- 情绪：狂怒、暴怒、杀意
- 场景：杀亲之仇、毁灭性背叛

【愤怒微表情的"场景适配"】

场景1：压抑愤怒
- 愤怒类型：内敛的愤怒
- 肌肉：AU4轻度 + AU23（嘴唇紧闭）
- 表情：眉毛微皱，嘴唇紧闭
- 适配：办公室冲突、压抑情绪

场景2：爆发愤怒
- 愤怒类型：外显的愤怒
- 肌肉：AUC + AU10（嘴唇张开）
- 表情：咬牙切齿，上唇提升
- 适配：激烈争吵、对抗

场景3：冷怒
- 愤怒类型：冰冷的愤怒
- 肌肉：AU4 + AU7 + AU23
- 表情：眉间皱纹 + 眼睛变窄 + 嘴唇紧闭
- 眼神：冷峻、盯着对方
- 适配：威胁、警告
```
