## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 恐惧/惊恐
**定位**: 这是"恐惧"情绪的微表情系统——基于FACS（面部动作编码系统），恐惧不是"害怕"，是眉毛内侧向上提升并聚拢（AU1+AU4）导致眉毛呈"倒八字形"+ 眼睛睁大上眼睑提升（AU5）+ 嘴唇向两侧水平拉伸（AU20）的协同运动。恐惧的微表情是"生存本能的面部信号"——睁大眼睛是为了扩大视野，嘴巴是为了快速呼吸和尖叫准备。
**核心功能**: 提供恐惧的完整FACS肌肉运动图谱、倒八字眉与瞪眼的触发器、恐惧与惊讶的区分、以及AI视频生成的精确提示词

## 第一部分：恐惧微表情的科学基础

### 什么是"恐惧"？——FACS层面的精确解析

```
【恐惧的微表情科学】

"恐惧" = 生存本能的面部激活

恐惧的肌肉运动（必需）：
- 皱眉肌（corrugator supercilii）+ 额肌（frontalis）内侧收缩 → 眉毛内侧向上提升并聚拢
- 提上睑肌（levator palpebrae superioris）激活 → 上眼睑提升，眼睛睁大
- 口轮匝肌（orbicularis oris）外侧收缩 → 嘴唇向两侧水平拉伸

【FACS编码】
- AU1（眉毛内侧提升）→ 眉毛内侧向上
- AU4（眉毛下降）→ 眉毛整体下降
- AU5（上眼睑提升）→ 上眼睑向上提升
- AU20（嘴唇水平拉伸）→ 嘴唇向两侧拉

【恐惧的"倒八字眉"】

恐惧的眉毛是独特的"倒八字形"：
- 内侧：向上提升（AU1）
- 外侧：向下压（AU4）
- 结果：眉毛内侧高于外侧

【恐惧 vs 惊讶】

恐惧和惊讶有相似的眼睛，但关键区别：
- 恐惧：AU1+AU4（眉毛内侧升外侧降）+ AU5
- 惊讶：AU1+AU2（整个眉毛提升）+ AU5

【恐惧的微表情分级】

Level 1：担心（轻度恐惧）
- FACS：AU1 + AU4轻度激活
- 肌肉：只有眉毛变化
- 可见：眉毛呈倒八字形，眼睑正常

Level 2：不安（中等恐惧）
- FACS：AU1 + AU4 + AU5轻度激活
- 肌肉：眉毛 + 眼睑变化
- 可见：倒八字眉 + 眼睛轻微睁大

Level 3：恐惧（标准恐惧）
- FACS：AU1 + AU4 + AU5 + AU20激活
- 肌肉：眉毛 + 眼睑 + 嘴部
- 可见：倒八字眉 + 瞪眼 + 嘴唇微张

Level 4：惊恐（强烈恐惧）
- FACS：AUC + 颤抖
- 肌肉：所有恐惧肌肉 + 身体反应
- 可见：极度瞪眼 + 嘴唇完全张开 + 颤抖

Level 5：恐怖（极强恐惧）
- FACS：AUC = 完全恐惧
- 肌肉：所有恐惧肌肉 + 尖叫准备
- 可见：尖叫表情 + 眼睛瞪到最大 + 尖叫
```

### 恐惧微表情的"眼睛-嘴巴连线"

```
【恐惧微表情的核心：眼睛-嘴巴连线】

为什么"瞪眼"和"张嘴"是恐惧的关键？

瞪眼（AU5）：
- 提上睑肌激活 → 上眼睑向上提升
- 眼睛睁大 → 巩膜（白眼球）暴露增加
- 瞪大是为了"扩大视野"，寻找逃跑路线

张嘴（AU20）：
- 口轮匝肌外侧收缩 → 嘴唇向两侧水平拉伸
- 嘴巴张开 → 为尖叫准备气道
- 张嘴也是"惊讶"的信号

【眼睛-嘴巴连线的三个关键点】

Point 1：眉毛（倒八字）
- AU1 + AU4 → 内侧升，外侧降
- 眉毛呈倒V或倒八字形
- 这是恐惧眉毛的决定性特征

Point 2：眼睛（瞪大）
- AU5 → 上眼睑大幅提升
- 巩膜暴露增加
- 瞳孔可能放大（应激反应）

Point 3：嘴巴（张开）
- AU20 → 嘴唇向两侧水平拉伸
- 嘴巴张开呈椭圆形或长方形
- 可能露出牙齿和牙龈
```

## 第二部分：恐惧微表情的技术执行

### 恐惧微表情的AI生成参数

```
【恐惧微表情的AI生成系统】

眼睑层：
- 上眼睑：大幅向上提升（AU5）
- 下眼睑：可能紧张，向上提升
- 瞳孔：可能放大

【AI生成的"恐惧"面部参数】

眉毛：
- 内侧：向上提升（AU1）
- 外侧：向下压（AU4）
- 整体：呈倒八字形（内侧高于外侧）
- 错误：平直或向下（那是愤怒）

眼睛：
- 上眼睑：大幅提升（AU5）
- 巩膜：暴露增加
- 瞳孔：可能放大
- 错误：正常或变小（那是愤怒）

鼻部：
- 正常或轻微扩张
- 恐惧时用嘴呼吸

嘴部：
- 水平拉伸：嘴唇向两侧拉（AU20）
- 形状：椭圆形或长方形
- 可能露出牙齿
- 错误：向下（那是悲伤）

【恐惧微表情的"触发器"】

Trigger 1：倒八字眉
- AI指令：inner eyebrows raised, outer eyebrows lowered, inverted V eyebrows, fear brows
- 效果：恐惧眉毛的决定性特征

Trigger 2：瞪大眼睛
- AI指令：eyes wide open, upper eyelid raised high, AU5, sclera exposed, pupils dilated
- 效果：恐惧眼睛的决定性特征

Trigger 3：嘴巴张开
- AI指令：mouth open horizontally, lips stretched laterally, AU20, mouth agape, lips parted
- 效果：恐惧嘴部的决定性特征

Trigger 4：下巴下降
- AI指令：jaw dropped, mandible lowered,下颌下降
- 效果：极度恐惧的嘴部特征

Trigger 5：瞳孔放大
- AI指令：pupils dilated,瞳孔放大,应激反应
- 效果：恐惧的生理反应
```

## 第三部分：恐惧微表情的情绪设计

### 恐惧微表情的"情绪梯度"

```
【恐惧的情绪梯度设计】

Gradient 1：担心（轻度恐惧）
- 肌肉：AU1 + AU4轻度激活
- 表情：倒八字眉，眼睑正常
- 情绪：担心、忧虑、不安
- 场景：预感不妙、轻微危险

Gradient 2：不安（中等恐惧）
- 肌肉：AU1 + AU4 + AU5轻度激活
- 表情：倒八字眉 + 眼睛轻微睁大
- 情绪：恐惧、担忧、害怕
- 场景：看到危险、感到威胁

Gradient 3：恐惧（标准恐惧）
- 肌肉：AU1 + AU4 + AU5 + AU20激活
- 表情：倒八字眉 + 瞪眼 + 嘴唇微张
- 情绪：恐惧、惊恐、害怕
- 场景：直接威胁、危险逼近

Gradient 4：惊恐（强烈恐惧）
- 肌肉：AUC + 颤抖
- 表情：极度瞪眼 + 嘴唇完全张开
- 情绪：惊恐、惊慌、恐慌
- 场景：生命危险、极度威胁

Gradient 5：恐怖（极强恐惧）
- 肌肉：所有恐惧肌肉 + 尖叫
- 表情：尖叫表情 + 眼睛瞪到最大
- 情绪：恐怖、惊骇、魂飞魄散
- 场景：面对死亡、极度创伤

【恐惧微表情的"场景适配"】

场景1：看到危险
- 恐惧类型：警觉的恐惧
- 肌肉：AU1 + AU4 + AU5
- 表情：倒八字眉 + 瞪眼，嘴部正常或微张
- 适配：发现危险、预感威胁

场景2：面对威胁
- 恐惧类型：明显的恐惧
- 肌肉：AU1 + AU4 + AU5 + AU20
- 表情：倒八字眉 + 瞪眼 + 嘴唇张开
- 适配：被枪指、面对凶手

场景3：惊恐尖叫
- 恐惧类型：尖叫的恐惧
- 肌肉：AUC + 尖叫
- 表情：极度瞪眼 + 嘴巴大张 + 尖叫
- 适配：恐怖时刻、极度创伤
```
