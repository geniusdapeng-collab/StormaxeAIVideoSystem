## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 压抑悲伤/无声落泪
**定位**: 这是"压抑悲伤"变体的微表情系统——悲伤的核心是"静默的崩溃"，没有声音，只有面部肌肉的"重力下沉"和眼泪的静静流淌。眉毛内角提升（AU1）+ 眼睑下垂 + 嘴角向下（AU15）+ 眼泪静静流下的协同运动。压抑悲伤的微表情是"无法承受的痛苦在无声地溢出"——眼泪是唯一的出口。
**核心功能**: 提供压抑悲伤的完整FACS肌肉运动图谱、无声落泪与痛哭的区分、眼泪流下的触发器、以及AI视频生成的精确提示词

## 第一部分：压抑悲伤微表情的科学基础

### 什么是"压抑悲伤"？——FACS层面的精确解析

```
【压抑悲伤的微表情科学】

"压抑悲伤" = 无法哭泣的痛苦

压抑悲伤的肌肉运动（必需）：
- 皱眉肌（corrugator supercilii）+ 额肌内侧 → 眉毛内角向上（AU1）
- 眼睑：眼睑下垂（上眼睑向下）
- 降口角肌（depressor anguli oris） → 嘴角向下（AU15）
- 眼泪：泪腺激活，眼泪静静流下

【压抑悲伤 vs 痛哭的区别】

压抑悲伤（静默）：
- 表情：轻微的面部变化
- 眼泪：静静流下，不出声
- 身体：可能僵硬

痛哭（有声）：
- 表情：剧烈的面部扭曲
- 眼泪：大量涌出，可能伴随鼻涕
- 身体：肩膀抖动、抽泣

【FACS编码】
- AU1（眉毛内侧提升）→ 眉毛内侧向上
- AU4（皱眉）→ 眉毛下降并聚拢
- AU15（嘴角向下）→ 嘴角向下
- 泪腺激活 → 眼泪流下

【压抑悲伤的微表情分级】

Level 1：隐忍
- 肌肉：AU15轻度激活
- 可见：嘴角轻微下压
- 眼泪：无或少量

Level 2：含泪
- 肌肉：AU1 + AU15
- 可见：金字塔眉 + 嘴角下垂 + 眼眶含泪
- 眼泪：眼眶中打转

Level 3：无声落泪
- 肌肉：AU1 + AU4 + AU15 + 泪腺
- 可见：完整悲伤表情 + 眼泪流下
- 眼泪：静静流下

Level 4：泣不成声（但压抑）
- 肌肉：AU1 + AU4 + AU15 + 强烈泪腺 + 喉部紧张
- 可见：悲伤表情 + 眼泪 + 喉咙哽咽
- 眼泪：大量流下，但无声音

Level 5：彻底崩溃（但静默）
- 肌肉：所有悲伤肌肉 + 大量眼泪 + 颤抖
- 可见：完全崩溃 + 眼泪横流 + 面部颤抖
- 眼泪：无法控制地流下
```

### 压抑悲伤微表情的"静默信号"

```
【压抑悲伤微表情的核心：静默信号】

为什么"无声"是压抑悲伤的关键？

静默的特点：
- 面部变化轻微（与内心痛苦不成比例）
- 眼泪流下，但无声
- 身体僵硬，抑制哭泣冲动

【静默信号的三个关键点】

Point 1：眼睑下垂
- 上眼睑向下（被眼眶肌肉拉动）
- 下眼睑内侧向上
- 眼眶看起来"空"

Point 2：眼泪流下
- 眼泪从眼眶静静流下
- 没有抽泣的声音
- 眼泪可能沿着脸颊流下

Point 3：嘴角下压
- 降口角肌激活（AU15）
- 嘴角向下
- 嘴部可能轻微颤抖
```

## 第二部分：压抑悲伤微表情的技术执行

### 压抑悲伤微表情的AI生成参数

```
【压抑悲伤微表情的AI生成系统】

眼睑层：
- 上眼睑：下垂
- 下眼睑：内侧向上牵拉
- 眼眶：可能有泪

【AI生成的"压抑悲伤"面部参数】

眉毛：
- 内侧：向上提升（AU1）
- 外侧：向下压（AU4）
- 整体：倒八字形或金字塔形

眼睛：
- 上眼睑：下垂（悲伤眼睑）
- 下眼睑：内侧向上
- 眼眶：含泪或眼泪流下
- 瞳孔：可能半闭

嘴部：
- 嘴角：向下（AU15）
- 可能轻微颤抖
- 嘴唇可能紧闭（抑制哭泣）

脸颊：
- 眼泪：沿着脸颊流下
- 脸颊：可能轻微下垂

【压抑悲伤微表情的"触发器"】

Trigger 1：眼泪流下
- AI指令：tears streaming, tears rolling down, silent tears, tears falling,眼泪流下
- 效果：压抑悲伤的核心标记

Trigger 2：眼睑下垂
- AI指令：drooping eyelids, eyelid descended, upper lid covering pupil,眼睑下垂
- 效果：悲伤眼睛的标志

Trigger 3：嘴角下压
- AI指令：mouth corners pulled down,嘴角向下, depressor anguli oris activated
- 效果：悲伤嘴部的决定性特征

Trigger 4：金字塔眉
- AI指令：pyramid eyebrows, inverted V brows, inner raised outer lowered,金字塔眉
- 效果：悲伤眉毛的标志

Trigger 5：静默
- AI指令：silent, no sound, holding back sobs, silent grief,静默
- 效果：压抑悲伤的核心特质
```

## 第三部分：压抑悲伤微表情的情绪设计

### 压抑悲伤微表情的"情绪梯度"

```
【压抑悲伤的情绪梯度设计】

Gradient 1：隐忍
- 肌肉：AU15轻度激活
- 可见：嘴角轻微下压
- 情绪：隐忍、不愿表达
- 场景：公众场合、必须坚强

Gradient 2：含泪
- 肌肉：AU1 + AU15 + 轻度泪腺
- 可见：金字塔眉 + 嘴角下垂 + 眼眶湿润
- 情绪：快要忍不住、但在控制
- 场景：告别、压抑场合

Gradient 3：无声落泪
- 肌肉：完整悲伤 + 泪腺
- 可见：眼泪静静流下
- 情绪：无法控制、但仍然安静
- 场景：深夜独处、失去希望

Gradient 4：泣不成声
- 肌肉：强烈悲伤 + 喉部紧张
- 可见：眼泪 + 喉咙哽咽（但无声）
- 情绪：极度悲伤、但抑制声音
- 场景：听到噩耗、不敢相信

Gradient 5：彻底崩溃（静默）
- 肌肉：所有悲伤肌肉 + 大量眼泪 + 颤抖
- 可见：完全崩溃 + 眼泪横流 + 面部扭曲
- 情绪：彻底无法承受
- 场景：最亲近的人去世

【压抑悲伤微表情的"场景适配"】

场景1：告别
- 类型：隐忍的告别
- 肌肉：AU15轻度
- 可见：嘴角下压、眼睛含泪
- 适配：葬礼、送行

场景2：深夜独处
- 类型：无声崩溃
- 肌肉：完整悲伤 + 泪腺
- 可见：眼泪流下、但安静
- 适配：独自一人、无人知晓

场景3：听到噩耗
- 类型：震惊的悲伤
- 肌肉：AU1 + AU4 + 强烈泪腺
- 可见：震惊表情 + 眼泪
- 适配：突然的坏消息
```
