## 技能概述

**交叉维度**: 微表情 × 告别 × 无声离别 × 肌肉级
**定位**: 这是"无声离别"的微表情系统——当人物经历最艰难的告别时，往往不是哭泣或拥抱，而是一种深层的、内敛的、几乎无声的离别。嘴角可能有一丝几乎不可见的颤抖，眼神深处有无法言说的情感，面部肌肉僵硬，仿佛在用力控制自己不让情绪崩溃。这是告别时最令人心碎的状态。
**核心功能**: 提供告别微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的告别提示词

## 第一部分：告别的科学基础

### 什么是"无声离别"？——依恋理论的情感崩溃

```
【告别的心理学机制】

告别 = 依恋系统激活 + 情感压抑

核心状态：
- 依恋系统被强烈激活
- 必须压抑情感以完成告别
- 面部肌肉僵硬控制
- 眼神成为唯一出口

【依恋理论】

约翰·鲍尔比（John Bowlby）的依恋理论：
- 依恋是人类的基本需求
- 分离触发依恋焦虑
- 告别激活分离痛苦

安全依恋 vs. 不安全依恋：
- 安全依恋：可以健康告别
- 不安全依恋：告别困难，可能过度情绪化

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU43: 眼睛变得"湿润"（即将崩溃）
- 轻微颤抖（控制情绪的努力）
- 组合：AU43+颤抖=告别眼睛

眉毛（Eyebrows）：
- AU1: 内侧提升（悲伤）
- AU4: 轻微聚拢（痛苦）
- 组合：AU1+AU4=告别的眉毛

嘴部（Lips）：
- AU17: 下巴提升（用力控制）
- AU15: 嘴角轻微下压（压抑悲伤）
- AU24: 嘴唇接触（阻止哭泣）
- 组合：AU17+AU15+AU24=无声的嘴

整体：
- 所有肌肉：僵硬（控制情绪）
- 面部：仿佛在说"我不能哭"

【告别的两阶段】

Stage 1：告别前的瞬间
- 特点：最后的凝视
- 微表情：深情的眼神
- 持续：数秒

Stage 2：告别时刻
- 特点：转身离开
- 微表情：僵硬的面部
- 持续：数秒

【典型情境】

1. 火车站送别
2. 机场安检口
3. 医院病房的最后一面
4. 葬礼上的告别
5. 离婚后的最后凝视
6. 老人送别子女
7. 父母送孩子上大学
8. 恋人分手
9. 朋友远行
10. 宠物临终
```

## 第三部分：情绪梯度设计

### 告别的五级梯度

```
【告别的情绪梯度】

Level 1：轻微告别（10%强度）
- 特点：即将告别的预感
- 微表情：轻微的眼神
- 身体：轻微的紧张

Level 2：明显告别（30%强度）
- 特点：告别前的凝视
- 微表情：深情的眼神
- 身体：明显的紧张

Level 3：深层告别（60%强度）
- 特点：告别时刻
- 微表情：眼睛湿润
- 身体：僵硬

Level 4：极度告别（85%强度）
- 特点：最后的触碰
- 微表情：无法控制的湿润
- 身体：用尽全力控制

Level 5：终极告别（100%强度）
- 特点：无声崩溃
- 微表情：完全崩溃但无声
- 身体：僵硬到颤抖

【告别类型】

Type 1：生离（Temporary Separation）
- 特点：知道会再见面
- 情绪：悲伤但有希望
- 场景：出差、送孩子

Type 2：死别（Permanent Separation）
- 特点：永远不会再见面
- 情绪：绝望、崩溃
- 场景：葬礼、临终

Type 3：断联（Forced Separation）
- 特点：被迫分开
- 情绪：愤怒、悲伤
- 场景：离婚、绝交
```

**【无声离别微表情的禁止词清单】**

```
❌ crying loudly（无声离别 = 压抑 = 不能大声哭泣）
❌ wailing（无声离别 = 控制 = 不能哀号）
❌ collapsing（无声离别 = 镇定 = 不能崩溃）
❌ expressing openly（无声离别 = 压抑 = 不能开放表达）
❌ shouting（无声离别 = 无声 = 不能喊叫）
❌ screaming（无声离别 = 无声 = 不能尖叫）
❌ breaking down（无声离别 = 控制 = 不能崩溃）
❌ losing composure（无声离别 = 镇定 = 不能失去镇定）
❌ showing weakness（无声离别 = 坚强 = 不能表现软弱）
❌ being emotional（无声离别 = 控制 = 不能情绪化）
```
