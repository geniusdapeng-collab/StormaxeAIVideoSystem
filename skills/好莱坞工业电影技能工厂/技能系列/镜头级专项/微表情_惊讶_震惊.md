## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 惊讶/震惊
**定位**: 这是"惊讶"情绪的微表情系统——基于FACS（面部动作编码系统），惊讶不是"吃惊"，是整个眉毛向上提升（AU1+AU2）+ 眼睛睁大上眼睑大幅提升（AU5）+ 下颌下降嘴巴张开（AU26）的协同运动。惊讶的微表情是"大脑在高速处理信息"的面部信号——所有面部肌肉都在"暂停"以给大脑更多资源来处理意外信息。
**核心功能**: 提供惊讶的完整FACS肌肉运动图谱、眉毛整体提升与眼睛睁大的触发器、惊讶与恐惧的区分、以及AI视频生成的精确提示词

## 第一部分：惊讶微表情的科学基础

### 什么是"惊讶"？——FACS层面的精确解析

```
【惊讶的微表情科学】

"惊讶" = 大脑处理意外信息的面部暂停

惊讶的肌肉运动（必需）：
- 额肌（frontalis）整体收缩 → 整个眉毛向上提升（双侧）
- 提上睑肌（levator palpebrae superioris）激活 → 上眼睑大幅提升
- 下颌肌（masseter）激活 → 下颌下降，嘴巴张开

【FACS编码】
- AU1（眉毛内侧提升）→ 眉毛内侧向上
- AU2（眉毛外侧提升）→ 眉毛外侧向上
- AU5（上眼睑提升）→ 上眼睑大幅向上
- AU26（下颌下降）→ 嘴巴张开

【惊讶的"双提升"】

惊讶眉毛 vs 其他情绪：
- 惊讶：AU1 + AU2（整个眉毛向上）= 眉毛平直或拱形向上
- 恐惧：AU1 + AU4（内侧升外侧降）= 倒八字形
- 悲伤：AU1 + AU4（内侧升外侧降）= 倒八字形

惊讶眼睛 vs 其他情绪：
- 惊讶：AU5（大幅提升）= 眼睛睁到最大
- 恐惧：AU5 + 瞳孔放大 = 眼睛睁大 + 瞳孔变化
- 愤怒：AU7（眼睑收紧）= 眼睛变窄

【惊讶的微表情分级】

Level 1：注意（轻度惊讶）
- FACS：AU2轻度激活
- 肌肉：只有额肌外侧
- 可见：眉毛轻微向上

Level 2：意外（中等惊讶）
- FACS：AU1 + AU2 + AU5轻度激活
- 肌肉：额肌 + 提上睑肌
- 可见：眉毛向上 + 眼睛睁大

Level 3：惊讶（标准惊讶）
- FACS：AU1 + AU2 + AU5 + AU26激活
- 肌肉：额肌 + 提上睑肌 + 下颌肌
- 可见：标准惊讶表情

Level 4：震惊（强烈惊讶）
- FACS：AUC + 嘴巴大张
- 肌肉：所有惊讶肌肉 + 嘴巴完全张开
- 可见：嘴巴大张 + 眼睛瞪大

Level 5：惊骇（极强惊讶）
- FACS：AUC = 完全惊讶
- 肌肉：所有惊讶肌肉 + 身体僵硬
- 可见：完全僵住 + 嘴巴大张
```

### 惊讶微表情的"暂停信号"

```
【惊讶微表情的核心：暂停信号】

为什么惊讶是"暂停"？

惊讶的面部特点：
- 眉毛：整体向上（不是内侧或外侧，是整个）
- 眼睛：睁到最大（不是变窄）
- 嘴巴：张开（不是紧闭或向下）
- 整体：所有肌肉都在"向上"和"张开"

惊讶的生理意义：
- 眼睛睁大 = 扩大视野，接收更多信息
- 嘴巴张开 = 准备呼吸/尖叫
- 眉毛向上 = 扩大视野范围
- 整体 = "暂停"所有活动来处理意外

【暂停信号的三个关键点】

Point 1：眉毛（整体提升）
- 额肌整体收缩 → 整个眉毛向上
- 眉毛呈平直或微拱形
- 双侧对称

Point 2：眼睛（最大睁大）
- 提上睑肌大幅激活 → 上眼睑提升到最大
- 巩膜（白眼球）大面积暴露
- 瞳孔可能正常或轻微放大

Point 3：嘴巴（张开）
- 下颌下降 → 嘴巴自然张开
- 张口度通常较小到中等
- 惊讶强烈时可能大张
```

## 第二部分：惊讶微表情的技术执行

### 惊讶微表情的AI生成参数

```
【惊讶微表情的AI生成系统】

眼睑层：
- 上眼睑：大幅向上提升（AU5）
- 下眼睑：可能轻微提升（被拉伸）
- 瞳孔：可能轻微放大

【AI生成的"惊讶"面部参数】

眉毛：
- 整体：向上提升（AU1 + AU2）
- 形状：平直或微拱形
- 双侧：对称
- 错误：内侧升外侧降（那是恐惧）

眼睛：
- 上眼睑：大幅提升（AU5）
- 巩膜：暴露增加
- 瞳孔：可能轻微放大
- 错误：变窄（那是愤怒）

鼻部：
- 正常或轻微扩张
- 惊讶时可能轻微上提

嘴部：
- 下颌下降：嘴巴张开（AU26）
- 张口度：轻度到中度
- 形状：椭圆形或垂直椭圆形
- 错误：紧闭（那是愤怒）

【惊讶微表情的"触发器"】

Trigger 1：眉毛整体向上
- AI指令：both eyebrows raised, entire brow elevated, AU1 AU2, flat or slightly arched brows
- 效果：惊讶眉毛的决定性特征

Trigger 2：眼睛最大睁大
- AI指令：eyes wide open, upper eyelid raised maximum, AU5, eyes at maximum aperture, fully exposed sclera
- 效果：惊讶眼睛的决定性特征

Trigger 3：嘴巴张开
- AI指令：mouth open, jaw dropped, AU26, mouth parted, lips separated
- 效果：惊讶嘴部的决定性特征

Trigger 4：双侧对称
- AI指令：bilateral symmetry, both sides equal, symmetric expression
- 效果：真惊讶 vs 假惊讶

Trigger 5：短暂持续
- AI指令：brief duration, expression appears suddenly then fades, rapid onset and offset
- 效果：惊讶是"暂停"不是"表情"
```

## 第三部分：惊讶微表情的情绪设计

### 惊讶微表情的"情绪梯度"

```
【惊讶的情绪梯度设计】

Gradient 1：注意（轻度惊讶）
- 肌肉：AU2轻度激活
- 表情：眉毛轻微向上
- 情绪：注意、警觉、好奇
- 场景：听到声音、发现异常

Gradient 2：意外（中等惊讶）
- 肌肉：AU1 + AU2 + AU5轻度激活
- 表情：眉毛向上 + 眼睛轻微睁大
- 情绪：意外、惊奇、诧异
- 场景：听到意外消息

Gradient 3：惊讶（标准惊讶）
- 肌肉：AU1 + AU2 + AU5 + AU26激活
- 表情：标准惊讶表情
- 情绪：惊讶、吃惊、震惊
- 场景：面对意外事件

Gradient 4：震惊（强烈惊讶）
- 肌肉：AUC + 嘴巴大张
- 表情：嘴巴大张 + 眼睛瞪大
- 情绪：震惊、惊骇、难以置信
- 场景：听到噩耗或重大消息

Gradient 5：惊骇（极强惊讶）
- 肌肉：所有惊讶肌肉 + 身体僵硬
- 表情：完全僵住 + 嘴巴大张
- 情绪：惊骇、魂飞魄散
- 场景：面对极端意外

【惊讶微表情的"场景适配"】

场景1：意外发现
- 惊讶类型：好奇的惊讶
- 肌肉：AU1 + AU2 + AU5轻度
- 表情：眉毛向上 + 眼睛睁大
- 适配：发现秘密、看到意外

场景2：听到消息
- 惊讶类型：消息冲击的惊讶
- 肌肉：AU1 + AU2 + AU5 + AU26
- 表情：标准惊讶
- 适配：听到好消息/坏消息

场景3：面对真相
- 惊讶类型：震惊的惊讶
- 肌肉：AUC + 嘴巴大张
- 表情：极度惊讶
- 适配：真相揭露、重大发现
```
