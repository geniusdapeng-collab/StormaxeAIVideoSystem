## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 厌恶皱眉/鼻翼翕动
**定位**: 这是"厌恶皱眉"变体的微表情系统——厌恶的核心是"鼻子在排斥"，是鼻翼翕动和上唇皱缩的协同运动。鼻肌收缩导致鼻梁出现横向皱纹（AU9）+ 提上唇肌导致上唇向上皱缩（AU10）+ 鼻翼向上翘起的协同运动。厌恶皱眉的微表情是"身体在说'不'"——鼻子和嘴唇联合起来拒绝有毒物质。
**核心功能**: 提供厌恶皱眉的完整FACS肌肉运动图谱、鼻翼翕动与兔子皱的触发器、嗅觉厌恶与视觉厌恶的区分、以及AI视频生成的精确提示词

## 第一部分：厌恶皱眉微表情的科学基础

### 什么是"厌恶皱眉"？——FACS层面的精确解析

```
【厌恶皱眉的微表情科学】

"厌恶皱眉" = 身体拒绝有毒物质

厌恶皱眉的肌肉运动（必需）：
- 鼻肌（compressor nasi / procerus）→ 鼻梁横向皱纹（AU9）
- 提上唇肌（levator labii superioris）→ 上唇向上皱缩（AU10）
- 颧小肌（zygomaticus minor）抑制 → 嘴角不能上扬
- 皱眉肌（corrugator supercilii）→ 眉间皱纹（AU4）

【厌恶皱眉的"兔子皱"】

厌恶最独特的特征是"兔子皱"：
- 提上唇肌 + 鼻肌同时收缩
- 上唇向上皱缩 → 像兔子露出门牙
- 鼻翼向上翘起 → 鼻梁出现横向皱纹

【嗅觉厌恶 vs 视觉厌恶】

嗅觉厌恶（主要鼻子）：
- 鼻肌和提上唇肌强烈收缩
- 鼻子明显皱起
- 可能伴随嗅探动作

视觉厌恶（主要面部）：
- 皱眉肌强烈收缩
- 眉头明显皱起
- 鼻子可能轻微皱起

【FACS编码】
- AU9（鼻肌）→ 鼻翼向上翕动，鼻梁皱纹
- AU10（上唇）→ 上唇向上提升皱缩
- AU4（皱眉）→ 眉间皱纹

【厌恶皱眉的微表情分级】

Level 1：轻微嫌弃
- 肌肉：AU9轻度激活
- 可见：鼻翼轻微翕动

Level 2：嫌弃
- 肌肉：AU9 + AU10轻度
- 可见：鼻子皱起 + 上唇微抬

Level 3：厌恶
- 肌肉：AU9 + AU10 + AU4激活
- 可见：兔子皱 + 眉头皱起

Level 4：强烈厌恶
- 肌肉：所有厌恶肌肉 + 可能作呕
- 可见：完全兔子皱 + 面部扭曲

Level 5：作呕
- 肌肉：所有厌恶肌肉 + 身体反应
- 可见：干呕、身体后仰
```

### 厌恶皱眉微表情的"鼻-嘴联动"

```
【厌恶皱眉微表情的核心：鼻-嘴联动】

为什么"兔子皱"是厌恶的关键？

鼻-嘴联动的特点：
- 鼻肌收缩 → 鼻翼向上
- 提上唇肌收缩 → 上唇向上
- 两者协同 = 兔子皱

【鼻-嘴联动的三个关键点】

Point 1：鼻翼翕动
- 鼻肌收缩
- 鼻翼向上翘起
- 鼻梁出现横向皱纹

Point 2：上唇皱缩
- 提上唇肌收缩
- 上唇向上提升
- 像兔子露出门牙

Point 3：眉头皱起
- 皱眉肌收缩
- 眉头聚拢
- 眉间出现垂直皱纹
```

```
- 下唇：可能轻微前突
- 牙齿：可能露出

【厌恶皱眉微表情的"触发器"】

- AI指令：brows furrowed,眉头皱起, AU4, frown,眉间皱纹
- 效果：厌恶的眉部特征

Trigger 5：鼻梁皱纹
- AI指令：transverse nasal wrinkles, procerus contracted,鼻梁皱纹
- 效果：厌恶的鼻梁特征
```

### 厌恶皱眉微表情的"情绪梯度"

```
【厌恶皱眉的情绪梯度设计】

- 可见：完全兔子皱 + 面部扭曲
- 情绪：强烈厌恶
- 场景：极端臭味或事物

Gradient 5：作呕
- 肌肉：所有厌恶肌肉 + 身体反应

- 类型：视觉厌恶

场景3：道德厌恶
- 类型：精神厌恶
- 肌肉：AU4 + AU9
- 可见：眉头皱起 + 鼻梁皱纹
```
