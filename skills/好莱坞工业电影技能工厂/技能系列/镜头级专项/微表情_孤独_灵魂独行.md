## 技能概述

**交叉维度**: 微表情 × 孤独 × 灵魂独行 × 肌肉级
**定位**: 这是"灵魂独行"的微表情系统——当人物经历深层的孤独感时，灵魂仿佛在人群中行走却无人回应。面部肌肉呈现一种"向内收缩"的状态，眼角微微下垂，嘴角有细微的拉力，眼神空洞而深远，仿佛看向一个无人能达到的内心世界。这是存在主义孤独的终极表达。
**核心功能**: 提供孤独感微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的孤独提示词

## 第一部分：孤独感的科学基础

### 什么是"灵魂独行"？——存在主义孤独的神经科学

```
【孤独感的心理学机制】

孤独 ≠ 寂寞（Loneliness ≠ Being Alone）

核心区别：
- 寂寞：物理上独处，希望有人陪伴
- 孤独：灵魂上无人理解，即使在人群中
- 存在主义孤独：意识到自我与他人的根本分离

【神经科学机制】

社会排斥脑区（Social Rejection Brain Regions）：
- 前扣带回皮层：与社会疼痛相同区域
- 孤独感 = 真实的身体疼痛
- 独处时大脑不区分社会排斥和物理疼痛

社会需求未满足：
- 依恋需求：人类进化需要社会连接
- 孤独感：当需求未满足时的信号
- 慢性孤独：持续的社会需求受挫

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU43: 眼睛变得"空洞"（灵魂不在）
- 眼睑轻微下垂（能量内收）
- 组合：AU43+轻微下垂=孤独的眼睛

眉毛（Eyebrows）：
- AU1: 内侧轻微提升（悲伤的痕迹）
- 眉毛间距微微扩大（"孤独的眉毛"）

嘴部（Lips）：
- AU17: 下巴轻微提升（压抑的情绪）
- AU15: 嘴角轻微下压（但不完全是悲伤）
- 组合：AU17+AU15=孤独的嘴

整体：
- 所有肌肉：向内收缩
- 面部：呈现"保护内部"的状态

【孤独感的三阶段】

Stage 1：表面孤独（人群中的孤独）
- 特点：物理上有他人，但情感上隔离
- 微表情：轻微的空洞
- 持续：长期

Stage 2：深层孤独（灵魂的独行）
- 特点：意识到自我的根本分离
- 微表情：明显的空洞
- 持续：可能永久

Stage 3：终极孤独（存在的荒原）
- 特点：人类处境的根本孤独
- 微表情：完全的空洞
- 持续：存在性的

【典型情境】

1. 在喧闹派对的角落独自站着
2. 家庭聚会中感到完全孤独
3. 深夜的城市中独自行走
4. 亲密关系中感到无法被理解
5. 失去挚爱后的孤独
6. 老年人的慢性孤独
7. 移民在新国家的文化孤独
8. 成功人士高处不胜寒的孤独
```

## 第三部分：情绪梯度设计

### 孤独感的五级梯度

```
【孤独感的情绪梯度】

Level 1：轻微孤独（10%强度）
- 特点：开始感到不被理解
- 微表情：轻微的空洞
- 眼神：开始看向内部

Level 2：明显孤独（30%强度）
- 特点：明显感到与他人隔离
- 微表情：明显的空洞
- 身体：开始内收

Level 3：深层孤独（60%强度）
- 特点：灵魂独行的感觉
- 微表情：深层的空洞
- 身体：完全内收

Level 4：极度孤独（85%强度）
- 特点：存在主义的荒原
- 微表情：完全的空洞
- 身体：仿佛在人群中消失

Level 5：终极孤独（100%强度）
- 特点：人类处境的根本孤独
- 微表情：灵魂出窍般的空洞
- 身体：仿佛不存在于这个世界

【孤独感的三型】

Type 1：人群中的孤独（Social Loneliness）
- 特点：物理上有他人，但情感上隔离
- 场景：派对、聚会、家庭

Type 2：失去的孤独（Grief Loneliness）
- 特点：失去挚爱后的孤独
- 场景：丧亲、分手、离别

Type 3：存在的孤独（Existential Loneliness）
- 特点：意识到自我的根本分离
- 场景：任何情境，任何时间
```
