**交叉维度**: 微表情 × 凝视 × 沉默对峙 × 肌肉级
**定位**: 这是"沉默对峙"的微表情系统——当两个人物处于对峙状态时，不使用语言，而是通过眼神的较量来传达权力、挑战、威胁或恐惧。两个凝视在空中交锋，无声但激烈。这是人类最原始的权力展示方式之一，跨越文化。
**核心功能**: 提供沉默对峙微表情的肌肉级科学解析、FACS编码、眼神设计、以及AI视频生成的对峙提示词

## 第一部分：沉默对峙的科学基础

### 什么是"沉默对峙"？——眼神的权力游戏

```
【沉默对峙的心理学机制】

沉默对峙 = 非语言权力展示（Non-verbal Power Display）

核心状态：
- 两个个体之间的眼神较量
- 权力、地位、意图的无声传达
- 战斗或逃跑前的最后确认
- 威慑与反威慑的博弈

【眼神对峳的生物学基础】

生物学背景：
- 灵长类动物的眼神威胁是进化产物
- 直视眼睛=挑战=威胁
- 眼神接触触发杏仁核（恐惧中枢）
- 眼神回避=屈服=顺从

权力动态：
- 上位者：直视、占据空间
- 下位者：回避、收缩空间
- 对峳：两个上位者（或假装上位者）的碰撞

【FACS肌肉级解析】

攻击者眼神（Aggressor Gaze）：
- AU5: 眼睛极度张开（上睑提肌收缩）
- AU7: 轻微眼睑下垂（部分关闭，制造威胁感）
- 组合：AU5+AU7=威胁性凝视
- 瞳孔：可能收缩（愤怒时）或放大（兴奋时）
- 眉毛：可能下压（AU4），增加威胁感

防御者眼神（Defender Gaze）：
- AU5: 眼睛张大（但不像攻击者那样极端）
- AU1+AU2: 眉毛提升（恐惧+警觉）
- AU4: 眉毛聚拢（但不是完全的威胁）
- 瞳孔：可能放大（恐惧）
- 组合：警觉+恐惧+防御

对峙者眼神（Standoff Gaze）：
- AU5: 固定张开
- AU7: 轻微眼睑下垂
- AU1+AU2: 固定提升
- AU4: 固定聚拢
- 瞳孔：收缩（极端控制状态）
- 组合：愤怒+恐惧+控制+决心

【沉默对峙的三阶段】

Stage 1：初始对峙（接触阶段）
- 眼神：第一次接触
- 微表情：评估对方
- 持续：1-3秒

Stage 2：权力较量（交锋阶段）
- 眼神：激烈的无声交锋
- 微表情：双方都在展示力量
- 持续：3-30秒

Stage 3：结果（决定阶段）
- 眼神：一方的眼神退让或双方同时退让
- 微表情：显示谁赢得对峙
- 持续：1-5秒

【典型情境】

1. 酒吧里两个男人看上了同一个女人
2. 帮派对峙时的眼神交锋
3. 审讯室里警察vs嫌疑人
4. 西部片里的枪手对决
5. 办公室政治中的无声较量
6. 夫妻冷战中不说话的凝视
7. 体育比赛中的对手对视
```

## 第二部分：眼神设计系统

### 沉默对峙的眼神设计

```
【沉默对峙的眼神七型】

Type 1：杀手凝视（Killer's Stare）
- 特点：眼睛微闭，直视，瞳孔收缩
- 传达：我是危险的，不要逼我
- 情境：帮派、杀手、危险人物
- 肌肉：AU7B-C + 瞳孔收缩

Type 2：权威凝视（Authoritative Stare）
- 特点：眼睛张开，眉毛提升，直视
- 传达：我是老板，你必须服从
- 情境：老板、警官、父母
- 肌肉：AU5 + AU1+AU2

Type 3：挑战凝视（Challenger's Stare）
- 特点：眼睛张开，眉毛聚拢，直视
- 传达：我不怕你，你不能强迫我
- 情境：对峳、竞争、权力斗争
- 肌肉：AU5 + AU4

Type 4：轻蔑凝视（Contemptuous Stare）
- 特点：眼睛半闭，嘴角上扬，一眼瞥视
- 传达：你不够格
- 情境：上级对下级，轻蔑
- 肌肉：AU7 + AU12 + 嘴角上扬

Type 5：恐惧凝视（Fearful Stare）
- 特点：眼睛张大，眉毛提升，瞳孔放大
- 传达：我很害怕但我不退让
- 情境：恐惧但决心面对
- 肌肉：AU5 + AU1+AU2 + 瞳孔放大

Type 6：死亡凝视（Deadly Stare）
- 特点：眼睛固定张大，眉毛固定聚拢，瞳孔固定收缩
- 传达：我已经决定要杀你
- 情境：最终对决，致命意图
- 肌肉：AU5C + AU1+AU2C + AU4C + 瞳孔收缩

Type 7：冷对峙（Cold Stare）
- 特点：眼睛微闭，眉毛平直，无表情
- 传达：你对我不重要，我不关心你
- 情境：冷酷对决，情感隔离
- 肌肉：AU7 + 眉毛平直

【眼神接触的微参数】

┌─────────────────────────────────────────────────────────────┐
│ 参数项                │ 推荐值              │ 说明           │
├─────────────────────────────────────────────────────────────┤
│ 眼神接触时长          │ 3-30秒              │ 持续对视       │
│ 眼神移动              │ 极少                 │ 保持固定       │
│ 眨眼频率              │ 0-3次/10秒         │ 极少眨眼       │
│ 眼皮下垂程度          │ 10-30%              │ 轻微下垂       │
│ 巩膜可见度            │ 80-100%             │ 大量暴露       │
│ 瞳孔大小              │ 收缩（20-30%）或放大│ 控制或恐惧     │
└─────────────────────────────────────────────────────────────┘
```

## 第三部分：情绪梯度设计

### 沉默对峙的五级梯度

```
【沉默对峙的情绪梯度】

Level 1：轻微不适（10%强度）
- 眼神：轻微接触，快速移开
- 微表情：评估+轻微警惕
- 持续：1-3秒
- 退出：快速移开眼神

Level 2：明显挑衅（30%强度）
- 眼神：持续接触，但不极端
- 微表情：眉毛轻微聚拢
- 持续：3-10秒
- 退出：一方先移开

Level 3：真正对峙（60%强度）
- 眼神：激烈交锋
- 微表情：AU5+AU4激活
- 持续：10-30秒
- 退出：一方明显退让

Level 4：极度紧张（85%强度）
- 眼神：最大强度交锋
- 微表情：AU5C+AU4C
- 持续：30-60秒
- 退出：一方可能动手

Level 5：致命对峙（100%强度）
- 眼神：死亡凝视
- 微表情：完全激活
- 持续：直到行动
- 退出：动手或完全屈服
```
