## 技能概述

**交叉维度**: 微表情 × 孤独 × 无人回应的寂静 × 肌肉级
**定位**: 这是"孤独"的微表情系统——当人物站在喧嚣的背景中，却完全被忽视。没有人回应，没有人注意到，没有人在乎。面部表情是一种"透明的消失"状态，嘴角可能微微下垂，眼睛可能空洞无神，整个人仿佛是一个幽灵。这是电影中最令人心碎的孤独时刻，每一个面部肌肉都在无声地呼喊着"请看见我"。
**核心功能**: 提供孤独时刻的肌肉级科学解析、FACS编码、透明感技巧、以及AI视频生成的孤独提示词

## 第一部分：孤独的科学基础

### 什么是"无人回应的寂静"？——孤独的神经机制

```
【孤独的心理学机制】

孤独时刻 = 被忽视 + 被遗忘 + 无人在乎 + 透明存在

核心状态：
- 在人群中但完全孤独
- 呼救但无人回应
- 存在但被无视
- 想要连接但无法连接

【孤独的文化隐喻】

孤独的象征意义：
- 透明的存在
- 被遗忘的灵魂
- 喧嚣中的沉默
- 连接失败的痛苦
- 存在性孤独

孤独时刻的特殊性：
- 在社交场景中
- 有机会但无能力
- 想要但不敢
- 渴望但压抑

【FACS肌肉级解析】

眼睑（Eyelids）：
- 眼睑：轻微下垂（活力降低）
- 眨眼频率：减慢或停止（缺乏刺激）
- 眼神：空洞/涣散（内转）

眉毛（Eyebrows）：
- AU1: 内侧轻微提升（悲伤/失落）
- AU4: 轻微下降（疲惫/无力）
- 整体：平静但低能量

眼眶（Orbital）：
- 眼眶轮匝肌：几乎不激活（缺乏情绪波动）
- 眼神：看向虚空/不聚焦

嘴角（Lips）：
- AU15: 嘴角轻微下沉（失落）
- AU17: 轻微下巴提升（嘟嘴/委屈）
- 整体：几乎无表情

整体：
- 面部：缺乏生气
- 对比：周围喧嚣 vs 内心寂静
- 透明感：仿佛不存在

【孤独时刻的两型】

Type 1：社交失败孤独
- 特点：尝试连接但失败
- 表现：眼神游移、嘴角下垂、轻微收缩
- 场景：派对中被忽视、聚会中独坐

Type 2：存在性孤独
- 特点：深层的生命孤独感
- 表现：眼神空洞、面部无表情、灵魂出窍
- 场景：深夜独坐、空虚的日常生活

【典型情境】

1. 派对中站在角落无人问津
2. 餐厅里独自用餐无人理会
3. 聚会照片中被裁掉
4. 电梯里被忽视的问候
5. 群聊中被遗忘的消息
6. 机场独自候机的身影
7. 地铁站台上孤独的等路人
8. 生日那天无人祝福
```

## 第三部分：孤独类型设计

### 孤独时刻的五型设计

```
【孤独时刻的五型设计】

Type 1：派对孤独
- 特点：在热闹派对中完全孤独
- 情绪：失落、被排斥、自我怀疑
- 场景：站在角落、无人问津
- 代表：《壁花少年》的派对场景

Type 2：餐桌孤独
- 特点：餐厅中独自用餐
- 情绪：尴尬、接受、倔强
- 场景：一人桌、周围是成双成对
- 代表：《美食、祈祷和恋爱》

Type 3：群聊孤独
- 特点：社交媒体上被忽视
- 情绪：焦虑、被遗忘、自我怀疑
- 场景：看着手机、等待回复
- 代表：《黑镜》系列的孤独科技

Type 4：生日孤独
- 特点：生日那天无人祝福
- 情绪：心碎、接受、倔强
- 场景：独自吹蜡烛、无人记起
- 代表：《被嫌弃的松子的一生》

Type 5：临终孤独
- 特点：生命的最后时刻无人陪伴
- 情绪：接受、恐惧、平静
- 场景：病房、最后一口气
- 代表：《爱》中临终告别

【五级孤独强度设计】

Level 1：轻微忽视（10%强度）
- 特点：偶尔被忽视
- 微表情：轻微失落
- 氛围：可以忍受

Level 2：明显忽视（30%强度）
- 特点：持续被忽视
- 微表情：明显失落
- 氛围：开始难受

Level 3：严重忽视（60%强度）
- 特点：完全被忽视
- 微表情：空洞+下沉
- 氛围：痛苦

Level 4：极端孤独（85%强度）
- 特点：彻底被遗忘
- 微表情：透明感
- 氛围：心碎

Level 5：存在性孤独（100%强度）
- 特点：深层的存在孤独
- 微表情：灵魂出窍
- 氛围：超脱
```
