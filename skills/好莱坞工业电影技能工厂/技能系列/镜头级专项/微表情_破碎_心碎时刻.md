## 技能概述

**交叉维度**: 微表情 × 破碎 × 心碎时刻 × 肌肉级
**定位**: 这是"心碎时刻"的微表情系统——当人物经历极端的情感创伤（如失恋、失去亲人、背叛）时，会产生一种"心碎"的主观体验。这种体验伴随着真实的生理疼痛反应（大脑成像显示心碎激活与身体疼痛相同的脑区），微表情呈现出一种"破碎"的、无法完全控制的状态。
**核心功能**: 提供心碎时刻微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的心碎提示词

## 第一部分：心碎时刻的科学基础

### 什么是"心碎时刻"？——情感疼痛的神经科学

```
【心碎时刻的神经科学机制】

心碎 = 社会性疼痛（Social Pain）= 真实的生理疼痛

科学发现：
- 心碎激活的大脑区域与身体疼痛相同（前扣带回皮层）
- 社会排斥=真实的疼痛体验
- 情感创伤=神经系统的"受伤"
- 心碎可以导致"章鱼壶心肌病"（应激性心肌病）

【心理学机制】

心碎阶段：
1. 否认（不相信）
2. 震惊（无法处理）
3. 否认崩溃（现实进入）
4. 情绪崩溃（完全的悲伤）
5. 接受（缓慢的恢复）

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU6+AU7: 眼睑收缩+提升（哭泣前的特征）
- AU43: 眼睛紧闭（眼泪即将流出）
- 组合：AU6+AU7+AU43=心碎的眼睛

眉毛（Eyebrows）：
- AU1: 内侧提升（悲伤的独特标志）
- 组合：AU1+AU43=心碎的眉毛

嘴部（Lips）：
- AU15: 嘴角向下压（强烈悲伤）
- AU17: 下巴提升（哭泣时的特征）
- 组合：AU15+AU17=心碎的嘴

眼泪系统：
- 泪腺激活：LR4+LR7（悲伤眼泪）
- 眼泪类型：情绪性眼泪（与反射性眼泪化学成分不同）
- 眼泪模式：先涓涓细流→后大量涌出

【心碎时刻的两阶段】

Stage 1：冲击阶段（冲击期）
- 微表情：震惊、否认、不相信
- 时间：数秒到数分钟
- 特点：面部"冻结"或"空白"

Stage 2：崩溃阶段（崩溃期）
- 微表情：完全的情绪崩溃
- 时间：数分钟到数小时
- 特点：无法控制的哭泣

【典型情境】

1. 收到分手短信的瞬间
2. 得知爱人去世的消息
3. 发现伴侣背叛
4. 目睹关系结束的最后一刻
5. 孩子的离去
6. 宠物的死亡
7. 多年友谊的破裂
```

## 第三部分：情绪梯度设计

### 心碎时刻的五级梯度

```
【心碎时刻的情绪梯度】

Level 1：冲击（10%强度）
- 微表情：震惊、空白
- 眼泪：无或极少
- 身体：冻结
- 触发：刚刚得知消息
- 持续：数秒到数分钟

Level 2：否认（30%强度）
- 微表情：不相信的表情
- 眼泪：开始涓涓细流
- 身体：轻微颤抖
- 触发：试图否认现实
- 持续：数分钟

Level 3：崩溃（60%强度）
- 微表情：完全的悲伤
- 眼泪：涓涓细流变涌出
- 身体：明显的颤抖
- 触发：现实进入
- 持续：数分钟到数小时

Level 4：高峰（85%强度）
- 微表情：无法控制的哭泣
- 眼泪：大量涌出
- 身体：剧烈颤抖
- 触发：情绪完全崩溃
- 持续：数分钟到数小时

Level 5：耗尽（100%强度）
- 微表情：虚弱、无力
- 眼泪：减少或停止
- 身体：疲惫、耗尽
- 触发：情绪耗尽
- 持续：直到休息
```
