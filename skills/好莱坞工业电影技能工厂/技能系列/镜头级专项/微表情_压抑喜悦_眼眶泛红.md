## 技能概述

**交叉维度**: 微表情 × 情绪变体 × 压抑喜悦/眼眶泛红
**定位**: 这是"压抑喜悦"变体的微表情系统——这是幸福和悲伤的复合体，是"喜极而泣"或"克制的幸福"。微妙的眼轮匝肌收缩（杜兴微笑）+ 眼眶泛红或含泪 + 嘴角上扬但可能轻微颤抖的协同运动。压抑喜悦的微表情是"幸福的溢出"——当幸福太多，眼睛成为泄洪口。
**核心功能**: 提供压抑喜悦的完整FACS肌肉运动图谱、眼眶泛红与含泪的触发器、幸福与压抑喜悦的区分、以及AI视频生成的精确提示词

## 第一部分：压抑喜悦微表情的科学基础

### 什么是"压抑喜悦"？——FACS层面的精确解析

```
【压抑喜悦的微表情科学】

"压抑喜悦" = 幸福溢出 + 悲伤保护

压抑喜悦的肌肉运动（复合）：
- 颧大肌（zygomaticus major）收缩 → 嘴角上扬（幸福）
- 眼轮匝肌（orbicularis oculi）收缩 → 眼睛变小（幸福杜兴）
- 眼轮匝肌内侧 + 皱眉肌 → 眼眶泛红（悲伤/感动）
- 眼轮匝肌 → 含泪（幸福太多）

【压抑喜悦的独特组合】

为什么压抑喜悦是独特的？

组合1：幸福 + 感动
- 颧大肌（嘴角）+ 眼轮匝肌（眼）+ 泪腺激活
- 结果：笑中带泪

组合2：幸福 + 克制的悲伤
- 颧大肌（嘴角）+ 皱眉肌（眉）+ 眼眶变化
- 结果：眼眶泛红

【FACS编码】
- AU12（颧大肌）→ 嘴角向上
- AU6（眼轮匝肌）→ 眼尾皱纹
- 泪腺激活 → 眼眶泛红或含泪
- 皱眉肌轻微 → 克制的情感

【压抑喜悦 vs 幸福 vs 悲伤】

压抑喜悦的独特之处：
- 幸福：AU6 + AU12（对称微笑）
- 悲伤：AU1 + AU4 + AU15（倒八字眉 + 嘴角下垂）
- 压抑喜悦：AU6 + AU12 + 眼眶泛红/含泪（嘴角上扬 + 眼周变化）

【压抑喜悦的微表情分级】

Level 1：含泪的微笑
- 肌肉：AU6 + AU12 + 轻度泪腺激活
- 可见：微笑 + 眼眶湿润

Level 2：喜极而泣
- 肌肉：AU6 + AU12 + 中度泪腺激活
- 可见：微笑 + 眼泪流出

Level 3：眼眶泛红
- 肌肉：AU6 + AU12 + 皱眉肌轻微 + 眼眶变化
- 可见：微笑 + 眼眶泛红 + 可能含泪

Level 4：哽咽微笑
- 肌肉：AU6 + AU12 + 眼眶变化 + 喉部紧张
- 可见：微笑 + 眼眶泛红 + 哽咽

Level 5：情感崩溃的微笑
- 肌肉：所有幸福肌肉 + 大量眼泪
- 可见：完全微笑 + 眼泪 + 可能颤抖
```

### 压抑喜悦微表情的"眼眶泛红"

```
【压抑喜悦微表情的核心：眼眶泛红】

为什么眼眶是压抑喜悦的关键？

眼眶区域的变化：
- 眼轮匝肌：收缩（杜兴微笑）
- 泪腺：激活（幸福太多）
- 眼眶：血管扩张 → 泛红

【眼眶泛红的三个关键点】

Point 1：眼眶湿润
- 泪腺轻微激活
- 眼眶区域看起来湿润
- 是"喜极而泣"的前兆

Point 2：眼眶泛红
- 血管轻微扩张
- 眼眶周围呈现微红色
- 是"感动"的可见信号

Point 3：含泪微笑
- 眼泪在眼眶中
- 嘴角上扬
- 眼泪 + 微笑 = 压抑喜悦
```

## 第二部分：压抑喜悦微表情的技术执行

### 压抑喜悦微表情的AI生成参数

```
【压抑喜悦微表情的AI生成系统】

眼眶层：
- 眼轮匝肌：收缩（杜兴微笑）
- 泪腺：激活（眼眶湿润）
- 血管：轻微扩张（眼眶泛红）

【AI生成的"压抑喜悦"面部参数】

眉毛：
- 形状：正常或轻微弯曲
- 可能的轻微提升（幸福）
- 错误：明显下降（那是悲伤）

眼睛：
- 上眼睑：轻微下垂（幸福/满足）
- 下眼睑：可能轻微提升
- 眼眶：泛红或含泪
- 瞳孔：可能放大（情感）
- 错误：睁大（那是惊讶）

嘴部：
- 嘴角：向上（幸福/AU12）
- 可能轻微颤抖
- 可能不完全张开（克制）

脸颊：
- 提升（颧大肌）
- 可能对称

【压抑喜悦微表情的"触发器"】

Trigger 1：眼眶泛红
- AI指令：eyes reddened, eye corners red, glistening eyes, teary eyes,眼眶泛红
- 效果：压抑喜悦的决定性标记

Trigger 2：含泪微笑
- AI指令：smile with tears, tearful smile, happy tears, smiling through tears,含泪微笑
- 效果：压抑喜悦的经典表达

Trigger 3：杜兴微笑
- AI指令：Duchenne smile, orbicularis oculi activated, crow's feet, genuine smile,杜兴微笑
- 效果：幸福的肌肉基础

Trigger 4：嘴角颤抖
- AI指令：slight tremor in smile, trembling smile,嘴角颤抖,控制情绪
- 效果：压抑情感的信号

Trigger 5：眼眶湿润
- AI指令：glistening eyes, moist eyes, watery eyes,眼眶湿润
- 效果：喜极而泣的前兆
```
