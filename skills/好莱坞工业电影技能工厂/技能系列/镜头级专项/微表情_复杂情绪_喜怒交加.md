## 技能概述

**交叉维度**: 微表情 × 复合情绪 × 喜怒交加/复杂情绪
**定位**: 这是"复杂情绪"变体的微表情系统——现实中的情绪很少是纯粹的，经典案例是"喜怒交加"：嘴角上扬（幸福AU12）的同时眉毛却呈金字塔形（悲伤AU1+AU4）。这是两种对立情绪同时激活的肌肉博弈。复杂情绪的微表情是"人性的真实"——真实的人很少只感受一种情绪。
**核心功能**: 提供复杂情绪的完整FACS肌肉运动图谱、矛盾肌肉的触发器、单一情绪与复杂情绪的区分、以及AI视频生成的精确提示词

## 第一部分：复杂情绪微表情的科学基础

### 什么是"复杂情绪"？——FACS层面的精确解析

```
【复杂情绪的微表情科学】

"复杂情绪" = 两种对立情绪同时激活

复杂情绪的肌肉运动（矛盾）：
- 颧大肌（zygomaticus major）→ 嘴角上扬（幸福）
- 皱眉肌（corrugator supercilii）+ 额肌 → 眉毛呈金字塔形（悲伤）
- 结果：微笑 + 金字塔眉 = 苦笑/喜怒交加

【喜怒交加的矛盾】

为什么喜怒交加是可能的？

矛盾1：嘴角向上（幸福）
- 颧大肌收缩
- 嘴角向上向后
- 是"我想笑"的冲动

矛盾2：眉毛向下（悲伤）
- 皱眉肌收缩
- 眉间向下
- 是"我想哭"的冲动

矛盾3：肌肉博弈
- 两种冲动同时激活
- 面部出现"不协调"
- 是"复杂情绪"的信号

【FACS编码】
- AU12（颧大肌）→ 嘴角向上
- AU1+AU4（额肌+皱眉肌）→ 金字塔眉
- AU6+AU7（眼轮匝肌）→ 眼尾皱纹（可能）

【复杂情绪的经典组合】

Combo 1：苦笑
- 幸福（AU12）+ 悲伤（AU1+AU4）
- 嘴角上扬 + 金字塔眉

Combo 2：喜恶交加
- 幸福（AU12）+ 厌恶（AU9+AU10）
- 嘴角上扬 + 兔子皱

Combo 3：怒喜交加
- 幸福（AU12）+ 愤怒（AU4）
- 嘴角上扬 + 眉间皱纹

Combo 4：惊恐交加
- 恐惧（AU1+AU2+AU5）+ 惊讶（AU1+AU2+AU5）
- 两者相似，可能增强

Combo 5：悲愤交加
- 悲伤（AU1+AU4+AU15）+ 愤怒（AU4+AU7）
- 金字塔眉 + 眼睑收紧

【复杂情绪的微表情分级】

Level 1：轻微矛盾
- 肌肉：两种情绪轻度激活
- 可见：面部轻微不协调

Level 2：明显矛盾
- 肌肉：两种情绪中等激活
- 可见：明显对立表情

Level 3：强烈矛盾
- 肌肉：两种情绪强烈激活
- 可见：剧烈面部冲突

Level 4：崩溃边缘
- 肌肉：一种情绪占上风
- 可见：情绪即将爆发

Level 5：情绪崩溃
- 肌肉：完全被一种情绪接管
- 可见：无法维持复杂状态
```

### 复杂情绪微表情的"矛盾信号"

```
【复杂情绪微表情的核心：矛盾信号】

为什么"矛盾"是复杂情绪的关键？

矛盾的特点：
- 两种情绪同时出现
- 面部肌肉相互"对抗"
- 结果是"不协调"的表情

【矛盾信号的三个关键点】

Point 1：嘴角-眉毛矛盾
- 嘴角向上（幸福）+ 眉毛向下（悲伤）
- 微笑 + 金字塔眉 = 苦笑

Point 2：眼睛-嘴巴矛盾
- 眼睛睁大（惊讶/恐惧）+ 嘴角上扬（幸福）
- 瞪大 + 微笑 = 惊讶的喜悦

Point 3：整体不协调
- 面部的不同区域表达不同情绪
- 结果是"复杂"的表情
```

## 第二部分：复杂情绪微表情的技术执行

### 复杂情绪微表情的AI生成参数

```
【复杂情绪微表情的AI生成系统】

矛盾层：
- 嘴角：上扬（幸福）+ 下压（悲伤）= 苦笑
- 眉毛：提升（惊讶）+ 下降（悲伤）= 复杂
- 眼睛：睁大（恐惧）+ 眼尾皱纹（幸福）= 复杂

【AI生成的"复杂情绪"面部参数】

眉毛：
- 一侧：可能提升（惊讶）
- 另一侧：可能下降（悲伤）
- 整体：不协调

眼睛：
- 可能同时有幸福眼轮匝肌和悲伤皱眉肌
- 可能同时有惊讶睁大和愤怒眼睑收紧
- 整体：混乱

嘴部：
- 嘴角：上扬（幸福）+ 下压（悲伤）= 苦笑
- 可能扭曲

【复杂情绪微表情的"触发器"】

Trigger 1：苦笑
- AI指令：bitter smile,苦笑, smiling with sad eyes,眼含泪的笑
- 效果：幸福+悲伤的组合

Trigger 2：假笑
- AI指令：forced smile,假笑, not reaching eyes,不达眼底的笑
- 效果：嘴角向上+眼轮匝肌不激活

Trigger 3：矛盾表情
- AI指令：conflicting signals,矛盾信号, mixed signals, mixed emotions
- 效果：复杂情绪的表达

Trigger 4：情绪波动
- AI指令：emotional fluctuation,情绪波动, shifting emotions,情绪转变
- 效果：复杂情绪的变化

Trigger 5：内心挣扎
- AI指令：internal conflict,内心挣扎, turmoil, emotional conflict
- 效果：复杂情绪的内在状态
```

## 第三部分：复杂情绪微表情的情绪设计

### 复杂情绪微表情的"情绪梯度"

```
【复杂情绪的情绪梯度设计】

Gradient 1：轻微矛盾（轻度复杂）
- 肌肉：两种情绪轻度激活
- 表情：轻微不协调
- 情绪：轻微复杂
- 场景：收到意外消息

Gradient 2：明显矛盾（中等复杂）
- 肌肉：两种情绪中等激活
- 表情：明显对立
- 情绪：明显复杂
- 场景：离别时的祝福

Gradient 3：强烈矛盾（强烈复杂）
- 肌肉：两种情绪强烈激活
- 表情：剧烈冲突
- 情绪：强烈复杂
- 场景：亲人去世但有遗产

Gradient 4：崩溃边缘（极度复杂）
- 肌肉：一种情绪开始占上风
- 表情：即将统一
- 情绪：即将崩溃
- 场景：极度压力下

Gradient 5：情绪崩溃（完全复杂）
- 肌肉：完全被一种情绪接管
- 表情：统一为一种情绪
- 情绪：崩溃
- 场景：极限突破

【复杂情绪微表情的"经典场景"】

场景1：离别时的祝福
- 类型：喜悲交加
- 肌肉：嘴角向上（幸福）+ 眉毛向下（悲伤）
- 表情：苦笑
- 适配：送别、告别

场景2：意外遗产
- 类型：喜惧交加
- 肌肉：嘴角向上（幸福）+ 眼睛睁大（惊讶）
- 表情：惊喜
- 适配：意外好消息

场景3：竞争对手成功
- 类型：悲愤交加
- 肌肉：嘴角向下（悲伤）+ 眉间皱纹（愤怒）
- 表情：嫉妒
- 适配：职场竞争
```
