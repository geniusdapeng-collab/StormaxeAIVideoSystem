## 技能概述

**交叉维度**: 微表情 × 闪电 × 瞬间启示 × 肌肉级
**定位**: 这是"闪电"的微表情系统——当人物在黑暗中被闪电照亮面部的瞬间，面部肌肉因为强烈的光线刺激和意识冲击而产生短暂的"冻结"反应。闪电作为宇宙级的视觉隐喻，代表命运的裁决、真相的揭露、顿悟的降临。面部表情在那一瞬间被永久定格，成为电影史上最具冲击力的画面之一。
**核心功能**: 提供闪电时刻的肌肉级科学解析、FACS编码、瞬间捕捉技巧、以及AI视频生成的闪电提示词

## 第一部分：闪电瞬间的科学基础

### 什么是"瞬间启示"？——闪电的光学与神经机制

```
【闪电瞬间的心理学机制】

闪电时刻 = 光线冲击 + 意识冻结 + 命运揭示

核心状态：
- 黑暗中的突然光明
- 面部被瞬间照亮
- 意识在那一刻冻结
- 命运的不可抗力

【闪电的文化隐喻】

闪电的象征意义：
- 命运的裁决
- 真相的揭露
- 神明的警告
- 顿悟的降临
- 毁灭与重生
- 力量与恐惧

闪电瞬间的特殊性：
- 极短（0.01-0.1秒）
- 极强（数十万勒克斯）
- 不可预测
- 不可控制

【FACS肌肉级解析】

眼睑（Eyelids）：
- 眼睑：瞬间收缩（对强光的本能反射）
- 眼轮匝肌：强力收缩
- 瞬目反射：完全抑制

眉毛（Eyebrows）：
- AU1+2: 内侧提升+外侧提升（惊讶的经典配置）
- 眉毛：可以处于各种位置（取决于闪电前后的情绪）

眼眶（Orbital）：
- 眼眶轮匝肌：强力收缩
- 这是"被闪电击中"的表情

嘴部（Lips）：
- AU26: 张开（倒吸一口凉气）
- AU25: 张开（震惊的嘴型）

整体：
- 面部：被闪电照亮
- 对比：黑暗中的光明
- 时间：冻结在那一刻

【闪电瞬间的两型】

Type 1：恐惧闪电
- 特点：闪电伴随恐惧
- 情绪：害怕、惊恐、敬畏
- 场景：暴风雨中的人物、面对神明

Type 2：顿悟闪电
- 特点：闪电伴随顿悟
- 情绪：震惊、明白、接受
- 场景：揭示真相、命运宣告、灵感降临

【典型情境】

1. 被闪电照亮的恐惧面孔
2. 黑暗中闪电揭示真相
3. 面对雷电的神罚
4. 顿悟时刻的闪电象征
5. 英雄在暴风雨中
6. 雷电下的孤独身影
7. 被闪电照亮的反派面孔
8. 闪电下的爱情宣言
```

## 第三部分：瞬间类型设计

### 闪电瞬间的五型设计

```
【闪电瞬间的五型设计】

Type 1：神罚闪电
- 特点：雷电象征神明惩罚
- 情绪：恐惧、敬畏、屈服
- 光线：冷白/蓝白闪电
- 色调：蓝色、银色

Type 2：命运宣告闪电
- 特点：闪电揭示不可改变的命运
- 情绪：震惊、接受、决心
- 光线：强烈背光
- 色调：金色、白色

Type 3：真相揭露闪电
- 特点：闪电照亮隐藏的真相
- 情绪：惊讶、明白、崩溃
- 光线：正面强光
- 色调：蓝白、银色

Type 4：顿悟闪电
- 特点：灵感或决定在闪电中降临
- 情绪：震惊、清晰、平静
- 光线：柔和背光
- 色调：暖色、金色

Type 5：毁灭闪电
- 特点：闪电代表毁灭或死亡
- 情绪：恐惧、绝望、接受
- 光线：极端对比
- 色调：深蓝、黑色

【五级闪电强度设计】

Level 1：微弱闪光（10%强度）
- 特点：远处的闪光
- 微表情：轻微惊讶
- 氛围：预感、紧张

Level 2：近处闪电（30%强度）
- 特点：可以看到闪电
- 微表情：明显惊讶
- 氛围：警惕、警觉

Level 3：直接击中（60%强度）
- 特点：闪电在附近
- 微表情：强烈惊讶+恐惧
- 氛围：震惊、恐惧

Level 4：极端闪电（85%强度）
- 特点：闪电就在眼前
- 微表情：极度恐惧/顿悟
- 氛围：超载、濒临崩溃

Level 5：命运闪电（100%强度）
- 特点：闪电完全笼罩
- 微表情：意识完全冻结
- 氛围：超越、超凡
```
