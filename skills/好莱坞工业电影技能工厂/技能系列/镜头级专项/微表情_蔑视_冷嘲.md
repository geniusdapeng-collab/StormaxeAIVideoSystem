## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 蔑视/冷嘲
**定位**: 这是"蔑视"情绪的微表情系统——基于FACS（面部动作编码系统），蔑视不是"厌恶"，是一侧嘴角独特地上扬和收缩（AU12/不对称的嘴角提升）+ 鼻子轻微哼气 + 眉毛可能一侧微微抬起的协同运动。蔑视的微表情是"道德优越感"的面部信号——不像厌恶那样扭曲面部，而是用"不屑"来表达"我比你高一等"。
**核心功能**: 提供蔑视的完整FACS肌肉运动图谱、不对称嘴角的触发器、蔑视与厌恶的区分、以及AI视频生成的精确提示词

## 第一部分：蔑视微表情的科学基础

### 什么是"蔑视"？——FACS层面的精确解析

```
【蔑视的微表情科学】

"蔑视" = 道德优越感的面部信号

蔑视的肌肉运动（必需）：
- 颧大肌（zygomaticus major）单侧收缩 → 一侧嘴角向上向后拉动
- 颧小肌（zygomaticus minor）可能参与 → 一侧嘴角提升
- 提上唇肌（levator labii superioris）可能轻微激活 → 鼻子可能哼气

【FACS编码】
- AU12（嘴角上扬）→ 一侧嘴角向上（通常左侧）
- AU14（嘴角收缩）→ 一侧嘴角收缩（嘴部不对称）
- 不对称性：蔑视是唯一"偏向"某一侧的基础情绪

【蔑视的"不对称微笑"】

蔑视的独特之处是"不对称"：
- 一侧嘴角：向上向后（AU12）
- 另一侧嘴角：保持平直或轻微下降
- 结果：一侧嘴角上扬 = "冷笑"或"歪笑"

【蔑视 vs 厌恶 vs 幸福】

蔑视、厌恶、幸福的嘴角：
- 蔑视：一侧嘴角上扬（不对称）
- 厌恶：两侧嘴角向下（AU15）
- 幸福：两侧嘴角向上（对称）

【蔑视的微表情分级】

Level 1：轻视（轻度蔑视）
- FACS：AU12轻度（单侧）
- 肌肉：单侧颧大肌
- 可见：一侧嘴角微微上扬

Level 2：鄙视（中等蔑视）
- FACS：AU12 + AU14（单侧）
- 肌肉：单侧颧大肌 + 颧小肌
- 可见：一侧嘴角上扬 + 鼻翼翕动

Level 3：蔑视（标准蔑视）
- FACS：AUC（单侧激活）
- 肌肉：单侧颧大肌 + 颧小肌 + 可能提上唇肌
- 可见：不对称冷笑 + 鼻子哼气

Level 4：嘲笑（强烈蔑视）
- FACS：AUC + 身体语言
- 肌肉：所有蔑视肌肉 + 眼神
- 可见：歪嘴笑 + 眼神俯视

Level 5：轻蔑（极强蔑视）
- FACS：AUC = 完全蔑视
- 肌肉：所有蔑视肌肉 + 头部动作
- 可见：歪嘴 + 头部后仰 + 眼神
```

### 蔑视微表情的"不对称冷笑"

```
【蔑视微表情的核心：不对称冷笑】

为什么"不对称"是蔑视的关键？

蔑视的独特信号：
- 嘴角：一侧向上（AU12）= "我不在乎"
- 不对称：只有一侧嘴角 = "你不配"
- 鼻子：可能哼气 = "切"

【不对称冷笑的三个关键点】

Point 1：单侧嘴角上扬
- 颧大肌单侧收缩 → 一侧嘴角向上向后
- 通常是左侧（部分研究显示）
- 但可以是任一侧

Point 2：鼻子哼气
- 提上唇肌轻微激活 → 鼻子哼气
- 鼻子可能皱起或扩张
- 表达"切"或"哼"

Point 3：眼神俯视
- 眉毛：可能一侧微微抬起
- 眼睛：俯视对方
- 头部：可能后仰
```

## 第二部分：蔑视微表情的技术执行

### 蔑视微表情的AI生成参数

```
【蔑视微表情的AI生成系统】

嘴角层：
- 单侧嘴角：向上向后（AU12）
- 对侧嘴角：平直或轻微下降
- 不对称：关键特征

【AI生成的"蔑视"面部参数】

眉毛：
- 一侧：可能微微抬起（轻微傲慢）
- 另一侧：相对平直
- 整体：不对称

眼睛：
- 眼神：俯视、轻蔑
- 眼睑：可能轻微下垂
- 瞳孔：正常

鼻部：
- 可能轻微哼气（提上唇肌）
- 鼻翼：可能轻微扩张

嘴部：
- 单侧嘴角：向上向后（AU12）
- 对侧嘴角：平直或下降
- 不对称微笑 = 蔑视

【蔑视微表情的"触发器"】

Trigger 1：单侧嘴角上扬
- AI指令：one corner of mouth raised, asymmetric smile, unilateral zygomaticus,嘴角一侧上扬
- 效果：蔑视的决定性特征

Trigger 2：不对称冷笑
- AI指令：smirk,歪嘴笑,歪嘴, one-sided curl, lopsided smile
- 效果：蔑视的嘴部表情

Trigger 3：鼻子哼气
- AI指令：nose huff, nostril flare, sniff of contempt,哼气,鼻子轻哼
- 效果：蔑视的呼吸特征

Trigger 4：眼神俯视
- AI指令：looking down nose,俯视眼神, contemptuous gaze,蔑视眼神
- 效果：蔑视的眼神表达

Trigger 5：眉毛不对称
- AI指令：one eyebrow slightly raised, asymmetric brows,一侧眉毛微抬
- 效果：蔑视的傲慢特征
```

## 第三部分：蔑视微表情的情绪设计

### 蔑视微表情的"情绪梯度"

```
【蔑视的情绪梯度设计】

Gradient 1：轻视（轻度蔑视）
- 肌肉：AU12轻度（单侧）
- 表情：一侧嘴角微微上扬
- 情绪：轻视、不屑、看不起
- 场景：听到愚蠢言论

Gradient 2：鄙视（中等蔑视）
- 肌肉：AU12 + AU14（单侧）
- 表情：不对称微笑 + 眼神俯视
- 情绪：鄙视、轻蔑、鄙夷
- 场景：看到低级行为

Gradient 3：蔑视（标准蔑视）
- 肌肉：单侧颧大肌 + 颧小肌 + 提上唇肌
- 表情：歪嘴冷笑 + 鼻子哼气
- 情绪：蔑视、嘲讽、讥笑
- 场景：面对不道德行为

Gradient 4：嘲笑（强烈蔑视）
- 肌肉：AUC + 眼神
- 表情：歪嘴笑 + 眼神俯视 + 眉毛
- 情绪：嘲笑、冷笑、轻蔑
- 场景：看到某人失败

Gradient 5：轻蔑（极强蔑视）
- 肌肉：所有蔑视肌肉 + 头部后仰
- 表情：歪嘴 + 头部后仰 + 俯视
- 情绪：极度轻蔑、自大、傲慢
- 场景：面对极度反感

【蔑视微表情的"场景适配"】

场景1：听到愚蠢言论
- 蔑视类型：轻视
- 肌肉：AU12轻度
- 表情：一侧嘴角微上扬
- 适配：听到无知言论

场景2：面对不道德
- 蔑视类型：道德蔑视
- 肌肉：AU12 + AU14 + 鼻子哼气
- 表情：歪嘴 + 哼气
- 适配：面对背叛、欺骗

场景3：嘲笑对手
- 蔑视类型：竞争蔑视
- 肌肉：歪嘴 + 眼神俯视
- 表情：歪嘴笑 + 俯视
- 适配：击败对手后的态度
```
