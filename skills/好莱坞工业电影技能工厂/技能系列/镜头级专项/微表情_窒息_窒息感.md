## 技能概述

**交叉维度**: 微表情 × 窒息 × 窒息感 × 肌肉级
**定位**: 这是"窒息感"的微表情系统——当人物经历极度紧张、压迫、焦虑或幽闭恐惧时，会产生一种主观的"无法呼吸"的感觉。这种窒息感不一定有物理原因，而是心理压力导致的生理反应。面部会呈现特定的缺氧模式，同时伴随着恐惧、绝望和求生的本能。
**核心功能**: 提供窒息感微表情的肌肉级科学解析、FACS编码、梯度设计、以及AI视频生成的窒息提示词

## 第一部分：窒息感的科学基础

### 什么是"窒息感"？——心理性缺氧的生理表现

```
【窒息感的心理学机制】

窒息感 = 心理性缺氧（Psychogenic Anoxia）

核心状态：
- 主观的"无法呼吸"感
- 心理压力导致的生理反应
- 非实际的缺氧但有缺氧表现
- 焦虑/恐惧/幽闭恐惧的身体化

【身体化机制】

身体化（Somatiation）：
- 心理焦虑转化为身体症状
- 呼吸系统是最常见的靶器官
- 主观窒息感≠实际缺氧
- 但表现与真实窒息相似

【FACS肌肉级解析】

眼睑（Eyelids）：
- AU2: 眉毛急剧提升（惊恐反应）
- AU5: 眼睛极度张大（恐慌）
- AU7: 眼睑颤抖（极度恐惧）
- 组合：AU2+AU5+AU7=窒息性恐惧

眉毛（Eyebrows）：
- AU1+AU2: 提升（惊恐的独特标志）
- AU4: 聚拢（痛苦）
- 组合：AU1+AU2+AU4=窒息性痛苦

嘴部（Lips）：
- AU25+AU26: 嘴唇张开，试图吸气
- AU27: 嘴张开到最大程度
- 组合：张开的嘴=窒息的挣扎

呼吸系统：
- 呼吸频率：急剧增加（过度换气）
- 呼吸深度：浅而快
- 呼吸模式：不规则，叹息
- 换气过度：导致头晕

【窒息感的三阶段】

Stage 1：初始恐慌（呼吸困难期）
- 呼吸：开始变得困难
- 微表情：开始出现窒息特征
- 持续：0.5-2分钟

Stage 2：高峰恐慌（完全窒息感）
- 呼吸：过度换气+窒息感
- 微表情：完全的窒息表现
- 持续：1-5分钟

Stage 3：崩溃/恢复（决定期）
- 呼吸：继续过度换气或崩溃
- 微表情：根据结果变化
- 持续：取决于情境

【典型情境】

1. 幽闭恐惧症患者被困在小空间
2. 焦虑症患者的恐慌发作
3. 面试前/演讲前的急性焦虑
4. 得知灾难性消息后的窒息
5. 高空恐惧症站在高处
6. 溺水感（虽然没有水）
7. 窒息式哭泣
```

## 第三部分：情绪梯度设计

### 窒息感的五级梯度

```
【窒息感的情绪梯度】

Level 1：轻微不适（10%强度）
- 呼吸：开始变得困难
- 微表情：轻微的嘴部动作
- 身体：轻微的肩膀提升
- 触发：压力开始增加
- 持续：1-3分钟

Level 2：明显焦虑（30%强度）
- 呼吸：明显加快
- 微表情：嘴部开始张开
- 身体：肩膀明显提升
- 触发：压力增加
- 持续：2-5分钟

Level 3：急性恐慌（60%强度）
- 呼吸：过度换气开始
- 微表情：张嘴+眼睛张大
- 身体：手可能抓向喉咙
- 触发：压力达到高峰
- 持续：3-10分钟

Level 4：极度恐慌（85%强度）
- 呼吸：严重的过度换气
- 微表情：完全的窒息表现
- 身体：无法控制的颤抖
- 触发：压力极限
- 持续：5-20分钟

Level 5：崩溃（100%强度）
- 呼吸：可能停止或崩溃
- 微表情：完全的恐慌
- 身体：可能失去意识
- 触发：超出承受极限
- 持续：直到外部介入
```
