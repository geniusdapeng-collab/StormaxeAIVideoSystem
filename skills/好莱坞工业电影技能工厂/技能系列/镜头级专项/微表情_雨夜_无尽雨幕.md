## 技能概述

**交叉维度**: 微表情 × 雨夜 × 无尽雨幕 × 肌肉级
**定位*: 这是"雨夜"的微表情系统——当人物站在无尽的雨中，面部被雨水打湿，眼神看向远方或天空，雨成为情绪的载体和隐喻。面部肌肉呈现一种"被冲刷"的状态，眼泪和雨水混在一起，无法分辨。雨的质感、湿漉漉的皮肤、水珠滑落——这是电影中最具诗意的孤独与重生时刻。
**核心功能*: 提供雨夜微表情的肌肉级科学解析、FACS编码、氛围设计、以及AI视频生成的雨夜提示词

## 第一部分：雨夜的科学基础

### 什么是"无尽雨幕"？——雨水的情感隐喻

```
【雨夜的心理学机制】

雨夜 = 自然的力量 + 情感的释放

核心状态：
- 雨水作为情感的载体
- 物理上的湿与内心情感的释放
- 自然的力量与人类的脆弱
- 雨的质感、湿漉漉的感觉

【雨的文化隐喻】

雨的象征意义：
- 悲伤与哭泣
- 净化与重生
- 孤独与沉思
- 爱情与浪漫
- 命运与无常

雨夜的特殊性：
- 黑暗增加私密感
- 雨声提供掩护
- 视觉上形成屏障
- 温度降低增加脆弱感

【FACS肌肉级解析】

眼睑（Eyelids）：
- 眼睑：被雨水打湿，轻轻下垂
- 眼睑上可能有水珠
- 眨眼时水珠被清除

眉毛（Eyebrows）：
- AU1: 内侧提升（悲伤或沉思）
- 被雨水打湿，重量增加

嘴部（Lips）：
- AU25: 张开（可能仰头感受雨）
- 可能有水珠在嘴唇上

整体：
- 皮肤：湿漉漉
- 脸上：水珠滑落
- 表情：沉浸在大自然中

【雨夜的两型】

Type 1：悲伤雨夜
- 特点：雨伴随悲伤
- 情绪：哭泣、净化
- 场景：失去、离别、失败

Type 2：重生雨夜
- 特点：雨伴随释放
- 情绪：解脱、平静
- 场景：接受、放下、新开始

【典型情境】

1. 站在雨中仰头感受
2. 雨中独自行走
3. 雨中等待
4. 雨中告别
5. 雨中的顿悟
6. 雨中的净化仪式
7. 雨中的爱情
8. 雨中面对命运
```

## 第三部分：氛围类型设计

### 雨夜的四型设计

```
【雨夜的四型设计】

Type 1：孤独雨夜
- 特点：独自站在雨中
- 情绪：悲伤、孤独、沉思
- 光线：暗淡街灯
- 色调：蓝色、灰色

Type 2：爱情雨夜
- 特点：在雨中与爱人在一起
- 情绪：浪漫、亲密、温暖
- 光线：温暖街灯
- 色调：暖色、对比

Type 3：净化雨夜
- 特点：接受雨水冲刷
- 情绪：释放、净化、新生
- 光线：月光、曙光
- 色调：银色、白色

Type 4：命运雨夜
- 特点：面对无法改变的命运
- 情绪：接受、平静、宿命
- 光线：闪光、闪电
- 色调：强烈对比

【五级雨量设计】

Level 1：细雨（10%强度）
- 特点：轻柔的雨丝
- 微表情：轻微湿意
- 氛围：沉思、忧郁

Level 2：小雨（30%强度）
- 特点：持续的细雨
- 微表情：明显湿意
- 氛围：忧愁、回忆

Level 3：中雨（60%强度）
- 特点：明显的降雨
- 微表情：湿透
- 氛围：悲伤、释放

Level 4：暴雨（85%强度）
- 特点：强烈的降雨
- 微表情：完全湿透
- 氛围：冲击、净化

Level 5：倾盆大雨（100%强度）
- 特点：极端的雨量
- 微表情：几乎睁不开眼
- 氛围：淹没、超载
```
