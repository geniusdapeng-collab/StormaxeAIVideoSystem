## 技能概述

**交叉维度**: 微表情 × 孤独 × 镜子里的陌生人 × 肌肉级
**定位**: 这是"镜子里的陌生人"的微表情系统——当人物看着镜子里的自己，突然产生一种强烈的陌生感。那张脸是谁？这个人是怎么变成这样的？镜子里的倒影不再是熟悉的自己，而是一个被生活、被时间、被孤独雕刻出来的陌生人。这种自我认同的断裂，是都市人最深层的存在危机。
**核心功能**: 提供镜子前自我陌生化时刻的肌肉级科学解析、FACS编码、解离技巧、以及AI视频生成的陌生人提示词

## 第一部分：镜子里的陌生人的科学基础

### 什么是"镜子里的陌生人"？——自我解离的神经机制

```
【镜子陌生感的心理学机制】

镜子陌生感 = 自我认同的断裂 + 生活的痕迹 + 时间的雕刻 + 存在的叩问

核心状态：
- 看着镜子里的自己，感到强烈陌生
- 质疑："这个人是我吗？"
- 意识到自己已经变了
- 对过去的自己感到疏离

心理学背景：
- 自我连续性断裂
- 人格面具的崩塌
- 生活的累积痕迹
- 孤独对自我的侵蚀

【镜子陌生感的文化隐喻】

文学与电影中的镜子意象：
- 《爱丽丝梦游仙境》的柴郡猫："我们都是疯子"
- 《镜子里的小女孩》的恐怖凝视
- 《搏击俱乐部》的自我对抗
- 《黑天鹅》的分裂与转化
- 《美国精神病人》的面具与人格

镜子作为自我认知的工具：
- 镜子是自我的外部投射
- 镜子倒了，自我就崩了
- 镜子里的自己是"他者"
- 镜子陌生感=自我异化

【FACS肌肉级解析】

眼睑（Eyelids）：
- 眼睑：轻微睁大（专注凝视）
- 眨眼：减少或停止（完全聚焦）
- 眼眶：轻微收缩（试图识别）

眉毛（Eyebrows）：
- AU1: 内侧提升（困惑）
- AU4: 下降（努力识别）
- 组合：皱眉+困惑的混合

眼眶（Orbital）：
- 眼轮匝肌：轻微收缩（眯眼/细看）
- 眼神：探索性凝视，像在看陌生人

嘴角（Lips）：
- AU12: 可能不对称（内心混乱外化）
- AU17: 下巴轻微下降（微微张嘴）
- 可能轻微歪斜

整体：
- 面部：专注、困惑、轻微恐惧
- 眼神：探索性，像在寻找熟悉感
- 表情：逐渐从困惑到确认陌生

【两型对比】

Type 1：突然觉醒型
- 特点：某一天突然发现镜子里的自己陌生
- 触发：生活事件、年龄、失败、失去
- 表现：震惊→否认→确认→接受
- 情绪：恐惧→失落→虚无

Type 2：渐进累积型
- 特点：逐渐发现自己越来越不像自己
- 触发：长期孤独、压力、重复生活
- 表现：模糊→清晰→确认→麻木
- 情绪：模糊→清晰→接受→超然

【典型情境】

1. 早晨洗漱时对着镜子，突然不认识自己
2. 试衣间里看到侧面的自己感到陌生
3. 商场橱窗的倒影让自己停下脚步
4. 浴室雾气散去后的镜子
5. 化妆/卸妆时发现面孔的陌生
6. 拍照后看照片，觉得不像自己
7. 医院/警局等特殊场所的镜子
8. 深夜独自面对镜子
```

## 第三部分：镜子里的陌生人的类型设计

### 五型设计

```
【镜子里的陌生人的五型设计】

Type 1：岁月痕迹型
- 特点：年龄/压力/生活的痕迹明显
- 情绪：惊讶→接受→感伤
- 细节：皱纹/白发/眼袋/憔悴
- 镜头：特写镜子→手指触碰→对比

Type 2：表情空洞型
- 特点：脸上没有表情，像戴面具
- 情绪：困惑→恐惧→麻木
- 细节：面无表情、眼神空洞
- 镜头：缓慢接近镜子→凝视→无表情

Type 3：身份断裂型
- 特点：完全不认识自己，感到恐惧
- 情绪：震惊→否认→崩溃
- 细节：扭曲的表情、不像自己
- 镜头：摇头/后退/双手捂脸

Type 4：角色扮演型
- 特点：镜子里的自己是"应该成为的人"
- 情绪：讽刺→虚无→讽刺
- 细节：社会面具vs真实自我
- 镜头：苦笑/歪头/审视

Type 5：双重存在型
- 特点：感觉镜子里外有两个自己
- 情绪：分裂→对抗→和解/崩溃
- 细节：眼神对视、自我对话
- 镜头：缓慢转头、眼神追踪

【五级陌生强度设计】

Level 1：轻微陌生（10%强度）
- 特点：觉得有点不像自己
- 微表情：轻微困惑
- 氛围：可以接受

Level 2：明显陌生（30%强度）
- 特点：明显感到陌生
- 微表情：困惑+轻微恐惧
- 氛围：不安

Level 3：强烈陌生（60%强度）
- 特点：完全不认识
- 微表情：震惊+后退
- 氛围：恐惧

Level 4：解离（85%强度）
- 特点：自我解离，感觉在看别人
- 微表情：空洞+超然
- 氛围：分裂

Level 5：身份崩溃（100%强度）
- 特点：完全失去自我认同
- 微表情：崩溃/疯狂/麻木
- 氛围：超现实
```
