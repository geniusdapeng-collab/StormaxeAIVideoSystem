## 技能概述

**交叉维度**: 微表情 × 基础情绪 × 悲伤/哀伤
**定位**: 这是"悲伤"情绪的微表情系统——基于FACS（面部动作编码系统），悲伤不是"哭"，是眼轮匝肌内侧收缩导致的"眉毛内角提升"（金字塔形眉毛）+ 眼睑下垂 + 嘴角下压的协同运动。悲伤的微表情是"静默的崩溃"——没有眼泪，只有面部肌肉的"重力下沉"。
**核心功能**: 提供悲伤的完整FACS肌肉运动图谱、眼周微表情触发器、无声悲伤与哭泣的区分、以及AI视频生成的精确提示词

## 第一部分：悲伤微表情的科学基础

### 什么是"悲伤"？——FACS层面的精确解析

```
【悲伤的微表情科学】

"悲伤" = 面部重力下沉 + 眼周收缩

悲伤的肌肉运动（必需）：
- 皱眉肌（corrugator supercilii）收缩 → 眉毛内角向上提升
- 眼轮匝肌（orbicularis oculi）内侧收缩 → 眼睑下垂
- 降眉间肌（procerus）收缩 → 眉毛内侧下压（形成金字塔）
- 降口角肌（depressor anguli oris）收缩 → 嘴角向下
- 下唇提升肌（depressor labii inferioris）激活 → 下唇微微抬起

【FACS编码】
- AU1（眉毛内侧提升）→ 眉毛内角向上
- AU4（眉毛下降）→ 眉毛整体下降（内侧更明显）
- AU9（鼻肌收缩）→ 鼻梁上方的皱纹
- AU15（嘴角向下）→ 嘴角向下（悲伤的主要标记）
- AU17（下唇抬起）→ 下唇中央抬起

【悲伤的"眉毛金字塔"】

悲伤的眉毛是独特的"金字塔形"：
- 内侧：提升（AU1）= 眉毛内角向上走
- 外侧：下降（AU4）= 眉毛外侧向下走
- 结果：眉毛呈倒八字形（倒V）或金字塔形

【悲伤的微表情分级】

Level 1：忧郁（轻度悲伤）
- FACS：AU15轻度激活
- 肌肉：只有降口角肌
- 可见：嘴角微微下压

Level 2：哀伤（中等悲伤）
- FACS：AU1 + AU15激活
- 肌肉：皱眉肌 + 降口角肌
- 可见：金字塔形眉毛 + 嘴角下垂

Level 3：悲痛（标准悲伤）
- FACS：AU1 + AU4 + AU15 + AU9激活
- 肌肉：皱眉肌 + 降口角肌 + 鼻肌
- 可见：完整金字塔眉 + 鼻梁皱纹 + 嘴角下垂

Level 4：悲伤（强烈悲伤）
- FACS：AUC + AU17 + 眼睑下垂
- 肌肉：所有悲伤肌肉 + 下唇抬起
- 可见：眼眶泛红 + 下唇中央抬起 + 眼周皮肤变化

Level 5：崩溃（极强悲伤）
- FACS：AUC = 完全悲伤
- 肌肉：所有悲伤肌肉 + 眼睑完全下垂 + 眼泪
- 可见：完全崩溃 + 眼泪 + 颤抖
```

### 悲伤微表情的"眼眶区"

```
【悲伤微表情的核心：眼眶区】

为什么眼眶是悲伤的关键？

眼眶周围的变化：
- 眼睑：内侧收缩 → 眼睑下垂（上眼睑向下遮盖瞳孔）
- 眉毛：内侧提升 + 外侧下降 → 金字塔形
- 眼角：内侧向上牵拉 → 眼角向上倾斜

【眼眶区的三个关键点】

Point 1：眼睑（下眼睑）
- 眼轮匝肌内侧收缩 → 眼睑内侧向下
- 眼睑下垂程度 = 悲伤程度
- 严重时上眼睑会遮盖瞳孔上部

Point 2：眉毛（内角提升）
- 皱眉肌收缩 → 眉毛内角向上
- 同时眼轮匝肌内侧收缩 → 眉毛内侧向上抬起
- 形成"悲伤的弧度"

Point 3：眼角（向上倾斜）
- 内侧眼角向上牵拉
- 眼角的连线与地平线形成"悲伤的夹角"
- 角度越大 = 越悲伤
```

## 第二部分：悲伤微表情的技术执行

### 悲伤微表情的AI生成参数

```
【悲伤微表情的AI生成系统】

眼睑层：
- 上眼睑：下垂（被内侧收缩向下拉）
- 下眼睑：内侧向上牵拉，轻微紧张
- 眼角：内侧向上，外侧保持或向下

【AI生成的"悲伤"面部参数】

眉毛：
- 内侧：向上提升（金字塔形/AU1）
- 外侧：向下压（AU4）
- 整体：呈倒V字形或金字塔形
- 错误：平直或向上拱（那是惊讶）

眼睛：
- 上眼睑：下垂（遮盖瞳孔上部）
- 下眼睑：内侧向上牵拉
- 瞳孔：可能半闭
- 错误：睁大（那是惊讶）

鼻部：
- 鼻梁：横向皱纹（鼻肌收缩/AU9）
- 鼻翼：正常或轻微扩张（用嘴呼吸）

嘴部：
- 嘴角：向下（降口角肌/AU15）
- 上唇：可能平直或向下
- 下唇：中央抬起（AU17）
- 错误：向上（那是幸福）

眼周：
- 眼眶：内侧向上牵拉
- 眉毛：金字塔形
- 眼角：向上倾斜

【悲伤微表情的"触发器"】

Trigger 1：金字塔形眉毛
- AI指令：inner eyebrows elevated, outer eyebrows depressed, inverted V eyebrows, pyramid shape
- 效果：悲伤眉毛的决定性特征

Trigger 2：眼睑下垂
- AI指令：eyelids drooping, upper eyelid descended, eye half-closed
- 效果：悲伤眼睛的状态

Trigger 3：嘴角下垂
- AI指令：mouth corners pulled down, depressor anguli oris activated,嘴角向下
- 效果：悲伤嘴部的决定性特征

Trigger 4：鼻梁皱纹
- AI指令：transverse nasal wrinkles, procerus contracted,鼻肌收缩
- 效果：悲伤的鼻部特征

Trigger 5：下唇中央抬起
- AI指令：lower lip central raised, depressor labii inferioris activated,下唇抬起
- 效果：强烈悲伤的嘴部特征
```

## 第三部分：悲伤微表情的情绪设计

### 悲伤微表情的"情绪梯度"

```
【悲伤的情绪梯度设计】

Gradient 1：忧郁（轻度悲伤）
- 肌肉：AU15轻度激活
- 表情：嘴角微下垂，眼睛正常
- 情绪：忧虑、失落、惆怅
- 场景：轻微失望、错过机会

Gradient 2：哀伤（中等悲伤）
- 肌肉：AU1 + AU15激活
- 表情：金字塔眉 + 嘴角下垂
- 情绪：悲伤、哀痛、难过
- 场景：失恋、失败、遗憾

Gradient 3：悲痛（标准悲伤）
- 肌肉：AU1 + AU4 + AU15 + AU9激活
- 表情：完整金字塔眉 + 鼻梁皱纹 + 嘴部下垂
- 情绪：痛苦、难受、伤心
- 场景：亲人去世、重大损失

Gradient 4：绝望（强烈悲伤）
- 肌肉：AUC + AU17 + 眼睑下垂
- 表情：眼眶变化 + 下唇抬起
- 情绪：绝望、崩溃、无助
- 场景：失去一切、无法挽回

Gradient 5：崩溃（极强悲伤）
- 肌肉：所有悲伤肌肉 + 眼泪
- 表情：完全崩溃 + 眼泪 + 颤抖
- 情绪：彻底崩溃、无法承受
- 场景：最亲近的人去世、灾难

【悲伤微表情的"场景适配"】

场景1：失恋
- 悲伤类型：哀伤的凝视
- 肌肉：AU1 + AU15中等激活
- 眼神：眼神空洞，看向远方
- 适配：回忆过去、独自悲伤

场景2：亲人去世
- 悲伤类型：崩溃的悲伤
- 肌肉：AUC + 眼泪
- 眼神：眼神涣散，无法聚焦
- 适配：葬礼、告别

场景3：失败
- 悲伤类型：压抑的悲伤
- 肌肉：AU15激活（克制眼泪）
- 表情：嘴角下垂，眉毛可能正常
- 适配：比赛失败、工作失误

场景4：遗憾
- 悲伤类型：轻微忧郁
- 肌肉：AU15轻度
- 表情：嘴角微下压，眼睑正常
- 适配：错过机会、后悔
```
