## 技能概述

**对应好莱坞行业**: 选角导演是演员与角色之间的"配对专家"——负责从万千演员中找到最适合每个角色的那个人。是让对的演员遇到对的角色的那个人。
**核心职责**: 演员评估、选角流程管理、演员与角色匹配、合同谈判

## 核心能力

1. **演员评估**: 一眼看出演员是否适合角色
2. **选角流程**: 高效组织大规模的试镜流程
3. **角色匹配**: 理解角色需求，匹配合适演员
4. **沟通协调**: 与演员、经纪人、出品方的沟通

### 第一部分：演员评估系统

#### 演员评估的五个维度

```
演员评估框架:

1. 外形 (Look)
   - 是否符合角色的外貌描述
   - 是否能在造型后变成角色
   - 镜头前的魅力

2. 表演能力 (Acting Ability)
   - 情感表达范围
   - 细腻程度
   - 与镜头的关系

3. 声音 (Voice)
   - 声音是否适合角色
   - 台词功底
   - 声音变化能力

4. 魅力 (Star Power)
   - 观众缘
   - 镜头吸引力
   - 商业价值（考虑项目类型）

5. 配合度 (Fit)
   - 与其他演员的化学反应
   - 工作态度
   - 可用性和档期
```

#### 试镜流程

```
试镜流程:

1. 初选 (Casting Call)
   - 发布选角公告
   - 收集资料/视频
   - 初步筛选

2. 回调 (Call Back)
   - 邀请候选演员
   - 与其他候选演员一起试戏
   - 测试化学反应

3. 定角 (Booking)
   - 与经纪人谈判
   - 确定合同
   - 官宣
```

### 第二部分：AI视频生成的选角指南

**AI生成时的角色-演员匹配提示词**:

| 角色类型 | 演员特征 | AI提示词 |

| 反派 | 有威胁感、有深度 | menacing antagonist, complex villain, screen presence |

| 喜剧角色 | 节奏感、有魅力 | comedic timing, charming, funny bone |
| 复杂角色 | 表演范围广、可信 | complex character, wide acting range, emotional depth |
