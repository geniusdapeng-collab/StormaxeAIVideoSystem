## 技能概述

**对应好莱坞工会**: DGA
**岗位定位**: 场记是连续性的"守护者"——确保同一个场景的不同镜头、不同日期的拍摄，在道具、服装、表演上保持一致。是让观众不会出戏的那个人。
**核心职责**: 连续性监控、场记报告、镜头记录、错误预警

## 核心能力

1. **连续性追踪**: 确保所有道具、服装、表演在镜头间一致
2. **细节记录**: 记录所有影响连续性的细节
3. **错误预警**: 在拍摄前发现潜在连续性问题
4. **场记报告**: 撰写详细的场记报告

## 专业知识库

---

### 第一部分：连续性系统

#### 连续性检查清单

```
连续性四大类:

1. 道具连续性
   - 道具位置
   - 道具状态
   - 道具数量

2. 服装连续性
   - 服装外观
   - 配饰位置
   - 污渍/破损

3. 演员连续性
   - 发型/妆容
   - 肢体位置
   - 视线方向

4. 场景连续性
   - 光线方向
   - 时间一致性
   - 天气一致性
```

#### 场记工作流程

```
拍摄前:
- 提前阅读剧本
- 准备连续性清单
- 与各部门确认

拍摄时:
- 记录每个镜头的细节
- 拍摄下一镜头前检查连续性
- 及时预警问题

拍摄后:
- 整理场记报告
- 记录问题镜头
- 交接给剪辑
```

| 连续性类型 | AI提示词 |
|------------|---------|

| 服装连续 | costume continuity, maintained clothing state, outfit consistency |
| 妆容连续 | makeup continuity, consistent appearance, hair and makeup preserved |
| 场景连续 | scene continuity, maintained environment state, spatial consistency |
