## 技能概述

**对应好莱坞工会**: IATSE Local 489
**岗位定位**: 动作指导是动作戏的"安全官+编舞师"——负责设计、编排和执行所有特技表演，确保动作场面既精彩又安全。
**核心职责**: 动作编排、安全管理、特技演员管理、动作预可视化

## 核心能力

1. **动作编排**: 设计真实且精彩的动作场面
2. **安全管理**: 确保所有特技表演安全
3. **人员管理**: 协调特技演员团队
4. **与拍摄结合**: 设计与摄影配合的动作方案

### 第一部分：动作编排基础

#### 动作戏的类型

```
动作戏分类:

1. 格斗 (Fight Choreography)
   - 拳斗
   - 刀剑
   - 徒手vs武器

2. 追逐 (Chase Sequences)
   - 车辆追逐
   - 步行追逐
   - 追捕逃脱

3. 战斗 (Battle Sequences)
   - 战争场面
   - 群斗
   - 武器战斗

4. 坠落 (Falls)
   - 低处坠落（翻跟头）
   - 中处坠落（窗户）
   - 高处坠落（特效辅助）

5. 车辆特技 (Vehicle Stunts)
   - 撞车
   - 飘移
   - 翻车
```

#### 动作编排原则

```
动作编排七原则:

1. 可读性 (Readability)
   - 观众能看清动作
   - 动作有清晰的开始和结束

2. 真实感 (Realism)
   - 动作符合物理规律
   - 格斗看起来能造成伤害

3. 安全第一 (Safety First)
   - 每个动作都要有安全措施
   - 必要时用特效替代

4. 摄影配合 (Camera Integration)
   - 动作设计要适合镜头角度
   - 预判镜头能看到什么

5. 节奏设计 (Rhythm Design)
   - 有快有慢
   - 有张有弛

6. 角色表达 (Character Expression)
   - 动作风格表达角色
   - 打斗方式体现性格

7. 创新设计 (Innovation)
   - 避免老套的动作
   - 设计独特的动作
```

### 第二部分：安全管理

#### 安全管理系统

```
特技安全五原则:

1. 风险评估
   - 每个动作都要评估风险
   - 制定风险控制方案

2. 逐步升级
   - 从简单动作开始
   - 逐步增加难度

3. 专业设备
   - 使用专业特技设备
   - 定期检查设备

4. 人员资质
   - 只有合格特技演员才能表演
   - 禁止非专业人员冒险

5. 应急预案
   - 准备急救措施
   - 现场有医护人员
```

#### 常见特技安全设备

```
安全设备清单:

坠落保护:
- 威亚 (Wire Rigging)
- 气垫 (Air Bags)
- 垫子 (Crash Mats)
- 安全网 (Safety Nets)

撞车保护:
- 表演车 (Stunt Cars)
- 缓冲区 (Barrriers)
- 拖车系统 (Track System)

火险保护:
- 防火服装
- 灭火器
- 消防人员在场

特殊保护:
- 护颈 (Stunt Neck Brace)
- 护具 (Protective Gear)
- 牙套 (Mouth Guard)
```
