## 技能概述

**对应好莱坞工会**: SAG-AFTRA（演员工会）+ IATSE Local 706（化妆/发型工会）
**岗位定位**: 定妆守卫是角色的"变身魔法师"——负责把演员变成角色，让观众相信银幕上的就是那个人。是演员进入角色的最后一步。
**核心职责**: 角色定妆、特殊妆容、特效化妆、演员舒适度管理、妆效维护

> **核心理念**: 化妆不是"让演员变漂亮"，而是"让演员变成角色"。最好的化妆让演员忘记自己的脸，忘记自己在化妆，只记得自己就是那个人。

## 核心能力

1. **角色转化**: 理解角色心理，把外在形象与内在特质结合
2. **技术实现**: 掌握各类化妆技术，从日常妆容到极端特效
3. **演员关系**: 管理演员的舒适度和信心
4. **时间管理**: 在有限时间内完成高质量妆容

#### 定妆的五个层次

```
化妆的五个层次（从基础到特效）:

1. 日常妆容 (Everyday Makeup)
   - 目的: 呈现演员最佳状态
   - 程度: 自然、轻微改善
   - 适用: 现代剧、写实剧

2. 角色妆容 (Character Makeup)
   - 目的: 建立角色身份
   - 程度: 中度改变外貌
   - 适用: 年代剧、职业剧

3. 年龄妆容 (Age Makeup)
   - 目的: 改变演员年龄
   - 技术: 皱纹、皮肤质感、发型
   - 适用: 老年/年轻角色

4. 创伤妆容 (Injury Makeup)
   - 目的: 模拟受伤效果
   - 技术: 假血、瘀伤、割伤
   - 适用: 动作、战争、医疗场景

5. 特效妆容 (Special Effects Makeup)
   - 目的: 创造不可能的外貌
   - 技术: 假体、塑形、复杂假皮
   - 适用: 恐怖、科幻、奇幻
```

### 第二部分：技术技能库

#### 基础化妆技术

```
日常妆容步骤:

Step 1: 护肤
- 清洁面部
- 保湿

Step 2: 底妆
- 妆前乳
- 粉底（调色+遮瑕）
- 遮瑕

Step 3: 轮廓
- 阴影：颧骨、下巴、发际线
- 高光：额头、鼻梁、颧骨

Step 4: 眼妆
- 眼影
- 眼线
- 睫毛

Step 5: 唇妆
- 唇线
- 口红/唇彩

Step 6: 定妆
- 散粉
- 定妆喷雾
```

#### 年龄化妆技术

```
年轻化妆（显老技术）:

特征:
- 皱纹：眼角、额头、嘴角
- 皮肤质感：松弛、下垂
- 颜色：斑点、老年斑
- 头发：白发、稀疏

技术:
1. 皱纹制作
   - 乳胶+假皮：深皱纹
   - 彩绘：细皱纹
   - 道具：皱纹贴

2. 老年斑
   - 油彩：色斑
   - 喷枪：均匀老年斑

3. 头发处理
   - 白发粉/喷雾
   - 发际线后移处理

年轻化妆（显年轻技术）:

特征:
- 皮肤紧致
- 光泽
- 圆润

技术:
1. 提亮
   - 高光产品
   - 光泽感

2. 质感
   - 柔焦效果
   - 光滑质感

3. 轮廓
   - 圆润感
   - 减少阴影
```

#### 特效化妆技术

```
特效化妆类型:

1. 假体制作 (Prosthetic)
   - 雕刻
   - 模具制作
   - 发泡乳胶/硅胶成型
   - 上色/毛发植入
   - 粘贴/融合

2. 塑形化妆 (Molding)
   - 面部塑形
   - 身体塑形
   - 特殊形状

3. 特效妆容 (Special Effects)
   - 瘀伤：彩绘+凝胶
   - 割伤：乳胶+血
   - 烧伤：彩绘+纹理
   - 骨折：假体+彩绘

4. 特殊效果
   - 冷冻效果
   - 腐烂效果
   - 怪物效果
```

### 第三部分：特殊妆容设计

#### 角色妆容设计方案

```
场景: 角色变化的全流程设计:

1. 初始形象（角色出场）
   - 年龄感/外貌特征
   - 社会地位/职业特征
   - 情绪状态

2. 渐变过程（故事发展）
   - 压力痕迹
   - 疲惫感
   - 受伤情况

3. 最终形象（高潮/结局）
   - 最大变化
   - 视觉对比
   - 象征意义

示例：失业中年男性
初始: 整洁、职业感、略带疲惫
中期: 眼袋、胡茬、衣服皱褶
结局: 消瘦、苍白、眼神空洞

示例：反派觉醒
初始: 正常妆容
中期: 轻微变化（眼神）
觉醒: 完全转化（疤痕、眼神、气质）
```

#### 年代妆容

```
各年代妆容特征:

1920s:
- 眉毛：细长弓形
- 眼妆：深色烟熏
- 唇：深红/浆果色
- 底妆：哑光象牙白
- 特点：Flapper风格

1940s:
- 眉毛：自然浓密
- 眼妆：深色眼线
- 唇：红色
- 底妆：粉色调
- 特点：好莱坞黄金时代

1960s:
- 眉毛：粗浓
- 眼妆：白色内眼线、蓝色/绿色
- 唇：裸色/粉色
- 底妆：哑光
- 特点：Mod风格

1980s:
- 眉毛：粗浓
- 眼妆：多彩、闪光
- 唇：粉色/珊瑚色
- 底妆：哑光
- 特点：夸张、对比

1990s:
- 眉毛：细
- 眼妆：烟熏
- 唇：裸色/棕色
- 底妆：自然哑光
- 特点：极简主义

2000s:
- 眉毛：浓密自然
- 眼妆：烟熏、金属感
- 唇：粉色/裸色
- 底妆：光泽
- 特点：Y2K风格
```

## 决策逻辑

### 化妆问题处理框架

```
遇到化妆问题时的处理:

问题1: 演员对妆容不满意
→ 倾听意见
→ 解释设计意图
→ 在合理范围内调整
→ 如果不合理，坚持专业判断

问题2: 妆容不持久
→ 使用更强固定产品
→ 增加补妆频率
→ 调整产品配方

问题3: 皮肤过敏
→ 立即停止
→ 使用低敏产品
→ 咨询医生

问题4: 时间不够
→ 简化妆容（减少层次）
→ 提前开始
→ 请求助手帮助
```
