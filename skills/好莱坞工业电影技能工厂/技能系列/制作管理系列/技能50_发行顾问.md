## 技能概述

**岗位定位**: 发行顾问是影视内容的"市场桥梁"——负责把影视作品推向全球市场，找到最佳发行渠道。
**核心职责**: 发行策略制定、平台对接、国际销售、市场定位

## 专业知识库

---

### 发行渠道体系

```
主流发行渠道:

1. 院线发行 (Theatrical)
   - 全球院线
   - 独立院线
   - 艺术院线

2. 流媒体平台 (Streaming)
   - Netflix
   - Amazon Prime
   - Apple TV+
   - Disney+
   - 院网同步

3. 电视发行 (TV)
   - 有线电视
   - 公共电视
   - 电视网

4. 国际销售 (International Sales)
   - 地区代理
   - 国际电影节
   - 海外流媒体

5. 其他渠道
   - DVD/Blu-ray（减少）
   - TVOD点播
   - AVOD广告支持
```

### 发行策略制定

```
发行策略四要素:

1. 市场定位
   - 目标受众
   - 竞争对手
   - 差异化

2. 渠道选择
   - 优先渠道
   - 时间窗口
   - 独家vs非独家

3. 营销配合
   - 档期选择
   - 营销资源
   - PR策略

4. 收益模型
   - 窗口期策略
   - 价格策略
   - 分成比例
```

### AI发行指南

**AI生成时的发行策略提示词**:

| 发行渠道 | AI提示词 |
|----------|---------|
| 流媒体 | streaming release, platform debut, digital-first |
| 院线 | theatrical release, cinema distribution, wide release |
| 独立发行 | indie distribution, film festival circuit, art house |
| 全球发行 | international distribution, worldwide release, global rollout |
