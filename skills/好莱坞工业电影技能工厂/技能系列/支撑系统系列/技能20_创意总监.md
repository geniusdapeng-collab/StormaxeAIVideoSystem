## 技能概述

**对应跨行业职位**: 创意总监是创意团队的"总舵手"——负责定义和守护作品的整体创意方向，确保所有创意决策都服务于统一愿景。是把多元创意整合成一致体验的那个人。
**核心职责**: 创意方向制定、团队创意管理、品质把控、跨部门创意协调、创新推动

> **核心理念**: 创意总监不是在"做创意"，而是在"判断创意"。知道什么是好的创意，什么时候该接受，什么时候该拒绝。最好的创意总监让整个团队知道自己要做什么，同时给团队足够的创作空间。

## 核心能力

1. **方向定义**: 用清晰的语言描述创意愿景
2. **创意判断**: 快速识别好的创意和不好的创意
3. **团队领导**: 激发团队创意，同时保持方向一致
4. **跨域整合**: 整合不同领域的创意，形成统一体验

### 第一部分：创意方向制定

#### 创意简报系统

```
创意简报（Creative Brief）的结构:

1. 背景 (Background)
   - 项目的历史
   - 为什么现在要做这个
   - 相关信息

2. 目标 (Objectives)
   - 商业目标
   - 创意目标
   - 受众目标

3. 受众 (Audience)
   - 主要受众是谁
   - 受众的特点
   - 如何触达受众

4. 竞争 (Competition)
   - 市场现状
   - 竞争作品
   - 差异化机会

5. 创意方向 (Creative Direction)
   - 整体感觉
   - 风格关键词
   - 参考作品

6. 限制 (Constraints)
   - 预算限制
   - 时间限制
   - 法律/道德限制

7. 交付物 (Deliverables)
   - 需要什么
   - 格式要求
   - 时间节点
```

#### 创意关键词系统

```
创意关键词的选择方法:

好关键词的特征:
1. 具体 → "冒险"比"好"好
2. 可感 → "温暖"比"品质"好
3. 有方向 → "复古未来"比"特别"好

关键词分类:

感觉类关键词:
- 冷/暖
- 硬/软
- 快/慢
- 轻/重
- 远/近

风格类关键词:
- 复古/现代
- 极简/繁复
- 写实/抽象
- 优雅/粗犷

情绪类关键词:
- 亲密/宏大
- 紧张/放松
- 神秘/透明
- 危险/安全

关键词组合示例:
- 温暖+亲密+复古 = 怀旧家庭剧
- 冷+宏大+危险 = 科幻惊悚
- 快+紧张+现代 = 动作片
```

### 第二部分：创意判断框架

#### 创意的评估维度

```
评估创意的四个维度:

1. 正确性 (Is it Right?)
   - 是否符合项目目标？
   - 是否符合受众需求？
   - 是否符合品牌/作品调性？

2. 独特性 (Is it Unique?)
   - 与竞品有什么不同？
   - 是否容易被复制？
   - 是否有原创性？

3. 可执行性 (Is it Achievable?)
   - 在现有资源下能实现吗？
   - 技术上可行吗？
   - 时间来得及吗？

4. 影响力 (Is it Impactful?)
   - 能引起目标受众的共鸣吗？
   - 能产生预期的效果吗？
   - 值得投入吗？

评估矩阵:
| 维度 | 1(差) | 2(及格) | 3(好) | 4(优秀) |
|------|--------|---------|--------|---------|
| 正确性 | 完全不符 | 部分符合 | 符合 | 完美匹配 |
| 独特性 | 司空见惯 | 略有不同 | 明显差异 | 行业领先 |
| 可执行性 | 不可行 | 有挑战 | 可行 | 轻松实现 |
| 影响力 | 无感 | 轻微 | 中等 | 强烈 |

决策规则:
- 四个维度都≥3 → 通过
- 任何一个维度=1 → 拒绝
- 其他情况 → 修改后重审
```

#### 创意拒绝技巧

```
拒绝创意的方法（保护团队关系）:

1. 肯定+拒绝
   "这个创意很有意思，但是..."

2. 目标对齐
   "这个创意和我们的目标不太一致..."

3. 提供方向
   "与其X，不如Y？"

4. 询问更多
   "能再解释一下这个创意的目的是什么吗？"

5. 延迟决定
   "让我想想/和其他人讨论一下..."

拒绝不是批评:
- 拒绝创意 ≠ 拒绝创意人
- 解释为什么，而不是评价好坏
- 提供替代方向
```

### 第三部分：团队创意管理

#### 创意团队结构

```
创意团队的常见结构:

1. 小型团队 (<10人)
   - 创意总监 + 设计师/写手
   - 直接沟通，高效
   - 适合：短周期项目

2. 中型团队 (10-30人)
   - 创意总监 + 创意组长 + 执行层
   - 分工明确，层级清晰
   - 适合：中周期项目

3. 大型团队 (30+人)
   - 创意总监 + 创意副总监 + 组长 + 执行层
   - 专业化分工
   - 适合：长周期/大型项目

创意会议类型:

1. 头脑风暴 (Brainstorm)
   - 目标: 收集尽可能多的想法
   - 规则: 不批评，鼓励疯狂想法
   - 产出: 想法列表

2. 创意评审 (Creative Review)
   - 目标: 评估和选择想法
   - 规则: 批评可以，但要有建设性
   - 产出: 入选方案

3. 方向确认 (Direction Alignment)
   - 目标: 确保团队方向一致
   - 规则: 只有一个方向被确认
   - 产出: 确认的创意方向
```

#### 激发团队创意

```
让团队保持创意活跃的方法:

1. 给予空间
   - 不要过度管理
   - 给团队自己探索的时间
   - 允许失败

2. 提供养分
   - 分享行业趋势
   - 组织创意参观
   - 鼓励跨界学习

3. 认可创意
   - 及时表扬好的创意
   - 让创意人被看见
   - 把好创意的来源纳入项目credit

4. 建立安全感
   - 创意被拒绝 ≠ 人的失败
   - 鼓励提出疯狂的想法
   - 惩罚的是抄袭，不是失败

5. 保持节奏
   - 创意需要节奏
   - 高强度项目后要有恢复期
   - 保持团队的好奇心
```

### 第四部分：跨域整合

#### 整合不同创意领域

```
跨域创意的挑战与机遇:

挑战:
1. 语言不通 → 领域有自己的术语
2. 优先级不同 → 美术看重视觉，编剧看重故事
3. 专业傲慢 → "我的领域更重要"

整合策略:

1. 共同目标
   - 找到所有领域的共同目标
   - 用共同目标而非各自目标驱动

2. 翻译语言
   - 把专业术语翻译成共同语言
   - 用感受和情绪沟通，而非技术细节

3. 角色尊重
   - 理解每个领域的价值
   - 不轻视任何领域的贡献

4. 冲突解决
   - 创意总监有最终决定权
   - 但要充分听取各方意见
   - 决定要有清晰的理由
```

## 决策逻辑

### 创意决策框架

```
创意总监的日常决策:

决策1: 是否接受这个创意？
→ 用四维评估框架

决策2: 团队创意方向不一致？
→ 回到共同目标
→ 用创意简报对齐
→ 创意总监做最终决定

决策3: 创意与商业冲突？
→ 首先理解商业约束
→ 然后寻找两全方案
→ 如果不可调和，优先商业（除非违背创意原则）

决策4: 创意枯竭？
→ 给团队休息时间
→ 引入外部刺激（参观/参考）
→ 降低当前项目强度
```
