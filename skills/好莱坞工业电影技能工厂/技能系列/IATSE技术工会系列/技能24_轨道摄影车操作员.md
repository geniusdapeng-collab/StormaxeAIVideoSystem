## 技能概述

**对应好莱坞工会**: IATSE Local 659（轨道摄影车操作工会）
**岗位定位**: 负责操作轨道摄影车（dolly）、摄影升降车（crane）、摄影摇臂（jib）等大型运动设备，在平滑轨迹上承载摄影机完成复杂运动
**核心职责**: 轨道铺设、运动路径规划、摄影车精确操作、与摄影机操作员协同

## 核心能力

1. **轨道系统搭建**: 根据场景需求选择和铺设轨道系统
2. **运动路径规划**: 设计平滑的运动轨迹，确保OFA到EFA的自然过渡
3. **摄影车精确操作**: 在运动中保持平稳、精确的起止点控制
4. **团队协同**: 与摄影机操作员、DP密切配合，传达运动节点信息

### 运动速度与情绪映射

**速度分级**（基于摄影车运动，不是摄影机运动）:

| 速度等级 | 移动速率 | 适用情绪 | AI生成关键词 |
|----------|----------|----------|-------------|
| 极慢 | <0.3m/s | 神圣、庄严、悲伤 | barely moving, glacial, imperceptible |
| 慢速 | 0.3-0.8m/s | 抒情、浪漫、思考 | slow dolly, measured pace, gentle |
| 中速 | 0.8-1.5m/s | 叙事、探索、自然 | steady dolly, natural pace |

### Dolly Push/Pull 的电影语汇

**推镜头 (Dolly In/Push In)**:
- 接近主体 → 强调、聚焦、压迫感
- 从远景推进到特写 → 揭示细节，聚焦情绪
- AI提示词: dolly push-in, camera push-in, approaching subject

**拉镜头 (Dolly Out/Pull Out)**:
- 远离主体 → 退场、揭示环境、释放
- 从特写拉到全景 → 揭示关系、环境、孤独感
- AI提示词: dolly pull-out, retreating, pull back

**横移 (Dolly Track/Sideways)**:
- 与主体运动方向垂直 → 建立关系、展示空间
- AI提示词: tracking shot, dolly sideways, lateral movement

**复合运动 (Combo Moves)**:
- 推+升 → 推进+提升戏剧性
- 拉+转 → 退场+转向揭示
- AI提示词: combo dolly move, compound movement

### 轨道运动节点设计

**起幅(OFA) → 路径 → 落幅(EFA) 标准**:

```markdown
## Dolly Move节点设计
- 预热区: 运动开始前0.5秒，摄影车已有初速度
- 加速区: 从0到目标速度，约1-2秒
- 匀速区: 主要运动段落
- 减速区: 接近落幅前0.5-1秒开始减速
- 停止区: 在落幅精确停止
```

**AI生成时的轨道运动描述格式**:
```
[运动类型], [速度], [起点位置], [终点位置],
[路径描述如有曲线], smooth dolly operation
```

## 决策逻辑

### 轨道选择决策树

```
场景需要运动吗？
  ├── 否 → 固定机位，无需轨道
  └── 是 → 运动方向？
            ├── 前后（推拉）→ 直线轨道
            ├── 左右（横移）→ 直线或弧形轨道
            ├── 环形 → 弧形或环路轨道
            └── 升降 → 摄影升降车或伸缩炮
```

### 运动节奏决策

**规则1: 叙事节奏匹配**
- 慢叙事 → 慢轨道运动（抒情）
- 快叙事 → 快轨道运动（紧张）

**规则2: 情绪焦点**
- 需要聚焦主体 → 推镜头（dolly in）
- 需要揭示环境 → 拉镜头（dolly out）
- 需要平衡主体与环境 → 侧移（track）
