## 技能概述

**对应好莱坞工会**: IATSE Local 44
**岗位定位**: 道具师是道具的"总管"——负责获取、创造、维护和协调所有道具。是让演员手里有东西可拿的那个人。
**核心职责**: 道具获取与管理、特殊道具制作、道具与叙事、拍摄现场道具管理

> **核心理念**: 道具不是装饰品，而是演员与故事之间的桥梁。最好的道具让演员的表演更加真实，让观众相信银幕上的世界。

## 核心能力

1. **道具获取**: 知道去哪里找道具，以及如何制作道具
2. **道具管理**: 建立和维护道具库存，确保拍摄时能找到
3. **叙事理解**: 理解道具如何推动故事
4. **细节把控**: 确保道具在镜头前看起来真实

### 第一部分：道具分类体系

#### 道具的三大类别

```
1. 随身道具 (Hand Props)
   - 演员实际拿在手里的道具
   - 必须能够使用
   - 最严格的质感要求

2. 场景道具 (Set Dressing Props)
   - 装饰场景的道具
   - 演员可能不使用
   - 丰富场景感

3. 特效道具 (Special Effects Props)
   - 用于特效的道具
   - 可能需要与特效配合
   - 特殊处理
```

#### 按叙事功能的分类

```
| 功能 | 描述 | 示例 |
|------|------|------|
| 身份道具 | 揭示角色身份 | 警徽、医生胸牌 |
| 动机道具 | 推动角色行动 | 情书、遗嘱 |
| 象征道具 | 代表主题/概念 | 红苹果代表诱惑 |
| 时代道具 | 确定年代背景 | 老式电话、复古汽车 |
| 连续性道具 | 连接场景 | 桌上的杯子（同一位置） |
```

### 第二部分：道具获取与管理

#### 道具来源

```
道具获取的四种途径:

1. 租赁 (Rental)
   - 从道具租赁公司租借
   - 适合：高端/昂贵道具
   - 优点：成本可控，可选多样
   - 缺点：需要提前预约，有损坏风险

2. 购买 (Purchase)
   - 从商店/网络/二手市场购买
   - 适合：日常道具、消耗品
   - 优点：灵活，可改造
   - 缺点：可能质量不佳

3. 制作 (Fabrication)
   - 道具师手工制作
   - 适合：特殊/不存在的道具
   - 优点：完全定制
   - 缺点：时间长，费用高

4. 借用 (Borrow)
   - 从博物馆/收藏家/个人借
   - 适合：珍贵/历史道具
   - 优点：真实
   - 缺点：限制多，运输风险
```

#### 道具管理系统

```
道具管理三要素:

1. 追踪 (Tracking)
   - 每个道具的唯一编号
   - 位置记录（仓库/现场）
   - 状态记录（新/旧/维修中）

2. 存储 (Storage)
   - 按类型分类
   - 按场景分类
   - 环境控制（温度/湿度/安全）

3. 维护 (Maintenance)
   - 使用前检查
   - 使用后清洁
   - 定期保养
```

### 第三部分：道具与叙事

#### 道具叙事设计

```
道具作为叙事工具的三个层次:

1. 人物揭示
   - 角色私人物品揭示性格
   - 角色随身物品揭示职业/地位

2. 情节推动
   - 关键道具推动剧情
   - 道具丢失/获得引发转折

3. 主题象征
   - 核心道具代表主题
   - 道具变化反映主题

道具设计检查:
- [ ] 这个道具对角色意味着什么？
- [ ] 这个道具对观众意味着什么？
- [ ] 这个道具如何推动故事？
- [ ] 镜头能看到道具的细节吗？
```

### 第四部分：AI视频生成的道具指南

#### 场景类型 → 道具策略

**AI生成时的道具提示词**:

| 场景类型 | 道具策略 | AI提示词 |
|----------|---------|---------|
| 家庭日常 | 真实、日常、有生活痕迹 | authentic family props, lived-in detail, personal items |
| 职场办公 | 职业特征、品牌、细节 | office props, professional items, desk clutter |
| 历史场景 | 年代准确、质感匹配 | period-accurate props, vintage items, era-specific |
| 科幻场景 | 未来感、简洁、品牌 | futuristic props, clean design, sci-fi elements |
| 恐怖场景 | 暗示性、隐藏细节 | ominous props, subtle details, horror atmosphere |
