## 技能概述

**对应好莱坞工会**: IATSE Local 705
**岗位定位**: 服装设计师是角色的"皮肤建筑师"——用布料讲述角色的故事，通过服装让观众读懂角色的身份、情感和社会地位。
**核心职责**: 角色服装设计、服装采购/制作、服装管理、服装与叙事整合

## 核心能力

1. **服装设计**: 理解角色，用服装表达角色
2. **服装制作**: 从设计图到成衣的完整流程
3. **视觉叙事**: 服装如何与故事和角色弧光配合
4. **团队管理**: 管理服装部门，高效完成工作

## 专业知识库

---

### 第一部分：服装设计基础

#### 服装与角色的关系

```
服装揭示角色的四个维度:

1. 社会身份
   - 职业
   - 社会地位
   - 经济状况

2. 心理状态
   - 当前情绪
   - 内在冲突
   - 变化过程

3. 文化背景
   - 时代
   - 地域
   - 文化归属

4. 叙事功能
   - 角色关系
   - 剧情推动
   - 主题象征
```

#### 服装设计流程

```
服装设计五步法:

Step 1: 角色分析
├── 读剧本，理解角色
├── 与导演沟通愿景
└── 确定设计方向

Step 2: 参考收集
├── 历史资料
├── 时尚参考
├── 布料参考
└── 制作Mood Board

Step 3: 草图设计
├── 多个设计方案
├── 确定主要款式
└── 确定配色

Step 4: 制作/采购
├── 制作特殊服装
├── 采购现成服装
└── 改造现有服装

Step 5: 试穿调整
├── 演员试穿
├── 摄影指导确认
└── 微调修改
```

### 第二部分：服装管理

#### 服装管理三阶段

```
前期制作 (Pre-Production):
- 设计服装
- 制作/采购服装
- 建立服装档案

拍摄期 (Production):
- 每日服装检查
- 服装熨烫/清洁
- 现场备用服装
- 记录穿着情况

后期制作 (Post-Production):
- 服装归还
- 清洁整理
- 入库存档
```

#### 服装档案系统

```
每件服装的档案应包含:
- 照片（正面/侧面/背面）
- 尺寸规格
- 面料成分
- 制作/采购信息
- 穿着记录
- 保养说明
```
