## 技能概述

**对应好莱坞工会**: IATSE Local 695
**岗位定位**: 声音部门的基础支持岗位，是整个录音系统的"万金油"——负责无线话筒管理、设备维护备份、第二录音系统操作。是录音组的多面手和后勤保障。
**核心职责**: 无线话筒管理、备份录音系统、设备管理、第二系统操作、场景换场支持

> **行业真相**: 在好莱坞的片场，录音助理是最被低估但最不可或缺的角色。录音师可以专心混音，Boom Operator可以专心操作吊杆，但录音助理必须什么都懂、什么都会。在小制作中，一个好的录音助理可以顶半个录音师。

## 核心能力

1. **无线系统管理**: 无线话筒的设置、频率协调、信号监测全流程管理
2. **第二系统操作**: 独立操作备份录音系统，确保主系统失效时的连续性
3. **设备维护**: 设备日常检查、故障排除、线材管理
4. **场景换场支持**: 快速转换场景话筒布置，最小化停机时间

### 第一部分：无线话筒系统（Wireless Lavaliere System）

#### 无线系统完整架构

```
无线录音系统组成:
┌─────────────────────────────────────────────────────────────────┐
│ 发射端 (Transmitter)                                             │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │  无线发射机 (Bodypack Transmitter)                         │ │
│  │  - 佩戴在演员身上/藏在服装下                              │ │
│  │  - 输入: 领夹话筒或线材输入                              │ │
│  │  - 输出: UHF无线信号                                      │ │
│  └───────────────────────────────────────────────────────────┘ │
│                              │                                   │
│                         UHF 无线信号                            │
│                              │                                   │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │  接收端 (Receiver)                                         │ │
│  │  - 安装在摄影机上或放在腰间包                              │ │
│  │  - 输入: UHF天线                                          │ │
│  │  - 输出: XLR音频信号到录音机                              │ │
│  └───────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

#### 无线频率管理（核心技能）

**为什么要频率管理**:
- 无线信号使用UHF频段（300MHz-3GHz）
- 同剧组多个无线设备必须使用不同频率避免干扰
- 频率重叠会产生互调干扰（Intermodulation）

**频率协调原则**:
```
互调失真计算（简化的三阶互调公式）:

对于三个频率 F1, F2, F3:
互调产物 = 2×F1 - F2, 2×F1 + F2, F1 + F2 - F3 ...

安全距离规则:
- 同频道间隔: ≥200kHz（理想≥500kHz）
- 三阶互调距离: ≥1MHz
- 与电视广播: ≥2MHz

实用频率表（北美常用）:
Channel A: 518.00 MHz
Channel B: 520.00 MHz
Channel C: 572.00 MHz
Channel D: 576.00 MHz
```

**频率扫描与选择流程**:
```
Step 1: 扫描（Scan）
  - 打开接收机，自动扫描当前环境的干扰信号
  - 记录：干净频率、干扰频率

Step 2: 分配（Assign）
  - 主要演员: 最干净的频率
  - 群众演员: 次干净的频率
  - 备用: 有干扰但可接受的频率

Step 3: 验证（Verify）
  - 在演员走戏时，实际测试信号强度
  - 确认无掉线、无噪声
```

#### 领夹话筒隐藏技巧

**演员佩戴标准位置**:

| 位置 | 描述 | 适用场景 | 声音特点 |
|------|------|---------|---------|
| 胸口中间 | 衬衫/领口下方15-30cm | 大多数场景 | 清晰，但有衣物摩擦 |
| 领口处 | 领带结/领口边缘 | 正装/西装 | 最佳隐藏，声音自然 |
| 耳后 | 头发下方 | 短发/无领口 | 需要头套辅助 |
| 锁骨处 | 锁骨凹陷处 | 运动场景 | 稳定，不易摩擦 |

**隐藏技巧**:
```
领夹话筒隐藏七原则:
1. 藏线: 沿领口内侧贴皮肤走线，用胶带固定
2. 防摩擦: 话筒头用柔软材料包裹，避免直接接触皮肤
3. 防衣服声: 领口处用别针固定衣服，减少移动
4. 防汗水: 演员出汗时，用防汗贴保护话筒
5. 方向: 话筒头朝上，对准嘴部
6. 角度: 与嘴部成30°-45°角
7. 测试: 演员移动、转身、抬手后检查是否产生噪音
```

#### 常见无线问题与解决

| 问题 | 原因 | 解决 |
|------|------|------|
| 掉线/信号丢失 | 超出接收范围/干扰 | 调整天线位置/换频率 |
| 噼啪声 | 电池电量低/干扰 | 换电池/换频率 |
| 持续底噪 | 增益过高/干扰 | 降低发射机增益/换频率 |
| 声音闷 | 话筒被遮挡/方向错误 | 调整位置/方向 |
| 忽大忽小 | 多径效应/人体遮挡 | 改变天线位置/演员转身 |

### 第二部分：备份系统管理

#### 第二录音系统操作

**为什么需要第二系统**:
```
备份原则（好莱坞铁律）:
主系统失败率: ~2-3%（每天都有可能发生）
第二系统重要性:
  - 电池没电
  - 线材断裂
  - 发射机故障
  - 接收机故障
  - 人为失误

第二系统 = 保险 = 不需要ADR（自动对白替换）
```

**第二系统的设置**:
```
第二系统配置:
1. 独立电源: 专用电池，不与主系统共用
2. 独立接收机: 独立天线，不同频率
3. 独立录音: 单独的录音设备或通道
4. 独立人员: 录音助理专职监控

第二系统监控:
- 录音助理戴耳机监听第二系统
- 实时监听：信号质量、音量、噪音
- 发现问题时，立即通知录音师
```

### 第三部分：设备管理与维护

#### 录音设备清单与检查

**每日设备检查表**:
```
录音设备每日检查:
□ 录音机: 电池/存储卡/功能测试
□ 主话筒: 外观检查/功能测试
□ 领夹话筒: 外观检查/信号测试
□ 无线发射机: 电池/信号测试
□ 无线接收机: 天线/信号测试
□ 耳机: 左右耳平衡/音量测试
□ 线材: 外观检查/连续性测试
□ 混音器: 通道测试/电平测试
□ 备件: 电池/线材/话筒头
□ 防风系统: 完整性检查
□ 防水盒: 干燥检查
```

**电池管理**:
```
电池使用规范（录音行业标准）:
1. 一次性锂电池（推荐）:
   - 录音机: 8×AA 锂电池 = 12小时
   - 无线发射机: 2×AA 锂电池 = 4-6小时
   - 优势: 稳定、低温表现好、不漏液

2. 镍氢充电电池:
   - 容量大，但电压波动大
   - 低温性能差
   - 需要充满电测试后再使用

3. 更换时机:
   - 录音机: 每8小时更换（或电量<50%）
   - 无线发射机: 每4小时更换
   - 永远不要等到没电才换

4. 备用电池规则:
   - 每部戏至少携带50%备用
   - 重要场景前更换新电池
   - 极寒/极热环境增加更换频率
```

#### 线材管理

```
录音常用线材规格:
1. XLR（平衡音频线）:
   - XLR 3针（话筒线）: 最常用，50-100英尺
   - XLR 5针（多声道）: 扩展器用
   - 长度规则: 越短越好，但留有余量

2. TRS（大三芯）:
   - 用于混音器到录音机
   - 平衡信号，抗干扰

3. TA3/TB3（迷你XLR）:
   - 领夹话筒连接发射机
   - 小型化设备使用

线材维护检查:
- 外观: 无破皮、无折弯、无裸露
- 连接: 插头无松动、无氧化
- 信号: 使用前进行连续性测试
- 存放: 绕圈存放，避免死结
```

### 第四部分：AI视频生成的声音系统指南

#### 无线话筒的AI声音模拟

**无线话筒 vs 吊杆话筒的声音差异**:

| 特征 | 吊杆话筒 | 无线领夹 |
|------|---------|---------|
| 声音特点 | 自然、空间感、干净 | 近讲效应、闷感、衣物声 |
| 距离 | 30-90cm | 10-20cm |
| 频率 | 全频段 | 低频增强，高频稍衰减 |
| 混响 | 带有自然空间感 | 较干 |
| 衣物摩擦 | 无 | 常见 |

**AI生成中的领夹话筒声音提示词**:
```
正向提示词（领夹话筒声音特征）:
- close-mic warmth, intimate dialogue sound
- chest resonance, natural speech proximity
- lavalier audio, hidden mic sound
- subtle clothing rustle, close presence

负向提示词（避免的问题）:
- muffled speech (话筒被遮挡)
- fabric noise (衣物摩擦声)
- boomy low-end (近讲效应过重)
- muffled by clothing (被衣服闷住)
```

#### 场景类型与话筒策略映射

```
室内安静对话（标准场景）:
→ 吊杆为主，领夹备份
→ AI提示词: natural room dialogue, clean speech

室内嘈杂对话（咖啡馆/餐厅）:
→ 领夹为主（近讲提高信噪比）
→ AI提示词: close intimate dialogue, background ambience

户外安静对话:
→ 吊杆为主，注意风噪
→ AI提示词: outdoor dialogue, natural environment

户外嘈杂（街头/活动）:
→ 领夹为主，强指向话筒辅助
→ AI提示词: urban dialogue, close-mic priority, environmental awareness

移动场景（走路/跑步）:
→ 领夹必备，Boom难以跟踪
→ AI提示词: mobile dialogue, footsteps integrated, natural movement
```

## 决策逻辑

### 设备问题紧急决策

```
录音设备故障时:
  │
  ├── 主录音机故障
  │     ├── 立即切换到第二录音机
  │     ├── 通知录音师
  │     └── 记录故障时间点，后续补录
  │
  ├── 无线信号丢失
  │     ├── 检查演员身上发射机（电池？开关？）
  │     ├── 改变接收机天线位置
  │     ├── 尝试备用频率
  │     └── 通知录音师，准备重拍
  │
  ├── 话筒损坏
  │     ├── 立即更换备用话筒
  │     ├── 重新测试信号
  │     └── 通知录音师
  │
  └── 存储卡/硬盘满
        ├── 立即更换存储介质
        ├── 旧介质备份
        └── 继续录音不停
```

### 场景换场话筒决策

```
换场时的决策:
  │
  ├── 场景完全改变（室内→室外）
  │     ├── 更换话筒类型（如需要）
  │     ├── 重新扫描频率
  │     └── 确认新场景基准音
  │
  ├── 同一场景，不同机位
  │     ├── 确认演员是否还在原位
  │     ├── 确认话筒是否需要重新布置
  │     └── 拍摄前测试信号
  │
  └── 同一机位，演员换衣服
        └── 检查领夹话筒是否还在原位
              ├── 还在 → 继续
              └── 重新布置 → 重新测试
```
