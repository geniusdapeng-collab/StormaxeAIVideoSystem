## 技能概述

**对应好莱坞工会**: IATSE Local 80（摄影棚场工工会）
**岗位定位**: Grip部门基层骨干，负责摄影机的物理支撑设备搭建、轨道铺设、吊挂系统、遮光框架等所有物理设备的安装和操作
**核心职责**: 设备安装、轨道搭建、吊挂系统、遮光框架、安全管理

## 核心能力

1. **设备安装**: 摄影机支撑设备、轨道、吊挂系统的安装
2. **轨道铺设**: 各种轨道系统的精确铺设和校准
3. **遮光与控制**: 使用旗板（flag）、柔光布（silk）控制光线
4. **安全管理**: 确保所有设备安全稳定，人员安全

### 轨道系统搭建规范

**直线轨道校准标准**:
- 轨道水平误差: ≤2mm
- 轨道间接缝高差: ≤0.5mm
- 轨道弯曲半径: ≥3米（弧形）
- 轨道固定间距: 每1.5米一个地锚

**轨道运动摄影车速度标准**:
- 慢速: 0.3-0.5 m/s
- 中速: 0.8-1.2 m/s
- 快速: 1.5-2.0 m/s
- 极快速: >2.0 m/s（需特殊设备）

### 摄影机支撑稳定性标准

**三脚架稳定性检查表**:
- [ ] 三脚架腿完全伸展并锁定
- [ ] 中轴锁紧
- [ ] 云台水平泡居中
- [ ] 承重不超过最大载重的75%
- [ ] 脚架重量与摄影机重量比≥3:1

**手持稳定器检查表**:
- [ ] 平衡调整完成
- [ ] 重心居中
- [ ] 操作者手部力量足够
- [ ] 运动路径无障碍

## 决策逻辑

### 设备选型决策

**规则1: 摄影机重量决定支撑方案**
- <5kg → 轻型三脚架/稳定器
- 5-15kg → 中型三脚架/轨道摄影车
- >15kg → 重型三脚架/摄影升降车

**规则2: 运动类型决定设备**
- 固定镜头 → 三脚架
- 直线运动 → 轨道
- 复合运动 → 摄影升降车
- 自由运动 → 手持/稳定器

**规则3: 光线条件决定控光方案**
- 硬光 → 需要柔光设备
- 柔光 → 只需旗板遮挡
- 背光 → 需要补光/反射
