## 技能概述

**对应好莱坞工会**: IATSE Local 600
**岗位定位**: 摄影组基础支持岗位，负责摄影报表、卷轴管理、镜头记录，是剧组信息的中央记录节点
**核心职责**: 场记表维护、摄影报表、卷轴更换、镜头间连续性追踪

## 核心能力

1. **场记表管理**: 维护详细的摄影报表，记录每个镜头的所有参数
2. **连续性追踪**: 追踪场景内的道具位置、演员状态、服装细节
3. **镜头报告**: 记录每个镜头的满意/不满意，标记NG原因
4. **卷轴管理**: 摄影机卷轴的安装、标记、存储

### AI视频生成中的连续性追踪

**连续性要素检查表**:

| 要素 | 检查点 | 典型错误 |
|------|--------|---------|
| 道具位置 | 物体在画面中的精确位置 | 移动了位置 |
| 服装 | 演员服装细节 | 扣子/领口变化 |
| 发型/妆容 | 头发、面部状态 | 妆容不一致 |
| 视线方向 | 眼睛看哪里 | 视线跳轴 |
| 轴线 | 运动方向保持 | 轴线跳切 |
| 时间状态 | 日/夜/晨/昏 | 光线方向不一致 |
| 天气 | 雨/雪/风 | 天气突变 |

## 决策逻辑

### 连续性问题识别

**硬连续性错误**（必须修复）:
- 视线方向相反（跳轴）
- 道具从一只手换到另一只
- 服装颜色/款式改变
- 轴线穿越

**软连续性错误**（可接受范围）:
- 光线角度微调
- 演员微表情差异
- 轻微道具位置偏差

### NG镜头分类

| NG类型 | 原因 | 记录代码 |
|--------|------|---------|
| 技术NG | 焦点/曝光/噪点 | NG-TECH |
| 表演NG | 演员失误 | NG-ACT |
| 连续性NG | 跳轴/道具错误 | NG-CONT |
| 导演NG | 导演叫停 | NG-DIR |
| 设备NG | 机器故障 | NG-EQ |
