## 技能概述

**对应好莱坞工会**: IATSE Local 695
**岗位定位**: 现场录音师的核心搭档，负责吊杆话筒的操作——在正确的时间把话筒放到正确的位置，同时保持正确的距离、角度和高度。是声音捕捉的第一执行者。
**核心职责**: 吊杆操作、空间收音、话筒位置管理、与摄影机协调、与演员协调

> **行业真相**: 现场录音的质量，70%取决于话筒员的技术。话筒放歪一寸，声音就可能毁了。一个顶级话筒员的价值可以顶得上整套录音设备。

## 核心能力

1. **吊杆精确操作**: 在保持画面外的同时，把话筒放到最佳拾音位置
2. **空间感知**: 理解和捕捉空间声场，让声音具有空间感
3. **预判移动**: 预判演员和摄影机的移动轨迹，提前调整话筒位置
4. **无声操作**: 在操作吊杆时消除机械噪音，不产生额外噪声

#### 吊杆操作的基本参数

**话筒与声源的标准距离**:
| 拾音类型 | 距离范围 | 适用场景 | 声音特征 |
|----------|----------|---------|---------|
| 近距离 | 15-30cm | 耳语、亲密对话 | 亲密、厚重、闷感 |
| 标准距离 | 30-60cm | 大多数对白 | 自然、平衡 |
| 中距离 | 60-100cm | 有一定空间感 | 略微软、有空间 |
| 远距离 | 100-150cm | 吊杆为主 | 自然空间、但信噪比下降 |
| 极远 | >150cm | 背景对白 | 环境感强、信噪比差 |

**角度原则**:
```
话筒角度图（侧视图）:

声源嘴部 ────────●─────── 话筒
                  │ 30-45°
                  │ (向下倾斜)
                  ↓

规则:
- 话筒轴线对准嘴部（不是对准鼻子或下巴）
- 向下倾斜30-45°（话筒头低于话筒尾部）
- 避免正对演员（减少呼吸声和喷麦）
```

**高度标准**:
| 镜头类型 | 话筒位置 | 高度 | 注意事项 |
|----------|---------|------|---------|
| 胸部以上镜头 | 画面上方 | 高于头顶至少15cm | 不能入画 |
| 特写镜头 | 画面上方 | 高于头顶30-45cm | 距离更远，角度更陡 |
| 过肩镜头 | 画面上方 | 高于肩膀至少10cm | 优先保证对白质量 |
| 俯拍镜头 | 画面上方 | 取决于构图 | 尽量靠近声源 |

### 第二部分：空间收音技术

#### 为什么空间感很重要

**单声道 vs 空间声**:
- 单声道（无空间信息）: 声音扁平，像从脑子里发出
- 双耳空间声: 声音有位置感，能感知声源方向和距离
- 空间收音正确的声音: 有"空气感"，能感知声源在空间中的位置

**空间收音的三个维度**:
1. **距离感**: 声源远近，声音的"干"与"湿"比例
2. **方向感**: 声源左右，声音的Pan位置
3. **空间感**: 房间/环境的"味道"，混响和反射

#### 空间收音技巧

**技巧1: 话筒与声源的距离控制**
```
距离与空间感关系:
- 太近: 声音干，缺少空间，但信噪比高
- 太远: 空间大，但混响盖过直达声，信噪比差
- 最佳: 直达声与混响比例 3:1（户外）到 2:1（室内）

实操经验:
- 户外安静: 45-90cm，能捕捉环境同时保持干净
- 户外嘈杂: 30-45cm，靠近声源提高信噪比
- 室内混响: 30-60cm，靠近以压直达声比例
```

**技巧2: 反射面的利用与规避**
```
声学反射示意图（俯视图）:
                    演员
                     ●
                     │
  地板              │ ← 吊杆
  ─────             │  45°角
  墙面 ←────●─────── ← 话筒在轴线上
  反射路径 →  ────────────────

规则:
- 话筒轴线对准嘴部，避开墙面反射
- 避开光滑表面（墙面、玻璃、地板）
- 地毯和软家具减少反射，吊杆可以更低
```

**技巧3: 声源追踪**
```
追踪原则:
演员移动 → 话筒必须同步移动

追踪策略:
1. 提前预判移动路线
2. 话筒移动速度 = 演员移动速度（平滑跟随）
3. 移动时保持距离恒定
4. 移动时保持角度恒定
5. 避免急停急转（产生机械声）

操作姿势:
- 小碎步跟随，不要迈大步
- 保持手臂微弯（减震）
- 身体转向，不是手臂扭转
```

### 第三部分：不同场景的操作策略

#### 场景类型操作手册

**场景1: 室内对话（标准双人对白）**

```
设置:
- 主吊杆话筒: 超心形短枪型（416/8078）
- 位置: 主要演员前方45cm-60cm，高于头顶20cm
- 角度: 话筒向下倾斜30-45°，对准嘴部

双人说:
- 优先拾取说话者的声音
- 说话者换人时，话筒平滑移向新说话者
- 移动时间: 1-2秒内完成
- 如果两人同时说话，话筒居中

拍摄反打时:
- 摄影机在另一侧时，话筒仍在主说话者方向
- 话筒可以在画面边缘上方
- 确认话筒不入画
```

**场景2: 移动跟随（走廊/楼梯/移动车内）**

```
设置:
- 话筒延长到最长（3-4m）
- 操作者站在演员斜后方
- 保持最小距离: 30cm（安全距离，避免入画）

移动规则:
1. 预判移动: 先看剧本，标记移动位置
2. 步法: 小碎步，降低重心，手臂微弯
3. 话筒角度: 随演员高度变化同步调整
4. 避免障碍物: 门框/墙角/灯架提前绕行
5. 保持通话: 与录音师保持口头沟通

风险点:
- 脚步声传声（通过吊杆传到话筒）
  → 踩脚跟，用脚尖着地
- 话筒撞到物体
  → 保持足够操作空间
- 话筒入画
  → 与摄影指导确认画面安全范围
```

**场景3: 户外嘈杂环境**

```
挑战:
- 背景噪音大（交通、风声、人群）
- 信噪比差
- 风噪严重

策略:
1. 话筒尽量靠近声源（30-45cm）
2. 使用枪型话筒（强指向性，抑制侧面噪音）
3. 防风系统必须完整（外部防风罩必备）
4. 话筒背对噪音源（利用枪型话筒的拒音特性）

风噪处理:
- 外部防风罩: 野猪毛罩，必须使用
- 风向: 话筒尾部对准风向（话筒头迎风）
- 屏风防风: 在话筒和风之间设置挡风屏障
- 严重风噪: 降低话筒高度，在演员身后
```

**场景4: 大场景/群杂**

```
挑战:
- 多个声源
- 没有明确的"对白"
- 需要捕捉群杂环境声

策略:
1. 识别主声源区域（领说者/主要角色）
2. 群杂话筒（通常两个）:
   - 一个指向领说者
   - 一个捕捉群杂整体
3. 单独录制群杂Room Tone（纯群杂背景）
4. 环境话筒单独录制（大全景时）

话筒布局:
  [群杂1] [领说者] [群杂2]
     │        │        │
     ●────────●────────●
     ↑        ↑        ↑
   吊杆1    吊杆2     吊杆3
```

### 第四部分：与摄影系统的协调

#### 话筒与摄影机的时间同步

**核心问题**: 摄影机位置决定话筒必须在哪里

```
协调原则:
1. 拍摄前: 与摄影指导确认摄影机位置和镜头
2. 话筒位置: 始终在画面上方/侧方，不能入画
3. 安全框: 了解镜头安全框，确保话筒在安全框外
4. 沟通机制: 摄影机位置变化时，立即通知话筒员

入画风险评估表:
| 镜头类型 | 话筒入画风险 | 安全操作 |
|----------|------------|---------|
| 特写 | 高风险 | 话筒必须在画面正上方，远距离 |
| 中景 | 中风险 | 可以从侧面45°进入 |
| 全景 | 低风险 | 话筒可以较低，但仍需避免 |
| 过肩 | 中风险 | 话筒在前景演员肩膀外侧 |
```

#### 典型协调错误与避免

**错误1: 话筒入画**
```
原因: 话筒位置超出安全框
解决:
- 拍摄前确认画面安全框
- 使用取景器或监看画面
- 话筒永远在演员头顶以上
```

**错误2: 话筒阴影入画**
```
原因: 光线在话筒上产生明显阴影
解决:
- 确认主光方向
- 话筒位置避开主光路径
- 必要时使用透光性好的吊杆
```

**错误3: 话筒移动时产生声音**
```
原因: 吊杆碰撞、手部移动、衣服摩擦
解决:
- 手握吊杆把手，不要碰吊杆臂
- 穿不发声音的服装（棉质，避免尼龙）
- 移动时保持手臂微弯，减震
- 远离硬表面（碰撞风险）
```

## 决策逻辑

### 话筒位置决策树

```
新场景开始：
  ├── 确认镜头类型（特写/中景/全景）
  │     ├── 特写 → 话筒更远（45-60cm），角度更陡
  │     ├── 中景 → 标准距离（30-45cm）
  │     └── 全景 → 可更近或更远，灵活掌握
  │
  ├── 确认摄影机位置
  │     ├── 画面上方是否安全
  │     ├── 轴线方向
  │     └── 入画风险
  │
  ├── 确认演员移动
  │     ├── 固定 → 标准位置
  │     └── 移动 → 预判路径，准备跟随
  │
  └── 确认环境噪音
        ├── 安静 → 标准操作
        ├── 嘈杂 → 靠近声源，背对噪音
        └── 有风 → 防风系统+背风操作
```
