## 技能概述

**对应好莱坞后期制作**: 特效总监是视觉特效的"总建筑师"——负责设计和监督电影中所有视觉特效的实现，从简单的画面清除到复杂的数字角色。是把不可能变成可能的那个人。
**核心职责**: VFX设计、技术方案制定、预可视化、拍摄现场指导、后期特效制作管理、预算和时间管理

> **核心理念**: 视觉特效不是"魔法"，而是"技术"和"艺术"的结合。最好的特效让观众相信这是真实的，同时让观众为画面之美而惊叹。

## 核心能力

1. **VFX设计**: 把导演的创意愿景转化为可实现的技术方案
2. **技术方案**: 制定最优的特效实现路径
3. **预可视化**: 在拍摄前预览特效效果
4. **现场指导**: 指导实拍中的特效元素

### 第一部分：VFX分类体系

#### VFX的两大类别

```
1. 现场特效 (Practical Effects / On-Set Effects)
   - 在拍摄现场实际执行
   - 化妆特效、模型、机关
   - 优点: 真实、可信
   - 缺点: 受物理限制

2. 数字特效 (Digital Effects / CGI)
   - 在后期用计算机生成
   - 建模、动画、合成
   - 优点: 无限可能
   - 缺点: 可能失去真实感
   - 费用: 按复杂度计费
```

#### 特效复杂度分类

```
VFX Shot复杂度分级:

Level 1: 简单合成
- 移除不需要的元素（电线、标记）
- 添加简单的天空/背景
- 时间: 1-3小时/镜头
- 费用: $500-2,000/镜头

Level 2: 中等合成
- 画面扩展（添加不存在的建筑）
- 简单的粒子效果（火花、烟）
- 物体跟踪替换
- 时间: 4-16小时/镜头
- 费用: $2,000-15,000/镜头

Level 3: 复杂合成
- 多元素合成
- 数字角色替换（部分身体）
- 复杂粒子系统
- 时间: 16-80小时/镜头
- 费用: $15,000-75,000/镜头

Level 4: 重量级CGI
- 完整数字角色
- 复杂环境（城市、海底、外星）
- 流体/毛发/布料模拟
- 时间: 80-400小时/镜头
- 费用: $75,000-500,000/镜头

Level 5: 史诗级特效
- 数字人群
- 完整数字环境
- 真实物理模拟
- 时间: 400+小时/镜头
- 费用: $500,000+/镜头
```

### 第二部分：VFX工作流程

#### 完整的VFX制作流程

```
VFX制作阶段:

Phase 1: 预生产（Pre-Production）
├── VFX概念设计
│   ├── 与导演沟通视觉愿景
│   ├── 参考图片/视频收集
│   └── 概念设计图制作
├── 预可视化（Pre-Visualization/Previz）
│   ├── 简单3D模型预览
│   ├── 镜头设计
│   └── 与实拍协调
└── 技术评估
    ├── 可行性测试
    ├── 预算评估
    └── 时间规划

Phase 2: 拍摄（Production/On-Set）
├── 特效现场指导
│   ├── 蓝绿幕搭建指导
│   ├── 跟踪点标记
│   └── 特效元素现场执行
├── 数据采集
│   ├── HDR环境光照采集
│   ├── 跟踪参考拍摄
│   └── 演员/道具数字扫描
└── 现场协调
    ├── 与DP协调光线
    ├── 与美术协调颜色
    └── 与导演确认效果

Phase 3: 后期制作（Post-Production）
├── 预演（Layout/Animation Blocking）
│   ├── 3D场景搭建
│   ├── 动画初版
│   └── 与导演确认方向
├── 细节制作（Detailing）
│   ├── 建模/纹理/材质
│   ├── 动画/模拟
│   └── 灯光/渲染
├── 合成（Compositing）
│   ├── 元素合成
│   ├── 颜色匹配
│   └── 最终输出
└── 质量控制（QC）
    ├── 内部审查
    └── 导演审查
```

### 第三部分：常见VFX技术

#### 绿幕/蓝幕技术

```
绿幕vs蓝幕选择:

绿色 (Green Screen):
- 优点: 摄影机最敏感，亮度高
- 缺点: 容易在演员身上产生绿色反射
- 适用: 大多数场景

蓝色 (Blue Screen):
- 优点: 减少演员身上的颜色反射
- 缺点: 亮度较低，需要更多光线
- 适用: 演员穿绿色服装时

Keying技术要求:
- 幕布颜色均匀
- 光线均匀，无阴影
- 演员与幕布保持足够距离
- 无绿/蓝色元素在演员身上

边缘处理 (Edge/Chroma Keying):
- 边缘干净（Hard Edge）
- 半透明边缘（Hair/Cloth）
- 阴影（Contact Shadow）
- 运动模糊边缘
```

#### 跟踪技术

```
Camera Tracking (Matchmoving):
目的: 让计算机模拟摄影机的运动

类型:
1. 2D跟踪
   - 跟踪画面中的2D点
   - 简单但不精确

2. 3D跟踪
   - 重建3D摄影机运动
   - 需要画面中有足够跟踪点
   - 用于合成元素

3. 物体跟踪
   - 跟踪特定物体
   - 用于物体替换

跟踪点标记要求:
- 拍摄前在场景中放置跟踪点
- 跟踪点需要有对比度
- 需要多个角度的跟踪点
- 避免跟踪点被遮挡
```

#### 数字环境创建

```
Digital Environment创建流程:

1. 参考采集
   - HDR环境光照
   - 360度照片/视频
   - 纹理照片

2. 建模
   - 几何建模
   - 细节雕刻
   - LOD优化

3. 纹理
   - 基础纹理
   - 细节纹理
   - 污渍/破损

4. 材质
   - PBR材质
   - 特殊材质（皮肤/布料/金属）

5. 灯光
   - HDRI照明
   - 人工灯光
   - 全局光照

6. 渲染
   - 路径追踪/光栅化
   - 降噪
   - 合成
```

### 第四部分：VFX的AI生成策略

#### 场景类型 → VFX策略

**AI生成时的VFX提示词**:

| 场景类型 | VFX复杂度 | 技术方案 | AI提示词 |
|----------|-----------|---------|---------|
| 简单背景替换 | L1-2 | 2D合成 | simple background replacement, clean plate, sky replacement |
| 物体移除 | L1-2 | 2D/3D跟踪 | object removal, wire removal, set extension |
| 数字角色 | L4-5 | Full CGI | digital double, performance capture, realistic CGI character |
| 爆炸/粒子 | L2-4 | 粒子系统 | explosion, fire and smoke, particle simulation |
| 数字环境 | L3-5 | Environment build | digital environment, full CG set, photorealistic world |
| 超自然效果 | L2-4 | 合成+元素 | supernatural effects, magical particles, ethereal glow |
| 天气效果 | L2-3 | 粒子+合成 | rain, snow, fog, weather effects, atmosphere |
| 时间变化 | L2-3 | 合成 | day-to-night, time-lapse, temporal effects |

#### AI生成与VFX的协作

```
AI视频生成 + VFX后期的工作流:

优势:
- AI可以生成概念图/预可视化
- AI可以生成部分简单VFX素材
- AI可以加速迭代过程

限制:
- AI生成难以满足影视级精度
- AI生成难以保证多镜头一致性
- AI生成难以处理复杂物理模拟

最佳实践:
1. AI用于:
   - 概念设计
   - 预可视化
   - 简单素材生成
   - 快速迭代

2. 传统VFX用于:
   - 影视级精度要求
   - 角色/面部
   - 复杂物理模拟
   - 最终输出
```

### 第五部分：VFX项目管理

#### VFX预算制定

```
VFX预算计算方法:

方法1: 按镜头复杂度
总费用 = Σ(每个镜头费用)

方法2: 按总屏幕时间
总费用 = 总VFX秒数 × 单价/秒

方法3: 按项目打包
总费用 = 打包价格 + 复杂度加成

预算分配参考:
| 阶段 | 占比 | 内容 |
|------|------|------|
| 预可视化 | 3-5% | Previz, 概念设计 |
| 现场数据 | 2-4% | 跟踪, 采集 |
| 后期制作 | 85-95% | 核心特效制作 |
| 质量控制 | 3-5% | QC, 修改 |
```

#### VFX时间规划

```
VFX制作时间参考:

对于一个10分钟VFX序列:
- 预可视化: 2-4周
- 拍摄: 1-2周（含VFX现场指导）
- 后期制作: 12-24周
- 总计: 14-30周

时间分配:
- Layout/Blocking: 15%
- 建模/资产: 25%
- 动画/效果: 30%
- 合成/输出: 20%
- 修改/QC: 10%

紧急项目加速:
- 加速费: +30-50%
- 人员翻倍: 时间减半（但费用翻倍）
- 质量风险: 增加
```

## 决策逻辑

### VFX决策框架

```
接到VFX需求时的决策流程:

Step 1: 评估需求
├── 导演的视觉愿景是什么？
├── 这个镜头是否真的需要VFX？
├── 是否有更简单的替代方案？

Step 2: 技术方案
├── 纯合成？需要建模？需要动画？
├── 现场拍摄如何配合？
└── 技术风险在哪里？

Step 3: 预算和时间
├── 估算制作时间
├── 估算费用
└── 与预算匹配

Step 4: 制作计划
├── 制作顺序
├── 里程碑设置
└── 沟通机制
```
