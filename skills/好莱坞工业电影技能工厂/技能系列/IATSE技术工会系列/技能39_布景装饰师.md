## 技能概述

**对应好莱坞工会**: IATSE Local 800
**岗位定位**: 布景装饰师是场景的"环境设计师"——负责选择和摆放场景中的所有装饰物，让场景看起来真实且有意义。
**核心职责**: 场景陈设、装饰采购/制作、场景历史/逻辑管理、现场陈设管理

## 核心能力

1. **环境设计**: 把空场景变成有故事的空间
2. **细节管理**: 无数细节道具的采购、分类、管理
3. **历史逻辑**: 确保场景陈设符合时代/故事逻辑
4. **团队管理**: 领导布景团队完成陈设工作

### 第一部分：场景陈设系统

#### 陈设的三层结构

```
场景陈设的三层:

1. 大件陈设 (Big Props / Set Dressing)
   - 家具（桌子、椅子、床、沙发）
   - 大型装饰物
   - 窗帘、地毯

2. 小件陈设 (Small Props / Set Dressing)
   - 桌上的物品
   - 墙上装饰
   - 书架内容物

3. 纹理/表面 (Textures)
   - 墙面质感
   - 地面材质
   - 织物纹理
```

#### 场景陈设原则

```
好的场景陈设的特征:

1. 有故事
   - 陈设物讲述住在在这里的人的故事
   - 不是随机摆放，而是有逻辑

2. 有层次
   - 前景、中景、背景都有内容
   - 镜头深入时，有东西可看

3. 有真实感
   - 物品有使用痕迹
   - 不是全新的超市陈设

4. 服务镜头
   - 重要位置放重要道具
   - 无关道具放在不显眼位置
```
