## 技能概述

**对应好莱坞工会**: IATSE Local 695（声音技术工会）
**岗位定位**: 声音部门最高技术权威，现场所有音频信号的捕捉、记录、监控全权负责。是声音素材质量的第一道防线和最后防线。
**核心职责**: 现场录音质量总负责、对白/环境音/效果音采集、录音系统搭建、现场混音、声音档案管理

> **重要性声明**: 在所有后期制作之前，现场录音是唯一不可替代的原始素材。一旦拍摄结束，环境、演员状态、空间特性永远无法复原。现场录音师是"声音守门人"，必须对每一个声音帧负责。

## 核心能力

1. **声音捕捉**: 在任何环境下捕捉最高质量的对白、环境音、效果音
2. **现场混音**: 在有限的条件下进行实时混音，优化录制素材
3. **声音诊断**: 快速识别并解决噪音、干扰、相位等问题
4. **设备管理**: 搭建和管理完整的现场录音系统

### 第一部分：声音物理基础（必须精通）

#### 声音的本质

**声波三要素**:
- **振幅 (Amplitude)**: 声音的强度，决定响度。单位：分贝(dB SPL)
- **频率 (Frequency)**: 声音的音高，决定音色。单位：赫兹(Hz)
- **相位 (Phase)**: 声波的时序关系，决定干涉和梳状滤波

**人耳听觉范围**:
- 可听频率: 20Hz - 20kHz
- 最敏感频率: 2kHz - 5kHz（语音清晰度核心区）
- 动态范围: 约120dB（从0dB SPL到痛阈120dB SPL）

**分贝刻度参考**（必须记住）:

| 场景 | dB SPL | 主观感受 | 录音质量风险 |
|------|--------|---------|------------|
| 录音室 | 20-30 | 极安静 | 无风险 |
| 安静的办公室 | 35-45 | 安静 | 低风险 |
| 正常对话（1m） | 60-70 | 舒适 | 无风险 |
| 大声对话 | 75-85 | 响亮 | 可能失真 |
| 街道交通 | 75-90 | 嘈杂 | 需要降噪 |
| 地铁/施工 | 90-110 | 震耳 | 严重干扰 |
| 摇滚音乐会 | 110-120 | 痛阈 | 无法录音 |

#### 频率与声音设计

**主要频率区间与录音策略**:

| 频率范围 | 包含声音 | 录音策略 | AI生成关键词 |
|----------|----------|----------|-------------|
| 20-80Hz | 低频轰鸣、Sub-bass | 需要大振膜麦克风，指向性强 | sub-bass rumble, deep low-end |
| 80-250Hz | 男声基础音、 bass | 避免同频噪音干扰 | warm low-mid, chest resonance |
| 250-500Hz | 语音浑浊区 | 控制这个频段避免闷感 | chest voice, boominess |
| 500Hz-2kHz | 语音清晰度核心 | 保证这个频段的干净 | speech clarity, presence |
| 2kHz-5kHz | 齿音、辅音、sibilance | 控制近讲效应，-3dB切忌过度 | sibilance, crisp consonants |
| 5kHz-10kHz | 空气感、光泽 | 增加空间感 | air, brightness, sparkle |
| 10kHz-20kHz | 泛音、细节 | 延音衰减、空气感 | natural reverb, ambience |

### 第二部分：麦克风技术（核心技术）

#### 麦克风类型与选型

**按换能原理分类**:

| 类型 | 振膜尺寸 | 指向性 | 频响特征 | 适用场景 |
|------|----------|--------|----------|---------|
| 大振膜电容 (LFDC) | >1英寸 | 多模式 | 平直、细节丰富 | 录音室、安静环境 |
| 小振膜电容 (SFDC) | <1英寸 | 多模式/全向 | 快速、精确 | 乐器、环境录音 |
| 动圈 (Dynamic) | - | 心形/超心形 | 厚实、压抑高频 | 舞台、现场 |
| 铝带 (Ribbon) | - | 双指向 | 温暖、复古 | 人声、乐器 |

**按指向性分类（最重要的选择）**:

| 指向性 | 拾音角度 | 特性 | 使用场景 |
|--------|----------|------|---------|
| 全向 (Omnidirectional) | 360° | 自然空间感，低近讲效应 | 环境音、合唱、安静的室内 |
| 心形 (Cardioid) | 120°前端 | 抑制背面噪音，标准录音 | 大多数对白场景 |
| 超心形 (Supercardioid) | 100°前端 | 更好的侧面抑制 | 有环境噪音的现场 |
| 枪型 (Shotgun) | 30-45°锥形 | 极强的前向拾音 | 吊杆话筒、远距离录音 |
| 8字形 (Figure-8) | 180°前后 | 极高侧面抑制 | 铝带话筒、立体声录音 |

#### 现场录音黄金标准：吊杆+领夹双系统

**双系统备份原则**（好莱坞现场录音最高标准）:

```
Primary System（主系统）: 吊杆话筒（Boom）
  → 位置：演员前方30-60cm，与嘴成30-45°角
  → 优先对白录音，干净、自然的空间感

Secondary System（备份系统）: 领夹无线话筒（Lavalier）
  → 位置：演员胸前15-30cm，隐藏于服装内
  → 备份对白，防止吊杆失效
  → 近讲效应明显，需要EQ修正
```

#### 近讲效应（Proximity Effect）

**定义**: 心形和超心形话筒靠近声源时，低频增强的现象。

**近讲效应曲线**:

| 话筒距离 | 低频增强量 | 声音特征 |
|----------|-----------|---------|
| >30cm | 无 | 自然、平直 |
| 15-30cm | +2-4dB | 温暖、厚重 |
| 5-15cm | +4-8dB | 厚重、近讲感 |
| <5cm | +8-12dB | 极度过载、闷感 |

**控制策略**:
- 吊杆保持30-60cm距离
- 如需靠近，使用全向话筒减少近讲效应
- 近讲效应严重时，用EQ在80-100Hz做高通滤波

### 第三部分：现场录音决策框架

#### 环境声评估七步法

**第一步：听（Listen）**
- 进入场景前，先静默30秒，闭上眼睛听
- 识别：噪音类型（持续/间歇/随机）、方位、高度

**第二步：测量（Measure）**
- 打开录音机，开启监听，放置测量话筒
- 记录：背景噪音dB SPL、各频段能量分布

**第三步：分类（Classify）**
- 噪音类型分类：
  - **A类（可控）**: HVAC空调（规律）、灯光嗡嗡（规律）
  - **B类（不可控）**: 交通（随机）、风声（变化）、人员走动（间歇）
  - **C类（灾难）**: 飞机、警笛、施工（无法录音）

**第四步：定位（Locate）**
- 确定噪音来源方位
- 确定吊杆最佳位置（噪音最小、声音最清晰）

**第五步：决策（Decide）**

| 背景噪音 | 对白信噪比 | 决策 |
|----------|-----------|------|
| <35dB SPL | >40dB | 优秀，直接录音 |
| 35-50dB SPL | 25-40dB | 良好，使用超心形话筒 |
| 50-65dB SPL | 10-25dB | 可接受，使用枪型+领夹双系统 |
| >65dB SPL | <10dB | 不可接受，考虑ADR或强降噪 |

**第六步：执行（Execute）**
- 布置话筒位置
- 设置录音参数（电平、增益、滤波器）
- 进行声音测试（Say "Testing 1-2-3"）

**第七步：验证（Verify）**
- 监听录音回放
- 检查：是否有削波、噪音是否可接受、是否有奇怪的反射

#### 室内录音空间声学判断

**混响时间判断**（RT60：声音衰减60dB所需时间）:

| RT60 | 空间感受 | 声音策略 |
|------|---------|---------|
| <0.3s | 极干、录音室感 | 自然录音，保留少量环境 |
| 0.3-0.5s | 干净、日常感 | 标准室内，自然混响 |
| 0.5-1.0s | 明显混响、中等 | 需要注意吊杆角度，避免过多反射 |
| 1.0-2.0s | 大空间感 | 混响明显，EQ削减低中频 |
| >2.0s | 巨大空间、洞穴感 | 严重混响问题，考虑ADR |

### 第四部分：录音参数与设备配置

#### 采样率与位深度的科学选择

| 配置 | 采样率 | 位深度 | 适用场景 | 文件规格 |
|------|--------|--------|---------|---------|
| 电影标准 | 48kHz | 24-bit | 所有电影/TV制作 | WAV |
| 音乐录制 | 96kHz | 24-bit | 高品质音乐 | WAV |
| 对白专录 | 48kHz | 24-bit | 对白干净版 | WAV |
| 参考/草稿 | 44.1kHz | 16-bit | 临时参考 | WAV |

**为什么是48kHz/24-bit？**
- 48kHz提供20Hz-20kHz的完整人耳可闻范围，额外余量用于处理
- 24-bit提供144dB的动态范围，确保不会削波
- 人耳可闻动态范围约120dB，24-bit足够

#### 电平设置标准（行业规范）

**0dB FS（满刻度）刻度认知**:
```
Peak Meter:
  -inf → -60 → -48 → -36 → -24 → -18 → -12 → -6 → 0dB FS
         |      |      |      |      |      |      |
       极低   很低    低     标准    高     警告   危险   削波

Recording Level (专业标准):
  -20dB FS → 峰值电平，用于音乐/动态大内容
  -18dB FS → 标准对白电平
  -12dB FS → 响亮对白/尖叫声
  -6dB FS → 极限，不应经常到达
  0dB FS → 绝对禁止，会削波失真
```

**对白录音电平标准**:
- 平均电平（Dialogue Level）: -18dB FS RMS
- 峰值电平（Peak）: 不超过-3dB FS
- 动态范围: 12-15dB（对话场景）
- 空间留余: 峰值与平均电平差约12-15dB

#### 对白清晰度优化（AI生成策略）

**室内近景对白**:
```
正面提示词:
  dialogue in quiet room, crisp speech clarity, close-mic warmth,
  subtle room ambience, natural reverb tail, no background noise

反面提示词:
  muffled dialogue, excessive reverb, room echo,
  boomy low-end, harsh sibilance
```

**室外环境对白**:
```
正面提示词:
  dialogue in outdoor setting, wind protection EQ, clear speech,

反面提示词:
  wind noise, traffic interference, harsh outdoor echo,
  muffled by wind, excessive background
```

**大空间/教堂/大厅**:
```
正面提示词:
  dialogue in cavernous space, natural reverb (RT60: 1.5-2.5s),

反面提示词:
  muddying reverb, incomprehensible echo,

```

## 决策逻辑

### 录音问题诊断决策树

```
录制声音有问题？
  ├── 削波失真（Peak at 0dB）
  │     ├── 话放增益过高 → 降低输入增益
  │     ├── 话筒太靠近声源 → 增加话筒距离
  │     └── 瞬态能量过大 → 降低灵敏度或移动话筒
  │
  ├── 低频轰鸣（Boominess）
  │     ├── 近讲效应 → 增加话筒距离
  │     ├── HVAC噪音 → 在100Hz做高通滤波
  │     └── 反射干扰 → 改变话筒角度避开反射面
  │
  ├── 闷感（Muffled）
  │     ├── 话筒被遮挡 → 移动话筒位置
  │     ├── 低频过多 → 高通滤波80-100Hz
  │     └── 高频衰减 → 检查话筒频率响应
  │
  ├── 房间混响（Reverberation）
  │     ├── 话筒太远 → 靠近声源
  │     ├── 环境太大 → 考虑ADR或强降噪
  │     └── 使用心形/超心形 → 提高信噪比
  │
  └── 底噪（Noise Floor）
        ├── 增益过高 → 降低前置放大
        ├── 环境噪音 → 找更安静的位置/时间
        └── 设备问题 → 检查线材和接口
```

### 环境录音决策矩阵

| 场景条件 | 推荐话筒 | 推荐技术 | 备份方案 |
|----------|---------|---------|---------|
| 安静室内，固定机位 | 大振膜电容 | 吊杆+领夹双录 | 领夹单独备份 |
| 嘈杂室外，有风 | 枪型+防风罩 | 超心形近讲 | 领夹强降噪 |
| 移动跟拍，有环境音 | 小振膜领夹 | 领夹主录 | 备用领夹 |
| 大场景，群杂 | 全向环境话筒 | 环境音单独录 | 多点收音 |
| 车内，小空间 | 界面话筒+领夹 | 车内混音 | ADR备选 |
