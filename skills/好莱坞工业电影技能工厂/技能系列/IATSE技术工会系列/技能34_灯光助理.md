### 第一部分：灯位搭建基础

#### 基本灯位术语

```
灯位代号（国际通用）:
- Kicker: 提亮阴影侧的辅助光
- Rim/Edge: 轮廓光，分离主体与背景
- Backlight: 背光，从后方照亮主体
- Hair Light: 头发光，照亮头发/肩部
- Eye Light: 眼神光，在眼睛里形成反光
- Fill: 填充光，补阴影侧

标准三点布光:
┌────────────────────────────────────────────────────────────┐
│                                                             │
│                        BACKLIGHT (背光)                    │
│                              ●                             │
│                                                             │
│         ●KICKER                                    ●FILL   │
│        (提亮)         ←─── SUBJECT ───→                  │
│                                                             │
│                         ●KEY LIGHT                          │
│                         (主光)                               │
│                                                             │
└────────────────────────────────────────────────────────────┘

主光(Key Light):
- 位置: 45°侧面，高于主体
- 角度: 与相机约45°-90°
- 高度: 高于主体眼睛约45°

填充光(Fill Light):
- 位置: 主光另一侧
- 亮度: 主光的1/4到1/2
- 作用: 填充主光产生的阴影

背光(Backlight/Rim):
- 位置: 主体正后方或侧后方
- 作用: 将主体与背景分离，增加立体感
```

#### 灯位搭建流程

```
标准灯位搭建流程:

Step 1: 理解指令
- 灯的类型
- 功率和数量
- 位置和角度
- 附件（柔光箱、色片等）

Step 2: 设备准备
- 从仓库领取设备
- 检查设备状态
- 准备所有配件

Step 3: 搭建灯位
- 安装灯架/吊挂设备
- 安装灯光设备
- 安装附件（色片、柔光布等）
- 连接电源

Step 4: 调整位置
- 按照指示放置灯架
- 调整高度
- 调整角度
- 固定位置

Step 5: 电力连接
- 连接电源线
- 确认电压匹配
- 打开电源测试

Step 6: 微调
- 根据Gaffer指示微调
- 调整亮度
- 调整角度
- 确认效果
```

#### 常见附件使用

```
灯光附件:

柔光附件:
- 柔光箱: 最常用的柔光设备
- 蝴蝶布: 大面积柔光
- 丝绸布: 轻柔光
- 网格(Grid): 控制光线范围

控光附件:
- 旗片(Flag): 阻挡光线
- 挡板(Barndoor): 控制光线边缘
- 蜂巢(Honeycomb): 聚光，减少散射
- 束光筒(Snoot): 聚光到小范围

色光附件:
- 色片(Gel): 改变光线颜色
- 色纸架(Snap): 快速安装色片

特殊附件:
- 降频器(Dimmer): 调节亮度
- 调焦器(Iris): 调整光斑大小
```

### 第二部分：设备操作规范

#### 灯架使用

```
常见灯架类型:

C-Stand (C型灯架):
- 最常用的灯架
- 可调节高度和角度
- 配有沙袋配重

Junior Stand (Junior灯架):
- 承重更大
- 用于大型灯具

Baby Stand (Baby灯架):
- 用于小型设备
- 高度有限

Overhead Pipe (顶管):
- 吊挂在天花板
- 用于免支架灯位

卡扣类型:
- Baby Pin (1-1/8"): 小型设备
- Junior Pin (1-1/2"): 大型设备

安全规则:
- 灯架必须用沙袋配重
- 沙袋重量 = 灯具重量的1.5倍
- 灯架腿必须完全展开
- 禁止将灯架放在不稳定表面
```

#### 安全操作规范

```
灯光安全五条铁律:

1. 热灯禁止触摸
   - 钨丝灯工作时表面温度可达300°C+
   - 等待冷却后再操作
   - 使用手套或工具操作

2. 防水禁止
   - 灯光设备禁止直接接触水
   - 雨天户外使用需要防雨罩
   - 湿手禁止操作设备

3. 电力安全
   - 操作前确认电源已关闭
   - 使用绝缘工具
   - 禁止私自改动电路
   - 定期检查电线破损

4. 防火措施
   - 热灯远离易燃物
   - 柔光布需要防火处理
   - 现场配备灭火器
   - 离开前确认所有热灯关闭

5. 高空作业
   - 使用安全梯/升降台
   - 禁止在灯架上攀爬
   - 重型灯具需要两人操作
```
