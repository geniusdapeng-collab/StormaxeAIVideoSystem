## 技能概述

**对应好莱坞工会**: IATSE Local 800
**岗位定位**: 美术指导是Production Designer的"执行臂膀"——负责把设计愿景转化为具体的美术实现，管理美术部门的日常工作。是把创意变成现实的那个人。
**核心职责**: 设计执行、美术组管理、预算控制、质量监督

## 核心能力

1. **设计执行**: 把设计图转化为实际场景
2. **团队管理**: 协调美术部门的各工种
3. **预算控制**: 确保美术设计在预算内完成
4. **品质把控**: 确保美术效果符合设计标准

### 第一部分：美术部门结构

#### 美术部门的层级

```
美术部门结构:

Production Designer (美术指导)
    │
    ├── Art Director (美术指导)
    │       │
    │       ├── Set Decorator (布景装饰师)
    │       │       ├── Lead Man
    │       │       └── Set Dressers
    │       │
    │       ├── Prop Master (道具师)
    │       │       └── Props Department
    │       │
    │       └── Construction Foreman (搭建工头)
    │               └── Construction Crew
    │
    └── Graphic Designer (平面设计师)
```

#### 各岗位协作

```
美术部门协作流程:

1. Production Designer 确定视觉方向
2. Art Director 执行具体设计
3. Set Decorator 负责场景陈设
4. Prop Master 负责道具
5. Construction 负责实体搭建
```

---

### 第二部分：美术执行管理

#### 设计到实现的转化

```
设计执行流程:

Step 1: 设计解读
- 理解Production Designer的意图
- 与设计师确认细节
- 识别潜在问题

Step 2: 技术转化
- 把设计图转化为施工图
- 计算材料和人工需求
- 评估时间线

Step 3: 资源调配
- 分配人员
- 采购材料
- 协调场地

Step 4: 制作监督
- 定期检查进度
- 确保质量
- 及时调整

Step 5: 最终验收
- 与Production Designer验收
- 修复问题
- 完成交付
```
