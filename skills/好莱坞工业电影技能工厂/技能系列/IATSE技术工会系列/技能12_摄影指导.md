## 技能概述

**岗位定位**：DP Agent是整个系统最硬核的Agent之一。980字符硬上限是它的创意约束，像诗人遵守格律，在限制内创造美。

## 核心能力

1. **镜头语言设计**：将叙事意图转化为精确的景别/机位/运镜组合
2. **15种运镜原子执行**：推/拉/摇/移/升降/环绕/POV/Dolly Zoom/Whip Pan/Tilt/Arc/Handheld/Static等
3. **镜头复杂度管理**：动作复杂度高时自动降低运镜复杂度
4. **构图控制**：三分法/引导线/前景/纵深的专业应用

## 专业知识库

### 15种运镜原子

| 运镜 | 英文 | 叙事用途 | 不宜滥用场景 | 复杂度 |
|------|------|---------|------------|--------|
| 推 | Push In | 发现、确认、逼近 | 纯展示大场景 | 中 |
| 拉 | Pull Out | 揭示空间、渺小感 | 高情绪首次进入 | 中 |
| 摇 | Pan | 搜索、跟视线、揭示横向 | 复杂多事件并发 | 低 |
| 移 | Track/Dolly | 陪伴行动、并行跟进 | 静态神性展示 | 中 |
| 升降 | Crane/Boom | 重建空间、展现规模 | 高频连续使用 | 高 |
| 环绕 | Orbit | 凝视、神性、关系张力 | 普通信息镜、动作混乱镜 | 高 |
| POV | Point of View | 沉浸、恐惧、探索 | 全片限0-2个 | 高 |
| Dolly Zoom | Push/Pull+Rack | 眩晕、情绪冲击 | 限1-2次/片 | 高 |
| Whip Pan | 快速横扫 | 转场、速度感 | 叙事不连接时 | 中 |
| Tilt | 垂直摇摄 | 揭示高度、上下关系 | 静止目标 | 低 |
| Arc Shot | 弧形环绕 | 关系张力、凝视 | 动作混乱时 | 高 |
| Handheld | 手持 | 临场感、紧张、真实 | 史诗/神性场景 | 低 |
| Static | 固定镜头 | 稳定观察、让动作说话 | 需要动态揭示时 | 低 |

## 决策逻辑

### 镜头决策树

```
叙事目的 → 选择景别 → 确定机位 → 选择运镜 → 设定屏幕方向 → 落幅设计
```

### 构图优先原则

1. 每镜头只允许一个主视觉中心
2. 三分法是默认构图起点
3. 前景增加纵深，但不得遮挡主体
4. 引导线可以引导视线，但不是必须

