## 技能概述

**对应好莱坞工会**: IATSE Local 728
**岗位定位**: 灯光师大助是灯光组的"后勤大总管"——负责所有灯光设备的日常管理、电力分配、与外景地电力系统的协调。是Gaffer最信任的左膀右臂。
**核心职责**: 设备管理、电力分配、灯光组日常运营、安全管理

> **核心理念**: 灯光师大助不需要懂光，只需要懂电。最好的大助让Gaffer不需要操心任何后勤问题，所有设备、电力、人员都在应该在的位置。

1. **电力管理**: 计算和分配整个剧组的电力需求
2. **设备管理**: 管理灯光设备的出入库、维护、保养
3. **人员调度**: 协调灯光助理的工作分配
4. **安全管理**: 确保所有电气操作安全合规

### 第一部分：电力系统管理

#### 电力计算基础

```
电力计算公式:
功率 (Watts) = 电压 (Volts) × 电流 (Amps)

常用电压:
- 北美: 120V / 240V
- 欧洲: 230V / 400V
- 香港: 220V

电流计算:
电流 (Amps) = 功率 (Watts) / 电压 (Volts)

保险丝/断路器规则:
- 安全负载: 断路器额定值的80%
- 例如: 20A断路器 → 安全最大负载 = 16A

电力需求计算示例:
1000W钨丝灯在120V下:
电流 = 1000W / 120V = 8.3A
安全考虑: 断路器应为20A

常见灯光设备功率:
| 设备 | 功率范围 |
|------|---------|
| 1K Fresnel | 1000W |
| 2K Fresnel | 2000W |
| 575W HMI | 575W (实际消耗约750VA) |
| 1200W HMI | 1200W (实际消耗约1500VA) |
| LED Panel | 50-400W |
| China Ball | 100-200W |
| Tweenie | 650W |
| Baby | 650W |
| Junior | 1000W |
| Senior | 2000W |
| Maxi Brute | 2400-3600W |
```

#### 电力来源类型

```
电力来源分类:

1. 发电车 (Generator)
   - 优点: 完全独立，不受外接限制
   - 缺点: 需要运输、加油、噪音
   - 容量: 200A-1000A
   - 适用: 外景、大功率需求

2. 配电箱 (Distro Box)
   - 优点: 连接建筑物电力系统
   - 缺点: 受建筑物容量限制
   - 容量: 100A-400A
   - 适用: 摄影棚、室内

3. 电池组 (Battery)
   - 优点: 完全无线、静音
   - 缺点: 容量有限、需要更换
   - 容量: 视电池组而定
   - 适用: 小型LED设备、便携场景

4. 建筑物电力
   - 优点: 直接使用、无额外成本
   - 缺点: 容量有限、可能有其他用户
   - 需要: 提前勘察、与建筑管理协调
   - 适用: 室内小场景
```

### 第二部分：设备管理系统

#### 灯光设备清单

```
常用灯光设备分类:

柔光设备:
- 1K Fresnel (Baby) → 柔和边缘光
- 2K Fresnel (Junior) → 主光/背光
- 4K Fresnel (Senior) → 强主光
- LED Panel → 可调色温、静音
- China Ball → 环境光、瞳孔灯
- 灯笼 (Celestial) → 柔和环境

硬光设备:
- Open Face → 无透镜、直接光
- PAR Can → 硬边光束
- Maxi Brute → 高强度硬光
-dede灯 → 聚光、追光效果

HMI (Hydrargyrum Medium-Arc Iodide):
- 575W HMI → 小场景日光
- 1200W HMI → 中等场景
- 2500W HMI → 大场景主光
- 4000W+ HMI → 超大场景/日光模拟

特效灯:
- 燃烧弹 (Practicals) → 真实光源
- 管灯 (Tubes) → 霓虹/LED效果
- 蜡烛/火焰 → 烛火效果
```

#### 设备管理流程

```
设备出入库流程:

收到设备:
1. 打开箱子检查外观
2. 通电测试（开关、功能）
3. 记录序列号
4. 如有问题，立即报告

归还设备:
1. 清洁设备
2. 整理线材
3. 检查配件完整性
4. 通电测试
5. 归还仓库

设备检查清单:
- [ ] 外壳无损坏
- [ ] 灯泡/灯管正常
- [ ] 调焦/俯仰调节顺畅
- [ ] 电源线无破损
- [ ] 支架/把手正常
- [ ] 通电测试正常
```
