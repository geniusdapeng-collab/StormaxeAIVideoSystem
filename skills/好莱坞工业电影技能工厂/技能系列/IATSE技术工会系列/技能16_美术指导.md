## 技能概述

**对应好莱坞工会**: IATSE Local 800（艺术指导工会）
**岗位定位**: 美术指导是电影的"视觉建筑师"——负责设计电影的视觉世界，把剧本中的场景文字转化为有情感、有风格、有逻辑的三维空间。是把导演的视觉愿景变成现实的那个人。
**核心职责**: 视觉风格设计、场景空间规划、色彩方案制定、美术预算管理、跨部门协调

> **核心理念**: 美术设计不是"装饰"，而是"叙事"。每一个道具的摆放、每一面墙的颜色、每一束光的角度，都在向观众传递信息。最好的美术设计让观众沉浸在故事世界里，却感觉不到设计的痕迹。

## 核心能力

1. **视觉叙事**: 用视觉元素讲述故事、表达情绪、塑造人物
2. **空间设计**: 设计真实且有表现力的三维空间
3. **风格统一**: 建立和维护全片统一的视觉语言
4. **预算管理**: 在有限预算内实现最大视觉效果

### 第一部分：视觉叙事原理

#### 什么是"视觉叙事"

```
美术设计的核心问题：如何用视觉元素讲故事？

视觉叙事三层:

1. 物理叙事（Physical Storytelling）
   - 空间如何反映角色
   - 道具如何暗示人物
   - 环境如何推动情节

2. 情绪叙事（Emotional Storytelling）
   - 色彩如何影响情绪
   - 光线如何塑造氛围
   - 质感如何创造触感

3. 文化叙事（Cultural Storytelling）
   - 时代感如何建立
   - 社会背景如何体现
   - 地域特征如何呈现
```

#### 空间与人物的关系

```
空间设计揭示人物:

1. 角色的空间 = 角色的心理

空间特征 → 人物心理:
| 空间特征 | 心理含义 |
|----------|---------|
| 狭小拥挤 | 压力、限制、困境 |
| 宽敞空旷 | 孤独、冷漠、权力 |
| 杂乱无章 | 混乱、失控、崩溃 |
| 整洁有序 | 控制、强迫、压抑 |
| 自然温馨 | 安全、归属、平静 |
| 人工冰冷 | 疏离、异化、无情 |

2. 变化的空间 = 变化的角色

角色变化 → 空间变化:
- 角色变富有 → 生活空间变大、变精致
- 角色变贫穷 → 生活空间变小、变破旧
- 角色变疯狂 → 空间扭曲、失衡
- 角色获救赎 → 空间变明亮、变开阔

3. 角色的"领地" = 角色的身份

领地从属关系:
- 主角的领地：最大、最复杂、最有细节
- 反派的领地：与主角形成对比
- 配角的领地：简洁、辅助叙事
```

### 第二部分：色彩理论与应用

#### 色彩心理学深度解析

```
色彩三要素与情绪:

1. 色相 (Hue)
   - 颜色的基本属性
   - 红/橙/黄/绿/蓝/紫

2. 明度 (Value)
   - 颜色的明暗程度
   - 高明度：轻盈、欢快、脆弱
   - 低明度：沉重、严肃、神秘

3. 饱和度 (Saturation)
   - 颜色的纯度/鲜艳程度
   - 高饱和：强烈、热情、戏剧性
   - 低饱和：克制、怀旧、真实

色彩情绪映射表:
| 色相 | 正向情绪 | 负向情绪 | 适用场景 |
|------|---------|---------|---------|
| 红 | 热情、爱情、力量 | 危险、愤怒、血腥 | 激情、战斗、恐怖 |
| 橙 | 温暖、活力、创造 | 焦虑、警告 | 喜剧、能量、危险 |
| 黄 | 快乐、希望、纯真 | 背叛、疯狂 | 喜剧、儿童、幻觉 |
| 绿 | 自然、成长、和平 | 嫉妒、毒药 | 自然、田园、疾病 |
| 蓝 | 平静、信任、忧郁 | 冷漠、悲伤、死亡 | 剧情、忧郁、科幻 |
| 紫 | 神秘、奢华、魔法 | 堕落、腐败 | 奇幻、宫廷、梦幻 |
| 白 | 纯净、神圣、和平 | 冰冷、死亡、空虚 | 神圣、极简、恐怖 |
| 黑 | 优雅、权力、神秘 | 死亡、邪恶、未知 | 犯罪、恐怖、正式 |
```

#### 色彩方案设计

```
五种经典色彩方案（Color Schemes）:

1. 单色方案 (Monochromatic)
   - 单一色相，不同明度和饱和度
   - 效果：统一、克制、优雅
   - 案例：《辛德勒的名单》黑白中的红衣女孩

2. 互补色方案 (Complementary)
   - 色轮上相对的两种颜色
   - 效果：强烈对比、戏剧性
   - 案例：《搏击俱乐部》蓝橙对比

3. 类似色方案 (Analogous)
   - 色轮上相邻的几种颜色
   - 效果：和谐、舒适、自然
   - 案例：《少年派的奇幻漂流》蓝绿橙日落

4. 分裂互补 (Split-Complementary)
   - 一种颜色+其互补色两侧的颜色
   - 效果：对比但柔和
   - 案例：《她》橙色主导+蓝色点缀

5. 三角色彩 (Triadic)
   - 色轮上等距的三种颜色
   - 效果：丰富、活泼、平衡
   - 案例：《布达佩斯大饭店》粉+红+蓝

色彩方案选择指南:
| 场景情绪 | 推荐方案 | 原因 |
|----------|---------|------|
| 浪漫爱情 | 类似色/单色 | 和谐温柔 |
| 动作紧张 | 互补色 | 强烈对比 |
| 神秘悬疑 | 低饱和单色 | 克制紧张 |
| 喜剧欢乐 | 高饱和类似色 | 活泼欢快 |
| 恐怖惊悚 | 低饱和+点缀色 | 压抑+突然刺激 |
| 历史剧情 | 低饱和分plits | 复古真实 |
```

### 第三部分：场景空间设计

#### 场景设计流程

```
美术指导的场景设计七步法:

Step 1: 读懂剧本
├── 理解场景的功能（叙事/情绪/人物）
├── 识别场景的关键词
└── 了解场景在整体中的位置

Step 2: 与导演沟通
├── 导演的视觉愿景是什么？
├── 是否有参考图片？
├── 色调/风格/质感的要求？
└── 这个场景与前后场景的关系？

Step 3: 收集参考资料
├── 真实空间照片
├── 绘画/摄影作品
├── 历史资料
└── 整理Mood Board

Step 4: 绘制草图/概念图
├── 粗略布局
├── 色彩倾向
├── 光线方向
└── 关键道具位置

Step 5: 制作微缩模型/3D预览
├── 验证空间比例
├── 验证镜头角度
└── 发现潜在问题

Step 6: 细化设计
├── 确定所有道具清单
├── 确定色彩规格
├── 确定材质规格
└── 编写美术设计说明书

Step 7: 实施与监督
├── 搭建/改造空间
├── 摆放道具
├── 色彩涂装
└── 持续调整直到满意
```

#### 不同场景类型的设计要点

```
室内场景类型设计:

| 场景类型 | 核心关注 | 常见问题 | 设计要点 |
|----------|---------|---------|---------|
| 住宅/家庭 | 真实感 | 太整洁/太刻意 | 留有生活痕迹 |
| 办公室 | 阶层感 | 缺乏个性 | 通过细节展示职业/地位 |
| 餐厅/酒吧 | 社交氛围 | 太过普通 | 用道具讲人群故事 |
| 医院 | 冰冷/希望 | 过于恐怖/过于乐观 | 平衡残忍与救赎 |
| 学校 | 年代感 | 过于现代/刻板 | 细节展示时代背景 |
| 监狱 | 压抑感 | 过于戏剧化 | 真实细节更重要 |

户外场景类型设计:

| 场景类型 | 核心关注 | 常见问题 | 设计要点 |
|----------|---------|---------|---------|
| 城市街道 | 时代/地域 | 过于干净 | 杂乱、涂鸦、招牌 |
| 自然景观 | 情绪表达 | 缺乏人工痕迹 | 人工痕迹暗示人类活动 |
| 废墟/废弃 | 时间感 | 过于干净 | 生长中的自然、残留物 |
| 夜景 | 灯光设计 | 过于明亮 | 黑暗中的点光源 |
| 恶劣天气 | 氛围强化 | 视觉疲劳 | 雨/雪/风要有变化 |
```

### 第四部分：道具设计与管理

#### 道具的三个层次

```
道具分类体系:

1. 戏用道具（Functional Props）
   - 角色实际使用的物品
   - 必须真实可用
   - 与角色行为直接相关

2. 陈设道具（Set Dressing）
   - 装饰性物品，营造空间氛围
   - 不需要真实使用
   - 讲述空间的故事

3. 效果道具（Special Effects Props）
   - 用于特效的物品
   - 通常需要特殊处理
   - 与视觉特效配合
```

#### 道具叙事设计

```
道具作为叙事工具:

1. 人物揭示道具
   - 放在角色空间中的私人物品
   - 揭示职业、兴趣、关系

2. 情节推动道具
   - 推动剧情发展的关键物品
   - 主角追寻/守护/逃避的对象

3. 主题象征道具
   - 代表电影主题的物品
   - 在关键时刻反复出现

4. 时代标记道具
   - 确定年代背景的物品
   - 让观众立即进入时代

道具设计检查表:
- [ ] 这个道具是角色的还是陈设的？
- [ ] 这个道具放在这里有什么故事？
- [ ] 镜头能看到这个道具吗？
- [ ] 这个道具在片中会出现几次？
- [ ] 这个道具与主题有什么关系？
```

### 第五部分：AI视频生成的美术设计指南

#### 场景类型 → 美术设计策略

**AI生成时的美术设计提示词**:

| 场景类型 | 色彩策略 | 空间策略 | AI提示词 |
|----------|---------|---------|---------|
| 童年回忆 | 暖色调、褪色感 | 温暖狭小 | warm nostalgic, faded colors, intimate childhood space |
| 噩梦场景 | 冷色调、低饱和 | 扭曲失衡 | cold desaturated nightmare, distorted space, unsettling |
| 浪漫场景 | 柔和色调、柔光 | 亲密空间 | soft romantic palette, intimate lighting, warm tones |
| 犯罪现场 | 冷色调、对比强 | 杂乱压抑 | crime scene palette, high contrast, claustrophobic |
| 科幻未来 | 冷色调、蓝色调 | 极简几何 | sci-fi cold palette, blue tones, minimalist geometry |
| 复古怀旧 | 暖色调、褪色 | 旧质感 | vintage warm palette, aged textures, nostalgic setting |

#### 视觉风格分类与AI提示词

```
经典视觉风格参考:

| 风格 | 色彩特点 | 空间特点 | AI提示词 |
|------|---------|---------|---------|
| 黑色电影 | 高对比、低明度 | 阴影丰富 | film noir, high contrast, deep shadows, noir aesthetic |
| 新黑色 | 现代+黑色电影 | 当代空间 | neo-noir, modern twist on classic noir |
| 极简主义 | 黑白或单色 | 极少物品 | minimalist, sparse, clean lines, monochromatic |
| 印象派 | 柔和色调 | 模糊边界 | impressionistic, soft focus, dreamy palette |
| 超现实 | 不协调色彩 | 扭曲空间 | surreal, discordant colors, warped space |
| 写实主义 | 低饱和 | 杂乱真实 | realistic, desaturated, lived-in detail |
| 表现主义 | 强烈色彩 | 夸张变形 | expressionist, bold colors, exaggerated forms |
```

## 决策逻辑

### 美术设计决策框架

```
接到美术设计需求时的决策流程:

Step 1: 理解需求
├── 场景功能（叙事/情绪/人物）
├── 导演风格偏好
├── 预算范围
└── 时间限制

Step 2: 建立视觉方向
├── 色彩方案选择
├── 空间风格定位
├── 质感方向确定
└── 制作Mood Board

Step 3: 设计实施
├── 场景设计图
├── 道具清单
├── 色彩规格
└── 材质要求

Step 4: 协调沟通
├── 与摄影指导协调光线
├── 与导演确认方向
├── 与制片确认预算
└── 与特效协调需求

Step 5: 质量控制
├── 场景搭建监督
├── 色彩涂装确认
├── 道具摆放指导
└── 最终美术审核
```
