## 技能概述

**对应好莱坞后期制作**: 调色师是电影的"光影雕塑师"——负责在数字后期重新塑造影片的光影、色彩和情绪。是把摄影机记录的原始数据转化为有灵魂的视觉体验的那个人。
**核心职责**: 色彩校正、情绪调色、风格建立、跨部门协作、输出格式管理

> **核心理念**: 调色不是"把画面调漂亮"，而是"用光影和色彩讲故事"。每一个调色决策都应该服务于叙事。最好的调色让观众沉浸在故事里，而不是注意到调色本身。

## 核心能力

1. **色彩科学**: 精通色彩理论、色彩空间、光学原理
2. **情绪塑造**: 用色彩和光影传递情绪、塑造角色、建立氛围
3. **技术精度**: 掌握专业的调色软件和工作流程
4. **叙事理解**: 理解故事需求，把视觉风格与叙事意图结合

### 第一部分：色彩科学基础

#### 色彩空间与色彩模型

```
色彩空间（Color Space）:

1. RGB色彩空间
   - 加法混色（光）
   - 用于显示器、摄影机
   - 三个通道：红、绿、蓝

2. YUV/YCbCr色彩空间
   - 亮度（Y）与色度（Cb/Cr）分离
   - 用于视频压缩
   - 调色时通常分解为Luminance和Chroma

3. LAB色彩空间
   - 亮度（L）与色度（a/b）分离
   - a = 绿-红，b = 蓝-黄
   - 最接近人眼感知的色彩模型
   - 调色时用于精确色彩调整

4. 调色常用的色彩空间选择:
   - Log素材: 使用Log空间调色（如Rec.709 Log, S-Log, LogC）
   - 输出标准: Rec.709（HD）或Rec.2020（4K/HDR）
```

#### 色彩理论深度解析

```
色彩三要素（对调色师最重要的）:

1. 色相 (Hue)
   - 颜色的基本属性
   - 调色中最难改变的部分
   - 需要通过HSL/LAB调整

2. 饱和度 (Saturation)
   - 颜色的纯度/鲜艳度
   - 最影响情绪的参数
   - 提升=更鲜艳，降低=更灰

3. 明度/亮度 (Luminance/Value)
   - 颜色的明暗程度
   - 影响对比度和层次感
   - 最影响视觉重量的参数

色彩校正三步骤（调色师的工作顺序）:

Step 1: 色彩校正（Color Correction）
├── 目的: 还原真实色彩
├── 内容:
│   ├── 白平衡校正（色温）
│   ├── 曝光校正（亮度）
│   ├── 对比度校正
│   └── 色彩平衡（黑白场）
└── 标准: 中性灰正确、白色正确、肤色正确

Step 2: 色彩匹配（Color Matching）
├── 目的: 让不同镜头的色彩一致
├── 内容:
│   ├── 同场景不同机位匹配
│   ├── 不同光线条件匹配
│   └── 白天/夜景衔接匹配
└── 工具: 色轮、矢量示波器

Step 3: 风格调色（Creative Grading）
├── 目的: 建立情绪和风格
├── 内容:
│   ├── 色调（冷/暖/对比）
│   ├── 色彩强调（特定颜色突出）
│   ├── 光影调整（高光/阴影着色）
│   └── 整体情绪塑造
└── 风格参考: 电影参考、导演意图
```

### 第二部分：调色工具与技术

#### 调色软件核心工具

```
专业调色软件工具箱:

1. 示波器（Scopes）- 客观分析
├── 波形示波器（Waveform）
│     ├── 显示亮度分布
│     ├── Y轴=亮度，X轴=时间
│     └── 用于曝光控制
├── 矢量示波器（Vectorscope）
│     ├── 显示色度分布
│     ├── 中心=灰度，边缘=饱和色
│     └── 用于肤色检查和色彩匹配
├── 直方图（Histogram）
│     ├── 显示像素分布
│     ├── 快速了解画面亮度分布
│     └── 辅助曝光决策
└── 色彩条（Color Bar）
      ├── 用于色彩校正参考
      └── 现在较少使用

2. 调色轮（Color Wheels）- 主观调整
├── Lift（提升）: 控制阴影
├── Gamma（伽马）: 控制中间调
├── Gain（增益）: 控制高光
└── Offset（偏移）: 整体偏移

3. 曲线（Curves）- 精确控制
├── RGB曲线: 整体对比度
├── 单通道曲线: 分通道调整
├── 色相vs饱和度曲线: 精确色彩控制
└── 色相vs亮度曲线: 色彩亮度关系

4. 节点（Nodes）- 工作流程
├── 串联节点: 按顺序执行
├── 并联节点: 同时应用不同调整
├── 混合节点: 叠加不同效果
└── 节点工作流是专业调色的基础
```

#### 专业调色技术

```
技术1: 一级校色（Primary Correction）

目的: 建立正确的曝光和色彩基础

步骤:
1. 查看波形，确定画面亮度范围
2. 设置黑场（Black Level）- 波形底部0-10IRE
3. 设置白场（White Level）- 波形顶部90-100IRE
4. 校正白平衡 - 矢量示波器检查肤色线
5. 校正色彩平衡 - 确保中性色正确

曝光判断标准:
| 区域 | 波形位置 | 调色决策 |
|------|---------|---------|
| 阴影 | 0-10 IRE | 不死黑，保留细节 |
| 中间调 | 40-60 IRE | 主要信息区，正常曝光 |
| 高光 | 90-100 IRE | 不过曝，保留质感 |
| 肤色 | 约70 IRE | 示波器肤色线位置 |


技术2: 二级校色（Secondary Correction）

目的: 精确调整画面特定部分

常用二级调整:
- 肤色选择: 只影响皮肤区域
- 天空选择: 只影响天空区域
- 阴影/高光分离: 独立调整明暗区域

技术:
- 色相选择（Hue Selection）
- 饱和度选择（Saturation Selection）
- 亮度选择（Luminance Selection）
- 软硬度（Soft/Hard）


技术3: 风格调色（Creative Grading）

目的: 建立情绪和风格

三种经典调色风格:

1. 对比风格（Contrast Grade）
   特点: 高对比、强阴影、鲜明色彩
   代表: 《权力的游戏》夜王场景
   参数:
   - 提升对比度
   - 压低高光
   - 提升阴影深度
   - 饱和度中高

2. 消色风格（Desaturated Grade）
   特点: 低饱和、灰色调、冷暖对比
   代表: 《逃离拉斯维加斯》
   参数:
   - 大幅降低饱和度
   - 保持一定对比度
   - 冷色调倾向
   - 肤色保持自然

3. 夸张风格（High-Key / Stylized）
   特点: 强色彩、高饱和、戏剧化
   代表: 《宿醉》
   参数:
   - 提高整体饱和度
   - 偏暖或偏冷色调
   - 提升高光
   - 压低阴影
```

### 第三部分：情绪调色——用色彩讲故事

#### 色调与情绪的映射

```
经典色调情绪库（调色师必备参考）:

| 色调类型 | 色彩特征 | 情绪效果 | 代表案例 |
|----------|---------|---------|---------|
| 暖色调 | 橙/黄为主 | 温暖、怀旧、亲密 | 《现代启示录》开头 |
| 冷色调 | 蓝/青为主 | 寒冷、孤独、疏离 | 《银翼杀手》 |
| 消色暖调 | 灰+暖点缀 | 怀旧、失落、伤逝 | 《拯救大兵瑞恩》诺曼底 |
| 消色冷调 | 灰+冷点缀 | 冷酷、现实、压抑 | 《黑客帝国》 |
| 高饱和暖调 | 鲜艳橙红 | 热烈、激情、欢乐 | 《疯狂的麦克斯》 |
| 高饱和冷调 | 鲜艳蓝绿 | 活力、科幻、异化 | 《遗落战境》 |
| 低对比柔调 | 灰蒙蒙、柔和 | 梦幻、回忆、迷离 | 《搏击俱乐部》梦境 |
| 高对比强调 | 黑白+单色点缀 | 强烈、警示、聚焦 | 《辛德勒的名单》红衣 |
| 日落金调 | 金色高光+蓝阴影 | 浪漫、黄金时刻 | 《爱在日落黄昏时》 |
| 青色替代 | 蓝绿代替暖色 | 科幻、冷漠、异化 | 《搏击俱乐部》办公室 |

情绪调色决策框架:
1. 确定场景情绪 → 选择色调类型
2. 确定场景时间 → 调整色温方向
3. 确定角色状态 → 调整对比度
4. 确定叙事阶段 → 调整饱和度
```

#### 肤色——调色最重要的颜色

```
肤色是调色的红线:

为什么肤色最重要:
- 人眼对人脸肤色的敏感度极高
- 肤色偏差会立即被感知
- 错误的肤色让观众产生不适感

肤色的正确范围:
- 示波器上肤色线约在70 IRE位置
- 矢量示波器上肤色点应在肤色刻度线上
- 色调应在橙黄范围内（色相约30-40°）

肤色检查三步骤:
Step 1: 矢量示波器检查
- 肤色点是否在肤色刻度线上？
- 是否偏离太远？

Step 2: 直视判断
- 肤色是否看起来"正常"？
- 是否偏红/偏绿/偏黄/偏蓝？

Step 3: 环境对比
- 肤色与环境是否协调？
- 是否与环境融合/对比？

常见肤色问题与修复:
| 问题 | 原因 | 修复方法 |
|------|------|---------|
| 偏红 | 白平衡偏暖/曝光过度 | 减少红色，降低色温 |
| 偏绿 | 白平衡偏绿/荧光灯 | 减少绿色，增加品红 |
| 偏黄 | 白平衡偏黄/钨丝灯 | 减少黄色，增加蓝色 |
| 偏蓝 | 白平衡偏蓝/日光 | 减少蓝色，增加暖色 |
| 灰暗 | 饱和度不足 | 轻微提升饱和度 |
| 惨白 | 曝光过度/饱和度低 | 降低曝光，提升饱和度 |
```

### 第四部分：场景类型调色手册

#### 不同场景的调色策略

```
场景1: 日景（Day Exterior）

调色目标: 还原自然日光、强化时间感

参数参考:
- 色温: 5500K-6500K（日光）
- 色调: 轻微偏暖（阳光感）
- 对比度: 中等偏高（阳光方向感）
- 饱和度: 自然饱和或略高

情绪变体:
- 明亮日景: 暖色调、高饱和、柔对比
- 正午日景: 冷色调、高对比、硬阴影
- 阴天日景: 偏冷、低对比、灰蓝调


场景2: 夜室内景（Night Interior）

调色目标: 建立夜晚氛围、人工光源真实感

参数参考:
- 色温: 2700K-3200K（钨丝灯）
- 色调: 暖色为主，可能有冷色点缀
- 对比度: 中等（室内光线柔和）
- 饱和度: 中等

情绪变体:
- 温馨夜景: 暖橙色调、柔和光线
- 紧张夜景: 冷暖对比、强阴影
- 神秘夜景: 低饱和、蓝紫调、暗部偏蓝


场景3: 夜室外景（Night Exterior）

调色目标: 黑暗中有可见度、保持夜晚感

参数参考:
- 色温: 混合（城市灯光多样）
- 色调: 蓝黑为主，灯光暖色点缀
- 对比度: 高对比（深黑+点光源）
- 饱和度: 中等（灯光色彩保留）

情绪变体:
- 城市夜景: 蓝黑底色、霓虹点缀
- 月夜场景: 冷蓝调、银白高光
- 街灯夜景: 暖黄底色、强烈明暗对比


场景4: 回忆/闪回（Memories/Flashbacks）

调色目标: 与现实区分、建立时间距离感

参数参考:
- 色温: 可偏暖或偏冷
- 色调: 与主时间线形成对比
- 对比度: 降低（柔化）
- 饱和度: 可降低（褪色感）

情绪变体:
- 童年回忆: 暖调、柔焦、褪色、饱和度略低
- 创伤回忆: 高对比、偏冷、锐化
- 梦境: 低饱和、蓝紫调、柔光


场景5: 动作/追逐（Action/Chase）

调色目标: 增强动感、强化速度感

参数参考:
- 色调: 根据场景时间设定
- 对比度: 高（强调动态）
- 饱和度: 根据需要（可提升紧张感）
- 特殊处理: 动态模糊配合

情绪变体:
- 阳光追逐: 暖色、高饱和、高对比
- 夜间追逐: 冷色、低饱和、蓝光点缀
```

## 决策逻辑

### 调色决策框架

```
调色工作流程:

Step 1: 观看与理解
├── 完整观看序列，理解叙事意图
├── 标记需要调整的镜头
├── 与导演/DP沟通风格方向
└── 确定参考影片或风格图片

Step 2: 建立基础（Primary Grade）
├── 曝光校正 - 确保正确亮度范围
├── 白平衡 - 确保色彩中性
├── 对比度 - 建立正确的明暗关系
└── 确保所有镜头基准一致

Step 3: 场景匹配（Scene Matching）
├── 选择参考镜头（最理想的镜头）
├── 匹配其他镜头到参考镜头
├── 检查接戏问题
└── 必要时逐帧调整

Step 4: 风格调色（Creative Grade）
├── 应用整体色调
├── 分离调整（天空/肤色/阴影）
├── 添加风格化处理
└── 保持肤色正确

Step 5: 最终检查（Final QC）
├── 检查所有镜头接戏
├── 检查肤色是否正确
├── 检查输出格式
└── 与导演最终确认
```
