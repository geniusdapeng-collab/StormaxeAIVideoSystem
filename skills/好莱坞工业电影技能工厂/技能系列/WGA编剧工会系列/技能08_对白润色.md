## 技能概述

**对应好莱坞工会**: WGA（编剧工会体系）
**岗位定位**: 对白润色是台词的"声音雕刻师"——负责让每一个角色的声音独特、真实、有力。是让观众相信"这个人真的会说这样的话"的那个人。
**核心职责**: 角色声音设计、台词精炼、潜台词挖掘、方言/口音处理、口语化改写

> **核心理念**: 对白不是"说话"，而是"表演"。每一句台词都是角色的一次行动。最好的对白让观众忘记这是写的，只会觉得"这个人真的就是这样说话的"。

## 核心能力

1. **角色声音识别**: 区分不同角色的声音，确保每个角色独一无二
2. **潜台词挖掘**: 在表面台词下挖掘角色的真实意图
3. **口语化改写**: 把书面语改成自然的口语
4. **信息密度控制**: 控制每句对白的信息量，避免"解说腔"

### 第一部分：角色声音设计

#### 什么是"角色声音"

```
角色声音的四个维度:

1. 词汇选择 (Word Choice)
   - 角色的教育背景、职业、地区影响词汇选择
   - 律师 vs 农民 vs 科学家 → 词汇完全不同

2. 句式结构 (Sentence Structure)
   - 角色的思维方式决定说话方式
   - 理性的人 → 完整句子、逻辑连接词
   - 感性的人 → 短句、感叹词、省略

3. 节奏韵律 (Rhythm & Cadence)
   - 角色说话的快慢、停顿、重音
   - 南方人 vs 纽约客 → 节奏完全不同

4. 语气态度 (Tone & Attitude)
   - 角色对他人的态度影响说话方式
   - 对上级 vs 对朋友 vs 对敌人 → 语气完全不同
```

#### 角色声音诊断工具

```
角色声音测试（Voice Test）:

当这个角色说"我爱你"时，他们会怎么说？

A角色（硬汉侦探）:
"我爱你。"
→ 实际说："我不喜欢说这些。但你知道的。"

B角色（天真少女）:
"我爱你。"
→ 实际说："我超级喜欢你！你不觉得我们很配吗？"

C角色（冷酷反派）:
"我爱你。"
→ 实际说："这是你应得的。因为你有用。"

D角色（老兵）:
"我爱你。"
→ 实际说："（沉默良久）...你是我的人了。"

角色声音设计表:
| 角色 | 说话长度 | 词汇水平 | 节奏 | 口头禅 | 回避话题 |
|------|---------|---------|------|--------|---------|
| 硬汉侦探 | 短 | 口语化 | 断断续续 | "该死" | 情感 |
| 天真少女 | 长 | 简单 | 流畅轻快 | "超级" | 无 |
| 冷酷反派 | 短 | 精准 | 冷峻平稳 | 无 | 感情 |
| 老兵 | 中等 | 朴实 | 沉重停顿 | "见鬼" | 过去 |
```

### 第二部分：对白的核心技巧

#### 技巧1: "说"与"做"的分离

```
对白的最高境界：角色说的话 ≠ 角色想做的事

例子（普通）:
"我不想去，但我觉得我应该去。"
→ 直白说出内心想法

例子（优秀）:
"给我拿外套。"
→ 通过行动暗示内心挣扎

规则：
- 当你能用行动展示时，不要用对白
- 对白应该补充行动，而不是重复行动
- 潜台词 > 表面意思

AI对白提示词:
"subtext-rich dialogue, characters say one thing but mean another,
imply emotion through action and word choice, undertone of tension"
```

#### 技巧2: 口语化改造

```
书面语 vs 口语 的对比:

书面语（剧本中不应该出现的）:
- "我非常感谢你的帮助。"
- "我认为这可能是一个错误的选择。"
- "我需要一些时间来思考这个问题。"
- "让我告诉你一些重要的事情。"

口语化改造:
- "谢了。" / "太谢谢了。"
- "搞不好是个坑。"
- "让我想想..."
- "哎，听我说。"

口语化的标志:
| 书面语 | 口语化 |
|--------|--------|
| 我认为 | 我觉得 / 我寻思 |
| 他说 | 他说 / 他管这叫 |
| 为什么不 | 干嘛不 |
| 不要 | 甭 / 别 |
| 非常 | 特 / 贼 / 超级 |
| 可以 | 成 / 行 / 中 |

AI口语化提示词:
"natural speech patterns, conversational dialogue,
contractions and elisions, incomplete sentences,
characters speak like real people, not exposition"
```

#### 技巧3: 信息密度控制

```
对白的信息密度三种模式:

1. 低密度（适合日常/关系场景）
   - 每句话只含1个信息点
   - 大量留白、停顿、废话
   - 示例：
     "吃了吗？"
     "吃了。"
     "吃的啥？"
     "面。"

2. 中密度（适合推进情节）
   - 每句话含1-2个信息点
   - 对话有明确目的
   - 示例：
     "他走了。"
     "什么时候？"
     "昨晚。"
     "说什么了吗？"
     "让我别等他。"

3. 高密度（适合高张力/揭示场景）
   - 每句话含2-3个信息点
   - 快节奏、多层意思
   - 示例：
     "他昨晚死了。"
     "怎么死的？"
     "官方说心脏病。你信吗？"
     "那实际上呢？"
     "我知道的是：他死前收到一封信，
      发件人是我们的老朋友。"

信息过载的错误:
- 在低压力场景中高密度 → 感觉像解说
- 在高压力场景中低密度 → 节奏拖沓
```

#### 技巧4: 潜台词设计

```
潜台词的三个层次:

1. 表面层（Surface）
   - 角色说的话的字典意思
   - 每个观众都能听到

2. 意图层（Intention）
   - 角色说这话的目的
   - 想让对方做什么/感受什么
   - 需要观众思考才能理解

3. 真实层（Truth）
   - 角色内心真正想表达的
   - 通常与表面意思相反或更深
   - 高级叙事才有

案例解析:
"你看起来真美。"
（表面）: 赞美
（意图）: 让她高兴/调情/讽刺
（真实）: 我很嫉妒你现在的生活

潜台词设计练习:
问自己三个问题:
1. 角色在说这话时，内心真正想说什么？
2. 角色想让对方做什么？
3. 角色的这句话会引发什么后果？
```

#### 技巧5: 重复与变化的艺术

```
对白的重复不是错误，而是技巧:

类型1: 关键词重复（强化主题）
- 角色在不同场景反复说同一个词
- 这个词成为角色的标志/主题
- 示例：《教父》中反复出现的"offer"

类型2: 句式重复（建立节奏）
- 相同句式在不同场景重复
- 形成观众的期待感
- 示例："你要么站在我这边，要么..."

类型3: 语义重复（深化情感）
- 不同的词说同样的意思
- 层层递进
- 示例：
  "我需要你。"
  "我离不开你。"
  "没有你我不知道该怎么办。"

类型4: 变化式重复（打破期待）
- 相同句式，但结尾变化
- 制造意外感
- 示例：
  第一次："你疯了吗？"
  第二次："你疯了吗！"（愤怒）
  第三次："你疯了吗..."（疲惫）
```

### 第三部分：特殊对白处理

#### 回忆/梦境对白

```
回忆场景对白特点:
- 声音质感：低沉、混响、褪色感
- 节奏：略慢于现实
- 信息：碎片化、不完整

AI处理提示词:
"memory dialogue, dreamlike quality, fragmented speech,
slightly slowed, reverbed audio, past tense implication"

梦境场景对白特点:
- 声音质感：极简、延迟、回声
- 节奏：不稳定、有停顿
- 信息：象征性、隐喻性

处理示例:
现实版："我不想离开你。"
回忆版："我...离开...不想...你..."
梦境版："你是谁？哦，不对，我认识你。
        你还在这里吗？等等...时间..."
```

#### 多角色群戏对白

```
群戏对白的设计原则:

1. 声部分配
   - 主角声部：清晰、主动
   - 配角声部：配合、回应
   - 背景声部：暗示、环境

2. 打断技巧
   - 自然打断（抢话）
   - 故意打断（制造冲突）
   - 被打断（制造悬念）

3. 交织节奏
   - 不是轮流说话
   - 而是像音乐一样交织

群戏对白句式:
| 场景 | 对白模式 | 示例 |
|------|---------|------|
| 家庭餐桌 | 交叉说话 | "爸！妈她——" "吃饭！" |
| 会议 | 轮流+打断 | "我认为——" "不对，我觉得——" |
| 争吵 | 对立说话 | "你总是！" "你才总是！" |
| 行动前 | 简短确认 | "A？" "Ready。" "Go。" |
```

#### 方言与口音

```
方言处理的原则:

1. 不用方言写全部对白
   - 找到标志性的词汇/语法
   - 让读者/观众感知到，但不难以阅读

2. 避免刻板印象
   - 不用方言丑化角色
   - 尊重地域文化

3. 发音→用词的转化
   - 发音无法写（除非用语音合成）
   - 用词可以体现地域

方言词汇示例（供参考）:
| 地域 | 特征词汇 |
|------|---------|
| 东北 | 嘎嘎的、整、咋整、唠嗑 |
| 北京 | 嘛、压根儿、得瑟、局气 |
| 上海 | 阿拉、伐、灵光、老卵 |
| 广东 | 丢、我哋、咁、唔 |
| 台湾 | 诶、什么喔、很机车、揪咪 |
```

### 第四部分：AI视频生成的对白指南

#### 场景类型 → 对白风格

**AI生成时的对白提示词**:

| 场景类型 | 对白风格 | AI提示词 |
|----------|---------|---------|
| 亲密对话 | 简短、亲密、口语化 | intimate dialogue, short exchanges, subtext, emotional |
| 冲突争吵 | 打断、交叠、情绪化 | heated argument, overlapping dialogue, emotional tension |
| 正式场合 | 完整、礼貌、有距离 | formal speech, measured pace, professional distance |
| 动作场景 | 简短、命令式、无废话 | quick commands, minimal dialogue during action |
| 揭示场景 | 慢、信息密集、高张力 | slow revelation, dense information, high tension |
| 幽默场景 | 节奏感、口语化、意外 | witty banter, punchy rhythm, unexpected turns |
| 回忆/梦境 | 碎片化、延迟、混响 | fragmented speech, slowed, dreamlike quality |

#### 对白质量评估

```
AI生成对白的质量标准:

1. 角色声音区分度（30%）
   - 能否分辨出是谁在说话？
   - 每个角色是否独特？

2. 潜台词丰富度（25%）
   - 表面意思下是否有更多意思？
   - 观众是否需要思考？

3. 口语化程度（20%）
   - 听起来像真人会说的话？
   - 还是有"编剧腔"？

4. 信息密度合适度（15%）
   - 每句话的信息量是否合适？
   - 是否有过多的"解说腔"？

5. 节奏感（10%）
   - 对白是否有快慢变化？
   - 是否有自然的停顿？
```

## 决策逻辑

### 对白修改决策框架

```
收到改写需求时的诊断流程:

Step 1: 角色声音诊断
├── 这是谁在说话？
├── 这个角色的声音特征是什么？
├── 现在的对白是否像这个角色？

Step 2: 场景目的诊断
├── 这个场景要完成什么？
├── 每句对白是否服务于场景目的？
├── 有没有多余的"解说腔"？

Step 3: 潜台词诊断
├── 角色内心真正想说什么？
├── 现在的对白是否表达了潜台词？
├── 潜台词是否足够丰富但不晦涩？

Step 4: 节奏诊断
├── 对白的节奏是否合适？
├── 是否有自然的停顿？
├── 长短句是否搭配得当？

Step 5: 改写执行
├── 保留好的部分
├── 修改问题部分
└── 确保改写后更符合角色和场景
```
