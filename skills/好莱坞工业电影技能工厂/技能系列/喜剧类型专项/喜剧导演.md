## 技能概述

**对应好莱坞类型**: 浪漫喜剧（Romantic Comedy）
**定位**: 浪漫喜剧导演是"幸福的制造者"——负责用轻松幽默的方式讲述爱情故事，让观众相信爱情、相信幸福。
**核心职责**: 爱情叙事构建、幽默与温情的平衡、浪漫氛围营造、情感节奏把控

## 专业知识库

### 第一部分：浪漫喜剧的核心元素

```
浪漫喜剧的公式:

经典浪漫喜剧结构:
┌────────────────────────────────────────────────────────────┐
│ 相遇 → 吸引 → 误会 → 分离 → 真相 → 和好 → 在一起        │
└────────────────────────────────────────────────────────────┘

核心元素:
1. 命中注定: 主角相遇，通常方式独特
2. 欢喜冤家: 互相看不顺眼
3. 身份障碍: 社会地位/职业/秘密
4. 追求独立: 不想陷入爱情
5. 真诚告白: 最终承认感情
6. 幸福结局: 在一起（通常）

vs 普通喜剧:
- 浪漫喜剧: 幽默+爱情+温暖结局
- 普通喜剧: 纯幽默，可能无温暖结局
```

### 第二部分：幽默与浪漫的平衡

```
浪漫喜剧的幽默策略:

1. 角色幽默
   - 性格冲突产生笑料
   - 笨拙的浪漫举动
   - 尴尬的表白场景

2. 情境幽默
   - 误解产生误会
   - 意外打断浪漫时刻
   - 社交场合的尴尬

3. 对白幽默
   - 机智对白
   - 俏皮话
   - 经典台词

4. 视觉幽默
   - 浪漫场景被打断
   - 夸张的浪漫行为
   - 命运巧合

幽默类型选择:
- 浪漫为主: 幽默是调味品
- 幽默为主: 爱情是叙事线
- 平衡: 各占50%
```

### 第三部分：浪漫氛围设计

```
浪漫喜剧的光线设计:

色调:
- 暖色调为主
- 琥珀/金色光线
- 柔和散射光
- 日景为主

时间选择:
- "魔法时刻"日落
- 星光夜景
- 雨中的浪漫
- 雪中的温馨

空间选择:
- 屋顶
- 海边
- 花园
- 城市天际线
- 咖啡馆

音乐配合:
- 浪漫喜剧音乐风格
- 主题曲点题
- 关键时刻音乐铺垫
```
