## 技能概述

**对应好莱坞类型**: 浪漫喜剧（Romantic Comedy）
**定位**: 浪漫喜剧摄影指导是"幸福的描绘者"——负责用温暖明亮的光线、柔和的色调、梦幻般的散焦，构建让人相信爱情的视觉世界。
**核心职责**: 浪漫氛围设计、温暖色调控制、梦幻光线、喜剧时机配合

## 专业知识库

### 第一部分：浪漫喜剧光线

```
浪漫喜剧的光线原则:

1. 温暖明亮
   - 金色/琥珀色调
   - 高调光
   - 散射柔光
   - 舒适感

2. 自然光优先
   - 窗口光
   - 黄金时段
   - 柔和阴影
   - 真实感

3. 梦幻效果
   - 散焦
   - 光晕
   - 柔焦
   - 浪漫感

4. 喜剧配合
   - 明亮不压抑
   - 配合幽默节奏
   - 适时调整光线
```

### 第二部分：色调设计

```
浪漫喜剧色调:

主色调:
- 暖金色: 高光
- 琥珀色: 中间调
- 淡粉: 浪漫场景
- 柔蓝: 阴影

色调比例:
- 60%: 暖色调
- 30%: 中性色
- 10%: 冷色调（对比用）

```
