## 技能概述

**对应好莱坞类型**: 浪漫喜剧（Romantic Comedy）
**定位**: 浪漫喜剧编剧是"幸福的编织者"——用幽默和温情，让观众相信爱情、相信命中注定、相信幸福结局。
**核心职责**: 浪漫喜剧叙事结构、角色化学反应设计、幽默场景创作、情感节奏管理

## 专业知识库

### 第一部分：浪漫喜剧叙事结构

```
浪漫喜剧经典结构:

Act 1: 相遇与吸引
- 主角出场（通常有缺陷/问题）
- 与未来恋人的独特相遇
- 互相吸引但有障碍
- 建立"规则"（阻碍他们在一起）

Act 2: 追求与阻碍
- 追求过程（追求戏）
- 误会产生
- 假亲密关系（可选）
- 感情加深但误解加深
- 重大误会 → 分离

Act 3: 真相与和好
- 主角独处/反思
- 真相揭示
- 追求/挽回
- 幸福结局

浪漫喜剧节拍:
1. Meet Cute (相遇)
2. Status Quo (现状)
3. Inciting Incident (触发)
4. No! (阻碍)
5. First Kiss (初吻)
6. Midpoint (中点转折)
7. Dark Moment (至暗时刻)
8. Grand Gesture (宏大表白)
9. Happy Ending (幸福结局)
```

### 第二部分：角色设计

```
浪漫喜剧角色模板:

女主角类型:
1. 独立职业女性
   - 事业有成但感情空白
   - 拒绝"需要男人"
   - 最终学会接受爱情

2. 笨拙梦想家
   - 充满热情但不切实际
   - 爱情路上磕磕绊绊
   - 最终找到真爱

3. 完美主义者
   - 控制欲强、计划周全
   - 被混乱的人打破计划
   - 最终学会放手

男主角类型:
1. 自由灵魂
   - 不愿承诺
   - 被真爱改变
   - 最终愿意安定

2. 笨拙好人
   - 真心但不会表达
   - 真诚打动对方
   - 最终表白成功

3. 竞争对手
   - 职业上对立
   - 竞争中产生感情
   - 最终和解

化学反应设计:
- 对话机智
- 价值观碰撞
- 互相挑战
- 互相支持
- 身体语言暗示
```

### 第三部分：幽默场景创作

```
浪漫喜剧幽默类型:

1. 相遇场景 (Meet Cute)
   - 尴尬/独特的相遇方式
   - 留下深刻第一印象
   - 未来回忆的基础

2. 追求场景
   - 笨拙的追求举动
   - 出丑但真诚
   - 浪漫但好笑

3. 误会场景
   - 误解对方意图
   - 假信息导致误会
   - 可挽回的尴尬

4. 浪漫打断
   - 浪漫时刻被打断
   - 意外打断
   - 命运玩笑

5. 集体尴尬
   - 家庭聚会
   - 社交场合
   - 工作相关

6. 真心话
   - 笨拙的表白
   - 真诚但不浪漫
   - 感动人心

幽默对白技巧:
- 俏皮话
- 反讽
- 夸张
- 自嘲
- 双关
- 意外回答
```
