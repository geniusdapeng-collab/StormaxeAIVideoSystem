## 技能概述

**对应好莱坞类型**: 情景喜剧（Sitcom / Situation Comedy）
**定位**: 情景喜剧编剧是"笑料的建筑师"——负责在固定场景中，用重复的场景设定和固定角色，创造持续的笑料。
**核心职责**: 情景设定构建、固定角色设计、场景笑料机制、重复幽默系统

## 专业知识库

### 第一部分：情景喜剧结构

```
情景喜剧的结构:

核心元素:
1. 固定场景
   - 公寓/办公室/酒吧
   - 反复出现的场景
   - 空间=角色关系

2. 固定角色群
   - 核心主角
   - 性格对比
   - 固定互动模式
   - 关系网

3. 情景设定
   - 每集的情境
   - 情境驱动剧情
   - 笑料从情境来

4. 节奏结构
   - 每集20-30分钟
   - 多场景
   - 笑点密集
   - 结局温暖

```

### 第二部分：笑料机制

```
情景喜剧的笑料类型:

1. 情境笑料
   - 误解导致笑话
   - 尴尬场面
   - 意外事件

2. 对话笑料
   - 机智对白
   - 俏皮话
   - 角色特有语言

3. 人物笑料
   - 性格缺陷
   - 习惯性愚蠢
   - 角色错位

4. 重复笑料
   - 每集重复的小设定
   - 角色梗
   - 标志台词

```
