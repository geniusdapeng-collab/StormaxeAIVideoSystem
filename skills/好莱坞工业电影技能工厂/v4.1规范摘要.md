# SHANHAISTORY FORGE v4.1 核心规范摘要

> 来源：SHANHAISTORY FORGE / NIRATH v4.1 Professional Film Production Control
> 完整文档：用户提供的v4.1完整版

---

## 核心目标（五大原则）

1. **镜头意图清晰**：每个镜头必须知道"为什么存在"
2. **叙事服务优先**：运镜/光影/材质/声音都服务于叙事与情绪
3. **角色与世界观稳定**：角色一致性不等于完全不变，允许戏剧性变化
4. **成片可交付**：可审、可改、可剪、可交付
5. **整体优先于单镜炫技**：单镜优秀但不服务整体叙事，不视为合格

## Scene Card 场次执行卡

### 必填字段

| 字段 | 说明 |
|------|------|
| 场次编号 | 如SC01/SC02 |
| 场次功能 | 建立/推进/冲突/揭示/回收 |
| 场次目标 | 可验证的叙事结果 |
| 情绪曲线 | 时间轴上的连续变化，不是标签堆砌 |
| 场次光线总策略 | Light Tier A/B/C/D及变化逻辑 |
| 场次色彩策略 | Color Script，主色系统+强调色 |
| 轴线设定 | 主轴方向、视线关系、机位限制 |
| 节奏走势 | 慢入—蓄力—爆发—余韵 |

### 情绪曲线示例
- 好奇 → 不安 → 敬畏
- 平静 → 警觉 → 恐惧 → 求生

## Light Tier 光线分级

| 档位 | 名称 | 色温参考 | 反差比 | 用途 |
|------|------|---------|--------|------|
| A | 明亮探索 | 5600K日光 | 低2:1 | 探索、发现 |
| B | 神秘低照 | 3200-4000K | 中4:1 | 未知、悬疑 |
| C | 对抗高反差 | 混合色温 | 高8:1+ | 冲突、觉醒 |
| D | 神圣显现 | 非现实金/青色 | 极高16:1+ | 显灵、启示 |

## 运镜与复杂度联动

| 主体动作复杂度 | 推荐运镜复杂度 | 示例 |
|--------------|----------------|------|
| 高（打斗、变形） | 低（固定/简单推拉） | 刑天苏醒，固定机位 |
| 中（行走、对话） | 中（摇、移、缓推） | 小G穿越林地 |
| 低（静坐、凝视） | 高（环绕、升降） | 凝视神坛，缓慢环绕 |

## Prompt构建规范

### 推荐结构

```
1. 主体与绑定 (Subject & Character Anchor)
2. 主动作 (Primary Action)
3. 表演或反应重点 (Performance Focus)
4. 空间环境 (Spatial Environment)
5. 镜头语言 (Camera Language)
6. 光线与材质 (Lighting & Material)
7. 声音/对白（如有）
8. 收束锚点 (Closing Anchor / EFA)
```

### 长度控制

| 场景类型 | 建议长度 |
|----------|----------|
| 简单动作镜 | 120-220字符 |
| 标准叙事镜 | 220-420字符 |
| Hero Shot | 320-520字符 |

## 连续性模式

| 模式 | 定义 | 技术实现 |
|------|------|---------|
| strict | 帧级衔接 | first_frame/last_frame绑定 |
| soft | 状态延续，不强制同构图 | 角色锚点+Light Tier延续 |
| none | 仅叙事连贯 | 无特殊技术绑定 |

## 质量评审（五维评分）

| 维度 | 权重 |
|------|------|
| 可读性 | 25% |
| 可控性 | 20% |
| 可剪性 | 20% |
| 情绪命中率 | 20% |
| 记忆点 | 15% |

**75分生死线**：总分≥75且情绪≥15通过

## 屏幕方向规则

| 方向 | 含义 |
|------|------|
| 左→右 | 前进/接近/主动 |
| 右→左 | 受阻/退却/逆行 |
| 向镜头 | 压迫/逼近/降临 |
| 离镜头 | 离开/被吞没/失落 |

---

## 转场意图类型

- 动作接续（Action Match Cut）
- 视线接续（Eyeline Match）
- 光线接续（Light Bridge）
- 情绪接续（Emotional Continuity）
- 声音先接（Sound Lead / L-Cut）
- 空间对比（Contrast Cut）
- 时间跳转（Time Jump）

---

## 五大禁止

1. 不允许跳过Scene Card与Shot Card直接写Prompt
2. 不允许绕过导演审片直接渲染
3. 不允许以单次生成结果代替正式镜头设计
4. 不允许单镜决策破坏场次整体成立
5. 不允许Prompt主体堆叠过多negative约束
