## 🚀 新增功能

### BidEval 内容比稿技能（v1.0.0-Peng）
**定位**：渲染配额是稀缺资源，LLM是无限的。在提交渲染前做"比稿闸门"，生成N个方案 → 评测选优 → 只送最优的去渲染。

**四大评测维度**：

| 维度 | 权重 | 子维度 |
|------|------|--------|
| 需求对齐度 | 30% | 主题覆盖、情感基调、关键场景、角色设定、规格匹配 |
| 剧本质量 | 25% | 结构清晰度、角色弧光、冲突设计、节奏感、转场自然度 |
| Seedance合规 | 25% | 提示词长度、6步公式、敏感内容、参数合法性、角色一致性 |
| 艺术感染力 | 20% | 画面想象力、情感共鸣、独特性、镜头语言、张力节奏 |

**子评估器**（4个）：
- `need-alignment.js` — 需求对齐度
- `script-quality.js` — 剧本质量
- `seedance-compliance.js` — Seedance 合规性
- `artistic-impact.js` — 艺术感染力

**集成方式**：
- Phase 3.7 集成到 director.js
- `--bid-count` 参数支持（默认1-不比稿，2-4-多方案评测）
- 比稿结果反馈给生成系统用于优化

### 四板块架构切分
1. **方案脚本制作系统**：分析用户需求，产出N个待评审脚本方案
2. **BidEval比稿闸门**：从N个方案中选出最佳方案，并给出修改意见
3. **渲染引擎**：分批次送去 Seedance 2.0 进行渲染
4. **后期制作**：把渲染回来的片段进行制作合成成片交付
