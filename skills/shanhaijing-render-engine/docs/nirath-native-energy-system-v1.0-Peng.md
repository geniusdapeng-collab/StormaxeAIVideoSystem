# Nirath原生能量体系统 v1.0-Peng

## 🚫 禁用声明

**Nirath世界绝对禁止"水晶"元素**：
- ❌ 水晶柱状体 / 水晶簇 / 水晶球
- ❌ 晶簇山脉 / 晶尘粒子 / 晶体结构
- ❌ 任何透明/折射/棱镜效果的矿物形态

**原因**：水晶概念在科幻/奇幻作品中被过度使用，已失去独特性。Nirath必须有自己独特的能量表现形式。

---

## ✅ Nirath原生能量体 —「光脉/虹脉」系统

### 核心概念

Nirath星球的能量不是以"水晶"形式储存，而是以**地质光脉网络**和**虹脉能量场**的形式存在。这是Nirath星球地质演化的自然结果，与地球水晶完全不同的物理机制。

### 1. 光脉岩柱 (Luminous Vein Rock)

**物理本质**：
- 地壳断裂带中的矿物脉络，因Nirath双恒星辐射激发而持续发光
- 不是透明晶体，而是**岩石内部的发光脉络**，类似熔岩裂缝但冷光
- 发光原理：双日光谱中的特定波长激活矿物中的荧光物质

**视觉特征**：
- 外观：暗色岩石基质 + 内部发光的脉络网络
- 颜色：紫金（富含锰矿物）、琥珀（含铁矿物）、蓝绿（含铜矿物）
- 质感：粗糙岩石表面，裂缝中渗出柔和光晕
- 非透明：光从裂缝渗出，而非穿透整体

**Prompt用语**：
```
光脉岩柱：暗色火山岩基质中，紫金色矿物脉络如血管般分布，双日暮光激发下内部发光，裂缝渗出柔和光晕，粗糙地质纹理，绝非水晶透明质感
```

### 2. 虹脉核心 (Halo Core)

**物理本质**：
- Nirath行星磁场在特定地质应力点聚焦形成的能量漩涡
- 不是球体水晶，而是**旋转的能量场**，边缘有光脉辐射状延伸
- 类似地球极光但固定在地质节点上

**视觉特征**：
- 外观：球状能量漩涡，核心明亮，边缘光脉如触须辐射
- 颜色：虹彩渐变，随磁场强度脉动
- 质感：能量场的流动感，非固体，非透明
- 动态：缓慢旋转，光脉如呼吸般明暗变化

**Prompt用语**：
```
虹脉核心：Nirath地脉节点上，旋转的能量漩涡球，核心紫金色光芒，边缘虹彩光脉辐射状延伸如触须，磁场脉动造成明暗呼吸效应，绝非水晶球，而是地质能量场的可视化
```

### 3. 浮空光脉山脉 (Floating Lumina Range)

**物理本质**：
- Nirath地壳断裂后，富含光脉矿物的岩层因磁场反重力效应悬浮
- 不是"浮空晶簇"，而是**地质板块碎片**，内部光脉网络发光

**视觉特征**：
- 外观：巨大岩石板块，内部光脉网络如神经网络
- 颜色：暗岩基质 + 内部紫金/琥珀光脉
- 质感：厚重岩石，非轻盈晶体
- 动态：缓慢漂移，光脉随板块运动明暗变化

**Prompt用语**：
```
浮空光脉山脉：地壳断裂的岩石板块悬浮空中，内部紫金光脉网络如神经网络般分布，双日暮光下裂缝发光，厚重地质质感，磁场漂移缓慢移动，绝非水晶浮空岛
```

### 4. 虹脉孢子 (Halo Spores)

**物理本质**：
- Nirath生物发光器官释放的孢子与地质光脉粉尘混合
- 不是"晶尘粒子"，而是**生物-地质混合发光微粒**

**视觉特征**：
- 外观：微小发光颗粒，在空中漂浮
- 颜色：粉金色（生物发光）+ 紫金（地质粉尘）混合
- 质感：柔软光点，非硬晶体
- 动态：随气流漂浮，受生物电场影响轻微脉动

**Prompt用语**：
```
虹脉孢子：生物发光孢子与地质光脉粉尘混合的微粒，粉金色与紫金双色漂浮，受生物电场影响轻微脉动，柔软光点质感，绝非晶尘粒子
```

### 5. 地脉节点 (Geothermal Node)

**物理本质**：
- Nirath地质应力集中点，光脉网络交汇处的能量溢出点
- 不是"能量水晶"，而是**地质能量溢出的自然现象**

**视觉特征**：
- 外观：地面裂缝中的光脉交汇，形成发光的节点
- 颜色：多色光脉交汇处的虹彩效果
- 质感：地面裂缝中的发光液体/气体溢出
- 动态：光脉如心跳般脉动，能量溢出如呼吸

**Prompt用语**：
```
地脉节点：地面地质裂缝中，多色光脉交汇形成发光节点，虹彩能量如液态光溢出，脉动节奏如行星心跳，地质能量场的自然可视化，绝非人工能量水晶
```

## 在Prompt中的使用原则

### 必须包含的否定声明
```
NO crystal NO transparent mineral NO crystal cluster NO crystal ball NO prismatic refraction
```

### 必须包含的肯定描述
```
Nirath native luminous vein rock, geological energy manifestation, 
internal light-emitting mineral networks, non-crystalline geological formations,
ambient glow from rock fractures, planetary geothermal phenomena
```

- ❌ 晶莹剔透
- ❌ 透明如冰
- ❌ 折射光芒

- ❌ 晶簇丛生

- ✅ 粗糙地质
