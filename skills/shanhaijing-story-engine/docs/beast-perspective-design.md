## 一、核心原则：感官体验 ≠ 第三人称解说

### 错误示范（第三人称解说）
```
镜头1：小G穿越峡谷，发现远古脚印
镜头2：刑天苏醒，注视小G
台词：刑天说"又一个闯入者"
```
**问题**：观众在"看纪录片"，不是"体验战魂记忆"。

### 正确示范（感官体验）
```
镜头1：战魂感应到地面微微震动，那么轻，像三千年前的猎户孩子
镜头2：战魂从岩石中渗出，看到一个渺小的生命——他不怕我吗？
台词："又一个脚步声...岩石记得，我不记得了"
```
**效果**：观众在刑天的感知里，感受他的孤独、厌倦、好奇。

## 二、三层错误定位

### 上游：Story Engine（故事引擎）
**问题**：默认第三人称客观视角
**修复**：增加 `perspectiveMode: 'beast-first-person'` 参数

```javascript
// story-engine.js
const STORY_CONFIG = {
  perspectiveMode: 'beast-first-person', // 新增
  // 默认：'third-person-objective'
  // 可选：'protagonist-first-person', 'beast-first-person', 'narrator-omniscient'
};
```

**视角模式定义**：
- `third-person-objective`：旁白客观解说（默认，不推荐用于神兽故事）
- `protagonist-first-person`：主角视角（孩子看到的世界）
- `beast-first-person`：神兽视角（战魂感受到的世界）← **推荐**
- `narrator-omniscient`：全知旁白（适用于科普/纪录片）

### 中游：Dialogue Engine（台词引擎）
**问题**：按幕分配角色（起幕旁白、承幕神兽、高潮主角）
**修复**：情感驱动台词，所有台词来自战魂低语

```javascript
// dialogue-engine.js
const DIALOGUE_MODE = {
  mode: 'beast-monologue', // 新增
  // 默认：'multi-character'（多角色分配）
  // 可选：'beast-monologue', 'protagonist-monologue', 'narrator-driven'
};
```

**神兽独白模式规则**：
1. 所有台词来自战魂的"内心活动"，不是"对外说话"
2. 没有"旁白"这个角色
3. 主角（孩子）极少说话，只在最动情时刻说一句
4. 台词情绪弧线：厌倦→困惑→触动→传承→释然

### 下游：Render Engine（渲染引擎）
**问题**：Prompt生成只拼接技术参数，没有把视角变成视觉语言
**修复**：在Prompt中注入"视角质感"

```javascript
// seedance-render-engine.js
const PERSPECTIVE_ANCHOR = {
  'beast-first-person': {
    cameraStyle: '从战魂的感知中拍摄，画面带有能量波动的微光质感',
    subjectDescription: '孩子的身影渺小如尘埃，在巨人的记忆中飘过',
    lightingNote: '光效反映战魂情绪：厌倦=暗淡，触动=脉动，传承=爆发'
  }
};
```

### 视角质感注入位置

```

[场景描述...]
[SCENE-ENRICHED...]

```

## 四、情感弧线设计

### 刑天战魂的五幕情感

| 幕 | 情感 | 能量状态 | 台词质感 | 画面质感 |
|----|------|---------|---------|---------|
| 起 | 厌倦中夹杂好奇 | 沉睡→微醒 | 低吟，像从大地深处传来 | 暗淡，微光 |
| 承 | 试探与困惑 | 防御→犹豫 | 沙哑，试探性的震动 | 明暗交替 |
| 转 | 触动与震动 | 共振→涌动 | 震颤，不敢相信 | 脉动光效 |
| 高潮 | 传承与爆发 | 峰值→传递 | 轰鸣，能量峰值 | 光芒爆发 |
| 合 | 释然与消散 | 退潮→余烬 | 余韵，像风中残烛 | 渐暗，但中心有光 |

## 五、声音设计规范

### 刑天声线：三层混合
1. **腹部共鸣**（基础层）：低沉的持续嗡鸣，像大地在呼吸
2. **脐部发声**（情感层）：沙哑的震动，随情绪起伏
3. **心灵感应**（信息层）：不是"听到"，是"感受到"的意念

### 声线参数
```javascript
const BEAST_VOICE_PROFILE = {
  speed: 0.7,      // 缓慢
  pitch: 0.8,      // 低沉
  emotion: 'epic', // 史诗
  reverb: 0.6,     // 强混响
  layers: ['abdominal-resonance', 'navel-vibration', 'telepathic-pulse']
};
```

## 六、台词质量检查清单

### 每句台词必须回答
1. **这是战魂在"感受"还是"解说"？** → 必须是感受
2. **有没有具体的记忆闪回？** → 必须有画面（猎户孩子/黄帝战士/母亲）
3. **有没有情感递进？** → 不能和上一句同情绪
4. **如果删掉这句，故事会断吗？** → 必须是必要信息

### 禁止出现的台词
- ❌ "我是刑天，上古战神"（自我介绍，解说）
- ❌ "这个孩子很勇敢"（评价，外部视角）
- ❌ "我要把战魂传给他"（计划，理性思维）
- ✅ "他的心跳和战魂在共振...像斩首前最后一瞬的心跳"（感受+记忆）
