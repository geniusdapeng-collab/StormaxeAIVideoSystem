## TL;DR

数据可视化 = 选择图表 + 设计布局 + 优化细节 + 讲好故事

## 1. 图表选择

### 1.1 图表分类
```
图表类型选择：

比较类：
├── 柱状图：类别间比较
├── 条形图：横向比较，类别多
├── 折线图：时间序列趋势
├── 雷达图：多维度对比
└── 词云图：文本词频对比

趋势类：
├── 折线图：连续变化
├── 面积图：累积趋势
├── 堆叠面积：部分趋势
├── 迷你图：空间节省
└── 回归线：趋势线

构成类：
├── 饼图：占比展示（<5类）
├── 环形图：饼图变体
├── 堆叠柱状图：时间构成
├── 瀑布图：变化构成
└── 树图：层级构成

关系类：
├── 散点图：两变量关系
├── 气泡图：三变量关系
├── 热力图：交叉关系
├── 桑基图：流向关系
└── 网络图：复杂关系

分布类：
├── 直方图：数值分布
├── 箱线图：统计分布
├── 小提琴图：密度分布
├── 点图：离散分布
└── 地图：地理分布
```

### 1.2 选择原则
```
图表选择原则：

数据维度：
├── 1维数据：单个数字/KPI
├── 2维数据：X-Y散点/折线
├── 3维数据：气泡图/3D图
├── 多维数据：雷达图/热力图
└── 层级数据：树图/旭日图

比较维度：
├── 类别比较：柱状图
├── 时间比较：折线图
├── 区域比较：地图
└── 多维度：雷达图

关系维度：
├── 相关性：散点图
├── 分布：直方图/箱线图
├── 构成：饼图/堆叠图
└── 流向：桑基图/流程图

受众需求：
├── 详细分析：表格+图表
├── 概览展示：大数字+KPI
├── 探索分析：交互图表
└── 讲故事：故事化图表
```

## 2. 设计原则

### 2.1 设计基础
```
可视化设计原则：

有效性：
├── 展示数据，不展示无关信息
├── 减少认知负担
├── 突出关键信息
└── 一图一结论

简洁性：
├── 删除无关元素
├── 减少视觉干扰
├── 留白适当
└── 避免过度装饰

准确性：
├── 保持数据比例
├── 不截断坐标轴
├── 不使用双Y轴
└── 图例清晰

可读性：
├── 字体大小合适
├── 颜色对比清晰
├── 标签完整
└── 注释说明

美观性：
├── 配色协调
├── 排版整齐
├── 风格统一
└── 比例协调
```

### 2.2 配色设计
```
配色设计指南：

配色原则：
├── 使用品牌色系
├── 避免彩虹色系
├── 颜色数量不超过7种
├── 考虑色盲友好
└── 明暗搭配

图表配色：
├── 连续色系：渐变色，从浅到深
├── 离散色系：不同色相，易区分
├── 强调色：关键数据用亮色
└── 背景色：浅色或白色

常用配色方案：
├── 单色系：蓝/绿/橙，深浅变化
├── 双色系：蓝橙互补
├── 多色系：彩虹色表，热力图
└── 灰度系：黑白灰打印友好

色盲友好：
├── 避免红绿组合
├── 使用蓝橙组合
├── 加图案/形状区分
└── 使用饱和度区分
```

## 3. 常见图表

### 3.1 柱状图
```
柱状图设计：

基础柱状图：
├── 直立柱形
├── 柱宽一致
├── 柱间距小于柱宽
└── 从0开始

横向柱状图：
├── 类别名称长时使用
├── 从左到右阅读
├── 降序排列
└── 顶部显示数值

分组柱状图：
├── 多系列比较
├── 颜色区分系列
├── 限制在3-4组
└── 图例清晰

堆叠柱状图：
├── 展示构成
├── 顶部对齐
├── 限制在4-5段
└── 标注关键数值

注意事项：
├── 避免3D效果
├── 不截断柱子
├── 标签完整
└── 排序有逻辑
```

### 3.2 折线图
```
折线图设计：

基础折线图：
├── 点和线连接
├── 线型清晰
├── 标记关键点
└── 标注数据点

多线图：
├── 最多5-7条线
├── 颜色区分
├── 虚实区分
└── 避免交叉过多

面积图：
├── 填充颜色
├── 半透明效果
├── 堆叠面积
└── 避免太多层

迷你图：
├── 无坐标轴
├── 占用空间小
├── 趋势展示
└── 配合大数字

注意事项：
├── 纵轴从0开始
├── 线型一致
├── 点大小适中
└── 数据点不过密
```

### 3.3 饼图
```
饼图设计：

基础饼图：
├── 12点钟方向开始
├── 顺时针排列
├── 大段顺时针排列
└── 最大段在右上

环形图：
├── 中间可放汇总
├── 更现代感
├── 数据清晰
└── 推荐使用

复合饼图：
├── 主图+副图
├── 展示细节
├── 饼中饼结构
└── 避免复杂

注意事项：
├── 不超过5-7段
├── 不做3D饼图
├── 不比较相似值
├── 标注百分比
└── 可用条形图替代
```
