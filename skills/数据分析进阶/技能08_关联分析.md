## TL;DR

关联分析 = 相关性分析 + 因果推断 + 回归分析 + 关系验证 + 应用落地

## 1. 相关性分析

### 1.1 相关系数
```
常用相关系数：

Pearson相关系数：
├── 衡量线性相关
├── 取值：-1到1
├── r = 1：完全正相关
├── r = -1：完全负相关
├── r = 0：不相关
├── 假设：连续变量、正态分布、线性关系
├── 公式：Cov(X,Y) / (σx * σy)
└── 适用：连续变量

Spearman秩相关系数：
├── 衡量单调相关
├── 基于秩次计算
├── 不假设分布
├── 对异常值不敏感
├── 公式：ρ = 1 - 6Σd²/n(n²-1)
└── 适用：有序数据、偏态数据

Kendall τ相关系数：
├── 基于秩次的等级相关
├── 计算一致对和不一致对
├── 对数据中的并列敏感
├── 公式：τ = (C - D) / (n(n-1)/2)
└── 适用：小样本、有序数据

点二列相关：
├── 一个二元变量 + 一个连续变量
├── 公式：rpb = (Mp - Mq) / σ * √(pq)
└── 适用：0-1变量与连续变量

Cramer's V：
├── 两个分类变量相关
├── 取值：0到1
├── 公式：V = √(χ² / (n * min(r-1,c-1)))
└── 适用：分类变量
```

### 1.2 相关性解读
```
相关系数解读：

强度判断：
|r| = 0.0-0.2：几乎无相关
|r| = 0.2-0.4：弱相关
|r| = 0.4-0.6：中等相关
|r| = 0.6-0.8：强相关
|r| = 0.8-1.0：非常强相关

相关方向：
正向相关：r > 0，一个↑另一个↑
负向相关：r < 0，一个↑另一个↓

显著性检验：
p < 0.05：相关显著
p < 0.01：相关高度显著
p ≥ 0.05：相关不显著

解读注意事项：
├── 相关不等于因果
├── 可能存在伪相关
├── 可能存在混淆变量
├── 非线性关系可能相关低
├── 数据范围影响相关性
└── 样本量影响显著性
```

## 2. 因果推断

### 2.1 因果关系
```
因果vs相关：

相关关系：
A和B同时变化
可能是因果，可能不是

因果关系：
A变化导致B变化
原因在前，结果在后
其他因素保持不变

因果推断困难：
├── 无法观测反事实
├── 混淆变量
├── 选择偏差
├── 双向因果
└── 时间滞后

因果推断方法：
├── 随机实验
├── 准实验设计
├── 工具变量
├── 倾向得分匹配
├── 双重差分
├── 断点回归
└── 因果图模型
```

### 2.2 因果推断方法
```
因果推断方法选择：

随机实验（A/B测试）：
├── 金标准
├── 随机分组，控制混淆
├── 内部效度高
├── 外部效度可能受限
└── 适用：可实验的场景

双重差分（DID）：
├── 处理组vs对照组
├── 控制时间趋势
├── 平行趋势假设
├── 公式：(Y处理后 - Y处理前) - (Y对照后 - Y对照前)
└── 适用：政策评估、活动评估

倾向得分匹配（PSM）：
├── 匹配相似的个体
├── 控制选择偏差
├── 最近邻匹配/半径匹配
├── 卡尺匹配
└── 适用：观察性研究

工具变量（IV）：
├── 使用外生变量
├── 两阶段最小二乘
├── 工具变量选择困难
├── 需要弱工具变量检验
└── 适用：内生性问题

断点回归（RDD）：
├── 临界点两侧比较
├── 局部随机化
├── 连续性假设
└── 适用：临界值场景
```

## 3. 回归分析

### 3.1 回归基础
```
回归分析类型：

线性回归：
├── 一元线性：Y = a + bX
├── 多元线性：Y = a + b1X1 + b2X2 + ...
├── 假设：线性、独立、正态、同方差
├── R²：解释力度
├── 系数：影响方向和大小
└── 适用：连续因变量

Logistic回归：
├── Y = 1 / (1 + e^-(a + bX))
├── OR值：优势比
├── 预测概率
├── 分类决策边界
└── 适用：二元分类

多元回归：
├── 多个自变量
├── 控制其他变量
├── 共线性问题
├── 变量选择
└── 适用：复杂关系

逐步回归：
├── 前向逐步
├── 后向逐步
├── 逐步筛选
├── AIC/BIC准则
└── 适用：变量筛选
```

### 3.2 回归诊断
```
回归诊断要点：

模型拟合：
├── R²：解释方差比例
├── 调整R²：考虑变量数
├── F检验：模型整体显著
├── t检验：系数显著
└── 残差分析

残差诊断：
├── 残差图：检查线性
├── QQ图：检查正态
├── 尺度图：检查同方差
├── 杠杆值：检查异常点
└── Durbin-Watson：检查自相关

多重共线性：
├── VIF > 10：存在共线性
├── 相关系数矩阵
├── 特征根分析
└── 处理：删除/合并/正则化

异方差检验：
├── White检验
├── Breusch-Pagan检验
├── 处理：加权最小二乘
└── 稳健标准误

异常值处理：
├── 学生化残差
├── Cook距离 > 4/n
├── 异常点分析
└── 决定是否剔除
```


### 4.2 常见陷阱
```
关联分析常见陷阱：

伪相关：
├── 两个变量共同受第三变量影响
├── 例如：冰淇淋销量和溺水率（夏季）
├── 解决：控制第三变量

虚假相关：
├── 纯属巧合
├── 数据范围过窄/过宽
├── 解决：扩大数据范围

因果倒置：
├── 把结果当原因
├── 例如：成绩好的学生更努力学习
├── 解决：时序验证

混淆变量：
├── 未控制的第三变量
├── 影响自变量和因变量
├── 解决：加入控制变量

选择偏差：
├── 样本选择偏差
├── 幸存者偏差
├── 自选择偏差
└── 解决：随机实验

中介效应：
├── A→B→C关系
├── A对C的直接/间接效应
├── 需要检验中介
└── 解决：中介分析
```
