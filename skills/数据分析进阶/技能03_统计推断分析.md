## TL;DR

统计推断 = 描述统计 + 概率分布 + 假设检验 + 置信区间 + 效果量

## 1. 描述统计

### 1.1 中心趋势
```
中心趋势指标：

均值（Mean）：
├── 计算：所有值求和除以个数
├── 优点：利用全部数据
├── 缺点：受极端值影响
└── 适用：对称分布

中位数（Median）：
├── 计算：排序后中间位置的值
├── 优点：不受极端值影响
├── 缺点：不利用全部数据
└── 适用：偏态分布

众数（Mode）：
├── 计算：出现最多的值
├── 优点：直观易懂
├── 缺点：不唯一
└── 适用：分类数据

选择建议：
├── 正态分布：均值
├── 偏态分布：中位数
├── 分类数据：众数
└── 多峰分布：多众数
```

### 1.2 离散程度
```
离散程度指标：

极差（Range）：
├── 定义：最大值 - 最小值
├── 优点：计算简单
├── 缺点：受极端值影响
└── 不常用

方差（Variance）：
├── 定义：离均差平方的平均
├── 优点：利用全部数据
├── 缺点：量纲是原数据的平方
└── 样本方差：除以n-1

标准差（Std）：
├── 定义：方差的平方根
├── 优点：与原数据量纲一致
├── 缺点：受极端值影响
└── 最常用

四分位距（IQR）：
├── 定义：Q3 - Q1
├── 优点：不受极端值影响
├── 缺点：丢失中间信息
└── 适用：识别异常值

变异系数（CV）：
├── 定义：标准差/均值
├── 优点：可比较不同数据集
├── 缺点：均值不能为0
└── 适用：比较离散程度
```

## 2. 概率分布

### 2.1 常用分布
```
常用概率分布：

正态分布：
├── 特征：钟形曲线，均值=中位数=众数
├── 参数：均值μ、标准差σ
├── 68-95-99.7法则
├── 应用：身高、测量误差、考试成绩

二项分布：
├── 特征：n次独立试验的成功次数
├── 参数：n（次数）、p（成功率）
├── 应用：抛硬币、转化率、点击率

泊松分布：
├── 特征：单位时间/面积内的事件次数
├── 参数：λ（平均发生率）
├── 应用：客服来电、网页点击、设备故障

指数分布：
├── 特征：事件间隔时间
├── 参数：λ（率参数）
├── 应用：等待时间、寿命分析

分布选择：
├── 连续数据，正态分布 → 正态检验
├── 计数数据 → 泊松分布
├── 二元结果 → 二项分布
└── 时间间隔 → 指数分布
```

### 2.2 中心极限定理
```
中心极限定理：

核心思想：
样本量足够大时，样本均值的分布近似正态分布

应用条件：
├── 样本量 n ≥ 30
├── 样本是独立同分布
└── 总体分布不限

实际应用：
├── 用样本推断总体
├── 计算置信区间
├── 进行假设检验
└── 预测总体均值

注意事项：
├── 样本量越大越好
├── 异常值要处理
├── 要有代表性
└── 独立性要满足
```

## 3. 假设检验

### 3.1 假设检验流程
```
假设检验六步法：

Step 1：明确问题
├── 原假设（H0）：通常是无差异/无效果
├── 备择假设（H1）：通常是有差异/有效果
└── 检验类型：双尾 or 单尾

Step 2：选择检验方法
├── 单样本t检验：一个均值与已知值比较
├── 两独立样本t检验：两组均值比较
├── 配对样本t检验：同一组前后比较
├── 卡方检验：比例/分布比较
├── ANOVA：多组均值比较
└── 相关性检验：变量关系

Step 3：确定显著性水平
├── α = 0.05（常用）
├── α = 0.01（严格）
├── α = 0.10（宽松）
└── 第一类错误 vs 第二类错误

Step 4：计算统计量
├── t = (x̄ - μ) / (s / √n)
├── z = (p̂ - p) / √(p(1-p)/n)
├── χ² = Σ(O-E)²/E
└── F = 组间方差/组内方差

Step 5：计算P值
├── P < α → 拒绝H0
├── P ≥ α → 不拒绝H0
└── P值越小，证据越强

Step 6：得出结论
├── 统计结论
├── 业务结论
└── 建议和行动
```

### 3.2 常用检验方法
```
常用检验方法选择：

两组比较：
| 数据类型 | 样本关系 | 方法 |
|----------|----------|------|
| 连续 | 独立 | 两独立样本t检验 |
| 连续 | 配对 | 配对样本t检验 |
| 二元 | 独立 | 卡方检验/Fisher精确 |
| 有序 | 独立 | Mann-Whitney U |
| 有序 | 配对 | Wilcoxon符号秩 |

多组比较：
| 数据类型 | 方法 |
|----------|------|
| 连续 | 单因素ANOVA |
| 连续 | Kruskal-Wallis H |
| 比例 | 卡方检验 |
| 有序 | Jonckheere-Terpstra |

相关分析：
| 数据类型 | 方法 |
|----------|------|
| 连续-连续 | Pearson相关 |
| 有序-有序 | Spearman相关 |
| 二元-连续 | 点二列相关 |
| 分类-连续 | Eta相关 |
```

## 4. 置信区间

### 4.1 置信区间原理
```
置信区间原理：

定义：
有95%的置信度认为总体参数落在区间内

计算公式：
均值置信区间：
x̄ ± t(α/2, n-1) × s/√n

比例置信区间：
p̂ ± z(α/2) × √(p̂(1-p̂)/n)

样本量计算：
n = (z × σ / E)²
其中 E 为允许误差

影响因素：
├── 置信水平：越高，区间越宽
├── 样本量：越大，区间越窄
├── 数据变异：越大，区间越宽
└── 允许误差：越小，区间越窄
```

### 4.2 置信区间应用
```
置信区间应用场景：

1. 业务指标估计
"我们有95%的把握，转化率在3.2%-3.8%之间"

2. AB测试结果
"新方案转化率提升1.2%，95%置信区间为[0.8%, 1.6%]，
由于区间不包含0，提升是显著的"

3. 用户调研
"用户满意度为4.2分，95%置信区间为[4.0, 4.4]，
说明满意度在中等偏上水平"

4. 预测区间
"下月销售额预测值500万，95%预测区间[450, 550]万"

解读注意事项：
├── 不是"95%的数据落在此区间"
├── 是"95%的置信区间包含真值"
├── 样本量影响区间宽度
└── 多个置信区间不一定包含真值
```

## 5. 效果量与统计功效

### 5.1 效果量
```
效果量指标：

Cohen's d（均值差异）：
├── d = 0.2：小效果
├── d = 0.5：中效果
├── d = 0.8：大效果
└── 计算：(x̄1 - x̄2) / s_pooled

Pearson r（相关系数）：
├── r = 0.1：小效果
├── r = 0.3：中效果
├── r = 0.5：大效果
└── 计算：协方差/标准差乘积

R²（决定系数）：
├── R² = 0.01：小效果
├── R² = 0.09：中效果
├── R² = 0.25：大效果
└── 解释：X解释Y的方差比例

OR值（优势比）：
├── OR = 1：无效果
├── OR > 1：正向效果
├── OR < 1：负向效果
└── 计算：odds1 / odds2

报告建议：
├── 报告统计显著性的同时报告效果量
├── 效果量比P值更重要
├── 小样本容易不显著但效果量大
└── 大样本容易显著但效果量小
```

### 5.2 统计功效
```
统计功效分析：

功效（Power）：
├── 定义：正确拒绝H0的概率
├── 通常设为 0.8 或 0.9
└── 功效越高，越不容易漏掉真效应

影响因素：
├── 样本量：越大，功效越高
├── 显著性水平：α越大，功效越高
├── 效果量：越大，功效越高
└── 总体方差：越小，功效越高

样本量计算：
根据目标功效、显著性水平、效果量计算所需样本量

应用场景：
├── 实验设计前计算样本量
├── 评估实验检测能力
├── 判断实验是否值得做
└── 解释不显著的结果

注意事项：
├── 功效不足 → 容易漏掉真效应
├── 功效过高 → 浪费资源
├── 0.8是常用标准
└── 不同场景要求不同
```
