## 1. AI写作场景

### 1.1 写作场景分类

AI写作场景：

商务写作：
├── 邮件撰写：工作邮件、商务邮件

├── 方案撰写：项目方案、商务提案
├── 合同撰写：合同条款、协议
└── 公文撰写：通知、公告、函件

营销写作：
├── 广告文案：品牌口号、广告语
├── 营销文章：推广软文、产品文案
├── 社交媒体：微博、公众号、小红书

内容创作：
├── 文章写作：博客文章、专栏

├── 用户指南：使用说明、操作手册

## 2. AI写作方法

### 2.1 写作流程
```
AI辅助写作流程：

Step 1：明确写作目的
├── 写给谁看
├── 要达成什么目的
├── 需要传达什么信息
└── 期望什么行动

Step 2：准备素材
├── 相关背景资料
├── 参考范文
├── 数据支撑
├── 关键信息点
└── 风格参考

Step 3：AI初稿生成
├── 选择合适的Prompt模板
├── 填写关键信息
├── 设置格式要求
├── 设定字数要求
└── 生成初稿

Step 4：人工审核
├── 检查内容准确性
├── 调整语气风格
├── 补充AI遗漏的信息
├── 删除多余内容
└── 优化逻辑结构

Step 5：修改完善
├── 润色语言
├── 调整格式
├── 添加个人风格
├── 终审校对
└── 最终定稿
```

### 2.2 Prompt技巧
```
AI写作Prompt技巧：

基础结构：
角色 + 任务 + 背景 + 要求 + 格式

技巧1：角色设定
"你是一位资深产品经理，擅长写PRD文档"
"你是一位专业文案，擅长写营销文案"

技巧2：示例引导
"参考以下风格：
[示例1]
[示例2]
请按类似风格撰写..."

技巧3：结构化输出
"请按以下结构输出：
1. 背景
2. 问题
3. 解决方案
4. 预期效果"

技巧4：迭代优化
"请先写一版大纲，我确认后再写完整内容"
"针对第3点，再详细展开一下"
"语气可以更正式一些"

技巧5：限制约束
"字数控制在500字以内"
"使用通俗易懂的语言，避免专业术语"
"不要使用emoji"
```

## 3. 写作场景实战

### 3.1 工作报告
```
工作报告写作：

周报Prompt：
"请帮我写一份周报，包含以下内容：
[本周完成的工作1]
[本周完成的工作2]
[本周遇到的问题]
[下周工作计划]

要求：
1. 成果要量化（用数据说话）
2. 问题要说明原因和解决方案
3. 计划要具体可执行
4. 字数300字左右
5. 格式清晰，使用项目符号"

月度总结Prompt：
"请帮我写一份月度工作总结，主题是[主题]。

包含：
1. 本月主要工作成果（3-5个亮点）
2. 遇到的问题和解决方案
3. 个人成长和收获
4. 下月工作计划和目标

要求：
1. 成果要突出，体现价值
2. 问题要客观，有深度分析
3. 成长要有具体事例
4. 计划要SMART
5. 字数800-1000字"
```

### 3.2 营销文案
```
营销文案写作：

产品文案Prompt：
"你是一位资深营销文案专家。请帮我写产品文案。

产品信息：
- 产品名称：[名称]
- 核心卖点：[卖点1、卖点2、卖点3]
- 目标用户：[用户画像]
- 使用场景：[场景描述]

要求：
1. 开头要有吸引力，抓住注意力
2. 突出产品核心卖点
3. 解决用户痛点
4. 唤起用户情感共鸣
5. 有明确的行动号召
6. 字数200-300字

风格：年轻、活力、有感染力"

小红书文案Prompt：
"请帮我写一篇小红书文案，推广[产品/服务]。

内容要点：
- 真实使用体验
- 解决什么问题
- 个人感受
- 推荐理由

要求：
1. 口语化，像真人分享
2. 开头要吸引人（钩子）
3. 中间有具体细节
4. 结尾有互动引导（提问/呼吁）
5. 适当使用emoji
6. 字数500-800字
7. 加Tag：[相关话题]"
```

### 3.3 内容创作
```
内容创作写作：

公众号文章Prompt：
"请帮我写一篇公众号文章。

主题：[文章主题]
目标读者：[读者画像]
文章目的：[传播目的]

内容要求：
1. 开头：用故事/数据/问题引入
2. 主体：3-5个观点，每个有案例支撑
3. 结尾：总结升华 + 互动引导

风格要求：
- 通俗易懂，有温度
- 有干货，有深度
- 有趣味，不枯燥
- 适当使用金句

格式：
- 使用小标题分节
- 关键数据加粗
- 重要观点单独成段
- 配图说明

字数：1500-2000字"

知识科普Prompt：
"请帮我写一篇科普文章，主题是[科普主题]。

受众：普通大众（非专业人士）

要求：
1. 从身边的现象/问题引入
2. 用通俗的语言解释原理
3. 穿插有趣的案例/故事
4. 结尾总结 + 延伸思考

风格：轻松有趣，像和朋友聊天

字数：1000-1500字"
```

### 4.1 AI润色技巧
```
AI润色方法：

语言润色：
原始文本：
[你的初稿]

润色要求：
"请帮我润色以下文字，使其更流畅、专业。

要求：
1. 保持原意不变
2. 语言更流畅自然
3. 用词更精准
4. 句式更丰富

润色后的文字："

风格调整：
原始文本：
[你的初稿]

风格要求：
"请将以下文字调整为[正式/口语/活泼/文艺]风格。

具体要求：
1. [具体风格要求]
2. [具体风格要求]
3. [具体风格要求]

调整后："

长度调整：
原始文本：
[你的初稿]

长度要求：
"请将以下文字：
1. 精简到200字以内 / 扩展到800字以上
2. 删除冗余表述
3. 保留核心信息

调整后："
```

### 4.2 常见问题处理
```
写作常见问题及处理：

问题1：不知道写什么
处理：
"帮我头脑风暴，关于[主题]可以写哪些角度？
列出10个有价值的选题方向"

问题2：开头不知道怎么写
处理：
"请为[主题]提供5种不同的开头方式：
1. 故事引入
2. 数据引入
3. 问题引入
4. 观点引入
5. 场景引入
每个50字左右"

问题3：内容太空洞
处理：
"请为[主题]补充具体案例和数据支撑"

问题4：不知道如何结尾
处理：
"请为[文章主题]提供3种结尾方式：
1. 总结升华
2. 行动号召
3. 开放式思考
每个100字左右"

问题5：逻辑不清晰
处理：
"请帮我梳理以下内容的逻辑结构，
并提出优化建议"
```
