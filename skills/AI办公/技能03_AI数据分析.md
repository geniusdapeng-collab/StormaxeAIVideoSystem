## 1. AI数据分析场景

### 1.1 应用场景
```
AI数据分析场景：

日常分析：
├── 数据解读：解释数据含义
├── 趋势分析：分析变化趋势
├── 对比分析：对比不同维度
├── 异常检测：发现数据异常
└── 原因分析：找数据变化原因

报告生成：
├── 数据描述：描述数据情况
├── 结论提炼：从数据中提炼结论
├── 报告撰写：生成数据分析报告
├── PPT生成：生成数据展示PPT
└── 建议输出：基于数据的建议

数据处理：
├── 数据清洗：处理异常值
├── 数据转换：格式转换
├── 数据分类：数据分组
├── 数据汇总：多维度汇总
└── 数据可视化：生成图表建议
```

### 1.2 数据类型处理
```
不同数据类型处理：

结构化数据（表格）：
处理方式：
├── 提供数据样本
├── 说明数据背景
├── 明确分析目的
└── 让AI进行解读

示例Prompt：
"请分析以下销售数据：
[粘贴数据表格]

请找出：
1. 增长最快的3个产品
2. 增长最快的产品线
3. 表现最差的区域
4. 数据中的异常点

并给出分析结论和建议"

非结构化数据（文本）：
处理方式：
├── 提取关键信息
├── 分类汇总
├── 情感分析
└── 主题聚类

示例Prompt：
"请分析以下用户反馈：
[粘贴反馈文本]

请总结：
1. 用户主要反馈的问题类别
2. 每类问题的数量和占比
3. 用户的整体情感倾向
4. 最需要改进的3个点"
```

## 2. AI分析Prompt技巧

### 2.1 分析Prompt模板
```
通用数据分析Prompt：

"请分析以下数据：
[粘贴数据]

分析要求：
1. 数据概述：数据的规模、维度等
2. 主要发现：3-5个核心发现
3. 趋势分析：变化趋势和规律
4. 异常识别：数据中的异常点
5. 原因推测：可能的原因分析
6. 结论建议：基于数据的建议

输出格式：
- 使用表格展示关键数据
- 用图表描述趋势（文字描述）
- 结论用加粗标注
- 建议要具体可执行"

对比分析Prompt：

"请对比分析以下两组数据：
[数据A]
[数据B]

分析维度：
1. 整体对比：规模、均值等
2. 差异分析：主要差异点
3. 变化对比：各自的变化趋势
4. 原因推测：差异的可能原因
5. 结论总结：对比得出的结论"
```

### 2.2 分析报告生成
```
数据分析报告Prompt：

"请基于以下数据生成数据分析报告：
[粘贴数据]

报告结构：
1. 执行摘要（100字）
2. 数据概览
3. 核心发现（3-5个）
4. 详细分析（每个发现展开）
5. 结论与建议
6. 附录（数据说明）

要求：
1. 结论要有数据支撑
2. 建议要具体可执行
3. 重要数据用表格展示
4. 趋势用图表描述
5. 字数1500-2000字"

趋势分析报告Prompt：

"请分析以下数据的变化趋势：
[粘贴数据，包含时间序列]

分析内容：
1. 整体趋势：是上升、下降还是平稳
2. 阶段特征：是否有明显的阶段划分
3. 周期性：是否有周期性规律
4. 异常点：是否有异常的峰值或低谷
5. 预测：基于趋势的未来预测

输出要求：
- 量化描述趋势（增长XX%）
- 说明关键转折点
- 提供预测数据"
```

## 3. 常见分析场景

### 3.1 销售数据分析
```
销售数据分析Prompt：

"请分析以下销售数据：
[粘贴销售数据]

分析维度：
1. 整体销售情况
   - 总销售额、订单量、客单价
   - 同比/环比变化
   
2. 产品维度
   - 热销产品TOP10
   - 产品结构占比
   - 产品趋势变化

3. 客户维度
   - 新老客户占比
   - 客户分层分析
   - 客户流失预警

4. 渠道维度
   - 各渠道销售贡献
   - 渠道效率对比
   - 渠道趋势变化

5. 时间维度
   - 月/周/日趋势
   - 季节性规律
   - 工作日vs节假日

输出要求：
- 用表格展示关键数据
- 用数字量化变化
- 给出结论和建议"
```

### 3.2 用户数据分析
```
用户数据分析Prompt：

"请分析以下用户数据：
[粘贴用户数据]

分析维度：
1. 用户规模
   - 总用户数、新增用户数
   - 同比/环比变化
   
2. 用户质量
   - 活跃用户数、活跃率
   - 留存率（次日/7日/30日）
   - 转化率

3. 用户行为
   - 用户使用时长/频率
   - 核心功能使用情况
   - 用户路径分析

4. 用户画像
   - 用户属性分布
   - 用户偏好分析
   - 典型用户特征

5. 用户生命周期
   - 新用户激活情况
   - 用户升级/流失情况
   - 关键行为分析

输出要求：
- 用数据说话
- 找出关键洞察
- 给出运营建议"
```

### 3.3 运营数据分析

4. 用户互动

- 给出具体建议"
```

## 4. 数据可视化建议

### 4.1 图表选择
```
AI生成图表建议：

趋势数据 → 折线图
- 时间序列数据
- 展示变化趋势

对比数据 → 柱状图
- 不同类别对比
- 同一类别不同时期对比

构成数据 → 饼图/环形图
- 各部分占比
- 限制在5-7个类别

关系数据 → 散点图
- 两个变量的关系
- 发现相关性

分布数据 → 直方图
- 数据分布情况
- 发现异常值

Prompt示例：
"基于以下数据，建议使用什么图表类型？
[粘贴数据]

请说明：
1. 推荐的图表类型
2. 为什么适合这种数据
3. 图表的关键元素（X轴、Y轴、颜色等）
4. 如何解读这种图表"
```

### 4.2 图表解读
```
AI图表解读Prompt：

"请解读以下图表：
[描述图表内容或粘贴图表]

请分析：
1. 图表显示的主要信息
2. 关键数据点和趋势
3. 数据中的异常
4. 可以得出的结论
5. 可能的原因解释

输出格式：
- 用文字描述图表
- 量化关键数据
- 给出结论"
```

```
