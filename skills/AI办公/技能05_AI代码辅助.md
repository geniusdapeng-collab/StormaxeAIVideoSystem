## 2. AI编程Prompt技巧

### 2.1 基础技巧
```

├── 说明使用的语言/框架

├── 说明现有代码结构
├── 提供数据格式

示例：
"在以下函数中添加参数验证：
```

### 3.1 数据处理
```
数据处理代码Prompt：

"用[语言]实现以下数据处理功能：

数据源：[数据格式]
处理逻辑：[详细逻辑]
输出格式：[期望输出]

要求：
1. 代码简洁清晰
2. 添加必要的注释
3. 包含错误处理
4. 考虑边界情况"

示例Prompt：
"用Python实现数据清洗功能：

输入：CSV文件，包含用户数据
- name：用户名（可能为空）
- email：邮箱（格式可能不规范）
- phone：手机号（可能有国家区号）

处理逻辑：
1. 去除首尾空格
2. 邮箱转小写，验证格式
3. 手机号统一格式，去除非数字字符
4. 删除name为空的行

输出：清洗后的CSV文件

请用pandas实现，代码要包含：
1. 读取CSV
2. 数据清洗函数
3. 导出CSV
4. 异常处理
5. 主程序"
```
