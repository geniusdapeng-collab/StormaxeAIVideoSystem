## TL;DR

拒绝 = 肯定对方 + 说明原因 + 提供替代 + 保持友好

## 1. 拒绝的意义

### 1.1 为什么要拒绝
```
拒绝的意义：
├── 保护时间：时间是有限资源
├── 专注重点：聚焦最重要的事
├── 避免超载：不做能力之外的事
├── 建立边界：让别人知道你的边界
├── 提升质量：少做才能做好
├── 长期可持续：保护自己不过度消耗
```

### 1.2 不拒绝的后果
```
不拒绝的后果：
├── 时间被占用：做不完自己的事
├── 质量下降：事情太多做不好
├── 影响健康：过度消耗
├── 关系受损：做不好反而影响关系
├── 形成恶性循环：别人觉得你好说话
└── 失去主动：被动应付
```

### 1.3 拒绝的顾虑
```
拒绝顾虑及应对：
├── 怕得罪人 → 优雅拒绝不得罪
├── 不好意思 → 拒绝的是事不是人
├── 担心影响关系 → 真诚的拒绝反而建立信任
├── 觉得自己应该帮忙 → 你的时间和别人一样重要
└── 不知道怎么拒绝 → 学习技巧
```

## 2. 拒绝原则

### 2.1 拒绝原则
```
拒绝原则：
├── 真诚友好：不伤害对方
├── 明确清晰：不要含糊不清
├── 及时回复：不要不回复
├── 提供替代：给出其他建议
├── 保持坚定：不要反复
└── 不必愧疚：你有权拒绝
```

### 2.2 拒绝vs推卸
```
拒绝 vs 推卸：

拒绝（OK）：
├── 基于合理原因
├── 提供替代方案
├── 真诚友好
└── 维护关系

推卸（NO）：
├── 只说不行
├── 不给理由
├── 态度冷漠
└── 影响关系
```

### 2.3 什么时候该拒绝
```
应该拒绝的情况：
├── 超出能力范围
├── 影响正常工作
├── 没有时间和精力
├── 不是自己的职责
├── 损害自己利益
├── 违反原则或价值观
└── 对方要求不合理
```

## 3. 拒绝技巧

### 3.1 拒绝公式
```
拒绝公式：
肯定 + 原因 + 替代 + 友好

示例：
"这个项目听起来很有价值（肯定）
但我目前手上有XX项目在做，时间确实排不开（原因）
你可以找XX，他之前做过类似的项目（替代）
后续有什么我能帮忙的随时说（友好）
"
```

### 3.2 拒绝话术
```
通用拒绝话术：

模板1（直接型）：
"谢谢你的邀请/请求
但我现在手上有XX任务在做
恐怕没有精力再接新的
抱歉帮不上忙
"

模板2（解释型）：
"谢谢想到我
这个任务确实很有价值
但目前我在负责XX项目
实在抽不出时间来
建议可以找XX试试
"

模板3（替代型）：
"感谢你的信任
但我可能不太适合
你可以考虑XX，他可能更合适
或者我们可以XX方式合作
"
```

### 3.3 具体场景拒绝

```
拒绝额外任务：
"谢谢信任，但目前我的工作量已经饱和
如果接这个任务可能会影响XX项目的进度
你可以找XX或者我们讨论下优先级

拒绝不合理要求：
"这个要求可能有点为难
从XX角度来看，可能会有XX问题
建议我们可以XX方式实现
"

拒绝无效会议：
"谢谢邀请
看了一下议题，我觉得可能不需要我参加
如果有关于XX的结论同步给我就行
或者我们可以换个方式，比如异步沟通
"
```

## 4. 拒绝不同对象

### 4.1 拒绝领导
```
拒绝领导要点：
├── 谨慎处理：领导还是要尊重
├── 给出理由：说明合理原因
├── 提供替代：找其他方案
├── 强调影响：说明接受的影响
├── 表达愿意：如果可以，愿意其他方式帮忙
└── 接受决定：最终听领导的

拒绝话术：
"领导，关于XX任务我很愿意做
但目前我在负责XX项目，如果再加这个任务
可能两个项目的进度都会受影响
建议可以XX方案，您觉得呢？
"
```

### 4.2 拒绝同事
```
拒绝同事要点：
├── 真诚友好：保持良好关系
├── 说明原因：让对方理解
├── 提供建议：帮助对方找其他方案
├── 及时回复：不要拖着不回复
├── 保持坚定：不要被说服就答应了
└── 关系维护：拒绝后仍然保持友好

拒绝话术：
"XX，感谢想到我
但最近手头任务确实比较紧
恐怕没办法很好地完成这个任务
建议你找XX试试，他可能时间上更宽裕
有什么其他能帮的随时说
"
```

### 4.3 拒绝下属
```
拒绝下属要点：
├── 解释原因：让下属理解
├── 给出方向：告诉下属可以怎么做
├── 保持鼓励：不要打击下属积极性
├── 给予支持：用其他方式支持
└── 维护信任：拒绝请求但不拒绝关系

拒绝话术：
"你的想法很好，很有创新性
但从目前的优先级来看，我们可能暂时不能做
主要原因是XX
你可以先做XX方面的准备
等时机成熟了我们再推进
"
```

## 5. 拒绝后续

### 5.1 保持友好
```
拒绝后关系维护：
├── 保持正常沟通
├── 主动关心对方
├── 提供其他帮助
├── 感谢对方理解
├── 承诺未来可能
└── 不要回避对方
```

### 5.2 提供替代
```
提供替代方案：
├── 推荐合适人选
├── 提供资源或信息
├── 建议其他时间
├── 提出简化方案
├── 协助寻找帮助
└── 承诺未来可能
```

## 6. 常见问题

### 6.1 对方坚持怎么办
```
对方坚持应对：
├── 坚定但友好：不要动摇
├── 再次解释：说明为什么
├── 提供替代：给出其他方案
├── 建议升级：可以找领导协调
├── 延迟回复：给我时间考虑
└── 学会说不：有时候就是不行
```

### 6.2 愧疚感怎么办
```
处理愧疚感：
├── 认识愧疚：你没有做错什么
├── 合理拒绝：拒绝的是事不是人
├── 对方会理解：真诚的拒绝对方能理解
├── 你也有权利：你有权保护自己
├── 拒绝不等于不帮忙
└── 长远来看：帮不了反而更不好
```

### 6.3 不知道怎么开口
```
不知道怎么开口：
├── 直接简单：直接说明就行
├── 先表示感谢：感谢对方想到你
├── 说明原因：给出真实原因
├── 提供替代：给个建议
├── 不要拖延：拖着不回复更不好
└── 练习就好：拒绝也是需要练习的
```
