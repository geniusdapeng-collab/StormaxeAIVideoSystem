## TL;DR

1v1沟通 = 选择时机 + 准备话题 + 真诚倾听 + 建设反馈 + 明确后续

## 1. 1v1沟通价值

### 1.1 为什么1v1重要
```
1v1沟通价值：
├── 深度连接：比群组沟通更深入
├── 建立信任：更容易建立信任关系
├── 解决问题：私下更容易讨论敏感问题
├── 获取反馈：更真实地获取反馈
├── 传递关怀：表达对对方的关注
└── 职业发展：获取职业建议和发展机会
```

### 1.2 常见场景
```
1v1沟通场景：
├── 领导约谈：工作反馈、职业发展
├── 主动约领导：获取指导、汇报想法
├── 下属沟通：绩效反馈、工作指导
├── 同事协作：解决分歧、深化合作
├── 新人融入：了解情况、获取帮助
└── 离职面谈：了解原因、保持关系
```

## 2. 沟通准备

### 2.1 明确目的
```
沟通前思考：
├── 我这次沟通想要达成什么？
├── 对方可能关心什么？
├── 我需要准备什么信息？
├── 对方可能的反应是什么？
└── 我有什么顾虑需要处理？
```

### 2.2 准备清单
```
沟通准备清单：
□ 明确沟通目的
□ 列出要讨论的话题
□ 收集必要的信息和数据
□ 准备可能的解决方案
□ 选择合适的时间和地点
□ 提前告知对方沟通内容（可选）
```

### 2.3 时机选择
```
选择时机原则：
├── 对方有时间：不打断对方工作
├── 环境私密：不会被打扰
├── 心情平和：避免对方情绪不好时
├── 时效性：根据话题紧迫程度选择
└── 频率适度：不要太频繁也不要太少
```

## 3. 沟通过程

### 3.1 开场技巧
```
开场方式：
├── 直接型："今天想和你聊一下XX"
├── 暖场型："最近工作怎么样？"
├── 感谢型："感谢你抽时间"
├── 说明型："想和你聊聊关于XX"
└── 开放型："有什么想和我聊的吗？"
```

### 3.2 倾听的艺术
```
倾听技巧：
├── 专注：放下手机，眼神注视
├── 回应：点头、嗯、"是的"
├── 复述："你的意思是XX，对吗？"
├── 追问：深入了解对方的想法
├── 记录：记下关键信息
└── 不打断：让对方说完再回应
```

### 3.3 表达技巧
```
表达公式：
事实 + 感受 + 诉求

示例：
"最近项目进度压力很大（事实）
我也感觉有点焦虑（感受）
想和你讨论下如何优化工作流程（诉求）
"

其他技巧：
├── 先说结论
├── 用数据支撑
├── 换位思考
├── 承认不足
└── 感谢对方
```

### 3.4 敏感话题处理
```
敏感话题处理原则：
├── 私下不公开：敏感话题不扩散
├── 具体不抽象：用具体事例说明
├── 对事不对人：讨论行为而非人格
├── 表达感受：分享自己的感受
├── 寻求理解：理解对方的立场
└── 建设解决：一起找解决方案
```

## 4. 不同场景沟通

### 4.1 与领导1v1
```
与领导沟通：
├── 准备充分：带上数据和方案
├── 主动汇报：让领导知道你的进展
├── 寻求指导：请教领导的经验
├── 表达诉求：职业发展、资源需求
├── 接受反馈：开放心态接受意见
└── 跟进执行：说完就要行动

沟通频率：
├── 例行1v1：每周/每两周一次
├── 随时沟通：有需要随时约
└── 重要决定：提前沟通
```

### 4.2 与下属1v1
```
与下属沟通：
├── 关心发展：关注下属的成长
├── 给予反馈：及时给予正向和建设性反馈
├── 解答疑惑：帮助解决工作中的困惑
├── 收集意见：了解下属的想法和建议
├── 激励认可：认可下属的成绩
└── 辅导帮助：帮助下属提升能力

沟通要点：
├── 鼓励下属先说
├── 多问开放式问题
├── 认真倾听
├── 给予具体反馈
├── 明确后续行动
```

### 4.3 与同事1v1
```
与同事沟通：
├── 解决分歧：私下讨论解决工作分歧
├── 深化合作：讨论如何更好协作
├── 获取信息：了解其他部门情况
├── 建立关系：培养工作友谊
├── 寻求支持：请求同事的支持配合
└── 表达感谢：感谢同事的帮助
```

## 5. 沟通结尾

### 5.1 总结要点
```
结尾总结：
├── 回顾讨论内容
├── 确认达成的共识
├── 明确各自的行动
└── 约定后续沟通

示例：
"我们来总结一下今天的沟通
第一，关于XX我们达成共识...
第二，接下来你需要做...
第三，我这边会...
下次我们什么时候再聊？
"
```

### 5.2 后续跟进
```
跟进要点：
├── 按约定行动
├── 及时反馈进展
├── 主动更新情况
├── 感谢对方的帮助
└── 保持定期沟通
```

## 6. 常见问题

### 6.1 紧张怎么办
```
缓解紧张：
├── 充分准备：准备充分会更有信心
├── 换个角度：把领导当普通人
├── 目标明确：知道要说什么
├── 提前到场：熟悉环境
├── 深呼吸：深呼吸缓解紧张
└� 降低期望：不是所有沟通都要完美
```

### 6.2 冷场怎么办
```
打破冷场：
├── 聊近况：从最近的工作聊起
├── 请教问题：问对方擅长的领域
├── 分享趣事：分享一些轻松的
├── 谈共同话题：找到共同兴趣
├── 从小事聊起：从简单话题切入
└── 沉默也没关系：有时候沉默也是沟通
```

### 6.3 被怼怎么办
```
被怼反应：
├── 保持冷静：不要立即反驳
├── 倾听理解：听对方说完
├── 确认理解："你的意思是XX？"
├── 回应感受：理解对方的情绪
├── 解释说明：温和地解释
├── 寻求共识：找到双方都能接受的
└── 求同存异：实在不一致就先放一放
```
