## 3. 专业建议

### A. 套餐设计

#### 套餐类型

| 类型 | 构成 | 适用 |
|------|------|------|
| 主食+配菜 | 主食+小菜+饮品 | 快餐 |
| 双人套餐 | 主食×2+配菜+饮品 | 正餐 |
| 商务套餐 | 主食+主菜+配菜+饮品 | 轻食 |
| 儿童套餐 | 孩子喜欢+健康 | 亲子 |

#### 套餐定价
```
套餐定价 = 单品成本×1.3 + 溢价
溢价空间：10-20%（用户感知优惠）
```

### B. 搭配推荐

#### 关联销售逻辑
```
1. 主食 → 推荐饮品
2. 辛辣 → 推荐饮品
3. 油腻 → 推荐解腻
4. 正餐 → 推荐水果
```

#### 推荐位置
```
1. 菜单分类推荐
2. 购物车页面
3. 下单确认页
4. 结算页凑单
```

### C. 满减引导

#### 凑单设计
```
1. 低价小食：5-15元
2. 引导文案："再满10元减3元"
3. 自动凑单推荐
```

### D. 转化技巧

#### 引导下单
```
1. 菜品描述：突出价值感
2. 精美图片：激发食欲
3. 用户评价：增加信任
4. 限时优惠：制造紧迫
```
