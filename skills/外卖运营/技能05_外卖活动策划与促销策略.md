## 3. 专业建议

### A. 满减设计

#### 满减梯度
```
第一档：引流档（门槛低，力度大）
- 满20减3

第二档：提升档（提升客单）
- 满40减8

第三档：利润档（高客单专属）
- 满80减20
```

#### 满减计算
```
毛利公式：
毛利 = 销售额 - 食材成本 - 平台抽成 - 活动支出

示例：
客单价50元，毛利60%（30元）
满减门槛40减8，平台抽成20%
实际收入：50×0.8-8=32元
食材成本：50×0.4=20元
毛利：32-20=12元（24%）
```
