## 2. 核心指标框架

### 经营漏斗

```
曝光 → 进店 → 下单 → 复购
  ↓      ↓      ↓      ↓
曝光量  进店率  下单率  复购率
```

### 核心指标

| 指标 | 含义 | 优秀值 |
|------|------|--------|
| 曝光量 | 看到店铺的人数 | 区域top20% |
| 进店率 | 进店人数/曝光 | >10% |
| 下单率 | 下单/进店 | >30% |
| 复购率 | 复购用户/总用户 | >40% |
| 客单价 | 每单平均金额 | 品类均值+20% |
| 毛利 | 收入-成本 | >25% |

| 复购率低 | 口味/质量/服务 | 检查用户评价 |
