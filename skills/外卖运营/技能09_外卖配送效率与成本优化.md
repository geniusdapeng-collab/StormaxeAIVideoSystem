## 2. 配送体系框架

### 配送模式对比

| 模式 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| 平台配送 | 覆盖广 | 成本高 | 初创期 |
| 自配送 | 成本低 | 范围小 | 品牌成熟 |
| 混合配送 | 灵活 | 管理复杂 | 中大型 |

### 配送成本结构

```
总成本 = 配送费 + 平台抽成 + 保险 + 差评扣款
```

## 3. 专业建议

### A. 出餐效率

#### 优化方法
```
1. 预制半成品
2. 动线优化
3. 打包标准化
4. 峰时备货
```

#### 时间管理
```
出餐流程：
接单 → 备料 → 制作 → 打包 → 出餐
  ↓     ↓     ↓     ↓      ↓
  30s  60s  180s  30s    300s = 10分钟
```

### B. 配送效率

#### 配送时效提升
```
1. 预制菜品提前做
2. 骑手等候区设置
3. 错峰出餐
4. 智能派单
```

### C. 配送成本

#### 成本优化策略
```
1. 合理设置配送范围
2. 阶梯定价引导非高峰
3. 自配送降成本
4. 平台活动减少扣罚
```

### D. 配送范围

#### 范围设置原则
```
1. 参考竞品范围
2. 考虑配送时效
3. 平衡成本和覆盖
4. 动态调整
```
