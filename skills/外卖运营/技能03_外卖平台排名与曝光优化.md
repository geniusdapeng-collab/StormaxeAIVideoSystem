## 3. 专业建议

### A. 销量提升

#### 基础优化
```
1. 优化菜单结构
2. 提升菜品质量
3. 合理定价策略
4. 积极参与活动
5. 引导好评
```

#### 新店冲榜策略
```
阶段1（1-7天）：积累基础销量
- 朋友下单+好评
- 限时特价引流
- 霸王餐活动

阶段2（8-14天）：稳定销量
- 持续优化转化率
- 参与平台活动
- 复购引导
```

### B. 评分优化

#### 好评引导
```
1. 随餐小卡片
2. 好评返券
3. 差评跟进解决
4. 及时回复评价
```

#### 差评应对
```
1. 24小时内响应
2. 真诚道歉
3. 解决问题
4. 引导下次改进
```

