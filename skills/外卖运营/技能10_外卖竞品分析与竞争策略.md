竞争策略核心：竞品分析→定位差异化→建立壁垒→持续监控，四步建立长期竞争优势

## 1. 场景分析

### 请求摘要
用户需要分析竞争对手并制定竞争策略

### 显性需求

- 获得可执行的竞争策略

### 场景分类
- `scenario_classification`: 战略规划
- `urgency`: normal

## 2. 竞品分析框架

### 分析维度

| 维度 | 内容 | 数据来源 |
|------|------|----------|
| 销量 | 单量/销售额 | 平台公开数据 |
| 价格 | 定价策略 | 店铺观察 |
| 活动 | 活动力度 | 店铺主页 |
| 评价 | 评分/评价 | 用户评价 |
| 菜单 | 菜品结构 | 菜单分析 |
| 定位 | 目标客群 | 综合判断 |

### 竞品选择

```
1. 直接竞品：同品类同区域
2. 间接竞品：同区域不同品类
3. 潜在竞品：有重合用户的品类
```

## 3. 专业建议

### A. 竞争定位

#### 定位方法
```
1. 差异化定位：找到独特价值
2. 补缺定位：填补市场空白
3. 跟随定位：模仿优秀竞品
4. 挑战定位：正面竞争
```

#### 定位要素
```
目标客群 + 核心价值 + 差异化特征 = 品牌定位
```

### B. 差异化策略

#### 差异化维度

| 维度 | 方式 | 示例 |
|------|------|------|
| 产品 | 特色菜品 | 独家口味 |
| 价格 | 定价策略 | 高性价比 |
| 服务 | 体验升级 | 更快配送 |
| 场景 | 使用场景 | 办公室下午茶 |
| 情感 | 品牌调性 | 年轻潮流 |

### C. 竞争壁垒

#### 常见壁垒

```
1. 产品壁垒：独家配方
2. 品牌壁垒：认知度
3. 规模壁垒：成本优势
4. 运营壁垒：效率优势
5. 客户壁垒：忠实用户
```

### D. 竞争监控

#### 监控指标
```
1. 竞品销量变化
2. 活动动态
3. 新品上架
4. 评价变化
5. 排名变化
```
