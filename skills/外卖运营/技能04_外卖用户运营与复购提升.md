用户运营核心：新客转化→分层运营→复购激励→沉睡唤醒，全链路提升用户价值

## 1. 场景分析

### 请求摘要
用户需要提升外卖用户的复购率和生命周期价值

### 显性需求

- 获得可执行的运营方案

### 场景分类

## 2. 用户分层模型

### RFM分层

| 分层 | 定义 | 运营策略 |
|------|------|----------|
| 高价值 | 高频+高额 | 专属优惠 |
| 潜力 | 高频+低额 | 提升客单 |
| 维护 | 低频+高额 | 唤醒召回 |
| 流失 | 低频+低额 | 沉睡唤醒 |

### 用户生命周期

```
新用户 → 成长用户 → 成熟用户 → 沉默用户 → 流失用户
   ↓        ↓          ↓          ↓          ↓
首单转化   复购引导   会员维护   活动唤醒   流失召回
```

## 3. 专业建议

### A. 新客转化

#### 首单转化策略
```
1. 新客专属券：首次下单立减
2. 新客礼物：随餐赠送
3. 首单低价：爆款引流
4. 快速响应：缩短接单配送时间
```

### B. 复购激励

#### 会员体系设计
```
等级设计：
- 普通会员：注册即享
- 银卡会员：累计5单
- 金卡会员：累计15单
- 黑卡会员：累计30单

权益设计：
- 积分倍率不同
- 专属优惠券
- 优先客服
- 免配送费
```

#### 复购活动
```
1. 周期性活动：会员日/周三优惠
2. 菜品推荐：根据历史推荐
3. 套餐组合：提升客单
4. 积分兑换：激励复购
```

### C. 精准营销

#### 用户画像应用
```
1. 消费偏好：推荐喜欢菜品
2. 消费时间：定时推送
3. 消费能力：差异化定价
4. 活跃周期：提前干预
```

### D. 沉睡唤醒

#### 唤醒策略
```
1. 定向优惠：沉睡用户专属券
2. 召回短信/推送
3. 新品通知
4. 流失原因调研
```
