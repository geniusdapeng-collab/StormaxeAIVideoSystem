## 3. 专业建议

### A. 店招设计

#### 设计原则
```
1. 品牌感强：突出品牌Logo和名称
2. 主打产品：展示核心爆款
3. 活动信息：优惠活动一目了然
4. 色彩统一：符合品牌调性
```

#### 常见风格

| 风格 | 特点 | 适用品类 |
|------|------|----------|
| 简约清新 | 白底浅色 | 轻食/沙拉 |
| 热闹烟火 | 实景拍摄 | 烧烤/火锅 |
| 品质感 | 黑金/深色 | 高端餐饮 |
| 卡通可爱 | 卡通形象 | 奶茶/甜品 |

### B. 菜品图片

#### 拍摄技巧
```
1. 光线：自然光或专业灯光
2. 角度：45度俯拍最佳
3. 构图：主体居中，留白合适
4. 背景：简洁干净
5. 后期：调色但不失真
```

#### 图片规范
```
- 分辨率：≥800×800
- 格式：JPG/PNG
- 大小：≤5MB
- 数量：每菜品至少2张
```

### C. 菜单分类设计

#### 分类原则
```
1. 逻辑清晰：按菜品类型分
2. 主次分明：主打分类靠前
3. 便于浏览：不超过8个分类
4. 特色突出：活动分类引流
```

#### 分类排序
```
1. 招牌/主打
2. 热销推荐
3. 特色分类
4. 常规分类
5. 套餐组合
6. 配菜饮品
7. 优惠活动
```

### D. 品牌统一

#### 视觉规范
```
1. 色彩：主色+辅色+点缀色
2. 字体：统一字体风格
3. 风格：整体调性一致
4. 元素：品牌符号贯穿
```
