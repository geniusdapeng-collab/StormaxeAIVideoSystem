### 本地化层次

| 层次 | 内容 | 复杂度 |

## 3. 专业建议

### A. 语言本地化

#### 翻译策略
```
自研 → 机翻+人工校对
外包 → 专业本地化公司
众包 → 社区翻译
```

#### 文本扩展处理
```css
/* 文本扩展预留 */
.button { min-width: 120px; padding: 8px 16px; }
/* 德语可能扩展30% */
```

#### i18n实现
```javascript
// 国际化框架
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';

i18n.use(initReactI18next).init({
  resources: {
    en: { translation: { "hello": "Hello" } },
    de: { translation: { "hello": "Hallo" } },
    ja: { translation: { "hello": "こんにちは" } }
  },
  lng: "en",
  fallbackLng: "en"
});
```

### B. 文化适配

#### 文化敏感点

| 区域 | 敏感内容 | 注意事项 |
|------|----------|----------|
| 中东 | 宗教符号 | 避免猪、十字 |
| 亚洲 | 颜色含义 | 日本：绿丧葬色 |
| 西方 | 数字 | 13不吉利 |
| 拉美 | 动物 | 蝴蝶寓意不同 |

#### 节日适配
- 美国：感恩节、黑色星期五
- 中国：春节、双十一
- 中东：斋月、开斋节
- 印度：排灯节

### C. 功能本地化

#### 支付方式
```
区域        主流支付
------------------------------
北美        Visa, Mastercard, PayPal
欧洲        Klarna, iDEAL, Giropay
东南亚        GrabPay, GoPay, TrueMoney
日本        Konbini, PayPay
```

#### 配送选项
- 美国：2-day shipping
- 欧洲：Standard delivery
- 东南亚：COD常见

### D. 视觉本地化

#### 图片调整
- 人像：使用目标市场模特
- 颜色：考虑文化联想
- 符号：避免文化忌讳

#### 布局适配
- 阿拉伯语：RTL（从右向左）
- 中文：文字竖排可能
- 德语：文本长需预留空间
