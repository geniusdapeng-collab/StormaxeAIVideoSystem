## 1. 场景分析

用户需要提升海外用户增长和留存

### 场景分类

- `urgency`: normal

## 2. AARRR增长框架

### 漏斗模型

```
Acquisition（获客）
    ↓
Activation（激活）
    ↓
Retention（留存）
    ↓
Revenue（变现）
    ↓
Referral（裂变）
```

### 关键指标

| 阶段 | 核心指标 | 优秀值 |
|------|----------|--------|
| 获客 | CAC | <LTV/3 |
| 激活 | 激活率 | >40% |
| 留存 | 次日/7日/30日留存 | 40%/20%/10% |
| 变现 | ARPU | 持续增长 |
| 裂变 | K因子 | >1 |

## 3. 专业建议

### A. 用户获取（Acquisition）

#### 获客渠道
```
1. 付费渠道：广告投放
2. 有机渠道：SEO/内容营销
3. 口碑渠道：推荐/裂变
4. 合作渠道：BD/换量
```

#### 优化策略
```
1. 降低CAC：优化渠道+素材
2. 提升质量：精准定向
3. 规模化：多渠道测试
```

### B. 用户激活（Activation）

#### 激活定义
```
新用户完成核心行为（如首次购买/发布内容）
```

#### 激活策略
```
1. 简化注册流程
2. 新手引导（Onboarding）
3. 首单优惠
4. 及时价值交付
```

### C. 用户留存（Retention）

#### 留存分析
```
1. 同期群分析
2. 用户分层（RFM）
3. 流失预警
```

#### 留存策略
```
1. 产品价值留存
2. 习惯养成
3. 会员体系
4. 个性化推送
```

### D. 会员体系设计

#### 会员权益
```
等级          权益
------------------------
普通          基础权益
银卡          积分倍率
金卡          专属折扣
黑卡          优先服务
```

#### 积分体系
```
获取：消费/行为/活动
消耗：抵扣/兑换/特权
```

### E. 复购与变现（Revenue）

#### 复购策略
```
1. 订阅服务
2. 会员订阅
3. 交叉销售
4. 个性化推荐
5. 及时促销
```

#### LTV计算
```
LTV = ARPU × 用户生命周期 × 毛利率

示例：
ARPU = $50/月
生命周期 = 12月
毛利率 = 30%
LTV = 50 × 12 × 0.3 = $180
```

### F. 裂变增长（Referral）

#### 裂变机制
```
1. 邀请奖励：双向得利
2. 分享返现：内容传播
3. 拼团：社交裂变
4. 分销：利益驱动
```
