## 2. 核心合规领域

### 四大合规支柱

| 领域 | 核心法规 | 违规处罚 |
|------|----------|----------|
| 隐私保护 | GDPR, CCPA | 全球营收4% |
| 消费权益 | 消费者保护法 | 罚款+诉讼 |
| 税务合规 | VAT, GST | 补税+罚款 |
| 知识产权 | 专利/商标 | 赔偿+下架 |

## 3. 专业建议

### A. 隐私保护（GDPR/CCPA）

#### GDPR要求
```
1. 合法基础：需有合法处理理由
2. 知情同意：清晰易懂的同意机制
3. 数据最小化：仅收集必要数据
4. 目的限制：用于指定目的
5. 存储限制：不超过必要时间
6. 安全措施：技术+组织措施
7. 问责原则：记录处理活动
```

#### 隐私政策必备内容
```
1. 数据控制者信息
2. 处理目的和法律基础
3. 数据类型
4. 数据接收者
5. 数据保留期限
6. 用户权利（访问/删除/导出）
7. 投诉权利
8. 联系方式
```

#### CCPA要点
- 消费者知情权
- 删除权
- 拒绝销售权
- 不歧视权利

### B. 消费保护

#### 常见要求
```
1. 退货政策（欧盟14天）
2. 清晰定价（无隐藏费用）
3. 产品安全认证
4. 虚假宣传禁止
5. 公平合同条款
```

#### 区域差异

| 区域 | 退货期 | 特殊要求 |
|------|--------|----------|
| 欧盟 | 14天 | 支持首单退货运费 |
| 美国 | 30天 | 各州不同 |
| 英国 | 14天 | 偏远地区例外 |
| 澳洲 |  Consumer Rights Act |

### C. 税务合规

#### 增值税/销售税

| 区域 | 税率 | 注册门槛 |
|------|------|----------|
| 欧盟 | 19-27% | €10,000远程销售 |
| 英国 | 20% | £0（必须注册） |
| 美国 | 0-10% | 各州不同 |
| 澳大利亚 | 10% | AUD $75,000 |

#### 税务注册
```
1. 确定应税区域
2. 申请税号
3. 设置申报周期
4. 配置税务计算
5. 按时申报缴纳
```

### D. 产品认证

| 认证 | 区域 | 产品类别 |
|------|------|----------|
| CE | 欧洲 | 电子/机械 |
| FCC | 美国 | 电子产品 |
| CPC | 美国 | 儿童产品 |
| PSE | 日本 | 电气产品 |
| CCC | 中国 | 强制目录产品 |

### E. 知识产权

#### 商标保护
```
1. 目标市场商标注册
2. 监测侵权
3. 域名保护
4. 专利申请（如适用）
```
