## 2. 物流方案框架

### 物流模式对比

| 模式 | 时效 | 成本 | 适用 |
|------|------|------|------|
| 直发 | 7-15天 | 低 | 小包/轻量 |
| 海外仓 | 2-5天 | 高 | 大件/高周转 |
| FBA | 2-5天 | 中高 | 亚马逊卖家 |
| 虚拟仓 | 3-7天 | 中 | 铺货/测款 |

### 物流成本结构
```
总成本 = 头程 + 清关费 + 仓储费 + 尾程 + 退货成本
```

## 3. 专业建议

### A. 头程运输

#### 运输方式
```
方式      时效      成本      适用
-----------------------------------------
空运      3-7天    高       轻小件/急货
海运      15-40天  低       大件/大批量
铁运      20-30天  中       欧洲专列
快递      3-7天    高       样品/高价值
```

#### 物流商选择
- 四大快递：DHL, UPS, FedEx, TNT
- 跨境专线：云途、燕文、递四方
- 邮政：EMS, ePacket

### B. 海外仓布局

#### 仓群策略
```
区域      仓库位置      覆盖
-----------------------------------------
美西      洛杉矶      美西/墨西哥
美东      纽约/新泽西  美东
欧洲      德国/英国    全欧洲
东南亚   新加坡/马来   东南亚
```

#### 库存管理
```python
# 安全库存计算
安全库存 = (最大日销量 × 采购周期) + 缓冲库存
补货点 = 日均销量 × 采购周期 + 安全库存
```

### C. 清关实务

#### 清关文件
```
1. 商业发票（Commercial Invoice）
2. 装箱单（Packing List）
3. 提单（Bill of Lading）
4. 原产地证书（Certificate of Origin）
5. 产品认证（CE, FCC等）
```

#### 关税计算
```
关税 = CIF价值 × 税率
综合税 = 关税 + 增值税 - 可抵扣进项
```

### D. 尾程配送

#### 最后一公里
```
区域      主流配送
-----------------------------------------
美国      USPS, UPS, FedEx
欧洲      DHL, Hermes, Yodel
东南亚   Grab, Lalamove
日本      佐川, 邮政
```

### E. 退货处理

#### 退货方案
```
方案1: 退回国（成本高，适合高价值）
方案2: 当地销毁（成本中，适合低价值）
方案3: 翻新再售（需海外处理能力）
方案4: 折扣清仓（快速回笼资金）
```
