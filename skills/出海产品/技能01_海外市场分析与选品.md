## TL;DR

海外市场分析核心：市场容量→竞争强度→盈利空间→进入难度，四维评估选定了最佳市场

## 1. 场景分析

## 2. 市场分析框架

### 四维评估模型

| 维度 | 评估指标 | 数据来源 |
|------|----------|----------|
| 市场容量 | TAM/SAM/SOM | Statista, eMarketer |
| 竞争强度 | 集中度、头部占比 | Jungle Scout, SimilarWeb |
| 盈利空间 | 利润率、加成空间 | 竞品定价分析 |
| 进入难度 | 法规、渠道、本地化 | 专家访谈 |

### 目标市场优先级

| 优先级 | 市场特征 | 适用策略 |
|--------|----------|----------|
| P0 | 容量大+竞争适中+盈利高 | 重点投入 |
| P1 | 容量大+竞争激烈 | 差异化切入 |
| P2 | 新兴市场 | 提前布局 |

## 3. 专业建议

### A. 区域市场分析

#### 成熟市场（美/欧/日）
- 特点：消费力强、要求高、竞争激烈
- 策略：品质优先、品牌化、高毛利

#### 新兴市场（东南亚/拉美/中东）
- 特点：增长快、价格敏感、渠道分散
- 策略：性价比、 本地化、 轻投入

### B. 选品方法论

#### 数据选品流程
```
1. 市场规模筛选 → 设定最低TAM
2. 竞争分析 → 排除过度竞争品类
3. 利润测算 → 毛利>30%优先
4. 供应链确认 → 交付周期<15天
5. 合规检查 → 法规/认证要求
```

#### 蓝海品类特征
- 搜索量上升但供给不足
- 头部品牌集中度<50%
- 复购率>20%
- 无明显巨头垄断
