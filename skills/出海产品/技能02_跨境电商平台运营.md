## 3. 专业建议

### A. Listing优化

#### 标题公式
```
品牌 + 核心关键词 + 特性/卖点 + 适用场景/人群 + 规格/颜色
```

#### 五点描述结构
```
1. 核心卖点（用户最关心的）
2. 差异化优势
3. 使用场景
4. 材质/工艺
5. 售后保障
```

#### 搜索词优化
- 250字符填满
- 核心词前置
- 避免重复
- 使用长尾词

### B. 站内流量获取

#### 关键词优化
```
步骤1: 收集关键词（Helium 10, MerchantWords）
步骤2: 分类排序（高搜索/低竞争）
步骤3: 布局到标题/五点/描述
步骤4: 定期优化迭代
```

#### 广告策略
- 起步期：自动广告跑词
- 成长期：手动广告精准投放
- 成熟期：品牌广告+展示型广告

### C. 评价管理

#### 索评策略
- 订单签收后及时索评
- 使用个性化模板
- 提供优质售后

#### 差评应对
- 24小时内响应
- 真诚道歉+解决方案
- 引导至线下沟通

### D. 利润控制

#### 成本结构
```
售价 = 采购 + 物流 + 平台费 + 营销 + 利润
```

#### 定价策略
- 竞品对标定价
- 阶梯定价
- 促销定价（ACOAS）
