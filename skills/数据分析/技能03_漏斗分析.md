## TL;DR

漏斗分析 = 定义阶段 → 追踪用户 → 计算转化 → 定位问题

## 1. 漏斗构建

### 阶段定义
| 阶段 | 定义 | 关键点 |
|------|------|--------|
| 入口 | 用户进入 | PV/UV |
| 核心步骤 | 关键行为 | 每步必记 |
| 目标 | 最终转化 | 成交/注册 |

### 示例：电商购买漏斗
```
商品曝光 → 点击详情 → 加入购物车 → 提交订单 → 支付成功
10000   →   3000   →    1000   →    800   →    600
         30%       33%        80%       75%
```

## 2. 分析方法

### 绝对转化
- 阶段间用户数差
- 流失人数

### 相对转化率
- 上一阶段→本阶段比例
- 环比对比

### 差异分析
- 同比/环比对比
- 分群对比

## 3. 优化策略

- 找出最大流失点
- 分析流失原因
- 针对性优化
