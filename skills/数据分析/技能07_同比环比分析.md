## TL;DR

同比 = 同期对比(同期)；环比 = 相邻周期对比

## 1. 核心概念

### 环比
- 与上一周期对比
- 反映短期变化
- 适用：日常监控

### 同比
- 与去年同周期对比
- 反映长期趋势
- 适用：季节性产品

## 2. 计算方法

### 环比增长率
```
环比增长率 = (本期值 - 上期值) / 上期值 × 100%
```

### 同比增长率
```
同比增长率 = (本期值 - 去年同期值) / 去年同期值 × 100%
```

## 3. 分析框架

### 步骤1：计算变化率
- 环比变化
- 同比变化

### 步骤2：判断变化类型
| 变化 | 可能原因 |
|------|----------|
| 环比↑同比↓ | 季节性，正常 |
| 环比↓同比↑ | 短期因素 |
| 环比↓同比↓ | 需关注 |
| 环比↑同比↑ | 健康增长 |

### 步骤3：归因分析
- 外部因素
- 内部因素
