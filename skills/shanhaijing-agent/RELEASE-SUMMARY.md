## 🎯 本次升级核心（两份外部建议书的完整实施）

### 1️⃣ 业务视角优化（上次）+ 风格辨识度构建（本次）= 导演层完整

| 维度 | 新增能力 | 价值 |
|------|---------|------|
| **导演思维** | 5层注入（阐释/张力/质量/A-B/日报） | Agent具备创作意图分析能力 |
| **风格配方** | 20位导演风格识别 + 60%+30%+10%混合 | 从"选风格"升级为"调配风格" |
| **风格DNA** | 20维量化参数（视觉/叙事/声音/节奏/时间） | 风格从"感觉"变成"可计算" |
| **冲突检测** | 硬冲突拦截 + 软冲突警告 | 避免"贝+维伦纽瓦"这种不可能组合 |
| **SRS评分** | 五维度辨识度评分（0-100） | 每部作品可量化"风格辨识度" |
| **漂移检测** | 镜头级色温/光比/饱和度监控 | 解决"第3镜诺兰、第5镜宫崎骏"问题 |
| **系列化** | 风格染色体（核心70%锁定+可变30%浮动） | 系列视频保持品牌一致性 |

### 2️⃣ v7.0 → v6.0 适配器（真实模块调用）

- 9个v6.0模块全部接入（故事引擎/渲染引擎/评估/后期/声音/角色/交付/舞蹈/导演）
- exec模式子进程调用，零改动v6.0代码
- 失败自动降级到mock模式

### 3️⃣ 飞书通知系统

- 权限请求/预算预警/风险告警/进度汇报/错误通知
- **新增**: 风格漂移告警 / 风格冲突提示 / SRS评分报告
- 队长 `open_id` 已配置，通知直达飞书
