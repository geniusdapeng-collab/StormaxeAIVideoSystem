# PPT制作技能 V1.0.0

**将高密度文字降维为可视化图表，直接生成可编辑PPTX文件**

---

## 技能简介

PPT制作技能是一款集视觉架构设计与文件生成于一体的智能工具，能够：
- 将用户输入的报告内容转化为专业的PPT视觉架构方案
- 直接生成可编辑的PPTX文件，支持飞书/微信直接发送
- 遵循"一图胜千言"原则，让读者3秒内理解页面逻辑

---

## 核心能力

### 1. 视觉架构设计

**降维打击**：将高密度文字"翻译"为可视化图表

| 内容类型 | 强制图表 | 说明 |
|---------|---------|------|
| 结构关系 | 金字塔/逻辑树 | 层级结构清晰展示 |
| 流程步骤 | 甘特图/时间轴 | 步骤逻辑显性化 |
| 对比分析 | 矩阵图/左右对比 | 优劣对比直观 |
| 组成占比 | 瀑布图/饼图 | 数据构成可视化 |
| 趋势变化 | 折线图/子弹图 | 变化趋势一眼可见 |

### 2. PPT文件生成

**支持多种幻灯片类型**：

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| title | 标题幻灯片 | 开场、结尾 |
| content | 内容幻灯片 | 文字要点 |
| chart | 图表幻灯片 | 数据卡片 |
| comparison | 对比幻灯片 | 前后对比 |
| process | 流程幻灯片 | 步骤展示 |
| table | 表格幻灯片 | 数据表格 |

## 使用方式

### 方式1：直接请求

```
用户：帮我制作一个关于"Q3工作汇报"的PPT
```

### 方式2：提供内容


## 输出内容

1. **PPT视觉架构方案**（Markdown格式）
   - A. PPT信息架构（目录）
   - B. 全局概览页
   - C. 分模块可视化方案
   - D. 统一图表库

2. **PPTX文件**
   - 可直接下载编辑
   - 支持飞书/微信发送

## 快速使用

```python
from generate_ppt import create_ppt

slides = [
    {'type': 'title', 'title': '标题', 'subtitle': '副标题'},
    {'type': 'content', 'title': '页面标题', 'bullets': ['要点1', '要点2']},
    {'type': 'chart', 'title': '图表', 'items': [{'title': '指标', 'value': '99%', 'desc': '说明'}]},
    {'type': 'comparison', 'title': '对比', 'left_title': '旧', 'left_items': [], 'right_title': '新', 'right_items': []},
]

output = create_ppt(slides, 'output.pptx')
```

## 视觉规范

### 色彩系统

- **主色**：蓝色 #0066CC
- **强调色**：绿色 #00994D（正向）
- **警示色**：红色 #CC0000（风险）
- **灰阶**：#646464 / #F0F0F0

### 文字规范

- 标题：结论句，<10字，动词开头
- 正文：短句为主，<2行
- 图注：<20字

### 层级排序

```
图形(70%) > 标题 > 关键数据 > 解释文字
```
