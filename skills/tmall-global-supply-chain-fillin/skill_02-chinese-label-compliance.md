# TL;DR

掌握天猫国际跨境电商中文标签设计、审核与合规管理，让每一件商品都有合规标签，顺利通过平台审核，避免海关扣货。

# 场景分析

## 典型场景
- **标签设计阶段**：新品上架前需要设计符合法规的中文标签，内容繁多容易出错
- **平台审核阶段**：天猫审核标签发现不合规，商品被下架，影响销售计划
- **海关查验阶段**：进口商品被海关查验标签不合格，扣货或退运，损失惨重
- **大促前夕**：活动商品量大，标签问题集中爆发来不及整改

## 核心挑战
1. **法规多且杂**：食品安全法、化妆品监管条例、产品标识规定等交叉重叠
2. **跨境特殊要求**：原产国、境内责任人、注册证号等跨境特有内容易漏
3. **格式要求严**：字号、位置、语言等细节要求多
4. **更新频繁**：法规政策变化，标签需要及时更新
5. **跨部门协作难**：标签涉及采购、设计、法务、运营多部门

## 正确认知
- **标签合规是商品上架的前提条件，不是可选项**
- **问题发现越晚，整改成本越高——上架前审核优于上架后整改**
- **中文标签是消费者信任的重要来源，合规本身就是竞争力**
- **标签是法定义务，虚假或缺失标签有法律风险**

# 决策框架

## 第一步：判断商品类型与适用法规

### 商品类型决定标签法规

| 商品类型 | 主要适用标准 | 标签审核机构 |
|----------|-------------|-------------|
| 预包装食品 | GB 7718、食品安全法 | 海关/天猫 |
| 化妆品 | GB 5296.3、化妆品监管 | 药监局/天猫 |
| 保健食品 | GB 7718+专项规定 | 食药监/天猫 |
| 服装/鞋类 | GB 5296系列、GB 18401 | 天猫 |
| 电器类 | 强制性标准、产品标识规定 | 天猫/3C认证 |
| 其他品类 | 产品质量法+专项规定 | 天猫 |

### 决策逻辑
```
商品上架前
    ↓
确认商品品类
    ↓
匹配适用标准（参考上表）
    ↓
确定必标内容清单
```

## 第二步：梳理必标内容清单

### 食品类标签必标内容

| 必标内容 | 要求 | 示例 |
|----------|------|------|
| 食品名称 | 反映食品真实属性 | "全脂奶粉"而非"营养奶粉" |
| 配料表 | 按用量降序排列 | 配料：生牛乳、白砂糖… |
| 净含量 | 法定计量单位 | 500g / 500ml |
| 生产日期 | 明确日期 | 2026/01/01 |
| 保质期 | 标明储存条件 | 常温保存12个月 |
| 贮存条件 | 如实标注 | 常温保存、避光等 |
| 原产国 | 标注原产国家 | 原产国：日本 |
| 境内代理商 | 名称+地址+电话 | ××株式会社 |
| 营养成分表 | 每100g/份含量 | 能量×kJ 蛋白质×g… |

### 化妆品标签必标内容

| 必标内容 | 要求 | 示例 |
|----------|------|------|
| 产品名称 | 注册或备案名称 | ××精华液 |
| 成分表 | 全部成分（含量≤1%可乱序） | 成分：水、丁二醇… |
| 净含量 | 法定计量单位 | 30ml |
| 生产企业 | 名称+地址 | ××株式会社 |
| 境内责任人 | 名称+地址 | ××贸易有限公司 |
| 生产批号 | 可追溯 | LOT:20260101 |
| 保质期 | 开封后使用期限 | 开封后12个月内用完 |
| 备案文号 | 国妆网备字+年份 | 国妆网备字2026001234 |
| 使用方法 | 简明易懂 | 洁面后取适量涂抹于面部 |

## 第三步：标签设计与制作

### 标签设计要点

| 要点 | 说明 |
|------|------|
| 文字规范 | 使用规范汉字，不得使用繁体字 |
| 字号要求 | 最小不得小于1.8mm（饮料酒≥2mm） |
| 语言要求 | 必须使用中文，可同时使用外文 |
| 位置要求 | 最小销售单元有标签，不拆分销售 |
| 有效期标注 | 生产日期+保质期，或生产批号+保质期 |

### 标签审核决策树

```
标签设计完成
    ↓
自检清单核对（必标内容是否齐全）
    ↓
格式规范检查（字号/语言/位置）
    ↓
专业审核（法务/合规人员）
    ↓
平台预审（天猫标签审核系统）
    ↓
通过 → 正式生产/印刷
失败 → 整改后重新提交
```

## 第四步：审核不通过处理

### 常见被拒原因

| 被拒原因 | 整改方案 | 预防措施 |
|----------|----------|----------|
| 缺少境内责任人 | 补充完整信息 | 提前收集供应商资料 |
| 成分表格式错误 | 按国标格式重排 | 使用标准模板 |
| 保质期标注不规范 | 重新标注 | 学习国标要求 |
| 原产国标注错误 | 核实后更正 | 核对海关单据 |
| 字号不达标 | 调整标签尺寸 | 设计时设置最小字号 |

### 审核申诉流程
```
审核失败通知
    ↓
查看具体驳回原因
    ↓
联系天猫审核小二确认
    ↓
整改完成后重新提交
    ↓
申诉（若认为误判）
```

# 专业建议

## 食品中文标签实操

### 保健食品特殊要求

| 项目 | 要求 |
|------|------|
| 蓝帽子标志 | 必须标注 |
| 批准文号 | 国食健字G/J+年份+序号 |
| 功效成分 | 明确标注含量 |
| 适宜人群 | 明确标注 |
| 不适宜人群 | 如实标注（如少年儿童） |
| 食用方法 | 明确每日用量 |
| 注意事项 | "本品不能代替药物" |

### 婴幼儿配方奶粉特殊要求

| 项目 | 要求 |
|------|------|
| 配料表 | 按国标要求强制标注乳糖等 |
| 营养成分 | 必须符合国标要求 |
| 段位标注 | 明确1/2/3段 |
| 注册号 | 标注配方注册号 |
| 喂养指南 | 必须提供冲调方法 |
| 警示说明 | "在医生指导下使用"等 |

## 化妆品中文标签实操

### 普通化妆品 vs 特殊化妆品

| 类型 | 区别 | 标签差异 |
|------|------|----------|
| 普通化妆品 | 备案制 | 标注备案号 |
| 特殊化妆品 | 注册制 | 标注注册号 |

### 进口化妆品标签检查清单

- [ ] 成分表是否完整（全部成分）
- [ ] 境内责任人信息是否准确

- [ ] 保质期标注是否规范
- [ ] 净含量单位是否正确
- [ ] 是否使用规范汉字
- [ ] 字号是否符合要求

## 标签审核流程优化

### 内部审核SOP

```
Step 1：设计稿审核（运营）
→ 对照品类标签清单
→ 核对必标内容

Step 2：合规审核（法务/合规）
→ 法规符合性检查
→ 格式规范性检查

Step 3：平台预审
→ 使用天猫标签预审工具
→ 提前发现问题

Step 4：印刷前终审
→ 最终确认无误
→ 方可印刷
```

### 标签模板管理
- 为每个品类建立标准标签模板
- 模板定期更新（法规变化时）
- 所有新商品必须使用标准模板
- 非标准情况需单独审批
