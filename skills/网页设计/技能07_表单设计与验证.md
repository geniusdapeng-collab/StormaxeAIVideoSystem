## TL;DR

表单设计核心：标签清晰 + 即时反馈 + 错误引导 + 智能验证

## 1. 场景分析

### 请求摘要
用户需要设计或优化网页表单

### 显性需求
- 掌握表单设计原则
- 获得可执行的表单代码

### 场景分类
- `scenario_classification`: 前端开发
- `urgency`: normal

## 2. 表单设计原则

| 原则 | 说明 |
|------|------|
| 标签清晰 | 每个输入有明确标签 |
| 即时反馈 | 输入时立即验证 |
| 错误引导 | 明确指出错误位置和原因 |
| 智能验证 | 失焦验证 vs 提交验证 |

## 3. 专业建议

### A. 表单结构

```html
<form novalidate>
  <div class="form-group">
    <label for="email">邮箱 <span class="required">*</span></label>
    <input 
      type="email" 
      id="email" 
      name="email"
      required
      autocomplete="email"
      aria-describedby="email-error"
    >
    <span id="email-error" class="error" role="alert"></span>
  </div>
  
  <button type="submit">提交</button>
</form>
```

### B. 即时验证

```javascript
const form = document.querySelector('form');
const inputs = form.querySelectorAll('input');

const validators = {
  email: (value) => {
    if (!value) return '请输入邮箱';
    if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value)) {
      return '请输入有效的邮箱地址';
    }
    return '';
  },
  password: (value) => {
    if (!value) return '请输入密码';
    if (value.length < 8) return '密码至少8位';
    if (!/[A-Z]/.test(value)) return '需包含大写字母';
    return '';
  }
};

// 失焦时验证
inputs.forEach(input => {
  input.addEventListener('blur', () => {
    const error = validators[input.name]?.(input.value);
    showError(input, error);
  });
  
  // 输入时清除错误
  input.addEventListener('input', () => {
    clearError(input);
  });
});
```

### D. 用户体验技巧

1. **必填项标识**：用 * 号，位置统一
2. **占位符**：用示例而非标签替代
3. **自动完成**：使用 autocomplete 属性
4. **密码眼睛**：切换密码可见性
5. **手机验证码**：倒计时按钮

```html
<!-- 自动完成示例 -->
<input type="email" autocomplete="email">
<input type="tel" autocomplete="tel">
<input type="name" autocomplete="name">
<input type="cc-number" autocomplete="cc-number">
```

### E. 常见验证规则

| 场景 | 正则 | 说明 |
|------|------|------|
| 手机号 | ^1[3-9]\d{9}$ | 11位数字 |
| 邮箱 | ^[^\s@]+@[^\s@]+\.[^\s@]+$ | 标准格式 |
| 身份证 | ^\d{17}[\dXx]$ | 18位 |
| URL | ^https?://.* | http/https开头 |
| 密码 | ^(?=.*[A-Z])(?=.*\d).{8,}$ | 8位+大写+数字 |
