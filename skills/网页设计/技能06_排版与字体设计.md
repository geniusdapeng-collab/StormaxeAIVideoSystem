## TL;DR

排版核心：足够的行高(1.5-1.8) + 合理的字重对比 + 舒适的字体大小(16px+)

## 1. 场景分析

### 请求摘要
用户需要为网页选择字体和设置排版

### 显性需求
- 掌握排版基础
- 获得可用的字体方案

### 场景分类
- `scenario_classification`: UI设计
- `urgency`: normal

## 2. 排版基础

### 字体分类
| 类型 | 特点 | 适用 | 示例 |
|------|------|------|------|
| 无衬线体 | 现代、简洁 | 正文、界面 | Inter, Noto Sans |
| 衬线体 | 传统、优雅 | 标题、文章 | Noto Serif, Source Serif |
| 等宽字体 | 规整 | 代码、数据 | JetBrains Mono |

### 黄金法则
- 正文字体：16px+，行高1.5-1.8
- 标题：字重700+，比例1.25-1.5
- 段落：最长60-75字符

## 3. 专业建议

### A. 中文字体方案

```css
/* 方案1：系统字体栈（最快） */
body {
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
}

/* 方案2：思源黑体（Google Fonts） */
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;500;700&display=swap');

body {
  font-family: 'Noto Sans SC', sans-serif;
}

/* 方案3：进阶衬线体 */
h1, h2 {
  font-family: 'Noto Serif SC', serif;
}
```

### B. 响应式字体

```css
/* 响应式缩放 */
html {
  font-size: clamp(14px, 2vw, 18px);
}

/* 或使用媒体查询 */
body { font-size: 16px; }
@media (max-width: 768px) {
  body { font-size: 14px; }
}
```

### C. 字体层级

```css
:root {
  --text-xs: 0.75rem;    /* 12px */
  --text-sm: 0.875rem;    /* 14px */
  --text-base: 1rem;     /* 16px */
  --text-lg: 1.125rem;   /* 18px */
  --text-xl: 1.25rem;    /* 20px */
  --text-2xl: 1.5rem;    /* 24px */
  --text-3xl: 1.875rem;  /* 30px */
  --text-4xl: 2.25rem;   /* 36px */
  
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.75;
}
```

### D. 字体加载优化

```html
<!-- 预加载关键字体 -->
<link rel="preload" href="/fonts/inter-var.woff2" as="font" type="font/woff2" crossorigin>

<!-- 使用font-display -->
@font-face {
  font-family: 'Inter';
  src: url('inter-var.woff2') format('woff2');
  font-display: swap;  /* 防止FOUT/FOIT */
}
```
