## TL;DR

配色核心：60-30-10法则 + 足够的对比度 + 符合品牌调性

## 1. 场景分析

## 2. 配色基础

### 60-30-10法则
| 比例 | 用途 | 示例 |
|------|------|------|
| 60% | 主色（背景） | 白色/浅灰 |
| 30% | 辅色（主体） | 品牌主色 |
| 10% | 强调色（CTA） | 亮橙/亮蓝 |

### 色彩心理学

| 色系 | 情感联想 | 适用场景 |
|------|----------|----------|
| 蓝色 | 信任、专业、科技 | 金融、SaaS、企业 |
| 绿色 | 自然、健康、成长 | 环保、健康、电商 |
| 橙色 | 活力、创意、促销 | 餐饮、电商、活动 |
| 紫色 | 奢华、创意、女性 | 时尚、奢侈品 |
| 红色 | 紧迫、热情、促销 | 电商、通知 |

## 3. 专业建议

### A. 配色方案类型

#### 单色系（最安全）
```css
:root {
  --primary: #2563eb;
  --primary-light: #3b82f6;
  --primary-dark: #1d4ed8;
  --surface: #f8fafc;
  --text: #1e293b;
}
```

#### 互补色（最强对比）
- 色轮上180度相对的颜色
- 适合需要强烈吸引注意力的场景

#### 类似色（最和谐）
- 色轮上相邻的30-60度
- 适合营造舒适、专业感

### B. 暗色模式配色

```css
@media (prefers-color-scheme: dark) {
  :root {
    --bg-primary: #0f172a;
    --bg-secondary: #1e293b;
    --text-primary: #f1f5f9;
    --text-secondary: #94a3b8;
    --accent: #38bdf8;
  }
}
```

### C. 无障碍对比度

WCAG 2.1 标准：
- 普通文本：4.5:1
- 大文本（18px+或14px粗体）：3:1
- UI组件：3:1
