# TL;DR

掌握科学的销量预测方法论，结合数据模型与业务经验，实现需求预测精准化、备货计划科学化，降低断货与积压双重风险。

# 场景分析

## 典型场景
- 大促备货：双11备多少货？备货不足错失GMV，备货过多清仓亏损
- 新品上市：第一次引进的品牌，完全没有历史数据怎么备货
- 季节性商品：护肤品有淡旺季，怎么提前规划
- 促销活动：店庆/会员日预计销量增长多少

## 核心挑战
1. **数据不足**：新品/新店没有历史参考
2. **波动大**：活动期间销量可能是日常的10倍
3. **多因素影响**：平台流量、竞品动作、价格变化
4. **时间跨度**：跨境备货周期长，要预测2-3个月后的需求

## 正确认知
- **预测是概率，不是确定**
- **数据+经验>纯数据**
- **滚动预测比一次性预测更准确**
- **误差不可避免，重点是控制在可接受范围**

# 决策框架

## 第一步：确定预测层级

### 预测层级体系

| 层级 | 范围 | 时间粒度 | 准确率要求 |
|------|------|----------|------------|
| 年度规划 | 全品类 | 月度 | 70-80% |
| 季度计划 | 品类 | 月度 | 75-85% |
| 月度计划 | SKU | 周度 | 80-90% |
| 周计划 | 爆款SKU | 日度 | 85-95% |

## 第二步：选择预测方法

### 预测方法对比

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 历史均值 | 稳定品 | 简单 | 无法捕捉趋势 |
| 移动平均 | 短期 | 稳定 | 滞后 |
| 趋势外推 | 上升/下降品 | 捕捉趋势 | 波动大不准 |
| 季节模型 | 季节性商品 | 考虑周期 | 需要多年数据 |
| 机器学习 | 数据充足 | 准确 | 需要技术能力 |
| 专家判断 | 新品/不确定 | 结合经验 | 主观偏差 |

## 第三步：建立预测模型

### 基础预测模型

#### 模型1：简单移动平均
```
预测值 = Σ(近N期销量) ÷ N
```

#### 模型2：加权移动平均
```
预测值 = Σ(近N期销量 × 权重) ÷ Σ权重
近期权重 > 远期权重
```

#### 模型3：指数平滑
```
预测值 = α × 上期实际值 + (1-α) × 上期预测值
α = 平滑系数（0<α<1）
```

#### 模型4：趋势外推
```
预测值 = 基期值 + 趋势值 × 预测期数
趋势值 = (本期值 - 上期值) ÷ 时间跨度
```

# 专业建议

## 常规销量预测

### 历史数据分析

#### 数据清洗
- 剔除异常值（大促、限流）
- 修正数据缺失
- 统一时间口径

#### 基础指标计算

| 指标 | 计算公式 | 意义 |
|------|----------|------|
| 日均销量 | 总销量÷天数 | 基础需求 |
| 销量标准差 | STD(每日销量) | 波动程度 |
| 环比增长率 | (本期-上期)÷上期 | 增长趋势 |
| 同比增长率 | (本期-去年同期)÷去年同期 | 季节性 |

### 预测流程

#### Step 1：识别基线
```
基线销量 = 近90天日均销量（剔除异常）
```

#### Step 2：趋势调整
```
趋势因子 = 过去90天 vs 再前90天增长率
调整后基线 = 基线销量 × (1 + 趋势因子)
```

#### Step 3：季节调整
```
季节因子 = 本月历史均值 ÷ 全年历史均值
调整后预测 = 调整后基线 × 季节因子
```

#### Step 4：营销调整
```
营销因子 = 1 + 活动预期增幅
最终预测 = 调整后预测 × 营销因子
```

## 大促销量预测

### 大促预测模型

#### 历史大促数据参考
| 活动 | 销量倍数（vs平日） | 备货系数 |
|------|-------------------|----------|
| 双11 | 8-15倍 | 1.2-1.5 |
| 618 | 5-10倍 | 1.2-1.3 |
| 黑五 | 6-12倍 | 1.2-1.4 |
| 99划算节 | 3-6倍 | 1.15-1.25 |
| 38节 | 2-4倍 | 1.1-1.2 |

#### 大促预测公式
```
大促销量预测 = 平日日均 × 大促倍数 × 天数 × 流量系数 × 安全系数
```

### 大促备货量计算

#### 公式
```
备货量 = 大促预测销量 + 预估退货量 + 安全库存 - 当前库存 - 在途库存
```

#### 分阶段备货
| 阶段 | 时间 | 备货量占比 | 说明 |
|------|------|------------|------|
| 第一波 | 活动前60天 | 60% | 基础备货 |
| 第二波 | 活动前30天 | 30% | 补充备货 |
| 第三波 | 活动前7天 | 10% | 应急备货 |

### 大促预测调整因素

| 因素 | 影响 | 调整方向 |
|------|------|----------|
| 流量变化 | ±30% | 实时监控 |
| 价格力度 | ±20% | 提前锁定 |
| 竞品动作 | ±15% | 动态调整 |
| 库存情况 | ±10% | 保守备货 |
| 历史误差 | ±10% | 经验修正 |

## 新品需求预测

### 新品预测方法

#### 方法1：类比法
```
找相似老品的销量作为参考
考虑：品类、品牌、价格段相似度
修正：新品种草期更长
```

#### 方法2：市场容量法
```
新品销量 = 市场容量 × 预期份额
市场容量 = 品类大盘 × 增长系数
预期份额 = 品牌力 × 投入资源
```

#### 方法3：专家法
```
结合运营、供应链、采购多方意见
给予不同权重
保守估计×信心系数
```

### 新品备货策略

| 阶段 | 策略 | 备货量 | 说明 |
|------|------|--------|------|
| 上市初期 | 保守 | 预估30天销量 | 快速反应 |
| 爬坡期 | 观察 | 预估45天销量 | 每周调整 |
| 稳定期 | 标准 | 预估60天销量 | 纳入常规 |

## 季节性预测

### 季节性识别

#### 季节因子计算
```
季节指数 = 月均销量 ÷ 年均日销量
>1 为旺季
<1 为淡季
=1 为平季
```

### 季节性商品分类

| 类型 | 特征 | 预测方法 |
|------|------|----------|
| 强季节 | 淡旺季明显（防晒、取暖器） | 季节模型 |
| 弱季节 | 季节影响小（食品） | 趋势+微调 |
| 反季节 | 提前备货（羽绒服春卖） | 预测前置 |

### 季节性预测策略

```
预测销量 = 趋势分量 + 季节分量 + 随机分量
```

#### 旺季备货建议
- 提前2-3个月开始补货
- 旺季前30天达到最高库存
- 旺季结束前2周开始清仓

## 预测精度管理

### 预测误差评估

#### 误差指标
| 指标 | 计算 | 目标 |
|------|------|------|
| MAPE | 平均绝对百分比误差 | <20% |
| MAE | 平均绝对误差 | - |
| RMSE | 均方根误差 | - |
| BIAS | 偏差率 | ±5% |

#### MAPE计算
```
MAPE = Σ(|实际值-预测值|÷实际值) ÷ N × 100%
```

### 预测准确性追踪

| 品类 | MAPE目标 | 说明 |
|------|----------|------|
| 爆款 | <10% | 数据充足，模型成熟 |
| 核心款 | <15% | 历史参考 |
| 普通款 | <20% | 波动较大 |
| 长尾款 | <30% | 数据稀少 |
| 新品 | <40% | 缺乏历史 |

### 预测改进机制

#### 每周复盘
- 追踪预测vs实际
- 分析误差原因
- 调整预测模型

#### 模型迭代
- 收集新数据
- 优化模型参数
- 测试新方法
