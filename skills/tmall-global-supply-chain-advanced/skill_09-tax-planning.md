# TL;DR

掌握天猫国际跨境电商全链路税务筹划，从进口关税到增值税，从综合税到所得税，在合规框架内实现税负最优化。

# 场景分析

## 典型场景
- 选品时不知道这个商品的税负率，算不清楚利润
- 同样的商品，走保税模式和直邮模式税负差很多，不知道怎么选
- 关税降了，但消费者感知不到，不知道该怎么利用政策红利
- 税局查账，不知道自己的税务处理有没有问题

## 核心挑战
1. **税种复杂**：关税、增值税、消费税、综合税...各种税怎么算
2. **模式差异**：保税、直邮、一般贸易税负差异大
3. **政策变化**：跨境电商政策年年变，跟不上
4. **合规风险**：节税过了头，变成偷税漏税

## 正确认知
- **合规是底线，筹划是手段**
- **税务成本是商品定价的重要组成**
- **政策红利要用足，但要在合规框架内**
- **专业的事交给专业的人，但运营要懂基础**

# 决策框架

## 第一步：了解税种结构

### 跨境电商主要税种

| 税种 | 征收环节 | 特点 |
|------|----------|------|
| 关税 | 进口时 | 按HS编码税率 |
| 增值税 | 进口/销售时 | 13%/9%/6% |
| 消费税 | 进口时 | 部分商品征收 |
| 综合税 | 跨境电商 | 优惠税率 |
| 所得税 | 企业经营 | 25%/小微15% |

## 第二步：选择进口模式

### 进口模式税负对比

| 模式 | 关税 | 增值税 | 综合税 | 适用 |
|------|------|--------|--------|------|
| 保税备货 | 0 | 0 | 按综合税 | 爆款批量 |
| 直邮 | 0 | 0 | 按综合税 | 长尾新品 |
| 一般贸易 | 按税率 | 13% | 无 | 大批量 |

## 第三步：制定筹划策略

### 筹划策略层次
| 层次 | 策略 | 风险 | 收益 |
|------|------|------|------|
| 合规层 | 正确归类、申报 | 无 | 避免处罚 |
| 优惠层 | 利用政策、限额 | 低 | 节省5-15% |
| 优化层 | 模式组合、结构 | 中 | 节省10-25% |

# 专业建议

## 进口关税筹划

### 关税计算

```
关税 = 完税价格 × 关税税率
```

### 完税价格确定

#### 组成
```
完税价格 = CIF价（货值 + 运费 + 保险费）
```

#### 价格申报原则
| 原则 | 说明 |
|------|------|
| 实际成交价 | 真实交易价格 |
| 完整价格 | 包含全部费用 |
| 唯一价格 | 不得有条件价格 |

### HS编码筹划

#### 归类原则
- 遵循协调制度
- 参考类注、章注
- 合理运用注释

#### 常见误区
| 误区 | 风险 | 正确做法 |
|------|------|----------|
| 低税率归类 | 查验+处罚 | 正确归类 |
| 模糊归类 | 争议+延误 | 明确归类 |
| 随意归类 | 合规风险 | 咨询专业 |

## 增值税筹划

### 增值税计算

```
应纳税额 = 销项税额 - 进项税额
销项税额 = 销售额 × 税率
```

### 进项税抵扣

#### 可抵扣项目
| 项目 | 税率 | 说明 |
|------|------|------|
| 采购成本 | 按采购税率 | 发票合规 |
| 物流成本 | 9% | 专票 |
| 仓储成本 | 6% | 专票 |
| 推广费 | 6% | 专票 |

#### 抵扣合规
- 发票合规（专票）
- 业务真实
- 三流一致（合同/发票/付款）

### 增值税优惠

| 优惠类型 | 适用条件 | 优惠内容 |
|----------|----------|----------|
| 小微优惠 | 年销售额<500万 | 税率优惠 |
| 即征即退 | 特定商品 | 先征后退 |
| 免税 | 特定服务 | 免征增值税 |

## 跨境电商综合税

### 综合税计算

```
综合税 = 完税价格 × 税率 ÷ (1 - 税率)
```

### 综合税税率表

| 商品类别 | 税率 | 优惠限额 |
|-----------|------|----------|
| 奶粉 | 9.1% | 单次500元免征 |
| 纸尿裤 | 9.1% | 单次500元免征 |
| 化妆品 | 23.06% | 单次5000元 |
| 保健品 | 9.1% | 单次500元免征 |
| 食品 | 9.1% | 单次500元免征 |
| 服装 | 20% | 无 |
| 电器 | 20% | 无 |

### 综合税筹划

#### 单次限额优化
| 策略 | 说明 | 效果 |
|------|------|------|
| 套装拆单 | 套装拆成多单 | 每单享免征 |
| 规格设计 | 单品定价<500 | 享受免征 |
| 组合销售 | 多品打包 | 控制单次金额 |

#### 订单结构优化
```
错误做法：
- 1000元/件的奢侈品 → 综合税230元

正确做法：
- 拆分为2单500元 → 综合税0元（享免征）
```

## 进口模式税负对比

### 保税备货模式

#### 税负计算
```
综合税 = 货值 × 综合税率 ÷ (1 - 综合税率)

示例：100元奶粉
综合税 = 100 × 9.1% ÷ (1 - 9.1%) = 10元
```

#### 适用场景
- 爆款、高销量商品
- 需要大量备货
- 成本敏感

### 直邮模式

#### 税负计算
```
综合税 = 完税价格 × 综合税率 ÷ (1 - 综合税率)
```

#### 适用场景
- 长尾商品
- 新品测试
- 高价值商品

### 一般贸易模式

#### 税负计算
```
关税 = 货值 × 关税税率
增值税 = (货值 + 关税) × 13%
消费税 = (货值 + 关税) ÷ (1 - 消费税率) × 消费税率
```

#### 适用场景
- 大批量采购
- 非跨境清单商品
- 有一般贸易资质

### 模式对比选择

```
选择决策树：

商品在跨境电商正面清单？
├── 是 → 跨境电商模式
│   ├── 销量>1000/月 → 保税备货
│   └── 销量<1000/月 → 直邮
└── 否 → 一般贸易模式
```

## FTA关税优惠

### 主要FTA协定

| 协定 | 覆盖国家 | 优惠幅度 |
|------|----------|----------|
| RCEP | 中日韩+东盟 | 关税逐步降至0 |
| 中国-东盟 | 东盟10国 | 0-10% |
| 中国-韩国 | 韩国 | 0-15% |
| 中国-澳大利亚 | 澳大利亚 | 0-10% |
| 中国-瑞士 | 瑞士 | 0-10% |

### 原产地优惠利用

#### RCEP累积规则
```
在RCEP成员国区域内加工的原材料，可以累积计算原产地成分
```

#### 关税优惠利用步骤
1. 确认商品是否在协定覆盖范围
2. 确认原产地标准
3. 准备原产地证明
4. 申报时勾选优惠

### FTA优惠申请

#### 申请流程
```
准备资料 → 申请原产地证 → 出口申报 → 进口享受优惠
```

#### 所需材料
| 材料 | 说明 |
|------|------|
| 原产地证明 | FORM E/R/P等 |
| 发票 | 商业发票 |
| 运输单据 | 提单等 |
| 签证机构 | 贸促会/海关 |
