# TL;DR

掌握供应链全链路成本分析方法，从采购到物流到库存，系统性识别降本机会，制定可落地的降本方案，实现真正的降本增效。

# 场景分析

## 典型场景
- 老板说"成本太高了，要降本20%"，不知道从哪下手
- 单纯的压价谈判，供应商利润被压没了，质量服务也跟着下降
- 降本方案定了，但执行不下去，供应商不配合
- 降本效果难衡量，降了多少不知道

## 核心挑战
1. **成本不透明**：只知道总成本，不知道成本在哪里
2. **降本误区**：以为降本就是压价，结果质量服务下降
3. **执行困难**：降本方案落地难，供应商不配合
4. **效果难衡量**：降了多少，ROI多少，说不清楚

## 正确认知
- **降本不是简单压价，是系统优化**
- **TCO思维比单价思维更重要**
- **降本的目的是提升竞争力，不是克扣供应商**
- **降本要可持续，不能牺牲质量和供应稳定性**

# 决策框架

## 第一步：成本透视

### 全链路成本结构

```
采购成本 = CIF价 + 关税 + 物流费 + 汇率损耗

运营成本 = 仓储费 + 操作费 + 配送费 + 退货处理

资金成本 = 库存占用资金 + 账期成本

总成本(TCO) = 采购成本 + 运营成本 + 资金成本
```

### 成本归因

| 成本项 | 归因对象 | 优化责任 |
|--------|----------|----------|
| CIF价 | 采购 | 采购谈判 |
| 关税 | 合规 | 政策利用 |
| 头程物流 | 物流 | 物流优化 |
| 仓储费 | 仓储 | 库存优化 |
| 操作费 | 仓储 | 流程优化 |
| 配送费 | 物流 | 快递议价 |
| 退货处理 | 运营 | 质量改进 |

## 第二步：识别降本机会

### 降本机会矩阵

```
         降本空间
              ↑
              │
    快速见效  │    战略优化
   (战术层)  │   (战略层)
              │
              ├──────────────→ 实施难度
              │
              │    长期投入  │   复杂协同
              │   (组织层)  │
```

### 降本机会分类

| 类型 | 特点 | 典型措施 |
|------|------|----------|
| 快速见效 | 实施周期短，效果明显 | 批量议价、快递议价 |
| 流程优化 | 需要流程调整 | 入库优化、打包标准化 |
| 战略优化 | 需要供应商协同 | VMI、联合采购 |
| 组织变革 | 需要组织调整 | 供应商整合、模式切换 |

## 第三步：制定降本方案

### 降本方案要素
- 降本目标（具体数字）
- 实施方案（谁做、怎么做）
- 资源投入（人、系统、钱）
- 时间计划（什么时候完成）
- ROI测算（投入产出比）

# 专业建议

## 采购降本策略

### 源头降本

#### 供应商整合
| 策略 | 说明 | 降本空间 |
|------|------|----------|
| 整合SKU | 同类商品集中采购 | 5-15% |
| 整合供应商 | 减少供应商数量 | 3-8% |
| 整合品类 | 全品类打包谈 | 5-10% |

#### 采购模式优化
| 模式 | 适用场景 | 降本空间 |
|------|----------|----------|
| 集中采购 | 多店铺/多品类 | 5-10% |
| 年度框架 | 稳定商品 | 3-8% |
| 战略合作 | 核心供应商 | 5-15% |

### 谈判降本

#### 谈判降本框架

```
谈判降本 = 基线成本 - (目标降幅 × 基线成本)

示例：
- 基线成本：100元
- 目标降幅：10%
- 目标成本：90元
- 谈判降幅：7%（实际达成）
- 实际成本：93元
```

#### 谈判策略

| 策略 | 适用场景 | 效果 |
|------|----------|------|
| 批量议价 | 采购量大 | 5-15% |
| 年度框架 | 长期合作 | 3-10% |
| 阶梯价格 | 增量采购 | 5-20% |
| 成本透明 | 信任关系 | 3-8% |

### 成本结构优化

#### 关税优化
| 策略 | 说明 | 适用 |
|------|------|------|
| 原产地优化 | 寻找优惠原产地 | 有替代产地 |
| 商品归类 | 合理HS编码归类 | 归类有争议 |
| 政策利用 | RCEP等自贸协定 | 有协定覆盖 |

#### 汇率管理
| 策略 | 说明 | 效果 |
|------|------|------|
| 远期锁汇 | 锁定未来汇率 | 避免波动 |
| 结算币种 | 优选强势币种 | 节省1-3% |
| 结算时机 | 汇率低点结算 | 看市场 |

## 物流降本策略

### 头程降本

#### 物流方案对比

| 方案 | 时效 | 成本 | 适用 |
|------|------|------|------|
| 整柜海运 | 20-30天 | 最低 | 大批量、不紧急 |
| 拼柜海运 | 25-35天 | 低 | 中等批量 |
| 空运 | 5-7天 | 高 | 小批量、紧急 |
| 快递 | 3-5天 | 最高 | 样品、紧急补货 |

#### 头程降本措施

| 措施 | 说明 | 降本空间 |
|------|------|----------|
| 批量发货 | 集中发运 | 5-15% |
| 散货集拼 | 多SKU拼柜 | 3-8% |
| 物流商比价 | 年度比价 | 5-10% |
| 路线优化 | 优选路线 | 3-5% |

### 尾程降本

#### 快递降本

| 措施 | 说明 | 降本空间 |
|------|------|----------|
| 协议价谈判 | 年度协议 | 10-25% |
| 阶梯运量 | 增量换折扣 | 5-15% |
| 快递整合 | 减少快递商 | 3-5% |
| 包装优化 | 减轻重量 | 2-5% |

## 仓储降本策略

### 仓储成本构成

| 成本项 | 影响因素 | 优化方向 |
|--------|----------|----------|
| 仓储费 | 库存量、库型 | 减少库存 |
| 操作费 | SKU数、订单量 | 批量操作 |
| 耗材费 | 包装材料 | 包装优化 |
| 货损费 | 仓储管理 | 流程改进 |

### 仓储降本措施

#### 库存优化

| 措施 | 说明 | 降本空间 |
|------|------|----------|
| 周转优化 | 提高库存周转 | 库存资金↓ |
| 呆滞清理 | 减少无效库存 | 仓储费↓ |
| 库存可视化 | 实时监控 | 积压↓ |

#### 操作优化

| 措施 | 说明 | 降本空间 |
|------|------|----------|
| 批量作业 | 减少零散操作 | 10-20% |
| 标准化 | 统一流程 | 5-10% |
| 自动化 | 引入设备 | 15-30% |

## 退货降本策略

### 退货成本结构

```
退货成本 = 退货商品成本 + 物流成本 + 操作成本 + 商品损耗
```

### 退货降本措施

| 策略 | 说明 | 降本空间 |
|------|------|----------|
| 质量改进 | 减少退货源头 | 退货率↓ |
| 详情优化 | 减少信息不对称 | 退货率↓ |
| 退货再利用 | 翻新/降价卖 | 损耗↓ |
| 退供处理 | 退回供应商 | 部分回收 |

## 降本ROI测算

### ROI计算

```
降本ROI = (降本金额 - 投入成本) ÷ 投入成本 × 100%
```

### 降本项目评估表

| 项目 | 降本金额 | 投入成本 | ROI | 优先级 |
|------|----------|----------|-----|--------|
| 快递议价 | 50万 | 5万 | 900% | 高 |
| VMI | 100万 | 30万 | 233% | 高 |
| 批量采购 | 30万 | 10万 | 200% | 中 |
| 系统优化 | 20万 | 50万 | -60% | 低 |
