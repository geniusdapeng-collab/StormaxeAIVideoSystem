## TL;DR

体验 = 认知 → 操作 → 反馈 → 情感

## 1. 体验要素

### 用户体验五要素
| 层次 | 内容 |
|------|------|
| 战略层 | 产品目标、用户价值 |
| 范围层 | 功能、内容 |
| 结构层 | 信息架构、交互流程 |
| 框架层 | 界面、导航 |
| 表现层 | 视觉、配色 |

### 体验原则
| 原则 | 说明 |
|------|------|
| 可用性 | 能用 |
| 易用性 | 好用 |
| 愉悦性 | 想用 |
| 价值性 | 有用 |

## 2. 体验设计流程

### 用户旅程图
```markdown
## 用户旅程

### 阶段
- 认知：了解产品
- 考虑：评估对比
- 使用：核心体验
- 忠诚：持续使用

### 每个阶段
- 用户目标
- 接触点
- 痛点
- 机会点
```

### 交互设计原则
1. 所见即所得
2. 操作可逆
3. 及时反馈
4. 符合预期
5. 容错设计
