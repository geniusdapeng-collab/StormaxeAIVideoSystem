# 🍜 美食摄影指南

**「让食物自己说话」** —— 一张好图，胜过千言万语。

---

## 一、构图法则三大绝招

### 1.1 三角形构图法 ⭐最稳

**核心原理**：画面主体形成稳定的三角形，引导视觉焦点

**实操方法**：
- 将主菜放在画面中央偏下
- 左右两侧摆放配菜、餐具或饮品
- 形成「主菜+副菜+装饰物」的稳定三角

**适用场景**：
- 中餐圆桌菜（主菜+2个小碟）
- 西餐摆盘（主盘+副盘+酒杯）
- 日料定食（主菜+小菜+味噌汤）

**⚠️ 大鹏避坑**：
> 三角形不要等边！稍微打破对称才更生动
> 主菜要最大最突出，副菜作为点缀不能抢戏

---

### 1.2 留白构图法 ⭐最高级

**核心原理**：给食物「呼吸」的空间，不贪心

**实操方法**：
- 食物占画面 60-70%
- 留白区域可放餐具、桌布、调料瓶
- 留白处可以是纯色桌面或背景布

**适用场景**：
- 高级餐厅氛围照
- 单品特写（咖啡、拉面、蛋糕）
- 文艺小清新风格

**⚠️ 大鹏避坑**：
> 留白不是留空！留白区域要有内容承接
> 背景杂乱时，用大光圈虚化或换角度

---

### 1.3 对角线构图法 ⭐最动感

**核心原理**：斜线打破呆板，动静结合

**实操方法**：
- 将长条形食物（法棍、面条、牛排）沿对角线摆放
- 餐具也可形成对角呼应
- 画面从左上到右下，或右上到左下

**适用场景**：
- 面条、意粉等长条形食物
- 牛排切开的剖面
- 手握咖啡杯的侧面

**⚠️ 大鹏避坑**：
> 背景不能有明显的水平线干扰
> 倾斜角度控制在 30-60 度，不要过于夸张

## 二、光线利用技巧

### 2.1 自然光是王道 🌞

**最佳时间**：
- 窗边自然光：上午 10 点前 / 下午 3 点后
- 避免正午阳光直射（硬光导致阴影过重）

**光线方向**：
```
    ☀️ 窗
    ↓
    ┌─────┐
    │ 🍽️ │  ← 食物放在光源同一侧
    │     │     光从侧面打过来
    └─────┘     阴影在另一侧
```

**实操技巧**：
- 让食物「面朝光源」
- 用白色餐巾纸或外卖纸袋当反光板补阴影面
- 拉上窗帘柔化强烈阳光

### 2.2 手机补光应急方案 📱

**没有自然光时**：
- 打开另一部手机的「手电筒」照向天花板
- 利用餐厅顶灯，但要避免直射产生反光
- 白色餐盘反光补亮

**⚠️ 大鹏避坑**：
> 不要开闪光灯！会把食物拍得惨白
> 食物表面有油光时，正面强光会过曝

## 三、手机摄影快速上手

### 3.1 基础参数设置

**iPhone 用户**：
- 关闭 HDR（避免食物颜色失真）
- 打开网格线（辅助构图）
- 对焦点锁定：长按屏幕锁定曝光

**安卓用户**：
- 专业模式：WB 白平衡设为「日光」
- 关闭 AI 美食模式（经常过度饱和）
- 测光点放在食物最亮处

### 3.2 放大倍数选择

| 食物类型 | 推荐倍数 | 原因 |
|---------|---------|------|
| 整桌菜 | 1x | 展现全貌 |
| 单份主食 | 1.5x-2x | 主体突出 |
| 细节特写 | 2x-3x | 纹理质感 |

**⚠️ 大鹏避坑**：
> 不要用数码变焦（画质会变渣）
> 近拍时用手动拉远而非数码放大

### 3.3 一秒提升质感的诀窍

**必做动作**：
1. **擦盘子**：用湿纸巾把盘边污渍擦干净
2. **理菜相**：把歪的摆正，散的归拢
3. **加点缀**：香菜叶、小米椒、柠檬片点缀
4. **等一等**：热气腾腾时拍最有食欲（凉了就没了）

**⚠️ 大鹏避坑**：
> 外卖拍照前别急着打开，等热气散了就「塌房」了
> 咖啡拉花拍照要快，拉花 30 秒就会「化掉」

## 四、朋友圈美食九宫格攻略

### 4.1 排版黄金法则

**首选 2+1 排布**（最常见）：

```
┌─────┬─────┬─────┐
│ 1号 │ 2号 │ 3号 │
│ 主角 │ 配角 │ 配角 │  ← 第一排放菜品全家福
├─────┼─────┼─────┤
│ 4号 │ 5号 │ 6号 │
│ 环境 │ 特写 │ 细节 │  ← 第二排放场景/细节
├─────┼─────┼─────┤
│ 7号 │ 8号 │ 9号 │
│ 人物 │ 人物 │ 美食 │  ← 第三排放人物互动
└─────┴─────┴─────┘
```

**1号位原则**：
- 1号位是视觉焦点，决定整体调性
- 放「最有食欲」或「最想炫耀」的一道菜
- 不要放空盘、环境图在 1 号位

### 4.2 不同场景的排版策略

**场景一：探店打卡**

```
推荐排法：1-1-1 竖图拼接
┌────┐ ┌────┐ ┌────┐
│全景│ │单品│ │氛围│
└────┘ └────┘ └────┘
→ 1号位放餐厅全景，2号位放招牌菜，3号位放环境
```

**场景二：家庭聚餐**

```
推荐排法：中间大图
┌────┬────┬────┐
│配角│ 中间 │配角│
│    │ 大图 │    │
├────┤     ├────┤
│细节│     │细节│
└────┴─────┴────┘
→ 中间放全家福或主菜特写
```

**场景三：单一美食分享**

```
推荐排法：1+1+1 连续故事
第1张：原材料/准备
第2张：刚上桌热气腾腾
第3张：动筷后的状态
→ 讲好一个「从生到熟」的故事
```

### 4.3 文案配合技巧

**标题党公式**：
```
【地点+感受】+【菜品名】
例：「又来这家宝藏小店！红烧肉绝了🐷」
```

**互动引导**：
- 「猜猜这是哪家？」→ 引发评论
- 「被这碗面馋哭了」→ 引发共鸣
- 「下次带XXX来」→ 引发好奇

**⚠️ 大鹏避坑**：
> 不要发空盘（说明不好吃或吃完了没拍到）
> 不要连续发美食（会被拉黑）
> 最佳频率：一天不超过 3 条美食

## 五、快速检查清单

拍照前过一遍「美食五连拍」：

- [ ] **角度**：45度俯拍（最安全）/ 正上方俯拍（更专业）/ 侧面拍（展示层次）
- [ ] **光线**：自然光优先，没有就用补光
- [ ] **背景**：整洁无杂物，杂乱就虚化
- [ ] **对焦**：点击食物主体锁定焦点
- [ ] **检查**：拍完放大看是否糊片

## 六、常见翻车现场急救

### 翻车1：餐厅灯光太暗

**急救方案**：
- 打开手机手电筒照天花板，用反射光
- 找窗边位置借光
- 调高手机曝光度（对焦框旁边的小太阳↑）

### 翻车2：颜色拍出来发灰

**急救方案**：
- 关闭 HDR
- 后期用 Snapseed「氛围」+10，「饱和度」+5
- 对着白色纸巾校准白平衡（iPhone 选 AEB）

### 翻车3：油光反光严重

**急救方案**：
- 换个角度拍，让光源在侧面而非正面
- 用浅色桌布吸光
- 轻微喷水雾制造「新鲜」感
