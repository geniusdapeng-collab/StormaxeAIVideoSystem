## 🚀 快手菜谱库

### 核心原则

**"炒菜10分钟 + 备菜20分钟 = 30分钟完整晚餐"**

> 💡 **小G提醒**：香香现在5个月+，如果是辅食阶段，需要单独准备。大人的快手菜+宝宝辅食，一次搞定两份！

---

### 🥇 顶级推荐：15分钟快手套餐

#### 【可乐鸡翅 + 蒜蓉西兰花 + 米饭】

| 菜品 | 烹饪时间 | 难度 | 适合宝宝 |
|------|----------|------|----------|
| 可乐鸡翅 | 12分钟 | ⭐ | ❌（含糖） |
| 蒜蓉西兰花 | 5分钟 | ⭐ | ✅（切碎） |
| 宝宝南瓜泥 | 10分钟 | ⭐ | ✅✅ |

**操作流程（12分钟鸡翅版）**：

```
⏰ 时间轴
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0:00  鸡翅正反面划刀，冷水下锅焯水去腥
0:03  捞出沥干，热锅少油煎至两面金黄
0:06  加可乐（没过鸡翅1/3）+ 生抽2勺 + 老抽半勺
0:08  大火收汁，汤汁变稠裹满鸡翅即可
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> 🎯 **Tip**：用无糖可乐或减半，大人吃健康一点

---

#### 【番茄炒蛋盖饭 + 紫菜蛋花汤】

| 菜品 | 烹饪时间 | 难度 | 宝宝辅食 |
|------|----------|------|----------|
| 番茄炒蛋 | 8分钟 | ⭐ | ✅（番茄去皮切碎） |
| 紫菜蛋花汤 | 5分钟 | ⭐ | ✅✅ |

**黄金番茄炒蛋秘诀**：

```
🥚 番茄炒蛋好吃的3个关键：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1️⃣ 鸡蛋打散加少许盐，煎至8分熟盛出
2️⃣ 番茄切块，热锅炒出汁（可加一点点水）
3️⃣ 鸡蛋回锅，加糖少许，翻炒均匀即可
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 🥉 懒人福音：微波炉/电饭煲料理

#### 【电饭煲焖饭】

```
🍚 一锅端焖饭（30分钟）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
食材：大米 + 腊肠/腊肉 + 土豆 + 胡萝卜 + 豌豆
调料：生抽 + 蚝油 + 少许盐

做法：
1. 所有食材切丁，大米洗净
2. 放入电饭煲，正常煮饭水位
3. 调料均匀淋在食材上
4. 煮饭键跳起后焖5分钟
5. 搅拌均匀，香气扑鼻！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> 🎯 **升级版**：加一颗鸡蛋，戳破拌匀，完美！

---

#### 【微波炉蒸蛋】

```
🥚 完美蒸蛋（宝宝最爱）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
食材：鸡蛋2个 + 温水（鸡蛋液的1-1.5倍）
调料：少许盐 + 几滴香油

做法：
1. 鸡蛋打散，加入温水搅拌均匀
2. 过筛去掉气泡（这是嫩滑关键！）
3. 盖上盘子，微波炉中高火3分钟
4. 出锅加香油 + 葱花

⏱️ 适合：6个月+宝宝，大人配米饭超赞
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```

```


### 👨‍👩‍👧 亲子厨房活动设计

```
⏱️ 完成后一起说："香香好棒！"
```
