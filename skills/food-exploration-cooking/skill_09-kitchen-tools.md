# 🫕 厨房装备指南

*适用场景：厨房装备选购、快手料理准备、收纳整理、装备升级*

## 🎯 一、必备厨具清单（快手料理版）

> 大鹏你平时熬夜写代码，周末又要陪香香，时间宝贵。快手料理的核心是：**工具要精不要多，每件都要用得上。**

### 1.1 锅具三件套（核心）

| 装备 | 推荐类型 | 选购要点 | 预算参考 |
|:---:|:---:|:---|:---:|
| **炒锅** | 32cm熟铁锅 | 窒化铁/窒化氮工艺，不粘靠养 | 300-600元 |
| **汤锅/奶锅** | 18-20cm不锈钢 | 304食品级，带蒸格 | 150-300元 |
| **煎锅** | 26cm不粘锅 | 陶瓷涂层 or 麦饭石（备孕/有娃家庭优先） | 200-400元 |

**🔥 小G吐槽：** 那些买了十几口锅结果吃灰的案例我见太多了！你又不是开饭馆，三口锅走天下足够了。炒锅负责爆炒、汤锅负责煮炖、煎锅负责蛋和饼，分工明确效率翻倍。

### 1.2 刀具基础款

| 装备 | 推荐 | 选购要点 | 预算参考 |
|:---:|:---|:---|:---:|
| **主刀** | 三德刀（18cm）or 中式菜刀 | 日系 VG10+/国产邓家刀 | 200-500元 |
| **小刀** | 水果刀/削皮刀 | 便宜好用就行 | 30-80元 |

**⚠️ 注意：** 别买整套刀具！99%的套装刀里那把小刀你用不上几次。买一把好主刀 + 一把小刀，够用到香香长大。

### 1.3 厨具配件

```
✅ 必备清单（精简版）
├── 木铲 / 硅胶铲（炒锅搭档）
├── 不锈钢漏勺 + 汤勺
├── 砧板 × 2（生/熟分开）
│   ├── 塑料砧板（生肉）- 百钻/妙洁
│   └── 木砧板 or 麦秆砧板（蔬果/熟食）
├── 保鲜盒套装（玻璃优先）
│   └── 500ml × 3 + 1L × 2
├── 计量勺/量杯（做烘焙或精确调味）
└── 厨房剪（剪食材/剪包装/剪香料）
```

## ☕ 二、咖啡/威士忌装备清单

> 大鹏你的两大心头好，手冲咖啡和单一麦芽威士忌。这部分不将就，要讲究。

### 2.1 手冲咖啡装备（核心配置）

```
☕ 手冲咖啡进阶路线（按需升级）

Level 1 - 入门（预算 500-800元）
├── 磨豆机：泰摩栗子C / Hario手摇磨
├── 手冲壶：Hario V60云朵壶 / 泰摩鱼smart
├── 滤杯套装：V60树脂滤杯 + 滤纸
├── 电子秤：小米厨房秤（精度0.1g）
└── 分享壶：玻璃 or 不锈钢

Level 2 - 进阶（预算 1500-2500元）
├── 磨豆机：卓卓Orphan / C40手摇
├── 手冲壶：Fellow Stagg EKG（温控壶）
├── 滤杯：Origami折纸滤杯
└── 分享壶：Hario玻璃

Level 3 - 发烧（预算 5000+）
├── 磨豆机：Mazzer /EK43（电动）
├── 手冲壶：Fellow Stagg EKG Pro
├── 滤杯：Kono MD-21 / Tetsu Kasuya联名
└── 配件：精准温控、预热设备
```

**💡 选购建议：**
- **磨豆机优先！** 磨豆机是咖啡品质的灵魂，省什么都别省磨。
- 豆子新鲜度比装备更重要，买新鲜烘焙的豆子比升级装备提升更明显。
- 大鹏你经常出差，建议选**手摇磨** + **便携滤杯**（比如Timemore Air、Origami Air），出差也能喝到好咖啡。

### 2.2 威士忌装备

```
🥃 威士忌老饕配置

核心装备
├── 格兰凯恩闻香杯（Glencairn）- 闻香必备
├── 古典杯 × 2-4（厚底防碰）
├── 酒壶 - 随身携带款（出差/露营）
└── 冰球模具 - 大块冰化得慢，逼格拉满

加分装备
├── 调酒壶 + 量杯（想玩Highball或古典鸡尾酒）
├── 威士忌石（不想化水又懒得冻冰）
└── 烈酒温度计（精准品鉴用）

❌ 不建议买的
├── 豪华酒架（占地儿，落灰）
├── 太多不同杯型（其实闻香杯够用了）
└── 整套调酒工具（除非你要开酒吧）
```

## 📦 三、厨房收纳方案

> 大鹏你是完美主义者，厨房乱糟糟肯定忍不了。收纳的核心是：**顺手比好看重要，好拿比藏起来重要。**

### 3.1 收纳原则

```
🗂️ 收纳黄金法则

1. 【伸手可得】原则
   → 灶台旁边30cm内：铲子、锅铲、调料（高频）
   → 水槽旁边：洗碗布、沥水架（中频）
   → 抽屉：工具、餐具（开抽屉拿）

2. 【按使用频率】分区
   → 第一层（眼平/手平）：每天用的
     - 炒锅、煎锅、常用调料、砧板
   → 第二层（下蹲）：每周用的
     - 备用锅、烤盘、保鲜盒
   → 第三层（垫脚/最上层）：每月用的
     - 节日才用的模具、备用一次性用品

3. 【台面无物】原则
   → 台面只留：刀、砧板、调味罐
   → 小家电尽量上墙 or 藏柜子里
   → 实在不行，用搁板or置物架
```

### 3.2 收纳神器推荐

| 神器 | 用途 | 推荐品牌/类型 | 预算 |
|:---|:---|:---|:---:|
| **磁吸刀架** | 刀具收纳+通风 | 实木磁吸刀架 | 100-200元 |
| **壁挂调料盒** | 灶台旁调料 | 宜家芬托 / 日式不锈钢 | 50-150元 |
| **抽屉分隔器** | 餐具分类 | 弹簧式分隔板 | 30-80元 |
| **旋转调料盘** | 角落调料收纳 | 360°旋转盘 | 50-100元 |
| **下挂置物架** | 橱柜门收纳 | 免打孔下挂篮 | 30-60元 |
| **玻璃保鲜盒** | 食材存放 | Pyrex / 乐扣乐扣 | 100-200元/套 |

### 3.3 咖啡装备收纳

```
☕ 咖啡角专属收纳（建议单独规划）

方案A - 开放式小推车
├── 第一层：磨豆机 + 手冲壶
├── 第二层：滤纸 + 分享壶
└── 轮子可移动，咖啡角随时搬家

方案B - 固定角落柜
├── 吊柜下挂杆：挂杯子
├── 台面：磨豆机 + 滤杯套装
└── 抽屉：咖啡豆（密封罐）、配件

💡 建议给咖啡角预留：
- 插座（电磨 or 热水壶）
- 靠近水槽（冲煮方便）
- 通风 or 排风扇（咖啡香气留香但油烟别串）
```

## 🚀 四、升级装备建议

> 大鹏你追求完美，这部分给有追求的你。升级不是买更多，而是**把高频工具升级到位**。

### 4.1 值得升级的单品（优先级排序）

```
🔥 高优先级（使用频率极高，升级体验明显）

1. 【磨豆机】
   - 现在用入门款？升级到 C40 / Orphan 级别
   - 研磨均匀度提升 → 咖啡品质明显提升
   - 预算：800-1500元

2. 【炒锅】
   - 现在用不粘锅？升级窒化铁锅
   - 开锅养锅后完全不粘，导热快
   - 预算：400-800元（优质熟铁）

3. 【刀具】
   - 现在用普通菜刀？升级邓家刀 or 三德刀
   - 切菜体验是质的飞跃
   - 预算：200-500元

4. 【手冲壶】
   - 现在用普通壶？升级温控壶
   - Fellow Stagg EKG（控温精准+颜值担当）
   - 预算：800-1200元
```

## 🔧 六、装备保养指南

### 铁锅开锅/养锅

```
1. 新锅清洗 → 擦干 → 中小火烘干
2. 猪油/肥肉擦锅内壁 → 小火冒烟 → 擦掉（重复3次）
3. 静置过夜 → 次日清洗 → 擦干 → 涂薄油
4. 养锅原则：热锅凉水洗 → 擦干 → 涂油
```

### 不粘锅保养

```
1. 避免空烧（超过30秒）
2. 不用铁铲，用木铲/硅胶铲
3. 冷却后再清洗（热锅不冲凉水）
4. 涂层划伤及时更换（影响健康）
5. 洗碗机慎用（加速涂层老化）
```

### 刀具保养

```
1. 定期磨刀（每月1次用磨刀棒，每天用磨刀器）
2. 不砍骨头（那是斧头的活）
3. 手洗不机洗（保护刃口）
4. 刀+砧板配套（好刀配软砧板更耐用）
```

## 🎁 附：给新晋奶爸大鹏的特别建议

> 香香2025年11月出生，现在应该快5个月了 🍼

**有娃家庭的厨房特别注意事项：**

1. **安全第一**
   - 刀具放高处 or 上锁抽屉
   - 小配件（电池、调料包）收好防误食
   - 插座保护盖安排上

2. **时间管理**
   - 周末趁香香睡觉备菜（切好分装）
   - 常用食材放顺手位置（减少抱娃找东西）
   - 洗碗机安排上（省时间陪娃）

3. **饮食升级**
   - 给香香做辅食需要：辅食机 or 料理棒、蒸锅、研磨碗
   - 备一套硅胶模具（冰格+蒸糕模具）
   - 婴儿食品存储用独立冰箱区域
