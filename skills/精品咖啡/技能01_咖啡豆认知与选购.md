## 2. 咖啡豆知识框架

### 主要品种

| 品种 | 特点 | 产地代表 |
|------|------|----------|
| 阿拉比卡 | 风味丰富 | 埃塞俄比亚、哥伦比亚 |
| 罗布斯塔 | 咖啡因高 | 越南、印尼 |
| 瑰夏 | 花香果酸 | 巴拿马、哥伦比亚 |
| 波旁 | 甜感好 | 巴西、危地马拉 |
| 铁皮卡 | 均衡优雅 | 牙买加、夏威夷 |

### 处理法区别

```
1. 日晒：风味浓郁、甜感好
2. 水洗：干净、酸质明亮
3. 蜜处理：甜感足、口感厚
4. 厌氧处理：风味独特、酒香
```

## 3. 专业建议

### A. 产地特点

#### 产地风味
```
非洲产区（埃塞/肯尼亚）：
- 花香、果酸、莓果
- 典型：耶加雪菲、西达摩

美洲产区（哥伦比亚/巴拿马）：
- 坚果、焦糖、均衡
- 典型：瑰夏、卡杜拉

亚洲产区（印尼/云南）：
- 木质、泥土、草本
- 典型：曼特宁、耶加雪菲
```

### B. 烘焙度选择

#### 烘焙特点
```
浅烘：酸质明亮、花果香
中烘：均衡、甜感好
中深：巧克力、坚果
深烘：焦糖、苦味
```

### C. 选购技巧

#### 选购要点
```
1. 看日期：新鲜最重要（2周内）
2. 看产地：好产地有好豆
3. 看品种：瑰夏>波旁>卡杜拉
4. 看处理法：根据口味选择
5. 看海拔：高海拔更优质
```

### D. 保存方法

#### 保存技巧
```
1. 避光：不要阳光直射
2. 密封：减少氧化
3. 常温：不要放冰箱
4. 单独存放：不要和有味物品放
5. 尽快喝完：1个月内最佳
```
