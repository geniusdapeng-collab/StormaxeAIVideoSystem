## 2. 全球产区框架

### 三大产区

| 产区 | 代表产地 | 风味特点 |
|------|----------|----------|
| 非洲 | 埃塞俄比亚、肯尼亚 | 花香、果酸、莓果 |
| 美洲 | 哥伦比亚、巴拿马 | 坚果、焦糖、均衡 |
| 亚洲 | 印尼、云南 | 木质、草本、厚重 |

### 风土要素

```
1. 海拔：越高越优质（1200m+）
2. 气候：温度、降雨
3. 土壤：矿物质含量
4. 处理法：日晒/水洗/蜜处理
5. 品种：基因多样性
```

## 3. 专业建议

### A. 非洲产区

#### 代表产地
```
埃塞俄比亚：
- 西达摩、耶加雪菲、瑰夏
- 花香、柑橘、茉莉
- 典型：柑橘、茉莉、莓果

肯尼亚：
- Nyeri、Kirinyaga
- 番茄、黑醋栗、明亮酸
- 典型：黑莓、番茄、酸质
```

### B. 美洲产区

#### 代表产地
```
哥伦比亚：
- 慧兰、娜玲峡谷
- 坚果、焦糖、均衡
- 典型：坚果、巧克力、顺滑

巴拿马：
- 翡翠庄园（瑰夏）
- 茉莉、柑橘、伯爵茶
- 典型：花香、茶感、精致
```

### C. 亚洲产区

#### 代表产地
```
印度尼西亚：
- 曼特宁、爪哇
- 木质、草本、泥土
- 典型：草药、黑巧克力

云南：
- 普洱、孟连
- 果酸、坚果、甜感
- 典型：水果、焦糖
```

### D. 庄园级别

#### 等级分类
```
竞标批次（Bid Lot）：
- 拍卖会精选
- 价格最高
- 品质顶级

微批次（Micro Lot）：
- 单一庄园
- 可追溯
- 品质优秀

合作社批次：
- 合作社交豆
- 品质稳定
- 性价比高
```
