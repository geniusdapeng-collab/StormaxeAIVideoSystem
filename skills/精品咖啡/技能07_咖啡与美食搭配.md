## 2. 搭配原则框架

### 搭配类型

| 搭配类型 | 原则 | 示例 |
|----------|------|------|
| 相似相配 | 风味相近 | 果酸+柑橘 |
| 对比相配 | 互补平衡 | 苦+甜 |
| 增味相配 | 共同增强 | 巧克力+坚果 |

### 基本原则

```
1. 平衡原则：不要一方压过另一方
2. 层次原则：先轻后重
3. 顺序原则：清洁口腔再品
4. 量比原则：咖啡为主，食物为辅
5. 个人原则：最终是个人体验
```

## 3. 专业建议

### A. 甜点搭配

#### 经典搭配
```
浅烘+水果甜点：
- 耶加雪菲+柠檬塔
- 肯尼亚+莓果蛋糕
- 效果：果香呼应

中烘+巧克力甜点：
- 哥伦比亚+布朗尼
- 危地马拉+芝士蛋糕
- 效果：苦甜平衡

深烘+焦糖甜点：
- 曼特宁+焦糖布丁
- 印尼+提拉米苏
- 效果：醇厚加倍
```

### B. 餐食搭配


### C. 顺序品鉴

#### 品鉴顺序
```
1. 先品单品：感受原味
2. 再尝食物：认识味道
3. 后品搭配：体验变化
4. 对比单独：感受差异
5. 记录感受：建立偏好
```
