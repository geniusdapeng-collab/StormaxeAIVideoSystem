## 3. 专业建议

### A. 磨豆机选择

#### 选择要点
```
1. 刀盘类型：锥刀 vs 平刀
2. 研磨均匀度：最重要指标
3. 档位调节：是否精准
4. 清理方便：是否好打理
5. 噪音大小：家用要考虑
```

### B. 滤杯选择

#### 滤杯类型
```
V60：
- 流速快，需要技术
- 风味清晰、层次分明

Kalita扇形：
- 流速慢，容易操作
- 口感圆润、甜感好

Origami折纸：
- 兼顾两者优点
- 适应性强
```

### C. 手冲壶选择

#### 选择要点
```
1. 控温功能：是否可调温
2. 壶嘴设计：细嘴还是鹤嘴
3. 容量大小：1L左右最佳
4. 材质选择：不锈钢/铜/珐琅
5. 握持手感：是否顺手
```

### D. 电子秤选择

#### 选择要点
```
1. 精度：0.1g最佳
2. 响应速度：是否快
3. 计时功能：是否带计时
4. 防水等级：使用寿命
5. 显示屏：是否清晰
```
