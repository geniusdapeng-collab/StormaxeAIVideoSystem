## 2. 咖啡发展史框架

### 三大浪潮

| 浪潮 | 时期 | 特点 |
|------|------|------|
| 第一波 | 1940-1960 | 速溶化，便利 |
| 第二波 | 1966-2000 | 意式化，连锁 |
| 第三波 | 2003-至今 | 精品化，单品 |

### 第三波特征

```
1. 产地溯源：知道豆子从哪来
2. 浅烘保留：突出产地风味
3. 手冲流行：慢下来品味
4. 风味描述：专业杯测语言
5. 匠人精神：追求极致
```

## 3. 专业建议

### A. 品鉴礼仪

#### 品鉴场合
```
1. 不加糖奶：品鉴单品原味
2. 安静品尝：专注感受风味
3. 轻声交流：不要大声喧哗
4. 尊重分享：不要独占豆子
5. 感谢冲煮：感谢冲煮者
```

### B. 社交礼仪

#### 咖啡社交
```
1. 点单礼仪：
- 可以询问推荐
- 不要过分挑剔
- 尊重咖啡师专业

2. 品饮礼仪：
- 等待降温再喝
- 不要吹凉
- 小口品尝

3. 交流礼仪：
- 分享感受
- 不懂就问
- 尊重不同喜好
```

### C. 商务场合

#### 商务咖啡
```
1. 约见选择：
- 安静咖啡馆
- 有座位可谈事

2. 点单礼仪：
- 简洁快速
- 不要犹豫太久

3. 交谈礼仪：
- 专注谈话
- 不要一直看手机

4. 付账礼仪：
- 发起方付款
- 不要斤斤计较
```
