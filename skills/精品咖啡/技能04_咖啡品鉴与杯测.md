## 3. 专业建议

### A. 杯测流程

#### 标准流程
```
1. 研磨：粗研磨
2. 闻干香：研磨香
3. 注水：93°C，1:16
4. 浸泡：4分钟
5. 破渣：搅动表面
6. 闻湿香：萃取香
7. 捞渣：去除浮渣
8. 啜吸： aerosol吸入
9. 品鉴：各维度评估
10. 记录：打分+描述
```

### B. 啜吸技巧

#### 啜吸方法
```
1. 用力吸气：形成喷雾
2. 覆盖舌面：全面感受
3. 引入空气：放大风味
4. 感受层次：前中后段
5. 记录余韵：回味变化
```

### C. 风味描述

#### 描述框架
```
1. 基础风味：酸/甜/苦
2. 风味类别：水果/花香/坚果
3. 具体风味：柑橘/草莓/杏仁
4. 联想词汇：像什么
5. 强度描述：浓/淡/适中
```

### D. 感官训练

#### 训练方法
```
1. 咖啡闻香瓶：系统训练
2. 水果品尝：建立记忆
3. 产地对比：了解差异
4. 盲测练习：提高准确
5. 记录习惯：建立数据库
```
