## 2. 手冲参数框架

### 萃取参数

| 参数 | 建议范围 | 影响 |
|------|----------|------|
| 粉量 | 15-20g | 浓度 |
| 水温 | 88-94°C | 萃取率 |
| 研磨 | 中细偏粗 | 萃取速度 |
| 时间 | 2:30-3:30 | 萃取时间 |
| 水量 | 1:12-1:15 | 浓度 |

### 萃取公式

```
TDS = 可溶物重量 / 咖啡液重量
萃取率 = TDS × 咖啡液重量 / 咖啡粉重量
```

## 3. 专业建议

### A. 冲煮流程

#### 经典流程
```
1. 预热：预热器具
2. 称豆：精准定量
3. 研磨：现磨现冲
4. 温杯：保持温度
5. 洗滤纸：去除纸味
6. 投粉：倒入粉碗
7. 闷蒸：30秒排气
8. 注水：分阶段
9. 萃取：等待滴滤
10. 享用：好好品尝
```

### B. 注水技巧

#### 注水方法
```
1. 中心注入：从中心开始
2. 螺旋轨迹：向外画圈
3. 水流稳定：不要忽大忽小
4. 分段注水：分2-3次
5. 避免触碰：不要碰到滤纸
```

### C. 常见问题

#### 问题调整
```
过萃（苦/涩）：
- 降低水温
- 调粗研磨
- 缩短时间

萃取不足（酸/淡）：
- 提高水温
- 调细研磨
- 延长时间
```

### D. 风味调整

#### 调整方向
```
想突出酸：
- 浅烘、高水温、快冲

想突出甜：
- 中烘、中水温、慢冲

想突出苦：
- 深烘、低水温、慢冲

想平衡：
- 均衡参数、中庸之道
```
