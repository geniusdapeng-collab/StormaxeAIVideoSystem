## 2. 意式参数框架

### 萃取参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 粉量 | 18-20g | 双份浓缩 |
| 液重 | 36-40g | 1:2比例 |
| 水温 | 90-94°C | 意式专用 |
| 压力 | 9bar | 标准萃取 |
| 时间 | 25-30秒 | 萃取时间 |

### 浓缩特征

```
好浓缩标准：
- crema：红棕色，厚实
- 香气：浓郁，持久
- 口感：醇厚，平衡
- 余韵：绵长，甜感
```

## 3. 专业建议

### A. 萃取要点

#### 萃取步骤
```
1. 称豆：精准定量
2. 研磨：现磨最佳
3. 布粉：均匀分布
4. 压粉：水平压实
5. 萃取：观察流速
6. 截流：适时停止
```

### B. 常见问题

#### 问题调整
```
流速太快（萃取不足）：
- 调细研磨
- 增加压粉力度
- 延长萃取时间

流速太慢（萃取过度）：
- 调粗研磨
- 减少压粉力度
- 缩短萃取时间
```

### C. 牛奶打发

#### 打发技巧
```
1. 蒸汽角度：15度斜角
2. 进气阶段：大气泡
3. 打发阶段：小气泡
4. 光泽阶段：丝绸感
5. 温度控制：55-65°C
```

### D. 拉花基础

#### 拉花方法
```
1. 融合：奶咖充分混合
2. 注入：中心高位
3. 摆动：形成图案
4. 收尾：干净利落
5. 练习：多练是关键

入门图案：
- 心形
- 叶子
- 郁金香
```
