### 评价维度

```
1. 咖啡品质：最重要

3. 服务态度：体验感
4. 性价比：值不值
5. 特色亮点：记忆点
```

## 3. 专业建议

### A. 探店准备

#### 信息收集
```
1. 线上筛选：
- 小红书/B站看评价
- 大众点评看评分
- 咖啡博主推荐

2. 初步判断：
- 豆子来源：是否标注
- 烘焙商：自有还是代烘
- 器具选择：专业程度

3. 目标明确：
- 想喝什么类型
- 环境要求
- 预算范围
```

### B. 实地体验

#### 体验要点
```
1. 第一印象：
- 门头装修
- 清洁程度
- 氛围感受

2. 核心体验：
- 点一杯招牌
- 观察萃取过程
- 感受出品质量

3. 细节观察：
- 器具选择
- 环境卫生
- 服务态度
```

### C. 多维评价

#### 评价框架
```
1. 咖啡维度（50%）：
- 豆子新鲜度
- 萃取稳定性
- 风味表现
- 与描述符合度

2. 环境维度（30%）：
- 空间舒适度
- 音乐氛围
- 光线环境
- 座位数量

3. 服务维度（20%）：
- 专业程度
- 热情态度
- 推荐能力
```

### D. 探店记录

#### 记录方法
```
1. 照片记录：
- 门面照
- 环境照
- 咖啡照
- 特写照

2. 笔记记录：
- 咖啡名称
- 价格
- 风味感受
- 综合评价

3. 收藏分享：
- 建立收藏夹
- 星级评分
- 推荐理由
```
