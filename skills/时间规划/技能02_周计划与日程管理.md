周计划核心：目标聚焦→时间分块→预留弹性→每日复盘，四步构建高效周计划

## 2. 周计划框架

### 周时间结构

| 时段 | 时间 | 用途 |
|------|------|------|
| 周一 | 目标启动 | 计划+启动 |
| 周二-周四 | 深度工作 | 高价值任务 |
| 周五 | 收尾+复盘 | 总结+下周规划 |
| 周末 | 休息+充电 | 家庭+自我 |

### 时间分块原则
```
1. 核心工作时间：3-4小时（高价值任务）
2. 协作时间：2小时（会议+沟通）
3. 琐碎时间：1小时（邮件+处理）
4. 学习时间：1小时（成长+提升）
```

## 3. 专业建议

### A. 周一规划

#### 规划步骤
```
1. 回顾上周完成情况
2. 明确本周核心目标（≤3个）
3. 分配到具体日程
4. 预留应急时间
```

#### 周一例会结构
```
1. 上周回顾（10分钟）
2. 本周目标确认（10分钟）
3. 任务分配（15分钟）
4. 风险识别（5分钟）
```

### B. 每日时间块

#### 典型时间块
```
上午（9-12点）：
- 9:00-9:30 处理邮件/消息
- 9:30-12:00 深度工作（高价值）

下午（14-18点）：
- 14:00-15:30 会议/协作
- 15:30-17:30 处理琐事
- 17:30-18:00 明日计划
```

### C. 应对变化

#### 临时任务处理
```
1. 记录但不立即响应
2. 评估紧急程度
3. 调整计划或延后
4. 保护核心时间
```

### D. 周五复盘

#### 复盘清单
```
1. 本周目标完成情况
2. 有效时间利用率
3. 关键成就
4. 需要改进的点
5. 下周优先事项
```
