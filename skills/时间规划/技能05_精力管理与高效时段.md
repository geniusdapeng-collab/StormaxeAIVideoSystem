精力管理核心：了解周期→匹配任务→主动恢复→保持节奏，四步实现持续高效

## 1. 场景分析

### 请求摘要
用户需要管理精力，在高效时段做重要的事

### 显性需求
- 掌握精力管理方法
- 了解自己的高效时段

### 场景分类

## 2. 精力管理框架

### 能量波动规律

```
典型工作日能量曲线：
06:00-08:00  起床适应（低）
08:00-10:00  第一个高峰
10:00-12:00  持续高效
12:00-14:00  午间低谷
14:00-16:00  下午恢复
16:00-18:00  第二个高峰
18:00-20:00  逐渐下降
20:00-22:00  适合创作
```

### 四维精力

| 维度 | 含义 | 恢复方式 |
|------|------|----------|
| 身体 | 体能 | 运动+睡眠 |
| 情绪 | 情感 | 社交+爱好 |
| 思维 | 认知 | 学习+思考 |
| 意志 | 动机 | 意义+价值 |

## 3. 专业建议

### A. 识别高效时段

#### 自我观察
```
1. 连续一周记录精力状态
2. 标记高效和低谷时段
3. 分析规律
4. 匹配任务
```

#### 常见类型
```
晨型人：6-12点高效
夜型人：10-24点高效
午后型：14-18点高效
```

### B. 任务匹配

#### 高效时段安排
```
高认知：深度工作、决策、创意
低精力：处理邮件、简单沟通
```

#### 能量低谷应对
```
1. 短暂休息（15分钟）
2. 轻度活动（走动）
3. 切换任务类型
4. 补充能量
```

### C. 精力恢复

#### 快速恢复
```
1. 90秒深呼吸
2. 户外走动5分钟
3. 听音乐10分钟
4. 小憩20分钟
```

#### 长期维护
```
1. 充足睡眠（7-8小时）
2. 规律运动
3. 健康饮食
4. 定期休假
```

### D. 保持节奏

#### 日节律
```
1. 固定起床时间
2. 固定高效时段
3. 固定休息时间
4. 避免频繁切换
```
