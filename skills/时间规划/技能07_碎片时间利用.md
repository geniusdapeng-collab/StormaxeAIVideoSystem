## 2. 碎片时间框架

### 常见碎片时间

| 场景 | 时长 | 状态 |
|------|------|------|
| 通勤 | 30-60分钟 | 移动 |
| 等人 | 5-30分钟 | 停留 |
| 排队 | 5-15分钟 | 停留 |
| 饭后 | 15-30分钟 | 休息 |
| 睡前 | 15-30分钟 | 放松 |

### 任务匹配

| 时长 | 适合任务 |
|------|----------|
| 5分钟 | 回消息、简单整理 |
| 15分钟 | 阅读、听书、学习 |
| 30分钟 | 深度学习、思考 |
| 60分钟 | 专注工作 |

## 3. 专业建议

### A. 场景化任务库

#### 通勤时间
```
地铁/公交（可看/可听）：
- 听播客/课程
- 阅读电子书
- 思考问题
- 规划日程

开车（仅可听）：
- 听音频课程
- 听新闻
- 听有声书
```

#### 等待时间
```
等人/排队：
- 处理消息
- 浏览资讯
- 快速学习
- 整理笔记
```

### B. 准备机制

#### 素材准备
```
1. 建立学习清单
2. 缓存离线内容
3. 准备待办事项
4. 设置提醒
```

#### 工具准备
```
- 稍后阅读：Pocket/Instapaper
- 笔记工具：印象笔记
- 音频：喜马拉雅/得到
- 阅读：微信读书
```

### C. 习惯养成

#### 微习惯策略
```
1. 从5分钟开始
2. 固定场景固定任务
3. 不追求完美
4. 记录完成情况
```
