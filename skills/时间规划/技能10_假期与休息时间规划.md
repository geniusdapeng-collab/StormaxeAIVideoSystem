## 2. 休息类型框架

### 休息分类

| 类型 | 时长 | 特点 |
|------|------|------|
| 微休息 | 5-15分钟 | 随时进行 |
| 日常休息 | 1-2小时 | 每天应有 |
| 周末充电 | 1-2天 | 深度恢复 |
| 长假规划 | 3-7天 | 系统修复 |

### 休息质量

```
真正的休息：
- 身心放松
- 精力恢复
- 心情愉悦
- 创造回忆

假休息：
- 焦虑空虚
- 比工作还累
- 无收获感
```

## 3. 专业建议

### A. 日常休息

#### 微休息方法
```
1. 90分钟周期：工作90分钟，休息15分钟
2. 眼睛休息：每30分钟看远处
3. 身体活动：每小时站起来走动
4. 快速恢复：深呼吸、伸展
```

#### 午间休息
```
1. 饭后散步15分钟
2. 短暂午睡（20分钟）
3. 避免：长时间刷手机
```

### B. 周末充电

#### 周末原则
```
1. 至少半天完全放空
2. 接触大自然
3. 社交/独居平衡
4. 避免：工作相关
```

#### 周末活动
```
选项1：社交恢复
- 和朋友聚会
- 家庭时间

选项2：独处恢复
- 阅读
- 创作
- 思考

选项3：自然恢复
- 户外运动
- 公园散步
```

### C. 长假规划

#### 假期结构
```
假期前半段：彻底放空
假期中间：安排活动
假期后半段：收心准备
```

#### 避免情况
```
× 特种兵式旅游
× 假期报复性加班
× 安排太满
× 结束前焦虑
```

### D. 休息仪式

#### 建立习惯
```
1. 固定休息时间
2. 创造休息氛围
3. 远离工作相关
4. 记录美好时刻
```
