## 2. 会议时间框架

### 会议类型时长

| 类型 | 时长 | 人数 |
|------|------|------|
| 站立会 | 15分钟 | 团队 |
| 快速同步 | 30分钟 | 2-5人 |
| 决策会 | 45分钟 | 3-7人 |
| 研讨会 | 60分钟 | 5-15人 |
| 复盘会 | 90分钟 | 团队 |

### 会议成本

```
会议成本 = 时长 × 人数 × 平均时薪

示例：
10人 × 1小时 × 500元 = 5000元/小时
```

## 3. 专业建议

### A. 会前准备

#### 必须准备
```
1. 明确会议目的
2. 准备议程（发给大家）
3. 提前阅读材料
4. 确认参会人
5. 预约合适时长
```

#### 议程模板
```
议程：
1. [议题1] 10分钟 负责人A
2. [议题2] 10分钟 负责人B
3. [议题3] 5分钟 总结
```

### B. 时间控制

#### 主持技巧
```
1. 提前声明时间
2. 严格执行议程
3. 适时打断发散
4. 时间提醒
5. 果断收尾
```

#### 时间提醒
```
- 还剩5分钟
- 需要加快进度
- 这个问题我们会后讨论
```

### C. 会议角色

#### 必要角色
```
主持人：控场+推进
记录人：记录+跟进
时间官：提醒时间
```

### D. 会后跟进

#### 必须做到
```
1. 24小时内发出纪要
2. 明确Action Item
3. 指定负责人和deadline
4. 跟进执行情况
```
