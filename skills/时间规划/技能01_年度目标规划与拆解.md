## TL;DR

目标规划核心：愿景→方向→目标→拆解→执行，五步构建完整目标体系

## 1. 场景分析

### 请求摘要
用户需要制定年度目标并拆解为可执行计划

### 显性需求
- 掌握目标制定方法
- 获得可执行的规划方案

### 场景分类
- `scenario_classification`: 战略规划
- `urgency`: normal

## 2. 目标规划框架

### 目标金字塔

```
愿景（3-5年）
    ↑
方向（1-3年）
    ↑
年度目标（O）
    ↑
关键结果（KR）
    ↑
季度/月度任务
```

### OKR模板

```
O（目标）：本年度核心目标
KR1：关键结果1（可量化）
KR2：关键结果2（可量化）
KR3：关键结果3（可量化）
```

## 3. 专业建议

### A. 年度目标制定

#### 目标来源
```
1. 人生愿景倒推
2. 角色责任驱动
3. 痛点解决导向
4. 机会把握牵引
```

#### SMART原则
```
S（Specific）：具体明确
M（Measurable）：可衡量
A（Achievable）：可达成
R（Relevant）：相关性
T（Time-bound）：有时限
```

### B. 目标拆解

#### 拆解逻辑
```
年度目标
    ↓ 拆解
季度里程碑
    ↓ 拆解
月度任务
    ↓ 拆解
周度行动
    ↓ 拆解
每日TODO
```

#### 量化指标
```
年度目标：收入100万
季度：25万/季
月均：8.3万/月
周均：2万/周
```

### C. 执行追踪

#### 复盘机制
```
1. 每日：TODO检查
2. 每周：周复盘（45分钟）
3. 每月：月复盘（2小时）
4. 每季：季复盘（半天）
```

#### 调整原则
```
- 外部环境变化：及时调整
- 目标达成困难：分析原因
- 方向错误：果断修正
```
