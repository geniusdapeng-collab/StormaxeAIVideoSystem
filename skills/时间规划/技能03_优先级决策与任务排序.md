## 2. 优先级框架

### 艾森豪威尔矩阵

| | 紧急 | 不紧急 |
|---|------|--------|
| **重要** | Ⅰ立即做 | Ⅱ计划做 |
| **不重要** | Ⅲ授权做 | Ⅳ删除 |

### 优先级排序

```
P0：必须做（不干会死）
P1：重要做（影响目标）
P2：应该做（提升效率）
P3：可以做（有空再做）
P4：不做（删除/推迟）
```

## 3. 专业建议

### A. 价值评估

#### 评估维度
```
1. 目标贡献度：离目标多远
2. 影响范围：影响多少人
3. 不可替代性：只有你能做
4. 复利效应：长期收益
```

#### 价值公式
```
价值 = 产出 × 影响面 × 时效性
```

### B. 紧急程度

#### 判断标准
```
1. 有deadline：即将到期
2. 依赖方等待：别人等着你
3. 连锁影响：影响后续任务
4. 机会窗口：错过不再有
```

### C. 决策流程

#### 每日优先级
```
1. 列出所有任务
2. 评估价值和紧急度
3. 按四象限分类
4. 确定执行顺序
5. 保护P0时间
```

### D. 常见误区

#### 避免做法
```
× 按时间顺序做
× 谁催先做谁
× 一直做简单的
× 追求100%完成
```
