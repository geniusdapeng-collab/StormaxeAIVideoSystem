## 2. 脚本结构框架

### 短视频黄金结构

```
0-3秒：黄金开头（吸引注意力）
3-15秒：价值输出（提供干货/情绪）
15-45秒：情绪共鸣（引发互动）
45-60秒：行动引导（点赞关注）
```

### 时长分配

| 时长 | 开头 | 核心 | 结尾 |
|------|------|------|------|
| 15秒 | 2秒 | 10秒 | 3秒 |
| 30秒 | 3秒 | 22秒 | 5秒 |
| 60秒 | 5秒 | 45秒 | 10秒 |

## 3. 专业建议

### A. 黄金3秒设计

#### 开头方法
```
1. 提问式：你知道...吗？
2. 反差式：居然可以...
3. 痛点式：你是不是也...
4. 悬念式：最后一个...
5. 干货式：3个技巧...
```

### B. 脚本模板

#### 口播脚本模板
```
【开场】（3秒）
[hook句式]

【正文】（价值内容）
1. 第一个点...
2. 第二个点...
3. 第三个点...

【结尾】（行动引导）
[总结+引导]
```

### C. 分镜设计

#### 分镜要素
```
1. 景别：远/全/中/近/特
2. 运镜：推/拉/摇/移/跟
3. 时长：每个镜头几秒
4. 内容：具体画面描述
5. 音效：配乐/音效
```

### D. 台词技巧

#### 台词原则
```
1. 口语化：听起来自然
2. 节奏感：长短句交错
3. 情绪感：有感染力
4. 记忆点：金句重复
```
