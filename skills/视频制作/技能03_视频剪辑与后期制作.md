## 2. 剪辑流程框架

### 标准流程

| 步骤 | 内容 | 时长占比 |
|------|------|----------|
| 1. 整理素材 | 分类/标记 | 20% |
| 2. 粗剪 | 搭建结构 | 15% |
| 3. 精剪 | 节奏调整 | 30% |
| 4. 调色 | 色彩校正 | 15% |
| 5. 配乐 | 音乐/音效 | 10% |
| 6. 输出 | 导出成片 | 10% |

## 3. 专业建议

### A. 节奏剪辑

#### 剪辑点选择
```
1. 动作连贯：动作中间切断
2. 情绪匹配：情绪高潮点
3. 音乐节拍：卡点剪辑
4. 节奏感：长短镜头交替
```

### B. 转场技巧

#### 常用转场
```
1. 硬切：直接切换，最常用
2. 叠化：渐变过渡
3. 缩放：画面缩放转场
4. 划入：滑动效果
5. 黑场：黑色过渡
```

### C. 调色基础

#### 基础调色
```
1. 曝光：调整整体亮度
2. 对比：增强画面层次
3. 饱和度：调整颜色浓度
4. 色温：冷暖色调
5. 风格化：统一色调
```

### D. 配乐选择

#### 音乐匹配
```
1. 节奏匹配：音乐节奏与画面匹配
2. 情绪匹配：音乐情绪与内容一致
3. 版权注意：使用正版音乐
4. 音量平衡：音乐不盖过人声
```
