## 2. 直播流程框架

### 直播结构

| 阶段 | 时长 | 目标 |
|------|------|------|
| 预热 | 30分钟 | 蓄水 |
| 开场 | 5分钟 | 留人 |
| 主体 | 50分钟 | 价值输出 |
| 转化 | 30分钟 | 成交 |
| 结尾 | 5分钟 | 引导关注 |

## 3. 专业建议

### A. 开播准备

#### 准备清单
```
1. 设备检查：摄像头/麦克风/灯光
2. 网络测试：流畅不卡顿
3. 场景布置：整洁有氛围
4. 物料准备：产品/话术/优惠
5. 人员分工：主播/场控/助理
```

### B. 开场留人

#### 留人话术
```
1. 欢迎互动：欢迎XX来到直播间
2. 价值预告：今天主要讲...
3. 福利预告：整点有福利
4. 互动提问：大家有什么问题
```

### C. 互动技巧

#### 互动方法
```
1. 实时互动：回答弹幕问题
2. 红包福袋：整点抽奖
3. 话题讨论：引发讨论
4. 连麦互动：增加参与感
```

### D. 促单话术

#### 转化技巧
```
1. 痛点挖掘：引发共鸣
2. 价值塑造：讲解卖点
3. 价格锚点：原价vs现价
4. 限时限量：制造紧迫
5. 行动引导：赶紧下单
```
