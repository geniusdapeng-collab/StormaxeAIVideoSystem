## 2. 数据指标框架

### 核心指标

| 指标 | 含义 | 优秀值 |
|------|------|--------|
| 完播率 | 看完全程比例 | >30% |
| 5秒完播 | 前5秒留存 | >50% |
| 点赞率 | 点赞/播放 | >3% |
| 评论率 | 评论/播放 | >0.5% |
| 转发率 | 转发/播放 | >0.3% |
| 关注转化 | 关注/播放 | >1% |

### 数据漏斗

```
曝光 → 播放 → 完播 → 互动 → 关注
  ↓      ↓      ↓      ↓      ↓
曝光量 播放率 完播率 互动率 涨粉率
```

## 3. 专业建议

### A. 问题诊断

#### 常见问题

| 问题 | 指标表现 | 优化方向 |
|------|----------|----------|
| 开头差 | 5秒完播低 | 优化黄金3秒 |
| 内容水 | 完播率低 | 精简内容 |
| 没共鸣 | 互动率低 | 增强情绪 |
| 不吸引 | 点赞率低 | 优化价值 |
| 不转发 | 转发率低 | 增加社交货币 |

### B. 数据分析方法

#### 对比分析
```
1. 横向对比：与同类型账号对比
2. 纵向对比：与自己历史对比
3. 平台对比：不同平台数据
```

### C. 优化迭代

#### 迭代方法
```
1. 找到问题视频
2. 分析数据指标
3. 定位具体问题
4. 制定优化方案
5. 测试验证效果
```
