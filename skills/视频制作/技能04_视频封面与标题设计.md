## 3. 专业建议

### A. 标题设计

#### 爆款标题公式
```
1. 数字+关键词：5个技巧...
2. 提问式：为什么...
3. 反差式：居然可以...
4. 痛点式：不会...怎么办
5. 干货式：...必看
```

#### 标题技巧
```
1. 关键词前置
2. 使用标点符号
3. 控制字数（15-25字）
4. 引发好奇
5. 制造紧迫感
```

### B. 封面设计

#### 视觉原则
```
1. 对比强烈：吸引注意力
2. 主体突出：人物/产品够大
3. 文字清晰：一句话说明
4. 色彩鲜明：吸引眼球
```

#### 常用模板
```
1. 人物+文字：真人出镜+标题
2. 产品+文字：产品展示+卖点
3. 场景+文字：场景+主题
4. 表情包+文字：吸睛+说明
```

### C. 平台适配

#### 抖音封面
```
- 竖屏为主
- 文字要大
- 人物居中
- 色彩鲜明
```

#### B站封面
```
- 横屏为主
- 可以更精致
- 信息量更大
- 适合二次元风格
```
