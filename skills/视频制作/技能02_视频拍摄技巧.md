## 3. 专业建议

### A. 基础构图

#### 常用构图
```
1. 三分法：画面分成九宫格
2. 中心构图：主体在画面中央
3. 对称构图：左右/上下对称
4. 引导线：利用线条引导视线
5. 框架构图：利用门窗等框架
```

### B. 运镜方法

#### 基础运镜
```
1. 推：靠近主体，突出细节
2. 拉：远离主体，展示环境
3. 摇：左右/上下旋转
4. 移：跟随主体移动
5. 跟：保持距离跟随
6. 升/降：垂直运动
```

### C. 光线运用

#### 自然光
```
1. 顺光：光线从前面照亮
2. 侧光：侧面光照，有立体感
3. 逆光：勾勒轮廓，氛围感
4. 顶光：正上方光照
```

### D. 稳定技巧

#### 防抖方法
```
1. 手持：双手握持，肘部夹紧
2. 稳定器：使用云台稳定器
3. 靠靠：依靠固定物体
4. 后期防抖：软件处理
```
