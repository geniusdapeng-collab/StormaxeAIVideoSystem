# 山海经40异兽统一部位表架构 v1.0-Peng
## 基于比较解剖学的标准化身体部位矩阵

---

## 一、架构设计原则

### 1.1 比较解剖学基础
参考真实动物界五大类群的解剖结构：
- **哺乳纲**（Mammalia）：四肢、毛发、外耳、齿系
- **鸟纲**（Aves）：双翼、羽毛、喙、气囊系统
- **爬行纲**（Reptilia）：鳞片、无足/四足、变温
- **两栖纲**（Amphibia）：皮肤呼吸、水陆双栖
- **昆虫纲**（Insecta）：六足、外骨骼、复眼、触角

### 1.2 山海经异兽的特殊性
异兽远超真实动物分类，存在：
- **部位数量变异**：六足（混沌）、九尾（九尾狐）、无足（烛龙）
- **部位位置变异**：腋下生眼（饕餮）、腹生口（刑天类）
- **复合形态**：人面蛇身（烛龙）、鸟身龙首（应龙）
- **超自然器官**：独角发光（白泽）、火精（烛龙）、量子计算天线（Nirath设定）

### 1.3 统一策略
- **基干部位**（所有异兽共有）：采用真实动物解剖学标准命名
- **变异部位**（部分异兽特有）：采用"部位+修饰符"扩展命名
- **超自然字段**（aura/system）：独立维度，不纳入物理部位表

## 三、扩展字段（跨部位属性）

### 3.1 解剖学数量字段
| 字段 | 说明 | 示例 |
|------|------|------|
| `limb_count` | 肢体总数（含翼） | 混沌=6（六足）、烛龙=0（无足） |
| `tail_count` | 尾巴数量 | 九尾狐=9、应龙/凤凰=1 |
| `wing_count` | 翅膀对数 | 混沌=2（四翼）、凤凰=1（双翼） |
| `head_count` | 头部数量 | 相柳=9、正常=1 |
| `eye_count` | 眼睛数量 | 饕餮=4（腋下+正常？） |

### 3.2 形态分类字段
| 字段 | 说明 | 枚举值 |
|------|------|--------|
| `body_shape` | 整体体型 | serpentine蛇形 / draconic龙形 / therian兽形 / avian鸟形 / amorphous无定形 / centaurine半人半兽 |
| `head_type` | 头部类型 | humanoid人面 / draconic龙首 / lupine狼首 / vulpine狐首 / avian鸟首 / ophidian蛇首 / bovine牛首 / none无头 |
| `coat_type` | 被覆类型 | fur毛发 / feathers羽毛 / scales鳞片 / shell壳 / skin皮肤 / chitin外骨骼 |
| `limb_type` | 肢体类型 | digitigrade趾行 / plantigrade跖行 / ungulate蹄行 / none无足 / wings翼 |

### 3.3 超自然Aura字段（独立维度，非物理部位）
| 字段 | 说明 |
|------|------|
| `aura_presence` | 威严气场：周围空气扭曲/凡人不敢直视 |
| `aura_wisdom` | 智慧之眼：看穿一切/知识流动 |
| `aura_divine` | 神圣光芒：强烈圣光/邪祟退散 |
| `aura_power` | 通天之力：连接天地/地面纹路蔓延 |
| `aura_malevolent` | 邪煞气场：恐惧/腐败/死亡气息 |

## 五、部位表数据填充规范

### 5.1 每个部位的5属性
```json
{
  "has": true/false,
  "material": "材质结构（解剖学描述）",
  "texture": "纹理细节（微观结构）",
  "color": "颜色系统",
  "light": "发光/光影特性",
  "detail": "关键识别特征（必须视觉化）"
}
```

### 5.2 材质描述分级
| 级别 | 描述精度 | 示例 |
|------|---------|------|
| L1 形态级 | 大类形态 | "羊身"、"蛇身" |
| L2 结构级 | 肌肉/骨骼 | "肌肉饱满有力，脊背隆起如山脊" |
| L3 组织级 | 毛发/皮肤 | "毛发分三层：外层白色蓬松长毛（20-30cm）" |
| L4 细胞级 | 微观质感 | "每根毛发独立物理模拟，次表面散射" |
| L5 分子级 | 光学特性 | "半透明象牙白，内部金色光芒如熔岩流动" |

**CG电影级要求：L3-L5**

### 5.3 颜色系统规范
采用科学色名+艺术色名双标注：
- 科学：RGB/HSV值或Pantone近似
- 艺术："象牙白"、"琥珀金"、"赤红如熔岩"、"青丘灵原蓝绿色"

## 八、架构优势

| 维度 | 传统文本features | 统一部位表v1.0 |
|------|-----------------|---------------|
| **系统性** | 随意堆砌 | 16个固定部位，无遗漏 |

| **视觉化** | 抽象难渲染 | 每个字段都对应视觉元素 |
| **电影级** | 依赖写手水平 | L3-L5材质标准自动达标 |
