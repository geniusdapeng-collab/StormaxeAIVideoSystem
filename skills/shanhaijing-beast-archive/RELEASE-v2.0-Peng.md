## 📐 档案标准（v2.0-Peng）

### 15维度完整版

每个档案包含以下15个维度：

1. **神兽身份信息**：名称、拼音、别名、出处、神话地位
2. **综合介绍**：核心特征、文化意义、一句话定位
3. **外观描述**：整体形态、头部、身体、四肢/翅膀/尾巴、动态特征（500+字）
4. **神通能力（5项）**：每项含视觉表现 + 科幻原理
5. **传说故事（3则）**：每则含视觉高潮点
6. **象征寓意**：文化象征 + 现代启示
7. **影视创作建议**：视觉呈现 + 特效设计 + 角色定位 + 参考风格
8. **Nirath星球融合设定**：栖息地 + 外星生态适应 + 与人类关系 + 科幻故事线
9. **相关神兽网络**：关系图谱
10. **档案元数据**：版本、编号、创建时间
11. **Prompt工程套件**：分层Prompt + 490字精编Prompt + 否定Prompt + 风格锚点 + 角色一致性锚点
12. **小G交互档案**：友好度 + 首次相遇剧情 + 任务触发 + 羁绊 + 奖励
13. **生产就绪参数**：渲染复杂度 + API预估 + 画幅/镜头/时长 + 灯光 + 相机运动
14. **生态位档案**：食物链位置 + 能量来源 + 共生关系 + 领地范围 + 生态影响
15. **版本与审计日志**：版本信息 + 审计日志 + 质量评分

### 质量标准

- **质量评分**：50/50（满分标准）
- **Prompt字数**：490字精编Prompt
- **外观描述**：700+字
- **科幻原理**：每项能力含具体科学术语
- **Nirath融合**：栖息地具体参数 + 3条科幻故事线
