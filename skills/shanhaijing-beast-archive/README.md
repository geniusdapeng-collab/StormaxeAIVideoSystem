## 档案结构说明

每只神兽的档案均包含以下 **10个标准维度**：

| 序号 | 字段名称 | 内容说明 |
|------|----------|----------|
| ① | **神兽名称** | 中文名称、拼音注音、别名别称 |
| ② | **山海经出处** | 具体篇章、原文记载、现代译文 |
| ③ | **神兽介绍** | 综合概述（150-250字），包含基本定位与神话地位 |
| ④ | **神兽外观** | 详细外形描述（200-350字），含体型、颜色、特征部位 |
| ⑤ | **神兽能力/神通** | 特殊能力和效果，分点列出 |
| ⑥ | **神兽故事** | 传说典故（300-400字），生动叙述 |
| ⑦ | **象征寓意** | 中国文化象征意义及对现代人的启示 |
| ⑧ | **影视创作建议** | 视觉呈现、特效设计、角色定位、参考风格等 |
| ⑨ | **与Nirath星球结合建议** | 科幻世界观下的栖息地、生态、故事线、科学解释 |
| ⑩ | **相关神兽** | 与之有关联的其他神兽（2-4个） |
