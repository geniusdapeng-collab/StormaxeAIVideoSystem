## 3. 专业建议

### A. 实验设计六步法
1. **明确问题**
   - 当前痛点是什么
   - 期望提升什么指标

2. **提出假设**
   - 如果...（变量变化）
   - 那么...（核心指标提升）
   - 因为...（业务理由）

3. **设计实验**
   - 实验组vs对照组
   - 样本量计算
   - 实验周期
   - 核心指标+辅助指标

4. **执行监控**
   - 样本分布检查
   - 异常值监控
   - 数据质量验证

5. **结果分析**
   - 统计显著性检验
   - 效果量计算
   - 细分维度分析

6. **决策迭代**
   - 全量/否定/迭代

### B. 常见实验类型
| 类型 | 适用场景 | 周期 |
|------|----------|------|
| UI实验 | 页面改版、按钮优化 | 1-2周 |
| 流程实验 | 转化路径优化 | 2-4周 |
| 策略实验 | 补贴、激励策略 | 2-4周 |
| 算法实验 | 推荐、匹配优化 | 4-8周 |

### C. 样本量计算
```python
# 简化公式
n = 16 * σ² / δ²
# σ: 标准差
# δ: 期望提升量
```
常用参考：
- 转化率提升10%: 每组约10000样本
- 留存率提升5%: 每组约5000样本

### D. 结果判断标准
| 指标 | 判断标准 |
|------|----------|
| 统计显著性 | p < 0.05 |
| 效果量 | 提升>5% |
| 置信区间 | 不包含0 |
