## TL;DR

激活 = 识别关键行为 → 设计引导 → 优化体验 → 持续追踪

## 1. 激活核心概念

### 什么是Aha Moment
用户第一次感受到产品价值的时刻，是从"试用"到"使用"的关键转折点。

### Aha Moment三要素
| 要素 | 说明 |
|------|------|
| 行为 | 用户做了什么 |
| 时间 | 多长时间内 | 
  | 感受 | 用户感受到什么价值 |

### 经典案例
- Facebook: 7天内加10个好友
- Dropbox: 上传第一个文件
- Slack: 发送第一条消息

## 2. 激活分析步骤

### 步骤一：定义激活行为
1. **筛选候选行为**
   - 与核心价值相关
   - 可量化、可追踪
   - 大概率带来留存

2. **验证行为价值**
   - 做过的用户vs没做过的用户
   - 留存率差异显著吗？
   - 差异>2倍通常有意义

### 步骤二：找到最优路径
- 用户从注册到完成关键行为
- 路径上有哪些流失点
- 哪些步骤可以优化

### 步骤三：设计激活引导
```markdown
## 激活引导设计

### 关键行为
- 行为：[具体行为]
- 目标：[X天内完成]

### 引导策略
1. 降低门槛
2. 及时反馈
3. 价值展示
4. 社会证明

### 优化方向
- 减少步骤
- 减少输入
- 增加动力
```

## 3. 激活策略库

### 新手引导优化
| 策略 | 适用场景 | 效果 |
|------|----------|------|
| 跳过引导 | 高确定性用户 | 减少流失 |
| 强制引导 | 低确定性用户 | 确保认知 |
| 渐进引导 | 复杂产品 | 降低认知 |
| 游戏化引导 | 年轻用户 | 增加兴趣 |

### 即时价值展示
- 首单优惠
- 个性化推荐
- 数据亮点
- 成就解锁

### 行为触发
- 行为邮件
- App推送
- 短信召回
- 弹窗提示

## 4. 激活指标体系

### 核心指标
- 激活率 = 完成关键行为用户 / 新用户
- 激活时间 = 注册到激活的平均时间
- 激活成本 = 激活花费 / 激活用户
