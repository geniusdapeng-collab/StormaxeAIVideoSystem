## 2. 创意方法

### 2.1 创意框架
```
创意生成框架：

SCAMPER法：
S - Substitute：替代
C - Combine：组合
A - Adapt：适配
M - Modify：修改
P - Put to other uses：改变用途
E - Eliminate：消除
R - Reverse：反转

创意提示词生成：
1. 选择一个基础概念
2. 应用SCAMPER变换
3. 生成新描述
4. 评估可行性

创意组合公式：
"[A元素] + [B元素] + [C元素]"

示例：
"赛博朋克 + 浮世绘 + 猫"
→ "cat in ukiyo-e style cyberpunk world"

主题发散法：
从主题出发，向外发散

主题：咖啡
├── 咖啡豆 → 森林、土地
├── 咖啡馆 → 复古、未来、温暖
├── 咖啡香气 → 可见的香气、魔法
├── 咖啡师 → 不同职业、不同世界
└── 喝咖啡的人 → 各种职业、各种情绪
```

### 2.2 创意模板
```
创意模板：

模板1：反差创意
"将[正常场景]置于[不可能场景]中"

示例：
"咖啡师在太空站制作咖啡"
"中世纪骑士使用智能手机"
"猫统治的现代社会"

模板2：融合创意
"[元素A] + [元素B] = 新场景"

示例：
"维多利亚时代 + 赛博朋克 = 蒸汽朋克"
"宫崎骏 + 皮克斯 = 温柔3D"
"浮世绘 + 现代东京 = 新派和风"

模板3：变形创意
"[正常物体]被赋予[非常规属性]"

示例：
"会说话的植物在城市中漫步"
"时间被具象化为流动的液体"
"情绪被可视化为云彩和天气"

模板4：视角创意
"[换一个视角/尺度/角度]看世界"

示例：
"从蚂蚁的视角看花园"
"从黑洞内部看宇宙"
"从未来的视角看21世纪"

模板5：故事创意
"[角色]在[场景]中[做某事]"

示例：
"一只旅行的狐狸在东京街头"
"时间旅行者被困在博物馆"
"失散的机器人朋友重逢"
```
