## 1. ControlNet控制

### 1.1 ControlNet基础
```
ControlNet简介：
ControlNet是一种神经网络结构，通过添加额外的条件来控制AI生成图像。

核心功能：
├── 姿态控制：通过骨骼图控制人物姿势
├── 边缘控制：通过线稿控制轮廓
├── 深度控制：通过深度图控制空间
├── 法线控制：通过法线图控制立体
├── 语义分割：通过分割图控制区域
└── 参考图：通过参考图控制内容

应用场景：
├── 精确构图：控制画面构图
├── 姿态控制：控制人物姿势
├── 建筑还原：还原建筑线稿
├── 产品展示：保持产品形状
├── 风格迁移：保持原图结构
└── 手部优化：精确控制手部

支持的模型：
├── Canny：边缘检测
├── M-LSD：直线检测
├── HED：软边缘检测
├── OpenPose：骨骼检测
├── Depth：深度图
├── Normal：法线图
├── Segmentation：分割图
└── Scribble：手绘稿
```

### 1.2 ControlNet使用
```
ControlNet使用技巧：

基础使用流程：
1. 准备控制图
   - 骨骼图/边缘图/深度图等
2. 上传控制图到ControlNet
3. 设置控制权重（weight）
4. 设置引导终止时机（end）
5. 选择预处理器
6. 生成图片

参数设置：

权重（Weight）：
├── 0.0-0.3：弱控制，AI发挥空间大
├── 0.4-0.6：中等控制，平衡
├── 0.7-1.0：强控制，严格遵循
└── 建议：一般0.5-0.8

引导终止（Start/End）：
├── Start：开始引导时机（0-1）
├── End：终止引导时机（0-1）
├── 0.0-0.5：前半段控制
├── 0.5-1.0：后半段控制
└── 全程：0.0-1.0

预处理器：
├── 无预处理：使用已处理好的图
├── 自动预处理：AI自动处理
└── 手动处理：外部工具处理

组合使用：
├── 多ControlNet组合
├── 不同权重分配
├── 不同时机控制
└── 实现复杂效果
```

## 2. 图生图技巧

### 2.1 图生图基础
```
图生图概念：
基于一张参考图片生成新的图片，可以保持参考图的某些特征。

核心参数：

提示词（Prompt）：
├── 参考图的内容描述
├── 想要改变的部分
├── 想要保持的特征
└── 风格和氛围描述

重绘幅度（Denoising Strength）：
├── 0.0-0.3：轻微改变，保持原图
├── 0.4-0.6：中等改变，平衡
├── 0.7-0.85：大幅改变，创新
├── 0.9-1.0：完全重绘
└── 建议：一般0.5-0.75

提示词引导（Prompt Strength）：
├── 控制提示词的影响程度
├── 数值越高，提示词权重越大
├── 与重绘幅度配合
└── 建议：7-12

种子值（Seed）：
├── 固定种子可复现
├── -1为随机种子
├── 调整参数后可对比
└── 保存好种子值
```

```
图生图应用场景：

创意融合：
```

## 3. 局部重绘

### 3.1 局部重绘基础
```
局部重绘概念：
在原图上选择特定区域进行重新生成，其他区域保持不变。

使用方式：

蒙版重绘：
├── 在图上涂抹需要重绘的区域
├── AI只重绘蒙版区域
├── 其他区域保持不变
├── 适合局部修改

手绘蒙版：
├── 用画笔手动涂抹区域
├── 边缘羽化效果更好
├── 避免硬边缘
└── 透明区域表示要重绘

提示词设置：
├── 描述想要生成的内容
├── 可以描述不希望出现的内容
├── 与原图风格保持一致
├── 适当的细节描述
└── 负面提示词同样重要
```

### 3.2 局部重绘技巧
```
局部重绘技巧：

技巧1：边缘处理
├── 羽化蒙版边缘
├── 避免生硬边界
├── 与周围融合自然
├── 使用较小画笔
└── 多次小幅调整

技巧2：内容连贯
├── 保持与周围风格一致
├── 光影方向统一
├── 透视关系正确
├── 色彩色调协调
└── 参考周围细节

技巧3：提示词调整
├── 明确描述重绘内容
├── 参考周围环境描述
├── 适当添加细节
├── 控制重绘范围
└── 负面提示词排除

技巧4：参数调整
├── 降低重绘幅度
├── 提高迭代次数
├── 调整CFG值
├── 使用更多步骤
└── 多次重绘叠加

技巧5：蒙版处理
├── 蒙版边缘模糊
├── 蒙版稍微放大
├── 避免太小区域
├── 复杂区域分次处理
└── 保持蒙版干净
```

## 4. 参数调优

### 4.1 核心参数
```
核心参数说明：

采样器（Sampler）：
├── Euler/Euler a：快速，细节一般
├── DPM++ 2M Karras：平衡，细节好
├── DPM++ SDE Karras：细节丰富，慢
├── DDIM：快速收敛
├── PLMS：DDIM改进
└── 建议：DPM++ 2M Karras

采样步数（Steps）：
├── 15-20：快速，适合简单场景
├── 20-30：标准，质量平衡
├── 30-50：高质量，细节丰富
├── 50+：极高细节，慢
└── 建议：25-35

CFG Scale：
├── 1-5：创意模式，发散
├── 6-8：平衡，细节好
├── 9-12：提示词引导强
├── 13-15：强引导，可能过饱和
├── 16+：极强引导，可能变形
└── 建议：7-12

尺寸（Size）：
├── 512×512：标准，快速
├── 768×768：较大，细节多
├── 1024×1024：大，高质量
├── 横向：16:9, 3:2
├── 纵向：9:16, 2:3
└── 根据用途选择
```

## 5. 工作流设计

### 5.1 标准化工作流
```
AI绘画工作流：

Phase 1：构思阶段
1. 明确需求
   - 设计目标
   - 目标受众
   - 使用场景
   - 尺寸规格

2. 参考收集
   - 风格参考
   - 构图参考
   - 色彩参考
   - 光影参考

Phase 2：生成阶段
3. 提示词编写
   - 结构化编写
   - 参考风格
   - 质量参数
   - 负面提示

4. 参数设置
   - 采样器选择
   - 步数设置
   - CFG调整
   - 尺寸确定

5. 测试生成
   - 生成2-4张
   - 评估效果
   - 记录参数
   - 选择最佳

Phase 3：优化阶段
6. 迭代优化
   - 调整提示词
   - 调整参数
   - 多次生成
   - 筛选最佳

7. 后期处理
   - 局部重绘
   - 放大处理
   - 色彩调整
   - 细节增强

Phase 4：交付阶段
8. 输出处理
   - 格式转换
   - 尺寸调整
   - 质量检查
   - 交付存档
```
