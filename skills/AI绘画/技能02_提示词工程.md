## TL;DR

提示词工程 = 结构化编写 + 风格控制 + 细节描述 + 参数设置 + 迭代优化

## 1. 提示词基础结构

### 1.1 提示词框架
```
提示词结构框架：

主体(subject)：
├── 核心主体：人物、动物、物体
├── 主体特征：年龄、服装、动作
├── 主体数量：单个、多个、群体
└── 主体状态：静态、动态

环境(environment)：
├── 背景设置：室内、室外、天空
├── 时间环境：白天、夜晚、日落
├── 天气氛围：晴天、雨天、雪天
├── 光线氛围：自然光、灯光、阴影
└── 空间氛围：狭窄、开阔、压迫

风格(style)：
├── 艺术风格：油画、水彩、素描
├── 摄影风格：纪实、风光、人像
├── 设计师风格：扁平、极简、3D
├── 时代风格：复古、现代、未来
└── 文化风格：中式、日式、欧美

构图(composition)：
├── 视角：正面、侧面、俯视、仰视
├── 取景：全景、中景、近景、特写
├── 角度：平视、高角度、低角度
├── 构图方式：三分法、对角线、中心
└── 焦点：清晰范围、虚化程度

参数(quality)：
├── 质量词：8K、4K、高清
├── 细节词：详细、精致
├── 渲染词：渲染引擎、材质
├── 光效词：光追、体积光
└── 氛围词：电影感、氛围感
```

### 1.2 提示词公式
```
通用提示词公式：

公式1：简单描述
"主体 + 风格"
示例："一只猫，水彩画风格"

公式2：标准描述
"主体 + 环境 + 风格"
示例："一只猫坐在窗台上，午后阳光，宫崎骏动画风格"

公式3：详细描述
"主体 + 动作 + 环境 + 风格 + 光效 + 构图"
示例："一只橘猫蜷缩在窗台，阳光洒落，动画电影风格，温暖色调，侧光，8K高清"

公式4：专业描述
"[主体描述]，[动作描述]，[环境描述]，
[时间天气]，[光线氛围]，[风格描述]，
[艺术家参考]，[构图方式]，[视角]，
[质量参数]，[细节参数]，[氛围参数]"

示例：
"A beautiful young woman with long flowing hair,
standing on a balcony overlooking a sunset ocean,
wearing a white flowing dress, gentle breeze blowing,
golden hour lighting, cinematic composition,
soft focus background, photorealistic,
8K resolution, detailed textures, dramatic lighting"
```

## 5. 高级技巧

### 5.1 提示词组合
```
提示词组合技巧：

技巧1：分层描述
从宏观到微观逐层描述
示例：
"宏大场景 → 具体物体 → 细节特征 → 质感描述"

技巧2：正反结合
正向描述想要的 + 负向排除不想要的
示例：
"beautiful landscape, sunset, golden light,
no text, no watermark, no cartoon style"

技巧3：权重调节
使用括号或数字调整词权重
SD语法：(word) 或 (word:1.5)
示例：
"((beautiful)) flower, (red:1.3) color"

技巧4：融合混合
使用blend、mix描述混合效果
示例：
"a mix of Van Gogh and Studio Ghibli style"

技巧5：否定提示词
明确排除不需要的元素
通用否定词：
"lowres, bad anatomy, bad hands, text, error,
missing fingers, extra digit, fewer digits,
cropped, worst quality, low quality,
normal quality, jpeg artifacts, signature,
watermark, username, blurry"

技巧6：关键词顺序
越靠前的词权重越高
重要描述放前面
示例：
"1girl, solo, long hair, smiling, cafe setting"
```

### 5.2 迭代优化
```
迭代优化流程：

第1轮：基础生成
- 简单描述主体和风格
- 生成1-2张
- 评估效果

第2轮：细节补充
- 根据效果添加细节
- 补充环境、构图
- 添加质量词
- 生成2-4张

第3轮：风格调整
- 调整风格词
- 添加艺术家参考
- 调整光效词
- 选择最佳

第4轮：精修完善
- 选中最佳图片
- 局部重绘
- 放大高清
- 最终输出

优化检查清单：
□ 主体是否清晰
□ 风格是否符合预期
□ 构图是否满意
□ 光影效果
□ 细节丰富度
□ 无不需要的元素
□ 分辨率是否足够
□ 色彩是否协调
```
