## 1. AI绘画基础

### 1.1 什么是AI绘画
```
AI绘画定义：
利用人工智能技术，根据文字描述或参考图片，自动生成图像的技术。

核心能力：
├── 文字生图：根据文字描述生成图片
├── 图片生图：基于参考图生成新图
├── 图片编辑：修改、重绘图片局部
├── 风格迁移：改变图片艺术风格
└── 图像增强：提升图片质量

发展历程：
├── 2015年：DeepDream开创
├── 2021年：DALL-E、Stable Diffusion出现
├── 2022年：Midjourney、SDXL发展
├── 2023年：SDXL Turbo、LCM加速
└── 2024-2026年：视频生成、图片生成能力大幅提升
```

### 1.2 AI绘画原理
```
AI绘画原理（简化理解）：

扩散模型（Diffusion）：
├── 前向扩散：给图片添加噪声
├── 反向扩散：从噪声还原图片
├── 条件控制：通过文字引导生成
└── 迭代生成：多轮逐步优化

关键技术：
├── CLIP：理解文字和图片的对应关系
├── VAE：压缩和解压图片
├── U-Net：图片特征提取和生成
└── LoRA：个性化风格微调

生成过程：
1. 输入文字描述
2. CLIP理解描述
3. 噪声图开始
4. 多轮迭代去噪
5. 生成最终图片
```

## 2. AI绘画工具

### 2.1 主流工具
```
主流AI绘画工具：

Midjourney：
├── 特点：艺术感强，图片精美

## 3. 基础使用流程

### 3.1 文字生图流程
```
文字生图基本流程：

Step 1：明确需求
├── 想要什么内容
├── 什么风格
├── 什么用途
└── 什么尺寸

Step 2：编写提示词
├── 主体描述：画什么
├── 风格描述：什么风格
├── 细节描述：什么细节
├── 参数设置：尺寸、质量等
└── 参考词：大师作品、艺术家风格

Step 3：参数设置
├── 尺寸比例：1:1、16:9、9:16等
├── 质量参数：质量、细节
├── 风格强度：风格影响程度
├── 生成数量：几张
└── 种子值：固定生成

Step 4：生成和选择
├── 生成多张图片
├── 选择最符合预期的
├── 可能需要调整提示词
└── 迭代优化

Step 5：后期处理
├── 图片放大
├── 图片修复
├── 局部调整
└── 格式转换
```

### 3.2 提示词基础
```
提示词基础结构：

基础公式：
主体 + 环境 + 风格 + 构图 + 参数

示例：
"一只可爱的橘猫，躺在阳光下的窗台上，
宫崎骏动画风格，温暖色调，
侧面视角，特写，8K，高清"

提示词分类：

主体词：
├── 人物：女孩、男孩、老人
├── 动物：猫、狗、鸟
├── 风景：城市、海边、森林
├── 物品：汽车、房子、花
└── 抽象：概念、情感

风格词：
├── 艺术风格：油画、水彩、素描
├── 艺术家：梵高、莫奈、宫崎骏
├── 时代风格：复古、现代、赛博朋克
├── 情绪风格：温暖、忧郁、活泼
└── 媒介风格：照片、插画、海报

质量词：
├── 高质量：8K、4K、HD
├── 细节丰富：detailed、intricate
├── 光影优秀：cinematic lighting
├── 逼真：photorealistic
└── 艺术感：masterpiece、best quality

负面提示词：
├── 低质量：low quality、blurry
├── 畸形：deformed、mutated
├── 不想要：水印、文字
├── 风格问题：anime、cartoon
└── 其他：extra limbs、missing
```

## 5. 注意事项

### 5.1 版权问题
```
AI绘画版权问题：

版权现状：
├── 不同工具政策不同
├── 地区法规不同
├── 商业使用需注意
└── 持续变化中

一般原则：
├── 生成内容可商用 → 查看具体工具政策
├── 使用他人风格 → 可能侵权
├── 使用真实艺术家风格 → 可能侵权
├── 生成的人物图像 → 避免商业滥用
└── 参考图片版权 → 仍需尊重原图版权

安全建议：
├── 使用无版权风险的风格
├── 避免模仿在世艺术家
├── 商业使用购买正版
├── 保留生成记录
└── 咨询法律意见

推荐做法：
├── 原创风格融合
├── 使用公共领域风格
├── 购买正版工具
├── 申请自己的版权
└── 注明AI辅助创作
