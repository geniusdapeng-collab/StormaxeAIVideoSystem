## 1. 风格类型

### 1.1 艺术绘画风格
```
艺术绘画风格：

写实风格：
├── 特点：逼真还原
├── 关键词：photorealistic, realistic, hyperrealistic
├── 适合：人物、产品、风景写实
├── 技巧：使用真实参考，详细描述
└── 缺点：可能缺乏艺术感

印象派：
├── 特点：光影变化，色彩斑斓
├── 关键词：impressionist, Monet style, loose brushstrokes
├── 适合：风景、自然、情绪表达
├── 技巧：使用莫奈、雷诺阿等风格词
└── 缺点：细节可能不够清晰

扁平插画：
├── 特点：简洁、几何、色彩鲜明
├── 关键词：flat illustration, vector art, minimalist
├── 适合：UI设计、海报、图标
├── 技巧：去除阴影，使用纯色
└── 缺点：可能显得单调

动漫风格：
├── 特点：日式动漫，角色可爱
├── 关键词：anime style, manga style, Studio Ghibli
├── 适合：角色设计、二次元内容
├── 技巧：使用宫崎骏、新海诚等风格
└── 缺点：与真实照片风格差距大

水彩风格：
├── 特点：柔和、通透、湿润
├── 关键词：watercolor painting, soft edges, translucent
├── 适合：插画、封面、装饰画
├── 技巧：描述纸张质感、水彩晕染
└── 缺点：细节表现有限
```

### 1.2 设计风格
```
设计风格类型：

极简主义：
├── 特点：简洁、功能、留白
├── 关键词：minimalist, clean, simple, whitespace
├── 适合：现代设计、品牌视觉
├── 搭配：简单构图、低饱和色
└── 元素：几何形状、无衬线字体

孟菲斯风格：
├── 特点：鲜艳、几何、活泼
├── 关键词：Memphis design, geometric patterns, vibrant
├── 适合：海报、封面、包装
├── 搭配：几何图案、对比色
└── 元素：圆形、三角形、波浪线

赛博朋克：
├── 特点：科技、霓虹、夜景
├── 关键词：cyberpunk, neon lights, futuristic, dark city
├── 适合：游戏、影视概念、壁纸
├── 搭配：霓虹灯光、潮湿街道、高楼
└── 元素：LED屏幕、霓虹招牌、雨滴

蒸汽朋克：
├── 特点：维多利亚、机械、复古科幻
├── 关键词：steampunk, Victorian sci-fi, gears, brass
├── 适合：游戏、影视概念、插画
├── 搭配：齿轮、黄铜、蒸汽、飞行器
└── 元素：机械装置、飞行机器、怀旧感

极繁主义：
├── 特点：复杂、华丽、密集
├── 关键词：maximalist, eclectic, ornate, detailed
├── 适合：奢侈品、装饰艺术
├── 搭配：丰富细节、复杂图案
└── 元素：花纹、装饰、层次
```

风格与场景匹配：

商业设计场景：

品牌设计 → 极简主义/扁平插画
包装设计 → 扁平插画/极繁主义
UI设计 → 极简主义/扁平插画

社交媒体 → 扁平插画/动漫风格/水彩
公众号配图 → 扁平插画/简约插画
视频封面 → 扁平插画/设计感强风格

艺术创作场景：
收藏作品 → 油画/写实/印象派

### 2.2 风格搭配原则
```
风格搭配原则：

主次分明：
├── 一个主风格
├── 1-2个辅助风格
├── 避免风格冲突
└── 保持视觉统一

调性一致：
├── 情绪调性：温暖/冷淡/活泼
├── 色彩调性：暖色/冷色/中性
├── 质感调性：粗糙/光滑/柔软
└── 整体协调

受众匹配：
├── 目标用户喜好
├── 品牌调性一致
├── 传播场景适配
└── 文化背景考虑

创意融合：
├── 风格叠加：油画+赛博朋克
├── 风格混搭：扁平+3D
├── 风格致敬：致敬某位艺术家
├── 原创风格：融合创造新风格
└── 避免风格堆砌

示例搭配：
油画+极简 = "极简油画风"
动漫+水彩 = "水彩动漫风"
赛博朋克+浮世绘 = "赛博浮世绘"
扁平+3D = "伪3D插画"
```

## 3. 风格融合

### 3.1 风格融合方法
```
风格融合技巧：

方法1：艺术家融合
公式："art in the style of [艺术家A] meets [艺术家B]"

示例：
"portrait in the style of Van Gogh meets Studio Ghibli"
→ 梵高笔触+宫崎骏氛围

方法2：风格融合
公式："[风格A] meets [风格B]"

示例：
"cyberpunk meets ukiyo-e"
→ 赛博朋克内容+浮世绘风格

方法3：时代融合
公式："[古代风格] meets [现代元素]"

示例：
"traditional Chinese painting meets modern minimalist design"
→ 国画意境+现代简约

方法4：媒介融合
公式："[媒介A] rendered as [媒介B]"

示例：
"photograph painted as watercolor"
→ 照片构图+水彩质感

方法5：主题融合
公式："[主题A] reimagined in the style of [主题B]"

示例：
"Greek mythology reimagined in cyberpunk style"
→ 希腊神话+赛博朋克世界观
```

## 4. 风格统一

### 4.1 系列作品风格
```
系列作品风格统一：

统一要素：
├── 主色调统一
├── 构图方式统一
├── 风格关键词统一
├── 画面比例统一
└── 细节层次统一

建立模板：
风格提示词模板：
"[主体描述], [统一风格词], 
[统一构图词], [统一色调词],
[统一光效词], best quality, masterpiece"

示例（治愈系插画系列）：
"soft pastel illustration, cozy scene,
3/4 view composition, warm color palette,
golden hour lighting, gentle brushstrokes,
same style consistency, best quality, masterpiece"

配色方案：
├── 主色：柔和粉/淡蓝/米白
├── 辅助色：淡绿/浅黄
├── 点缀色：珊瑚粉/薄荷绿
└── 保持统一

构图规范：
├── 景别：全身/中景/近景
├── 视角：平视/俯视/仰视
├── 三分法：主体位置
└── 留白：上下左右留白比例
```
