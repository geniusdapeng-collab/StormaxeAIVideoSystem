# 叙事转场类型 v5.1-Peng

> 转场不仅是技术操作，更是叙事工具。每种转场类型都有特定的情绪和叙事功能。

## 五大叙事转场类型

### 1. 硬切 (Hard Cut)
- **叙事功能**: 节奏突变、时间跳跃、空间转换
- **情绪效果**: 直接、无缓冲、制造意外感
- **适用场景**: 动作场景、悬念揭示、时间线跳转
- **提示词标注**: `；镜头切换：`
- **示例**: 战斗画面 → 黑屏 → 战场废墟

### 2. 渐隐渐显 (Fade In/Out)
- **叙事功能**: 时间流逝、章节分隔、情绪沉淀
- **情绪效果**: 温柔过渡、留白呼吸、暗示时间变化
- **适用场景**: 章节之间、日夜转换、回忆与现实交替
- **提示词标注**: `渐暗→渐亮：`
- **示例**: 黄昏场景渐暗 → 清晨场景渐亮

### 3. 匹配剪辑 (Match Cut)
- **叙事功能**: 主题连接、时空跨越、隐喻关联
- **情绪效果**: 诗意连贯、恍然大悟、视觉惊喜
- **适用场景**: 形状匹配、动作连贯、象征性转场
- **提示词标注**: `匹配剪辑：`
- **示例**: 旋转的陀螺 → 旋转的地球

### 4. 蒙太奇叠化 (Montage Dissolve)
- **叙事功能**: 时间压缩、过程展示、情绪积累
- **情绪效果**: 流动感、时间加速、渐进变化
- **适用场景**: 训练过程、季节变化、情绪渐变
- **提示词标注**: `叠化：`
- **示例**: 春天花开 → 叠化 → 秋天落叶

### 5. 声音先入 (J-Cut / L-Cut)
- **叙事功能**: 提前引入下一场景、制造期待、无缝连接
- **情绪效果**: 好奇心、 anticipation、沉浸感
- **适用场景**: 对话场景、环境音引导、悬念铺垫
- **提示词标注**: `声音先入：`
- **示例**: 听到爆炸声（当前画面未变）→ 切换到爆炸现场

## 转场选择规则

| 前镜头情绪 | 后镜头情绪 | 推荐转场 |
|-----------|-----------|---------|
| 高张力 → 高张力 | 硬切 |
| 高张力 → 低张力 | 渐隐渐显 |
| 低张力 → 高张力 | 声音先入 |
| 同主题 → 同主题 | 匹配剪辑 |
| 时间压缩 | 蒙太奇叠化 |
| 回忆/梦境 | 渐隐渐显 |
| 现实跳跃 | 硬切 |

## 转场标注格式

在分镜表中，每个镜头的 `transitionType` 字段应包含：
```json
{
  "transitionTo": "S02",
  "transitionType": "硬切",
  "transitionNote": "节奏突变，战斗爆发",
  "transitionDuration": "0.5s"
}
```

## 后期制作转场时长建议

| 转场类型 | 建议时长 |
|---------|---------|
| 硬切 | 0帧（瞬时） |
| 渐隐渐显 | 1-2秒 |
| 匹配剪辑 | 0帧+形状关联 |
| 蒙太奇叠化 | 1-3秒 |
| 声音先入 | 音频提前0.5-1秒 |
