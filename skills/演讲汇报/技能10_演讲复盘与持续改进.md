### 场景分类

## 2. 复盘框架

### 复盘流程

```
1. 收集反馈：获取多方面反馈
2. 自我评估：自己的感受
3. 问题分析：找出关键问题
4. 制定计划：改进措施
5. 持续练习：执行计划
```

### 反馈来源

| 来源 | 方式 | 价值 |
|------|------|------|
| 听众 | 问卷/访谈 | 真实感受 |
| 同事 | 直接反馈 | 专业意见 |
| 录像 | 自我观察 | 客观视角 |
| 数据 | 演讲数据 | 量化指标 |

### A. 反馈收集

#### 收集方法
```
1. 问卷：结构化反馈

4. 录像：回放分析

```

#### 分析方法
```
1. 分类整理：按类型分类
2. 优先级：找出关键问题
3. 根本原因：5Why分析法
4. 影响分析：问题影响

```

### C. 改进计划

#### 计划制定
```
1. 明确目标：具体可衡量
2. 具体措施：可执行
3. 时间计划：时间表
4. 资源需求：需要什么
5. 评估标准：怎么衡量
```

### D. 持续练习

#### 练习方法
```
1. 针对性练习：针对问题
2. 刻意练习：专注改进
3. 反馈循环：持续反馈

```
