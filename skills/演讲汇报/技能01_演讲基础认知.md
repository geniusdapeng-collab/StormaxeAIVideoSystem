## 2. 演讲本质框架

### 演讲是什么

```
演讲 = 信息传递 + 情感连接 + 价值创造

核心要素：
1. 内容：有价值的信息
2. 表达：清晰的呈现
3. 连接：与听众共鸣
4. 行动：引导改变
```

## 3. 专业建议

### A. 核心素养

#### 演讲者素养
```
1. 专业能力：领域知识
2. 表达能力：清晰传达
3. 心理素质：稳定自信
4. 应变能力：临场处理
5. 学习能力：持续提升
```

### B. 演讲准备

#### 准备清单
```
1. 明确目标：演讲目的是什么
2. 了解听众：他们是谁
3. 确定主题：核心信息
4. 收集素材：支撑材料
5. 设计结构：逻辑框架
6. 制作PPT：视觉辅助
7. 反复演练：熟练表达
```

### C. 基本流程

#### 演讲流程
```
1. 准备阶段：内容+PPT+演练
2. 开场阶段：吸引注意力
3. 主体阶段：传递核心信息
4. 结尾阶段：强化记忆
5. 问答阶段：回答问题
6. 跟进阶段：后续联系
```

### D. 常见误区

#### 避免误区
```
× 内容太多：贪多嚼不烂
× 照本宣科：缺乏互动
× 忽视听众：自说自话
× 准备不足：临场发挥
× 追求完美：不敢开始
```
