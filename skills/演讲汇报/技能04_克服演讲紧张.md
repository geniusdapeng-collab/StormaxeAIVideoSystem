## TL;DR

克服紧张核心：接纳紧张→充分准备→掌握技巧→正向暗示，四步建立演讲自信

## 1. 场景分析

### 请求摘要
用户演讲时感到紧张

### 显性需求
- 掌握克服紧张方法
- 获得可执行的心理建设方案

### 场景分类
- `scenario_classification`: 心理调适
- `urgency`: high

## 2. 紧张原因框架

### 常见原因

| 原因 | 表现 | 解决方法 |
|------|------|----------|
| 经验不足 | 害怕出错 | 多练习 |
| 完美主义 | 怕表现不好 | 接纳不完美 |
| 缺乏准备 | 心里没底 | 充分准备 |
| 被关注感 | 害怕注视 | 转移焦点 |
| 生理反应 | 手抖心跳 | 生理调节 |

### 紧张循环

```
紧张 → 生理反应 → 更紧张 → 失控
  ↓
打破：接纳+准备+技巧
```

## 3. 专业建议

### A. 生理调节

#### 即时放松技巧
```
1. 深呼吸：4-7-8呼吸法
2. 放松肌肉：紧张-放松练习
3. 走动：释放能量
4. 喝水：缓解口干
5. 微笑：欺骗大脑
```

### B. 心理建设

#### 认知调整
```
1. 接纳紧张：正常现象
2. 转移焦点：关注内容
3. 正面暗示：我能行
4. 降低预期：完成>完美
5. 价值视角：分享>表演
```

### C. 充分准备

#### 准备清单
```
1. 内容熟练：倒背如流
2. 场地熟悉：提前到场
3. 设备测试：确保正常
4. 听众了解：知己知彼
5. 预案准备：应对意外
```

### D. 练习方法

#### 脱敏训练
```
1. 模拟演练：对着镜子
2. 录音录像：自我观察
3. 小场开始：逐步升级
4. 同伴反馈：互相练习
5. 真实上场：积累经验
```
