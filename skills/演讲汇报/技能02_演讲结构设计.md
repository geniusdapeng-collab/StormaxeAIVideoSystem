
## 3. 专业建议

### A. 开场设计

#### 开场方法
```
1. 提问式：引发思考
2. 故事式：建立连接
3. 数据式：展示价值
4. 震撼式：制造冲击
5. 幽默式：拉近距离
```

### B. 主体结构

#### 常用结构
```
1. 问题-解决方案
2. 时间顺序
3. 空间顺序
4. 重要性顺序
5. 对比结构
6. 金字塔结构
```

### C. 结尾设计

#### 结尾方法
```
1. 总结式：回顾要点
2. 呼吁式：行动号召
3. 展望式：未来愿景
4. 故事式：情感共鸣
5. 金句式：留下记忆
```
