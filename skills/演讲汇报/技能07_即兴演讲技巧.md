## 2. 即兴框架

### 黄金结构

```
1. 开头（20%）：承接/表态
2. 主体（60%）：观点+论证
3. 结尾（20%）：总结/升华
```

### 即兴模型

| 模型 | 结构 | 适用场景 |
|------|------|----------|
| PREP | 观点-理由-举例-总结 | 观点表达 |
| 故事 | 背景-冲突-解决-结果 | 经验分享 |
| 否定 | 现象-问题-原因-方案 | 问题讨论 |
| 时间 | 过去-现在-未来 | 发展回顾 |

## 3. 专业建议

### A. 思维技巧

#### 快速思维
```
1. 关键词法：抓取核心词
2. 联想发散：由点及面
3. 分类思考：从不同维度
4. 举例论证：用故事支撑
5. 升华高度：上升到价值
```

### B. 素材积累

#### 常用素材
```
1. 亲身经历
2. 名人故事
3. 行业案例
4. 数据事实
5. 金句名言
```

### C. 万能话术

#### 开头话术
```
1. 关于这个问题...
2. 我分享一个观点...
3. 从我的经验来看...
4. 刚才 XX 说的...
```
