## 3. 专业建议

### A. 提问技巧

#### 提问类型
```
1. 封闭式：是否/对错
2. 开放式：为什么/怎么
3. 引导式：引导思考
4. 挑战式：引发讨论
```

#### 提问时机
```
1. 开场：破冰
2. 中间：保持注意力
3. 转折：过渡
4. 结尾：强化记忆
```

### B. 气氛调动

#### 调动方法
```
1. 幽默：适当玩笑
2. 故事：引发共鸣
3. 案例：贴近实际
4. 数据：震撼展示
5. 视觉：吸引眼球
```

### C. 反馈处理

#### 回应技巧
```
1. 积极肯定：无论对错
2. 引导深入：追问为什么
3. 总结提炼：归纳观点
4. 感谢参与：鼓励参与
5. 控制节奏：避免跑题
```

### D. 互动设计

#### 设计原则
```
1. 提前准备：设计互动点
2. 自然融入：不显突兀
3. 时间控制：适度即可
4. 风险评估：避免冷场
5. 备选方案：准备预案
```
