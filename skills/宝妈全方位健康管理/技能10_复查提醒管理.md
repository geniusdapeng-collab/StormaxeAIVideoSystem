## TL;DR

**复查核心原则 = 按时间表查 + 就医前准备 + 有症状不等待**

HbA1c每3个月 | 眼底诊断时即查 | 尿蛋白/血脂每6个月
急性症状（DKA/HHS/胸痛/视力骤降）→ 立即急诊，不等复查

## 2. 就医准备清单

### 就诊前3天

```
□ 整理最近30天血糖记录（SMBG日志或CGM报告）
□ 整理饮食记录（重点：是否有特殊情况，如外食、节日）
□ 回顾用药情况（是否规律服药，有无漏服，有无新加药物）
□ 列出想问医生的问题（3个以内，优先最重要的）
□ 确认就诊时间和科室
□ 准备既往病历和检查报告
```

### 就诊当天

```
□ 空腹就诊（部分检查需要，如空腹血糖和血脂）
□ 带上血糖记录本/APP截图
□ 带上近期用药清单（药名+剂量+服药时间）
□ 记录医生说的关键信息（可以录音，需事先告知医生）
□ 如有需要，计划下一步检查的时间
□ 带好医保卡和身份证
```

### 就诊时向医生汇报的三大核心信息

```
【主诉】
GDM产后7个月，FBG持续≥12 mmol/L已7个月

【血糖数据汇总】
- 最近1个月空腹血糖范围：X至Y mmol/L
- 最近1个月餐后2h血糖范围：X至Y mmol/L
- 最近一次HbA1c：X%（如已测）
- 低血糖事件（近1个月）：□无  □有，共N次，最低调到X mmol/L

【当前治疗方案】
- 口服药：□二甲双胍  □其他：___  剂量：___
- 胰岛素：□是  剂型：___  剂量：___
- 饮食/运动控制：□是

【想向医生提出的问题（≤3个）】
1. _______________________
2. _______________________
3. _______________________
```

## 4. 预约挂号攻略

### 挂号渠道

| 渠道 | 适用情况 | 说明 |
|------|---------|------|
| 医院官方App/公众号 | 首选，大多数三甲医院 | 提前7–14天放号 |
| 114/当地预约平台 | 统一预约，覆盖多家医院 | 提前7天 |
| 好大夫在线 | 专家加号/在线问诊 | 适合专家号 |
| 支付宝/微信城市服务 | 快速挂号 | 覆盖医院有限 |
| 医院现场挂号 | 紧急情况 | 热门科室可能无号 |

### 抢号技巧

- 大医院热门科室提前7天00:00抢号（如北京协和医院）
- 眼底/内分泌等热门专科提前14天
- 第一次就诊后直接在医生处约下次复诊（最靠谱）
- 糖妈妈可同时约内分泌科+眼科，半天解决

### 异地就医（如有必要去上级医院）

- 提前在国家医保服务平台App做异地就医备案
- 带好当地医院的检查报告（避免重复检查）
- 热门专家可先在好大夫在线问诊，确认需要面诊再前往

## 5. HbA1c解读

| HbA1c值 | 血糖控制状态 | 说明 |
|---------|------------|------|
| < 6.0% | 接近正常 | 可能接近缓解，需维持 |
| 6.0%–6.5% | 良好 | 低于大多数成人目标 |
| 6.5%–7.0% | 达标 | 一般控制目标 |
| 7.0%–7.5% | 基本达标 | 需进一步优化 |
| 7.5%–8.0% | 控制不佳 | 需调整方案 |
| > 8.0% | 差 | 需显著干预 |

**注**：具体目标因人而异，需结合年龄、低血糖风险、病程等因素由医生制定。
