## TL;DR

**用药核心原则 = 胰岛素最安全 + 高血糖比药物更危险 + 不自行停药**

胰岛素：✅完全安全（不入乳汁）| 二甲双胍：✅风险低 | GLP-1RA：⚠️证据有限 | SGLT2i：❌不推荐

---

## 1. 哺乳期降糖药安全性速查表

| 药物类别 | 代表药物 | 哺乳期 | 说明 |
|---------|---------|--------|------|
| 胰岛素 | 门冬、赖脯、甘精、德谷 | ✅ 完全安全 | 不进入乳汁，哺乳期首选降糖药 |
| 双胍类 | 二甲双胍 | ✅ 风险低 | 乳汁中浓度极低（<1%母体浓度），美国儿科学会认为兼容 |
| DPP-4抑制剂 | 西格列汀、利格列汀 | ⚠️ 证据有限 | 利格列汀可能更安全，总体风险低 |
| GLP-1RA | 司美格鲁肽、度拉糖肽、艾塞那肽 | ⚠️ 证据有限 | 不建议哺乳期新启动，既往使用者建议停药或回避 |
| SGLT2抑制剂 | 恩格列净、达格列净、卡格列净 | ❌ 不推荐 | 可能影响婴儿水分平衡/泌尿生殖系统发育 |
| 磺脲类 | 格列本脲、格列美脲 | ⚠️ 谨慎 | 可进入乳汁，可能引起婴儿低血糖，优选格列美脲 |
| α-糖苷酶抑制剂 | 阿卡波糖、伏格列波糖 | ⚠️ 证据有限 | 几乎不吸收，理论安全，但缺乏临床数据 |
| 噻唑烷二酮类 | 罗格列酮、吡格列酮 | ❌ 不推荐 | 禁用于哺乳期 |

**核心原则**：高血糖本身对母婴都有害，不能因为哺乳延误治疗。胰岛素是哺乳期最安全的降糖药选择。

---

## 2. 各药物服用要点

### 胰岛素

| 要点 | 说明 |
|------|------|
| 剂型选择 | 基础胰岛素（甘精/地特/德谷）+ 餐前速效（门冬/赖脯）是哺乳期标准方案 |
| 与哺乳关系 | 与哺乳无时间冲突，餐前注射即可 |
| 低血糖风险 | 是主要风险，尤其是夜间哺乳后需注意 |
| 储存 | 未开封2–8°C冷藏，开封后室温(<30°C)可存28–42天 |

### 二甲双胍

| 要点 | 说明 |
|------|------|
| 服用时间 | 餐后立即服用（减少胃肠道反应）|
| 起始剂量 | 通常0.5g，2次/日开始，逐步加量 |
| 哺乳期剂量 | 多数研究支持常规剂量，注意婴儿大便变化（可能有轻微腹泻）|
| 胃肠道反应 | 恶心/腹泻：餐后服用，从小剂量开始逐步适应 |

## 3. 漏服/忘记服药处理

### 二甲双胍漏服

| 想起时间 | 处理方式 |
|---------|---------|
| 想起来时距下次服药≥2h | 可补服 |
| 想起来时距下次服药<2h | 跳过本次，不补服，不加倍 |
| 想起时已过餐后2h | 测血糖再判断是否补服 |

### 胰岛素漏打

| 情况 | 处理 |
|------|------|
| 餐前速效胰岛素漏打 | 餐后尽快补打（≤30min内），或咨询医生调整 |
| 基础胰岛素漏打 | 不要加倍，联系医生评估是否补打 |
| 漏打超过1次 | 立即联系医生，不自行连续补打 |
| 夜间基础胰岛素漏打 | 测睡前血糖，高于目标则补打，低于目标则观察，次日就医评估 |

### 一般原则

```
漏服处理三步：
1. 立即测血糖，了解当前血糖状态
2. 评估漏服药物对血糖的可能影响
3. 不要在下次服药时加倍剂量（除非医生明确指示）
4. 如有疑问，联系医生或药师
```

## 4. 常见副作用及处理

### 二甲双胍副作用

| 副作用 | 发生率 | 处理 |
|--------|--------|------|
| 胃肠道反应（恶心/腹胀/腹泻）| 20%–30% | 餐后立即服用，从小剂量开始，逐步加量 |
| 维生素B12缺乏（长期使用）| 5%–10% | 每年监测B12水平，必要时补充 |
| 乳酸酸中毒 | 极罕见 | 严重肝肾疾病者禁用 |

### 胰岛素副作用

| 副作用 | 处理 |
|--------|------|
| 低血糖（最常见）| 见血糖监测技能——15-15规则 |
| 体重增加 | 正常现象，与剂量相关，注意饮食控制 |
| 注射部位反应（红肿/硬结）| 轮换注射部位，每次间隔2–3cm |
| 过敏反应（极罕见）| 立即就医，换用其他胰岛素剂型 |

### 磺脲类副作用

| 副作用 | 处理 |
|--------|------|
| 低血糖（风险高于二甲双胍）| 及时加餐，熟悉低血糖识别和处理 |
| 体重增加 | 注意饮食，咨询医生是否可以联用二甲双胍 |

## 5. 用药与哺乳的时机安排

| 药物 | 与哺乳的关系 | 建议服用时机 |
|------|------------|------------|
| 胰岛素 | 无冲突 | 餐前注射，可正常哺乳 |
| 二甲双胍 | 乳汁中浓度极低 | 哺乳后立即服用，婴儿暴露最小化 |
| 格列美脲 | 可进入乳汁 | 哺乳后立即服用，监测婴儿有无低血糖表现 |
| 西格列汀 | 证据有限 | 哺乳后服用，观察婴儿状态 |

---

## 6. 就医指征（用药相关）

| 情况 | 建议 |
|------|------|
| 血糖持续偏高（FBG>11.1 mmol/L）| 联系医生调整降糖方案 |
| 频繁低血糖（>3次/周）| 立即联系医生，可能是药物剂量过大 |
| 无法耐受药物副作用 | 联系医生换药 |
| 想停药 | 不自行停药，先与医生讨论 |
