## TL;DR

**睡眠核心原则 = 总量够用 + 节律稳定 + 皮质醇管理**

睡眠<6小时 → 胰岛素敏感性↓30–40% → 血糖难控
固定作息时间 + 白天补觉 + 压力管理 = 睡眠优化三板斧

## 1. 睡眠与血糖的关系

### 机制链条

```
睡眠不足/睡眠片段化
    ↓
交感神经激活 + 皮质醇分泌↑（尤其是深夜和凌晨）
    ↓
肝脏糖异生增加 + 胰岛素敏感性↓30–40%
    ↓
空腹血糖↑ + 整体血糖控制恶化
```

### 睡眠不足的量化影响

| 睡眠时长 | 胰岛素敏感性变化 | 空腹血糖变化 |
|---------|--------------|-----------|
| 8–9小时（充足）| 基准 | 基准 |
| 6–7小时 | ↓约15%–20% | ↑约0.5–1.0 mmol/L |
| 5–6小时 | ↓约20%–30% | ↑约1.0–1.5 mmol/L |
| <5小时 | ↓约30%–40% | ↑约1.5–2.5 mmol/L |

### 哺乳期睡眠的特殊性

- 产后7个月的妈妈通常夜间仍需哺乳1–2次
- 每次哺乳醒来的睡眠片段化，会打断深睡眠周期
- 夜间皮质醇本应处于低值，频繁夜醒会让皮质醇"误以为"需要应激
- 睡眠片段化还会导致日间疲劳、情绪不稳、进食决策失控（更容易选择高GI食物）

### 糖妈妈睡眠常见问题

| 问题 | 表现 | 可能原因 |
|------|------|---------|
| 哺乳打断睡眠 | 婴儿醒来后难以快速再入睡 | 催乳素干扰，焦虑思维 |

| 睡眠片段化 | 一夜醒>3次，无法进入深睡眠 | 哺乳/焦虑/疼痛 |

## 3. 皮质醇与血糖管理

### 皮质醇的日节律

```
时间        皮质醇水平    生理状态
06:00–08:00  全天最高峰值  觉醒反应
08:00–12:00  快速下降     日间活动期
12:00–18:00  逐渐下降     相对稳定
18:00–20:00  轻微波动     晚间过渡
20:00–22:00  开始下降     准备入睡
22:00–06:00  全天最低值   深睡眠期
```

### 皮质醇管理与血糖

- **皮质醇↑ = 血糖↑**：肝脏糖异生增加，胰岛素敏感性下降
- **哺乳期特殊性**：夜间哺乳后如果焦虑/睡不着，皮质醇会在本应低谷的时候反跳
- **凌晨皮质醇高峰**：有些人会在凌晨2–4点出现皮质醇小高峰，这就是早醒失眠的生理原因

### 皮质醇管理技巧

| 时段 | 目标 | 具体做法 |
|------|------|---------|
| 早晨（6–8点）| 充分利用皮质醇高峰 | 起床后接触自然光15min，激活日节律 |
| 日间（9–17点）| 保持活跃，不打盹>30min | 规律活动，午睡不超过13:00 |
| 傍晚（17–20点）| 逐渐放松，降低皮质醇 | 做放松活动，避免高强度工作 |
| 夜间（20点后）| 主动降皮质醇 | 关强光，温水澡，冥想/深呼吸 |
| 睡前（22点前）| 皮质醇降到最低 | 固定睡前仪式，不看手机 |

## 4. 睡眠卫生优化方案

### 环境优化

| 要素 | 建议 |
|------|------|
| 温度 | 卧室18–22°C，偏凉更好睡 |
| 光线 | 全黑环境（遮光窗帘），夜间哺乳用最暗的小夜灯 |
| 声音 | 安静，可接受白噪音（风扇/空调声）|
| 床垫/枕头 | 支撑良好，产后盆底友好（不过软）|
| 手机 | 充电放在够不着的地方，睡前1小时不看 |

### 睡前仪式（帮助大脑识别"该睡了"）

```
睡前30–60分钟：
  1. 关闭强光，调暗室内灯光
  2. 温水淋浴或泡脚（15min，40°C左右）
  3. 轻柔拉伸或瑜伽放松体式
  4. 5–10分钟冥想或深呼吸（4-7-8呼吸法）
  5. 如有需要，记录焦虑想法（写在纸上）然后放下
```

### 4-7-8呼吸法（降低皮质醇，助眠）

```
步骤：
1. 用鼻子吸气，数4下
2. 屏住呼吸，数7下
3. 用嘴缓慢呼气，数8下
4. 重复3–5次

原理：延长呼气激活副交感神经，对抗"战斗或逃跑"反应
```

## 5. 夜间哺乳与睡眠管理

### 减少哺乳对睡眠打断的策略

| 策略 | 说明 |
|------|------|
| 同床不同床之争 | WHO推荐母婴同室但不同床，减少哺乳距离 |
| 瓶喂分担 | 配偶或家人用提前挤出的母乳/配方奶喂一次夜间 |
| 侧躺哺乳 | 妈妈侧躺喂，减少起身打断睡眠 |
| 床旁冰箱 | 存储当天挤出的母乳，夜间快速瓶喂 |
| 记录哺乳时间 | 避免不必要叫醒（婴儿可能只是哼唧而非真正饿）|

### 白天补觉策略

- 日间每次小睡不超过60–90分钟（一个完整睡眠周期）
- 最佳补觉时间：13:00–15:00（午后皮质醇低谷期）
- 即使30分钟的小睡也有显著恢复效果
- 哺乳妈妈可与婴儿同步睡眠（婴儿睡你也睡）

## 6. 作息优化建议

### 固定作息时间表

| 时间 | 活动 | 说明 |
|------|------|------|
| 06:30–07:00 | 起床 | 固定时间，不赖床 |
| 07:00–07:30 | 晨间活动（洗漱、早餐、接触阳光）| 阳光激活日节律 |
| 07:30–21:00 | 日间活动 | 规律哺乳、进食、工作、运动 |
| 21:00–21:30 | 睡前仪式开始 | 关强光，放松活动 |
| 21:30–22:00 | 尝试入睡 | 固定就寝时间窗口 |
| 22:00–06:00 | 睡眠时段 | 目标7–8小时睡眠 |
| 13:00–14:00 | 午睡（可选，30–60min）| 不影响夜间睡眠 |

### 咖啡因管理

- 哺乳期咖啡因摄入限制：<200mg/日（约2杯中杯咖啡）
- 下午14:00后不摄入咖啡因（半衰期5–6小时，14:00喝的咖啡到20:00还有一半）
- 警惕隐藏咖啡因：奶茶、巧克力、可乐、某些止痛药

## 7. 就医指征

| 情况 | 建议 |
|------|------|
| 持续失眠>2周 | 建议睡眠医学科或心理科就诊 |
| 严重打鼾/呼吸暂停 | 建议睡眠呼吸暂停筛查 |
| 早醒+情绪低落持续>2周 | 建议心理/精神科EPDS筛查 |
| 日间极度嗜睡无法自控 | 建议就医排除甲状腺等问题 |
