## TL;DR

**饮食核心原则 = 够用不超 + 低GI优先 + 顺序控糖 + 营养均衡**

每日热量不<1800kcal，碳水不<175g，蛋白质≥70g
食物顺序：蔬菜 → 蛋白质 → 主食（降餐后血糖1–4 mmol/L）

---

## 1. 每日总热量计算

### 计算公式

```
Step 1：计算标准体重（SBW）
  SBW(kg) = 身高(cm) - 105

Step 2：计算每日总热量
  每日总热量 = SBW × (25–30) kcal/kg × 活动系数 × 哺乳系数

Step 3：哺乳期热量增加
  哺乳期：+300–500 kcal/日（纯母乳取高值，混合喂养取低值）
```

### 活动系数

| 活动量 | 系数 | 说明 |
|--------|------|------|
| 卧床/极少活动 | 20–25 | 产后初期 |
| 轻度活动（日常行走）| 25–30 | 轻体力 |
| 中度活动（规律运动）| 30–35 | 有氧运动≥150min/周 |

### 示例（身高165cm，轻体力，哺乳期）

```
SBW = 165 - 105 = 60kg
基础 = 60 × 27.5(取中间值) = 1650kcal
活动系数 = 1.1（取27.5/25的折算值）
实际基础 = 1650 × 1.1 = 1815kcal
哺乳增加 = +450kcal（取中间值）
每日总热量 ≈ 2265kcal
```

### 热量档位（哺乳期参考）

| 档位 | 每日热量 | 适用情况 |
|------|---------|---------|
| 保守档 | 1800–2000 kcal | 体重较高需减重，或混合喂养 |
| 标准档 | 2000–2300 kcal | 标准体重，纯母乳喂养 |
| 高档 | 2300–2500 kcal | 体重偏低，需增重，或双胞胎哺乳 |

## 2. 宏量营养素配比

### 每日摄入目标（哺乳期2型糖尿病）

| 营养素 | 占总热量比例 | 每日参考量 | 每克热量 |
|--------|------------|----------|---------|
| 碳水化合物 | 45%–55% | ≥175g/日 | 4kcal/g |
| 蛋白质 | 15%–20% | ≥70–80g/日 | 4kcal/g |
| 脂肪 | 25%–30% | 50–75g/日 | 9kcal/g |
| 膳食纤维 | — | 25–35g/日 | 0 |

### 各营养素核心原则

**碳水化合物（最重要的血糖影响因素）**：

- 每日不<175g（影响乳汁产量）
- 优先低GI碳水（全谷物、豆类、低GI水果）
- 限制精制糖（<25g/日）和高GI碳水
- 碳水均匀分配到各餐（避免一餐过少或过多）

**蛋白质（哺乳期需求量增加）**：

- 每日≥70–80g（普通成人约60g）
- 优质蛋白占比≥50%：鱼、禽、蛋、奶、豆制品
- 每餐都要有蛋白质（平稳餐后血糖）

**脂肪**：

- 饱和脂肪<7%总热量（约<15g/日）
- 限制动物油、棕榈油
- 优先不饱和脂肪：橄榄油、菜籽油、坚果、深海鱼
- Omega-3脂肪：每周吃2–3次深海鱼（三文鱼、鲈鱼）

**膳食纤维**：

- 每日25–35g
- 来源：蔬菜（500g/日）、全谷物、豆类

## 3. GI/GL速查

### GI（血糖生成指数）分类

| 分类 | GI值 | 对血糖影响 | 代表食物 |
|------|------|----------|---------|
| 低GI | ≤55 | 血糖缓慢上升 | 燕麦、糙米、藜麦、绿豆、苹果、梨 |
| 中GI | 56–69 | 血糖中速上升 | 荞麦、全麦面包、红薯、菠萝 |
| 高GI | ≥70 | 血糖快速飙升 | 白米饭、白馒头、土豆泥、西瓜 |

## 5. 食物进食顺序（降餐后血糖1–4 mmol/L）

**最佳进食顺序（研究证实有效）**：

```
第1步：先吃蔬菜（不限量，500g以上）
  ↓ 等待3–5分钟，让膳食纤维先行
第2步：再吃蛋白质（鱼/肉/蛋/豆腐）
  ↓ 蛋白质刺激GLP-1分泌
第3步：最后吃主食（碳水）
  ↓ 此时碳水被纤维和蛋白质"包裹"，吸收减慢
```

**原理**：先吃蔬菜和蛋白质 → 胃排空延迟 → 碳水吸收减慢 → 餐后血糖峰值降低

**额外好处**：增加饱腹感，减少主食摄入量

## 6. 典型餐单示例（2000–2200kcal，哺乳期）

### 早餐（约450–500kcal）

```
主食：燕麦50g（煮成300g燕麦粥）
蛋白质：鸡蛋1个（60g）+ 牛奶200mL
水果：蓝莓50g
饮品：温水或无糖豆浆
烹饪油：0（燕麦无需加油）

→ 碳水约50g，蛋白质约22g
→ 早餐后血糖通常较平稳
```

### 午餐（约550–650kcal）

```
主食：糙米饭75g（生重，熟重约200g）
蔬菜：西兰花200g + 胡萝卜50g（清炒或蒜蓉）
蛋白质：鲈鱼100g（清蒸，少油）
豆制品：嫩豆腐100g（凉拌）
烹饪油：橄榄油10g
汤：番茄蛋汤（少盐）

→ 碳水约65g，蛋白质约40g
→ 鱼类富含Omega-3，对哺乳有益
```

### 晚餐（约500–600kcal）

```
主食：藜麦50g（生重）+ 杂粮馒头半个（30g）
蔬菜：菠菜200g + 木耳50g（凉拌）
蛋白质：鸡胸肉80g（烤或炒）
菌菇：香菇100g（炒菜）
烹饪油：橄榄油8g

→ 碳水约55g，蛋白质约38g
→ 晚餐不宜过晚（睡前3小时完成）
```

### 上午加餐（约100–150kcal）

```
无糖酸奶150g + 坚果10g（核桃/杏仁）

→ 蛋白质补充，饱腹感好
→ 避免血糖过低导致午餐前饥饿
```

### 下午加餐（约100–150kcal）

```
苹果1个（小个，约150g）+ 奶酪20g

→ 水果提供维生素，奶酪补充蛋白质和钙
```

### 睡前加餐（如血糖偏低或需注射胰岛素）

```
牛奶200mL + 全麦饼干1–2块

→ 预防夜间低血糖，保障睡眠
```

## 7. 哺乳期关键营养素

| 营养素 | 每日需求 | 食物来源 | 说明 |
|-------|---------|---------|------|
| 钙 | 1000mg | 奶制品（牛奶/酸奶）、豆制品、芝麻 | 哺乳期需求比孕期还高 |
| 铁 | 18mg | 瘦红肉（猪里脊、牛肉）、动物肝脏 | 产后7个月是铁缺乏高发期 |
| DHA | 200–300mg | 深海鱼（鲈鱼、三文鱼）、藻油 | 有助婴儿神经发育 |
| 维生素D | 600–800IU | 日照（每日15min）、强化奶 | VD促进钙吸收 |
| 碘 | 240μg | 海带、紫菜、加碘盐 | 影响婴儿甲状腺发育 |
| 蛋白质 | ≥70–80g | 鱼/肉/蛋/奶/豆 | 保障乳汁蛋白质量 |
| 水分 | ≥2000mL | 白水、汤、牛奶 | 哺乳期需水量增加 |

## 8. 外出就餐/外卖选择指南

| 场景 | 推荐选择 | 需注意 |
|------|---------|--------|
| 中式快餐 | 盖浇饭→换成杂粮饭，多点蔬菜 | 不要把菜汁拌饭（油盐重）|
| 面食 | 选粗粮面/荞麦面，多加蔬菜和蛋 | 清汤面优于油炸/酱拌 |
| 西式快餐 | 烤鸡腿堡（去皮）+ 蔬菜沙拉 | 不要薯条和含糖饮料 |
| 日料 | 刺身、烤鱼、味噌汤 + 糙米饭 | 酱油蘸料少用（盐重）|
| 火锅 | 清汤锅底，多涮蔬菜和鱼/肉 | 麻酱蘸料换成酱油+醋 |
| 咖啡奶茶 | 美式/拿铁（纯奶），不要糖 | 注意咖啡因<200mg/日 |
