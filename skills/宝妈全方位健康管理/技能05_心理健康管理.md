## TL;DR

**心理健康核心原则 = 先保安全底线 + 再做筛查 + 最后给支持**

EPDS总分≥13分 → 建议心理/精神科就诊
第10题>0分（有自我伤害想法）→ 立即联系危机干预热线
压力和抑郁会双向加重血糖，形成恶性循环

---

## 1. GDM与产后抑郁的双向关系

### 统计数据

- GDM女性产后抑郁（PPD）患病率：**6.5%–48.4%**（研究差异大）
- 约 **15%–25%** 的GDM妈妈会在产后第一年经历抑郁或焦虑症状
- 抑郁会使糖尿病自我管理能力下降：血糖监测频率降低、饮食控制变差、运动依从性降低

### 恶性循环

```
高血糖 → 身体不适/疲劳 → 情绪低落
    ↓
抑郁/焦虑 → 皮质醇↑ → 胰岛素抵抗↑ → 血糖更难控
    ↓
睡眠差 → 情绪更差 → 血糖更高
```

## 2. EPDS筛查（爱丁堡产后抑郁量表）

### 使用说明

**过去7天内，你是否有以下感受？**（0-3分评分，0=从不，3=总是）

| 题号 | 问题 | 0 | 1 | 2 | 3 |
|------|------|---|---|---|---|---|
| 1 | 我能笑并看到事情有趣的一面 | 0 | 1 | 2 | 3 |
| 2 | 我以愉快的心情看待未来 | 0 | 1 | 2 | 3 |
| 3 | 当事情出错时，我会不必要地责备自己 | 3 | 2 | 1 | 0 |
| 4 | 我无缘无故地感到焦虑和担心 | 0 | 1 | 2 | 3 |
| 5 | 我无缘无故地感到害怕和惊慌 | 0 | 1 | 2 | 3 |
| 6 | 事情使我感到不知所措 | 0 | 1 | 2 | 3 |
| 7 | 我很不开心，以至于睡眠有困难 | 0 | 1 | 2 | 3 |
| 8 | 我感到悲伤和痛苦 | 0 | 1 | 2 | 3 |
| 9 | 我因心情不好而哭泣 | 0 | 1 | 2 | 3 |
| 10 | 我有伤害自己的想法 | 0 | 1 | 2 | 3 |

### 评分标准

| 总分 | 风险等级 | 建议 |
|------|---------|------|
| 0–9分 | 🟢 低风险 | 继续观察，保持心理自我监测 |
| 10–12分 | 🟡 中等风险 | 建议寻求专业心理支持，2周后复查EPDS |
| ≥13分 | 🔴 高风险 | 强烈建议心理/精神科就诊 |
| 第10题 > 0 | ⚠️ 极高风险 | 立即联系危机干预热线或就医 |

### ⚠️ 第10题特别说明

**第10题："我有伤害自己的想法"**

- **> 0分**（偶尔/有时/总是）：无论总分多少，属于极高风险
- 必须立即获得专业帮助
- 不要独自承受，告诉身边人或拨打危机热线

## 3. 压力来源快速识别

| 来源 | 表现 | 影响 |
|------|------|------|
| 照顾婴儿 | 疲惫、睡眠不足 | 生理应激+心理压力 |
| 血糖管理 | 焦虑、无力感 | 双重任务叠加压力 |
| 家庭关系 | 矛盾、失落感 | 情绪支持缺失 |
| 身份转变 | 自我认同混乱 | 存在价值感动摇 |
| 身体健康 | 身体形象、精力下降 | 自我效能感降低 |
| 社会支持 | 孤立、缺乏理解 | 孤独感加重抑郁 |

## 4. 压力缓解方案

### 即时减压（3分钟内可做）

**4-7-8呼吸法**：
```
吸气4秒 → 屏气7秒 → 呼气8秒，重复3–5次
原理：激活副交感神经，对抗"战斗或逃跑"反应
```

**5-4-3-2-1接地技术**（当感到焦虑/恐慌发作时）：
```
说出5样你看到的东西
4样你能摸到的东西
3样你能听到的声音
2样你能闻到的气味
1样你能尝到的味道
→ 把注意力从焦虑思绪拉回当下
```

**渐进式肌肉放松**：
从脚趾到头顶，逐部位收紧（5秒）→ 放松（10秒），感受紧张和放松的对比

### 中期方案（坚持1–4周）

| 方法 | 操作 | 频率 |
|------|------|------|
| 冥想 | Insight Timer/冥想App，10分钟引导冥想 | 每日 |
| 日记 | 记录3件当天感恩的事（哪怕很小）| 每日 |
| 户外散步 | 30分钟快走，接触自然光 | 每周≥3次 |
| 社交联系 | 与理解你的朋友/支持群体通话≥1次 | 每周 |
| 兴趣爱好 | 保留至少30分钟/天做自己喜欢的事 | 每日 |

### 就医指征

| 情况 | 建议 |
|------|------|
| EPDS ≥ 13分持续2周以上 | 建议心理/精神科就诊 |
| 第10题 > 0分 | 立即联系危机干预热线 |
| 无法正常照顾婴儿 | 建议心理/精神科就诊 |
| 严重影响家庭功能 | 建议家庭治疗+个体治疗 |

## 6. 配偶支持指南（供宝爸参考）

| 配偶做法 | 效果 |
|---------|------|
| "我听到你说……"（复述她的感受）| 让她感到被理解 |
| "你做得很好"（具体表扬）| 提升自我效能 |
| 分担换尿布/哄睡/洗澡等任务 | 减少她的非哺乳负担 |
| 陪同产检/复查 | 表达重视，让她不孤单 |
| 不说"你想太多了" / "你别这么敏感" | 避免让她感觉被否定 |
