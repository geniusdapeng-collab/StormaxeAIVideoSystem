## TL;DR

**并发症核心原则 = 早查早发现 + 血糖血压双控 + 预防为主**

HbA1c每3个月 | 血脂每6个月 | 眼底诊断时即查 | 尿微量白蛋白每6个月
有症状不等待复查时间表 → 立即就医

---

## 1. 并发症分类

### 微血管并发症（慢性高血糖损伤小血管）

| 类型 | 受累器官 | 发病机制 |
|------|---------|---------|
| 糖尿病视网膜病变 | 眼底血管 | 持续高血糖损伤视网膜微血管 |
| 糖尿病肾病 | 肾脏 | 高血糖损伤肾小球 |
| 糖尿病神经病变 | 周围神经 | 高血糖损伤神经纤维 |

### 大血管并发症（慢性高血糖损伤大血管）

| 类型 | 受累部位 | 发病机制 |
|------|---------|---------|
| 心血管疾病 | 心脏/冠状动脉 | 动脉粥样硬化加速 |
| 脑血管疾病 | 脑部血管 | 动脉粥样硬化+高血压 |
| 外周血管疾病 | 下肢血管 | 动脉粥样硬化导致供血不足 |

## 2. 筛查时间表

| 检查项目 | 首次时间 | 常规频率 | 目标值 |
|---------|---------|---------|--------|
| **HbA1c** | 诊断时 | 每3个月 | < 7.0%（大多数成人）|
| **空腹血糖/餐后血糖** | 诊断时 | 每日居家监测 | 空腹4.4–7.0 mmol/L |
| **血压** | 诊断时 | 每次就诊/每周自测 | < 130/80 mmHg |
| **血脂全套（TC/TG/LDL-C/HDL-C）** | 诊断时 | 每6个月 | LDL-C < 2.6 mmol/L（高危<1.8）|
| **尿微量白蛋白/ACR** | 诊断时 | 每6个月 | ACR < 3 mg/mmol |
| **估算肾小球滤过率（eGFR）** | 诊断时 | 每6–12个月 | > 60 mL/min/1.73m² |
| **眼底检查（散瞳）** | 诊断时即查 | 每1–2年 | 无视网膜病变 |
| **足部检查** | 诊断时 | 每次就诊/年 | 无溃疡/感染 |
| **神经检查（10g尼龙丝/音叉）** | 诊断时 | 每年 | 无明显异常 |
| **心电图（ECG）** | 诊断时 | 诊断时及每年 | 无明显异常 |
| **骨密度（DXA）** | 诊断时 | 每1–2年 | 无骨质疏松 |

**注**：HbA1c未达标或病程较长者，筛查频率应加密

## 3. 各类并发症早期信号

### 3.1 视网膜病变

| 分期 | 症状 | 严重程度 |
|------|------|---------|
| 早期（非增殖期）| 通常无明显症状，体检发现 | 轻 |
| 中期 | 视力模糊、眼前黑点（飞蚊症）| 中 |
| 晚期（增殖期）| 视野缺损、视力急剧下降、眼前有帘幕感 | 重 |
| 威胁视力 | 视网膜脱离、玻璃体出血 | 极重 |

**就医指征**：
- 突然视力下降
- 眼前闪光感
- 视野缺损（看东西有暗区）
- 眼前漂浮物突然增多

### 3.2 糖尿病肾病

| 分期 | 指标 | 说明 |
|------|------|------|
| 正常 | ACR < 3 mg/mmol，eGFR正常 | 无损伤 |
| 微量白蛋白尿 | ACR 3–30 mg/mmol | 早期肾病，可逆 |
| 大量白蛋白尿 | ACR > 30 mg/mmol | 临床肾病期 |
| 肾衰竭 | eGFR < 15 mL/min | 需透析或移植 |

**早期信号**：
- 尿液泡沫增多（蛋白尿）
- 眼睑/下肢水肿
- 血压难以控制

**就医指征**：尿蛋白试纸持续阳性，或 ACR > 3 mg/mmol

### 3.3 神经病变

| 类型 | 症状 | 部位 |
|------|------|------|
| 周围神经病变（最常见）| 手脚麻木/刺痛/灼热感/感觉减退 | 从脚趾/手指开始，对称分布 |
| 自主神经病变 | 消化不良（胃轻瘫）/ 体位性低血压/心率异常/性功能障碍 | 内脏 |
| 运动神经病变 | 肌肉无力/萎缩 | 少见 |

**就医指征**：
- 持续麻木影响睡眠
- 疼痛影响日常生活
- 足部感觉完全丧失（感觉不到热水/外伤）

### 3.4 心血管疾病

| 类型 | 症状 |
|------|------|
| 冠心病 | 运动后胸闷/胸痛（心绞痛），休息可缓解 |
| 心力衰竭 | 呼吸困难（平卧加重）、下肢水肿、极度疲劳 |
| 脑血管病 | 突然头晕/头痛、面臂腿无力、言语不清 |

**就医指征**：
- 胸痛/胸闷持续>5分钟不缓解
- 呼吸困难突然加重
- 任何突然的神经系统异常

## 4. HbA1c与并发症的关系

| HbA1c水平 | 平均血糖 | 并发症风险 |
|-----------|---------|----------|
| 6.0%–6.5% | 7.0–7.8 mmol/L | 低 |
| 6.5%–7.0% | 7.8–8.6 mmol/L | 轻度升高 |
| 7.0%–7.5% | 8.6–9.5 mmol/L | 中度升高 |
| 7.5%–8.0% | 9.5–10.5 mmol/L | 显著升高 |
| > 8.0% | > 10.5 mmol/L | 极高 |

**研究数据**：HbA1c每降低1%，微血管并发症风险降低37%，心肌梗死风险降低14%

## 5. 复查准备清单

每次复查前，整理以下信息：

```
【复查准备清单】
□ 最近血糖记录（30天SMBG日志或CGM报告）
□ 当前所有药物清单及剂量
□ 有无新出现的症状（眼/足/尿/心脏）
□ 最近一次各项检查的结果（HbA1c/血脂/尿蛋白等）
□ 想问医生的问题（≤3个）
□ 带好既往病历和检查报告
```
