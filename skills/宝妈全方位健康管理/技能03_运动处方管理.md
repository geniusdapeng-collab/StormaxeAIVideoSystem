## TL;DR

**运动核心原则 = 先评估后运动 + FITT渐进 + 血糖保驾护航**

有禁忌先就医，无禁忌从低强度开始
有氧每周≥150分钟（中等强度），抗阻每周≥2次
运动前血糖<5.0先进食，运动后及时补水和加餐

---

## 1. ACSM 2022 糖尿病运动推荐（基准）

| 维度 | 推荐 |
|------|------|
| 有氧频率 | ≥3天/周（间隔不超过2天）|
| 有氧时长 | ≥150分钟/周（每次≥10min，建议30–60min）|
| 有氧强度 | 中等强度（能说话但不能唱歌，RPE 12–13）|
| 抗阻频率 | ≥2天/周（非连续日）|
| 抗阻强度 | 8–15次/组，2–3组/动作，≥5种动作 |
| 柔韧+平衡 | 每周≥2–3次，每次≥10–30min |

### 运动强度自感判断（RPE量表）

| RPE | 主观感受 | 对应心率（参考）| 适合阶段 |
|-----|---------|--------------|---------|
| 6–7 | 很轻松 | <50%最大心率 | 启动期 |
| 8–9 | 轻松 | 50%–60%最大心率 | 启动期/恢复期 |
| 10–11 | 稍用力 | 60%–70%最大心率 | 适应期 |
| 12–13 | 中等用力（能说话不能唱歌）| 70%–75%最大心率 | 稳定期 |
| 14–15 | 较费力 | 75%–85%最大心率 | 提高期 |
| 16–20 | 非常费力 | >85%最大心率 | 一般不推荐 |

**最大心率估算**：220 - 年龄（岁）

## 2. 产后运动启动时间表

### 不同分娩方式的启动时间

| 分娩方式 | 启动时间 | 初始运动 |
|---------|---------|---------|
| 阴道分娩（无并发症）| 产后数天至1周 | 床旁站立、呼吸训练、盆底训练 |
| 阴道分娩（有会阴撕裂）| 产后2–4周 | 从轻度步行开始，避免盆底受力 |
| 剖宫产 | 产后6–8周 | 需医生评估后启动，从床旁站立开始 |

### 产后分阶段运动方案

**阶段1：启动期（产后1–2周）**

| 运动 | 时长/频率 | 强度 |
|------|---------|------|
| 床旁站立 | 5–10min，每日2–3次 | RPE 6–7 |
| 呼吸训练（腹式呼吸）| 5–10min，每日2次 | 很轻松 |
| 盆底训练（Kegel）| 每次收缩保持5–10s，10–15次/组，2组/日 | 轻度 |
| 室内步行 | 5–10min，每日2–3次（绕床/绕房间）| RPE 7–8 |

**阶段2：适应期（产后3–6周）**

| 运动 | 时长/频率 | 强度 |
|------|---------|------|
| 户外平地步行 | 15–20min，每日2次 | RPE 9–10 |
| 弹力带抗阻（上肢）| 1组×10次，每日1次 | 轻度 |
| 呼吸训练 | 10min，每日2次 | 轻松 |

**阶段3：进展期（产后6–12周）**

| 运动 | 时长/频率 | 强度 |
|------|---------|------|
| 快走/固定自行车 | 30min，每周3–5天 | RPE 12–13 |
| 抗阻训练（上肢+下肢）| 2组×8–12次，每周2天 | 中等 |
| 柔韧性训练 | 10–15min，每日或隔日 | 拉伸感无疼痛 |

**阶段4：稳定期（产后12周后）**

| 运动 | 时长/频率 | 强度 |
|------|---------|------|
| 中等强度有氧 | 30–45min/次，每周3–5天 | RPE 12–13 |
| 抗阻训练 | 2–3组×8–15次，每周2天（间隔日）| 中等 |
| 平衡训练（单脚站等）| 10–20min，每周2–3天 | 轻度 |

## 3. FITT运动处方模板

### 有氧运动FITT

| 维度 | 说明 |
|------|------|
| F（Frequency频率）| 3–5天/周，间隔不超过连续2天 |
| I（Intensity强度）| 中等强度（RPE 12–13，心率60%–75%最大心率）|
| T（Time时长）| 每次30–60min，从10min开始逐步增加 |
| T（Type类型）| 步行、固定自行车、游泳、椭圆机等 |

### 抗阻运动FITT

| 维度 | 说明 |
|------|------|
| F（Frequency频率）| 2–3天/周（非连续日）|
| I（Intensity强度）| 中等（8–15次/组，感觉用力但能完成）|
| T（Time时长）| 每次20–30min，≥5个不同肌群 |
| T（Type类型）| 自身体重、弹力带、轻哑铃（1–2kg起步）|

**主要肌群覆盖**：胸、背、肩、臂、核心（腹横肌）、髋、腿

## 4. 运动前血糖管理

### 运动前血糖决策

| 运动前血糖 | 处理 |
|-----------|------|
| < 5.0 mmol/L | 先进食15g碳水，等30min再运动，运动后测血糖 |
| 5.0–7.0 mmol/L | 先进食10g碳水，等15min再运动 |
| 7.0–10.0 mmol/L | 正常运动，可正常哺乳 |
| 10.0–13.9 mmol/L | 以轻度运动为主（如步行），避免高强度 |
| > 13.9 mmol/L | 暂缓运动，排查原因（饮食/药物/应激），等血糖降至<13.9再运动 |

### 运动前加餐建议（10–15g碳水）

| 加餐 | 碳水含量 |
|------|---------|
| 1个小苹果（150g）| 约15g |
| 4块全麦饼干 | 约15g |
| 1小盒酸奶（150g）| 约12g |
| 半根香蕉 | 约15g |

## 5. 哺乳期运动注意事项

### 运动与哺乳的时间安排

| 时机 | 哺乳建议 |
|------|---------|
| 运动前 | 哺乳后排空乳房，减少运动时溢乳不适 |
| 运动后 | 等待30min再哺乳（运动后乳酸短暂升高）|
| 运动后立即 | 如乳房胀满，可挤出部分前奶，30min后再哺乳 |

### 乳酸问题

- 剧烈运动（>80%最大心率，持续>60min）后，乳汁乳酸会在30–90min内升高
- 中等强度运动对乳汁乳酸无显著影响
- 对大多数婴儿无害，但部分婴儿可能不喜欢运动后的乳汁味道

### 运动后补水

运动中每15–20min补充100–200mL水
运动后根据体重下降情况补水（每减重1kg补1000mL）
哺乳期额外补水：+500–700mL/日（结合运动强度）

## 6. 运动禁忌症

### 绝对禁忌症（需医生评估后才能运动）

| 禁忌情况 | 说明 |
|---------|------|
| 增殖期视网膜病变（PDR）| 任何屏气/Valsalva动作可能导致眼压骤变 |
| 严重心血管疾病（不稳定心绞痛、心衰）| 运动可能诱发心血管事件 |
| 严重高血压（>180/110mmHg）| 运动进一步升高血压 |
| 严重神经病变（足部溃疡/坏疽）| 运动加重足部损伤 |
| 急性全身感染/发热 | 加重代谢紊乱 |
| DKA/HHS急性期 | 需先纠正代谢紊乱 |

### 相对禁忌症（需医生评估后决定运动方案）

| 情况 | 说明 |
|------|------|
| HbA1c > 9% | 从极低强度开始，避免高强度运动 |
| 频繁低血糖（>3次/周）| 运动前加餐优化，睡前血糖监测 |
| 妊娠期高血压史 | 避免仰卧位运动（压迫下腔静脉）|
| 产后腹直肌分离>2指 | 避免平板支撑、仰卧起坐等腹压动作 |
| 耻骨联合分离 | 避免髋外展/内收运动 |

## 7. 渐进式4周运动方案（产后12周后/无禁忌者）

**第1周**（建立习惯）

```
运动：每日2次 × 10min 室内步行（绕客厅/走廊）
时间：早晨起床后 + 下午
强度：RPE 8–9（轻松）
目标：适应每日运动的节奏，不追求强度
```

**第2周**（逐步增加时长）

```
运动：每日2次 × 15–20min 户外步行
时间：早晨 + 傍晚
强度：RPE 9–10（稍用力）
加入：弹力带手臂抗阻 1组 × 10次
```

**第3周**（提升强度）

```
运动：每日30min 快走
时间：早晨或傍晚，选择固定时间
强度：RPE 12–13（能说话不能唱歌）
加入：弹力带上下肢抗阻 2组 × 10次
```

**第4周**（稳定维持）

```
运动：每日30–40min 有氧 + 每周2次抗阻训练
有氧：快走/固定自行车，RPE 12–13
抗阻：2–3组 × 8–12次，覆盖5个肌群
```

## 8. 运动降糖效果

| 运动类型 | HbA1c改善 | 效果持续时间 |
|---------|-----------|------------|
| 有氧运动 | -0.5% to -0.8% | 2–48小时（最后一次运动后）|
| 抗阻运动 | -0.3% to -0.5% | 24–72小时 |
| 有氧+抗阻联合 | -0.8% to -1.2% | 持续改善 |

**注意**：运动降糖效果是可累积的，坚持3个月以上才能看到HbA1c的显著改善
