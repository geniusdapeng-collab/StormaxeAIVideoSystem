## 技能包简介

本技能包包含10个健康维度的专属技能，覆盖糖妈妈日常健康的方方面面：

| # | 技能 | 文件名 | 核心价值 |
|---|------|--------|---------|
| 01 | 血糖监测管理 | 技能01_血糖监测管理.md | 记录、分析、预警 |
| 02 | 饮食营养管理 | 技能02_饮食营养管理.md | 个性化餐单、GI/GL |
| 03 | 运动处方管理 | 技能03_运动处方管理.md | FITT方案、哺乳期安全 |
| 04 | 睡眠作息管理 | 技能04_睡眠作息管理.md | 皮质醇、睡眠质量 |
| 05 | 心理健康管理 | 技能05_心理健康管理.md | EPDS筛查、压力缓解 |
| 06 | 用药提醒管理 | 技能06_用药提醒管理.md | 安全性速查、哺乳期用药 |
| 07 | 并发症预防管理 | 技能07_并发症预防管理.md | 筛查时间表、早期信号 |
| 08 | 哺乳期特殊护理 | 技能08_哺乳期特殊护理.md | 哺乳与血糖平衡 |
| 09 | 家人支持协调 | 技能09_家人支持协调.md | 配偶赋能、家庭分工 |
| 10 | 复查提醒管理 | 技能10_复查提醒管理.md | HbA1c、眼底等复查 |

## 快速开始

**血糖高了/低了** → 技能01  
**不知道吃什么** → 技能02  
**想运动但不确定安全** → 技能03  
**睡眠不好** → 技能04  
**心情不好/焦虑** → 技能05  
**用药有疑问** → 技能06  
**不知道该复查什么** → 技能10  
**哺乳有疑问** → 技能08

## 核心原则

1. **哺乳优先**：所有建议不能损害哺乳和乳汁质量
2. **证据驱动**：基于《中国2型糖尿病防治指南2024》《ADA Standards of Care 2024》等高等级证据
3. **安全底线**：急性并发症（DKA/HHS）识别与处理是每个技能的底线
4. **全家协同**：健康管理不是宝妈一个人的事
