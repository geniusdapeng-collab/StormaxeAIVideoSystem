**家庭支持核心原则 = 明确分工 + 精准沟通 + 配偶赋能 + 寻求外部支持**

## 3. 与配偶有效沟通框架

### 原则：非暴力沟通（NVC）

```
观察 → 感受 → 需求 → 请求
```

### 场景1：希望配偶分担夜间哺乳

**❌ 不用说（引发防御）**：
"你天天睡得像猪一样，我累死了！"

**✅ 应该说（具体请求）**：
"宝宝昨晚10点和凌晨2点各醒一次，我这两周总共只睡了4小时/天，血糖都控制不好了。如果你能用奶瓶喂一次（我提前挤出来），我可以多睡1–2小时，对我血糖控制很重要。我们可以一起看看怎么安排。"

### 场景2：希望配偶支持饮食管理

**❌ 不用说**：
"你做的饭太难吃了，这个不能吃那个不能吃。"

**✅ 应该说**：
"医生说我的情况和普通糖尿病不一样，需要按高纤维、低GI的原则来吃。如果你能帮我参考这个原则来买菜做饭，我血糖会控制得好很多。我们可以一起研究几个低GI食谱，你来挑喜欢的菜，我来定原则。"

### 场景3：配偶觉得"没什么大不了的"

**❌ 不用说**：
"你不懂，别说了。"

**✅ 应该说**：
"医生说我的空腹血糖已经持续12+有7个月了，再不控制会有眼睛、肾脏这些并发症。我的情况和普通糖尿病还不一样，哺乳期更难处理。我需要你和我一起认真对待这件事，不只是帮忙做家务，而是真正理解这个状态有多难。"

### 场景4：配偶想帮忙但不知道做什么

**直接给清单**：
```
你能帮我做的事（选1–2件你愿意做的）：
□ 每天洗碗（让我有时间测血糖/吃饭）
□ 晚上给宝宝换尿布（让我哺乳后能马上睡）
□ 帮我记血糖记录（我念你写）
□ 每周陪我散步30分钟（一举两得）
□ 提醒我按时吃药
□ 周末让我补觉2小时，你带娃
```

## 4. 与祖辈协作要点

### 基本原则

| 原则 | 说明 |
|------|------|
| 明确核心分工 | 哺乳以宝妈为主，抚育和家务由祖辈协助 |
| 尊重代际差异 | 观念不同是正常的，不强求改变 |
| 聚焦安全底线 | 食品安全、运动安全、婴儿护理安全等原则必须坚持 |
| 避免正面冲突 | 通过配偶间接沟通，不要直接对质 |
| 具体感激表达 | 不是笼统的"谢谢"，而是具体说"谢谢妈帮我做了XX" |

### 安全底线清单（必须坚持）

| 类别 | 具体要求 |
|------|---------|
| 食品安全 | 不给婴儿添加糖/盐；辅食制作符合月龄 |
| 运动安全 | 不剧烈摇晃婴儿；婴儿运动符合发育阶段 |
| 血糖管理 | 不给婴儿喂含糖食物；理解哺乳期妈妈需要规律饮食 |
| 哺乳支持 | 不催促断奶；支持宝妈的哺乳决定 |

### 祖辈带娃常见分歧处理

| 分歧 | 处理方式 |
|------|---------|
| 婴儿喂养观念不同 | 通过配偶沟通，以科学依据说明 |
| 给婴儿额外喂水/糖 | 温和说明WHO和儿科指南不建议 |
| 过度包办 | 具体解释为什么需要让婴儿自己探索 |
| 干涉断奶时机 | 明确这是宝妈的决定 |
