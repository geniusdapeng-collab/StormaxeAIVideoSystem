# 技能01：血糖监测管理

## 核心功能

1. **血糖数据记录**：接收 SMBG 数据，存入记录体系
2. **趋势分析**：分析7天/30天血糖趋势，给出趋势判断
3. **异常识别**：识别高血糖/低血糖/血糖波动过大，给出等级判断
4. **处理建议**：根据异常等级给出居家处理或就医建议
5. **就医准备**：生成血糖报告供内分泌科就诊使用

## 血糖控制目标（哺乳期T2DM）

| 时点 | 目标范围 | 说明 |
|------|---------|------|
| 空腹 | 4.4–7.0 mmol/L | 每日必测 |
| 餐后1h | < 10.0 mmol/L | 评估餐后峰值 |
| 餐后2h | 4.4–9.0 mmol/L | 哺乳期略放宽 |
| 睡前 | 6.0–10.0 mmol/L | 避免夜间低血糖 |
| 凌晨 | 4.4–7.0 mmol/L | 警惕无症状低血糖 |

## SMBG 频率建议

| 情况 | 频率 |
|------|------|
| 单纯饮食/口服药控制 | 每周2–4次（空腹+餐后） |
| 胰岛素治疗 | 每日多点（空腹+三餐后+睡前） |
| 血糖不稳定期 | 每日7点法（空腹+三餐前后+睡前） |
| 哺乳前/后 | 必要时加测 |

## 异常处理决策树

```
血糖值 → 判断等级 → 处理
                       
FBG > 16.7 mmol/L    → 🔴极高危 → 立即就医（排查DKA/HHS）
FBG 11.1–16.7 mmol/L → 🟠高危   → 4h内就医，增加监测
FBG 7.0–11.1 mmol/L  → 🟡中危   → 当日饮食复盘，次日复查
FBG < 3.9 mmol/L     → 🔴低血糖 → 15-15规则处理
FBG 3.9–7.0 mmol/L   → 🟢正常   → 继续保持
```

## 低血糖「15-15规则」

1. 立即口服15–20g速效碳水（4颗葡萄糖片/150mL果汁/1汤匙蜂蜜/3块方糖）
2. 等待15分钟复测
3. 如仍≤3.9 mmol/L，重复一次
4. 血糖恢复后进食含蛋白质的小食

## 急性并发症识别红线

| 症状 | 可能情况 | 紧急度 |
|------|---------|--------|
| 极度口渴、多尿、脱水 | HHS前驱 | 🔴极高 |
| 恶心呕吐、腹痛、呼吸深快 | DKA早期 | 🔴极高 |
| 嗜睡、意识改变 | 严重HHS/DKA | 🔴极高 |
| FPG持续>11.1 | 严重高血糖 | 🟠高 |

## 使用限制

- ❌ 不提供降糖药物剂量调整建议
- ❌ 不替代医生诊断
- ⚠️ FBG > 16.7 mmol/L 或有不适症状 → 立即就医
