## TL;DR

**哺乳核心原则 = 哺乳降血糖 + 高血糖不损乳汁 + 急性乳腺炎立即就医**

哺乳本身每天消耗约500kcal，相当于温和运动
高血糖不影响乳汁基本营养成分，继续哺乳对母婴都有益
涨奶→增加哺乳/挤奶；乳腺炎→立即就医+继续哺乳（除非重度）

## 1. Q&A：血糖与哺乳

### Q：血糖高能哺乳吗？

**A：能，且应该继续哺乳。**

- 哺乳本身可降低母亲T2DM风险30%–50%（大规模研究证实）
- 高血糖**不影响**乳汁的基本营养成分（蛋白质、脂肪、乳糖、免疫球蛋白含量不受血糖影响）
- 胰岛素是安全的，**不进入乳汁**
- 继续哺乳对婴儿的代谢和免疫系统都有长期保护作用
- 哺乳对婴儿肠道菌群建立、代谢健康、免疫力都有正面影响

### Q：哺乳会影响血糖吗？

**A：会，哺乳有降血糖作用，但夜间哺乳需防低血糖。**

| 影响 | 说明 |
|------|------|
| 降血糖 | 哺乳每天额外消耗约300–500kcal，相当于温和运动 |
| 改善胰岛素敏感性 | 催乳素升高可暂时改善胰岛素敏感性 |
| 夜间低血糖风险 | 夜间哺乳后血糖可能偏低（尤其胰岛素治疗者）|

**夜间防低血糖方案**：
- 哺乳前测血糖，< 5.0 mmol/L先进食15g碳水
- 床旁常备葡萄糖片
- 配偶知道低血糖识别方法

### Q：哺乳期降糖药怎么选？

**A：胰岛素首选，二甲双胍安全。**

详见技能06——用药提醒管理。

## 2. 涨奶/堵奶处理

### 涨奶严重程度分级

| 严重程度 | 表现 | 感觉 |
|---------|------|------|
| 轻度 | 乳房微胀，乳汁流畅 | 轻微不适，婴儿可正常吸出 |
| 中度 | 乳房胀满，明显变硬 | 胀痛，婴儿吸吮费力 |
| 重度 | 乳房硬如石头，皮肤紧绷发红 | 剧烈疼痛，可能伴发热 |

### 涨奶处理方案

| 严重程度 | 核心措施 |
|---------|---------|
| 轻度 | 继续按需哺乳，冷敷（卷心菜叶/冷敷袋）|
| 中度 | 增加哺乳次数（每2–3小时一次），哺乳前轻柔按摩乳房，手挤奶排空部分乳汁 |
| 重度 | 继续哺乳（婴儿是最强吸乳器），手挤奶排空，联系哺乳顾问或就医 |

### 涨奶预防措施

- **按需哺乳**：不要攒奶，乳房胀满就喂
- **每次哺乳后**：如有剩余，挤出来储存
- **哺乳姿势**：确保婴儿含接正确（有效吸吮）
- **哺乳内衣**：支撑良好但不过紧
- **避免**：突然断奶、哺乳间隔过长、压迫乳房

## 3. 乳腺炎识别与处理

### 乳腺炎早期信号

| 信号 | 说明 |
|------|------|
| 乳房局部红、肿、热、痛 | 炎症反应 |
| 发热（>38.5°C）| 全身反应 |
| 全身不适/肌肉酸痛/寒战 | 类似流感症状 |
| 乳汁变化 | 颜色/味道改变（有时宝宝会拒绝吃）|

### 乳腺炎处理

| 情况 | 处理 |
|------|------|
| 疑似早期 | 继续频繁哺乳/挤奶，休息，补充水分，必要时用退热药（对乙酰氨基酚）|
| 24–48小时无改善 | 立即就医（可能需要抗生素）|
| 重度（高热+乳房大面积红肿）| 急诊就医（排查化脓性乳腺炎）|

**重要**：乳腺炎时**应该继续哺乳**（除非重度化脓），婴儿的吸吮是最好的疏通方式。停乳会让乳腺炎加重。

**化脓性乳腺炎（需外科处理）信号**：乳房局部有波动感（好像有脓液）、高热持续不退、乳房皮肤紫暗。

## 4. 怎么知道奶够不够

### 婴儿摄入充足的信号

| 指标 | 判断 |
|------|------|
| 尿布 | 每日湿尿布6–8片（7月龄婴儿）|
| 大便 | 大便次数减少（可能几天一次），颜色金黄 |
| 体重 | 每月体重增长约0.5–1kg（7月龄婴儿生长速度放缓）|
| 精神状态 | 精神良好，醒着时活跃 |
| 吃奶表现 | 有效吞咽，能主动松开乳头 |

### 增加乳汁分泌的方法

| 方法 | 说明 |
|------|------|
| 增加哺乳频率 | 婴儿吸吮是泌乳最强刺激 |
| 确保充分含接 | 有效吸吮才能有效泌乳 |
| 充足水分 | 每日≥2000mL |
| 充足热量 | 不<1800kcal/日 |
| 充足蛋白质 | ≥70–80g/日 |
| 充分休息 | 疲劳会抑制泌乳素 |
| 哺乳后挤奶 | 排空乳房=增加产奶 |

## 5. 断奶时机与方案

### 断奶时机建议

| 月龄 | 考量 |
|------|------|
| 6个月内 | WHO推荐纯母乳6月+持续母乳至2岁+ |
| 7–12个月 | 已添加辅食，母乳仍应是重要营养来源 |
| 12个月+ | 可根据母亲血糖控制、生活质量综合决定 |
| 血糖控制差 | 先控糖，不必急于断奶 |

### 哺乳期减重注意事项

- 减重**不要过快**（>2kg/月会影响乳汁量）
- 哺乳期热量不<1800kcal
- 减重目标是改善代谢，不是快速瘦身

### 温和断奶方案（推荐）

| 周次 | 方法 |
|------|------|
| 第1–2周 | 逐渐减少一顿哺乳（从白天开始），用挤出的母乳瓶喂替代 |
| 第3–4周 | 再减少一顿，引入配方奶或牛奶（12月龄后可用纯牛奶）|
| 第5–6周 | 继续减少，直到完全离乳 |
| 全程 | 观察婴儿情绪，给予更多拥抱和陪伴 |

**不要**：突然断奶（母婴都不适）、在乳头上涂辣椒/苦瓜（心理创伤）

## 6. 乳汁质量保障

| 要素 | 每日需求 | 说明 |
|------|---------|------|
| 热量 | ≥1800kcal/日 | 低于此影响乳汁分泌 |
| 蛋白质 | ≥70–80g/日 | 保障乳汁蛋白质量 |
| 水分 | ≥2000mL/日 | 哺乳期需水量增加 |
| DHA | 200–300mg/日 | 深海鱼或藻油补充 |
| 钙 | 1000mg/日 | 奶制品/豆制品/芝麻 |

**禁忌**：
- ❌ 饮酒（酒精会进入乳汁）
- ⚠️ 咖啡因 < 200mg/日（约2杯咖啡）
