# TL;DR

按场景分类的亲子沟通话术模板，帮你学会用正确的方式夸奖、批评、询问、共情，建立良好的亲子沟通习惯。

# 场景分析

## 典型场景
很多父母不知道怎么跟孩子说话：想夸奖但不知道怎么夸，想批评但一开口就是"不许xxx"，想问孩子事情但问不出话......错误的沟通方式会影响亲子关系和孩子的心理健康。

## 核心挑战
1. **夸奖失当**：只会说"真棒"，导致孩子只做给大人看
2. **批评打击**：用否定式语言伤害孩子自尊
3. **询问困难**：问什么都回答"不知道"
4. **情感疏离**：不知道跟孩子说什么

## 正确认知
- **语言输入是孩子语言发展的关键**
- **怎么说是孩子学习的模板**
- **沟通质量比时长更重要**

# 专业建议

## 夸奖话术（场景化模板）

### 原则
- **夸过程不夸结果**：努力>聪明
- **夸具体不夸笼统**：说清楚好在哪
- **夸行为不夸天赋**：你很努力>你真聪明

### 话术模板

#### 场景1：当孩子完成一件事时

❌ 错误：
> "真棒！"
> "你最厉害了！"

✅ 正确：
> "你自己把积木搭好了，你一定很努力对不对？"
> "这幅画用了好多颜色，你是怎么想到的？"

#### 场景2：当孩子有进步时

❌ 错误：
> "你怎么现在才会"

✅ 正确：
> "我看到你这次比上次多坚持了5分钟，你是故意练习了对吗？"

#### 场景3：当孩子帮助别人时

✅ 正确：
> "你把玩具分给小朋友，他们一定很开心，你是个很会分享的宝贝。"

### 夸婴儿的话术（0-1岁）

虽然婴儿听不懂，但**语言输入很重要**：

| 场景 | 推荐话术 |
|------|----------|
| 换尿布 | "我们来换尿布啦，干干净净好舒服" |
| 喂奶 | "饿了对吗，妈妈/爸爸给你喂neinei" |
| 抱起来 | "来，妈妈/爸爸抱抱" |
| 洗澡 | "洗澡澡啦，水温刚刚好" |
| 户外 | "今天天气真好，阳光暖暖的" |

**原则**：你在做什么就说什么，这是最好的语言输入

## 批评话术（场景化模板）

### 原则
- **描述行为不说人格**：这件事做错了≠你是个坏孩子
- **表达感受不攻击**：我很难过≠你怎么这么烦
- **给出选择不命令**：可以这样做≈必须这样做

### 话术模板

#### 场景1：当孩子做错事时

❌ 错误：
> "你怎么这么笨！"
> "不许这样！"

✅ 正确：
> "我看到你把水洒在地上了，地滑会让人摔倒。我们一起擦干净好吗？"
> "我知道你想玩水，但不是在墙上画。下次想画画我们用纸好吗？"

#### 场景2：当孩子打人时

❌ 错误：
> "你不许打人！"
> "再打我打你！"

✅ 正确：
> "我看到你推了小朋友，他摔倒了一定很疼。"
> "我们用手做什么会让人开心？对了，握手！来，我们握握手。"

#### 场景3：当孩子说谎时

❌ 错误：
> "你竟然撒谎！"
> "小孩子不能撒谎！"

✅ 正确：
> "我看到杯子碎了，是你打碎的吗？没关系，告诉我真相，我们一起解决。"
> "我不生气，我只是想知道发生了什么。"

## 询问话术（场景化模板）

### 原则
- **开放式问题**比"是不是"好
- **观察式开场**：我看到......我注意到......
- **给孩子选择**：你想A还是B

### 话术模板

#### 场景1：放学后/见面时

❌ 错误：
> "今天在幼儿园/家里怎么样？"
> "乖不乖？"

✅ 正确：
> "今天有什么开心的事想告诉我吗？"
> "我看到你书包里有幅画，能给我讲讲吗？"

#### 场景2：睡前

✅ 正确：
> "今天最开心的事是什么？"
> "今天有什么困难吗？你是怎么解决的？"
> "如果今天可以重来，你想改变什么？"

#### 场景3：玩游戏时

✅ 正确：
> "你在搭什么？"
> "这个城堡里住着谁？"
> "你想让我做什么角色？"

### 问婴儿的话术

对5月龄的香香，你可以说：

| 场景 | 推荐话术 |
|------|----------|
| 看到她笑 | "香香笑了，是不是很开心呀？" |
| 她发出声音 | "哦～你在说话呢～爸爸听见了" |
| 给她看东西 | "你看，这是小鸭子，嘎嘎嘎" |
| 换衣服 | "我们来换新衣服啦" |

## 共情话术（场景化模板）

### 原则
- **先接纳情绪，再解决问题**
- **说出孩子的感受**
- **不要否定孩子的感受**

### 话术模板

#### 场景1：孩子摔跤哭了

❌ 错误：
> "不疼不疼，不哭"
> "没事没事"
> "都怪这个地！"

✅ 正确：
> "摔疼了对不对？爸爸看到了，你一定很疼。来，爸爸抱抱。"

#### 场景2：孩子害怕

✅ 正确：
> "你害怕大灰狼吗？我知道你害怕，妈妈/爸爸在这里保护你。"

#### 场景3：孩子失望

✅ 正确：
> "你想去公园但是下雨了对吗？你一定很失望。我也觉得很可惜。"

#### 场景4：孩子生气

✅ 正确：
> "我看到你很生气。你想搭的积木倒了，真的很让人沮丧对不对？"

## 指令话术（场景化模板）

### 原则
- **简短清晰**：一句话不超过10个字
- **正面表达**：告诉做什么，而不是不做什么
- **给选择**：在可接受范围内让孩子选

### 话术模板

#### 场景1：叫孩子吃饭

❌ 错误：
> "不要玩了！"
> "还不来吃饭！"

✅ 正确：
> "香香，吃饭啦。"
> "你是想自己走过来还是让爸爸抱过来？"

#### 场景2：让孩子收拾玩具

❌ 错误：
> "你怎么又不收拾！"

✅ 正确：
> "我们一起收拾玩具吧。你先收这一堆，我收那一堆，看谁先收完？"

## 日常话术建议

### 五大"魔法句"

| 类型 | 推荐句式 |
|------|----------|
| 感谢 | "谢谢你帮妈妈/爸爸......" |
| 请求 | "你能帮妈妈/爸爸......吗？" |
| 道歉 | "对不起，妈妈/爸爸刚才......" |
| 表扬 | "我看到你......你一定很......" |
| 好奇 | "我很好奇你是怎么想到的？" |

### 每日亲子对话清单

周末陪香香时，可以这样说：

| 时段 | 推荐话术 |
|------|----------|
| 早上 | "香香，早上好！睡得好吗？" |
| 换尿布 | "我们洗干净，香香好舒服" |
| 喂奶 | "neinei time～香香饿了对吗" |
| 户外 | "你看，树叶在动，风吹过来的" |
| 傍晚 | "今天太阳下山了，天黑了" |
| 洗澡 | "洗澡澡啦～香香喜欢水对不对" |
| 睡前 | "天黑了，香香要睡觉了，晚安" |

## 破坏亲子关系的"禁语"清单

### 绝对禁止
| 禁语 | 伤害 |
|------|------|
| "你怎么这么笨" | 打击自尊 |
| "不要你了" | 破坏安全感 |
| "你不听话我就不喜欢你了" | 有条件的爱 |
| "看看别人家孩子" | 比较伤害 |
| "我数三下" | 威胁式教育 |
| "你怎么这么烦" | 否定感受 |
| "男孩子/女孩子不能哭" | 压抑情绪 |

### 应该替换为

| 禁语 | 替换 |
|------|------|
| "你怎么这么笨" | "这件事有点难，我们一起想想办法" |
| "不要你了" | "我有点生气，但我永远爱你" |
| "看看别人" | "你上次......做得很好" |
| "数三下" | "等你准备好了我们就......" |

# 质量检查

## 话术自检
- [ ] 我今天夸奖时说了具体的事情吗？
- [ ] 我批评时只说了行为，没有否定人格吗？
- [ ] 我有没有用威胁/命令的语气？
- [ ] 我有没有否定孩子的感受？

## 记住
- **你说话的方式就是孩子学习的方式**
- **先处理情绪，再解决问题**
- **接纳感受不是纵容行为**
- **每一天都是练习的机会**
