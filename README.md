# Stormaxe AI Video System

AI Video Generation System powered by ShanhaiStory Forge. End-to-end pipeline from story to Seedance rendering.

## Overview

Stormaxe AI is a comprehensive AI video generation system built around the ShanhaiStory Forge pipeline. It provides:

- **Story Generation**: AI-powered narrative creation with character development
- **Shot Design**: Cinematic shot planning and composition
- **Prompt Engineering**: Optimized prompts for Seedance rendering
- **Rendering Pipeline**: End-to-end video generation workflow
- **Quality Assurance**: Automated quality checks and compliance

## Skills

The system is organized as a collection of specialized skills:

### Core Pipeline
- `shanhaijing-director` - Main pipeline orchestrator
- `shanhaijing-story-engine` - Narrative generation
- `shanhaijing-shot-design` - Shot planning
- `shanhaijing-render-engine` - Rendering workflow

### Content Creation
- `shanhaijing-bestiary` - Beast/creature design
- `shanhaijing-character-manager` - Character management
- `shanhaijing-world-engine` - World building

### Quality & Compliance
- `shanhaijing-quality-oracle` - Quality assessment
- `shanhaijing-promptforge` - Prompt optimization
- `shanhaijing-narrative-consistency` - Story consistency checks
