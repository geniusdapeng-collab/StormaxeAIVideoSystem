## 1. 世界观核心设定 Worldview Core

Nirath（奈拉斯），地球在46亿年前的原始形态。这是一颗被双恒星系统眷顾的生命摇篮——体积约为地球的1.7倍，自转周期28小时，表面71%被发光海洋覆盖。与潘多拉不同，Nirath不是外星异域，而是我们星球被遗忘的"童年"。

这里的"山海"并非比喻——大陆板块如同沉睡巨兽的脊背隆起，海洋是液态星光，而大气中漂浮的孢子群在黄昏时分会形成类似《山海经》中所述"五色祥云"的光晕现象。

> **视觉锚点**: 阿凡达潘多拉级别的生物发光 + 地球原始太古代的海洋与火山 + 《山海经》神兽图腾的抽象形态融入地貌

## 15. 核心美学原则 Core Aesthetic Principles

> **"这不是外星世界。这是地球在做梦——梦见自己46亿年前的样子，梦见那些后来被写成神话的真实。"**

1. **熟悉中的陌生**: 每个场景都应有可识别的地质形态（海洋、山脉、草原、洞穴），但其材质、尺度和发光属性必须超越地球经验
2. **科学中的诗意**: 生物发光、超导悬浮、超临界流体——所有奇观基于真实科学概念的夸张延伸
3. **神话的地质学**: 《山海经》的每一则记述都对应一个地质或天文现象的真实起源
4. **光的叙事**: 在Nirath，光不是照明工具——光是生命，是记忆，是尚未诞生的神话
